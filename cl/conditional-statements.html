Introduction

Conditional statements are fundamental in programming as they allow the execution of code based on specific conditions. In CL (Control Language), conditional statements help control the flow of a program by making decisions. This section will cover the basics to advanced usage of conditional statements in CL.

Basic Conditional Statements

IF Statement

The IF statement is used to execute a block of code if a specified condition is true.

Syntax

IF COND(logical-expression) THEN(DO)
    /* Code to execute if condition is true */
ENDDO

Example

DCL VAR(&AGE) TYPE(*DEC) LEN(3 0)
CHGVAR VAR(&AGE) VALUE(20)

IF COND(&AGE >= 18) THEN(DO)
    SNDPGMMSG MSG('You are an adult.')
ENDDO

In this example, the message "You are an adult." is sent if the variable &AGE is 18 or older.

ELSE Statement

The ELSE statement is used to execute a block of code if the condition in the IF statement is false.

Syntax

IF COND(logical-expression) THEN(DO)
    /* Code to execute if condition is true */
ENDDO
ELSE DO
    /* Code to execute if condition is false */
ENDDO

Example

DCL VAR(&AGE) TYPE(*DEC) LEN(3 0)
CHGVAR VAR(&AGE) VALUE(16)

IF COND(&AGE >= 18) THEN(DO)
    SNDPGMMSG MSG('You are an adult.')
ENDDO
ELSE DO
    SNDPGMMSG MSG('You are a minor.')
ENDDO

In this example, the message "You are a minor." is sent if the variable &AGE is less than 18.

Advanced Conditional Statements

ELSEIF Statement

The ELSEIF statement allows for multiple conditions to be checked in sequence.

Syntax

IF COND(logical-expression1) THEN(DO)
    /* Code to execute if condition1 is true */
ENDDO
ELSEIF COND(logical-expression2) THEN(DO)
    /* Code to execute if condition2 is true */
ENDDO
ELSE DO
    /* Code to execute if none of the above conditions are true */
ENDDO

Example

DCL VAR(&AGE) TYPE(*DEC) LEN(3 0)
CHGVAR VAR(&AGE) VALUE(20)

IF COND(&AGE < 13) THEN(DO)
    SNDPGMMSG MSG('You are a child.')
ENDDO
ELSEIF COND(&AGE < 18) THEN(DO)
    SNDPGMMSG MSG('You are a teenager.')
ENDDO
ELSE DO
    SNDPGMMSG MSG('You are an adult.')
ENDDO

In this example, the message "You are an adult." is sent if the variable &AGE is 18 or older.

Nested IF Statements

Nested IF statements are IF statements placed inside another IF statement to check multiple conditions.

Syntax

IF COND(logical-expression1) THEN(DO)
    IF COND(logical-expression2) THEN(DO)
        /* Code to execute if both conditions are true */
    ENDDO
ENDDO

Example

DCL VAR(&AGE) TYPE(*DEC) LEN(3 0)
DCL VAR(&MEMBER) TYPE(*CHAR) LEN(1)
CHGVAR VAR(&AGE) VALUE(20)
CHGVAR VAR(&MEMBER) VALUE('Y')

IF COND(&AGE >= 18) THEN(DO)
    IF COND(&MEMBER = 'Y') THEN(DO)
        SNDPGMMSG MSG('You are an adult member.')
    ENDDO
ENDDO

In this example, the message "You are an adult member." is sent if the variable &AGE is 18 or older and &MEMBER is 'Y'.

Summary

Conditional statements in CL are crucial for controlling the flow of a program. Starting with basic IF and ELSE statements, you can create complex decision-making structures using ELSEIF and nested IF statements. Understanding and mastering these constructs will significantly enhance your ability to write efficient and effective CL programs.
