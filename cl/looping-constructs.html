Introduction to Looping Constructs

Looping constructs are essential in programming as they allow you to execute a block of code multiple times. In CL, looping constructs help automate repetitive tasks, making your scripts more efficient and easier to manage.

Types of Looping Constructs in CL

In CL, there are primarily two types of looping constructs:

  • DO Loop
  • DOWHILE Loop

DO Loop

The DO loop is used to execute a block of code a specific number of times. It is useful when you know in advance how many times you want to execute the loop.

Syntax

DO var FROM start TO end BY step
    // Code to be executed
ENDDO

Example

DCL VAR(&I) TYPE(*INT) LEN(4)
DO VAR(&I) FROM(1) TO(5) BY(1)
    SNDPGMMSG MSG('Iteration ' *CAT &I)
ENDDO
  • Explanation:
    • DCL VAR(&I) TYPE(*INT) LEN(4): Declares a variable &I of type integer.
    • DO VAR(&I) FROM(1) TO(5) BY(1): Starts a loop with &I ranging from 1 to 5, incrementing by 1.
    • SNDPGMMSG MSG('Iteration ' *CAT &I): Sends a program message concatenating the string 'Iteration ' with the current value of &I.
    • ENDDO: Ends the loop.

DOWHILE Loop

The DOWHILE loop executes a block of code as long as a specified condition is true. It is useful when the number of iterations is not known in advance and depends on a condition.

Syntax

DOWHILE (condition)
    // Code to be executed
ENDDO

Example

DCL VAR(&I) TYPE(*INT) LEN(4) VALUE(1)
DOWHILE (&I <= 5)
    SNDPGMMSG MSG('Iteration ' *CAT &I)
    CHGVAR VAR(&I) VALUE(&I + 1)
ENDDO
  • Explanation:
    • DCL VAR(&I) TYPE(*INT) LEN(4) VALUE(1): Declares a variable &I of type integer and initializes it to 1.
    • DOWHILE (&I <= 5): Starts a loop that continues as long as &I is less than or equal to 5.
    • SNDPGMMSG MSG('Iteration ' *CAT &I): Sends a program message concatenating the string 'Iteration ' with the current value of &I.
    • CHGVAR VAR(&I) VALUE(&I + 1): Increments the value of &I by 1.
    • ENDDO: Ends the loop.

Advanced Looping Techniques

Nested Loops

You can nest loops within other loops to handle more complex scenarios.

Example

DCL VAR(&I) TYPE(*INT) LEN(4)
DCL VAR(&J) TYPE(*INT) LEN(4)
DO VAR(&I) FROM(1) TO(3) BY(1)
    DO VAR(&J) FROM(1) TO(2) BY(1)
        SNDPGMMSG MSG('Outer loop ' *CAT &I *CAT ' Inner loop ' *CAT &J)
    ENDDO
ENDDO
  • Explanation:
    • The outer loop runs from 1 to 3.
    • The inner loop runs from 1 to 2 for each iteration of the outer loop.
    • A message is sent for each combination of &I and &J.

Loop Control Statements

  • LEAVE: Exits the loop immediately.
  • ITERATE: Skips the remaining code in the current iteration and proceeds to the next iteration.

Example with LEAVE and ITERATE

DCL VAR(&I) TYPE(*INT) LEN(4)
DO VAR(&I) FROM(1) TO(10) BY(1)
    IF COND(&I = 5) THEN(LEAVE)
    IF COND(&I MOD 2 = 0) THEN(ITERATE)
    SNDPGMMSG MSG('Iteration ' *CAT &I)
ENDDO
  • Explanation:
    • The loop runs from 1 to 10.
    • If &I equals 5, the LEAVE statement exits the loop.
    • If &I is even, the ITERATE statement skips the remaining code and proceeds to the next iteration.
    • A message is sent only for odd values of &I that are not 5.

Conclusion

Looping constructs in CL are powerful tools that help automate repetitive tasks and manage complex logic. By understanding and utilizing DO and DOWHILE loops, along with advanced techniques like nested loops and control statements, you can write more efficient and effective CL scripts. Practice these constructs with various examples to gain a deeper understanding and enhance your programming skills in CL.
