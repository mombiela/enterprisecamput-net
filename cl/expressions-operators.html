Introduction

Expressions and operators are fundamental components in Control Language (CL) programming. They allow you to perform calculations, make decisions, and manipulate data. This section will guide you from the basics to advanced usage of expressions and operators in CL.

Basic Concepts

What is an Expression?

  • An expression is a combination of variables, constants, operators, and functions that are evaluated to produce a value.
  • Example: 3 + 5 is an expression that evaluates to 8.

Types of Operators

Operators are special symbols or keywords that are used to perform operations on variables and values. They can be categorized into:

  • Arithmetic Operators
  • Comparison Operators
  • Logical Operators
  • Assignment Operators

Arithmetic Operators

Arithmetic operators are used to perform mathematical calculations.

| Operator | Description | Example | Result | |----------|-------------|---------|--------| | + | Addition | 5 + 3 | 8 | | - | Subtraction | 5 - 3 | 2 | | * | Multiplication | 5 * 3 | 15 | | / | Division | 6 / 3 | 2 | | % | Modulus | 5 % 3 | 2 |

Example

DCL VAR(&RESULT) TYPE(*DEC) LEN(5 2)
CHGVAR VAR(&RESULT) VALUE(5 + 3)

In this example, &RESULT will be set to 8.

Comparison Operators

Comparison operators are used to compare two values.

| Operator | Description | Example | Result | |----------|-------------|---------|--------| | = | Equal to | 5 = 3 | False| | <> | Not equal to| 5 <> 3| True | | > | Greater than| 5 > 3 | True | | < | Less than | 5 < 3 | False| | >= | Greater than or equal to | 5 >= 3 | True | | <= | Less than or equal to | 5 <= 3 | False |

Example

DCL VAR(&IS_EQUAL) TYPE(*LGL)
CHGVAR VAR(&IS_EQUAL) VALUE(5 = 3)

In this example, &IS_EQUAL will be set to *FALSE.

Logical Operators

Logical operators are used to combine multiple conditions.

| Operator | Description | Example | Result | |----------|-------------|---------|--------| | & | AND | *TRUE & *FALSE | *FALSE | | | | OR | *TRUE | *FALSE | *TRUE | | ! | NOT | ! *TRUE | *FALSE |

Example

DCL VAR(&RESULT) TYPE(*LGL)
CHGVAR VAR(&RESULT) VALUE((*TRUE & *FALSE) | *TRUE)

In this example, &RESULT will be set to *TRUE.

Assignment Operators

Assignment operators are used to assign values to variables.

| Operator | Description | Example | Result | |----------|-------------|---------|--------| | = | Assign | VAR = 5 | VAR is set to 5 |

Example

DCL VAR(&NUMBER) TYPE(*DEC) LEN(5 0)
CHGVAR VAR(&NUMBER) VALUE(10)

In this example, &NUMBER will be set to 10.

Advanced Usage

Combining Operators

You can combine different types of operators in a single expression to perform complex calculations and evaluations.

Example

DCL VAR(&RESULT) TYPE(*DEC) LEN(5 2)
CHGVAR VAR(&RESULT) VALUE((5 + 3) * 2 / (4 - 2))

In this example, &RESULT will be set to 8.00.

Using Functions in Expressions

You can also use built-in functions within expressions to perform more advanced operations.

Example

DCL VAR(&DATE) TYPE(*CHAR) LEN(8)
CHGVAR VAR(&DATE) VALUE(%DATE())

In this example, &DATE will be set to the current system date.

Conclusion

Expressions and operators are essential tools in CL programming. They enable you to perform calculations, make decisions, and manipulate data effectively. By mastering these concepts, you can write more efficient and powerful CL programs. Practice combining different operators and using functions within your expressions to become proficient in CL programming.
