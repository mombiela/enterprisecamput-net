Introduction

Operators are special symbols that perform operations on variables and values. Dart supports a wide range of operators, from basic arithmetic to more advanced bitwise and logical operators. Understanding these operators is crucial for writing efficient and effective Dart code.

Arithmetic Operators

Arithmetic operators are used to perform common mathematical operations.

  • + Addition
  • - Subtraction
  • * Multiplication
  • / Division
  • % Modulus (remainder)
  • ~/ Integer Division

Examples

void main() {
  int a = 10;
  int b = 3;

  print(a + b);  // Output: 13
  print(a - b);  // Output: 7
  print(a * b);  // Output: 30
  print(a / b);  // Output: 3.3333333333333335
  print(a % b);  // Output: 1
  print(a ~/ b); // Output: 3
}

Assignment Operators

Assignment operators are used to assign values to variables.

  • = Assign
  • += Add and assign
  • -= Subtract and assign
  • *= Multiply and assign
  • /= Divide and assign
  • %= Modulus and assign

Examples

void main() {
  int a = 5;
  a += 3; // Equivalent to a = a + 3
  print(a); // Output: 8

  a -= 2; // Equivalent to a = a - 2
  print(a); // Output: 6

  a *= 2; // Equivalent to a = a * 2
  print(a); // Output: 12

  a /= 3; // Equivalent to a = a / 3
  print(a); // Output: 4.0

  a %= 2; // Equivalent to a = a % 2
  print(a); // Output: 0.0
}

Comparison Operators

Comparison operators are used to compare two values and return a boolean result.

  • == Equal to
  • != Not equal to
  • > Greater than
  • < Less than
  • >= Greater than or equal to
  • <= Less than or equal to

Examples

void main() {
  int a = 5;
  int b = 3;

  print(a == b); // Output: false
  print(a != b); // Output: true
  print(a > b);  // Output: true
  print(a < b);  // Output: false
  print(a >= b); // Output: true
  print(a <= b); // Output: false
}

Logical Operators

Logical operators are used to combine multiple boolean expressions.

  • && Logical AND
  • || Logical OR
  • ! Logical NOT

Examples

void main() {
  bool a = true;
  bool b = false;

  print(a && b); // Output: false
  print(a || b); // Output: true
  print(!a);     // Output: false
}

Bitwise Operators

Bitwise operators are used to perform bit-level operations.

  • & Bitwise AND
  • | Bitwise OR
  • ^ Bitwise XOR
  • ~ Bitwise NOT
  • << Left shift
  • >> Right shift

Examples

void main() {
  int a = 5;  // 0101 in binary
  int b = 3;  // 0011 in binary

  print(a & b);  // Output: 1 (0001 in binary)
  print(a | b);  // Output: 7 (0111 in binary)
  print(a ^ b);  // Output: 6 (0110 in binary)
  print(~a);     // Output: -6 (inverts all bits)
  print(a << 1); // Output: 10 (1010 in binary)
  print(a >> 1); // Output: 2 (0010 in binary)
}

Conditional (Ternary) Operator

The conditional operator is a shorthand for an if-else statement.

  • condition ? expr1 : expr2

Example

void main() {
  int a = 5;
  int b = 3;

  String result = a > b ? 'a is greater' : 'b is greater';
  print(result); // Output: a is greater
}

Null-aware Operators

Null-aware operators are used to handle null values more gracefully.

  • ?? If null
  • ??= Assign if null
  • ?. Conditional member access

Examples

void main() {
  int? a;
  int b = 3;

  print(a ?? b); // Output: 3

  a ??= 10;
  print(a); // Output: 10

  String? name;
  print(name?.length); // Output: null
}

Summary

Operators in Dart are essential tools for performing a wide range of operations, from basic arithmetic to complex logical and bitwise manipulations. Understanding how to use these operators effectively will help you write more efficient and readable Dart code. Practice using these operators in different scenarios to become more comfortable with their syntax and functionality.
