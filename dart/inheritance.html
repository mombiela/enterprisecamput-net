Inheritance in Dart
Mixins ►

Inheritance is a fundamental concept in object-oriented programming that allows a class to inherit properties and methods from another class. This helps in code reusability and establishing a natural hierarchy between classes.

Key Concepts

  • Superclass (Parent Class): The class whose properties and methods are inherited.
  • Subclass (Child Class): The class that inherits from another class.
  • extends Keyword: Used to indicate that a class is inheriting from another class.
  • Method Overriding: The ability of a subclass to provide a specific implementation of a method that is already defined in its superclass.
  • super Keyword: Used to refer to the superclass and access its properties and methods.

Basic Inheritance

In Dart, you can create a subclass by using the extends keyword. Here’s a simple example:

class Animal {
  void makeSound() {
    print('Animal makes a sound');
  }
}

class Dog extends Animal {
  void bark() {
    print('Dog barks');
  }
}

void main() {
  Dog dog = Dog();
  dog.makeSound(); // Inherited method
  dog.bark(); // Subclass method
}

Explanation

  • Animal Class: This is the superclass with a method makeSound.
  • Dog Class: This is the subclass that extends Animal and adds a new method bark.
  • main Function: Creates an instance of Dog and calls both the inherited makeSound method and the bark method.

Method Overriding

A subclass can override a method from its superclass to provide a specific implementation. Use the @override annotation to indicate that you are overriding a method.

class Animal {
  void makeSound() {
    print('Animal makes a sound');
  }
}

class Dog extends Animal {
  @override
  void makeSound() {
    print('Dog barks');
  }
}

void main() {
  Dog dog = Dog();
  dog.makeSound(); // Outputs: Dog barks
}

Explanation

  • @override Annotation: Indicates that the makeSound method in Dog is overriding the method in Animal.
  • Output: When makeSound is called on a Dog object, the overridden method in Dog is executed.

Using `super` Keyword

The super keyword is used to refer to the superclass and can be used to call its methods and constructors.

class Animal {
  void makeSound() {
    print('Animal makes a sound');
  }
}

class Dog extends Animal {
  @override
  void makeSound() {
    super.makeSound(); // Calls the superclass method
    print('Dog barks');
  }
}

void main() {
  Dog dog = Dog();
  dog.makeSound(); 
  // Outputs:
  // Animal makes a sound
  // Dog barks
}

Explanation

  • super.makeSound(): Calls the makeSound method of the superclass Animal before executing the makeSound method in Dog.

Constructors and Inheritance

When a subclass is instantiated, the constructor of the superclass is called first. You can explicitly call a superclass constructor using the super keyword.

class Animal {
  String name;
  
  Animal(this.name);
  
  void display() {
    print('Animal: $name');
  }
}

class Dog extends Animal {
  String breed;
  
  Dog(String name, this.breed) : super(name);
  
  @override
  void display() {
    super.display();
    print('Breed: $breed');
  }
}

void main() {
  Dog dog = Dog('Buddy', 'Golden Retriever');
  dog.display();
  // Outputs:
  // Animal: Buddy
  // Breed: Golden Retriever
}

Explanation

  • Animal Constructor: Takes a name parameter.
  • Dog Constructor: Takes name and breed parameters and calls the superclass constructor with super(name).
  • display Method: Overridden in Dog to include additional information about the breed.

Summary

Inheritance in Dart allows you to create a class hierarchy where subclasses inherit properties and methods from their superclasses. Key concepts include the extends keyword for defining inheritance, method overriding, and the super keyword for accessing superclass methods and constructors. Understanding these concepts is crucial for writing reusable and maintainable code in Dart.
