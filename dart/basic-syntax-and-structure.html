Introduction

Dart is a client-optimized language for fast apps on any platform. It is developed by Google and is used to build mobile, desktop, server, and web applications. Understanding the basic syntax and structure of Dart is essential for writing efficient and readable code.

Key Concepts

  • Variables and Data Types
  • Functions
  • Control Flow Statements
  • Comments
  • Imports

Variables and Data Types

In Dart, variables are used to store data. Dart is a statically typed language, which means you need to specify the type of a variable when you declare it.

Declaring Variables

void main() {
  int age = 25; // Integer
  double height = 5.9; // Double
  String name = 'John Doe'; // String
  bool isStudent = true; // Boolean
}

Type Inference

Dart can infer the type of a variable based on the assigned value.

void main() {
  var age = 25; // Dart infers that age is an int
  var height = 5.9; // Dart infers that height is a double
  var name = 'John Doe'; // Dart infers that name is a String
  var isStudent = true; // Dart infers that isStudent is a bool
}

Functions

Functions are the building blocks of a Dart program. They allow you to encapsulate code into reusable blocks.

Defining Functions

void main() {
  printGreeting();
  print(add(5, 3));
}

void printGreeting() {
  print('Hello, World!');
}

int add(int a, int b) {
  return a + b;
}

Arrow Functions

For simple functions, Dart provides a shorthand syntax known as arrow functions.

int add(int a, int b) => a + b;

Control Flow Statements

Control flow statements are used to control the execution flow of the program.

If-Else Statements

void main() {
  int age = 20;

  if (age >= 18) {
    print('You are an adult.');
  } else {
    print('You are a minor.');
  }
}

Switch Statements

void main() {
  String grade = 'A';

  switch (grade) {
    case 'A':
      print('Excellent');
      break;
    case 'B':
      print('Good');
      break;
    case 'C':
      print('Fair');
      break;
    case 'D':
      print('Poor');
      break;
    default:
      print('Fail');
  }
}

Loops

For Loop

void main() {
  for (int i = 0; i < 5; i++) {
    print(i);
  }
}

While Loop

void main() {
  int i = 0;

  while (i < 5) {
    print(i);
    i++;
  }
}

Comments

Comments are used to explain code and are ignored by the compiler.

Single-line Comments

// This is a single-line comment

Multi-line Comments

/*
This is a 
multi-line comment
*/

Imports

Imports are used to include libraries and other Dart files into your program.

Importing Core Libraries

import 'dart:math';

Importing Custom Files

import 'my_custom_file.dart';

Conclusion

Understanding the basic syntax and structure of Dart is crucial for writing efficient and readable code. This includes knowing how to declare variables, define functions, use control flow statements, write comments, and import libraries. Mastering these basics will provide a strong foundation for more advanced Dart programming concepts.
