Introduction to Sets

Sets are a collection of unique items. Unlike lists, sets do not allow duplicate elements. In Dart, sets are represented by the Set class.

Key Concepts

  • Uniqueness: Each element in a set must be unique.
  • Unordered: The elements in a set are not stored in any particular order.
  • Set Literals: Sets can be created using set literals, similar to list literals but with curly braces.

Creating Sets

You can create a set in Dart using the Set constructor or set literals.

Using Set Constructor

void main() {
  Set<int> numbers = Set();
  numbers.add(1);
  numbers.add(2);
  numbers.add(3);
  print(numbers); // Output: {1, 2, 3}
}

Using Set Literals

void main() {
  Set<String> fruits = {'apple', 'banana', 'orange'};
  print(fruits); // Output: {apple, banana, orange}
}

Basic Operations on Sets

Adding Elements

You can add elements to a set using the add method.

void main() {
  Set<int> numbers = {1, 2, 3};
  numbers.add(4);
  print(numbers); // Output: {1, 2, 3, 4}
}

Removing Elements

You can remove elements using the remove method.

void main() {
  Set<int> numbers = {1, 2, 3, 4};
  numbers.remove(3);
  print(numbers); // Output: {1, 2, 4}
}

Checking for Elements

You can check if a set contains an element using the contains method.

void main() {
  Set<String> fruits = {'apple', 'banana', 'orange'};
  print(fruits.contains('banana')); // Output: true
  print(fruits.contains('grape')); // Output: false
}

Advanced Set Operations

Union

The union of two sets is a set containing all elements from both sets.

void main() {
  Set<int> setA = {1, 2, 3};
  Set<int> setB = {3, 4, 5};
  Set<int> unionSet = setA.union(setB);
  print(unionSet); // Output: {1, 2, 3, 4, 5}
}

Intersection

The intersection of two sets is a set containing only the elements that are present in both sets.

void main() {
  Set<int> setA = {1, 2, 3};
  Set<int> setB = {3, 4, 5};
  Set<int> intersectionSet = setA.intersection(setB);
  print(intersectionSet); // Output: {3}
}

Difference

The difference between two sets is a set containing elements that are in the first set but not in the second set.

void main() {
  Set<int> setA = {1, 2, 3};
  Set<int> setB = {3, 4, 5};
  Set<int> differenceSet = setA.difference(setB);
  print(differenceSet); // Output: {1, 2}
}

Iterating Over Sets

You can iterate over a set using a for loop or a forEach method.

Using a For Loop

void main() {
  Set<String> fruits = {'apple', 'banana', 'orange'};
  for (var fruit in fruits) {
    print(fruit);
  }
}

Using forEach Method

void main() {
  Set<String> fruits = {'apple', 'banana', 'orange'};
  fruits.forEach((fruit) => print(fruit));
}

Conclusion

Sets in Dart are a powerful way to manage collections of unique items. They provide various methods for adding, removing, and checking elements, as well as performing advanced operations like union, intersection, and difference. Understanding sets and their operations can help you write more efficient and effective Dart code.

By mastering sets, you can handle unique collections of data with ease, ensuring that your applications are both performant and logically sound.
