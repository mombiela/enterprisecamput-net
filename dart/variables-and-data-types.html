Introduction

In Dart, variables are used to store data that can be manipulated and referenced throughout your program. Understanding variables and data types is fundamental to programming in Dart, as it allows you to handle different kinds of data effectively.

Variables

Variables in Dart are used to store values. You can declare a variable using the var, final, or const keywords.

Declaring Variables

  • var: The type of the variable is inferred by the compiler.
  • final: A variable that can be set only once.
  • const: A compile-time constant.
void main() {
  var name = 'Alice'; // String type inferred
  final age = 30; // int type inferred, cannot be changed
  const pi = 3.14; // double type, compile-time constant
}

Variable Naming Rules

  • Must begin with a letter or underscore (_).
  • Can contain letters, digits, and underscores.
  • Case-sensitive.
void main() {
  var firstName = 'Alice';
  var _lastName = 'Smith';
  var age1 = 25;
}

Data Types

Dart is a statically typed language, which means that every variable has a type. The common data types in Dart include:

  • Numbers: int and double
  • Strings: String
  • Booleans: bool
  • Lists: List
  • Maps: Map

Numbers

Dart has two types of numbers: int for integers and double for floating-point numbers.

void main() {
  int age = 25;
  double height = 5.9;
}

Strings

Strings represent a sequence of characters. They are enclosed in single or double quotes.

void main() {
  String greeting = 'Hello, World!';
  String name = "Alice";
}

Booleans

Booleans represent true or false values.

void main() {
  bool isStudent = true;
  bool isEmployed = false;
}

Lists

Lists are ordered collections of objects. They can be of fixed or growable length.

void main() {
  List<int> numbers = [1, 2, 3, 4, 5];
  List<String> names = ['Alice', 'Bob', 'Charlie'];
}

Maps

Maps are collections of key-value pairs. Keys and values can be of any type.

void main() {
  Map<String, int> phoneBook = {
    'Alice': 12345,
    'Bob': 67890,
  };
}

Type Inference

Dart can infer the type of a variable based on the assigned value. This is done using the var keyword.

void main() {
  var name = 'Alice'; // Inferred as String
  var age = 30; // Inferred as int
  var height = 5.9; // Inferred as double
}

Type Annotation

You can explicitly specify the type of a variable.

void main() {
  String name = 'Alice';
  int age = 30;
  double height = 5.9;
}

Type Conversion

Sometimes you need to convert one data type to another. Dart provides methods for type conversion.

String to int

void main() {
  var one = int.parse('1'); // 1
}

String to double

void main() {
  var onePointOne = double.parse('1.1'); // 1.1
}

int to String

void main() {
  String oneAsString = 1.toString(); // '1'
}

double to String

void main() {
  String piAsString = 3.14159.toStringAsFixed(2); // '3.14'
}

Summary

Understanding variables and data types is crucial for effective programming in Dart. Variables store data, and data types define the kind of data a variable can hold. Dart supports various data types, including numbers, strings, booleans, lists, and maps. You can declare variables using var, final, and const, and Dart can infer types or you can explicitly annotate them. Additionally, Dart provides methods for type conversion to handle different data types seamlessly.

By mastering these concepts, you will be well-equipped to handle data in your Dart programs efficiently.
