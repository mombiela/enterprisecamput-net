Introduction

In this section of the Digital Analytics course, we will focus on practical data analysis exercises. These exercises are designed to help you apply the theoretical concepts learned in real-world situations. We will work with different types of data and use various tools and techniques to analyze them.

Objectives

  • Apply data analysis techniques in practical situations.
  • Use software tools for data analysis.
  • Interpret and communicate data analysis results.

Necessary Tools

  • Python: Programming language used for data manipulation and analysis.
  • Pandas: Python library for data manipulation.
  • Matplotlib/Seaborn: Python libraries for data visualization.
  • Jupyter Notebook: Interactive environment for programming in Python.

Exercise 1: Exploratory Data Analysis (EDA)

Description

We will perform an exploratory analysis of a dataset to understand its structure, identify patterns, and detect anomalies.

Steps

  1. Load the dataset:

    • We will use a sample dataset, such as the iris dataset.

  2. Inspect the data:

    • Check the first rows of the dataset.
    • Obtain a statistical summary of the variables.

  3. Visualize the data:

    • Create scatter plots to visualize relationships between variables.
    • Create histograms to understand the distribution of variables.

Example Code

import pandas as pd
import seaborn as sns
import matplotlib.pyplot as plt

# Load the dataset
df = sns.load_dataset('iris')

# Inspect the first rows
print(df.head())

# Statistical summary
print(df.describe())

# Scatter plot
sns.pairplot(df, hue='species')
plt.show()

# Histograms
df.hist(bins=20, figsize=(10, 8))
plt.show()

Explanation

  • Load the dataset: We use the load_dataset function from Seaborn to load the iris dataset.
  • Inspect the first rows: We use head() to see the first rows of the DataFrame.
  • Statistical summary: We use describe() to obtain a statistical summary of the variables.
  • Scatter plot: We use Seaborn's pairplot to create scatter plots between all variables.
  • Histograms: We use Pandas' hist to create histograms of the variables.

Exercise 2: Data Cleaning

Description

We will clean a dataset by removing null values and duplicates, and transforming categorical data.

Steps

  1. Load the dataset:

    • We will use a dataset with null values and duplicates.

  2. Remove null values:

    • Identify and remove rows with null values.

  3. Remove duplicates:

    • Identify and remove duplicate rows.

  4. Transform categorical data:

    • Convert categorical variables into numerical variables.

Example Code

# Load the dataset
df = pd.read_csv('data.csv')

# Remove null values
df = df.dropna()

# Remove duplicates
df = df.drop_duplicates()

# Transform categorical data
df['category'] = df['category'].astype('category').cat.codes

print(df.head())

Explanation

  • Load the dataset: We use Pandas' read_csv to load the dataset.
  • Remove null values: We use dropna() to remove rows with null values.
  • Remove duplicates: We use drop_duplicates() to remove duplicate rows.
  • Transform categorical data: We convert the categorical column category into numerical codes using astype('category').cat.codes.

Exercise 3: Correlation Analysis

Description

We will analyze the correlation between variables in a dataset to identify significant relationships.

Steps

  1. Load the dataset:

    • We will use a sample dataset.

  2. Calculate the correlation matrix:

    • Use Pandas' corr() function.

  3. Visualize the correlation matrix:

    • Create a heatmap to visualize the correlations.

Example Code

# Load the dataset
df = pd.read_csv('data.csv')

# Calculate the correlation matrix
correlation_matrix = df.corr()

# Visualize the correlation matrix
sns.heatmap(correlation_matrix, annot=True, cmap='coolwarm')
plt.show()

Explanation

  • Load the dataset: We use Pandas' read_csv to load the dataset.
  • Calculate the correlation matrix: We use corr() to calculate the correlation matrix.
  • Visualize the correlation matrix: We use Seaborn's heatmap to create a heatmap of the correlation matrix.
