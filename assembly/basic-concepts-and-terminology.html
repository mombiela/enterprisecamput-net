Introduction

Assembly language is a low-level programming language that is closely related to machine code. It provides a way to write programs that are directly executed by the computer's CPU. Understanding the basic concepts and terminology is crucial for anyone starting with assembly language programming.

Key Concepts

Machine Code

  • Definition: The set of binary instructions that a computer's CPU executes directly.
  • Example: 10110000 01100001 (This could be a machine code instruction to move the value 97 into a register).

Assembly Language

  • Definition: A human-readable representation of machine code using mnemonics.
  • Example: MOV AL, 61h (This is an assembly instruction equivalent to the machine code example above).

Assembler

  • Definition: A tool that converts assembly language code into machine code.
  • Example: NASM (Netwide Assembler), MASM (Microsoft Macro Assembler).

Registers

  • Definition: Small, fast storage locations within the CPU used to hold data temporarily.
  • Common Registers:
    • AX, BX, CX, DX: General-purpose registers.
    • SP: Stack Pointer.
    • BP: Base Pointer.
    • SI, DI: Source Index, Destination Index.

Mnemonics

  • Definition: Human-readable instructions in assembly language that correspond to machine code instructions.
  • Example: ADD, SUB, MOV, JMP.

Opcode

  • Definition: The portion of a machine language instruction that specifies the operation to be performed.
  • Example: In the instruction MOV AL, 61h, MOV is the mnemonic, and its corresponding opcode in machine code might be B0.

Operands

  • Definition: The data on which the operation is performed.
  • Example: In MOV AL, 61h, AL and 61h are operands.

Instruction Set

  • Definition: The complete set of instructions that a CPU can execute.
  • Example: x86, ARM.

Basic Assembly Instructions

Data Movement

  • MOV: Transfers data from one location to another. 
    MOV AX, BX  ; Move the contents of BX into AX
MOV AL, 61h ; Move the hexadecimal value 61 into AL

Arithmetic Operations

  • ADD: Adds two operands. 
    ADD AX, BX  ; Add the contents of BX to AX
  • SUB: Subtracts the second operand from the first. 
    SUB AX, BX  ; Subtract the contents of BX from AX

Control Flow

  • JMP: Jumps to a specified address. 
    JMP 0040h   ; Jump to the address 0040h

Comparison and Conditional Jumps

  • CMP: Compares two operands. 
    CMP AX, BX  ; Compare AX with BX
  • JE: Jump if equal. 
    JE 0040h    ; Jump to 0040h if the comparison was equal

Example Program

Here is a simple assembly program that adds two numbers and stores the result in a register:

section .data
    num1 db 10h  ; Define byte with value 10h
    num2 db 20h  ; Define byte with value 20h

section .text
    global _start

_start:
    mov al, [num1]  ; Move the value of num1 into AL
    add al, [num2]  ; Add the value of num2 to AL
    ; AL now contains the result (30h)
    
    ; Exit the program (specific to Linux)
    mov eax, 1      ; System call number (sys_exit)
    int 0x80        ; Call kernel

Conclusion

Understanding the basic concepts and terminology of assembly language is the first step towards mastering this powerful programming language. Key concepts such as machine code, mnemonics, registers, and the instruction set form the foundation of assembly programming. By familiarizing yourself with these terms and practicing simple instructions, you will be well-prepared to delve deeper into more complex assembly language programming topics.
