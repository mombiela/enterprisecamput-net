Conditional Branching in Assembly
Loops and Iteration ►

Conditional branching is a fundamental concept in assembly language programming that allows the flow of execution to change based on certain conditions. This topic will cover the basics of conditional branching, including key instructions, examples, and exercises to solidify your understanding.

Key Concepts

  • Conditional Branch Instructions: Instructions that alter the flow of execution based on the evaluation of a condition.
  • Flags: Special registers that store the results of various operations and are used to determine the outcome of conditional branches.
  • Comparison Instructions: Instructions used to compare values and set the appropriate flags.

Conditional Branch Instructions

Conditional branch instructions in assembly language allow the program to make decisions and execute different code paths based on certain conditions. Here are some common conditional branch instructions:

  • JE (Jump if Equal): Jumps to a specified address if the zero flag (ZF) is set.
  • JNE (Jump if Not Equal): Jumps to a specified address if the zero flag (ZF) is not set.
  • JG (Jump if Greater): Jumps to a specified address if the sign flag (SF) is equal to the overflow flag (OF) and the zero flag (ZF) is not set.
  • JL (Jump if Less): Jumps to a specified address if the sign flag (SF) is not equal to the overflow flag (OF).
  • JGE (Jump if Greater or Equal): Jumps to a specified address if the sign flag (SF) is equal to the overflow flag (OF).
  • JLE (Jump if Less or Equal): Jumps to a specified address if the sign flag (SF) is not equal to the overflow flag (OF) or the zero flag (ZF) is set.

Flags

Flags are special registers that store the results of various operations. The most commonly used flags for conditional branching are:

  • Zero Flag (ZF): Set if the result of an operation is zero.
  • Sign Flag (SF): Set if the result of an operation is negative.
  • Overflow Flag (OF): Set if an arithmetic overflow occurs.
  • Carry Flag (CF): Set if an arithmetic carry or borrow occurs.

Comparison Instructions

Comparison instructions are used to compare values and set the appropriate flags. The most common comparison instruction is:

  • CMP (Compare): Compares two operands and sets the flags based on the result.

Example

Let's look at an example of conditional branching in assembly language. This example will compare two numbers and print a message based on the comparison.

section .data
    msg_equal db "Numbers are equal", 0
    msg_greater db "First number is greater", 0
    msg_less db "Second number is greater", 0

section .bss

section .text
    global _start

_start:
    ; Initialize the numbers
    mov eax, 5
    mov ebx, 10

    ; Compare the numbers
    cmp eax, ebx

    ; Conditional branching
    je equal
    jg greater
    jl less

equal:
    ; Print "Numbers are equal"
    mov edx, len_equal
    mov ecx, msg_equal
    jmp print

greater:
    ; Print "First number is greater"
    mov edx, len_greater
    mov ecx, msg_greater
    jmp print

less:
    ; Print "Second number is greater"
    mov edx, len_less
    mov ecx, msg_less

print:
    ; Print the message
    mov ebx, 1
    mov eax, 4
    int 0x80

    ; Exit the program
    mov eax, 1
    int 0x80

section .data
    len_equal equ $ - msg_equal
    len_greater equ $ - msg_greater
    len_less equ $ - msg_less

In this example:

  • We initialize two numbers in registers eax and ebx.
  • We use the CMP instruction to compare the values in eax and ebx.
  • Based on the comparison, we use conditional branch instructions (JE, JG, JL) to jump to the appropriate label.
  • Each label contains code to print a message indicating the result of the comparison.

Exercises

  1. Basic Comparison: Write an assembly program that compares two numbers and prints whether they are equal, the first number is greater, or the second number is greater.
  2. Multiple Conditions: Modify the example program to handle more complex conditions, such as checking if a number is positive, negative, or zero.
  3. Loop with Conditional Branching: Write a program that uses a loop and conditional branching to find the maximum value in an array of integers.

Conclusion

Conditional branching is a powerful feature in assembly language that allows you to control the flow of your program based on specific conditions. By understanding and using conditional branch instructions, flags, and comparison instructions, you can create more dynamic and responsive programs. Practice the provided exercises to strengthen your understanding and become proficient in using conditional branching in assembly language.
