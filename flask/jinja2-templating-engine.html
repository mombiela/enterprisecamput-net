Introduction to Jinja2

Jinja2 is a powerful templating engine for Python, used extensively with the Flask web framework to create dynamic web pages. It allows you to embed Python-like expressions in your HTML files, making it easier to generate HTML dynamically.

Key Concepts

  • Templates: HTML files with embedded Jinja2 syntax.
  • Variables: Placeholders for dynamic content.
  • Control Structures: Logic for loops and conditionals.
  • Filters: Functions to modify variables.
  • Macros: Reusable blocks of code.

Setting Up Jinja2 in Flask

To use Jinja2 with Flask, you need to set up your project structure correctly.

Project Structure

my_flask_app/
│
├── app.py
├── templates/
│   ├── base.html
│   └── index.html
└── static/
    └── style.css

Basic Flask Application

from flask import Flask, render_template

app = Flask(__name__)

@app.route('/')
def index():
    return render_template('index.html', title='Home Page')

if __name__ == '__main__':
    app.run(debug=True)

Using Variables in Templates

Variables in Jinja2 are enclosed in double curly braces {{ }}.

Example

<!-- index.html -->
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>{{ title }}</title>
</head>
<body>
    <h1>Welcome to {{ title }}</h1>
</body>
</html>

Explanation

  • {{ title }}: This will be replaced by the value of the title variable passed from the Flask route.

Control Structures

Jinja2 supports control structures like loops and conditionals.

Loops

<!-- index.html -->
<ul>
    {% for item in items %}
        <li>{{ item }}</li>
    {% endfor %}
</ul>

Conditionals

<!-- index.html -->
{% if user %}
    <p>Welcome, {{ user }}!</p>
{% else %}
    <p>Welcome, Guest!</p>
{% endif %}

Explanation

  • {% for item in items %}: Loop through each item in the items list.
  • {% if user %}: Check if the user variable is defined.

Filters

Filters are used to modify variables. They are applied using the pipe | symbol.

Example

<!-- index.html -->
<p>{{ name | upper }}</p>
<p>{{ date | dateformat('%Y-%m-%d') }}</p>

Explanation

  • {{ name | upper }}: Converts the name variable to uppercase.
  • {{ date | dateformat('%Y-%m-%d') }}: Formats the date variable.

Macros

Macros are reusable blocks of code that can be defined and called within templates.

Example

<!-- macros.html -->
{% macro render_item(item) %}
    <li>{{ item }}</li>
{% endmacro %}
<!-- index.html -->
{% from 'macros.html' import render_item %}
<ul>
    {% for item in items %}
        {{ render_item(item) }}
    {% endfor %}
</ul>

Explanation

  • {% macro render_item(item) %}: Defines a macro named render_item.
  • {% from 'macros.html' import render_item %}: Imports the render_item macro from macros.html.

Conclusion

Jinja2 is an essential tool for creating dynamic web pages in Flask. By understanding its key concepts—templates, variables, control structures, filters, and macros—you can build more interactive and user-friendly web applications. Practice using these features to become proficient in Jinja2 and enhance your Flask projects.
