Introduction

User authentication is a crucial aspect of web development, ensuring that users can securely log in and access their data. In this section, we'll cover the basics of user authentication in Flask, progressing from beginner to advanced concepts.

Setting Up Flask

Before diving into user authentication, ensure you have Flask installed and set up.

pip install Flask

Create a basic Flask application:

from flask import Flask

app = Flask(__name__)

@app.route('/')
def home():
    return "Hello, Flask!"

if __name__ == '__main__':
    app.run(debug=True)

Basic User Authentication

Creating a User Model

To handle user data, we'll create a User model. For simplicity, we'll use a dictionary to store user information.

users = {
    "user1": {"username": "user1", "password": "password1"},
    "user2": {"username": "user2", "password": "password2"}
}

Login Route

Create a route to handle user login.

from flask import request, jsonify

@app.route('/login', methods=['POST'])
def login():
    data = request.get_json()
    username = data.get('username')
    password = data.get('password')
    
    user = users.get(username)
    if user and user['password'] == password:
        return jsonify({"message": "Login successful!"}), 200
    return jsonify({"message": "Invalid credentials"}), 401

Protecting Routes

To protect certain routes, we can use a simple session-based approach.

from flask import session

app.secret_key = 'supersecretkey'

@app.route('/protected')
def protected():
    if 'username' in session:
        return f"Hello, {session['username']}!"
    return "You are not logged in!", 401

@app.route('/login', methods=['POST'])
def login():
    data = request.get_json()
    username = data.get('username')
    password = data.get('password')
    
    user = users.get(username)
    if user and user['password'] == password:
        session['username'] = username
        return jsonify({"message": "Login successful!"}), 200
    return jsonify({"message": "Invalid credentials"}), 401

@app.route('/logout')
def logout():
    session.pop('username', None)
    return "Logged out!"

Intermediate User Authentication

Using Flask-Login

Flask-Login provides user session management for Flask. Install Flask-Login:

pip install Flask-Login

Setting Up Flask-Login

Integrate Flask-Login into your application.

from flask_login import LoginManager, UserMixin, login_user, login_required, logout_user, current_user

login_manager = LoginManager()
login_manager.init_app(app)

class User(UserMixin):
    def __init__(self, id):
        self.id = id

@login_manager.user_loader
def load_user(user_id):
    return User(user_id)

@app.route('/login', methods=['POST'])
def login():
    data = request.get_json()
    username = data.get('username')
    password = data.get('password')
    
    user = users.get(username)
    if user and user['password'] == password:
        user_obj = User(username)
        login_user(user_obj)
        return jsonify({"message": "Login successful!"}), 200
    return jsonify({"message": "Invalid credentials"}), 401

@app.route('/protected')
@login_required
def protected():
    return f"Hello, {current_user.id}!"

@app.route('/logout')
@login_required
def logout():
    logout_user()
    return "Logged out!"

Advanced User Authentication

Password Hashing

Storing plain text passwords is insecure. Use a hashing library like werkzeug.security to hash passwords.

from werkzeug.security import generate_password_hash, check_password_hash

# Hashing passwords
users = {
    "user1": {"username": "user1", "password": generate_password_hash("password1")},
    "user2": {"username": "user2", "password": generate_password_hash("password2")}
}

# Verifying passwords
@app.route('/login', methods=['POST'])
def login():
    data = request.get_json()
    username = data.get('username')
    password = data.get('password')
    
    user = users.get(username)
    if user and check_password_hash(user['password'], password):
        user_obj = User(username)
        login_user(user_obj)
        return jsonify({"message": "Login successful!"}), 200
    return jsonify({"message": "Invalid credentials"}), 401

Using Flask-SQLAlchemy

For a more robust solution, integrate Flask-SQLAlchemy to manage user data in a database.

pip install Flask-SQLAlchemy
from flask_sqlalchemy import SQLAlchemy

app.config['SQLALCHEMY_DATABASE_URI'] = 'sqlite:///users.db'
db = SQLAlchemy(app)

class User(db.Model, UserMixin):
    id = db.Column(db.Integer, primary_key=True)
    username = db.Column(db.String(150), unique=True, nullable=False)
    password = db.Column(db.String(150), nullable=False)

# Create the database
db.create_all()

# Add a user
new_user = User(username='user1', password=generate_password_hash('password1'))
db.session.add(new_user)
db.session.commit()

@app.route('/login', methods=['POST'])
def login():
    data = request.get_json()
    username = data.get('username')
    password = data.get('password')
    
    user = User.query.filter_by(username=username).first()
    if user and check_password_hash(user.password, password):
        login_user(user)
        return jsonify({"message": "Login successful!"}), 200
    return jsonify({"message": "Invalid credentials"}), 401

Conclusion

User authentication is a fundamental aspect of web development. Starting from basic session management to using Flask-Login and integrating Flask-SQLAlchemy, we've covered various methods to implement user authentication in Flask. By following these steps, you can ensure your application is secure and provides a seamless user experience.
