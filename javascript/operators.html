Introduction

Operators are fundamental building blocks in JavaScript that allow you to perform various operations on variables and values. Understanding how to use operators effectively is crucial for writing efficient and effective code.

Types of Operators

Arithmetic Operators

Arithmetic operators are used to perform mathematical operations.

  • Addition (+)

    let a = 5;
let b = 3;
let sum = a + b; // sum is 8

  • Subtraction (-)

    let difference = a - b; // difference is 2

  • Multiplication (*)

    let product = a * b; // product is 15

  • Division (/)

    let quotient = a / b; // quotient is 1.6667

  • Modulus (%)

    let remainder = a % b; // remainder is 2

  • Exponentiation (**)

    let power = a ** b; // power is 125

Assignment Operators

Assignment operators are used to assign values to variables.

  • Assignment (=)

    let x = 10;

  • Addition Assignment (+=)

    x += 5; // x is now 15

  • Subtraction Assignment (-=)

    x -= 3; // x is now 12

  • Multiplication Assignment (*=)

    x *= 2; // x is now 24

  • Division Assignment (/=)

    x /= 4; // x is now 6

  • Modulus Assignment (%=)

    x %= 5; // x is now 1

Comparison Operators

Comparison operators are used to compare two values and return a boolean result.

  • Equal (==)

    let isEqual = (a == b); // false

  • Strict Equal (===)

    let isStrictEqual = (a === b); // false

  • Not Equal (!=)

    let isNotEqual = (a != b); // true

  • Strict Not Equal (!==)

    let isStrictNotEqual = (a !== b); // true

  • Greater Than (>)

    let isGreaterThan = (a > b); // true

  • Less Than (<)

    let isLessThan = (a < b); // false

  • Greater Than or Equal (>=)

    let isGreaterThanOrEqual = (a >= b); // true

  • Less Than or Equal (<=)

    let isLessThanOrEqual = (a <= b); // false

Logical Operators

Logical operators are used to combine multiple boolean expressions.

  • Logical AND (&&)

    let andResult = (a > 0 && b > 0); // true

  • Logical OR (||)

    let orResult = (a > 0 || b < 0); // true

  • Logical NOT (!)

    let notResult = !(a > 0); // false

Bitwise Operators

Bitwise operators are used to perform bit-level operations.

  • AND (&)

    let bitwiseAnd = a & b; // 1 (binary: 0101 & 0011 = 0001)

  • OR (|)

    let bitwiseOr = a | b; // 7 (binary: 0101 | 0011 = 0111)

  • XOR (^)

    let bitwiseXor = a ^ b; // 6 (binary: 0101 ^ 0011 = 0110)

  • NOT (~)

    let bitwiseNot = ~a; // -6 (binary: ~0101 = 1010)

  • Left Shift (<<)

    let leftShift = a << 1; // 10 (binary: 0101 << 1 = 1010)

  • Right Shift (>>)

    let rightShift = a >> 1; // 2 (binary: 0101 >> 1 = 0010)

Ternary Operator

The ternary operator is a shorthand for the if-else statement.

  • Syntax: condition ? expr1 : expr2 
    let age = 18;
let canVote = (age >= 18) ? 'Yes' : 'No'; // canVote is 'Yes'

Type Operators

Type operators are used to determine the type of a variable.

  • typeof

    let typeOfA = typeof a; // 'number'

  • instanceof

    let isInstanceOf = a instanceof Number; // false

Conclusion

Operators are essential tools in JavaScript that allow you to manipulate data and control the flow of your programs. By mastering arithmetic, assignment, comparison, logical, bitwise, ternary, and type operators, you can write more efficient and effective code. Practice using these operators in different scenarios to become proficient in their usage.
