Groovy is a powerful, optionally typed, and dynamic language, with static-typing and static compilation capabilities, for the Java platform aimed at improving developer productivity. It can be used both as a scripting language and for writing full-fledged applications. This section will focus on using Groovy for scripting, covering the basics to more advanced scripting techniques.

Introduction to Groovy Scripting

Groovy scripts are typically used for automating tasks, manipulating files, and performing quick calculations. They are often used in build automation tools like Gradle.

Key Concepts

  • Dynamic Typing: Groovy allows you to write scripts without explicitly defining types.
  • Syntactic Sugar: Groovy provides a more concise and readable syntax compared to Java.
  • Interoperability: Groovy can seamlessly integrate with Java libraries and frameworks.

Hello World Script

Let's start with a simple "Hello World" script in Groovy.

println 'Hello, World!'

This script demonstrates the simplicity of Groovy syntax. The println function prints the string to the console.

Variables and Data Types

Groovy supports various data types and allows dynamic typing, making it flexible for scripting.

Declaring Variables

You can declare variables without specifying their type.

def name = 'Groovy'
def age = 10
println "Name: $name, Age: $age"

Common Data Types

  • String: Textual data.
  • Integer: Whole numbers.
  • Float/Double: Decimal numbers.
  • Boolean: True or false values.

Example: Data Types

def message = 'Hello, Groovy!'
def number = 42
def price = 19.99
def isAvailable = true

println "Message: $message"
println "Number: $number"
println "Price: $price"
println "Available: $isAvailable"

Control Structures

Groovy supports standard control structures such as loops and conditionals.

If-Else Statements

def number = 10

if (number > 0) {
    println 'Positive number'
} else {
    println 'Non-positive number'
}

For Loops

for (i in 1..5) {
    println "Iteration: $i"
}

While Loops

def count = 5

while (count > 0) {
    println "Count: $count"
    count--
}

Functions and Closures

Functions and closures are essential for writing reusable code in Groovy scripts.

Defining Functions

def greet(name) {
    return "Hello, $name!"
}

println greet('Groovy')

Using Closures

Closures are anonymous blocks of code that can be assigned to variables and passed around.

def square = { number -> number * number }

println square(4)  // Output: 16

File Operations

Groovy makes it easy to work with files, which is a common task in scripting.

Reading Files

def file = new File('example.txt')
file.eachLine { line ->
    println line
}

Writing Files

def file = new File('output.txt')
file.withWriter { writer ->
    writer.writeLine('Hello, Groovy!')
}

Advanced Scripting Techniques

As you become more comfortable with Groovy scripting, you can explore more advanced techniques.

Using Groovy Shell

The Groovy Shell (groovysh) is an interactive command-line shell for evaluating Groovy expressions.

groovy:000> println 'Hello from Groovy Shell!'

Integrating with Java Libraries

Groovy can easily use Java libraries, making it a versatile scripting language.

import java.util.Date

def now = new Date()
println "Current date and time: $now"

Conclusion

Groovy is a versatile and powerful scripting language that can significantly enhance your productivity. Its dynamic typing, concise syntax, and seamless Java integration make it an excellent choice for various scripting tasks. By mastering the basics and exploring advanced techniques, you can leverage Groovy to automate tasks, manipulate files, and integrate with Java libraries effectively.
