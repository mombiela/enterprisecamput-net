Introduction to Lists

Lists are a fundamental data structure in Groovy, used to store ordered collections of elements. They are dynamic, meaning their size can change, and they can hold elements of different types.

Key Concepts

  • Dynamic Size: Lists can grow and shrink as needed.
  • Ordered Collection: Elements in a list maintain their order.
  • Heterogeneous Elements: Lists can contain elements of different types.

Creating Lists

Creating lists in Groovy is straightforward. You can use square brackets [] to define a list.

def emptyList = []
def numberList = [1, 2, 3, 4, 5]
def mixedList = [1, "Groovy", true, 3.14]

Explanation

  • emptyList is an empty list.
  • numberList contains integers.
  • mixedList contains an integer, a string, a boolean, and a floating-point number.

Accessing List Elements

You can access elements in a list using their index, starting from 0.

def numberList = [1, 2, 3, 4, 5]
println(numberList[0])  // Outputs: 1
println(numberList[2])  // Outputs: 3

Explanation

  • numberList[0] accesses the first element.
  • numberList[2] accesses the third element.

Modifying Lists

Lists in Groovy are mutable, meaning you can change their contents.

Adding Elements

You can add elements using the << operator or the add() method.

def numberList = [1, 2, 3]
numberList << 4
numberList.add(5)
println(numberList)  // Outputs: [1, 2, 3, 4, 5]

Removing Elements

You can remove elements using the remove() method or by specifying the index.

def numberList = [1, 2, 3, 4, 5]
numberList.remove(2)  // Removes the element at index 2
println(numberList)   // Outputs: [1, 2, 4, 5]
numberList.remove((Integer) 4)  // Removes the element with value 4
println(numberList)   // Outputs: [1, 2, 5]

Updating Elements

You can update elements by assigning a new value to a specific index.

def numberList = [1, 2, 3, 4, 5]
numberList[1] = 10
println(numberList)  // Outputs: [1, 10, 3, 4, 5]

Iterating Over Lists

You can iterate over lists using loops or Groovy's each method.

Using a `for` Loop

def numberList = [1, 2, 3, 4, 5]
for (num in numberList) {
    println(num)
}

Using the `each` Method

def numberList = [1, 2, 3, 4, 5]
numberList.each { num ->
    println(num)
}

Advanced List Operations

Groovy provides several advanced operations for lists, such as filtering, mapping, and reducing.

Filtering

You can filter lists using the findAll method.

def numberList = [1, 2, 3, 4, 5]
def evenNumbers = numberList.findAll { it % 2 == 0 }
println(evenNumbers)  // Outputs: [2, 4]

Mapping

You can transform lists using the collect method.

def numberList = [1, 2, 3, 4, 5]
def squaredNumbers = numberList.collect { it * it }
println(squaredNumbers)  // Outputs: [1, 4, 9, 16, 25]

Reducing

You can reduce lists to a single value using the inject method.

def numberList = [1, 2, 3, 4, 5]
def sum = numberList.inject(0) { acc, num -> acc + num }
println(sum)  // Outputs: 15

Conclusion

Lists are a versatile and powerful data structure in Groovy. They allow you to store, access, modify, and iterate over collections of elements efficiently. By understanding the basics and advanced operations of lists, you can leverage their full potential in your Groovy programs.
