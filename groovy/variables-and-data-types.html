Variables and Data Types in Groovy
Operators ►

Introduction

In this section, we will cover the basics of variables and data types in Groovy. Understanding these concepts is crucial as they form the foundation of any programming language. We will start with the basics and gradually move to more advanced topics.

Variables

Variables are used to store data that can be manipulated and retrieved throughout your program.

Declaring Variables

  • In Groovy, you can declare a variable using the def keyword or by specifying the type explicitly.
  • Groovy is a dynamically typed language, meaning you don't always need to specify the type of a variable.

Using def Keyword

def name = "John Doe"
def age = 30
def isStudent = true

Specifying the Type

String name = "John Doe"
int age = 30
boolean isStudent = true

Variable Naming Rules

  • Variable names must start with a letter, a dollar sign $, or an underscore _.
  • Subsequent characters can be letters, digits, dollar signs, or underscores.
  • Variable names are case-sensitive.

Data Types

Groovy supports various data types, which can be broadly categorized into primitive types and object types.

Primitive Data Types

  • byte: 8-bit signed integer
  • short: 16-bit signed integer
  • int: 32-bit signed integer
  • long: 64-bit signed integer
  • float: 32-bit floating-point number
  • double: 64-bit floating-point number
  • char: 16-bit Unicode character
  • boolean: true or false

Examples

byte b = 10
short s = 100
int i = 1000
long l = 10000L
float f = 10.5F
double d = 20.5
char c = 'A'
boolean bool = true

Object Data Types

  • String: A sequence of characters
  • List: An ordered collection of elements
  • Map: A collection of key-value pairs
  • Range: A sequence of numbers or characters

Examples

String str = "Hello, Groovy!"
List<Integer> numbers = [1, 2, 3, 4, 5]
Map<String, Integer> ageMap = ["John": 30, "Jane": 25]
Range range = 1..10

Type Inference

Groovy can infer the type of a variable based on the value assigned to it. This is particularly useful for simplifying code.

Example

def message = "Hello, World!"  // Groovy infers this as a String
def count = 42                // Groovy infers this as an Integer
def price = 19.99             // Groovy infers this as a Double

Type Conversion

Groovy allows for both implicit and explicit type conversions.

Implicit Type Conversion

Groovy can automatically convert between compatible types.

Example

int num = 10
double result = num + 5.5  // num is implicitly converted to double

Explicit Type Conversion

You can explicitly convert types using type casting.

Example

double num = 10.5
int result = (int) num  // Explicitly cast double to int

Summary

In this section, we covered the basics of variables and data types in Groovy. We learned how to declare variables using the def keyword and by specifying types explicitly. We also explored the various primitive and object data types supported by Groovy, along with examples. Additionally, we discussed type inference and type conversion, both implicit and explicit.

Understanding these fundamental concepts is essential as they form the building blocks for more advanced programming topics. Make sure to practice by writing your own Groovy scripts to reinforce these concepts.
