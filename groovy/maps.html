Maps in Groovy
Ranges ►

Maps in Groovy are a powerful and flexible data structure that allow you to store key-value pairs. They are similar to dictionaries in Python or hash maps in Java. This section will guide you from the basics of creating and using maps to more advanced techniques and best practices.

Introduction to Maps

  • Definition: A map is a collection of key-value pairs where each key is unique.
  • Syntax: Maps in Groovy are defined using the [:] notation.

Creating Maps

You can create a map in Groovy using different methods:

// Using the literal notation
def map = [key1: 'value1', key2: 'value2']

// Using the map constructor
def map = new HashMap()
map.put('key1', 'value1')
map.put('key2', 'value2')

Accessing Map Elements

You can access elements in a map using the key:

def map = [name: 'John', age: 30]

// Accessing values
println map['name']  // Output: John
println map.age      // Output: 30

Modifying Maps

You can add, update, or remove elements in a map:

def map = [name: 'John', age: 30]

// Adding a new key-value pair
map['city'] = 'New York'

// Updating an existing key-value pair
map.age = 31

// Removing a key-value pair
map.remove('name')

println map  // Output: [age:31, city:New York]

Advanced Map Operations

Iterating Over Maps

You can iterate over the keys, values, or entries in a map:

def map = [name: 'John', age: 30, city: 'New York']

// Iterating over keys
map.keySet().each { key ->
    println key
}

// Iterating over values
map.values().each { value ->
    println value
}

// Iterating over entries
map.each { key, value ->
    println "$key: $value"
}

Filtering Maps

You can filter maps based on certain conditions:

def map = [name: 'John', age: 30, city: 'New York']

// Filtering entries where the value is a string
def filteredMap = map.findAll { key, value -> value instanceof String }

println filteredMap  // Output: [name:John, city:New York]

Transforming Maps

You can transform maps using the collectEntries method:

def map = [name: 'John', age: 30, city: 'New York']

// Transforming the map to uppercase keys
def transformedMap = map.collectEntries { key, value ->
    [(key.toUpperCase()): value]
}

println transformedMap  // Output: [NAME:John, AGE:30, CITY:New York]

Common Map Methods

Here are some common methods you can use with maps:

| Method | Description | Example Usage | |-----------------|---------------------------------------------|----------------------------------------| | put | Adds a key-value pair to the map | map.put('key', 'value') | | get | Retrieves the value for a given key | map.get('key') | | remove | Removes a key-value pair from the map | map.remove('key') | | containsKey | Checks if the map contains a specific key | map.containsKey('key') | | containsValue | Checks if the map contains a specific value | map.containsValue('value') | | size | Returns the number of key-value pairs | map.size() | | clear | Removes all key-value pairs from the map | map.clear() |

Conclusion

Maps in Groovy are a versatile and essential data structure that allow you to store and manipulate key-value pairs efficiently. From basic creation and access to advanced operations like filtering and transforming, Groovy provides a rich set of features to work with maps. Understanding these concepts will greatly enhance your ability to manage and manipulate data in your Groovy applications.
