Introduction

Groovy is an object-oriented programming language for the Java platform. It is a powerful, dynamic language with features similar to those of Python, Ruby, and Smalltalk. Groovy integrates seamlessly with all existing Java objects and libraries, making it an excellent choice for Java developers looking to enhance their productivity.

In this section, we will write our first Groovy script. This will help you get familiar with the basic syntax and structure of Groovy.

Setting Up the Environment

Before we write our first script, we need to set up the environment:

  1. Install Java Development Kit (JDK): Groovy runs on the Java platform, so you need to have JDK installed.
  2. Install Groovy: You can download and install Groovy from the official website or use a package manager like SDKMAN.

Writing Your First Script

Let's start with a simple "Hello, World!" script to get a feel for Groovy's syntax.

Hello, World!

Create a new file named HelloWorld.groovy and add the following code:

println 'Hello, World!'

Explanation

  • println: This is a Groovy method that prints the specified message to the console, followed by a newline character.
  • 'Hello, World!': This is a string literal in Groovy.

To run the script, open your terminal or command prompt, navigate to the directory containing HelloWorld.groovy, and execute the following command:

groovy HelloWorld.groovy

You should see the output:

Hello, World!

Basic Syntax and Structure

Let's explore some basic Groovy syntax and structure through examples.

Variables

Groovy is dynamically typed, meaning you don't have to explicitly declare the type of a variable.

def name = 'Groovy'
println name
  • def: Used to declare a variable.
  • name: The variable name.
  • 'Groovy': The value assigned to the variable.

Data Types

Groovy supports various data types, including numbers, strings, lists, and maps.

def number = 42
def text = 'Groovy is cool'
def list = [1, 2, 3, 4]
def map = [name: 'Groovy', type: 'language']

Control Structures

Groovy supports standard control structures like if-else, for loops, and while loops.

If-Else

def age = 18
if (age >= 18) {
    println 'Adult'
} else {
    println 'Minor'
}

For Loop

for (i in 1..5) {
    println i
}

While Loop

def count = 0
while (count < 5) {
    println count
    count++
}

Exercises

Try the following exercises to practice writing Groovy scripts:

  1. Print Your Name: Write a script that prints your name.
  2. Sum of Numbers: Write a script that calculates and prints the sum of numbers from 1 to 10.
  3. List Manipulation: Create a list of your favorite fruits and print each fruit using a loop.

Conclusion

In this section, we wrote our first Groovy script and explored some basic syntax and structures. We covered variables, data types, and control structures. These fundamentals are essential as we move forward to more advanced topics in Groovy.

By practicing the exercises, you will gain confidence in writing simple Groovy scripts. In the next section, we will dive deeper into Groovy's features and explore more complex concepts.
