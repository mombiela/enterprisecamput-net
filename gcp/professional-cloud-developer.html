# Example: Deploying a simple Python application on App Engine

# app.yaml configuration file
runtime: python39

# main.py - Simple Flask application
from flask import Flask

app = Flask(__name__)

@app.route('/')
def hello():
    return 'Hello, GCP!'

if __name__ == '__main__':
    app.run(host='0.0.0.0', port=8080)

Explanation:

  1. app.yaml: This file specifies the runtime environment for the application. In this case, it's set to Python 3.9.
  2. main.py: A simple Flask application that defines a single route (/) which returns "Hello, GCP!" when accessed.

To deploy this application:

  1. Ensure you have the Google Cloud SDK installed and authenticated.
  2. Navigate to the directory containing app.yaml and main.py.
  3. Run the command: 
    gcloud app deploy
  4. Once deployed, you can access your application via the URL provided by GCP.
