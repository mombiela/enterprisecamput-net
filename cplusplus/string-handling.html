Introduction

String handling is a fundamental aspect of programming in C++. This topic will cover the basics of strings, various operations that can be performed on strings, and advanced string handling techniques. By the end of this section, you will have a strong understanding of how to manipulate and utilize strings in your C++ programs.

Basics of Strings

In C++, strings can be handled using two primary methods: C-style strings and the std::string class from the Standard Library.

C-Style Strings

  • C-style strings are arrays of characters terminated by a null character ('\0').
  • They are defined using character arrays.
#include <iostream>

int main() {
    char greeting[] = "Hello, World!";
    std::cout << greeting << std::endl;
    return 0;
}

`std::string` Class

  • The std::string class provides a more flexible and powerful way to handle strings.
  • It is part of the C++ Standard Library and requires the inclusion of the <string> header.
#include <iostream>
#include <string>

int main() {
    std::string greeting = "Hello, World!";
    std::cout << greeting << std::endl;
    return 0;
}

String Operations

Concatenation

  • Concatenation is the process of joining two strings together.

Using C-Style Strings

#include <iostream>
#include <cstring>

int main() {
    char str1[20] = "Hello, ";
    char str2[] = "World!";
    strcat(str1, str2); // Concatenates str2 to str1
    std::cout << str1 << std::endl;
    return 0;
}

Using std::string

#include <iostream>
#include <string>

int main() {
    std::string str1 = "Hello, ";
    std::string str2 = "World!";
    std::string result = str1 + str2; // Concatenates str1 and str2
    std::cout << result << std::endl;
    return 0;
}

Length of a String

  • Determining the length of a string is a common operation.

Using C-Style Strings

#include <iostream>
#include <cstring>

int main() {
    char str[] = "Hello, World!";
    std::cout << "Length of string: " << strlen(str) << std::endl;
    return 0;
}

Using std::string

#include <iostream>
#include <string>

int main() {
    std::string str = "Hello, World!";
    std::cout << "Length of string: " << str.length() << std::endl;
    return 0;
}

Accessing Characters

  • Individual characters in a string can be accessed using array indexing.

Using C-Style Strings

#include <iostream>

int main() {
    char str[] = "Hello";
    std::cout << "First character: " << str[0] << std::endl;
    return 0;
}

Using std::string

#include <iostream>
#include <string>

int main() {
    std::string str = "Hello";
    std::cout << "First character: " << str[0] << std::endl;
    return 0;
}

Substrings

  • Extracting a substring from a string is a common requirement.

Using std::string

#include <iostream>
#include <string>

int main() {
    std::string str = "Hello, World!";
    std::string substr = str.substr(7, 5); // Extracts "World"
    std::cout << "Substring: " << substr << std::endl;
    return 0;
}

Advanced String Handling

String Streams

  • String streams allow you to perform input and output operations on strings.
#include <iostream>
#include <sstream>
#include <string>

int main() {
    std::string str = "123 456 789";
    std::istringstream iss(str);
    int a, b, c;
    iss >> a >> b >> c;
    std::cout << "Extracted numbers: " << a << ", " << b << ", " << c << std::endl;
    return 0;
}

Regular Expressions

  • Regular expressions provide a powerful way to search and manipulate strings.
#include <iostream>
#include <regex>
#include <string>

int main() {
    std::string str = "The quick brown fox jumps over the lazy dog.";
    std::regex pattern("(\\w+)"); // Matches words
    std::sregex_iterator begin(str.begin(), str.end(), pattern);
    std::sregex_iterator end;
    for (std::sregex_iterator i = begin; i != end; ++i) {
        std::smatch match = *i;
        std::cout << match.str() << std::endl;
    }
    return 0;
}

Conclusion

String handling in C++ ranges from basic operations with C-style strings to more advanced manipulations using the std::string class and other utilities like string streams and regular expressions. Understanding these concepts is crucial for effective text processing and manipulation in your programs. Practice these techniques to become proficient in handling strings in C++.
