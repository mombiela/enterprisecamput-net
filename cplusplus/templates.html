Templates in C++
Exception Handling ►

Templates are a powerful feature in C++ that allow you to write generic and reusable code. They enable functions and classes to operate with generic types, which can be specified when the function or class is instantiated. This section will guide you from the basics of templates to more advanced uses.

Introduction to Templates

Templates allow you to define functions and classes with a placeholder for the data type. This makes your code more flexible and reusable.

Function Templates

Function templates allow you to create a function that can operate on different data types without rewriting the entire function for each type.

Example:

#include <iostream>
using namespace std;

template <typename T>
T add(T a, T b) {
    return a + b;
}

int main() {
    cout << add(3, 4) << endl;       // Outputs 7
    cout << add(3.5, 4.5) << endl;   // Outputs 8.0
    return 0;
}
  • template <typename T>: This line declares a template with a placeholder type T.
  • T add(T a, T b): This function takes two parameters of type T and returns a value of type T.

Class Templates

Class templates allow you to create a class that can handle any data type.

Example:

#include <iostream>
using namespace std;

template <typename T>
class Box {
private:
    T value;
public:
    Box(T val) : value(val) {}
    T getValue() { return value; }
};

int main() {
    Box<int> intBox(123);
    Box<double> doubleBox(456.78);

    cout << intBox.getValue() << endl;       // Outputs 123
    cout << doubleBox.getValue() << endl;    // Outputs 456.78
    return 0;
}
  • template <typename T>: This line declares a template with a placeholder type T.
  • Box<T>: This class can now handle any data type specified when an object is created.

Advanced Templates

Template Specialization

Template specialization allows you to define a different implementation of a template for a specific data type.

Example:

#include <iostream>
using namespace std;

template <typename T>
class Box {
public:
    T value;
    Box(T val) : value(val) {}
    void print() {
        cout << "Generic template: " << value << endl;
    }
};

// Specialization for int
template <>
class Box<int> {
public:
    int value;
    Box(int val) : value(val) {}
    void print() {
        cout << "Specialized template for int: " << value << endl;
    }
};

int main() {
    Box<double> doubleBox(3.14);
    Box<int> intBox(123);

    doubleBox.print();  // Outputs: Generic template: 3.14
    intBox.print();     // Outputs: Specialized template for int: 123

    return 0;
}
  • template <> class Box<int>: This line declares a specialized version of the Box class for int type.
  • The specialized class can have a different implementation from the generic template.

Template Parameters

Templates can have non-type parameters, which allow you to pass values to templates.

Example:

#include <iostream>
using namespace std;

template <typename T, int size>
class Array {
private:
    T arr[size];
public:
    void set(int index, T value) {
        if (index >= 0 && index < size) {
            arr[index] = value;
        }
    }
    T get(int index) {
        if (index >= 0 && index < size) {
            return arr[index];
        }
        return T(); // Return default value of T
    }
};

int main() {
    Array<int, 5> intArray;
    intArray.set(0, 10);
    intArray.set(1, 20);

    cout << intArray.get(0) << endl;  // Outputs 10
    cout << intArray.get(1) << endl;  // Outputs 20

    return 0;
}
  • template <typename T, int size>: This line declares a template with a type parameter T and a non-type parameter size.
  • Array<int, 5>: This creates an Array object that can hold 5 integers.

Conclusion

Templates in C++ provide a powerful way to write generic, reusable code. They allow you to create functions and classes that can operate with any data type, reducing redundancy and increasing flexibility. By understanding and utilizing templates, you can write more efficient and maintainable C++ programs.
