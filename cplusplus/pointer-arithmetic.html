Pointer Arithmetic in C++
Pointers to Pointers ►

Pointer arithmetic is a powerful feature in C++ that allows you to perform operations on pointers to navigate through arrays and other data structures. This topic will cover the basics of pointer arithmetic, including incrementing, decrementing, and the difference between pointers.

Key Concepts

  • Pointer Basics: Understanding what pointers are and how they work.
  • Pointer Increment and Decrement: How to move pointers through memory.
  • Pointer Addition and Subtraction: Performing arithmetic operations on pointers.
  • Pointer Comparison: Comparing pointers to determine their relative positions in memory.
  • Pointer and Array Relationship: How pointers and arrays are closely related in C++.

Pointer Basics

A pointer is a variable that stores the memory address of another variable. Pointers are declared using the * operator.

int var = 10;
int *ptr = &var; // ptr now holds the address of var

Pointer Increment and Decrement

Pointers can be incremented (++) or decremented (--) to point to the next or previous memory location of the type they point to.

int arr[] = {10, 20, 30, 40};
int *ptr = arr; // ptr points to the first element of arr

ptr++; // Now ptr points to the second element of arr
std::cout << *ptr << std::endl; // Outputs 20

ptr--; // Now ptr points back to the first element of arr
std::cout << *ptr << std::endl; // Outputs 10

Pointer Addition and Subtraction

You can add or subtract integers to/from pointers to move them by multiple elements.

int arr[] = {10, 20, 30, 40};
int *ptr = arr; // ptr points to the first element of arr

ptr = ptr + 2; // Now ptr points to the third element of arr
std::cout << *ptr << std::endl; // Outputs 30

ptr = ptr - 1; // Now ptr points to the second element of arr
std::cout << *ptr << std::endl; // Outputs 20

Pointer Comparison

Pointers can be compared using relational operators (==, !=, <, >, <=, >=). This is useful for determining the relative positions of elements in an array.

int arr[] = {10, 20, 30, 40};
int *ptr1 = &arr[1]; // Points to the second element
int *ptr2 = &arr[3]; // Points to the fourth element

if (ptr1 < ptr2) {
    std::cout << "ptr1 points to an earlier element than ptr2" << std::endl;
} else {
    std::cout << "ptr1 does not point to an earlier element than ptr2" << std::endl;
}

Pointer and Array Relationship

In C++, arrays and pointers are closely related. The name of an array acts as a pointer to its first element.

int arr[] = {10, 20, 30, 40};
int *ptr = arr; // Equivalent to int *ptr = &arr[0];

std::cout << *ptr << std::endl; // Outputs 10
std::cout << *(ptr + 1) << std::endl; // Outputs 20

Example: Iterating Through an Array Using Pointers

#include <iostream>

int main() {
    int arr[] = {10, 20, 30, 40, 50};
    int *ptr = arr; // Points to the first element of arr

    for (int i = 0; i < 5; ++i) {
        std::cout << *(ptr + i) << " "; // Access each element using pointer arithmetic
    }

    return 0;
}

Conclusion

Pointer arithmetic is a fundamental concept in C++ that allows for efficient manipulation of arrays and other data structures. By understanding how to increment, decrement, add, subtract, and compare pointers, you can navigate through memory effectively. Remember that pointer arithmetic is always performed in terms of the size of the data type to which the pointer points, ensuring that you move correctly through memory locations.
