Pointers to Pointers in C++
References ►

Introduction

Pointers are a fundamental concept in C++ that allow you to directly manipulate memory. A pointer to a pointer is a form of multiple indirection or a chain of pointers. This concept is crucial for understanding complex data structures and dynamic memory allocation.

Key Concepts

  • Pointer Basics: A pointer is a variable that stores the memory address of another variable.
  • Pointer to Pointer: A pointer to a pointer is a variable that stores the address of another pointer.

Understanding Pointers to Pointers

Basic Pointer Recap

Before diving into pointers to pointers, let's quickly recap basic pointers.

int var = 10;
int *ptr = &var; // ptr holds the address of var

In this example:

  • var is an integer variable.
  • ptr is a pointer to an integer, holding the address of var.

Pointer to Pointer

A pointer to a pointer stores the address of another pointer.

int var = 10;
int *ptr = &var; // ptr holds the address of var
int **pptr = &ptr; // pptr holds the address of ptr

In this example:

  • var is an integer variable.
  • ptr is a pointer to an integer, holding the address of var.
  • pptr is a pointer to a pointer, holding the address of ptr.

Visual Representation

To better understand, let's visualize the memory addresses:

| Variable | Value | Address | |----------|-------|---------| | var | 10 | 0x100 | | ptr | 0x100 | 0x200 | | pptr | 0x200 | 0x300 |

Accessing Values

To access the value of var using pptr:

#include <iostream>
using namespace std;

int main() {
    int var = 10;
    int *ptr = &var;
    int **pptr = &ptr;

    cout << "Value of var: " << var << endl;
    cout << "Value using ptr: " << *ptr << endl;
    cout << "Value using pptr: " << **pptr << endl;

    return 0;
}

Output:

Value of var: 10
Value using ptr: 10
Value using pptr: 10

Practical Applications

Dynamic Memory Allocation

Pointers to pointers are often used in dynamic memory allocation, especially for multi-dimensional arrays.

#include <iostream>
using namespace std;

int main() {
    int rows = 3, cols = 4;
    int **arr = new int*[rows];

    for(int i = 0; i < rows; ++i)
        arr[i] = new int[cols];

    // Assign values
    for(int i = 0; i < rows; ++i)
        for(int j = 0; j < cols; ++j)
            arr[i][j] = i + j;

    // Print values
    for(int i = 0; i < rows; ++i) {
        for(int j = 0; j < cols; ++j)
            cout << arr[i][j] << " ";
        cout << endl;
    }

    // Deallocate memory
    for(int i = 0; i < rows; ++i)
        delete[] arr[i];
    delete[] arr;

    return 0;
}

Output:

0 1 2 3 
1 2 3 4 
2 3 4 5

Function Pointers

Pointers to pointers can also be used with function pointers to create more complex and flexible code structures.

#include <iostream>
using namespace std;

void func(int **pptr) {
    **pptr = 20;
}

int main() {
    int var = 10;
    int *ptr = &var;
    int **pptr = &ptr;

    cout << "Before func call: " << var << endl;
    func(pptr);
    cout << "After func call: " << var << endl;

    return 0;
}

Output:

Before func call: 10
After func call: 20

Conclusion

Pointers to pointers are a powerful feature in C++ that allow for multiple levels of indirection. They are essential for dynamic memory allocation, handling multi-dimensional arrays, and creating complex data structures. Understanding this concept will significantly enhance your ability to write efficient and flexible C++ programs.
