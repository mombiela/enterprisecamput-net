Lambda Expressions in C++
Move Semantics ►

Lambda expressions, also known as lambda functions or closures, are a powerful feature in C++ that allow you to define anonymous functions directly within your code. They are particularly useful for short, throwaway functions that are not reused elsewhere.

Introduction to Lambda Expressions

  • Definition: A lambda expression is an anonymous function that can capture variables from its surrounding scope.
  • Syntax: The basic syntax of a lambda expression is: 
    [capture](parameters) -> return_type {
    // function body
};

Basic Syntax and Usage

  • Capture Clause: The capture clause (the square brackets []) is used to specify which variables from the surrounding scope are accessible inside the lambda.
  • Parameters: The parameter list is similar to that of a regular function.
  • Return Type: The return type is optional and can often be inferred by the compiler.
  • Function Body: The function body contains the code to be executed when the lambda is called.

Example: Basic Lambda Expression

#include <iostream>

int main() {
    auto add = [](int a, int b) -> int {
        return a + b;
    };

    std::cout << "Sum: " << add(3, 4) << std::endl; // Output: Sum: 7
    return 0;
}

In this example:

  • auto add declares a lambda that takes two integers and returns their sum.
  • The lambda is called with the arguments 3 and 4, and the result is printed.

Capturing Variables

Lambda expressions can capture variables from their surrounding scope in several ways:

  • By Value: Captures a copy of the variable.
  • By Reference: Captures a reference to the variable.
  • Implicit Capture: Automatically captures variables used in the lambda by value or by reference.

Example: Capturing by Value and Reference

#include <iostream>

int main() {
    int x = 10;
    int y = 20;

    auto captureByValue = [x, y]() {
        return x + y;
    };

    auto captureByReference = [&x, &y]() {
        return x + y;
    };

    x = 15;
    y = 25;

    std::cout << "Capture by value: " << captureByValue() << std::endl; // Output: 30
    std::cout << "Capture by reference: " << captureByReference() << std::endl; // Output: 40

    return 0;
}

In this example:

  • captureByValue captures x and y by value, so changes to x and y after the lambda is defined do not affect the result.
  • captureByReference captures x and y by reference, so changes to x and y are reflected in the lambda's result.

Advanced Features

Mutable Lambdas

By default, lambdas that capture variables by value are not allowed to modify those variables. To allow modification, you can use the mutable keyword.

#include <iostream>

int main() {
    int x = 10;

    auto mutableLambda = [x]() mutable {
        x += 5;
        return x;
    };

    std::cout << "Mutable lambda: " << mutableLambda() << std::endl; // Output: 15
    std::cout << "Original x: " << x << std::endl; // Output: 10

    return 0;
}

Generic Lambdas

C++14 introduced generic lambdas, which allow you to use auto in the parameter list to create a lambda that can accept arguments of any type.

#include <iostream>

int main() {
    auto genericLambda = [](auto a, auto b) {
        return a + b;
    };

    std::cout << "Sum of integers: " << genericLambda(3, 4) << std::endl; // Output: 7
    std::cout << "Sum of doubles: " << genericLambda(3.5, 4.5) << std::endl; // Output: 8

    return 0;
}

Practical Applications

Using Lambdas with Standard Algorithms

Lambdas are often used with standard algorithms like std::sort, std::for_each, and std::transform.

#include <iostream>
#include <vector>
#include <algorithm>

int main() {
    std::vector<int> numbers = {1, 2, 3, 4, 5};

    std::for_each(numbers.begin(), numbers.end(), [](int &n) {
        n *= 2;
    });

    for (int n : numbers) {
        std::cout << n << " "; // Output: 2 4 6 8 10
    }

    return 0;
}

In this example:

  • std::for_each applies the lambda to each element in the vector, doubling each value.

Conclusion

Lambda expressions in C++ provide a concise and powerful way to define anonymous functions. They are particularly useful for short, one-off functions and can capture variables from their surrounding scope. Understanding how to use lambdas effectively can greatly enhance your ability to write clean and efficient C++ code.
