What is a Function?

  • A function is a reusable block of code that performs a specific task.
  • Functions help in breaking down complex problems into smaller, manageable pieces.
  • They promote code reusability and improve readability.

Syntax of a Function

  • A function in C++ consists of a return type, a function name, and a parameter list enclosed in parentheses.
  • The body of the function is enclosed in curly braces {}.

Example:

#include <iostream>

// Function declaration
int add(int a, int b);

int main() {
    int result = add(5, 3); // Function call
    std::cout << "The sum is: " << result << std::endl;
    return 0;
}

// Function definition
int add(int a, int b) {
    return a + b;
}

Function Declaration and Definition

  • Function Declaration (Prototype): Tells the compiler about the function name, return type, and parameters. It ends with a semicolon.
  • Function Definition: Contains the actual body of the function.

Example:

// Declaration
int multiply(int, int);

// Definition
int multiply(int x, int y) {
    return x * y;
}

Return Type

  • The return type specifies the type of value the function will return.
  • If a function does not return any value, its return type is void.

Example:

void printMessage() {
    std::cout << "Hello, World!" << std::endl;
}

Parameters and Arguments

  • Parameters: Variables listed in the function declaration/definition.
  • Arguments: Values passed to the function when it is called.

Example:

void greet(std::string name) {
    std::cout << "Hello, " << name << "!" << std::endl;
}

int main() {
    greet("Alice"); // Argument: "Alice"
    return 0;
}

Default Parameters

  • C++ allows functions to have default parameter values.
  • If no argument is provided, the default value is used.

Example:

void display(int x = 10) {
    std::cout << "Value: " << x << std::endl;
}

int main() {
    display();    // Uses default value 10
    display(5);   // Uses provided value 5
    return 0;
}

Function Overloading

  • Function overloading allows multiple functions to have the same name with different parameters.
  • The compiler differentiates them by their parameter lists.

Example:

void print(int i) {
    std::cout << "Integer: " << i << std::endl;
}

void print(double f) {
    std::cout << "Double: " << f << std::endl;
}

void print(std::string s) {
    std::cout << "String: " << s << std::endl;
}

int main() {
    print(10);
    print(3.14);
    print("Hello");
    return 0;
}

Inline Functions

  • Inline functions are defined with the inline keyword.
  • They suggest to the compiler to insert the function's body where the function call is, to reduce the overhead of a function call.

Example:

inline int square(int x) {
    return x * x;
}

int main() {
    std::cout << "Square of 5: " << square(5) << std::endl;
    return 0;
}

Recursion

  • A function that calls itself is known as a recursive function.
  • Recursion is useful for tasks that can be defined in terms of similar subtasks.

Example:

int factorial(int n) {
    if (n <= 1) return 1;
    else return n * factorial(n - 1);
}

int main() {
    std::cout << "Factorial of 5: " << factorial(5) << std::endl;
    return 0;
}

Summary

  • Functions are essential for breaking down complex problems and promoting code reusability.
  • Understanding function declaration, definition, parameters, return types, and overloading is crucial.
  • Inline functions and recursion are advanced concepts that offer performance benefits and elegant solutions to certain problems.
  • Practice writing and using functions to become proficient in structuring your C++ programs effectively.
