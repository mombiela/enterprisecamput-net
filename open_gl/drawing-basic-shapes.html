Introduction

In this section, we will explore how to draw basic shapes using OpenGL. This is fundamental for anyone starting with OpenGL as it lays the groundwork for more complex graphics programming. We will cover the following shapes:

  • Points
  • Lines
  • Triangles
  • Quads (Quadrilaterals)

Setting Up OpenGL

Before we start drawing shapes, ensure you have a basic OpenGL setup. This includes initializing OpenGL context, creating a window, and setting up a rendering loop. Here is a minimal setup using GLFW and GLEW:

#include <GL/glew.h>
#include <GLFW/glfw3.h>

int main() {
    // Initialize GLFW
    if (!glfwInit()) {
        return -1;
    }

    // Create a windowed mode window and its OpenGL context
    GLFWwindow* window = glfwCreateWindow(640, 480, "OpenGL Setup", NULL, NULL);
    if (!window) {
        glfwTerminate();
        return -1;
    }

    // Make the window's context current
    glfwMakeContextCurrent(window);

    // Initialize GLEW
    if (glewInit() != GLEW_OK) {
        return -1;
    }

    // Main loop
    while (!glfwWindowShouldClose(window)) {
        // Render here

        // Swap front and back buffers
        glfwSwapBuffers(window);

        // Poll for and process events
        glfwPollEvents();
    }

    glfwTerminate();
    return 0;
}

Drawing Points

Points are the simplest primitive in OpenGL. To draw a point, you need to specify its coordinates.

Example

void drawPoint() {
    glBegin(GL_POINTS); // Start drawing points
    glVertex2f(0.0f, 0.0f); // Specify the point's coordinates
    glEnd(); // End drawing points
}

Explanation

  • glBegin(GL_POINTS); tells OpenGL that we are starting to draw points.
  • glVertex2f(0.0f, 0.0f); specifies the coordinates of the point.
  • glEnd(); ends the drawing process.

Drawing Lines

Lines are defined by two points. OpenGL provides several types of lines, but we'll start with the basic line.

Example

void drawLine() {
    glBegin(GL_LINES); // Start drawing lines
    glVertex2f(-0.5f, 0.0f); // First point
    glVertex2f(0.5f, 0.0f); // Second point
    glEnd(); // End drawing lines
}

Explanation

  • glBegin(GL_LINES); tells OpenGL that we are starting to draw lines.
  • glVertex2f(-0.5f, 0.0f); specifies the first point of the line.
  • glVertex2f(0.5f, 0.0f); specifies the second point of the line.
  • glEnd(); ends the drawing process.

Drawing Triangles

Triangles are one of the most important primitives in OpenGL, as they are used to construct more complex shapes.

Example

void drawTriangle() {
    glBegin(GL_TRIANGLES); // Start drawing triangles
    glVertex2f(0.0f, 0.5f); // First vertex
    glVertex2f(-0.5f, -0.5f); // Second vertex
    glVertex2f(0.5f, -0.5f); // Third vertex
    glEnd(); // End drawing triangles
}

Explanation

  • glBegin(GL_TRIANGLES); tells OpenGL that we are starting to draw triangles.
  • glVertex2f(0.0f, 0.5f); specifies the first vertex of the triangle.
  • glVertex2f(-0.5f, -0.5f); specifies the second vertex of the triangle.
  • glVertex2f(0.5f, -0.5f); specifies the third vertex of the triangle.
  • glEnd(); ends the drawing process.

Drawing Quads

Quads are defined by four vertices. However, modern OpenGL has deprecated quads in favor of using two triangles to form a quad.

Example

void drawQuad() {
    glBegin(GL_QUADS); // Start drawing quads
    glVertex2f(-0.5f, 0.5f); // First vertex
    glVertex2f(0.5f, 0.5f); // Second vertex
    glVertex2f(0.5f, -0.5f); // Third vertex
    glVertex2f(-0.5f, -0.5f); // Fourth vertex
    glEnd(); // End drawing quads
}

Explanation

  • glBegin(GL_QUADS); tells OpenGL that we are starting to draw quads.
  • glVertex2f(-0.5f, 0.5f); specifies the first vertex of the quad.
  • glVertex2f(0.5f, 0.5f); specifies the second vertex of the quad.
  • glVertex2f(0.5f, -0.5f); specifies the third vertex of the quad.
  • glVertex2f(-0.5f, -0.5f); specifies the fourth vertex of the quad.
  • glEnd(); ends the drawing process.

Conclusion

In this section, we covered the basics of drawing shapes in OpenGL, including points, lines, triangles, and quads. Understanding these primitives is crucial as they form the foundation for more complex graphics programming. Practice drawing these shapes and experiment with different coordinates to get comfortable with OpenGL's coordinate system and rendering pipeline.
