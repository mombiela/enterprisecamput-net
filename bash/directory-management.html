Directory management is a fundamental skill when working with Bash. This topic will cover essential commands and techniques for navigating and manipulating directories in a Unix-like operating system.

Navigating Directories

Navigating through directories is one of the first tasks you'll need to master in Bash.

`pwd` - Print Working Directory

The pwd command displays the current directory you are in.

pwd

Example output:

/home/user

`cd` - Change Directory

The cd command is used to change the current directory.

cd /path/to/directory

Example:

cd /home/user/Documents

`ls` - List Directory Contents

The ls command lists the contents of a directory.

ls

Example:

ls /home/user/Documents

Common `ls` Options

  • -l: Long listing format
  • -a: Include hidden files
  • -h: Human-readable file sizes

Example:

ls -lah /home/user/Documents

Creating and Removing Directories

`mkdir` - Make Directory

The mkdir command creates a new directory.

mkdir new_directory

Example:

mkdir /home/user/new_folder

`rmdir` - Remove Directory

The rmdir command removes an empty directory.

rmdir empty_directory

Example:

rmdir /home/user/old_folder

`rm -r` - Remove Directory and Contents

The rm -r command removes a directory and its contents recursively.

rm -r directory_with_contents

Example:

rm -r /home/user/folder_to_delete

Moving and Renaming Directories

`mv` - Move or Rename

The mv command is used to move or rename directories.

mv old_directory new_directory

Example:

mv /home/user/old_folder /home/user/new_folder

Copying Directories

`cp -r` - Copy Directory and Contents

The cp -r command copies a directory and its contents recursively.

cp -r source_directory destination_directory

Example:

cp -r /home/user/source_folder /home/user/destination_folder

Viewing Directory Structure

`tree` - Directory Tree View

The tree command displays directories and files in a tree-like format. Note that tree might need to be installed on your system.

tree

Example:

tree /home/user/Documents

Summary

In this topic, we covered the essential commands for directory management in Bash:

  • pwd to print the current working directory.
  • cd to change directories.
  • ls to list directory contents.
  • mkdir to create new directories.
  • rmdir and rm -r to remove directories.
  • mv to move or rename directories.
  • cp -r to copy directories.
  • tree to view directory structures.

Mastering these commands will significantly enhance your ability to navigate and manage the filesystem efficiently. Practice these commands to become comfortable with directory management in Bash.
