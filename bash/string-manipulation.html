String Manipulation in Bash
Arrays in Bash ►

String manipulation is a fundamental skill in Bash scripting. This topic will guide you from basic to advanced string operations, providing clear explanations, examples, and exercises to solidify your understanding.

Basic String Operations

Concatenation

Concatenation is the process of joining two or more strings together.

str1="Hello"
str2="World"
result="$str1 $str2"
echo $result  # Output: Hello World

String Length

You can find the length of a string using the ${#string} syntax.

str="Hello"
length=${#str}
echo $length  # Output: 5

Substring Extraction

Extract a substring using the ${string:position:length} syntax.

str="Hello World"
substr=${str:6:5}
echo $substr  # Output: World

Intermediate String Operations

String Replacement

Replace a substring within a string using the ${string/substring/replacement} syntax.

str="Hello World"
new_str=${str/World/Bash}
echo $new_str  # Output: Hello Bash

Case Conversion

Convert strings to uppercase or lowercase using tr.

str="Hello World"
uppercase=$(echo $str | tr '[:lower:]' '[:upper:]')
echo $uppercase  # Output: HELLO WORLD

lowercase=$(echo $str | tr '[:upper:]' '[:lower:]')
echo $lowercase  # Output: hello world

String Splitting

Split a string into an array based on a delimiter.

str="apple,banana,cherry"
IFS=',' read -r -a array <<< "$str"
echo ${array[0]}  # Output: apple
echo ${array[1]}  # Output: banana
echo ${array[2]}  # Output: cherry

Advanced String Operations

Regular Expressions

Use regular expressions for complex string matching and manipulation.

str="Hello123"
if [[ $str =~ [0-9]+ ]]; then
  echo "String contains numbers"
else
  echo "String does not contain numbers"
fi
# Output: String contains numbers

String Comparison

Compare strings using conditional expressions.

str1="Hello"
str2="World"

if [ "$str1" == "$str2" ]; then
  echo "Strings are equal"
else
  echo "Strings are not equal"
fi
# Output: Strings are not equal

Advanced String Replacement

Use parameter expansion for more complex replacements.

str="Hello World World"
new_str=${str//World/Bash}
echo $new_str  # Output: Hello Bash Bash

Summary

String manipulation in Bash is a powerful tool for any programmer. From basic operations like concatenation and length calculation to more advanced techniques like regular expressions and complex replacements, mastering these skills will greatly enhance your scripting capabilities. Practice the provided examples and exercises to become proficient in handling strings in Bash.
