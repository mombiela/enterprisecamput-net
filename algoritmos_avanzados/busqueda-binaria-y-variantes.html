Introduction to Binary Search

Binary search is an efficient algorithm for finding an element in an ordered list. Its efficiency lies in repeatedly dividing the search range in half, significantly reducing the number of comparisons needed.

  • Purpose: Find the position of an element in an ordered list.
  • Time Complexity: O(log n), where n is the number of elements in the list.
  • Requirements: The list must be previously ordered.

Basic Binary Search Algorithm

The binary search algorithm follows these steps:

  1. Initialize two indices, low and high, representing the search range limits.
  2. Calculate the middle index (mid) of the current range.
  3. Compare the element at mid with the target value (target).
  4. If the target is equal to the element at mid, return mid.
  5. If the target is less than the element at mid, adjust high to mid - 1.
  6. If the target is greater than the element at mid, adjust low to mid + 1.
  7. Repeat steps 2-6 until low is greater than high.

Example Code in Python

def binary_search(arr, target):
    low = 0
    high = len(arr) - 1

    while low <= high:
        mid = (low + high) // 2
        if arr[mid] == target:
            return mid
        elif arr[mid] < target:
            low = mid + 1
        else:
            high = mid - 1

    return -1  # Element not found

# Example usage
arr = [1, 3, 5, 7, 9, 11]
target = 7
result = binary_search(arr, target)
print(f'Element found at index: {result}')

Variants of Binary Search

Recursive Binary Search

Binary search can also be implemented recursively. The logic is similar to the iterative version, but a function calls itself with updated parameters.

Example Code in Python

def recursive_binary_search(arr, target, low, high):
    if low > high:
        return -1  # Element not found

    mid = (low + high) // 2
    if arr[mid] == target:
        return mid
    elif arr[mid] < target:
        return recursive_binary_search(arr, target, mid + 1, high)
    else:
        return recursive_binary_search(arr, target, low, mid - 1)

# Example usage
arr = [1, 3, 5, 7, 9, 11]
target = 7
result = recursive_binary_search(arr, target, 0, len(arr) - 1)
print(f'Element found at index: {result}')

Binary Search for the First and Last Element

In some cases, it is necessary to find the first or last occurrence of an element in an ordered list that may contain duplicate elements.

Search for the First Element

def binary_search_first(arr, target):
    low = 0
    high = len(arr) - 1
    result = -1

    while low <= high:
        mid = (low + high) // 2
        if arr[mid] == target:
            result = mid
            high = mid - 1  # Continue searching in the left half
        elif arr[mid] < target:
            low = mid + 1
        else:
            high = mid - 1

    return result

# Example usage
arr = [1, 3, 5, 7, 7, 7, 9, 11]
target = 7
result = binary_search_first(arr, target)
print(f'First occurrence of the element at index: {result}')

Search for the Last Element

def binary_search_last(arr, target):
    low = 0
    high = len(arr) - 1
    result = -1

    while low <= high:
        mid = (low + high) // 2
        if arr[mid] == target:
            result = mid
            low = mid + 1  # Continue searching in the right half
        elif arr[mid] < target:
            low = mid + 1
        else:
            high = mid - 1

    return result

# Example usage
arr = [1, 3, 5, 7, 7, 7, 9, 11]
target = 7
result = binary_search_last(arr, target)
print(f'Last occurrence of the element at index: {result}')

Conclusion

Binary search is a fundamental algorithm in computer science due to its efficiency and simplicity. Understanding both the iterative and recursive implementations, as well as its variants for finding the first and last element, is crucial for solving search problems in ordered lists. Practicing these algorithms with different types of data and scenarios will help consolidate their understanding and application in real-world situations.
