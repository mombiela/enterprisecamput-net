Introduction

Webpack is a powerful module bundler for JavaScript applications. It takes modules with dependencies and generates static assets representing those modules. This section will guide you through setting up your first Webpack project from scratch.

Prerequisites

Before you start, ensure you have the following installed:

  • Node.js (LTS version recommended)
  • npm (comes with Node.js)

Step-by-Step Guide

1. Initialize Your Project

First, create a new directory for your project and navigate into it:

mkdir my-webpack-project
cd my-webpack-project

Initialize a new Node.js project:

npm init -y

This command creates a package.json file with default settings.

2. Install Webpack and Webpack CLI

Next, install Webpack and Webpack CLI as development dependencies:

npm install --save-dev webpack webpack-cli

3. Create the Project Structure

Create the following directory structure:

my-webpack-project/
├── src/
│   └── index.js
├── dist/
├── package.json
└── webpack.config.js

4. Configure Webpack

Create a webpack.config.js file in the root of your project:

const path = require('path');

module.exports = {
  entry: './src/index.js',
  output: {
    filename: 'bundle.js',
    path: path.resolve(__dirname, 'dist'),
  },
  mode: 'development',
};
  • entry: The entry point of your application.
  • output: Where Webpack should output the bundled files.
  • mode: Set to 'development' for easier debugging.

5. Add a Script to package.json

Add a build script to your package.json:

"scripts": {
  "build": "webpack"
}

6. Create a Simple JavaScript File

Create a simple index.js file in the src directory:

console.log('Hello, Webpack!');

7. Build the Project

Run the build script:

npm run build

This command will generate a bundle.js file in the dist directory.

8. Create an HTML File

Create an index.html file in the dist directory to include the bundled JavaScript:

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Webpack Project</title>
</head>
<body>
  <script src="bundle.js"></script>
</body>
</html>

9. Open the HTML File

Open index.html in your browser to see the output. You should see "Hello, Webpack!" in the console.

Summary

In this section, you learned how to set up a basic Webpack project. You:

  • Initialized a Node.js project.
  • Installed Webpack and Webpack CLI.
  • Created a project structure.
  • Configured Webpack.
  • Built and served a simple JavaScript application.

This foundational setup will serve as the basis for more advanced Webpack configurations and optimizations in future lessons.
