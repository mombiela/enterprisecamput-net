Introduction to Haskell

Haskell is a purely functional programming language with strong static typing and lazy evaluation. It is named after the logician Haskell Curry. Haskell is known for its expressive type system, high-level abstractions, and concise syntax, making it a powerful tool for both academic research and practical software development.

Key Features of Haskell

  • Purely Functional: Functions in Haskell are pure, meaning they do not have side effects. This leads to more predictable and maintainable code.
  • Strong Static Typing: Haskell uses a robust type system to catch errors at compile time, reducing runtime errors.
  • Lazy Evaluation: Haskell evaluates expressions only when their values are needed, which can lead to performance improvements and the ability to work with infinite data structures.
  • Immutability: Data in Haskell is immutable by default, which simplifies reasoning about code and enhances concurrency.

Haskell Syntax and Basics

Hello World Example

Let's start with a simple "Hello, World!" program in Haskell:

main :: IO ()
main = putStrLn "Hello, World!"
  • main is the entry point of a Haskell program.
  • putStrLn is a function that outputs a string followed by a newline.
  • IO () indicates that main performs an input/output action and returns a unit type ().

Basic Data Types

Haskell has several basic data types:

  • Int: Fixed-precision integer.
  • Integer: Arbitrary-precision integer.
  • Float: Single-precision floating-point number.
  • Double: Double-precision floating-point number.
  • Char: A single character.
  • Bool: Boolean type with values True or False.
  • String: A list of characters.

Example:

myInt :: Int
myInt = 42

myFloat :: Float
myFloat = 3.14

myChar :: Char
myChar = 'a'

myBool :: Bool
myBool = True

myString :: String
myString = "Hello, Haskell!"

Functions

Functions are first-class citizens in Haskell. Here's how you define and use functions:

-- Function to add two numbers
add :: Int -> Int -> Int
add x y = x + y

-- Using the function
result = add 5 3  -- result is 8
  • add is a function that takes two Int arguments and returns an Int.
  • The type signature Int -> Int -> Int indicates that add takes two Int values and returns an Int.

Lists

Lists are a fundamental data structure in Haskell. They are homogeneous, meaning all elements must be of the same type.

-- A list of integers
numbers :: [Int]
numbers = [1, 2, 3, 4, 5]

-- A list of characters (a string)
chars :: [Char]
chars = ['H', 'a', 's', 'k', 'e', 'l', 'l']

Pattern Matching

Pattern matching is a powerful feature in Haskell that allows you to destructure data types.

-- Function to get the head of a list
headOfList :: [a] -> a
headOfList (x:_) = x
headOfList [] = error "Empty list"

-- Using the function
firstElement = headOfList [1, 2, 3]  -- firstElement is 1
  • headOfList uses pattern matching to extract the head of a list.
  • (x:_) matches a non-empty list, binding x to the head.
  • [] matches an empty list and raises an error.

Conclusion

Haskell is a powerful and expressive language that emphasizes functional programming principles. Its strong type system, lazy evaluation, and immutability make it a unique and valuable tool for both academic and practical applications. By understanding the basics of Haskell, such as its syntax, data types, and core features, you can start to appreciate the elegance and power of functional programming.
