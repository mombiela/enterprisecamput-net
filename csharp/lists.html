Introduction to Lists

Lists in C# are a part of the System.Collections.Generic namespace and provide a way to store collections of objects. Unlike arrays, lists can dynamically grow and shrink in size.

  • Namespace: System.Collections.Generic
  • Class: List<T>

Key Concepts

  • Dynamic Sizing: Lists can automatically resize themselves as elements are added or removed.
  • Type Safety: Lists are strongly typed, meaning you can only store elements of a specified type.
  • Zero-based Indexing: Like arrays, lists are zero-indexed.

Creating and Initializing Lists

To create a list, you need to specify the type of elements it will store.

using System;
using System.Collections.Generic;

class Program
{
    static void Main()
    {
        // Creating a list of integers
        List<int> numbers = new List<int>();

        // Initializing a list with values
        List<string> fruits = new List<string> { "Apple", "Banana", "Cherry" };

        // Output the list
        foreach (string fruit in fruits)
        {
            Console.WriteLine(fruit);
        }
    }
}

Explanation

  • List<int> numbers = new List<int>();: Creates an empty list of integers.
  • List<string> fruits = new List<string> { "Apple", "Banana", "Cherry" };: Creates and initializes a list of strings with three elements.

Adding and Removing Elements

You can add elements to a list using the Add method and remove them using the Remove or RemoveAt methods.

class Program
{
    static void Main()
    {
        List<int> numbers = new List<int> { 1, 2, 3 };

        // Adding elements
        numbers.Add(4);
        numbers.Add(5);

        // Removing elements
        numbers.Remove(2); // Removes the first occurrence of 2
        numbers.RemoveAt(0); // Removes the element at index 0

        // Output the list
        foreach (int number in numbers)
        {
            Console.WriteLine(number);
        }
    }
}

Explanation

  • numbers.Add(4);: Adds the element 4 to the end of the list.
  • numbers.Remove(2);: Removes the first occurrence of the element 2.
  • numbers.RemoveAt(0);: Removes the element at index 0.

Accessing Elements

You can access elements in a list using the indexer.

class Program
{
    static void Main()
    {
        List<string> fruits = new List<string> { "Apple", "Banana", "Cherry" };

        // Accessing elements
        string firstFruit = fruits[0];
        string secondFruit = fruits[1];

        Console.WriteLine(firstFruit); // Output: Apple
        Console.WriteLine(secondFruit); // Output: Banana
    }
}

Explanation

  • fruits[0]: Accesses the first element in the list.
  • fruits[1]: Accesses the second element in the list.

Iterating Over a List

You can iterate over a list using a foreach loop or a for loop.

class Program
{
    static void Main()
    {
        List<string> fruits = new List<string> { "Apple", "Banana", "Cherry" };

        // Using foreach loop
        foreach (string fruit in fruits)
        {
            Console.WriteLine(fruit);
        }

        // Using for loop
        for (int i = 0; i < fruits.Count; i++)
        {
            Console.WriteLine(fruits[i]);
        }
    }
}

Explanation

  • foreach (string fruit in fruits): Iterates over each element in the list.
  • for (int i = 0; i < fruits.Count; i++): Iterates over the list using an index.

Advanced Operations

Sorting a List

You can sort a list using the Sort method.

class Program
{
    static void Main()
    {
        List<int> numbers = new List<int> { 3, 1, 4, 1, 5, 9 };

        // Sorting the list
        numbers.Sort();

        // Output the sorted list
        foreach (int number in numbers)
        {
            Console.WriteLine(number);
        }
    }
}

Explanation

  • numbers.Sort();: Sorts the list in ascending order.

Finding Elements

You can find elements in a list using methods like Contains, Find, and FindAll.

class Program
{
    static void Main()
    {
        List<string> fruits = new List<string> { "Apple", "Banana", "Cherry" };

        // Check if the list contains a specific element
        bool hasApple = fruits.Contains("Apple");

        // Find the first element that matches a condition
        string fruitWithA = fruits.Find(f => f.Contains("a"));

        // Find all elements that match a condition
        List<string> fruitsWithA = fruits.FindAll(f => f.Contains("a"));

        Console.WriteLine(hasApple); // Output: True
        Console.WriteLine(fruitWithA); // Output: Banana
        foreach (string fruit in fruitsWithA)
        {
            Console.WriteLine(fruit); // Output: Banana, Cherry
        }
    }
}

Explanation

  • fruits.Contains("Apple");: Checks if "Apple" is in the list.
  • fruits.Find(f => f.Contains("a"));: Finds the first element that contains the letter "a".
  • fruits.FindAll(f => f.Contains("a"));: Finds all elements that contain the letter "a".

Conclusion

Lists in C# are a versatile and powerful way to manage collections of objects. They offer dynamic sizing, type safety, and a variety of methods for adding, removing, accessing, and manipulating elements. By understanding and utilizing lists, you can write more efficient and flexible C# programs.
