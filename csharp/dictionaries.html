Dictionaries in C#
Stacks and Queues ►

Dictionaries are a fundamental data structure in C# that allow you to store key-value pairs. They are highly efficient for lookups, insertions, and deletions. This topic will guide you from the basics to advanced usage of dictionaries in C#.

Introduction to Dictionaries

  • Definition: A dictionary is a collection of key-value pairs where each key is unique.
  • Namespace: System.Collections.Generic
  • Class: Dictionary<TKey, TValue>

Creating a Dictionary

To create a dictionary, you need to specify the types for the keys and values.

using System;
using System.Collections.Generic;

class Program
{
    static void Main()
    {
        // Creating a dictionary with string keys and int values
        Dictionary<string, int> ages = new Dictionary<string, int>();
    }
}

Adding Elements

You can add elements to a dictionary using the Add method or the indexer.

ages.Add("Alice", 30);
ages["Bob"] = 25;

Accessing Elements

Access elements by their key using the indexer.

int aliceAge = ages["Alice"];
Console.WriteLine($"Alice's age is {aliceAge}");

Removing Elements

Remove elements using the Remove method.

ages.Remove("Alice");

Checking for Keys and Values

Use ContainsKey and ContainsValue to check for the existence of keys and values.

bool hasAlice = ages.ContainsKey("Alice");
bool hasAge30 = ages.ContainsValue(30);

Iterating Over a Dictionary

You can iterate over a dictionary using a foreach loop.

foreach (KeyValuePair<string, int> kvp in ages)
{
    Console.WriteLine($"Key: {kvp.Key}, Value: {kvp.Value}");
}

Advanced Dictionary Operations

Initializing with Values

You can initialize a dictionary with values using collection initializer syntax.

Dictionary<string, int> ages = new Dictionary<string, int>
{
    { "Alice", 30 },
    { "Bob", 25 },
    { "Charlie", 35 }
};

Updating Values

You can update values by assigning a new value to an existing key.

ages["Bob"] = 26;

Merging Dictionaries

You can merge two dictionaries using the Union method from LINQ.

using System.Linq;

Dictionary<string, int> moreAges = new Dictionary<string, int>
{
    { "David", 40 },
    { "Eve", 22 }
};

var mergedAges = ages.Union(moreAges).ToDictionary(kvp => kvp.Key, kvp => kvp.Value);

Handling Missing Keys

Use TryGetValue to safely handle missing keys.

if (ages.TryGetValue("Alice", out int aliceAge))
{
    Console.WriteLine($"Alice's age is {aliceAge}");
}
else
{
    Console.WriteLine("Alice is not in the dictionary.");
}

Performance Considerations

  • Time Complexity: Average O(1) for lookups, insertions, and deletions.
  • Memory Usage: Can be higher compared to other collections due to the need to store keys and values.

Common Use Cases

  • Configuration Settings: Storing application settings where keys are setting names and values are setting values.
  • Caching: Storing frequently accessed data for quick retrieval.
  • Counting Occurrences: Counting the number of occurrences of items in a collection.

Conclusion

Dictionaries are a versatile and efficient data structure in C#. They provide fast lookups, insertions, and deletions, making them ideal for a wide range of applications. By understanding how to create, manipulate, and iterate over dictionaries, you can leverage their power in your C# programs.
