Operators in C#

Introduction

Operators are special symbols in C# that perform operations on variables and values. They are fundamental to programming as they allow you to manipulate data and variables in various ways. This section will cover the different types of operators available in C#, from basic arithmetic to more advanced operators.

Arithmetic Operators

Arithmetic operators are used to perform basic mathematical operations.

  • + : Addition
  • - : Subtraction
  • * : Multiplication
  • / : Division
  • % : Modulus (remainder of a division)

Example

int a = 10;
int b = 3;

int sum = a + b;        // 13
int difference = a - b; // 7
int product = a * b;    // 30
int quotient = a / b;   // 3
int remainder = a % b;  // 1

Console.WriteLine($"Sum: {sum}, Difference: {difference}, Product: {product}, Quotient: {quotient}, Remainder: {remainder}");

Relational Operators

Relational operators are used to compare two values.

  • == : Equal to
  • != : Not equal to
  • > : Greater than
  • < : Less than
  • >= : Greater than or equal to
  • <= : Less than or equal to

Example

int a = 10;
int b = 3;

bool isEqual = (a == b);       // false
bool isNotEqual = (a != b);    // true
bool isGreater = (a > b);      // true
bool isLess = (a < b);         // false
bool isGreaterOrEqual = (a >= b); // true
bool isLessOrEqual = (a <= b);    // false

Console.WriteLine($"Equal: {isEqual}, Not Equal: {isNotEqual}, Greater: {isGreater}, Less: {isLess}, Greater or Equal: {isGreaterOrEqual}, Less or Equal: {isLessOrEqual}");

Logical Operators

Logical operators are used to combine conditional statements.

  • && : Logical AND
  • || : Logical OR
  • ! : Logical NOT

Example

bool a = true;
bool b = false;

bool andResult = a && b; // false
bool orResult = a || b;  // true
bool notResult = !a;     // false

Console.WriteLine($"AND: {andResult}, OR: {orResult}, NOT: {notResult}");

Assignment Operators

Assignment operators are used to assign values to variables.

  • = : Simple assignment
  • += : Add and assign
  • -= : Subtract and assign
  • *= : Multiply and assign
  • /= : Divide and assign
  • %= : Modulus and assign

Example

int a = 10;

a += 5; // a = a + 5;  => 15
a -= 3; // a = a - 3;  => 12
a *= 2; // a = a * 2;  => 24
a /= 4; // a = a / 4;  => 6
a %= 4; // a = a % 4;  => 2

Console.WriteLine($"Final value of a: {a}");

Bitwise Operators

Bitwise operators are used to perform operations on binary representations of numbers.

  • & : Bitwise AND
  • | : Bitwise OR
  • ^ : Bitwise XOR
  • ~ : Bitwise NOT
  • << : Left shift
  • >> : Right shift

Example

int a = 5;  // 0101 in binary
int b = 3;  // 0011 in binary

int andResult = a & b; // 0001 => 1
int orResult = a | b;  // 0111 => 7
int xorResult = a ^ b; // 0110 => 6
int notResult = ~a;    // 1010 => -6 (in 2's complement form)
int leftShift = a << 1; // 1010 => 10
int rightShift = a >> 1; // 0010 => 2

Console.WriteLine($"AND: {andResult}, OR: {orResult}, XOR: {xorResult}, NOT: {notResult}, Left Shift: {leftShift}, Right Shift: {rightShift}");

Conditional (Ternary) Operator

The conditional operator is a shorthand for an if-else statement.

  • ?: : Conditional

Example

int a = 10;
int b = 20;

string result = (a > b) ? "a is greater" : "b is greater";

Console.WriteLine(result); // Output: b is greater

Null-Coalescing Operators

Null-coalescing operators are used to provide a default value for nullable types.

  • ?? : Null-coalescing
  • ??= : Null-coalescing assignment

Example

int? a = null;
int b = a ?? 5; // If a is null, b will be 5

int? c = null;
c ??= 10; // If c is null, assign 10 to c

Console.WriteLine($"b: {b}, c: {c}"); // Output: b: 5, c: 10

Summary

Operators in C# are essential for performing various operations on data. From basic arithmetic to more complex bitwise operations, understanding how to use these operators effectively is crucial for any C# programmer. Practice using these operators in different scenarios to become proficient in manipulating data and writing efficient code.
