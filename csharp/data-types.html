Introduction to Data Types

Data types are fundamental to any programming language, and C# is no exception. They define the kind of data that can be stored and manipulated within a program. Understanding data types is crucial for writing efficient and error-free code.

Primitive Data Types

C# provides several built-in data types, also known as primitive data types. These are the most basic types and are used to store simple values.

Integer Types

  • int: Represents a 32-bit signed integer.
  • long: Represents a 64-bit signed integer.
  • short: Represents a 16-bit signed integer.
  • byte: Represents an 8-bit unsigned integer.
int age = 25;
long distance = 9876543210L;
short temperature = -5;
byte level = 255;

Floating-Point Types

  • float: Represents a single-precision floating-point number.
  • double: Represents a double-precision floating-point number.
  • decimal: Represents a 128-bit precise decimal value, suitable for financial calculations.
float pi = 3.14F;
double e = 2.7182818284;
decimal price = 19.99M;

Boolean Type

  • bool: Represents a boolean value that can be either true or false.
bool isActive = true;
bool isCompleted = false;

Character Type

  • char: Represents a single 16-bit Unicode character.
char grade = 'A';
char symbol = '#';

String Type

  • string: Represents a sequence of characters.
string greeting = "Hello, World!";
string name = "Alice";

Complex Data Types

C# also supports more complex data types that can store multiple values or more complex structures.

Arrays

Arrays are collections of elements of the same type.

int[] numbers = { 1, 2, 3, 4, 5 };
string[] names = { "Alice", "Bob", "Charlie" };

Lists

Lists are similar to arrays but are more flexible and part of the System.Collections.Generic namespace.

using System.Collections.Generic;

List<int> scores = new List<int> { 90, 85, 100 };
scores.Add(95);

Dictionaries

Dictionaries store key-value pairs and are part of the System.Collections.Generic namespace.

using System.Collections.Generic;

Dictionary<string, int> ages = new Dictionary<string, int>();
ages["Alice"] = 30;
ages["Bob"] = 25;

Nullable Types

Nullable types allow value types to be assigned null.

int? nullableInt = null;
nullableInt = 5;

Type Conversion

Type conversion is the process of converting one data type to another. C# supports both implicit and explicit conversions.

Implicit Conversion

Implicit conversions are automatically handled by the compiler.

int num = 123;
double doubleNum = num; // Implicit conversion from int to double

Explicit Conversion

Explicit conversions require a cast operator.

double doubleNum = 123.45;
int num = (int)doubleNum; // Explicit conversion from double to int

Summary

Understanding data types is essential for any C# programmer. This topic covered:

  • Primitive data types like int, float, bool, char, and string.
  • Complex data types like arrays, lists, and dictionaries.
  • Nullable types and type conversion.

By mastering these concepts, you'll be well-equipped to handle data effectively in your C# programs.
