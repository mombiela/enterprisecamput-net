Lambda Expressions in C#
Arrays ►

Lambda expressions are a powerful feature in C# that allow you to write inline expressions or functions. They are particularly useful for writing concise and readable code, especially when working with collections and LINQ (Language Integrated Query). This topic will guide you from the basics to advanced usage of lambda expressions in C#.

What are Lambda Expressions?

  • Definition: A lambda expression is an anonymous function that you can use to create delegates or expression tree types.
  • Syntax: The basic syntax of a lambda expression is (input parameters) => expression or statement block.

Basic Syntax

(parameters) => expression
(parameters) => { statements }

Example

Func<int, int> square = x => x * x;
Console.WriteLine(square(5)); // Output: 25

Why Use Lambda Expressions?

  • Conciseness: Lambda expressions allow you to write less code.
  • Readability: They can make your code more readable by removing boilerplate code.
  • Flexibility: They are versatile and can be used in various scenarios, such as LINQ queries, event handling, and more.

Basic Usage

Single Parameter

Func<int, int> increment = x => x + 1;
Console.WriteLine(increment(5)); // Output: 6

Multiple Parameters

Func<int, int, int> add = (x, y) => x + y;
Console.WriteLine(add(3, 4)); // Output: 7

No Parameters

Action greet = () => Console.WriteLine("Hello, World!");
greet(); // Output: Hello, World!

Advanced Usage

Using Lambda Expressions with LINQ

Lambda expressions are extensively used with LINQ to perform operations on collections.

List<int> numbers = new List<int> { 1, 2, 3, 4, 5 };
var evenNumbers = numbers.Where(n => n % 2 == 0).ToList();
evenNumbers.ForEach(n => Console.WriteLine(n)); // Output: 2 4

Expression Trees

Lambda expressions can be used to create expression trees, which are data structures representing code.

Expression<Func<int, bool>> isEven = x => x % 2 == 0;
Console.WriteLine(isEven); // Output: x => (x % 2 == 0)

Capturing Variables

Lambda expressions can capture variables from their enclosing scope.

int factor = 2;
Func<int, int> multiply = x => x * factor;
Console.WriteLine(multiply(3)); // Output: 6

Common Use Cases

Event Handling

Lambda expressions can be used to handle events in a concise manner.

button.Click += (sender, e) => { Console.WriteLine("Button clicked!"); };

Sorting Collections

You can use lambda expressions to define custom sorting logic.

List<string> names = new List<string> { "Alice", "Bob", "Charlie" };
names.Sort((a, b) => a.Length.CompareTo(b.Length));
names.ForEach(name => Console.WriteLine(name)); // Output: Bob Alice Charlie

Conclusion

Lambda expressions are a versatile and powerful feature in C#. They allow you to write concise and readable code, especially when working with collections and LINQ. By understanding and utilizing lambda expressions, you can significantly improve the efficiency and readability of your code.

Summary

  • Lambda expressions are anonymous functions that can be used to create delegates or expression tree types.
  • They provide a concise and readable way to write inline expressions or functions.
  • Lambda expressions are extensively used with LINQ, event handling, and custom sorting.
  • Understanding lambda expressions can greatly enhance your C# programming skills.

By mastering lambda expressions, you will be well-equipped to write more efficient and maintainable C# code.
