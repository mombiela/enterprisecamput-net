What is LINQ?

Language Integrated Query (LINQ) is a powerful feature in C# that allows developers to write queries directly within the C# language. LINQ provides a consistent way to query various data sources, such as collections, databases, XML, and more.

  • Consistent Syntax: LINQ uses a uniform syntax to query different data sources.
  • Type Safety: Queries are checked at compile-time, reducing runtime errors.
  • IntelliSense Support: Visual Studio provides IntelliSense support for LINQ queries, making it easier to write and debug.

Basic LINQ Syntax

LINQ queries are written using query syntax or method syntax. Both achieve the same result but have different syntactical approaches.

Query Syntax

Query syntax resembles SQL and is easier for those familiar with SQL to understand.

int[] numbers = { 2, 3, 4, 5, 6 };

var evenNumbers = from num in numbers
                  where num % 2 == 0
                  select num;

foreach (var num in evenNumbers)
{
    Console.WriteLine(num);
}

Method Syntax

Method syntax uses extension methods and lambda expressions.

int[] numbers = { 2, 3, 4, 5, 6 };

var evenNumbers = numbers.Where(num => num % 2 == 0);

foreach (var num in evenNumbers)
{
    Console.WriteLine(num);
}

LINQ Operators

LINQ provides a variety of operators to perform different operations on data. Here are some key operators:

  • Where: Filters elements based on a condition.
  • Select: Projects each element into a new form.
  • OrderBy: Sorts elements in ascending order.
  • GroupBy: Groups elements that share a common attribute.
  • Join: Joins two sequences based on a key.

Example: Using Multiple Operators

string[] names = { "John", "Jane", "Jake", "Jessie", "James" };

var filteredNames = names
    .Where(name => name.StartsWith("J"))
    .OrderBy(name => name)
    .Select(name => name.ToUpper());

foreach (var name in filteredNames)
{
    Console.WriteLine(name);
}

LINQ with Different Data Sources

LINQ to Objects

LINQ can be used to query in-memory collections like arrays, lists, etc.

List<int> numbers = new List<int> { 1, 2, 3, 4, 5 };

var oddNumbers = from num in numbers
                 where num % 2 != 0
                 select num;

foreach (var num in oddNumbers)
{
    Console.WriteLine(num);
}

LINQ to SQL

LINQ can be used to query databases. This requires setting up a data context.

using (var context = new DataContext())
{
    var customers = from customer in context.Customers
                    where customer.City == "Seattle"
                    select customer;

    foreach (var customer in customers)
    {
        Console.WriteLine(customer.Name);
    }
}

LINQ to XML

LINQ can be used to query XML documents.

XDocument doc = XDocument.Load("books.xml");

var books = from book in doc.Descendants("book")
            where (int)book.Element("price") > 20
            select book.Element("title").Value;

foreach (var title in books)
{
    Console.WriteLine(title);
}

Exercises

  1. Basic Filtering: Write a LINQ query to filter out even numbers from an array of integers.
  2. Projection: Use LINQ to select the names of students who scored above 80 from a list of student objects.
  3. Sorting: Write a LINQ query to sort a list of products by their price in descending order.
  4. Grouping: Group a list of employees by their department and display the department names along with the count of employees in each department.
  5. Joining: Use LINQ to join two lists: one containing customer IDs and another containing orders, and display the customer IDs along with their order details.

Summary

LINQ is a versatile and powerful feature in C# that simplifies data querying across different data sources. By providing a consistent syntax and leveraging the power of the C# language, LINQ enhances productivity and code readability. Whether you are working with in-memory collections, databases, or XML documents, LINQ offers a unified approach to data manipulation.
