Introduction

In this section, students will apply the knowledge acquired about data structures in practical projects. These projects are designed to consolidate learning and provide practical experience in implementing and manipulating various data structures.

Objectives of the Final Projects

  • Apply theoretical concepts in practical scenarios.
  • Develop problem-solving skills using data structures.
  • Improve the ability to write efficient and well-structured code.
  • Encourage teamwork and collaboration (if the projects are group-based).

Project Descriptions

Project 1: Library Management System

Description: Develop a library management system that allows basic operations such as adding, deleting, searching, and listing books. Use an appropriate data structure for each operation.

Requirements:

  • Implement a linked list to store the books.
  • Each book should have attributes such as title, author, ISBN, and year of publication.
  • Functionalities:
    • Add a new book.
    • Delete a book by ISBN.
    • Search for a book by title or author.
    • List all books in the library.

Code Example:

class Book:
    def __init__(self, title, author, isbn, year):
        self.title = title
        self.author = author
        self.isbn = isbn
        self.year = year
        self.next = None

class Library:
    def __init__(self):
        self.head = None

    def add_book(self, book):
        if not self.head:
            self.head = book
        else:
            current = self.head
            while current.next:
                current = current.next
            current.next = book

    def delete_book(self, isbn):
        current = self.head
        previous = None
        while current and current.isbn != isbn:
            previous = current
            current = current.next
        if previous is None:
            self.head = current.next
        elif current:
            previous.next = current.next

    def search_book(self, title):
        current = self.head
        while current:
            if current.title == title:
                return current
            current = current.next
        return None

    def list_books(self):
        current = self.head
        while current:
            print(f'Title: {current.title}, Author: {current.author}, ISBN: {current.isbn}, Year: {current.year}')
            current = current.next

Project 2: Task Management System

Description: Create a task management system that allows users to add, delete, search, and list tasks. Use an appropriate data structure to handle the tasks.

Requirements:

  • Implement a stack to manage the tasks.
  • Each task should have attributes such as description, due date, and priority.
  • Functionalities:
    • Add a new task.
    • Delete the most recent task.
    • Search for a task by description.
    • List all pending tasks.

Code Example:

class Task:
    def __init__(self, description, due_date, priority):
        self.description = description
        self.due_date = due_date
        self.priority = priority

class TaskManagement:
    def __init__(self):
        self.stack = []

    def add_task(self, task):
        self.stack.append(task)

    def delete_task(self):
        if self.stack:
            return self.stack.pop()
        return None

    def search_task(self, description):
        for task in self.stack:
            if task.description == description:
                return task
        return None

    def list_tasks(self):
        for task in reversed(self.stack):
            print(f'Description: {task.description}, Due Date: {task.due_date}, Priority: {task.priority}')

Project 3: Contact Management System

Description: Develop a contact management system that allows adding, deleting, searching, and listing contacts. Use an appropriate data structure for each operation.

Requirements:

  • Implement a binary search tree (BST) to store the contacts.
  • Each contact should have attributes such as name, phone number, and email.
  • Functionalities:
    • Add a new contact.
    • Delete a contact by name.
    • Search for a contact by name.
    • List all contacts in alphabetical order.

Code Example:

class Contact:
    def __init__(self, name, phone, email):
        self.name = name
        self.phone = phone
        self.email = email
        self.left = None
        self.right = None

class ContactManagement:
    def __init__(self):
        self.root = None

    def add_contact(self, contact):
        if not self.root:
            self.root = contact
        else:
            self._add(self.root, contact)

    def _add(self, node, contact):
        if contact.name < node.name:
            if node.left is None:
                node.left = contact
            else:
                self._add(node.left, contact)
        else:
            if node.right is None:
                node.right = contact
            else:
                self._add(node.right, contact)

    def delete_contact(self, name):
        self.root = self._delete(self.root, name)

    def _delete(self, node, name):
        if not node:
            return node
        if name < node.name:
            node.left = self._delete(node.left, name)
        elif name > node.name:
            node.right = self._delete(node.right, name)
        else:
            if not node.left:
                return node.right
            elif not node.right:
                return node.left
            temp = self._min_value_node(node.right)
            node.name = temp.name
            node.phone = temp.phone
            node.email = temp.email
            node.right = self._delete(node.right, temp.name)
        return node

    def _min_value_node(self, node):
        current = node
        while current.left:
            current = current.left
        return current

    def search_contact(self, name):
        return self._search(self.root, name)

    def _search(self, node, name):
        if not node or node.name == name:
            return node
        if name < node.name:
            return self._search(node.left, name)
        return self._search(node.right, name)

    def list_contacts(self):
        self._inorder(self.root)

    def _inorder(self, node):
        if node:
            self._inorder(node.left)
            print(f'Name: {node.name}, Phone: {node.phone}, Email: {node.email}')
            self._inorder(node.right)
