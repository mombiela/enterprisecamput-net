Introduction

Data structures are fundamental in computer science and are used to organize and store data efficiently. In this topic, we will explore how data structures are applied in real-world situations, providing clear examples and practical exercises to help IT professionals understand their importance and application.

Key Concepts

  • Data Structures: Ways to organize and store data to facilitate access and modification.
  • Common Types: Arrays, linked lists, stacks, queues, trees, graphs, hash tables.
  • Practical Applications: Use of data structures in real-world applications such as databases, operating systems, networks, and more.

Real-Life Application Examples

Arrays

Arrays are a simple yet powerful data structure. They are used to store elements of the same type in a contiguous sequence of memory.

Example: Flight Reservation System

In a flight reservation system, arrays can be used to store the list of available seats on a plane.

# Array representing seats on a plane
seats = [True, True, False, True, False]  # True indicates the seat is available

# Reserve a seat
def reserve_seat(seats, seat_number):
    if seats[seat_number]:
        seats[seat_number] = False
        return "Seat successfully reserved"
    else:
        return "Seat is already occupied"

print(reserve_seat(seats, 2))  # Output: Seat is already occupied
print(reserve_seat(seats, 1))  # Output: Seat successfully reserved

Linked Lists

Linked lists are a data structure where each element (node) contains a value and a reference to the next node in the sequence.

Example: Task Management System

In a task management system, linked lists can be used to maintain a list of pending tasks.

class Node:
    def __init__(self, data):
        self.data = data
        self.next = None

class LinkedList:
    def __init__(self):
        self.head = None

    def add_task(self, task):
        new_node = Node(task)
        new_node.next = self.head
        self.head = new_node

    def show_tasks(self):
        current = self.head
        while current:
            print(current.data)
            current = current.next

# Create a task list
task_list = LinkedList()
task_list.add_task("Buy milk")
task_list.add_task("Send email")
task_list.add_task("Call the client")

task_list.show_tasks()
# Output:
# Call the client
# Send email
# Buy milk

Stacks

Stacks are data structures that follow the LIFO (Last In, First Out) principle. They are commonly used in situations where the last element added is the first to be removed.

Example: Web Browser

In a web browser, stacks are used to implement the "back" and "forward" functionality.

class Stack:
    def __init__(self):
        self.items = []

    def is_empty(self):
        return self.items == []

    def push(self, item):
        self.items.append(item)

    def pop(self):
        return self.items.pop()

# Create a browsing history stack
history = Stack()
history.push("page1.com")
history.push("page2.com")
history.push("page3.com")

print(history.pop())  # Output: page3.com
print(history.pop())  # Output: page2.com

Queues

Queues are data structures that follow the FIFO (First In, First Out) principle. They are used in situations where the first element added is the first to be removed.

Example: Printing System

In a printing system, queues are used to manage print jobs in the order they were received.

class Queue:
    def __init__(self):
        self.items = []

    def is_empty(self):
        return self.items == []

    def enqueue(self, item):
        self.items.insert(0, item)

    def dequeue(self):
        return self.items.pop()

# Create a print job queue
print_queue = Queue()
print_queue.enqueue("Document1")
print_queue.enqueue("Document2")
print_queue.enqueue("Document3")

print(print_queue.dequeue())  # Output: Document1
print(print_queue.dequeue())  # Output: Document2
