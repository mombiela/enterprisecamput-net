Introduction

In this section of the Data Structures course, we will focus on graphs, a fundamental structure in computer science and computing. Graphs are used to model relationships and connections between different entities. Through practical exercises, we will learn to implement and manipulate graphs in different contexts.

Key Concepts

  • Graph: Set of nodes (or vertices) and edges (or links) that connect pairs of nodes.
  • Types of Graphs:
    • Directed: Edges have a direction.
    • Undirected: Edges do not have a direction.
    • Weighted: Edges have an associated weight.
    • Unweighted: Edges do not have weight.
  • Graph Representations:
    • Adjacency List: Uses lists to store adjacent nodes.
    • Adjacency Matrix: Uses a matrix to represent connections between nodes.

Exercise 1: Implementation of an Undirected Graph

Objective

Implement an undirected graph using an adjacency list.

Instructions

  1. Create a Graph class.
  2. Implement methods to add nodes and edges.
  3. Implement a method to print the graph.

Code Example

class Graph:
    def __init__(self):
        self.graph = {}

    def add_node(self, node):
        if node not in self.graph:
            self.graph[node] = []

    def add_edge(self, node1, node2):
        if node1 in self.graph and node2 in self.graph:
            self.graph[node1].append(node2)
            self.graph[node2].append(node1)

    def print_graph(self):
        for node in self.graph:
            print(f"{node}: {self.graph[node]}")

# Example of use
g = Graph()
g.add_node("A")
g.add_node("B")
g.add_node("C")
g.add_edge("A", "B")
g.add_edge("A", "C")
g.print_graph()

Explanation

  • Graph Class: Defines the structure of the graph.
  • add_node Method: Adds a node to the graph.
  • add_edge Method: Adds an edge between two nodes.
  • print_graph Method: Prints the adjacency list of the graph.

Exercise 2: Implementation of a Weighted Directed Graph

Objective

Implement a weighted directed graph using an adjacency list.

Instructions

  1. Create a WeightedDirectedGraph class.
  2. Implement methods to add nodes and edges with weights.
  3. Implement a method to print the graph.

Code Example

class WeightedDirectedGraph:
    def __init__(self):
        self.graph = {}

    def add_node(self, node):
        if node not in self.graph:
            self.graph[node] = []

    def add_edge(self, node1, node2, weight):
        if node1 in self.graph and node2 in self.graph:
            self.graph[node1].append((node2, weight))

    def print_graph(self):
        for node in self.graph:
            print(f"{node}: {self.graph[node]}")

# Example of use
g = WeightedDirectedGraph()
g.add_node("A")
g.add_node("B")
g.add_node("C")
g.add_edge("A", "B", 5)
g.add_edge("A", "C", 10)
g.print_graph()

Explanation

  • WeightedDirectedGraph Class: Defines the structure of the weighted directed graph.
  • add_node Method: Adds a node to the graph.
  • add_edge Method: Adds a weighted edge between two nodes.
  • print_graph Method: Prints the adjacency list of the graph with weights.

Exercise 3: Graph Search

Objective

Implement graph search algorithms: BFS (Breadth-First Search) and DFS (Depth-First Search).

Instructions

  1. Create methods for BFS and DFS in the Graph class.
  2. Implement the logic of the search algorithms.

Code Example

class Graph:
    def __init__(self):
        self.graph = {}

    def add_node(self, node):
        if node not in self.graph:
            self.graph[node] = []

    def add_edge(self, node1, node2):
        if node1 in self.graph and node2 in self.graph:
            self.graph[node1].append(node2)
            self.graph[node2].append(node1)

    def bfs(self, start):
        visited = set()
        queue = [start]

        while queue:
            node = queue.pop(0)
            if node not in visited:
                print(node, end=" ")
                visited.add(node)
                queue.extend(self.graph[node])

    def dfs(self, start, visited=None):
        if visited is None:
            visited = set()
        visited.add(start)
        print(start, end=" ")

        for neighbor in self.graph[start]:
            if neighbor not in visited:
                self.dfs(neighbor, visited)

# Example of use
g = Graph()
g.add_node("A")
g.add_node("B")
g.add_node("C")
g.add_node("D")
g.add_edge("A", "B")
g.add_edge("A", "C")
g.add_edge("B", "D")
print("BFS:")
g.bfs("A")
print("\nDFS:")
g.dfs("A")

Explanation

  • bfs Method: Implements breadth-first search.
  • dfs Method: Implements depth-first search.
