Introduction

In this section of the course on Data Structures, we will focus on lists, one of the most fundamental and versatile data structures in programming. Lists allow you to store collections of elements, which can be of any data type, and offer a variety of useful operations to manipulate these elements.

Key Concepts

  • List Definition: A list is an ordered collection of elements that can be accessed by their index.
  • Basic Operations: Insertion, deletion, access, and search of elements.
  • Types of Lists: Linked lists, double lists, circular lists.

Practical Exercises

Exercise 1: Creation and Access to Elements

Objective: Create a list and access its elements.

Instructions:

  1. Create a list with the numbers from 1 to 5.
  2. Access the first and last element of the list.
  3. Print all the elements of the list.

Example Code:

# Creation of the list
my_list = [1, 2, 3, 4, 5]

# Access to the first and last element
first_element = my_list[0]
last_element = my_list[-1]

# Print the elements
print("First element:", first_element)
print("Last element:", last_element)
print("All elements:", my_list)

Exercise 2: Modification of Elements

Objective: Modify elements within a list.

Instructions:

  1. Change the second element of the list to 10.
  2. Add the number 6 to the end of the list.
  3. Remove the third element of the list.

Example Code:

# Initial list
my_list = [1, 2, 3, 4, 5]

# Modification of the second element
my_list[1] = 10

# Add a new element to the end
my_list.append(6)

# Remove the third element
del my_list[2]

# Print the modified list
print("Modified list:", my_list)

Exercise 3: Iteration and Sum of Elements

Objective: Iterate over a list and calculate the sum of its elements.

Instructions:

  1. Iterate over each element of the list.
  2. Calculate the sum of all elements.

Example Code:

# Initial list
my_list = [1, 2, 3, 4, 5]

# Initialize sum
sum = 0

# Iterate and sum elements
for element in my_list:
    sum += element

# Print the sum
print("Sum of elements:", sum)

Exercise 4: Nested Lists

Objective: Work with lists within lists (nested lists).

Instructions:

  1. Create a list that contains three lists, each with three numbers.
  2. Access the second element of the third list.
  3. Print all the elements of the nested list.

Example Code:

# Creation of the nested list
nested_list = [
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9]
]

# Access to the second element of the third list
element = nested_list[2][1]

# Print the element
print("Second element of the third list:", element)

# Print all elements of the nested list
for sublist in nested_list:
    for item in sublist:
        print(item, end=' ')
    print()

Conclusion

Lists are an essential data structure in programming, offering flexibility and efficiency for storing and manipulating collections of elements. Through these practical exercises, we have explored how to create, modify, iterate, and work with nested lists. Understanding and mastering the use of lists is fundamental for any IT professional, as they are widely used in various applications and algorithms.
