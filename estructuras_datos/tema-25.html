Introduction

In this section of the Data Structures course, we will learn about basic operations with trees. Trees are hierarchical data structures consisting of nodes connected by edges. Each node contains a value and can have children, forming a tree-like structure.

Key Concepts

  • Node: Basic element of a tree that contains a value or data.
  • Root: The top node of a tree, which has no parents.
  • Child: Node that descends from another node.
  • Parent: Node that has one or more children.
  • Leaf: Node that has no children.
  • Subtree: Tree formed by a node and all its descendants.
  • Height: Length of the longest path from the root to a leaf.
  • Depth: Length of the path from the root to a specific node.

Basic Operations

1. Insertion

Insertion is the process of adding a new node to the tree. The location of the new node depends on the type of tree (e.g., binary search tree).

Code Example: Insertion in a Binary Search Tree (BST)

class Node:
    def __init__(self, key):
        self.left = None
        self.right = None
        self.val = key

def insert(root, key):
    if root is None:
        return Node(key)
    else:
        if root.val < key:
            root.right = insert(root.right, key)
        else:
            root.left = insert(root.left, key)
    return root

# Usage example
root = Node(50)
root = insert(root, 30)
root = insert(root, 20)
root = insert(root, 40)
root = insert(root, 70)
root = insert(root, 60)
root = insert(root, 80)

2. Search

Search is the process of finding a node with a specific value in the tree.

Code Example: Search in a Binary Search Tree (BST)

def search(root, key):
    if root is None or root.val == key:
        return root
    if root.val < key:
        return search(root.right, key)
    return search(root.left, key)

# Usage example
result = search(root, 40)
if result:
    print("Node found:", result.val)
else:
    print("Node not found")

3. Deletion

Deletion is the process of removing a node from the tree. This process can be more complex as several cases must be considered (node without children, with one child, or with two children).

Code Example: Deletion in a Binary Search Tree (BST)

def minValueNode(node):
    current = node
    while current.left is not None:
        current = current.left
    return current

def deleteNode(root, key):
    if root is None:
        return root
    if key < root.val:
        root.left = deleteNode(root.left, key)
    elif key > root.val:
        root.right = deleteNode(root.right, key)
    else:
        if root.left is None:
            return root.right
        elif root.right is None:
            return root.left
        temp = minValueNode(root.right)
        root.val = temp.val
        root.right = deleteNode(root.right, temp.val)
    return root

# Usage example
root = deleteNode(root, 20)

4. Traversals

Traversals are methods to visit all the nodes of a tree in a specific order. The most common types are:

  • In-order: Visits the left subtree, the root node, and then the right subtree.
  • Pre-order: Visits the root node, the left subtree, and then the right subtree.
  • Post-order: Visits the left subtree, the right subtree, and then the root node.

Code Example: Traversals in a Binary Tree

def inorder(root):
    if root:
        inorder(root.left)
        print(root.val, end=' ')
        inorder(root.right)

def preorder(root):
    if root:
        print(root.val, end=' ')
        preorder(root.left)
        preorder(root.right)

def postorder(root):
    if root:
        postorder(root.left)
        postorder(root.right)
        print(root.val, end=' ')

# Usage example
print("In-order:")
inorder(root)
print("\nPre-order:")
preorder(root)
print("\nPost-order:")
postorder(root)

Conclusion

In this section, we have covered the basic operations with trees, including insertion, search, deletion, and traversals. These operations are fundamental for working with trees and are applicable in various data structures and algorithms. Be sure to practice these concepts and code examples to consolidate your understanding.
