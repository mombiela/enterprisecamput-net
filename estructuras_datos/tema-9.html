Introduction

A stack is a linear data structure that follows the LIFO (Last In, First Out) principle, which means that the last element to enter is the first to leave. This concept is fundamental in computer science and is used in various applications such as memory management, expression evaluation, and algorithm implementation.

Characteristics of a Stack

  • LIFO (Last In, First Out): The last inserted element is the first to be removed.
  • Basic operations:
    • Push: Add an element to the stack.
    • Pop: Remove the top element from the stack.
    • Peek/Top: Get the top element without removing it.
    • isEmpty: Check if the stack is empty.
    • Size: Get the number of elements in the stack.

Stack Implementation

Stacks can be implemented in various ways, the most common being:

  • Using Arrays: A fixed-size stack.
  • Using Linked Lists: A dynamic-size stack.

Implementation using Arrays

class StackArray:
    def __init__(self, capacity):
        self.capacity = capacity
        self.stack = [None] * capacity
        self.top = -1

    def push(self, item):
        if self.top == self.capacity - 1:
            raise Exception("Stack Overflow")
        self.top += 1
        self.stack[self.top] = item

    def pop(self):
        if self.top == -1:
            raise Exception("Stack Underflow")
        item = self.stack[self.top]
        self.stack[self.top] = None
        self.top -= 1
        return item

    def peek(self):
        if self.top == -1:
            raise Exception("Stack is empty")
        return self.stack[self.top]

    def isEmpty(self):
        return self.top == -1

    def size(self):
        return self.top + 1

Implementation using Linked Lists

class Node:
    def __init__(self, value):
        self.value = value
        self.next = None

class StackLinkedList:
    def __init__(self):
        self.head = None
        self.count = 0

    def push(self, item):
        new_node = Node(item)
        new_node.next = self.head
        self.head = new_node
        self.count += 1

    def pop(self):
        if self.head is None:
            raise Exception("Stack Underflow")
        item = self.head.value
        self.head = self.head.next
        self.count -= 1
        return item

    def peek(self):
        if self.head is None:
            raise Exception("Stack is empty")
        return self.head.value

    def isEmpty(self):
        return self.head is None

    def size(self):
        return self.count

Comparison between Implementations

| Characteristic | Array | Linked List | |----------------------|--------------------------------|--------------------------------| | Size | Fixed | Dynamic | | Space | Pre-allocated | Allocated as needed | | Push Complexity | O(1) | O(1) | | Pop Complexity | O(1) | O(1) | | Memory Usage | More efficient if size is known | Less efficient due to pointers |

Practical Example

Below is a practical example of how to use a stack to check if a sequence of parentheses is balanced.

def is_balanced(expression):
    stack = StackArray(len(expression))
    for char in expression:
        if char in "({[":
            stack.push(char)
        elif char in ")}]":
            if stack.isEmpty():
                return False
            top_char = stack.pop()
            if not matches(top_char, char):
                return False
    return stack.isEmpty()

def matches(opening, closing):
    opens = "({["
    closes = ")}]"
    return opens.index(opening) == closes.index(closing)

# Example usage
expression = "{[()]}"
print(is_balanced(expression))  # Output: True

Conclusion

Stacks are a fundamental data structure in computer science, with applications in various areas such as memory management and expression evaluation. Understanding how to implement and use stacks is essential for any IT professional. Implementations may vary according to specific needs, whether using arrays for a fixed size or linked lists for a dynamic size.
