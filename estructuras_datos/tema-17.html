Introduction to Queues

Queues are linear data structures that follow the FIFO (First In, First Out) principle, meaning the first element to enter is the first to leave. They are widely used in scenarios where sequential processing of elements is required, such as task management, printing systems, and handling requests on servers.

Key Concepts

  • Enqueue: Operation to add an element to the end of the queue.
  • Dequeue: Operation to remove the element from the front of the queue.
  • Front: Operation to get the element at the front of the queue without removing it.
  • Rear: Operation to get the element at the end of the queue without removing it.
  • IsEmpty: Operation to check if the queue is empty.
  • IsFull: Operation to check if the queue is full (in the case of queues with limited capacity).

Queue Implementation

Below is a basic implementation of a queue using a list in Python.

class Queue:
    def __init__(self):
        self.items = []

    def is_empty(self):
        return len(self.items) == 0

    def enqueue(self, item):
        self.items.append(item)
        print(f"Enqueued: {item}")

    def dequeue(self):
        if self.is_empty():
            return "Queue is empty"
        return self.items.pop(0)

    def front(self):
        if self.is_empty():
            return "Queue is empty"
        return self.items[0]

    def rear(self):
        if self.is_empty():
            return "Queue is empty"
        return self.items[-1]

    def size(self):
        return len(self.items)

# Usage example
queue = Queue()
queue.enqueue(1)
queue.enqueue(2)
queue.enqueue(3)
print("Front element:", queue.front())
print("Rear element:", queue.rear())
print("Dequeued:", queue.dequeue())
print("Queue size:", queue.size())

Practical Exercises

  1. Implement a Circular Queue: Modify the above implementation to work as a circular queue with limited capacity.
  2. Print System Simulation: Use the queue to simulate a print system where tasks are processed in the order they arrive.
  3. Palindrome Verification: Use a queue to verify if a word is a palindrome (reads the same left to right and right to left).

Comparison of Queues and Stacks

| Feature | Queue | Stack | |---------|-------|-------| | Processing Order | FIFO (First In, First Out) | LIFO (Last In, First Out) | | Main Operations | Enqueue, Dequeue | Push, Pop | | Usage Examples | Task management, printing systems | Undo/redo in editors, expression evaluation |

Conclusion

Queues are a fundamental data structure in computer science, especially useful in situations where processing order is crucial. Understanding the basic operations and their implementation is essential for any IT professional. Practicing with examples and practical exercises will help consolidate these concepts and prepare for more advanced applications.
