Introduction

BigQuery is a fully-managed, serverless data warehouse that enables scalable analysis over petabytes of data. It uses SQL (Structured Query Language) for querying and managing data. This section will cover the basic SQL syntax required to get started with BigQuery.

Key Concepts

  • SQL (Structured Query Language): A standard language for accessing and manipulating databases.
  • BigQuery: A Google Cloud Platform service for data warehousing and analytics.
  • Dataset: A collection of tables.
  • Table: A collection of rows and columns.
  • Query: A request for data or information from a database.

Basic SQL Statements

SELECT Statement

The SELECT statement is used to query data from a database.

SELECT column1, column2, ...
FROM table_name;

Example:

SELECT name, age
FROM employees;

Explanation: This query selects the name and age columns from the employees table.

WHERE Clause

The WHERE clause is used to filter records.

SELECT column1, column2, ...
FROM table_name
WHERE condition;

Example:

SELECT name, age
FROM employees
WHERE age > 30;

Explanation: This query selects the name and age columns from the employees table where the age is greater than 30.

ORDER BY Clause

The ORDER BY clause is used to sort the result set.

SELECT column1, column2, ...
FROM table_name
ORDER BY column1 [ASC|DESC];

Example:

SELECT name, age
FROM employees
ORDER BY age DESC;

Explanation: This query selects the name and age columns from the employees table and sorts the results by age in descending order.

LIMIT Clause

The LIMIT clause is used to specify the number of records to return.

SELECT column1, column2, ...
FROM table_name
LIMIT number;

Example:

SELECT name, age
FROM employees
LIMIT 5;

Explanation: This query selects the name and age columns from the employees table and returns only the first 5 records.

Aggregation Functions

Aggregation functions perform a calculation on a set of values and return a single value.

  • COUNT(): Returns the number of rows.
  • SUM(): Returns the total sum of a numeric column.
  • AVG(): Returns the average value of a numeric column.
  • MAX(): Returns the maximum value.
  • MIN(): Returns the minimum value.

Example:

SELECT COUNT(*), AVG(age), MAX(age), MIN(age)
FROM employees;

Explanation: This query returns the total number of employees, the average age, the maximum age, and the minimum age from the employees table.

GROUP BY Clause

The GROUP BY clause groups rows that have the same values into summary rows.

SELECT column1, COUNT(*)
FROM table_name
GROUP BY column1;

Example:

SELECT department, COUNT(*)
FROM employees
GROUP BY department;

Explanation: This query groups the employees by department and returns the number of employees in each department.

JOIN Clause

The JOIN clause is used to combine rows from two or more tables.

  • INNER JOIN: Returns records that have matching values in both tables.
  • LEFT JOIN: Returns all records from the left table, and the matched records from the right table.
  • RIGHT JOIN: Returns all records from the right table, and the matched records from the left table.
  • FULL OUTER JOIN: Returns all records when there is a match in either left or right table.

Example:

SELECT employees.name, departments.department_name
FROM employees
INNER JOIN departments ON employees.department_id = departments.id;

Explanation: This query selects the name from the employees table and the department_name from the departments table where the department_id in employees matches the id in departments.

Conclusion

Understanding basic SQL syntax is essential for querying and managing data in BigQuery. This section covered fundamental SQL statements such as SELECT, WHERE, ORDER BY, LIMIT, aggregation functions, GROUP BY, and JOIN. Mastering these basics will provide a strong foundation for more advanced SQL queries and data analysis in BigQuery.
