Introduction

In this section, we will explore how to filter and sort data in BigQuery. These operations are fundamental for data analysis, allowing you to retrieve specific subsets of data and organize them in a meaningful order. We will cover basic to advanced techniques, ensuring you have a comprehensive understanding of these essential skills.

Filtering Data

Basic Filtering with WHERE Clause

The WHERE clause is used to filter records that meet certain conditions.

Example:

SELECT *
FROM `project.dataset.table`
WHERE age > 30;
  • This query selects all columns from the table where the age column has values greater than 30.

Combining Conditions with AND/OR

You can combine multiple conditions using AND and OR.

Example:

SELECT *
FROM `project.dataset.table`
WHERE age > 30 AND gender = 'M';
  • This query selects records where age is greater than 30 and gender is 'M'.

Filtering with IN and BETWEEN

The IN operator allows you to specify multiple values in a WHERE clause, and BETWEEN selects values within a given range.

Example:

SELECT *
FROM `project.dataset.table`
WHERE country IN ('USA', 'Canada', 'UK');
  • This query selects records where the country is either 'USA', 'Canada', or 'UK'.
SELECT *
FROM `project.dataset.table`
WHERE age BETWEEN 20 AND 30;
  • This query selects records where age is between 20 and 30, inclusive.

Using NULL in Filters

Handling NULL values is crucial in filtering data.

Example:

SELECT *
FROM `project.dataset.table`
WHERE email IS NOT NULL;
  • This query selects records where the email column is not NULL.

Sorting Data

Basic Sorting with ORDER BY

The ORDER BY clause is used to sort the result set by one or more columns.

Example:

SELECT *
FROM `project.dataset.table`
ORDER BY age;
  • This query sorts the records by the age column in ascending order.

Sorting in Descending Order

To sort in descending order, use the DESC keyword.

Example:

SELECT *
FROM `project.dataset.table`
ORDER BY age DESC;
  • This query sorts the records by the age column in descending order.

Sorting by Multiple Columns

You can sort by multiple columns by specifying them in the ORDER BY clause.

Example:

SELECT *
FROM `project.dataset.table`
ORDER BY age DESC, name ASC;
  • This query sorts the records first by age in descending order and then by name in ascending order.

Advanced Filtering and Sorting

Filtering with Subqueries

Subqueries can be used within the WHERE clause for more complex filtering.

Example:

SELECT *
FROM `project.dataset.table`
WHERE age > (SELECT AVG(age) FROM `project.dataset.table`);
  • This query selects records where age is greater than the average age of all records in the table.

Filtering with Regular Expressions

BigQuery supports filtering with regular expressions using the REGEXP_CONTAINS function.

Example:

SELECT *
FROM `project.dataset.table`
WHERE REGEXP_CONTAINS(email, r'@gmail\.com$');
  • This query selects records where the email column ends with '@gmail.com'.

Sorting with Complex Expressions

You can sort data using complex expressions in the ORDER BY clause.

Example:

SELECT name, age, salary
FROM `project.dataset.table`
ORDER BY (salary / age) DESC;
  • This query sorts records by the result of the expression (salary / age) in descending order.

Conclusion

Filtering and sorting are essential operations in data analysis with BigQuery. By mastering these techniques, you can efficiently retrieve and organize data to gain valuable insights. Practice these concepts with various datasets to become proficient in writing effective BigQuery queries.
