Introduction

In this course, we will build a social network using Django, a high-level Python web framework. This project will take you from beginner to advanced level, covering essential concepts and features of Django.

Setting Up the Environment

  • Install Python and Django
  • Setting up a virtual environment
  • Creating a new Django project

Code Example: Setting Up a Django Project

# Install Django
pip install django

# Create a new Django project
django-admin startproject social_network

# Navigate into the project directory
cd social_network

# Start the development server
python manage.py runserver

Creating the User Model

  • Using Django's built-in User model
  • Extending the User model with a Profile model

Code Example: Extending the User Model

# models.py
from django.contrib.auth.models import User
from django.db import models

class Profile(models.Model):
    user = models.OneToOneField(User, on_delete=models.CASCADE)
    bio = models.TextField(max_length=500, blank=True)
    location = models.CharField(max_length=30, blank=True)
    birth_date = models.DateField(null=True, blank=True)

    def __str__(self):
        return self.user.username

User Registration and Authentication

  • Creating registration forms
  • Handling user login and logout
  • Password reset functionality

Code Example: User Registration Form

# forms.py
from django import forms
from django.contrib.auth.models import User
from django.contrib.auth.forms import UserCreationForm

class SignUpForm(UserCreationForm):
    email = forms.EmailField(max_length=200, help_text='Required')

    class Meta:
        model = User
        fields = ('username', 'email', 'password1', 'password2')

Building the Social Network Features

  • Creating and managing posts
  • Implementing user profiles
  • Adding a follow system

Code Example: Creating a Post Model

# models.py
from django.db import models
from django.contrib.auth.models import User

class Post(models.Model):
    author = models.ForeignKey(User, on_delete=models.CASCADE)
    content = models.TextField()
    created_at = models.DateTimeField(auto_now_add=True)

    def __str__(self):
        return self.content[:20]

Implementing the Follow System

  • Creating a Follow model
  • Managing follow relationships
  • Displaying followers and following lists

Code Example: Follow Model

# models.py
class Follow(models.Model):
    follower = models.ForeignKey(User, related_name='following', on_delete=models.CASCADE)
    following = models.ForeignKey(User, related_name='followers', on_delete=models.CASCADE)
    created_at = models.DateTimeField(auto_now_add=True)

    def __str__(self):
        return f"{self.follower} follows {self.following}"

Advanced Features

  • Adding comments and likes to posts
  • Implementing notifications
  • Real-time updates with WebSockets

Code Example: Adding Comments to Posts

# models.py
class Comment(models.Model):
    post = models.ForeignKey(Post, related_name='comments', on_delete=models.CASCADE)
    author = models.ForeignKey(User, on_delete=models.CASCADE)
    content = models.TextField()
    created_at = models.DateTimeField(auto_now_add=True)

    def __str__(self):
        return self.content[:20]

Testing and Deployment

  • Writing unit tests for your application
  • Setting up a production environment
  • Deploying your Django application

Code Example: Writing a Simple Test

# tests.py
from django.test import TestCase
from django.contrib.auth.models import User
from .models import Post

class PostModelTest(TestCase):
    def setUp(self):
        self.user = User.objects.create_user(username='testuser', password='12345')
        self.post = Post.objects.create(author=self.user, content='Just a test post')

    def test_post_content(self):
        self.assertEqual(self.post.content, 'Just a test post')
