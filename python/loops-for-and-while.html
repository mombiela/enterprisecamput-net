Loops: for and while
Control Flow Tools ►

Introduction

Loops are fundamental constructs in programming that allow you to execute a block of code multiple times. In Python, the two primary types of loops are for loops and while loops. This section will cover both types of loops, their syntax, use cases, and provide examples and exercises to help you understand how to use them effectively.

For Loops

A for loop is used for iterating over a sequence (such as a list, tuple, dictionary, set, or string). This is less like the for keyword in other programming languages and works more like an iterator method as found in other object-oriented programming languages.

Syntax

for variable in sequence:
    # Code block to execute

Example

fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    print(fruit)

Explanation: This loop iterates over each item in the fruits list and prints it.

Key Concepts

  • Iteration: The process of looping through the items in a sequence.
  • Sequence: A collection of items that can be iterated over (e.g., lists, strings, tuples).
  • Loop Variable: The variable that takes the value of the item in the sequence on each iteration.

Nested For Loops

You can also nest for loops to iterate over multiple sequences.

Example

adj = ["red", "big", "tasty"]
fruits = ["apple", "banana", "cherry"]
for a in adj:
    for f in fruits:
        print(a, f)

Explanation: This nested loop prints each combination of adjectives and fruits.

While Loops

A while loop repeatedly executes a block of code as long as a given condition is true.

Syntax

while condition:
    # Code block to execute

Example

i = 1
while i < 6:
    print(i)
    i += 1

Explanation: This loop prints numbers from 1 to 5. The loop continues to execute as long as i is less than 6.

Key Concepts

  • Condition: A boolean expression that determines whether the loop will continue to execute.
  • Infinite Loop: A loop that never ends because the condition is always true. Be cautious to avoid these.

Break and Continue Statements

  • Break: Exits the loop prematurely.
  • Continue: Skips the rest of the code inside the loop for the current iteration and moves to the next iteration.

Example with Break

i = 1
while i < 10:
    print(i)
    if i == 5:
        break
    i += 1

Explanation: This loop prints numbers from 1 to 5 and then exits when i equals 5.

Example with Continue

i = 0
while i < 6:
    i += 1
    if i == 3:
        continue
    print(i)

Explanation: This loop prints numbers from 1 to 6, except for 3.

Comparing For and While Loops

| Feature | For Loop | While Loop | |------------------|-------------------------------|----------------------------------| | Use Case | Iterating over a sequence | Repeating until a condition changes | | Syntax Simplicity| More concise for sequences | Requires manual condition update | | Readability | High for fixed iterations | High for condition-based loops |

Exercises

  1. For Loop Exercise: Write a for loop that prints the squares of numbers from 1 to 10.

    for i in range(1, 11):
    print(i**2)

  2. While Loop Exercise: Write a while loop that prints numbers from 10 down to 1.

    i = 10
while i > 0:
    print(i)
    i -= 1

  3. Nested Loop Exercise: Write a nested loop that prints a multiplication table from 1 to 5.

    for i in range(1, 6):
    for j in range(1, 6):
        print(i * j, end=' ')
    print()

Conclusion

Loops are essential for performing repetitive tasks efficiently. The for loop is ideal for iterating over sequences, while the while loop is suitable for condition-based repetition. Understanding how to use these loops, along with control statements like break and continue, will significantly enhance your ability to write efficient and effective Python code. Practice the exercises provided to solidify your understanding and become proficient in using loops in Python.
