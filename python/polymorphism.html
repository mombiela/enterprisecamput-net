Polymorphism in Python
Encapsulation ►

Polymorphism is a core concept in object-oriented programming (OOP) that allows objects of different classes to be treated as objects of a common superclass. It is a powerful feature that enables flexibility and integration in code.

What is Polymorphism?

  • Definition: Polymorphism means "many shapes" and it allows methods to do different things based on the object it is acting upon.
  • Purpose: It helps in designing systems that are more modular and easier to extend.

Types of Polymorphism

  1. Compile-time Polymorphism (Method Overloading):

    • Not directly supported in Python.
    • Achieved using default arguments or variable-length arguments.

  2. Run-time Polymorphism (Method Overriding):

    • Supported in Python.
    • Allows a subclass to provide a specific implementation of a method that is already defined in its superclass.

Method Overloading in Python

Python does not support traditional method overloading as seen in other languages like Java or C++. However, you can achieve similar functionality using default arguments or variable-length arguments.

Example of Method Overloading Using Default Arguments

class MathOperations:
    def add(self, a, b=0, c=0):
        return a + b + c

math_op = MathOperations()
print(math_op.add(1))       # Output: 1
print(math_op.add(1, 2))    # Output: 3
print(math_op.add(1, 2, 3)) # Output: 6

Example of Method Overloading Using Variable-length Arguments

class MathOperations:
    def add(self, *args):
        return sum(args)

math_op = MathOperations()
print(math_op.add(1))       # Output: 1
print(math_op.add(1, 2))    # Output: 3
print(math_op.add(1, 2, 3)) # Output: 6

Method Overriding in Python

Method overriding allows a subclass to provide a specific implementation of a method that is already defined in its superclass.

Example of Method Overriding

class Animal:
    def sound(self):
        return "Some generic sound"

class Dog(Animal):
    def sound(self):
        return "Bark"

class Cat(Animal):
    def sound(self):
        return "Meow"

animals = [Dog(), Cat(), Animal()]

for animal in animals:
    print(animal.sound())

Output:

Bark
Meow
Some generic sound

Polymorphism with Inheritance

Polymorphism is often used with inheritance to allow a single interface to be used for a general class of actions.

Example of Polymorphism with Inheritance

class Shape:
    def area(self):
        pass

class Rectangle(Shape):
    def __init__(self, width, height):
        self.width = width
        self.height = height

    def area(self):
        return self.width * self.height

class Circle(Shape):
    def __init__(self, radius):
        self.radius = radius

    def area(self):
        return 3.14 * self.radius * self.radius

shapes = [Rectangle(2, 3), Circle(5)]

for shape in shapes:
    print(shape.area())

Output:

6
78.5

Polymorphism with Functions and Objects

Polymorphism can also be achieved by using functions that can take objects of different classes and perform actions based on their class type.

Example of Polymorphism with Functions

class Bird:
    def fly(self):
        return "Flying high"

class Airplane:
    def fly(self):
        return "Flying at 30,000 feet"

def let_it_fly(flying_object):
    print(flying_object.fly())

bird = Bird()
airplane = Airplane()

let_it_fly(bird)       # Output: Flying high
let_it_fly(airplane)   # Output: Flying at 30,000 feet

Summary

Polymorphism is a fundamental concept in OOP that allows for flexibility and integration in code. It can be achieved in Python through method overloading (using default or variable-length arguments) and method overriding. By using polymorphism, you can design systems that are more modular and easier to extend, making your code more maintainable and scalable.
