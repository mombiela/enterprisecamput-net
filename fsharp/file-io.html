Introduction to File I/O

File I/O (Input/Output) is a fundamental aspect of programming that allows you to read from and write to files. In F#, you can handle file operations using the System.IO namespace, which provides various classes and methods to work with files and directories.

Reading Files

Reading Entire File

To read the entire content of a file, you can use the File.ReadAllText method.

open System.IO

let readFile path =
    let content = File.ReadAllText(path)
    printfn "File Content:\n%s" content

// Example usage
readFile "example.txt"

Reading File Line by Line

To read a file line by line, you can use the File.ReadLines method.

open System.IO

let readFileLines path =
    let lines = File.ReadLines(path)
    for line in lines do
        printfn "%s" line

// Example usage
readFileLines "example.txt"

Writing Files

Writing Entire Content

To write a string to a file, you can use the File.WriteAllText method.

open System.IO

let writeFile path content =
    File.WriteAllText(path, content)
    printfn "Content written to %s" path

// Example usage
writeFile "example.txt" "Hello, F# World!"

Writing Line by Line

To write multiple lines to a file, you can use the File.WriteAllLines method.

open System.IO

let writeFileLines path lines =
    File.WriteAllLines(path, lines)
    printfn "Lines written to %s" path

// Example usage
writeFileLines "example.txt" [| "Line 1"; "Line 2"; "Line 3" |]

Appending to Files

To append content to an existing file, you can use the File.AppendAllText or File.AppendAllLines methods.

Appending Text

open System.IO

let appendToFile path content =
    File.AppendAllText(path, content)
    printfn "Content appended to %s" path

// Example usage
appendToFile "example.txt" "\nAppended line"

Appending Lines

open System.IO

let appendLinesToFile path lines =
    File.AppendAllLines(path, lines)
    printfn "Lines appended to %s" path

// Example usage
appendLinesToFile "example.txt" [| "Appended Line 1"; "Appended Line 2" |]

Working with Binary Files

For binary file operations, you can use FileStream along with BinaryReader and BinaryWriter.

Writing Binary Data

open System.IO

let writeBinaryFile path data =
    use fs = new FileStream(path, FileMode.Create, FileAccess.Write)
    use bw = new BinaryWriter(fs)
    bw.Write(data)
    printfn "Binary data written to %s" path

// Example usage
writeBinaryFile "example.bin" 12345

Reading Binary Data

open System.IO

let readBinaryFile path =
    use fs = new FileStream(path, FileMode.Open, FileAccess.Read)
    use br = new BinaryReader(fs)
    let data = br.ReadInt32()
    printfn "Binary data read from %s: %d" path data

// Example usage
readBinaryFile "example.bin"

Handling Exceptions

File operations can fail due to various reasons (e.g., file not found, access denied). It is important to handle exceptions properly.

open System.IO

let safeReadFile path =
    try
        let content = File.ReadAllText(path)
        printfn "File Content:\n%s" content
    with
    | :? FileNotFoundException ->
        printfn "File not found: %s" path
    | :? UnauthorizedAccessException ->
        printfn "Access denied to file: %s" path
    | ex ->
        printfn "An error occurred: %s" (ex.Message)

// Example usage
safeReadFile "nonexistent.txt"

Summary

In this section, we covered the basics of File I/O in F#. You learned how to:

  • Read files entirely or line by line.
  • Write and append text or lines to files.
  • Work with binary files using FileStream, BinaryReader, and BinaryWriter.
  • Handle exceptions that may occur during file operations.

Understanding File I/O is crucial for many applications, and mastering these techniques will help you handle data storage and retrieval efficiently in your F# programs.
