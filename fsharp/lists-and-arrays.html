Introduction

In F#, lists and arrays are fundamental data structures that allow you to store collections of elements. Understanding how to use these structures effectively is crucial for any F# programmer. This section will cover the basics of lists and arrays, their differences, and how to manipulate them.

Lists

Lists in F# are immutable, ordered collections of elements of the same type. They are defined using square brackets and semicolons to separate elements.

Creating Lists

To create a list, you can use the following syntax:

let numbers = [1; 2; 3; 4; 5]
let fruits = ["apple"; "banana"; "cherry"]

Common List Operations

  • Accessing Elements: Use the List.item function or the .[index] syntax.
  • Appending Elements: Use the @ operator to concatenate lists.
  • Mapping: Use the List.map function to apply a function to each element.
let firstElement = List.item 0 numbers
let secondElement = numbers.[1]
let combinedList = numbers @ [6; 7; 8]
let squaredNumbers = List.map (fun x -> x * x) numbers

Pattern Matching with Lists

Pattern matching is a powerful feature in F# that allows you to deconstruct lists easily.

let rec sumList lst =
    match lst with
    | [] -> 0
    | head :: tail -> head + sumList tail

let total = sumList numbers

Arrays

Arrays in F# are mutable, fixed-size collections of elements of the same type. They are defined using square brackets and vertical bars.

Creating Arrays

To create an array, you can use the following syntax:

let numbersArray = [| 1; 2; 3; 4; 5 |]
let fruitsArray = [| "apple"; "banana"; "cherry" |]

Common Array Operations

  • Accessing Elements: Use the .[index] syntax.
  • Modifying Elements: Directly assign a new value to an index.
  • Mapping: Use the Array.map function to apply a function to each element.
let firstElementArray = numbersArray.[0]
numbersArray.[1] <- 10
let squaredNumbersArray = Array.map (fun x -> x * x) numbersArray

Array vs List

| Feature | List (Immutable) | Array (Mutable) | |-----------------|------------------|-----------------| | Mutability | Immutable | Mutable | | Size | Dynamic | Fixed | | Performance | Slower for updates| Faster for updates| | Syntax | [1; 2; 3] | [| 1; 2; 3 |] |

Advanced Topics

Higher-Order Functions

Both lists and arrays support higher-order functions like map, filter, and fold.

let evenNumbers = List.filter (fun x -> x % 2 = 0) numbers
let sumOfArray = Array.fold (+) 0 numbersArray

Parallel Processing with Arrays

Arrays can be processed in parallel using the Array.Parallel module for performance improvements.

let squaredNumbersParallel = Array.Parallel.map (fun x -> x * x) numbersArray

Conclusion

Lists and arrays are essential data structures in F#. Lists are immutable and dynamically sized, making them suitable for functional programming paradigms. Arrays are mutable and fixed in size, offering better performance for certain operations. Understanding when and how to use these structures will greatly enhance your ability to write efficient and effective F# code.
