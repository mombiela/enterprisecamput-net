Introduction

F# is a functional-first programming language that also supports object-oriented and imperative programming. Understanding the basic syntax and structure of F# is crucial for writing efficient and readable code. This section will cover the fundamental elements of F# syntax and structure, providing a solid foundation for further learning.

Key Concepts

Comments

  • Single-line comments: Use // to comment out a single line.
  • Multi-line comments: Use (* ... *) to comment out multiple lines.
// This is a single-line comment

(* This is a 
   multi-line comment *)

Variables and Values

  • Immutable values: Defined using the let keyword.
  • Mutable values: Defined using the let mutable keyword.
let x = 10  // Immutable value
let mutable y = 20  // Mutable value
y <- 30  // Changing the value of y

Basic Data Types

  • Integer: int
  • Floating-point: float
  • Boolean: bool
  • String: string
let a: int = 5
let b: float = 3.14
let c: bool = true
let d: string = "Hello, F#"

Functions

  • Defined using the let keyword.
  • Can be nested and support recursion.
let add x y = x + y  // Simple function

let rec factorial n =  // Recursive function
    if n <= 1 then 1
    else n * factorial (n - 1)

Conditional Expressions

  • Use if ... then ... else for conditional logic.
let isEven n =
    if n % 2 = 0 then
        true
    else
        false

Pattern Matching

  • Powerful feature for deconstructing data types.
let describeNumber n =
    match n with
    | 0 -> "Zero"
    | 1 -> "One"
    | _ -> "Other"

Lists

  • Immutable collections of elements.
  • Created using square brackets [].
let numbers = [1; 2; 3; 4; 5]
let firstNumber = List.head numbers
let remainingNumbers = List.tail numbers

Tuples

  • Group multiple values into a single compound value.
  • Created using parentheses ().
let person = ("John", 30)
let name, age = person

Records

  • Define custom data types with named fields.
type Person = { Name: string; Age: int }
let john = { Name = "John"; Age = 30 }

Discriminated Unions

  • Define a type that can be one of several named cases.
type Shape =
    | Circle of float
    | Rectangle of float * float

let area shape =
    match shape with
    | Circle radius -> 3.14 * radius * radius
    | Rectangle (width, height) -> width * height

Conclusion

Understanding the basic syntax and structure of F# is essential for writing effective code. This section covered the foundational elements, including comments, variables, data types, functions, conditional expressions, pattern matching, lists, tuples, records, and discriminated unions. With these basics, you are well-equipped to delve deeper into more advanced features and paradigms of F# programming.
