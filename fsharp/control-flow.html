Introduction

Control flow is a fundamental concept in programming that allows you to dictate the order in which statements are executed. In F#, control flow is managed using conditionals and loops. This section will cover the basics of these constructs, providing examples and exercises to help you understand and apply them effectively.

Conditionals

If-Else Statements

The if-else statement is used to execute code based on a condition.

Syntax

if condition then
    // code to execute if condition is true
else
    // code to execute if condition is false

Example

let number = 10

if number > 0 then
    printfn "The number is positive."
else
    printfn "The number is non-positive."

Nested If-Else Statements

You can nest if-else statements to handle multiple conditions.

Example

let number = 0

if number > 0 then
    printfn "The number is positive."
elif number < 0 then
    printfn "The number is negative."
else
    printfn "The number is zero."

Pattern Matching

Pattern matching is a powerful feature in F# that allows you to match values against patterns and execute code based on the match.

Syntax

match expression with
| pattern1 -> // code to execute if pattern1 matches
| pattern2 -> // code to execute if pattern2 matches
| _ -> // code to execute if no patterns match

Example

let number = 10

match number with
| n when n > 0 -> printfn "The number is positive."
| n when n < 0 -> printfn "The number is negative."
| _ -> printfn "The number is zero."

Loops

For Loops

The for loop is used to iterate over a range of values.

Syntax

for variable in start .. end do
    // code to execute for each iteration

Example

for i in 1 .. 5 do
    printfn "Iteration %d" i

While Loops

The while loop is used to execute code as long as a condition is true.

Syntax

while condition do
    // code to execute while condition is true

Example

let mutable count = 5

while count > 0 do
    printfn "Count is %d" count
    count <- count - 1

Recursive Functions

In F#, loops can often be replaced with recursive functions, which are functions that call themselves.

Example

let rec factorial n =
    if n <= 1 then 1
    else n * factorial (n - 1)

printfn "Factorial of 5 is %d" (factorial 5)

Exercises

Exercise 1: Basic If-Else

Write a function that takes an integer and returns a string indicating whether the number is positive, negative, or zero.

Exercise 2: Pattern Matching

Write a function that takes a day of the week (as a string) and returns whether it is a weekday or a weekend.

Exercise 3: For Loop

Write a function that prints the first 10 numbers in the Fibonacci sequence using a for loop.

Exercise 4: While Loop

Write a function that takes an integer and prints all the even numbers from 0 to that integer using a while loop.

Exercise 5: Recursive Function

Write a recursive function to calculate the greatest common divisor (GCD) of two numbers.

Conclusion

In this section, we covered the basics of control flow in F#, including conditionals and loops. We explored if-else statements, pattern matching, for loops, while loops, and recursive functions. Understanding these constructs is crucial for writing efficient and effective F# code. Make sure to practice the exercises to solidify your understanding.
