Introduction

Perl, known for its text processing capabilities, is a versatile and powerful programming language. Understanding its basic syntax and structure is essential for writing efficient and readable code. This section will cover the fundamental elements of Perl syntax, including variables, operators, control structures, and basic input/output operations.

Variables

Variables in Perl are used to store data. There are three main types of variables:

  • Scalars: Store single values (numbers, strings).
  • Arrays: Store ordered lists of scalars.
  • Hashes: Store sets of key-value pairs.

Scalars

Scalars are prefixed with a dollar sign ($).

# Scalar variables
my $name = "John";
my $age = 30;
my $pi = 3.14;

Arrays

Arrays are prefixed with an at sign (@).

# Array variables
my @colors = ("red", "green", "blue");
print $colors[0]; # Outputs: red

Hashes

Hashes are prefixed with a percent sign (%).

# Hash variables
my %fruit_colors = (
    apple  => "red",
    banana => "yellow",
    grape  => "purple"
);
print $fruit_colors{"apple"}; # Outputs: red

Operators

Perl supports various operators for performing operations on variables.

Arithmetic Operators

  • Addition: +
  • Subtraction: -
  • Multiplication: *
  • Division: /
  • Modulus: %
my $sum = 5 + 10; # 15
my $product = 4 * 3; # 12

String Operators

  • Concatenation: .
my $greeting = "Hello, " . "world!"; # Hello, world!

Comparison Operators

  • Numeric: ==, !=, <, >, <=, >=
  • String: eq, ne, lt, gt, le, ge
if (5 == 5) {
    print "Numbers are equal\n";
}

if ("apple" eq "apple") {
    print "Strings are equal\n";
}

Control Structures

Control structures manage the flow of the program.

Conditional Statements

  • if, elsif, else
my $number = 10;

if ($number > 10) {
    print "Number is greater than 10\n";
} elsif ($number == 10) {
    print "Number is equal to 10\n";
} else {
    print "Number is less than 10\n";
}

Looping Statements

  • while
  • for
  • foreach
# while loop
my $i = 0;
while ($i < 5) {
    print "i = $i\n";
    $i++;
}

# for loop
for (my $j = 0; $j < 5; $j++) {
    print "j = $j\n";
}

# foreach loop
my @numbers = (1, 2, 3, 4, 5);
foreach my $num (@numbers) {
    print "num = $num\n";
}

Input and Output

Basic input and output operations are essential for interacting with users and files.

Standard Input

Using the <STDIN> filehandle to read input from the user.

print "Enter your name: ";
my $name = <STDIN>;
chomp($name); # Remove newline character
print "Hello, $name!\n";

Standard Output

Using the print function to display output.

print "This is a simple message\n";

Conclusion

Understanding the basic syntax and structure of Perl is the first step in mastering the language. This section covered the essentials, including variables, operators, control structures, and basic input/output operations. With these fundamentals, you can start writing simple Perl scripts and gradually move on to more complex tasks. Continue practicing and exploring to deepen your understanding and proficiency in Perl.
