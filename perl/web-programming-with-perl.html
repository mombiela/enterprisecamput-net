Writing Your First Perl CGI Script

  • Creating a simple Perl CGI script.
  • Explanation of the script structure.
  • Running the script on a web server.
#!/usr/bin/perl
use strict;
use warnings;
use CGI;

my $cgi = CGI->new;
print $cgi->header;
print $cgi->start_html('Hello World');
print $cgi->h1('Hello World from Perl CGI!');
print $cgi->end_html;

Handling Form Data

  • Creating HTML forms.
  • Retrieving form data using Perl CGI.
  • Validating and processing form input.
#!/usr/bin/perl
use strict;
use warnings;
use CGI;

my $cgi = CGI->new;
print $cgi->header;
print $cgi->start_html('Form Example');
print $cgi->start_form;
print $cgi->textfield(-name=>'name');
print $cgi->submit;
print $cgi->end_form;

if ($cgi->param) {
    my $name = $cgi->param('name');
    print $cgi->h1("Hello, $name!");
}
print $cgi->end_html;

Database Integration

  • Connecting to a database using DBI.
  • Performing CRUD operations.
  • Using ORM (Object-Relational Mapping) with DBIx::Class.
use DBI;

my $dbh = DBI->connect("DBI:mysql:database=testdb;host=localhost", "user", "password", {'RaiseError' => 1});

my $sth = $dbh->prepare("SELECT * FROM users");
$sth->execute();

while (my @row = $sth->fetchrow_array) {
    print "User: $row[0], Email: $row[1]\n";
}

$sth->finish;
$dbh->disconnect;
