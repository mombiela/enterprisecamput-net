Associative Arrays in REXX
Data Structures ►

Associative arrays, also known as stem variables in REXX, are a powerful feature that allows you to create and manipulate collections of data indexed by strings. This topic will guide you from the basics to advanced usage of associative arrays in REXX.

Introduction to Associative Arrays

Associative arrays in REXX are implemented using stem variables. A stem variable is a collection of variables that share a common prefix, known as the stem name. Each element in the collection is accessed using a unique index.

Key Concepts

  • Stem Variable: A variable that acts as a prefix for a collection of related variables.
  • Index: A string or number used to access a specific element in the stem variable.
  • Element: An individual item in the associative array, accessed using the stem name and an index.

Basic Syntax

To define and use a stem variable in REXX, you follow this syntax:

stem.index = value

Here, stem is the stem name, index is the unique identifier for the element, and value is the data you want to store.

Example

/* Define a stem variable */
students.1 = "Alice"
students.2 = "Bob"
students.3 = "Charlie"

/* Access elements */
say students.1  /* Output: Alice */
say students.2  /* Output: Bob */
say students.3  /* Output: Charlie */

Advanced Usage of Associative Arrays

As you become more comfortable with stem variables, you can start using more advanced features and techniques.

Dynamic Indexing

You can use dynamic values for the index, allowing for more flexible and powerful data structures.

/* Using dynamic indexing */
do i = 1 to 3
    students.i = "Student" || i
end

/* Access elements dynamically */
do i = 1 to 3
    say students.i  /* Output: Student1, Student2, Student3 */
end

Multi-Dimensional Associative Arrays

REXX allows you to create multi-dimensional associative arrays by using multiple indices.

/* Define a multi-dimensional associative array */
grades.Alice.Math = 95
grades.Alice.English = 88
grades.Bob.Math = 78
grades.Bob.English = 85

/* Access elements */
say grades.Alice.Math     /* Output: 95 */
say grades.Bob.English    /* Output: 85 */

Iterating Over Associative Arrays

You can iterate over all elements in a stem variable using the do loop and the stem. prefix.

/* Define a stem variable */
students.1 = "Alice"
students.2 = "Bob"
students.3 = "Charlie"

/* Iterate over elements */
i = 1
do while students.i \= ""
    say students.i
    i = i + 1
end

Checking for Existence

You can check if an element exists in a stem variable using the symbol function.

/* Define a stem variable */
students.1 = "Alice"

/* Check for existence */
if symbol("students.2") = "VAR" then
    say "Student 2 exists"
else
    say "Student 2 does not exist"

Conclusion

Associative arrays, or stem variables, in REXX provide a flexible and powerful way to manage collections of data. From basic usage to advanced techniques like dynamic indexing and multi-dimensional arrays, stem variables can greatly enhance your REXX programming capabilities. Practice using these features to become proficient in handling complex data structures in REXX.
