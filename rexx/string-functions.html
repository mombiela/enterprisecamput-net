Introduction

REXX (Restructured Extended Executor) is a programming language that is particularly strong in string manipulation. This section will cover various string functions in REXX, from basic to advanced, providing clear explanations, examples, and exercises.

Basic String Functions

LENGTH

  • Purpose: Returns the length of a string.
  • Syntax: LENGTH(string)
/* Example */
string = "Hello, World!"
say LENGTH(string)  /* Output: 13 */

SUBSTR

  • Purpose: Extracts a substring from a string.
  • Syntax: SUBSTR(string, start, length)
/* Example */
string = "Hello, World!"
say SUBSTR(string, 8, 5)  /* Output: World */

POS

  • Purpose: Finds the position of a substring within a string.
  • Syntax: POS(needle, haystack, start)
/* Example */
string = "Hello, World!"
say POS("World", string)  /* Output: 8 */

LEFT and RIGHT

  • Purpose: Extracts a specified number of characters from the left or right side of a string.
  • Syntax: LEFT(string, length) and RIGHT(string, length)
/* Example */
string = "Hello, World!"
say LEFT(string, 5)   /* Output: Hello */
say RIGHT(string, 6)  /* Output: World! */

Intermediate String Functions

STRIP

  • Purpose: Removes leading and/or trailing spaces from a string.
  • Syntax: STRIP(string, option, char)
/* Example */
string = "   Hello, World!   "
say STRIP(string)  /* Output: "Hello, World!" */

OVERLAY

  • Purpose: Overlays a substring onto a string.
  • Syntax: OVERLAY(new, old, start, length)
/* Example */
string = "Hello, World!"
say OVERLAY("REXX", string, 8, 4)  /* Output: Hello, REXX! */

TRANSLATE

  • Purpose: Translates characters in a string.
  • Syntax: TRANSLATE(string, to, from)
/* Example */
string = "Hello, World!"
say TRANSLATE(string, "123", "Hlo")  /* Output: 1e22, W2r2d! */

VERIFY

  • Purpose: Verifies characters in a string against a set of allowed characters.
  • Syntax: VERIFY(string, reference, option)
/* Example */
string = "Hello, World!"
say VERIFY(string, "ABCDEFGHIJKLMNOPQRSTUVWXYZabcdefghijklmnopqrstuvwxyz, !")  /* Output: 0 */

Advanced String Functions

WORD

  • Purpose: Extracts a specific word from a string.
  • Syntax: WORD(string, n)
/* Example */
string = "Hello, World! Welcome to REXX."
say WORD(string, 3)  /* Output: Welcome */

WORDPOS

  • Purpose: Finds the position of a word within a string.
  • Syntax: WORDPOS(word, string)
/* Example */
string = "Hello, World! Welcome to REXX."
say WORDPOS("Welcome", string)  /* Output: 3 */

WORDS

  • Purpose: Counts the number of words in a string.
  • Syntax: WORDS(string)
/* Example */
string = "Hello, World! Welcome to REXX."
say WORDS(string)  /* Output: 5 */

DELSTR

  • Purpose: Deletes a substring from a string.
  • Syntax: DELSTR(string, start, length)
/* Example */
string = "Hello, World!"
say DELSTR(string, 8, 6)  /* Output: Hello, */

INSERT

  • Purpose: Inserts a substring into a string.
  • Syntax: INSERT(new, old, start, length, pad)
/* Example */
string = "Hello, World!"
say INSERT("REXX", string, 8)  /* Output: Hello, REXXXWorld! */

Summary

In this section, we covered a range of string functions in REXX, from basic to advanced. These functions are essential for effective string manipulation, which is a common task in many programming scenarios. Understanding and mastering these functions will significantly enhance your ability to work with text data in REXX.

Exercises

  1. Write a REXX script to extract the domain name from an email address.
  2. Create a function that capitalizes the first letter of each word in a string.
  3. Write a script to count the number of vowels in a given string.
  4. Develop a function that reverses the words in a sentence while keeping the words themselves in the original order.

By practicing these exercises, you will reinforce your understanding of REXX string functions and improve your problem-solving skills.
