Introduction

In this section, we will explore arrays and compound variables in REXX. Arrays and compound variables are essential for handling collections of data efficiently. We will cover the basics of arrays, how to create and manipulate them, and delve into more advanced topics such as multi-dimensional arrays and dynamic arrays.

Key Concepts

  • Arrays: A collection of elements that can be accessed using an index.
  • Compound Variables: Variables that can hold multiple values, often used to simulate arrays in REXX.
  • Indexing: The method of accessing individual elements within an array or compound variable.

Simple Arrays in REXX

REXX does not have built-in support for traditional arrays. Instead, it uses compound variables to simulate arrays.

Creating and Using Compound Variables

Compound variables in REXX are created using a base name followed by one or more subscripts.

/* Simple array using compound variables */
array.1 = "Apple"
array.2 = "Banana"
array.3 = "Cherry"

say array.1  /* Output: Apple */
say array.2  /* Output: Banana */
say array.3  /* Output: Cherry */

Looping Through Arrays

You can loop through arrays using a DO loop.

/* Loop through the array */
do i = 1 to 3
    say array.i
end

Multi-Dimensional Arrays

REXX allows you to create multi-dimensional arrays by using multiple subscripts.

Example of a Two-Dimensional Array

/* Two-dimensional array */
matrix.1.1 = "A1"
matrix.1.2 = "A2"
matrix.2.1 = "B1"
matrix.2.2 = "B2"

say matrix.1.1  /* Output: A1 */
say matrix.2.2  /* Output: B2 */

Looping Through Multi-Dimensional Arrays

/* Loop through a two-dimensional array */
do i = 1 to 2
    do j = 1 to 2
        say matrix.i.j
    end
end

Dynamic Arrays

REXX allows you to dynamically add elements to arrays.

Adding Elements Dynamically

/* Adding elements dynamically */
array.4 = "Date"
array.5 = "Elderberry"

do i = 1 to 5
    say array.i
end

Practical Example: Storing and Retrieving Data

Let's create a simple program that stores and retrieves student names and their grades.

/* Program to store and retrieve student grades */
students.1 = "Alice"
grades.1 = 85
students.2 = "Bob"
grades.2 = 90
students.3 = "Charlie"
grades.3 = 78

/* Retrieve and display student names and grades */
do i = 1 to 3
    say students.i "scored" grades.i
end

Conclusion

In this section, we have covered the basics of arrays and compound variables in REXX. We learned how to create and manipulate simple and multi-dimensional arrays using compound variables. We also explored how to dynamically add elements to arrays and loop through them. Understanding these concepts is crucial for efficiently handling collections of data in REXX programs. Practice these examples and try creating your own arrays to become more comfortable with these concepts.
