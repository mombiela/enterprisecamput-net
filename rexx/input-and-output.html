Introduction

Input and output (I/O) operations are fundamental in any programming language, allowing programs to interact with users and other systems. In REXX, I/O operations are straightforward and easy to learn, making it an excellent language for beginners and advanced users alike.

Basic Input

Using the `PULL` Instruction

The PULL instruction is used to read input from the user. It waits for the user to type something and then stores the input in a variable.

/* Example of using PULL to get user input */
say "Please enter your name:"
pull name
say "Hello, " name "!"
  • Explanation:
    • say "Please enter your name:" displays a prompt to the user.
    • pull name waits for the user to type something and stores it in the variable name.
    • say "Hello, " name "!" outputs a greeting using the input.

Basic Output

Using the `SAY` Instruction

The SAY instruction is used to display output to the user.

/* Example of using SAY to display output */
say "Welcome to the REXX programming course!"
  • Explanation:
    • say "Welcome to the REXX programming course!" prints the message to the screen.

File Input and Output

Reading from a File

To read from a file, you use the LINEIN function. This function reads a line from a file.

/* Example of reading from a file */
fileName = "example.txt"
file = open(fileName, "r")
if file = "" then do
  say "Error: Could not open file."
  exit
end

line = linein(file)
do while line \= ""
  say line
  line = linein(file)
end

call close file
  • Explanation:
    • fileName = "example.txt" specifies the file to read.
    • file = open(fileName, "r") opens the file for reading.
    • if file = "" then do checks if the file was successfully opened.
    • line = linein(file) reads a line from the file.
    • do while line \= "" loops through the file until the end.
    • say line prints each line to the screen.
    • call close file closes the file.

Writing to a File

To write to a file, you use the LINEOUT function. This function writes a line to a file.

/* Example of writing to a file */
fileName = "output.txt"
file = open(fileName, "w")
if file = "" then do
  say "Error: Could not open file."
  exit
end

line = "This is a line of text."
call lineout file, line

call close file
  • Explanation:
    • fileName = "output.txt" specifies the file to write to.
    • file = open(fileName, "w") opens the file for writing.
    • if file = "" then do checks if the file was successfully opened.
    • line = "This is a line of text." specifies the line to write.
    • call lineout file, line writes the line to the file.
    • call close file closes the file.

Advanced I/O Operations

Using Streams

REXX provides stream functions for more advanced file operations. The STREAM function can be used to query and manipulate file streams.

/* Example of using STREAM function */
fileName = "example.txt"
file = open(fileName, "r")
if file = "" then do
  say "Error: Could not open file."
  exit
end

position = stream(file, "seek", "end")
say "The file size is:" position "bytes."

call close file
  • Explanation:
    • position = stream(file, "seek", "end") moves the file pointer to the end and returns the file size.
    • say "The file size is:" position "bytes." prints the file size.

Conclusion

Understanding input and output operations in REXX is crucial for creating interactive and useful programs. By mastering basic I/O with PULL and SAY, file I/O with LINEIN and LINEOUT, and advanced operations with STREAM, you can handle a wide range of tasks in your REXX programs. Practice these concepts with various examples to become proficient in REXX I/O operations.
