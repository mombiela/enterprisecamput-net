Introduction

In this section of the course on Technological Architecture, we will explore the fundamental components of a technological system. Understanding these components is crucial for designing, implementing, and maintaining efficient and scalable systems.

Main Components

Hardware

Hardware is the physical foundation of any technological system. It includes all the physical devices and equipment necessary for the system's operation.

  • Servers: Machines that provide services to other programs or devices.
  • Storage Devices: Hard disk drives (HDD), solid-state drives (SSD), and network-attached storage (NAS) systems.
  • Networks: Routers, switches, and other devices that facilitate communication between different parts of the system.
  • Input/Output Devices: Keyboards, mice, monitors, printers, etc.

Software

Software is the set of instructions that tells the hardware what to do. It can be divided into several categories:

  • Operating Systems: Software that manages the hardware and provides services for other programs. Examples include Windows, Linux, and macOS.
  • Applications: Programs designed to perform specific tasks. Examples include web browsers, word processors, and database management systems.
  • Middleware: Software that connects different applications or services. Examples include application servers and messaging systems.

Networks

Networks allow communication between different components of the technological system. They can be local (LAN) or global (WAN).

  • Internet Protocol (IP): Defines how data is sent and received over the network.
  • Hypertext Transfer Protocol (HTTP/HTTPS): Used for communication on the web.
  • Transmission Control Protocol (TCP): Provides reliable and ordered communication between applications.

Databases

Databases are systems that store and manage data in a structured manner.

  • Relational Databases (RDBMS): Use tables to store data and SQL to manage it. Examples include MySQL, PostgreSQL, and Oracle.
  • NoSQL Databases: Use different data models such as documents, graphs, and key-value pairs. Examples include MongoDB, Cassandra, and Redis.

Security

Security is a critical component that protects the system and data against unauthorized access and attacks.

  • Authentication and Authorization: Identity verification and access control.
  • Encryption: Data protection through encryption techniques.
  • Firewalls: Systems that control network traffic based on security rules.

Code Example

Below is an example of how to configure a connection to a database in Python using a relational database like PostgreSQL.

import psycopg2

try:
    # Connection to the database
    connection = psycopg2.connect(
        user="your_user",
        password="your_password",
        host="127.0.0.1",
        port="5432",
        database="your_database"
    )

    cursor = connection.cursor()
    # Execute a query
    cursor.execute("SELECT version();")
    record = cursor.fetchone()
    print("Connected to - ", record, "\n")

except (Exception, psycopg2.Error) as error:
    print("Error while connecting to the database", error)
finally:
    # Close the connection
    if connection:
        cursor.close()
        connection.close()
        print("Database connection closed")

Conclusion

Understanding the components of a technological system is essential for any IT professional. From hardware and software to networks and security, each component plays a crucial role in the system's operation. As we progress through the course, we will delve deeper into each of these components to better understand how they integrate and work together.
