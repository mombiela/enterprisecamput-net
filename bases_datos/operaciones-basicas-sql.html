Introduction

In this section, you will learn the basic operations in SQL that are fundamental for interacting with relational databases. These operations include creating, reading, updating, and deleting data, commonly known by their acronym CRUD (Create, Read, Update, Delete).

Creating Data (INSERT)

The INSERT operation is used to add new records to a table in the database.

Basic Syntax

INSERT INTO table_name (column1, column2, column3, ...)
VALUES (value1, value2, value3, ...);

Example

Suppose we have a table called employees with the columns id, name, age, and department.

INSERT INTO employees (id, name, age, department)
VALUES (1, 'Juan Pérez', 30, 'Human Resources');

Reading Data (SELECT)

The SELECT operation is used to query and retrieve data from a table.

Basic Syntax

SELECT column1, column2, ...
FROM table_name
WHERE condition;

Example

To select all employees from the "Human Resources" department:

SELECT name, age
FROM employees
WHERE department = 'Human Resources';

Updating Data (UPDATE)

The UPDATE operation is used to modify existing data in a table.

Basic Syntax

UPDATE table_name
SET column1 = value1, column2 = value2, ...
WHERE condition;

Example

To update "Juan Pérez's" age to 31:

UPDATE employees
SET age = 31
WHERE name = 'Juan Pérez';

Deleting Data (DELETE)

The DELETE operation is used to delete records from a table.

Basic Syntax

DELETE FROM table_name
WHERE condition;

Example

To delete "Juan Pérez" from the employees table:

DELETE FROM employees
WHERE name = 'Juan Pérez';

Comparison of CRUD Operations

| Operation | Description | Basic Syntax | |-----------|-------------|--------------| | INSERT | Adds new records to a table | INSERT INTO table_name (col1, col2, ...) VALUES (val1, val2, ...); | | SELECT | Retrieves data from a table | SELECT col1, col2, ... FROM table_name WHERE condition; | | UPDATE | Modifies existing data in a table | UPDATE table_name SET col1 = val1, col2 = val2, ... WHERE condition; | | DELETE | Deletes records from a table | DELETE FROM table_name WHERE condition; |

Conclusion

The basic operations in SQL are essential for any IT professional working with databases. Mastering INSERT, SELECT, UPDATE, and DELETE will allow you to manage and manipulate data effectively. Practice these commands with different scenarios and conditions to strengthen your understanding and skills in SQL.
