Introduction

Statistics is a fundamental tool in social sciences as it allows for the analysis of data and the extraction of meaningful conclusions about social phenomena. This topic will cover key concepts of statistics applied to social sciences, providing examples and practical exercises to facilitate understanding.

Key Concepts

  • Population and Sample:

    • Population: Complete set of individuals or elements being studied.
    • Sample: Subset of the population selected for the study.

  • Variables:

    • Qualitative: Describe qualities or categories (e.g., gender, marital status).
    • Quantitative: Describe numerical quantities (e.g., income, age).

  • Measures of Central Tendency:

    • Mean: Arithmetic average of a set of data.
    • Median: Central value of an ordered set of data.
    • Mode: Value that appears most frequently in a set of data.

  • Measures of Dispersion:

    • Range: Difference between the maximum and minimum values.
    • Variance: Measure of the dispersion of data relative to the mean.
    • Standard Deviation: Square root of the variance.

  • Probability Distributions:

    • Normal Distribution: Symmetrical bell-shaped distribution.
    • Binomial Distribution: Distribution of outcomes of experiments with two possible results (success or failure).

Practical Examples

Example 1: Calculation of Mean, Median, and Mode

Suppose we have the following data on the monthly income (in dollars) of a group of individuals:

$500, $600, $700, $800, $900, $1000, $1100, $1200, $1300, $1400

  • Mean:

    incomes = [500, 600, 700, 800, 900, 1000, 1100, 1200, 1300, 1400]
mean = sum(incomes) / len(incomes)
print("Mean:", mean)

  • Median:

    incomes.sort()
n = len(incomes)
median = (incomes[n//2 - 1] + incomes[n//2]) / 2 if n % 2 == 0 else incomes[n//2]
print("Median:", median)

  • Mode:

    from collections import Counter
mode = Counter(incomes).most_common(1)[0][0]
print("Mode:", mode)

Example 2: Normal Distribution

Suppose we want to generate a normal distribution of data on the height of individuals in a population:

import numpy as np
import matplotlib.pyplot as plt

# Generate data
mu, sigma = 170, 10  # mean and standard deviation
heights = np.random.normal(mu, sigma, 1000)

# Plot
plt.hist(heights, bins=30, density=True, alpha=0.6, color='g')
plt.title('Normal Distribution of Heights')
plt.xlabel('Height (cm)')
plt.ylabel('Frequency')
plt.show()

Conclusion

Statistics in social sciences is essential for interpreting data and making informed decisions. Understanding concepts such as mean, median, mode, and probability distributions allows professionals to effectively analyze social phenomena. The examples and exercises provided in this topic are a solid foundation for applying statistics in social research and studies.
