Your First Kotlin Program
Basic Syntax and Structure ►

Welcome to your journey into Kotlin programming! In this section, we will guide you through writing your first Kotlin program. By the end of this lesson, you will have a basic understanding of Kotlin syntax and how to run a simple Kotlin application.

Setting Up Your Environment

Before we dive into writing code, let's ensure you have everything set up to run Kotlin programs.

  • Install the JDK: Kotlin runs on the Java Virtual Machine (JVM), so you need to have the Java Development Kit (JDK) installed. You can download it from Oracle's website or use an open-source alternative like OpenJDK.
  • Install an IDE: While you can write Kotlin code in any text editor, using an Integrated Development Environment (IDE) like IntelliJ IDEA (which has excellent Kotlin support) will make your life easier. You can download it from JetBrains' website.

Writing Your First Kotlin Program

Let's start with a simple "Hello, World!" program. This is a traditional first program that prints "Hello, World!" to the console.

Step-by-Step Guide

  1. Create a New Project:

    • Open IntelliJ IDEA.
    • Select "Create New Project".
    • Choose "Kotlin" and then "JVM | IDEA".
    • Click "Next" and configure your project settings (name, location, etc.).
    • Click "Finish".

  2. Create a Kotlin File:

    • Right-click on the src folder in the Project view.
    • Select "New" -> "Kotlin File/Class".
    • Name your file Main.kt.

  3. Write the Code:

    • Open Main.kt and type the following code:
    fun main() {
    println("Hello, World!")
}

Explanation of the Code

  • fun main() { ... }: This defines a function named main. In Kotlin, the main function is the entry point of the application.
  • println("Hello, World!"): This is a standard library function that prints the specified message to the console.

Running Your Program

To run your program, follow these steps:

  1. Build the Project:

    • Click on the "Build" menu and select "Build Project".

  2. Run the Program:

    • Right-click on Main.kt in the Project view.
    • Select "Run 'MainKt'".

You should see "Hello, World!" printed in the Run window at the bottom of the IDE.

Key Concepts

  • Functions: In Kotlin, functions are declared using the fun keyword.
  • Standard Library Functions: Functions like println are part of Kotlin's standard library and provide common functionality.
  • Entry Point: The main function serves as the entry point of a Kotlin application.

Conclusion

Congratulations! You've written and run your first Kotlin program. This simple "Hello, World!" example introduced you to the basic structure of a Kotlin application, including how to define a function and use standard library functions. As we progress through this course, you'll learn more about Kotlin's powerful features and how to use them to build robust applications. Keep experimenting and happy coding!
