Introduction to Collections

Kotlin provides a rich set of collection types that are both mutable and immutable. Collections are used to store groups of related objects.

  • List: An ordered collection of elements.
  • Set: A collection of unique elements.
  • Map: A collection of key-value pairs.

List

A List is an ordered collection that allows duplicate elements.

val fruits = listOf("Apple", "Banana", "Cherry")
println(fruits) // Output: [Apple, Banana, Cherry]

MutableList

A MutableList is a List with methods to add, remove, and update elements.

val fruits = mutableListOf("Apple", "Banana")
fruits.add("Cherry")
println(fruits) // Output: [Apple, Banana, Cherry]

Set

A Set is a collection that contains no duplicate elements.

val numbers = setOf(1, 2, 3, 3)
println(numbers) // Output: [1, 2, 3]

MutableSet

A MutableSet is a Set with methods to add, remove, and update elements.

val numbers = mutableSetOf(1, 2, 3)
numbers.add(4)
println(numbers) // Output: [1, 2, 3, 4]

Map

A Map is a collection of key-value pairs.

val countryCodes = mapOf("US" to "United States", "CA" to "Canada")
println(countryCodes) // Output: {US=United States, CA=Canada}

MutableMap

A MutableMap is a Map with methods to add, remove, and update key-value pairs.

val countryCodes = mutableMapOf("US" to "United States")
countryCodes["CA"] = "Canada"
println(countryCodes) // Output: {US=United States, CA=Canada}

Functional Operations on Collections

Kotlin collections come with a variety of functional operations that make it easy to manipulate data.

  • map: Transforms each element in the collection.
  • filter: Filters elements based on a condition.
  • reduce: Reduces the collection to a single value.
  • forEach: Performs an action for each element.
  • sortedBy: Sorts the collection based on a selector function.

map

Transforms each element in the collection.

val numbers = listOf(1, 2, 3)
val doubled = numbers.map { it * 2 }
println(doubled) // Output: [2, 4, 6]

filter

Filters elements based on a condition.

val numbers = listOf(1, 2, 3, 4, 5)
val evenNumbers = numbers.filter { it % 2 == 0 }
println(evenNumbers) // Output: [2, 4]

reduce

Reduces the collection to a single value.

val numbers = listOf(1, 2, 3, 4)
val sum = numbers.reduce { acc, num -> acc + num }
println(sum) // Output: 10

forEach

Performs an action for each element.

val fruits = listOf("Apple", "Banana", "Cherry")
fruits.forEach { println(it) }
// Output:
// Apple
// Banana
// Cherry

sortedBy

Sorts the collection based on a selector function.

val fruits = listOf("Banana", "Apple", "Cherry")
val sortedFruits = fruits.sortedBy { it }
println(sortedFruits) // Output: [Apple, Banana, Cherry]

Conclusion

Kotlin's collection framework provides a powerful and flexible way to work with groups of data. Understanding the different types of collections and their functional operations allows you to write more concise and readable code. By mastering these concepts, you can efficiently manipulate data and perform complex transformations with ease.
