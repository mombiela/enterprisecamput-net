Installing Git
Basic Git Terminology ►

Git is a powerful version control system that is essential for modern software development. This section will guide you through the process of installing Git on various operating systems. By the end of this section, you will have Git installed and ready to use on your machine.

Installing Git on Windows

  1. Download the Git Installer:

    • Visit the official Git website: Git for Windows.
    • Click on the "Download" button to get the latest version of the Git installer.

  2. Run the Installer:

    • Locate the downloaded .exe file and double-click to run it.
    • Follow the installation wizard steps. The default settings are generally suitable for most users.

  3. Configure Git:

    • After installation, open the Git Bash terminal from the Start menu.
    • Set your username and email: 
      git config --global user.name "Your Name"
git config --global user.email "your.email@example.com"

Installing Git on macOS

  1. Using Homebrew (Recommended):

    • If you don't have Homebrew installed, you can install it by running the following command in your terminal: 
      /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
    • Once Homebrew is installed, install Git by running: 
      brew install git

  2. Using the Git Installer:

    • Visit the official Git website: Git for macOS.
    • Download and run the installer package.

  3. Configure Git:

    • Open the Terminal application.
    • Set your username and email: 
      git config --global user.name "Your Name"
git config --global user.email "your.email@example.com"

Installing Git on Linux

  1. Using the Package Manager:

    • For Debian-based distributions (e.g., Ubuntu): 
      sudo apt update
sudo apt install git
    • For RPM-based distributions (e.g., Fedora): 
      sudo dnf install git

  2. Configure Git:

    • Open your terminal.
    • Set your username and email: 
      git config --global user.name "Your Name"
git config --global user.email "your.email@example.com"

Verifying the Installation

After installing Git, you should verify that it has been installed correctly. Open your terminal or Git Bash and run the following command:

git --version

You should see output similar to:

git version 2.x.x

This confirms that Git is installed and ready to use.

Summary

In this section, you learned how to install Git on Windows, macOS, and Linux. You also configured Git with your username and email, which is essential for tracking changes and contributions. Now that Git is installed, you are ready to start using it for version control in your projects.
