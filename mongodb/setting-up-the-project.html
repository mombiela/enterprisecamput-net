Setting up a MongoDB project is the first step towards building applications that leverage MongoDB's powerful NoSQL database capabilities. This section will guide you through the process of installing MongoDB, setting up your development environment, and creating your first MongoDB project.

Installing MongoDB

Before you can start using MongoDB, you need to install it on your machine. Follow these steps to install MongoDB on different operating systems:

Windows

  1. Download MongoDB:

    • Visit the MongoDB Download Center and download the Windows installer.

  2. Run the Installer:

    • Double-click the downloaded .msi file and follow the installation instructions.
    • Choose the "Complete" setup type.

  3. Set Up the Environment Variable:

    • Add the MongoDB bin directory (C:\Program Files\MongoDB\Server\<version>\bin) to your system's PATH environment variable.

  4. Verify Installation:

    • Open Command Prompt and type mongo --version to verify the installation.

macOS

  1. Install Homebrew (if not already installed):

    /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

  2. Install MongoDB:

    brew tap mongodb/brew
brew install mongodb-community@<version>

  3. Start MongoDB:

    brew services start mongodb/brew/mongodb-community

  4. Verify Installation:

    • Open Terminal and type mongo --version to verify the installation.

Linux

  1. Import the Public Key:

    wget -qO - https://www.mongodb.org/static/pgp/server-<version>.asc | sudo apt-key add -

  2. Create a List File:

    echo "deb [ arch=amd64,arm64 ] https://repo.mongodb.org/apt/ubuntu focal/mongodb-org/<version> multiverse" | sudo tee /etc/apt/sources.list.d/mongodb-org-<version>.list

  3. Install MongoDB:

    sudo apt-get update
sudo apt-get install -y mongodb-org

  4. Start MongoDB:

    sudo systemctl start mongod

  5. Verify Installation:

    • Open Terminal and type mongo --version to verify the installation.

Setting Up the Development Environment

Once MongoDB is installed, you need to set up your development environment. This involves choosing a programming language, installing necessary drivers, and setting up an Integrated Development Environment (IDE).

Choosing a Programming Language

MongoDB supports various programming languages. Some popular choices include:

  • JavaScript (Node.js)
  • Python
  • Java
  • C#

For this course, we'll use Node.js due to its popularity and ease of use.

Installing Node.js and npm

  1. Download Node.js:

    • Visit the Node.js website and download the LTS version.

  2. Install Node.js:

    • Follow the installation instructions for your operating system.

  3. Verify Installation:

    node --version
npm --version

Setting Up an IDE

Choose an IDE that supports JavaScript development. Some popular choices include:

  • Visual Studio Code
  • WebStorm
  • Atom

For this course, we'll use Visual Studio Code.

  1. Download Visual Studio Code:

    • Visit the Visual Studio Code website and download the installer.

  2. Install Visual Studio Code:

    • Follow the installation instructions for your operating system.

Creating Your First MongoDB Project

Now that your development environment is set up, let's create your first MongoDB project.

Initialize a Node.js Project

  1. Create a Project Directory:

    mkdir my-mongodb-project
cd my-mongodb-project

  2. Initialize the Project:

    npm init -y

Install MongoDB Driver

  1. Install the MongoDB Driver: 
    npm install mongodb

Connect to MongoDB

Create a file named app.js and add the following code to connect to MongoDB:

const { MongoClient } = require('mongodb');

// Connection URL
const url = 'mongodb://localhost:27017';
const client = new MongoClient(url);

// Database Name
const dbName = 'myDatabase';

async function main() {
  // Use connect method to connect to the server
  await client.connect();
  console.log('Connected successfully to server');
  const db = client.db(dbName);

  // Close the connection
  await client.close();
}

main().catch(console.error);

Run the Project

  1. Run the Application:

    node app.js

  2. Verify Connection:

    • You should see the message "Connected successfully to server" in the console.

Conclusion

In this section, we covered the essential steps to set up a MongoDB project. You learned how to install MongoDB on different operating systems, set up your development environment with Node.js and Visual Studio Code, and create a simple Node.js application that connects to a MongoDB database. With your environment ready, you're now prepared to dive deeper into MongoDB and start building powerful applications.
