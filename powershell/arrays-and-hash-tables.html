Arrays and Hash Tables in PowerShell

Introduction

In this section, we will explore two fundamental data structures in PowerShell: Arrays and Hash Tables. Understanding these structures is crucial for efficient data manipulation and storage in your scripts.

Arrays

What is an Array?

An array is a collection of items stored at contiguous memory locations. In PowerShell, arrays can store multiple items of different types.

Creating Arrays

You can create an array by enclosing items in @().

# Creating an array of integers
$numbers = @(1, 2, 3, 4, 5)

# Creating an array of strings
$fruits = @("Apple", "Banana", "Cherry")

# Creating a mixed array
$mixedArray = @(1, "Hello", 3.14, $true)

Accessing Array Elements

Array elements are accessed using their index, starting from 0.

# Accessing the first element
$firstFruit = $fruits[0]  # Output: Apple

# Accessing the third element
$thirdNumber = $numbers[2]  # Output: 3

Modifying Arrays

You can modify array elements by assigning new values to specific indices.

# Changing the second fruit
$fruits[1] = "Blueberry"  # $fruits now contains "Apple", "Blueberry", "Cherry"

Array Methods

PowerShell arrays come with several useful methods.

# Adding an element to the array
$numbers += 6  # $numbers now contains 1, 2, 3, 4, 5, 6

# Removing an element from the array
$numbers = $numbers - 3  # $numbers now contains 1, 2, 4, 5, 6

Iterating Over Arrays

You can use loops to iterate over array elements.

# Using a foreach loop
foreach ($fruit in $fruits) {
    Write-Output $fruit
}

Hash Tables

What is a Hash Table?

A hash table is a collection of key-value pairs, where each key is unique. Hash tables are useful for storing data that needs to be quickly retrieved using a key.

Creating Hash Tables

You can create a hash table using @{}.

# Creating a hash table
$person = @{
    Name = "John"
    Age = 30
    Occupation = "Engineer"
}

Accessing Hash Table Elements

Hash table elements are accessed using their keys.

# Accessing the value associated with the key 'Name'
$name = $person["Name"]  # Output: John

# Accessing the value using dot notation
$age = $person.Age  # Output: 30

Modifying Hash Tables

You can add or modify elements in a hash table by assigning values to keys.

# Adding a new key-value pair
$person["City"] = "New York"

# Modifying an existing value
$person.Age = 31

Hash Table Methods

Hash tables come with several useful methods.

# Checking if a key exists
$exists = $person.ContainsKey("Name")  # Output: True

# Removing a key-value pair
$person.Remove("Occupation")

Iterating Over Hash Tables

You can use loops to iterate over hash table elements.

# Using a foreach loop to iterate over keys
foreach ($key in $person.Keys) {
    Write-Output "$key: $($person[$key])"
}

Conclusion

Arrays and hash tables are powerful data structures in PowerShell that allow you to store and manipulate collections of data efficiently. Arrays are ideal for ordered collections, while hash tables are perfect for key-value pairs. Understanding how to create, access, modify, and iterate over these structures will significantly enhance your scripting capabilities in PowerShell.
