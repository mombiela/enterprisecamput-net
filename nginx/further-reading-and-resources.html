Example Configurations

  • Basic HTTP Server Configuration:

    server {
    listen 80;
    server_name example.com;

    location / {
        root /usr/share/nginx/html;
        index index.html index.htm;
    }
}
    • This configuration sets up a basic HTTP server listening on port 80, serving files from /usr/share/nginx/html.

  • Reverse Proxy Configuration:

    server {
    listen 80;
    server_name example.com;

    location / {
        proxy_pass http://localhost:3000;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
    }
}
    • This configuration sets up NGINX as a reverse proxy, forwarding requests to a backend server running on localhost:3000.
