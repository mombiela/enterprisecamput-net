State management is a crucial concept in React development. It allows you to manage and synchronize the state of your application efficiently. This topic will guide you through the basics of state management, from understanding what state is to implementing advanced state management techniques.

What is State?

State refers to the data that a component maintains. It can change over time and affect how the component renders and behaves.

  • Local State: Managed within a single component.
  • Global State: Shared across multiple components.

Managing State in React

Using useState Hook

The useState hook is the simplest way to manage local state in a functional component.

import React, { useState } from 'react';

function Counter() {
  const [count, setCount] = useState(0);

  return (
    <div>
      <p>You clicked {count} times</p>
      <button onClick={() => setCount(count + 1)}>
        Click me
      </button>
    </div>
  );
}
  • useState: Initializes the state.
  • setCount: Updates the state.

Using useReducer Hook

For more complex state logic, useReducer can be a better option.

import React, { useReducer } from 'react';

const initialState = { count: 0 };

function reducer(state, action) {
  switch (action.type) {
    case 'increment':
      return { count: state.count + 1 };
    case 'decrement':
      return { count: state.count - 1 };
    default:
      throw new Error();
  }
}

function Counter() {
  const [state, dispatch] = useReducer(reducer, initialState);

  return (
    <div>
      <p>Count: {state.count}</p>
      <button onClick={() => dispatch({ type: 'increment' })}>
        +
      </button>
      <button onClick={() => dispatch({ type: 'decrement' })}>
        -
      </button>
    </div>
  );
}
  • useReducer: Manages complex state logic.
  • dispatch: Sends actions to the reducer function.

Global State Management

Context API

The Context API allows you to share state across multiple components without passing props down manually at every level.

import React, { createContext, useContext, useState } from 'react';

const CountContext = createContext();

function CounterProvider({ children }) {
  const [count, setCount] = useState(0);

  return (
    <CountContext.Provider value={{ count, setCount }}>
      {children}
    </CountContext.Provider>
  );
}

function Counter() {
  const { count, setCount } = useContext(CountContext);

  return (
    <div>
      <p>Count: {count}</p>
      <button onClick={() => setCount(count + 1)}>
        Increment
      </button>
    </div>
  );
}

function App() {
  return (
    <CounterProvider>
      <Counter />
    </CounterProvider>
  );
}
  • createContext: Creates a context object.
  • useContext: Consumes the context value.

Redux

Redux is a popular library for managing global state in larger applications.

// store.js
import { createStore } from 'redux';

const initialState = { count: 0 };

function counterReducer(state = initialState, action) {
  switch (action.type) {
    case 'increment':
      return { count: state.count + 1 };
    case 'decrement':
      return { count: state.count - 1 };
    default:
      return state;
  }
}

const store = createStore(counterReducer);

export default store;

// App.js
import React from 'react';
import { Provider, useDispatch, useSelector } from 'react-redux';
import store from './store';

function Counter() {
  const count = useSelector(state => state.count);
  const dispatch = useDispatch();

  return (
    <div>
      <p>Count: {count}</p>
      <button onClick={() => dispatch({ type: 'increment' })}>
        Increment
      </button>
      <button onClick={() => dispatch({ type: 'decrement' })}>
        Decrement
      </button>
    </div>
  );
}

function App() {
  return (
    <Provider store={store}>
      <Counter />
    </Provider>
  );
}

export default App;
  • createStore: Creates a Redux store.
  • Provider: Makes the Redux store available to the app.
  • useSelector: Accesses state from the Redux store.
  • useDispatch: Dispatches actions to the Redux store.

Conclusion

State management is a fundamental aspect of React development. Understanding how to manage both local and global state efficiently can greatly enhance the performance and maintainability of your applications. Start with the basics using useState and useReducer, and then explore more advanced techniques like the Context API and Redux for larger applications.
