Introduction to Pipes

Pipes in Angular are a powerful feature that allows you to transform data directly in your templates. They can be used to format data, such as dates, numbers, and strings, making your templates cleaner and more readable.

Key Concepts

  • Pipe: A function that takes in data as input and transforms it to a desired output.
  • Built-in Pipes: Angular provides several built-in pipes for common data transformations.
  • Custom Pipes: You can create your own pipes for custom data transformations (covered in another section).

Common Built-in Pipes

Angular provides several built-in pipes that you can use out of the box:

  1. DatePipe: Formats a date value according to locale rules.
  2. UpperCasePipe: Transforms text to uppercase.
  3. LowerCasePipe: Transforms text to lowercase.
  4. CurrencyPipe: Transforms a number to a currency string.
  5. DecimalPipe: Transforms a number into a string with a specified decimal format.
  6. PercentPipe: Transforms a number to a percentage string.
  7. JsonPipe: Converts a value into a JSON string.
  8. SlicePipe: Creates a new array or string containing a subset (slice) of the elements.

Using Built-in Pipes

DatePipe

The DatePipe is used to format date values.

<p>{{ today | date }}</p>
<p>{{ today | date:'short' }}</p>
<p>{{ today | date:'fullDate' }}</p>
<p>{{ today | date:'yyyy-MM-dd' }}</p>

UpperCasePipe and LowerCasePipe

These pipes transform text to uppercase or lowercase, respectively.

<p>{{ 'hello world' | uppercase }}</p>
<p>{{ 'HELLO WORLD' | lowercase }}</p>

CurrencyPipe

The CurrencyPipe formats a number as currency.

<p>{{ 1234.56 | currency }}</p>
<p>{{ 1234.56 | currency:'USD' }}</p>
<p>{{ 1234.56 | currency:'EUR':'symbol':'4.2-2' }}</p>

DecimalPipe

The DecimalPipe formats a number with a specified decimal format.

<p>{{ 1234.56 | number }}</p>
<p>{{ 1234.56 | number:'3.1-2' }}</p>

PercentPipe

The PercentPipe formats a number as a percentage.

<p>{{ 0.1234 | percent }}</p>
<p>{{ 0.1234 | percent:'2.2-2' }}</p>

JsonPipe

The JsonPipe converts an object into a JSON string.

<p>{{ { name: 'John', age: 30 } | json }}</p>

SlicePipe

The SlicePipe creates a new array or string containing a subset (slice) of the elements.

<p>{{ 'Hello World' | slice:0:5 }}</p>
<p>{{ [1, 2, 3, 4, 5] | slice:1:3 }}</p>

Examples and Exercises

Example 1: Formatting Dates

Create a component that displays the current date in various formats using the DatePipe.

import { Component } from '@angular/core';

@Component({
  selector: 'app-date-example',
  template: `
    <p>Default: {{ today | date }}</p>
    <p>Short: {{ today | date:'short' }}</p>
    <p>Full Date: {{ today | date:'fullDate' }}</p>
    <p>Custom: {{ today | date:'yyyy-MM-dd' }}</p>
  `
})
export class DateExampleComponent {
  today: number = Date.now();
}

Exercise 1: Using CurrencyPipe

Create a component that displays a product price in different currencies using the CurrencyPipe.

import { Component } from '@angular/core';

@Component({
  selector: 'app-currency-example',
  template: `
    <p>USD: {{ price | currency:'USD' }}</p>
    <p>EUR: {{ price | currency:'EUR' }}</p>
    <p>GBP: {{ price | currency:'GBP' }}</p>
  `
})
export class CurrencyExampleComponent {
  price: number = 1234.56;
}

Exercise 2: Transforming Text

Create a component that displays a string in uppercase and lowercase using UpperCasePipe and LowerCasePipe.

import { Component } from '@angular/core';

@Component({
  selector: 'app-text-transform-example',
  template: `
    <p>Original: {{ text }}</p>
    <p>Uppercase: {{ text | uppercase }}</p>
    <p>Lowercase: {{ text | lowercase }}</p>
  `
})
export class TextTransformExampleComponent {
  text: string = 'Hello Angular';
}

Conclusion

Built-in pipes in Angular provide a convenient way to format and transform data directly in your templates. By understanding and utilizing these pipes, you can make your templates cleaner and more readable. Practice using these pipes in different scenarios to become more comfortable with their syntax and capabilities.
