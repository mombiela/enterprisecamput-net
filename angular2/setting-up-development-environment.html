Setting Up the Development Environment
Your First Angular App ►

Setting up the development environment is the first crucial step in your journey to mastering Angular 2+. This section will guide you through the necessary tools and configurations to get started with Angular development.

Prerequisites

Before setting up Angular, ensure you have the following prerequisites installed on your system:

  • Node.js and npm: Angular requires Node.js and npm (Node Package Manager). You can download and install them from Node.js official website.

Checking Node.js and npm Installation

To verify if Node.js and npm are installed, open your terminal or command prompt and run the following commands:

node -v
npm -v

You should see the version numbers of Node.js and npm, respectively.

Installing Angular CLI

Angular CLI (Command Line Interface) is a powerful tool that simplifies the process of creating and managing Angular applications. To install Angular CLI globally on your system, use the following command:

npm install -g @angular/cli

Verifying Angular CLI Installation

After installation, verify that Angular CLI is installed correctly by running:

ng version

This command will display the version of Angular CLI and other related packages.

Creating a New Angular Project

With Angular CLI installed, you can now create a new Angular project. Use the following command to create a new project named my-angular-app:

ng new my-angular-app

Project Setup Options

During the project creation process, Angular CLI will prompt you to choose some configuration options:

  • Would you like to add Angular routing?: Choose Yes or No depending on whether you need routing in your application.
  • Which stylesheet format would you like to use?: Select your preferred stylesheet format (CSS, SCSS, SASS, LESS, Stylus).

Running the Angular Application

Navigate to your project directory and start the development server using the following commands:

cd my-angular-app
ng serve

Accessing the Application

Once the server is running, open your web browser and navigate to http://localhost:4200/. You should see the default Angular welcome page.

Development Tools and Extensions

To enhance your development experience, consider installing the following tools and extensions:

  • Visual Studio Code (VS Code): A popular code editor with excellent support for Angular.
  • Angular Language Service: An extension for VS Code that provides rich editing and IntelliSense for Angular templates.

Installing Angular Language Service in VS Code

  1. Open VS Code.
  2. Go to the Extensions view by clicking the Extensions icon in the Activity Bar on the side of the window.
  3. Search for "Angular Language Service" and click Install.

Summary

In this section, you have learned how to set up the development environment for Angular 2+. You installed Node.js and npm, set up Angular CLI, created a new Angular project, and ran the development server. Additionally, you explored some useful development tools and extensions to enhance your coding experience.

By ensuring your development environment is correctly set up, you are now ready to dive deeper into Angular and start building powerful web applications.
