Introduction

In this section, we will explore how to implement core features in Swift. This includes understanding the fundamental building blocks of the language, such as variables, constants, functions, and control flow. By the end of this section, you will be able to create basic Swift programs and understand how to structure your code effectively.

Variables and Constants

Variables and constants are the basic storage units in Swift. They allow you to store and manipulate data.

  • Variables: Use var to declare a variable that can change.
  • Constants: Use let to declare a constant that cannot change.

Example:

var myVariable = 42
myVariable = 50
print(myVariable) // Output: 50

let myConstant = 42
// myConstant = 50 // This will cause an error because myConstant is a constant
print(myConstant) // Output: 42

Data Types

Swift is a type-safe language, which means it encourages you to be clear about the types of values your code can work with.

  • Common Data Types:
    • Int: Integer numbers
    • Double and Float: Floating-point numbers
    • Bool: Boolean values
    • String: Textual data

Example:

let integer: Int = 10
let floatingPoint: Double = 10.5
let boolean: Bool = true
let text: String = "Hello, Swift!"

Functions

Functions are self-contained chunks of code that perform a specific task.

  • Defining a Function: Use the func keyword.
  • Calling a Function: Use the function name followed by parentheses.

Example:

func greet(name: String) -> String {
    return "Hello, \(name)!"
}

let greeting = greet(name: "Alice")
print(greeting) // Output: Hello, Alice!

Control Flow

Control flow statements allow you to control the execution of your code based on certain conditions.

  • If Statements: Execute code based on a condition.
  • Switch Statements: Choose from multiple possible values.
  • Loops: Repeat a block of code multiple times.

Example:

// If Statement
let condition = true
if condition {
    print("Condition is true")
} else {
    print("Condition is false")
}

// Switch Statement
let number = 3
switch number {
case 1:
    print("One")
case 2:
    print("Two")
case 3:
    print("Three")
default:
    print("Other number")
}

// For Loop
for i in 1...5 {
    print(i)
}

// While Loop
var count = 5
while count > 0 {
    print(count)
    count -= 1
}

Collections

Swift provides powerful collection types such as arrays, sets, and dictionaries to store groups of values.

  • Arrays: Ordered collections of values.
  • Sets: Unordered collections of unique values.
  • Dictionaries: Unordered collections of key-value pairs.

Example:

// Array
var fruits: [String] = ["Apple", "Banana", "Cherry"]
fruits.append("Date")
print(fruits) // Output: ["Apple", "Banana", "Cherry", "Date"]

// Set
var uniqueNumbers: Set<Int> = [1, 2, 3, 3, 4]
print(uniqueNumbers) // Output: [2, 3, 1, 4]

// Dictionary
var capitals: [String: String] = ["France": "Paris", "Japan": "Tokyo"]
capitals["Italy"] = "Rome"
print(capitals) // Output: ["France": "Paris", "Japan": "Tokyo", "Italy": "Rome"]

Conclusion

In this section, we covered the core features of Swift, including variables, constants, data types, functions, control flow, and collections. These building blocks are essential for writing any Swift program. Understanding and mastering these concepts will provide a strong foundation for more advanced topics in Swift programming.
