In this section, we will learn how to create a basic project for a REST API. This project will serve as a foundation to build and expand functionalities in future lessons. We will use Node.js and Express for this purpose.

Prerequisites

Before starting, make sure you have the following components installed:

  • Node.js (version 14 or higher)
  • npm (Node Package Manager)

Step 1: Setting Up the Development Environment

Installing Node.js and npm

If you don't have Node.js and npm installed yet, you can download and install them from nodejs.org.

Creating the Project Directory

  1. Open your terminal or command line.

  2. Create a new directory for your project and navigate into it:

    mkdir my-rest-api-project
cd my-rest-api-project

Initializing the Project with npm

  1. Initialize a new Node.js project:

    npm init -y

    This will create a package.json file with the default configuration.

Step 2: Installing Dependencies

For this project, we will use Express, a minimalist framework for Node.js that makes it easy to create REST APIs.

  1. Install Express:

    npm install express

Step 3: Creating the Basic Server

Project Structure

Organize your project as follows:

my-rest-api-project/
├── node_modules/
├── package.json
└── index.js

Server Configuration

  1. Create a file named index.js in the root directory of your project.

  2. Open index.js and add the following code to set up a basic server with Express:

    const express = require('express');
const app = express();
const port = 3000;

app.get('/', (req, res) => {
    res.send('Hello, world!');
});

app.listen(port, () => {
    console.log(`Server listening at http://localhost:${port}`);
});

Running the Server

  1. In your terminal, run the following command to start the server:

    node index.js

  2. Open your web browser and navigate to http://localhost:3000. You should see the message "Hello, world!".

Step 4: Creating Basic Routes

GET Routes

  1. Add a new GET route in index.js to handle requests to /api/greeting:

    app.get('/api/greeting', (req, res) => {
    res.json({ message: 'Hello from the API!' });
});

POST Routes

  1. To handle POST requests, we first need to configure Express to process JSON data. Add the following line before defining the routes:

    app.use(express.json());

  2. Add a new POST route in index.js to handle requests to /api/greeting:

    app.post('/api/greeting', (req, res) => {
    const name = req.body.name;
    res.json({ message: `Hello, ${name}!` });
});

Complete Code Example

The complete index.js file should look like this:

const express = require('express');
const app = express();
const port = 3000;

app.use(express.json());

app.get('/', (req, res) => {
    res.send('Hello, world!');
});

app.get('/api/greeting', (req, res) => {
    res.json({ message: 'Hello from the API!' });
});

app.post('/api/greeting', (req, res) => {
    const name = req.body.name;
    res.json({ message: `Hello, ${name}!` });
});

app.listen(port, () => {
    console.log(`Server listening at http://localhost:${port}`);
});

Conclusion

In this lesson, we have set up a basic REST API project using Node.js and Express. We have learned to:

  • Set up a development environment.
  • Create a basic server.
  • Define GET and POST routes.

This basic project will serve as a foundation to add more functionalities and explore advanced concepts in future lessons. Keep practicing and experimenting with new routes and functionalities!
