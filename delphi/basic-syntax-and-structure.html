Introduction to Delphi/Object Pascal

Delphi, also known as Object Pascal, is a high-level, compiled, strongly typed language that supports structured and object-oriented programming. It is known for its readability and ease of use, making it an excellent choice for both beginners and experienced programmers.

Key Concepts

  • Case Sensitivity: Delphi is not case-sensitive.
  • Comments: Single-line comments use //, and multi-line comments use {} or (* *).
  • Identifiers: Names for variables, types, functions, etc., which must start with a letter or underscore and can be followed by letters, digits, or underscores.
  • Reserved Words: Keywords that have a special meaning in Delphi and cannot be used as identifiers.

Basic Program Structure

A simple Delphi program consists of the following parts:

  1. Program Heading: Declares the program name.
  2. Uses Clause: Lists units that the program uses.
  3. Variable Declarations: Declares variables.
  4. Begin-End Block: Contains the main code.

Example

program HelloWorld;

uses
  SysUtils;

begin
  WriteLn('Hello, World!');
end.

Explanation

  • program HelloWorld;: Declares the program name.
  • uses SysUtils;: Includes the SysUtils unit, which provides various utility functions.
  • begin ... end.: The main block of code where the program execution starts and ends.
  • WriteLn('Hello, World!');: Outputs the text "Hello, World!" to the console.

Variables and Data Types

Variable Declaration

Variables in Delphi are declared using the var keyword followed by the variable name and type.

var
  age: Integer;
  name: String;

Common Data Types

| Data Type | Description | |-----------|---------------------------------------| | Integer | Whole numbers | | Real | Floating-point numbers | | Char | Single character | | String | Sequence of characters | | Boolean | True or False values |

Example

var
  age: Integer;
  price: Real;
  initial: Char;
  name: String;
  isStudent: Boolean;
begin
  age := 25;
  price := 19.99;
  initial := 'A';
  name := 'Alice';
  isStudent := True;
end.

Explanation

  • age := 25;: Assigns the value 25 to the variable age.
  • price := 19.99;: Assigns the value 19.99 to the variable price.
  • initial := 'A';: Assigns the character 'A' to the variable initial.
  • name := 'Alice';: Assigns the string "Alice" to the variable name.
  • isStudent := True;: Assigns the boolean value True to the variable isStudent.

Control Structures

Conditional Statements

Delphi supports if, if-else, and case statements for conditional execution.

Example: if-else

var
  age: Integer;
begin
  age := 18;
  if age >= 18 then
    WriteLn('Adult')
  else
    WriteLn('Minor');
end.

Looping Statements

Delphi supports for, while, and repeat-until loops.

Example: for loop

var
  i: Integer;
begin
  for i := 1 to 10 do
    WriteLn('Number: ', i);
end.

Explanation

  • for i := 1 to 10 do: Loops from 1 to 10.
  • WriteLn('Number: ', i);: Outputs the current value of i.

Procedures and Functions

Procedures

Procedures are blocks of code that perform a task but do not return a value.

procedure Greet;
begin
  WriteLn('Hello!');
end;

begin
  Greet;
end.

Functions

Functions are similar to procedures but return a value.

function Add(a, b: Integer): Integer;
begin
  Add := a + b;
end;

begin
  WriteLn('Sum: ', Add(5, 3));
end.

Explanation

  • procedure Greet;: Declares a procedure named Greet.
  • function Add(a, b: Integer): Integer;: Declares a function named Add that takes two integers and returns an integer.
  • Add := a + b;: Returns the sum of a and b.

Conclusion

Understanding the basic syntax and structure of Delphi/Object Pascal is crucial for writing effective programs. This includes knowing how to declare variables, use control structures, and define procedures and functions. With these fundamentals, you can start building more complex applications and explore advanced features of the language.
