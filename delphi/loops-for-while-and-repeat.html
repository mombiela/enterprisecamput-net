Loops: For, While, and Repeat
Case Statements ►

Introduction

Loops are fundamental constructs in programming that allow you to execute a block of code multiple times. In Delphi/Object Pascal, there are three primary types of loops: For, While, and Repeat. This section will guide you through each type, providing clear explanations, examples, and exercises to solidify your understanding.

For Loop

The For loop is used when you know in advance how many times you want to execute a statement or a block of statements.

Syntax

for counter := startValue to endValue do
begin
  // Statements to execute
end;

Example

program ForLoopExample;
var
  i: Integer;
begin
  for i := 1 to 5 do
  begin
    WriteLn('Iteration: ', i);
  end;
end.

In this example, the loop will print the numbers 1 through 5.

Key Concepts

  • Counter Variable: The variable that controls the loop.
  • Start and End Values: Define the range of the loop.
  • Step Value: Implicitly 1 in a For loop, but can be modified using the downto keyword for decrementing.

Exercises

  1. Write a For loop that prints the even numbers from 2 to 10.
  2. Modify the above loop to print numbers in reverse order using downto.

While Loop

The While loop is used when the number of iterations is not known beforehand. It continues to execute as long as a specified condition is true.

Syntax

while condition do
begin
  // Statements to execute
end;

Example

program WhileLoopExample;
var
  i: Integer;
begin
  i := 1;
  while i <= 5 do
  begin
    WriteLn('Iteration: ', i);
    i := i + 1;
  end;
end.

In this example, the loop will print the numbers 1 through 5.

Key Concepts

  • Condition: The loop continues as long as this condition is true.
  • Initialization and Update: Ensure the loop variable is properly initialized and updated to avoid infinite loops.

Exercises

  1. Write a While loop that prints the numbers from 10 to 1.
  2. Modify the above loop to print only odd numbers.

Repeat Loop

The Repeat loop is similar to the While loop but guarantees that the loop body is executed at least once.

Syntax

repeat
  // Statements to execute
until condition;

Example

program RepeatLoopExample;
var
  i: Integer;
begin
  i := 1;
  repeat
    WriteLn('Iteration: ', i);
    i := i + 1;
  until i > 5;
end.

In this example, the loop will print the numbers 1 through 5.

Key Concepts

  • Condition: The loop continues until this condition becomes true.
  • Guaranteed Execution: The loop body is executed at least once, regardless of the condition.

Exercises

  1. Write a Repeat loop that prints the numbers from 1 to 10.
  2. Modify the above loop to stop when a random number between 1 and 10 equals 7.

Comparison of Loops

| Feature | For Loop | While Loop | Repeat Loop | |---------------|--------------------------------|---------------------------------|---------------------------------| | Initialization| Implicit in the loop header | Must be done before the loop | Must be done before the loop | | Condition | Defined by start and end values| Evaluated before each iteration | Evaluated after each iteration | | Execution | Executes zero or more times | Executes zero or more times | Executes one or more times |

Conclusion

Understanding loops is crucial for effective programming. The For loop is ideal for a known number of iterations, the While loop is useful for conditions evaluated before the loop, and the Repeat loop ensures the loop body is executed at least once. Practice using these loops with the provided exercises to deepen your understanding and become proficient in Delphi/Object Pascal.
