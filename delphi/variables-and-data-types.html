Introduction

In this section, we will explore the fundamental concepts of variables and data types in Delphi/Object Pascal. Understanding these concepts is crucial for writing efficient and error-free code. We will cover the basics of declaring variables, different data types available in Delphi, and how to use them effectively.

Variables

Variables are used to store data that can be manipulated by the program. In Delphi, variables must be declared before they can be used.

Declaring Variables

To declare a variable in Delphi, you use the var keyword followed by the variable name and its type.

var
  age: Integer;
  name: String;
  isStudent: Boolean;
  • age is an integer variable.
  • name is a string variable.
  • isStudent is a boolean variable.

Variable Naming Rules

  • Must start with a letter or underscore.
  • Can contain letters, digits, and underscores.
  • Cannot be a reserved keyword.

Assigning Values

Values can be assigned to variables using the assignment operator (:=).

age := 25;
name := 'John Doe';
isStudent := True;

Data Types

Data types define the kind of data a variable can hold. Delphi provides several built-in data types.

Integer Types

Used to store whole numbers.

| Type | Range | Size (bytes) | |----------|------------------------------|--------------| | Byte | 0 to 255 | 1 | | ShortInt | -128 to 127 | 1 | | Integer | -2,147,483,648 to 2,147,483,647 | 4 | | LongInt | -2,147,483,648 to 2,147,483,647 | 4 | | Int64 | -9,223,372,036,854,775,808 to 9,223,372,036,854,775,807 | 8 |

Floating-Point Types

Used to store numbers with fractional parts.

| Type | Range | Size (bytes) | |----------|------------------------------|--------------| | Single | 1.5 x 10^-45 to 3.4 x 10^38 | 4 | | Double | 5.0 x 10^-324 to 1.7 x 10^308 | 8 | | Extended | 3.6 x 10^-4951 to 1.1 x 10^4932 | 10 |

Boolean Type

Used to store True or False.

var
  isActive: Boolean;
begin
  isActive := True;
end;

Character Types

Used to store single characters or strings.

| Type | Description | |--------|-------------------------------| | Char | Single character (1 byte) | | String | Sequence of characters |

var
  initial: Char;
  fullName: String;
begin
  initial := 'A';
  fullName := 'Alice Johnson';
end;

Enumerated Types

User-defined data types that consist of a set of named values.

type
  TDays = (Sun, Mon, Tue, Wed, Thu, Fri, Sat);
var
  today: TDays;
begin
  today := Mon;
end;

Subrange Types

Define a subset of an existing ordinal type.

type
  TAge = 0..120;
var
  myAge: TAge;
begin
  myAge := 30;
end;

Advanced Topics

Records

Records are used to group different types of data together.

type
  TPerson = record
    Name: String;
    Age: Integer;
    IsStudent: Boolean;
  end;
var
  person: TPerson;
begin
  person.Name := 'John Doe';
  person.Age := 25;
  person.IsStudent := True;
end;

Arrays

Arrays are used to store multiple values of the same type.

var
  numbers: array[1..5] of Integer;
begin
  numbers[1] := 10;
  numbers[2] := 20;
  numbers[3] := 30;
  numbers[4] := 40;
  numbers[5] := 50;
end;

Dynamic Arrays

Dynamic arrays can change size at runtime.

var
  dynArray: array of Integer;
begin
  SetLength(dynArray, 3);
  dynArray[0] := 1;
  dynArray[1] := 2;
  dynArray[2] := 3;
end;

Conclusion

Understanding variables and data types is fundamental to programming in Delphi/Object Pascal. This knowledge allows you to store and manipulate data effectively. As you progress, you'll encounter more complex data structures and learn how to use them to solve real-world problems. Practice declaring variables, assigning values, and using different data types to become proficient in Delphi programming.
