Conditional Statements in Delphi/Object Pascal
Loops: For, While, and Repeat ►

Conditional statements are fundamental in programming as they allow the execution of code based on certain conditions. In Delphi/Object Pascal, conditional statements include if, case, and else. This topic will cover these statements from a beginner to an advanced level.

If Statements

The if statement is used to execute a block of code if a specified condition is true.

Basic If Statement

The simplest form of an if statement checks a single condition.

var
  number: Integer;
begin
  number := 10;
  if number > 5 then
    WriteLn('Number is greater than 5');
end.

If-Else Statement

The if-else statement provides an alternative block of code to execute if the condition is false.

var
  number: Integer;
begin
  number := 3;
  if number > 5 then
    WriteLn('Number is greater than 5')
  else
    WriteLn('Number is 5 or less');
end.

Nested If Statements

You can nest if statements to check multiple conditions.

var
  number: Integer;
begin
  number := 7;
  if number > 5 then
  begin
    if number < 10 then
      WriteLn('Number is between 6 and 9');
  end
  else
    WriteLn('Number is 5 or less');
end.

Case Statements

The case statement is used to execute one of several blocks of code based on the value of an expression.

Basic Case Statement

A simple case statement checks the value of a single variable.

var
  grade: Char;
begin
  grade := 'B';
  case grade of
    'A': WriteLn('Excellent');
    'B': WriteLn('Good');
    'C': WriteLn('Average');
    'D': WriteLn('Below Average');
    'F': WriteLn('Fail');
  else
    WriteLn('Invalid grade');
  end;
end.

Case Statement with Ranges

The case statement can also handle ranges of values.

var
  score: Integer;
begin
  score := 85;
  case score of
    90..100: WriteLn('Grade A');
    80..89: WriteLn('Grade B');
    70..79: WriteLn('Grade C');
    60..69: WriteLn('Grade D');
    0..59: WriteLn('Grade F');
  else
    WriteLn('Invalid score');
  end;
end.

Advanced Conditional Statements

Combining Conditions with Logical Operators

You can combine multiple conditions using logical operators such as and, or, and not.

var
  age: Integer;
  hasLicense: Boolean;
begin
  age := 18;
  hasLicense := True;
  if (age >= 18) and hasLicense then
    WriteLn('You can drive')
  else
    WriteLn('You cannot drive');
end.

Using Functions in Conditions

You can call functions within conditional statements to make decisions based on more complex logic.

function IsEven(number: Integer): Boolean;
begin
  Result := (number mod 2) = 0;
end;

var
  number: Integer;
begin
  number := 4;
  if IsEven(number) then
    WriteLn('Number is even')
  else
    WriteLn('Number is odd');
end.

Conclusion

Conditional statements are essential for controlling the flow of a program. Delphi/Object Pascal provides powerful constructs like if, else, and case to handle various conditions. By mastering these statements, you can write more dynamic and responsive programs. Practice using these constructs with different scenarios to become proficient in their use.
