Introduction to Arrays

Arrays are a fundamental data structure in Delphi/Object Pascal, allowing you to store multiple values of the same type in a single variable. This section will cover the basics of arrays, including their declaration, initialization, and usage.

Key Concepts

  • Array Declaration: How to declare arrays in Delphi/Object Pascal.
  • Array Initialization: How to initialize arrays with values.
  • Accessing Array Elements: How to read and write array elements.
  • Static vs. Dynamic Arrays: Understanding the difference between static and dynamic arrays.

Declaring Arrays

In Delphi/Object Pascal, arrays can be declared in various ways depending on their type and size.

Static Arrays

Static arrays have a fixed size that is determined at compile time.

var
  Numbers: array[1..5] of Integer;

In this example:

  • Numbers is an array that can hold 5 integers.
  • The index ranges from 1 to 5.

Dynamic Arrays

Dynamic arrays can change size at runtime.

var
  Numbers: array of Integer;

In this example:

  • Numbers is a dynamic array of integers.
  • The size of the array is not specified at declaration.

Initializing Arrays

After declaring an array, you can initialize it with values.

Static Arrays

Static arrays can be initialized element by element or using a loop.

var
  Numbers: array[1..5] of Integer;
begin
  Numbers[1] := 10;
  Numbers[2] := 20;
  Numbers[3] := 30;
  Numbers[4] := 40;
  Numbers[5] := 50;
end;

Dynamic Arrays

Dynamic arrays need to be resized before initialization.

var
  Numbers: array of Integer;
begin
  SetLength(Numbers, 5); // Resize the array to hold 5 elements
  Numbers[0] := 10;
  Numbers[1] := 20;
  Numbers[2] := 30;
  Numbers[3] := 40;
  Numbers[4] := 50;
end;

Accessing Array Elements

You can access and modify array elements using their index.

var
  Numbers: array[1..5] of Integer;
begin
  Numbers[1] := 10;
  WriteLn(Numbers[1]); // Outputs: 10
end;

Static vs. Dynamic Arrays

Understanding the differences between static and dynamic arrays is crucial for efficient memory management and performance.

| Feature | Static Arrays | Dynamic Arrays | |-----------------|---------------------------------|---------------------------------| | Size | Fixed at compile time | Can be resized at runtime | | Memory Allocation | Allocated on the stack | Allocated on the heap | | Flexibility | Less flexible | More flexible | | Performance | Faster access | Slower access due to heap allocation |

Advanced Array Operations

As you become more proficient, you can perform more complex operations with arrays.

Multidimensional Arrays

Delphi/Object Pascal supports multidimensional arrays.

var
  Matrix: array[1..3, 1..3] of Integer;
begin
  Matrix[1, 1] := 1;
  Matrix[1, 2] := 2;
  Matrix[1, 3] := 3;
  // Continue initializing other elements
end;

Dynamic Multidimensional Arrays

Dynamic multidimensional arrays can also be created and resized.

var
  Matrix: array of array of Integer;
begin
  SetLength(Matrix, 3, 3); // Create a 3x3 matrix
  Matrix[0][0] := 1;
  Matrix[0][1] := 2;
  Matrix[0][2] := 3;
  // Continue initializing other elements
end;

Array of Records

You can create arrays of complex types like records.

type
  TPerson = record
    Name: string;
    Age: Integer;
  end;

var
  People: array[1..3] of TPerson;
begin
  People[1].Name := 'Alice';
  People[1].Age := 30;
  // Continue initializing other elements
end;

Conclusion

Arrays are a powerful and versatile data structure in Delphi/Object Pascal. Understanding how to declare, initialize, and manipulate arrays is essential for efficient programming. Whether you are working with static or dynamic arrays, mastering these concepts will allow you to handle collections of data effectively. As you advance, you can explore more complex array operations, such as multidimensional arrays and arrays of records, to further enhance your programming skills.
