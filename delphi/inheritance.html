Inheritance in Delphi/Object Pascal
Polymorphism ►

Inheritance is a fundamental concept in Object-Oriented Programming (OOP) that allows a class to inherit properties and methods from another class. This helps in code reusability and establishing a hierarchical relationship between classes.

Introduction to Inheritance

  • Definition: Inheritance allows a new class (derived class) to inherit attributes and methods from an existing class (base class).
  • Purpose: Promotes code reuse, reduces redundancy, and establishes a natural hierarchy.

Basic Concepts

Base Class and Derived Class

  • Base Class: The class whose properties and methods are inherited.
  • Derived Class: The class that inherits from the base class.

Syntax

In Delphi/Object Pascal, inheritance is implemented using the class keyword followed by the TObject base class or any other user-defined class.

type
  TBaseClass = class
    procedure ShowMessage; virtual;
  end;

  TDerivedClass = class(TBaseClass)
    procedure ShowMessage; override;
  end;

Virtual and Override Methods

  • Virtual Method: A method in the base class that can be overridden in a derived class.
  • Override Method: A method in the derived class that overrides a virtual method in the base class.
procedure TBaseClass.ShowMessage;
begin
  Writeln('Message from Base Class');
end;

procedure TDerivedClass.ShowMessage;
begin
  Writeln('Message from Derived Class');
end;

Implementing Inheritance

Creating a Base Class

Define a base class with some properties and methods.

type
  TAnimal = class
  public
    procedure Speak; virtual;
  end;

procedure TAnimal.Speak;
begin
  Writeln('The animal makes a sound');
end;

Creating a Derived Class

Define a derived class that inherits from the base class and overrides its methods.

type
  TDog = class(TAnimal)
  public
    procedure Speak; override;
  end;

procedure TDog.Speak;
begin
  Writeln('The dog barks');
end;

Using the Classes

Instantiate the classes and call their methods to see inheritance in action.

var
  Animal: TAnimal;
  Dog: TDog;
begin
  Animal := TAnimal.Create;
  Dog := TDog.Create;

  Animal.Speak; // Output: The animal makes a sound
  Dog.Speak;    // Output: The dog barks

  Animal.Free;
  Dog.Free;
end;

Advanced Concepts

Abstract Classes and Methods

  • Abstract Class: A class that cannot be instantiated and is meant to be a base class for other classes.
  • Abstract Method: A method that must be overridden in a derived class.
type
  TShape = class
  public
    procedure Draw; virtual; abstract;
  end;

  TCircle = class(TShape)
  public
    procedure Draw; override;
  end;

procedure TCircle.Draw;
begin
  Writeln('Drawing a circle');
end;

Inheritance and Constructors

Constructors in derived classes should call the constructor of the base class to ensure proper initialization.

type
  TPerson = class
  private
    FName: string;
  public
    constructor Create(AName: string); virtual;
  end;

  TEmployee = class(TPerson)
  private
    FEmployeeID: Integer;
  public
    constructor Create(AName: string; AEmployeeID: Integer); reintroduce;
  end;

constructor TPerson.Create(AName: string);
begin
  FName := AName;
end;

constructor TEmployee.Create(AName: string; AEmployeeID: Integer);
begin
  inherited Create(AName);
  FEmployeeID := AEmployeeID;
end;

Conclusion

Inheritance in Delphi/Object Pascal is a powerful feature that promotes code reuse and establishes a clear hierarchy between classes. By understanding and utilizing inheritance, you can create more organized and maintainable code. Remember to use virtual and override methods to customize behavior in derived classes and consider abstract classes and methods for more advanced designs.
