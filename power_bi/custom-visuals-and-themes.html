Introduction

In this section, we will explore how to enhance your Power BI reports by using custom visuals and themes. Custom visuals allow you to create unique and interactive data visualizations, while themes enable you to apply consistent styling across your reports.

Custom Visuals

What are Custom Visuals?

Custom visuals are additional visualization types that you can import into Power BI to extend its default capabilities. These visuals can be created by the Power BI community or by developers using the Power BI Visuals SDK.

Importing Custom Visuals

To import custom visuals into your Power BI report:

  1. Go to the Visualizations pane.
  2. Click on the ellipsis (...) and select "Import from marketplace" or "Import from file".
  3. Choose the desired custom visual and add it to your report.

Example: Using a Custom Visual

Let's import and use a custom visual called "Word Cloud".

  1. Import the Word Cloud Visual:

    - Go to the Visualizations pane.
- Click on the ellipsis (`...`) and select "Import from marketplace".
- Search for "Word Cloud" and click "Add".

  2. Add Data to the Word Cloud:

    - Drag a text field (e.g., Product Names) to the Word Cloud visual.
- Adjust the settings to customize the appearance.

Creating Custom Visuals

For advanced users, creating custom visuals involves:

  • Setting up a development environment with Node.js and the Power BI Visuals SDK.
  • Writing TypeScript code to define the visual's behavior.
  • Packaging the visual and importing it into Power BI.

Example: Basic Custom Visual Code

Here is a simple example of a custom visual using TypeScript:

module powerbi.extensibility.visual {
    export class SimpleVisual implements IVisual {
        private target: HTMLElement;

        constructor(options: VisualConstructorOptions) {
            this.target = options.element;
        }

        public update(options: VisualUpdateOptions) {
            this.target.innerHTML = `<div style="color:blue;">Hello, Power BI!</div>`;
        }
    }
}
  • Explanation:
    • SimpleVisual is a class implementing the IVisual interface.
    • The constructor initializes the visual.
    • The update method updates the visual with new data or settings.

Themes

What are Themes?

Themes in Power BI allow you to apply a consistent look and feel to your reports by defining colors, fonts, and other visual elements.

Applying a Theme

To apply a theme to your Power BI report:

  1. Go to the Home tab.
  2. Click on "Switch Theme".
  3. Choose from the available themes or import a custom theme file.

Creating Custom Themes

Custom themes are defined using JSON files. Here is an example of a simple custom theme:

{
  "name": "Custom Theme",
  "dataColors": ["#FF5733", "#33FF57", "#3357FF"],
  "background": "#FFFFFF",
  "foreground": "#000000",
  "tableAccent": "#FF5733"
}
  • Explanation:
    • dataColors: Defines the colors used in charts.
    • background: Sets the background color of the report.
    • foreground: Sets the text color.
    • tableAccent: Defines the accent color for tables.

Example: Applying a Custom Theme

  1. Create a JSON file with the theme definition:

    {
  "name": "My Custom Theme",
  "dataColors": ["#FF5733", "#33FF57", "#3357FF"],
  "background": "#F0F0F0",
  "foreground": "#333333",
  "tableAccent": "#FF5733"
}

  2. Import the Custom Theme:

    - Go to the Home tab.
- Click on "Switch Theme" and select "Import Theme".
- Choose the JSON file you created.

Conclusion

Custom visuals and themes are powerful tools in Power BI that allow you to create more engaging and visually appealing reports. By importing or creating custom visuals, you can extend the default visualization capabilities of Power BI. Themes help ensure a consistent and professional look across your reports. Experiment with these features to enhance your Power BI projects and deliver more impactful insights.
