Publishing and Sharing Reports
Introduction to Power BI Service ►

Introduction

In this section, we will explore how to publish and share reports in Power BI. This is an essential skill for making your data insights accessible to others, whether they are team members, stakeholders, or the general public.

Publishing Reports

Steps to Publish a Report

  1. Prepare Your Report: Ensure your report is complete and all data visualizations are accurate.
  2. Save Your Report: Save your Power BI Desktop file (.pbix) locally.
  3. Sign In to Power BI Service: Open Power BI Desktop and sign in to your Power BI account.
  4. Publish: Click on the 'Publish' button in the Home tab.
Home -> Publish -> Select Workspace -> Publish

Selecting a Workspace

  • My Workspace: Personal workspace for individual use.
  • Shared Workspaces: Collaborative spaces for team projects.

Example

1. Click 'Publish' in Power BI Desktop.
2. Choose 'My Workspace' or a specific shared workspace.
3. Click 'Select' to publish the report.

Sharing Reports

Sharing Options

  1. Direct Sharing: Share reports directly with specific users.
  2. Embed in SharePoint Online: Embed reports in SharePoint for easy access.
  3. Publish to Web: Make reports publicly accessible via a URL.

Direct Sharing

  • Steps:
    1. Open the report in Power BI Service.
    2. Click on the 'Share' button.
    3. Enter the email addresses of the recipients.
    4. Set permissions (view or edit).
    5. Click 'Send'.
Report -> Share -> Enter Emails -> Set Permissions -> Send

Example

1. Open the report in Power BI Service.
2. Click 'Share'.
3. Enter 'john.doe@example.com'.
4. Set permission to 'Can view'.
5. Click 'Send'.

Embed in SharePoint Online

  • Steps:
    1. Open the report in Power BI Service.
    2. Click on 'File' -> 'Embed in SharePoint Online'.
    3. Copy the embed code.
    4. Paste the embed code into a SharePoint Online page.
File -> Embed in SharePoint Online -> Copy Embed Code -> Paste in SharePoint

Publish to Web

  • Steps:
    1. Open the report in Power BI Service.
    2. Click on 'File' -> 'Publish to web'.
    3. Generate the embed code or URL.
    4. Share the URL or embed code as needed.
File -> Publish to web -> Generate Embed Code/URL -> Share

Managing Permissions

Setting Permissions

  • View Only: Users can view the report but cannot make changes.
  • Edit: Users can view and make changes to the report.

Example

1. Open the report in Power BI Service.
2. Click 'Share'.
3. Enter 'jane.doe@example.com'.
4. Set permission to 'Can edit'.
5. Click 'Send'.

Conclusion

Publishing and sharing reports in Power BI is a straightforward process that enhances collaboration and data accessibility. By following the steps outlined above, you can ensure your reports reach the right audience with the appropriate permissions. Whether you are sharing directly, embedding in SharePoint, or publishing to the web, Power BI provides flexible options to suit your needs.
