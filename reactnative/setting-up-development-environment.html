Setting Up the Development Environment
Hello World: Your First React Native App ►

Setting up the development environment is the first crucial step in your journey to becoming proficient in React Native. This section will guide you through the process of setting up your environment on different operating systems, ensuring you have all the necessary tools to start building React Native applications.

Prerequisites

Before diving into the setup, ensure you have the following prerequisites:

  • Basic knowledge of JavaScript.
  • A code editor (Visual Studio Code is recommended).
  • Node.js installed on your machine.

Installing Node.js and npm

Node.js is a JavaScript runtime that allows you to run JavaScript on the server side. npm (Node Package Manager) comes with Node.js and helps you manage dependencies.

  1. Download and Install Node.js:

    • Go to the Node.js website.
    • Download the LTS version for your operating system.
    • Follow the installation instructions.

  2. Verify Installation:

    node -v
npm -v

Setting Up React Native CLI

React Native CLI is a command-line interface that helps you create, build, and run React Native projects.

  1. Install React Native CLI:

    npm install -g react-native-cli

  2. Verify Installation:

    react-native --version

Setting Up the Development Environment on macOS

Installing Xcode

Xcode is required for iOS development.

  1. Download Xcode:

    • Open the App Store on your Mac.
    • Search for Xcode and install it.

  2. Install Command Line Tools:

    xcode-select --install

Installing Android Studio

Android Studio is required for Android development.

  1. Download Android Studio:

    • Go to the Android Studio website.
    • Download and install Android Studio.

  2. Set Up Android SDK:

    • Open Android Studio.
    • Go to Preferences > Appearance & Behavior > System Settings > Android SDK.
    • Select the SDK Platforms tab and ensure Android 10.0 (Q) is checked.
    • Select the SDK Tools tab and ensure Android SDK Build-Tools, Android Emulator, and Android SDK Platform-Tools are checked.

Setting Up Environment Variables

  1. Open Terminal and Edit Profile:

    nano ~/.bash_profile

  2. Add the Following Lines:

    export ANDROID_HOME=$HOME/Library/Android/sdk
export PATH=$PATH:$ANDROID_HOME/emulator
export PATH=$PATH:$ANDROID_HOME/tools
export PATH=$PATH:$ANDROID_HOME/tools/bin
export PATH=$PATH:$ANDROID_HOME/platform-tools

  3. Save and Close:

    • Press CTRL + X, then Y, and Enter.

  4. Apply Changes:

    source ~/.bash_profile

Setting Up the Development Environment on Windows

Installing Android Studio

  1. Download Android Studio:

    • Go to the Android Studio website.
    • Download and install Android Studio.

  2. Set Up Android SDK:

    • Open Android Studio.
    • Go to File > Settings > Appearance & Behavior > System Settings > Android SDK.
    • Select the SDK Platforms tab and ensure Android 10.0 (Q) is checked.
    • Select the SDK Tools tab and ensure Android SDK Build-Tools, Android Emulator, and Android SDK Platform-Tools are checked.

Setting Up Environment Variables

  1. Open Environment Variables:

    • Right-click on This PC > Properties > Advanced system settings.
    • Click on Environment Variables.

  2. Add New System Variables:

    • Click New under System variables.
    • Add ANDROID_HOME with the path to your Android SDK (e.g., C:\Users\<Your-Username>\AppData\Local\Android\Sdk).
    • Add PATH entries for platform-tools and tools.

Creating Your First React Native Project

  1. Create a New Project:

    npx react-native init MyFirstProject

  2. Navigate to Project Directory:

    cd MyFirstProject

  3. Run the Project:

    • For iOS: 
      npx react-native run-ios
    • For Android: 
      npx react-native run-android

Conclusion

Setting up the development environment for React Native might seem daunting at first, but following these steps will ensure you have everything you need to start building your applications. Whether you're on macOS or Windows, having Node.js, React Native CLI, and the necessary tools for iOS and Android development will set you up for success. Happy coding!
