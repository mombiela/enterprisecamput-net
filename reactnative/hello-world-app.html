Introduction

In this section, we will guide you through creating your first React Native app. This will be a simple "Hello World" application, which is a traditional starting point for learning any new programming language or framework. By the end of this section, you will have a basic understanding of how to set up a React Native environment and create a simple app.

Setting Up the Environment

Before we start coding, we need to set up our development environment.

Prerequisites

  • Node.js and npm (Node Package Manager)
  • React Native CLI or Expo CLI
  • A code editor (e.g., Visual Studio Code)
  • Android Studio or Xcode (for running the app on an emulator)

Installation Steps

  1. Install Node.js and npm: Download and install Node.js from nodejs.org. npm is included with Node.js.

  2. Install React Native CLI: Open your terminal and run:

    npm install -g react-native-cli

  3. Install Expo CLI (Optional): If you prefer using Expo, run:

    npm install -g expo-cli

  4. Set Up Android Studio or Xcode:

    • For Android: Download and install Android Studio from developer.android.com.
    • For iOS: Ensure you have Xcode installed from the Mac App Store.

Creating Your First React Native App

Now that our environment is set up, let's create our first app.

Using React Native CLI

  1. Create a New Project:

    npx react-native init HelloWorldApp

    This command creates a new React Native project named HelloWorldApp.

  2. Navigate to the Project Directory:

    cd HelloWorldApp

  3. Run the App:

    • For Android: 
      npx react-native run-android
    • For iOS: 
      npx react-native run-ios

Using Expo CLI

  1. Create a New Project:

    expo init HelloWorldApp

    Choose a template (e.g., blank) and follow the prompts.

  2. Navigate to the Project Directory:

    cd HelloWorldApp

  3. Run the App:

    expo start

    This will open a new tab in your browser with a QR code. Scan the QR code with the Expo Go app on your mobile device to run the app.

Writing the "Hello World" Code

Now, let's modify the default code to display "Hello World".

Editing the App Component

Open App.js in your code editor and replace its content with the following code:

import React from 'react';
import { Text, View, StyleSheet } from 'react-native';

const App = () => {
  return (
    <View style={styles.container}>
      <Text style={styles.text}>Hello World</Text>
    </View>
  );
};

const styles = StyleSheet.create({
  container: {
    flex: 1,
    justifyContent: 'center',
    alignItems: 'center',
    backgroundColor: '#F5FCFF',
  },
  text: {
    fontSize: 20,
    textAlign: 'center',
    margin: 10,
  },
});

export default App;

Explanation

  • Imports: We import React, Text, View, and StyleSheet from react-native.
  • App Component: This is a functional component that returns a View containing a Text element.
  • Styles: We define some basic styles using StyleSheet.create.

Running the App

After saving your changes, your app should automatically reload and display "Hello World" on the screen.

Conclusion

Congratulations! You have successfully created your first React Native app. In this section, you learned how to:

  • Set up your development environment.
  • Create a new React Native project using both React Native CLI and Expo CLI.
  • Write and run a simple "Hello World" application.

This foundational knowledge will be crucial as we move on to more advanced topics in React Native development.
