Introduction to Tables

Tables are a powerful data type in MATLAB that allow you to store data of different types and sizes in a single container. They are particularly useful for organizing and analyzing data sets that include both numeric and non-numeric data.

Key Concepts

  • Table Creation: How to create tables from arrays, cell arrays, or by importing data.
  • Table Properties: Understanding table dimensions, variable names, and row names.
  • Data Access: Methods to access and modify data within tables.
  • Table Operations: Common operations such as sorting, filtering, and joining tables.

Creating Tables

You can create tables in MATLAB using the table function. Here are some basic ways to create tables:

From Arrays

% Create arrays
Age = [25; 30; 35];
Height = [5.5; 6.0; 5.8];
Weight = [150; 180; 165];

% Create table
T = table(Age, Height, Weight);
disp(T);

This code creates a table T with three columns: Age, Height, and Weight.

From Cell Arrays

% Create cell arrays
Names = {'John'; 'Jane'; 'Jim'};
Scores = {85, 90, 88};

% Create table
T = table(Names, Scores);
disp(T);

This code creates a table T with two columns: Names and Scores.

Importing Data

You can also create tables by importing data from files such as CSV, Excel, or text files.

% Import data from a CSV file
T = readtable('data.csv');
disp(T);

Table Properties

Tables have properties that define their structure and metadata.

Dimensions

You can get the size of a table using the size function.

% Get table dimensions
[rows, cols] = size(T);
disp(['Rows: ', num2str(rows), ', Columns: ', num2str(cols)]);

Variable Names

Variable names can be accessed and modified using the Properties.VariableNames property.

% Get variable names
varNames = T.Properties.VariableNames;
disp(varNames);

% Set new variable names
T.Properties.VariableNames = {'Age', 'Height_in', 'Weight_lb'};
disp(T);

Row Names

Similarly, row names can be accessed and modified.

% Set row names
T.Properties.RowNames = {'Person1', 'Person2', 'Person3'};
disp(T);

Accessing and Modifying Data

You can access and modify table data using various indexing methods.

By Variable Name

% Access data by variable name
heights = T.Height_in;
disp(heights);

% Modify data by variable name
T.Height_in = T.Height_in * 2.54; % Convert inches to cm
disp(T);

By Row and Variable Name

% Access specific element
height_person1 = T{'Person1', 'Height_in'};
disp(height_person1);

% Modify specific element
T{'Person1', 'Height_in'} = 170;
disp(T);

Table Operations

MATLAB provides several functions to perform operations on tables.

Sorting

% Sort table by Age
T = sortrows(T, 'Age');
disp(T);

Filtering

% Filter rows where Age is greater than 30
T_filtered = T(T.Age > 30, :);
disp(T_filtered);

Joining Tables

You can join two tables using the join function.

% Create another table
Gender = {'M'; 'F'; 'M'};
T2 = table(Gender, 'RowNames', {'Person1', 'Person2', 'Person3'});

% Join tables
T_joined = join(T, T2);
disp(T_joined);

Conclusion

Tables in MATLAB are versatile and powerful for managing heterogeneous data. By understanding how to create, manipulate, and perform operations on tables, you can effectively organize and analyze complex data sets. Practice creating tables from different data sources and performing various operations to become proficient in using tables in MATLAB.
