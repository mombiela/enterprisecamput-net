Introduction

In this section, we will cover the fundamental operations and functions in MATLAB. This includes basic arithmetic operations, matrix manipulations, and the creation and use of functions. By the end of this section, you will be able to perform essential calculations and write simple functions in MATLAB.

Basic Arithmetic Operations

MATLAB is designed for numerical computations, and understanding basic arithmetic operations is crucial.

  • Addition (+)
  • Subtraction (-)
  • Multiplication (*)
  • Division (/ and \)
  • Exponentiation (^)

Examples

a = 5;
b = 3;

% Addition
c = a + b; % c = 8

% Subtraction
d = a - b; % d = 2

% Multiplication
e = a * b; % e = 15

% Division
f = a / b; % f ≈ 1.6667
g = a \ b; % g = 0.6 (equivalent to b / a)

% Exponentiation
h = a ^ b; % h = 125

Matrix Operations

MATLAB excels at matrix operations, which are fundamental in many scientific and engineering applications.

  • Matrix Addition and Subtraction
  • Matrix Multiplication
  • Element-wise Operations
  • Transpose

Examples

A = [1, 2; 3, 4];
B = [5, 6; 7, 8];

% Matrix Addition
C = A + B; % C = [6, 8; 10, 12]

% Matrix Subtraction
D = A - B; % D = [-4, -4; -4, -4]

% Matrix Multiplication
E = A * B; % E = [19, 22; 43, 50]

% Element-wise Multiplication
F = A .* B; % F = [5, 12; 21, 32]

% Transpose
G = A'; % G = [1, 3; 2, 4]

Built-in Functions

MATLAB provides a vast library of built-in functions for various tasks, including mathematical computations, data analysis, and visualization.

  • Mathematical Functions: sum, mean, max, min, sqrt, abs
  • Trigonometric Functions: sin, cos, tan, asin, acos, atan
  • Statistical Functions: median, std, var

Examples

% Mathematical Functions
x = [1, 2, 3, 4, 5];

total = sum(x); % total = 15
average = mean(x); % average = 3
maximum = max(x); % maximum = 5
minimum = min(x); % minimum = 1
square_root = sqrt(x); % square_root = [1, 1.4142, 1.7321, 2, 2.2361]
absolute = abs([-1, -2, -3]); % absolute = [1, 2, 3]

% Trigonometric Functions
angle = pi / 4; % 45 degrees

sine = sin(angle); % sine ≈ 0.7071
cosine = cos(angle); % cosine ≈ 0.7071
tangent = tan(angle); % tangent = 1

Creating Functions

Creating custom functions in MATLAB allows you to encapsulate code for reuse and better organization.

Syntax

function [output1, output2, ...] = functionName(input1, input2, ...)
    % Function description
    % Detailed explanation goes here

    % Function code
    output1 = ...;
    output2 = ...;
end

Example

function [area, circumference] = circleProperties(radius)
    % circleProperties calculates the area and circumference of a circle
    % given its radius.

    % Area of the circle
    area = pi * radius^2;

    % Circumference of the circle
    circumference = 2 * pi * radius;
end

% Usage
[radiusArea, radiusCircumference] = circleProperties(5);
% radiusArea = 78.5398
% radiusCircumference = 31.4159

Conclusion

In this section, we covered the basics of MATLAB operations and functions. You learned how to perform arithmetic and matrix operations, use built-in functions, and create custom functions. These foundational skills are essential for more advanced topics in MATLAB programming. Practice these concepts with various examples to solidify your understanding and prepare for more complex tasks.
