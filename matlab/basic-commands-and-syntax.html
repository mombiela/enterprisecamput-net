Introduction

MATLAB (Matrix Laboratory) is a high-performance language for technical computing. It integrates computation, visualization, and programming in an easy-to-use environment. This section will cover the basic commands and syntax necessary to get started with MATLAB.

Basic Commands

Understanding basic commands is essential for navigating MATLAB and performing simple operations.

  • clc: Clears the command window.
  • clear: Removes variables from the workspace.
  • close all: Closes all figure windows.
  • who: Lists current variables in the workspace.
  • whos: Lists current variables along with their details.
  • help: Provides help for MATLAB functions.
  • doc: Opens the documentation for MATLAB functions.

Example

clc;          % Clears the command window
clear;        % Clears all variables from the workspace
close all;    % Closes all figure windows

Variables and Data Types

Variables in MATLAB are used to store data, and MATLAB supports various data types.

  • Numeric Types: double, single, int8, int16, int32, int64, uint8, uint16, uint32, uint64
  • Characters and Strings: char, string
  • Logical: true, false
  • Arrays and Matrices: Fundamental to MATLAB

Example

a = 5;                % Integer
b = 3.14;             % Floating-point number
c = 'Hello';          % Character array (string)
d = true;             % Logical true
e = [1, 2, 3; 4, 5, 6]; % 2x3 Matrix

Basic Operations

MATLAB supports a variety of basic operations, including arithmetic, relational, and logical operations.

Arithmetic Operations

  • Addition: +
  • Subtraction: -
  • Multiplication: *
  • Division: /
  • Exponentiation: ^

Example

x = 10;
y = 3;
sum = x + y;          % Addition
difference = x - y;   % Subtraction
product = x * y;      % Multiplication
quotient = x / y;     % Division
power = x ^ y;        % Exponentiation

Relational Operations

  • Equal to: ==
  • Not equal to: ~=
  • Greater than: >
  • Less than: <
  • Greater than or equal to: >=
  • Less than or equal to: <=

Example

a = 5;
b = 10;
isEqual = (a == b);          % False
isNotEqual = (a ~= b);       % True
isGreater = (a > b);         % False
isLess = (a < b);            % True
isGreaterOrEqual = (a >= b); % False
isLessOrEqual = (a <= b);    % True

Logical Operations

  • AND: &
  • OR: |
  • NOT: ~

Example

p = true;
q = false;
andResult = p & q;   % False
orResult = p | q;    % True
notResult = ~p;      % False

Control Flow

Control flow statements allow you to control the execution of code based on conditions.

If-Else Statements

x = 10;
if x > 5
    disp('x is greater than 5');
else
    disp('x is not greater than 5');
end

Switch-Case Statements

choice = 'A';
switch choice
    case 'A'
        disp('Choice is A');
    case 'B'
        disp('Choice is B');
    otherwise
        disp('Unknown choice');
end

Loops

For Loop

for i = 1:5
    disp(['Iteration: ', num2str(i)]);
end

While Loop

i = 1;
while i <= 5
    disp(['Iteration: ', num2str(i)]);
    i = i + 1;
end

Functions

Functions in MATLAB allow you to encapsulate code for reuse and better organization.

Defining a Function

function result = addNumbers(a, b)
    result = a + b;
end

Calling a Function

sum = addNumbers(3, 4);  % Calls the function and stores the result in 'sum'
disp(sum);               % Displays 7

Conclusion

This section has introduced the basic commands and syntax in MATLAB, including variable declaration, basic operations, control flow, and functions. Mastering these basics is crucial for progressing to more advanced topics in MATLAB. Practice these commands and try writing simple scripts to solidify your understanding.
