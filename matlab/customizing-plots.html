Introduction

Customizing plots in MATLAB is essential for creating clear, informative, and visually appealing graphics. This section will guide you through various customization techniques, from basic to advanced levels.

Basic Plot Customization

Changing Line Styles and Colors

  • Line Styles: MATLAB offers different line styles such as solid (-), dashed (--), dotted (:), and dash-dot (-.).
  • Colors: You can specify colors using short names ('r' for red, 'g' for green, etc.), RGB triplets, or hexadecimal color codes.
x = 0:0.1:10;
y = sin(x);

figure;
plot(x, y, 'r--'); % Red dashed line
title('Sine Wave');
xlabel('x');
ylabel('sin(x)');

Adding Titles and Labels

  • Title: Use the title function to add a title to your plot.
  • Axis Labels: Use xlabel and ylabel to label the x-axis and y-axis, respectively.
title('Sine Wave');
xlabel('x');
ylabel('sin(x)');

Legends

  • Legend: Use the legend function to add a legend to your plot.
x = 0:0.1:10;
y1 = sin(x);
y2 = cos(x);

figure;
plot(x, y1, 'b-', x, y2, 'r--');
legend('sin(x)', 'cos(x)');

Intermediate Plot Customization

Grid and Box

  • Grid: Use the grid on and grid off commands to toggle the grid.
  • Box: Use the box on and box off commands to toggle the box around the plot.
grid on;
box on;

Axis Limits and Ticks

  • Axis Limits: Use the xlim and ylim functions to set the limits of the x-axis and y-axis.
  • Ticks: Use the xticks and yticks functions to customize the tick marks.
xlim([0 10]);
ylim([-1 1]);
xticks(0:2:10);
yticks(-1:0.5:1);

Multiple Plots

  • Hold On/Off: Use hold on to overlay multiple plots and hold off to stop overlaying.
x = 0:0.1:10;
y1 = sin(x);
y2 = cos(x);

figure;
plot(x, y1, 'b-');
hold on;
plot(x, y2, 'r--');
hold off;

Advanced Plot Customization

Subplots

  • Subplot: Use the subplot function to create multiple plots in a single figure.
x = 0:0.1:10;
y1 = sin(x);
y2 = cos(x);

figure;
subplot(2, 1, 1); % 2 rows, 1 column, 1st plot
plot(x, y1);
title('Sine Wave');

subplot(2, 1, 2); % 2 rows, 1 column, 2nd plot
plot(x, y2);
title('Cosine Wave');

Annotations

  • Text Annotations: Use the text function to add text annotations to your plot.
  • Arrows and Shapes: Use the annotation function to add arrows and shapes.
x = 0:0.1:10;
y = sin(x);

figure;
plot(x, y);
text(5, 0, 'Midpoint');
annotation('arrow', [0.5, 0.7], [0.5, 0.7]);

Customizing Plot Appearance

  • Figure Properties: Use the set function to customize figure properties such as background color, size, etc.
  • Axes Properties: Customize axes properties using the set function.
figure;
set(gcf, 'Color', [0.8 0.8 0.8]); % Set figure background color

axes;
set(gca, 'FontSize', 14, 'LineWidth', 2); % Set axes properties

Conclusion

Customizing plots in MATLAB allows you to create more informative and visually appealing graphics. By mastering these techniques, you can effectively communicate your data and results. Start with basic customizations and gradually explore more advanced features to enhance your plots.
