Introduction

Arrays are fundamental data structures in MATLAB, used to store and manipulate collections of data. This section will guide you through the basics of creating arrays, accessing their elements, and performing common operations on them. By the end of this section, you will be comfortable working with arrays in MATLAB, from basic creation to advanced manipulation techniques.

Creating Arrays

1. Creating Row and Column Vectors

  • Row Vector: A row vector is a 1xN array, where N is the number of elements.
  • Column Vector: A column vector is an Nx1 array.
% Creating a row vector
row_vector = [1, 2, 3, 4, 5];

% Creating a column vector
column_vector = [1; 2; 3; 4; 5];

2. Creating Matrices

  • Matrix: A matrix is a 2D array with M rows and N columns.
% Creating a 2x3 matrix
matrix = [1, 2, 3; 4, 5, 6];

3. Using Built-in Functions

  • zeros: Creates an array of all zeros.
  • ones: Creates an array of all ones.
  • eye: Creates an identity matrix.
  • rand: Creates an array with random values between 0 and 1.
% Creating a 3x3 matrix of zeros
zero_matrix = zeros(3, 3);

% Creating a 2x4 matrix of ones
one_matrix = ones(2, 4);

% Creating a 4x4 identity matrix
identity_matrix = eye(4);

% Creating a 3x3 matrix with random values
random_matrix = rand(3, 3);

Accessing and Modifying Array Elements

1. Indexing

  • MATLAB uses 1-based indexing.
  • You can access elements using parentheses ().
% Accessing the element at row 2, column 3
element = matrix(2, 3);

% Accessing the entire second row
second_row = matrix(2, :);

% Accessing the entire third column
third_column = matrix(:, 3);

2. Modifying Elements

  • You can modify elements by assigning new values.
% Changing the element at row 1, column 2 to 10
matrix(1, 2) = 10;

% Changing the entire first row
matrix(1, :) = [7, 8, 9];

Common Array Operations

1. Arithmetic Operations

  • You can perform element-wise operations using .*, ./, and .^.
% Element-wise multiplication
result = matrix .* 2;

% Element-wise division
result = matrix ./ 2;

% Element-wise power
result = matrix .^ 2;

2. Matrix Operations

  • MATLAB supports standard matrix operations like addition, subtraction, multiplication, and transposition.
% Matrix addition
result = matrix + ones(2, 3);

% Matrix multiplication
result = matrix * eye(3);

% Matrix transposition
transposed_matrix = matrix';

3. Reshaping Arrays

  • You can change the shape of an array using the reshape function.
% Reshaping a 2x3 matrix to a 3x2 matrix
reshaped_matrix = reshape(matrix, 3, 2);

Advanced Array Manipulation

1. Concatenation

  • You can concatenate arrays horizontally or vertically.
% Horizontal concatenation
h_concat = [matrix, ones(2, 3)];

% Vertical concatenation
v_concat = [matrix; ones(2, 3)];

2. Logical Indexing

  • Use logical conditions to index and manipulate arrays.
% Creating a logical array
logical_array = matrix > 3;

% Using logical indexing to modify elements
matrix(matrix > 3) = 0;

3. Finding Elements

  • Use functions like find to locate elements that meet a condition.
% Finding indices of elements greater than 3
indices = find(matrix > 3);

Conclusion

In this section, we covered the basics of creating and manipulating arrays in MATLAB. You learned how to create vectors and matrices, access and modify their elements, perform common arithmetic and matrix operations, reshape arrays, and use advanced techniques like concatenation and logical indexing. Mastering these skills will enable you to handle data efficiently and perform complex calculations in MATLAB.
