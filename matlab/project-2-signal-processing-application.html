Introduction

In this project, we will explore how to use MATLAB for signal processing applications. Signal processing is a crucial aspect of many engineering and scientific applications, including audio processing, communications, and biomedical engineering. By the end of this project, you will have a solid understanding of how to manipulate and analyze signals using MATLAB.

Key Concepts

  • Signal Representation: Understanding how signals are represented in MATLAB.
  • Filtering: Applying different types of filters to signals.
  • Fourier Transform: Using the Fourier Transform to analyze the frequency components of signals.
  • Signal Visualization: Techniques for visualizing signals in both time and frequency domains.

Signal Representation

Signals in MATLAB are typically represented as vectors or matrices. Let's start with a simple example of creating and plotting a sine wave.

Example: Creating and Plotting a Sine Wave

% Define parameters
fs = 1000; % Sampling frequency in Hz
t = 0:1/fs:1-1/fs; % Time vector from 0 to 1 second
f = 5; % Frequency of the sine wave in Hz

% Create sine wave
x = sin(2*pi*f*t);

% Plot the sine wave
figure;
plot(t, x);
title('Sine Wave');
xlabel('Time (s)');
ylabel('Amplitude');

In this example:

  • fs is the sampling frequency.
  • t is the time vector.
  • f is the frequency of the sine wave.
  • x is the sine wave signal.

Filtering

Filtering is used to remove unwanted components from a signal. MATLAB provides various built-in functions for filtering.

Example: Applying a Low-Pass Filter

% Create a noisy signal
x_noisy = x + 0.5*randn(size(x));

% Design a low-pass filter
fc = 10; % Cutoff frequency in Hz
[b, a] = butter(6, fc/(fs/2)); % 6th order Butterworth filter

% Apply the filter
x_filtered = filter(b, a, x_noisy);

% Plot the original and filtered signals
figure;
subplot(2, 1, 1);
plot(t, x_noisy);
title('Noisy Signal');
xlabel('Time (s)');
ylabel('Amplitude');

subplot(2, 1, 2);
plot(t, x_filtered);
title('Filtered Signal');
xlabel('Time (s)');
ylabel('Amplitude');

In this example:

  • x_noisy is the original sine wave with added noise.
  • butter designs a Butterworth low-pass filter.
  • filter applies the designed filter to the noisy signal.

Fourier Transform

The Fourier Transform is used to analyze the frequency components of a signal. MATLAB's fft function computes the Fast Fourier Transform.

Example: Computing the Fourier Transform

% Compute the Fourier Transform
X = fft(x);

% Compute the frequency vector
N = length(x);
f = (0:N-1)*(fs/N);

% Plot the magnitude of the Fourier Transform
figure;
plot(f, abs(X));
title('Magnitude of Fourier Transform');
xlabel('Frequency (Hz)');
ylabel('Magnitude');

In this example:

  • fft computes the Fourier Transform of the signal.
  • f is the frequency vector.
  • abs(X) gives the magnitude of the Fourier Transform.

Signal Visualization

Visualizing signals helps in understanding their characteristics. MATLAB provides various plotting functions for this purpose.

Example: Visualizing a Signal in Time and Frequency Domains

% Plot the signal in the time domain
figure;
subplot(2, 1, 1);
plot(t, x);
title('Signal in Time Domain');
xlabel('Time (s)');
ylabel('Amplitude');

% Compute and plot the Fourier Transform
X = fft(x);
f = (0:N-1)*(fs/N);

subplot(2, 1, 2);
plot(f, abs(X));
title('Signal in Frequency Domain');
xlabel('Frequency (Hz)');
ylabel('Magnitude');

In this example:

  • The signal is plotted in both time and frequency domains for better understanding.

Conclusion

In this project, we covered the basics of signal processing using MATLAB. We learned how to represent signals, apply filters, compute the Fourier Transform, and visualize signals. These skills are fundamental for anyone working with signal processing applications. Continue practicing with different types of signals and filters to deepen your understanding.
