Introduction to Object-Oriented Programming (OOP)

Object-Oriented Programming (OOP) is a programming paradigm that uses "objects" to design applications and computer programs. MATLAB supports OOP, allowing you to create classes, objects, and use inheritance and polymorphism.

Key Concepts of OOP

  • Class: A blueprint for creating objects (a particular data structure).
  • Object: An instance of a class.
  • Properties: Variables that belong to an object or class.
  • Methods: Functions that belong to an object or class.
  • Inheritance: Mechanism to create a new class using properties and methods of an existing class.
  • Polymorphism: Ability to process objects differently based on their class.

Creating Classes and Objects in MATLAB

Defining a Class

In MATLAB, a class is defined using the classdef keyword. Below is an example of a simple class definition:

classdef Circle
    properties
        Radius
    end
    methods
        function obj = Circle(r)
            if nargin > 0
                obj.Radius = r;
            end
        end
        function area = getArea(obj)
            area = pi * obj.Radius^2;
        end
    end
end

Creating an Object

To create an object of the Circle class, you use the class constructor:

c = Circle(5);
disp(c.Radius); % Output: 5
disp(c.getArea()); % Output: 78.5398

Properties and Methods

Properties

Properties are variables that belong to a class. They are defined within the properties block.

classdef Car
    properties
        Make
        Model
        Year
    end
end

Methods

Methods are functions that operate on objects of the class. They are defined within the methods block.

classdef Car
    properties
        Make
        Model
        Year
    end
    methods
        function obj = Car(make, model, year)
            obj.Make = make;
            obj.Model = model;
            obj.Year = year;
        end
        function age = getCarAge(obj)
            currentYear = year(datetime('now'));
            age = currentYear - obj.Year;
        end
    end
end

Example Usage

myCar = Car('Toyota', 'Camry', 2015);
disp(myCar.Make); % Output: Toyota
disp(myCar.getCarAge()); % Output: (current year - 2015)

Inheritance

Defining a Subclass

Inheritance allows a new class to inherit properties and methods from an existing class.

classdef ElectricCar < Car
    properties
        BatteryCapacity
    end
    methods
        function obj = ElectricCar(make, model, year, batteryCapacity)
            obj@Car(make, model, year); % Call superclass constructor
            obj.BatteryCapacity = batteryCapacity;
        end
        function range = getRange(obj)
            range = obj.BatteryCapacity * 5; % Example calculation
        end
    end
end

Example Usage

myElectricCar = ElectricCar('Tesla', 'Model S', 2020, 100);
disp(myElectricCar.Make); % Output: Tesla
disp(myElectricCar.getRange()); % Output: 500

Polymorphism

Method Overriding

Polymorphism in MATLAB can be achieved through method overriding, where a subclass provides a specific implementation of a method that is already defined in its superclass.

classdef SportsCar < Car
    methods
        function speed = getTopSpeed(obj)
            speed = 200; % Example speed
        end
    end
end

classdef ElectricSportsCar < SportsCar
    methods
        function speed = getTopSpeed(obj)
            speed = 250; % Overridden method
        end
    end
end

Example Usage

mySportsCar = SportsCar('Ferrari', '488', 2019);
disp(mySportsCar.getTopSpeed()); % Output: 200

myElectricSportsCar = ElectricSportsCar('Tesla', 'Roadster', 2022);
disp(myElectricSportsCar.getTopSpeed()); % Output: 250

Summary

In this section, we explored the basics of Object-Oriented Programming in MATLAB. We covered:

  • The fundamental concepts of OOP.
  • How to define classes and create objects.
  • The use of properties and methods.
  • Implementing inheritance to create subclasses.
  • Achieving polymorphism through method overriding.

Understanding these concepts will allow you to design and implement more complex and modular programs in MATLAB, leveraging the power of OOP to create reusable and maintainable code.
