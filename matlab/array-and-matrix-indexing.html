Array and Matrix Indexing in MATLAB
Advanced Array Functions ►

Introduction

Array and matrix indexing is a fundamental concept in MATLAB that allows you to access and manipulate elements within arrays and matrices. This topic covers the basics of indexing, advanced indexing techniques, and practical examples to help you become proficient in handling arrays and matrices in MATLAB.

Basic Indexing

  • One-Dimensional Arrays (Vectors):
    • Access elements using parentheses ().
    • Indexing starts at 1 (not 0 as in many other programming languages).
% Create a row vector
row_vector = [10, 20, 30, 40, 50];

% Access the third element
third_element = row_vector(3); % Output: 30
  • Two-Dimensional Arrays (Matrices):
    • Use row and column indices to access elements.
    • Syntax: matrix(row, column).
% Create a 3x3 matrix
matrix = [1, 2, 3; 4, 5, 6; 7, 8, 9];

% Access the element in the second row, third column
element = matrix(2, 3); % Output: 6

Advanced Indexing Techniques

  • Colon Operator (:):
    • Used to access a range of elements or entire rows/columns.
% Access all elements in the second row
second_row = matrix(2, :); % Output: [4, 5, 6]

% Access all elements in the third column
third_column = matrix(:, 3); % Output: [3; 6; 9]
  • Logical Indexing:
    • Use logical conditions to index arrays.
% Create a logical array where elements > 4
logical_index = matrix > 4;

% Use logical array to index the matrix
filtered_elements = matrix(logical_index); % Output: [5, 6, 7, 8, 9]
  • Linear Indexing:
    • Treats the matrix as a single column vector.
% Access the fifth element in column-major order
linear_element = matrix(5); % Output: 5

Modifying Elements

  • Single Element Modification:
% Modify the element in the first row, second column
matrix(1, 2) = 20; % matrix becomes [1, 20, 3; 4, 5, 6; 7, 8, 9]
  • Multiple Elements Modification:
% Modify an entire row
matrix(2, :) = [40, 50, 60]; % matrix becomes [1, 20, 3; 40, 50, 60; 7, 8, 9]

Special Indexing Functions

  • end Keyword:
    • Represents the last index in a dimension.
% Access the last element in the vector
last_element = row_vector(end); % Output: 50

% Access the last element in the first row
last_element_row = matrix(1, end); % Output: 3
  • sub2ind and ind2sub:
    • Convert between subscript and linear indices.
% Convert subscript to linear index
linear_index = sub2ind(size(matrix), 3, 2); % Output: 8

% Convert linear index to subscript
[row, col] = ind2sub(size(matrix), 8); % Output: row = 3, col = 2

Practical Examples

  • Example 1: Extracting a Submatrix
% Extract a 2x2 submatrix from the top-left corner
submatrix = matrix(1:2, 1:2); % Output: [1, 20; 40, 50]
  • Example 2: Replacing Elements Based on Condition
% Replace all elements greater than 5 with 0
matrix(matrix > 5) = 0; % matrix becomes [1, 20, 3; 40, 50, 0; 7, 8, 9]

Conclusion

Understanding array and matrix indexing in MATLAB is crucial for efficient data manipulation and analysis. By mastering basic and advanced indexing techniques, you can perform complex operations with ease. Practice these concepts with various examples to solidify your understanding and become proficient in MATLAB programming.
