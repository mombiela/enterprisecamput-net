Introduction to 3D Plotting

3D plotting in MATLAB allows you to visualize data in three dimensions, providing a deeper understanding of data relationships and structures. This section will guide you from the basics to advanced techniques in 3D plotting.

Basic 3D Plots

Creating a Simple 3D Plot

To create a basic 3D plot, you can use the plot3 function. This function requires three vectors: X, Y, and Z.

% Define the data
X = 1:10;
Y = rand(1, 10);
Z = rand(1, 10);

% Create the 3D plot
figure;
plot3(X, Y, Z, '-o');
xlabel('X-axis');
ylabel('Y-axis');
zlabel('Z-axis');
title('Simple 3D Plot');
grid on;

Scatter Plots

For scattered data points, use the scatter3 function.

% Define the data
X = rand(1, 50);
Y = rand(1, 50);
Z = rand(1, 50);

% Create the 3D scatter plot
figure;
scatter3(X, Y, Z, 'filled');
xlabel('X-axis');
ylabel('Y-axis');
zlabel('Z-axis');
title('3D Scatter Plot');
grid on;

Surface Plots

Surface plots are useful for visualizing functions of two variables.

Creating a Surface Plot

Use the surf function to create a surface plot.

% Define the data
[X, Y] = meshgrid(-5:0.5:5, -5:0.5:5);
Z = sin(sqrt(X.^2 + Y.^2));

% Create the surface plot
figure;
surf(X, Y, Z);
xlabel('X-axis');
ylabel('Y-axis');
zlabel('Z-axis');
title('Surface Plot');
colorbar;

Mesh Plots

Mesh plots are similar to surface plots but display only the edges.

% Define the data
[X, Y] = meshgrid(-5:0.5:5, -5:0.5:5);
Z = sin(sqrt(X.^2 + Y.^2));

% Create the mesh plot
figure;
mesh(X, Y, Z);
xlabel('X-axis');
ylabel('Y-axis');
zlabel('Z-axis');
title('Mesh Plot');

Advanced 3D Plotting Techniques

Contour Plots

Contour plots can be added to surface plots to show level curves.

% Define the data
[X, Y] = meshgrid(-5:0.5:5, -5:0.5:5);
Z = sin(sqrt(X.^2 + Y.^2));

% Create the surface plot with contours
figure;
surf(X, Y, Z);
hold on;
contour3(X, Y, Z, 20, 'k');
xlabel('X-axis');
ylabel('Y-axis');
zlabel('Z-axis');
title('Surface Plot with Contours');
colorbar;

Customizing 3D Plots

You can customize 3D plots by changing properties like color, lighting, and view angles.

% Define the data
[X, Y] = meshgrid(-5:0.5:5, -5:0.5:5);
Z = sin(sqrt(X.^2 + Y.^2));

% Create the surface plot
figure;
surf(X, Y, Z);
xlabel('X-axis');
ylabel('Y-axis');
zlabel('Z-axis');
title('Customized Surface Plot');
colorbar;

% Customize the plot
shading interp; % Smooth color transitions
colormap jet; % Change color map
lighting phong; % Add lighting
camlight headlight; % Add a light source
view([-30, 30]); % Change view angle

Conclusion

3D plotting in MATLAB provides powerful tools to visualize and analyze data in three dimensions. Starting from basic plots like plot3 and scatter3, you can move to more complex visualizations like surface and mesh plots. Advanced customization options allow you to enhance the readability and aesthetics of your plots. By mastering these techniques, you can effectively communicate complex data insights.
