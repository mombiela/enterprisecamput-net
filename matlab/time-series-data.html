Introduction to Time Series Data

Time series data is a sequence of data points collected or recorded at specific time intervals. It is widely used in various fields such as finance, economics, environmental studies, and engineering.

Key Concepts

  • Time Series: A series of data points indexed in time order.
  • Timestamp: The specific time at which a data point is recorded.
  • Frequency: The interval at which data points are recorded (e.g., daily, monthly).
  • Trend: The long-term movement in a time series.
  • Seasonality: Regular patterns or cycles in a time series.

Working with Time Series Data in MATLAB

Creating Time Series Data

MATLAB provides several ways to create and manipulate time series data. The datetime and timetable functions are particularly useful.

Example: Creating a Simple Time Series

% Create a datetime array for the timestamps
timeStamps = datetime(2023,1,1):caldays(1):datetime(2023,1,10);

% Create a corresponding data array
data = rand(1, 10); % Random data for illustration

% Combine into a timetable
tsData = timetable(timeStamps', data');
disp(tsData);

Explanation:

  • datetime(2023,1,1):caldays(1):datetime(2023,1,10) creates a range of dates from January 1, 2023, to January 10, 2023.
  • rand(1, 10) generates random data points.
  • timetable(timeStamps', data') creates a timetable with the timestamps and data.

Plotting Time Series Data

Visualizing time series data is crucial for analysis.

Example: Plotting Time Series Data

% Plot the time series data
plot(tsData.timeStamps, tsData.data);
xlabel('Date');
ylabel('Value');
title('Time Series Data');
grid on;

Explanation:

  • plot(tsData.timeStamps, tsData.data) plots the data points against their timestamps.
  • xlabel, ylabel, and title add labels and a title to the plot.
  • grid on adds a grid to the plot for better readability.

Advanced Time Series Analysis

Decomposing Time Series

Decomposing a time series into its components (trend, seasonality, and residuals) helps in understanding the underlying patterns.

Example: Decomposing Time Series

% Generate a sample time series with trend and seasonality
timeStamps = datetime(2023,1,1):caldays(1):datetime(2023,12,31);
data = 0.05*(1:length(timeStamps))' + 2*sin(2*pi*(1:length(timeStamps))'/365) + randn(length(timeStamps),1);

% Create a timetable
tsData = timetable(timeStamps', data);

% Decompose the time series
[trend, seasonal, residual] = decompose(tsData.data, 'model', 'additive');

% Plot the components
figure;
subplot(3,1,1);
plot(tsData.timeStamps, trend);
title('Trend');
subplot(3,1,2);
plot(tsData.timeStamps, seasonal);
title('Seasonality');
subplot(3,1,3);
plot(tsData.timeStamps, residual);
title('Residuals');

Explanation:

  • decompose function is used to decompose the time series into trend, seasonal, and residual components.
  • The subplot function is used to create multiple plots in a single figure for better visualization.

Forecasting Time Series

Forecasting future values based on historical data is a common application of time series analysis.

Example: Forecasting with ARIMA Model

% Load sample data
load(fullfile(matlabroot,'examples','econ','Data_Airline.mat'))

% Fit an ARIMA model
Mdl = arima(1,1,1);
EstMdl = estimate(Mdl, Data);

% Forecast future values
numPeriods = 12; % Forecast for the next 12 periods
[forecast, forecastMSE] = forecast(EstMdl, numPeriods, 'Y0', Data);

% Plot the forecast
figure;
plot(Data, 'Color', [0, 0.4470, 0.7410]);
hold on;
plot(length(Data)+1:length(Data)+numPeriods, forecast, 'r', 'LineWidth', 2);
title('ARIMA Forecast');
xlabel('Time');
ylabel('Value');
legend('Historical Data', 'Forecast');
grid on;

Explanation:

  • arima(1,1,1) specifies an ARIMA model with one autoregressive term, one differencing term, and one moving average term.
  • estimate fits the ARIMA model to the data.
  • forecast predicts future values based on the fitted model.

Conclusion

Time series analysis in MATLAB involves creating, visualizing, decomposing, and forecasting time series data. By understanding these fundamental concepts and techniques, you can effectively analyze and interpret time series data for various applications.
