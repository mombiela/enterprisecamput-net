Introduction

In Machine Learning, evaluation metrics are crucial for measuring the performance of models. These metrics allow us to understand how well a model is working and compare it with other models. In this topic, we will explore the most common metrics used to evaluate classification and regression models.

Metrics for Classification Models

Accuracy

  • Definition: The proportion of correct predictions over the total predictions.
  • Formula: \[ \text{Accuracy} = \frac{\text{Number of correct predictions}}{\text{Total number of predictions}} \]

Precision

  • Definition: The proportion of true positives over the total positive predictions.
  • Formula: \[ \text{Precision} = \frac{\text{True Positives}}{\text{True Positives} + \text{False Positives}} \]

Recall (Sensitivity)

  • Definition: The proportion of true positives over the total actual positives.
  • Formula: \[ \text{Recall} = \frac{\text{True Positives}}{\text{True Positives} + \text{False Negatives}} \]

F1-Score

  • Definition: The harmonic mean of precision and recall.
  • Formula: \[ \text{F1-Score} = 2 \times \frac{\text{Precision} \times \text{Recall}}{\text{Precision} + \text{Recall}} \]

Code Example

from sklearn.metrics import accuracy_score, precision_score, recall_score, f1_score

# Suppose we have the following true labels and predictions
y_true = [0, 1, 1, 0, 1, 0, 1, 1]
y_pred = [0, 1, 0, 0, 1, 0, 1, 1]

# Calculate the metrics
accuracy = accuracy_score(y_true, y_pred)
precision = precision_score(y_true, y_pred)
recall = recall_score(y_true, y_pred)
f1 = f1_score(y_true, y_pred)

print(f"Accuracy: {accuracy}")
print(f"Precision: {precision}")
print(f"Recall: {recall}")
print(f"F1-Score: {f1}")

Metrics for Regression Models

Mean Absolute Error (MAE)

  • Definition: The mean of the absolute differences between the predictions and the actual values.
  • Formula: \[ \text{MAE} = \frac{1}{n} \sum_{i=1}^{n} |y_i - \hat{y}_i| \]

Mean Squared Error (MSE)

  • Definition: The mean of the squares of the differences between the predictions and the actual values.
  • Formula: \[ \text{MSE} = \frac{1}{n} \sum_{i=1}^{n} (y_i - \hat{y}_i)^2 \]

Root Mean Squared Error (RMSE)

  • Definition: The square root of the MSE.
  • Formula: \[ \text{RMSE} = \sqrt{\frac{1}{n} \sum_{i=1}^{n} (y_i - \hat{y}_i)^2} \]

Coefficient of Determination (R²)

  • Definition: The proportion of the variance in the dependent variable that is predictable from the independent variables.
  • Formula: \[ R^2 = 1 - \frac{\sum_{i=1}^{n} (y_i - \hat{y}i)^2}{\sum{i=1}^{n} (y_i - \bar{y})^2} \]

Code Example

from sklearn.metrics import mean_absolute_error, mean_squared_error, r2_score
import numpy as np

# Suppose we have the following true labels and predictions
y_true = np.array([3.0, -0.5, 2.0, 7.0])
y_pred = np.array([2.5, 0.0, 2.0, 8.0])

# Calculate the metrics
mae = mean_absolute_error(y_true, y_pred)
mse = mean_squared_error(y_true, y_pred)
rmse = np.sqrt(mse)
r2 = r2_score(y_true, y_pred)

print(f"MAE: {mae}")
print(f"MSE: {mse}")
print(f"RMSE: {rmse}")
print(f"R²: {r2}")

Comparison of Metrics

| Metric | Model Type | Interpretation | |---------------|-----------------|-------------------------------------------------------------------------------| | Accuracy | Classification | Proportion of correct predictions. | | Precision | Classification | Proportion of true positives over the total positive predictions. | | Recall | Classification | Proportion of true positives over the total actual positives. | | F1-Score | Classification | Harmonic mean of precision and recall. | | MAE | Regression | Mean of the absolute differences between the predictions and the actual values. | | MSE | Regression | Mean of the squares of the differences between the predictions and the actual values. | | RMSE | Regression | Square root of the MSE. | | R² | Regression | Proportion of the variance explained by the model. |

Conclusion

Evaluation metrics are essential for understanding the performance of Machine Learning models. The choice of the appropriate metric depends on the type of problem and the specific objectives of the project. It is crucial to know and correctly apply these metrics to develop effective models and make informed decisions.
