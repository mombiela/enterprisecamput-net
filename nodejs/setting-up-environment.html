Setting up your development environment is the first step to becoming proficient in Node.js. This section will guide you through the process of installing Node.js, setting up a code editor, and verifying your installation.

Installing Node.js

Node.js is a JavaScript runtime built on Chrome's V8 JavaScript engine. To start developing with Node.js, you need to install it on your machine.

Steps to Install Node.js

  1. Download Node.js:

    • Go to the official Node.js website.
    • Download the LTS (Long Term Support) version for your operating system (Windows, macOS, or Linux).

  2. Install Node.js:

    • Run the installer and follow the on-screen instructions.
    • The installer will also install npm (Node Package Manager), which is essential for managing Node.js packages.

  3. Verify Installation:

    • Open your terminal or command prompt.
    • Run the following commands to check if Node.js and npm are installed correctly: 
      node -v
npm -v
    • You should see the version numbers of Node.js and npm.

Setting Up a Code Editor

A good code editor can significantly enhance your productivity. For Node.js development, Visual Studio Code (VS Code) is highly recommended due to its powerful features and extensions.

Installing Visual Studio Code

  1. Download VS Code:

    • Visit the Visual Studio Code website.
    • Download the installer for your operating system.

  2. Install VS Code:

    • Run the installer and follow the on-screen instructions.

  3. Install Extensions:

    • Open VS Code.
    • Go to the Extensions view by clicking on the Extensions icon in the Activity Bar on the side of the window or by pressing Ctrl+Shift+X.
    • Search for and install the following extensions:
      • Node.js Extension Pack: Provides a collection of extensions for Node.js development.
      • ESLint: Helps in identifying and fixing problems in your JavaScript code.
      • Prettier - Code formatter: Ensures consistent code formatting.

Creating Your First Node.js Project

Now that you have Node.js and a code editor set up, let's create a simple Node.js project to ensure everything is working correctly.

Steps to Create a Node.js Project

  1. Create a Project Directory:

    • Open your terminal or command prompt.
    • Create a new directory for your project and navigate into it: 
      mkdir my-node-project
cd my-node-project

  2. Initialize the Project:

    • Run the following command to create a package.json file, which will manage your project's dependencies: 
      npm init -y

  3. Create an Entry Point File:

    • In your project directory, create a file named index.js.
    • Open index.js in your code editor and add the following code: 
      console.log('Hello, Node.js!');

  4. Run Your Node.js Application:

    • In your terminal, run the following command: 
      node index.js
    • You should see the output Hello, Node.js!.

Summary

In this section, you have learned how to set up your development environment for Node.js. You installed Node.js and npm, set up Visual Studio Code with essential extensions, and created a simple Node.js project to verify your setup. With your environment ready, you are now prepared to dive deeper into Node.js development.
