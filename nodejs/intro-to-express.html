Express.js is a minimal and flexible Node.js web application framework that provides a robust set of features to develop web and mobile applications. It facilitates the rapid development of Node-based web applications.

What is Express.js?

Express.js is a web application framework for Node.js, designed for building web applications and APIs. It is the de facto standard server framework for Node.js.

  • Minimal: Express is a minimalistic framework that provides a thin layer of fundamental web application features.
  • Flexible: It is highly flexible and allows developers to structure their applications as they see fit.
  • Robust: Express provides a robust set of features for web and mobile applications.

Setting Up Express.js

To start using Express.js, you need to have Node.js installed on your machine. You can install Express.js using npm (Node Package Manager).

Installation Steps

  1. Initialize a new Node.js project:

    mkdir myapp
cd myapp
npm init -y

  2. Install Express.js:

    npm install express

Creating a Basic Express.js Application

Let's create a simple Express.js application to understand its basic structure.

Basic Application Structure

  1. Create a file named app.js:

    const express = require('express');
const app = express();
const port = 3000;

app.get('/', (req, res) => {
  res.send('Hello World!');
});

app.listen(port, () => {
  console.log(`Example app listening at http://localhost:${port}`);
});

  2. Run the application:

    node app.js

  3. Access the application: Open your browser and navigate to http://localhost:3000. You should see "Hello World!" displayed.

Explanation of Code

  • Importing Express:

    const express = require('express');

    This line imports the Express module.

  • Creating an Express application:

    const app = express();

    This line initializes a new Express application.

  • Defining a route:

    app.get('/', (req, res) => {
  res.send('Hello World!');
});

    This defines a route for the root URL (/). When a GET request is made to the root URL, the server responds with "Hello World!".

  • Starting the server:

    app.listen(port, () => {
  console.log(`Example app listening at http://localhost:${port}`);
});

    This starts the server and listens on port 3000. When the server starts, it logs a message to the console.

Key Features of Express.js

  • Middleware: Functions that execute during the lifecycle of a request to the server.
  • Routing: Defines how an application responds to a client request to a particular endpoint.
  • Template Engines: Allows you to use static template files in your application.

Middleware

Middleware functions are functions that have access to the request object (req), the response object (res), and the next middleware function in the application’s request-response cycle.

Example of a simple middleware:

app.use((req, res, next) => {
  console.log('Time:', Date.now());
  next();
});

Routing

Routing refers to how an application’s endpoints (URIs) respond to client requests.

Example of defining routes:

app.get('/about', (req, res) => {
  res.send('About Page');
});

app.post('/submit', (req, res) => {
  res.send('Form Submitted');
});

Template Engines

Express supports various template engines like Pug, EJS, etc., to generate HTML dynamically.

Example using Pug:

  1. Install Pug:

    npm install pug

  2. Set up Pug in Express:

    app.set('view engine', 'pug');
app.set('views', './views');

  3. Create a Pug template (views/index.pug):

    html
  head
    title= title
  body
    h1= message

  4. Render the Pug template:

    app.get('/', (req, res) => {
  res.render('index', { title: 'Hey', message: 'Hello there!' });
});

Conclusion

Express.js is a powerful and flexible framework for building web applications with Node.js. It provides a minimalistic yet robust set of features that make it easy to develop web and mobile applications. By understanding the basics of setting up an Express application, defining routes, using middleware, and integrating template engines, you can start building your own web applications efficiently.
