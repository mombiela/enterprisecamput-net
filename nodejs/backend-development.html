Introduction to Node.js

  • What is Node.js?
    • Node.js is a JavaScript runtime built on Chrome's V8 JavaScript engine.
    • It allows you to run JavaScript on the server side.
  • Why use Node.js for backend development?
    • Non-blocking, event-driven architecture.
    • High performance for I/O-bound tasks.
    • Large ecosystem with npm (Node Package Manager).

Setting Up Your Node.js Environment

  • Installing Node.js and npm
    • Download and install from the official Node.js website.
    • Verify installation: 
      node -v
npm -v
  • Setting up a project
    • Initialize a new Node.js project: 
      mkdir my-node-app
cd my-node-app
npm init -y

Basic Concepts

  • Modules and require
    • Creating and using modules: 
      // math.js
function add(a, b) {
  return a + b;
}
module.exports = add;

// app.js
const add = require('./math');
console.log(add(2, 3)); // Output: 5
  • Asynchronous programming
    • Callbacks: 
      const fs = require('fs');
fs.readFile('example.txt', 'utf8', (err, data) => {
  if (err) throw err;
  console.log(data);
});
    • Promises: 
      const fs = require('fs').promises;
fs.readFile('example.txt', 'utf8')
  .then(data => console.log(data))
  .catch(err => console.error(err));
    • Async/Await: 
      const fs = require('fs').promises;
async function readFile() {
  try {
    const data = await fs.readFile('example.txt', 'utf8');
    console.log(data);
  } catch (err) {
    console.error(err);
  }
}
readFile();

Building a Simple Web Server

  • Using the HTTP module
    • Creating a basic server: 
      const http = require('http');
const server = http.createServer((req, res) => {
  res.statusCode = 200;
  res.setHeader('Content-Type', 'text/plain');
  res.end('Hello, World!\n');
});
server.listen(3000, '127.0.0.1', () => {
  console.log('Server running at http://127.0.0.1:3000/');
});

Express.js Framework

  • Introduction to Express.js
    • Simplifies server creation and routing.
    • Installation: 
      npm install express
  • Basic usage
    • Creating an Express server: 
      const express = require('express');
const app = express();
const port = 3000;

app.get('/', (req, res) => {
  res.send('Hello, World!');
});

app.listen(port, () => {
  console.log(`Server running at http://localhost:${port}/`);
});
  • Middleware
    • Using middleware for request processing: 
      app.use((req, res, next) => {
  console.log('Time:', Date.now());
  next();
});

Working with Databases

  • Connecting to a MongoDB database using Mongoose
    • Installation: 
      npm install mongoose
    • Basic usage: 
      const mongoose = require('mongoose');
mongoose.connect('mongodb://localhost/mydatabase', { useNewUrlParser: true, useUnifiedTopology: true });

const db = mongoose.connection;
db.on('error', console.error.bind(console, 'connection error:'));
db.once('open', () => {
  console.log('Connected to the database');
});
  • Defining schemas and models
    • Example schema and model: 
      const userSchema = new mongoose.Schema({
  name: String,
  age: Number,
  email: String
});

const User = mongoose.model('User', userSchema);

const newUser = new User({ name: 'John Doe', age: 30, email: 'john@example.com' });
newUser.save((err) => {
  if (err) return console.error(err);
  console.log('User saved successfully');
});

Advanced Topics

  • Authentication with JWT (JSON Web Tokens)
    • Installation: 
      npm install jsonwebtoken
    • Generating and verifying tokens: 
      const jwt = require('jsonwebtoken');
const secretKey = 'your-secret-key';

// Generating a token
const token = jwt.sign({ username: 'john_doe' }, secretKey, { expiresIn: '1h' });
console.log('Token:', token);

// Verifying a token
jwt.verify(token, secretKey, (err, decoded) => {
  if (err) {
    console.error('Token verification failed:', err);
  } else {
    console.log('Decoded token:', decoded);
  }
});
  • Real-time communication with Socket.io
    • Installation: 
      npm install socket.io
    • Basic usage: 
      const http = require('http');
const socketIo = require('socket.io');
const server = http.createServer();
const io = socketIo(server);

io.on('connection', (socket) => {
  console.log('A user connected');
  socket.on('disconnect', () => {
    console.log('User disconnected');
  });
});

server.listen(3000, () => {
  console.log('Server running at http://localhost:3000/');
});

Conclusion

  • Recap of key concepts:
    • Node.js basics: modules, asynchronous programming.
    • Building web servers with HTTP and Express.js.
    • Connecting to databases with Mongoose.
    • Advanced topics like JWT authentication and real-time communication with Socket.io.

By following this structured course, you will gain a solid foundation in backend development with Node.js, enabling you to build robust and scalable web applications. Happy coding!
