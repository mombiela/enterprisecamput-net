Forms and Inputs in Ionic
Introduction to Ionic Navigation ►

Introduction

Forms and inputs are fundamental components in any application, allowing users to interact with the app by providing data. In Ionic, creating forms and handling user input is straightforward, thanks to its rich set of UI components and Angular's powerful form handling capabilities.

Basic Form Elements

Ionic provides a variety of form elements that can be used to capture user input. These elements are styled to match the look and feel of your Ionic application.

Text Inputs

Text inputs are used to capture single-line text data from the user.

<ion-item>
  <ion-label position="floating">Username</ion-label>
  <ion-input type="text"></ion-input>
</ion-item>
  • <ion-item>: A container for form elements.
  • <ion-label>: A label for the input field.
  • <ion-input>: The input field itself.

Password Inputs

Password inputs are used to capture sensitive information like passwords.

<ion-item>
  <ion-label position="floating">Password</ion-label>
  <ion-input type="password"></ion-input>
</ion-item>

Email Inputs

Email inputs are specifically for capturing email addresses.

<ion-item>
  <ion-label position="floating">Email</ion-label>
  <ion-input type="email"></ion-input>
</ion-item>

Number Inputs

Number inputs are used to capture numeric data.

<ion-item>
  <ion-label position="floating">Age</ion-label>
  <ion-input type="number"></ion-input>
</ion-item>

Advanced Form Elements

Ionic also provides more advanced form elements for capturing different types of data.

Select Inputs

Select inputs allow users to choose from a list of options.

<ion-item>
  <ion-label>Favorite Fruit</ion-label>
  <ion-select>
    <ion-select-option value="apple">Apple</ion-select-option>
    <ion-select-option value="banana">Banana</ion-select-option>
    <ion-select-option value="cherry">Cherry</ion-select-option>
  </ion-select>
</ion-item>

Date Inputs

Date inputs allow users to select a date.

<ion-item>
  <ion-label>Date of Birth</ion-label>
  <ion-datetime display-format="MM/DD/YYYY"></ion-datetime>
</ion-item>

Toggle Inputs

Toggle inputs are used for binary choices, like on/off switches.

<ion-item>
  <ion-label>Enable Notifications</ion-label>
  <ion-toggle></ion-toggle>
</ion-item>

Form Validation

Form validation is crucial for ensuring that the data entered by the user is correct and complete. Angular provides powerful tools for form validation, which can be easily integrated with Ionic forms.

Template-Driven Forms

Template-driven forms are simple and suitable for basic use cases.

<form #myForm="ngForm">
  <ion-item>
    <ion-label position="floating">Username</ion-label>
    <ion-input type="text" name="username" ngModel required></ion-input>
  </ion-item>
  <div *ngIf="myForm.controls.username?.invalid && myForm.controls.username?.touched">
    Username is required.
  </div>
  <ion-button type="submit" [disabled]="myForm.invalid">Submit</ion-button>
</form>

Reactive Forms

Reactive forms provide more control and are suitable for complex scenarios.

import { Component } from '@angular/core';
import { FormBuilder, FormGroup, Validators } from '@angular/forms';

@Component({
  selector: 'app-form-example',
  templateUrl: './form-example.component.html'
})
export class FormExampleComponent {
  myForm: FormGroup;

  constructor(private fb: FormBuilder) {
    this.myForm = this.fb.group({
      username: ['', Validators.required],
      email: ['', [Validators.required, Validators.email]]
    });
  }

  onSubmit() {
    if (this.myForm.valid) {
      console.log(this.myForm.value);
    }
  }
}
<form [formGroup]="myForm" (ngSubmit)="onSubmit()">
  <ion-item>
    <ion-label position="floating">Username</ion-label>
    <ion-input formControlName="username"></ion-input>
  </ion-item>
  <div *ngIf="myForm.get('username').invalid && myForm.get('username').touched">
    Username is required.
  </div>

  <ion-item>
    <ion-label position="floating">Email</ion-label>
    <ion-input formControlName="email"></ion-input>
  </ion-item>
  <div *ngIf="myForm.get('email').invalid && myForm.get('email').touched">
    Valid email is required.
  </div>

  <ion-button type="submit" [disabled]="myForm.invalid">Submit</ion-button>
</form>

Handling Form Submission

Handling form submission involves capturing the data entered by the user and processing it, such as sending it to a server.

<form (ngSubmit)="onSubmit(myForm)" #myForm="ngForm">
  <ion-item>
    <ion-label position="floating">Username</ion-label>
    <ion-input type="text" name="username" ngModel required></ion-input>
  </ion-item>
  <ion-button type="submit" [disabled]="myForm.invalid">Submit</ion-button>
</form>
import { Component } from '@angular/core';

@Component({
  selector: 'app-form-example',
  templateUrl: './form-example.component.html'
})
export class FormExampleComponent {
  onSubmit(form) {
    if (form.valid) {
      console.log(form.value);
    }
  }
}

Conclusion

Forms and inputs are essential for capturing user data in any application. Ionic provides a comprehensive set of form elements that are easy to use and integrate with Angular's form handling capabilities. Whether you are building simple forms with template-driven approaches or complex forms with reactive forms, Ionic has the tools you need to create robust and user-friendly forms.
