Introduction

In this course, we will guide you through the process of creating a weather app using Ionic. This course is designed to take you from a beginner to an advanced level, covering all the essential concepts and techniques required to build a fully functional weather application.

Prerequisites

Before starting this course, you should have:

  • Basic knowledge of HTML, CSS, and JavaScript.
  • Node.js and npm installed on your machine.
  • Basic understanding of Angular (since Ionic is built on Angular).

Setting Up the Environment

To get started with Ionic, you need to set up your development environment.

Installing Ionic CLI

First, install the Ionic CLI globally using npm:

npm install -g @ionic/cli

Creating a New Ionic Project

Create a new Ionic project using the following command:

ionic start weather-app blank --type=angular

Navigate to your project directory:

cd weather-app

Building the User Interface

In this section, we will build the basic user interface for our weather app.

Creating Pages

Ionic uses Angular's routing system to navigate between pages. Let's create a home page and a details page.

ionic generate page home
ionic generate page details

Designing the Home Page

Open src/app/home/home.page.html and add the following code to create a basic layout:

<ion-header>
  <ion-toolbar>
    <ion-title>Weather App</ion-title>
  </ion-toolbar>
</ion-header>

<ion-content>
  <ion-item>
    <ion-label position="floating">Enter City</ion-label>
    <ion-input [(ngModel)]="city"></ion-input>
  </ion-item>
  <ion-button expand="full" (click)="getWeather()">Get Weather</ion-button>
  <ion-card *ngIf="weather">
    <ion-card-header>
      <ion-card-title>{{ weather.name }}</ion-card-title>
    </ion-card-header>
    <ion-card-content>
      <p>Temperature: {{ weather.main.temp }}°C</p>
      <p>Condition: {{ weather.weather[0].description }}</p>
    </ion-card-content>
  </ion-card>
</ion-content>

Adding Logic to Home Page

Open src/app/home/home.page.ts and add the following code to handle user input and fetch weather data:

import { Component } from '@angular/core';
import { HttpClient } from '@angular/common/http';

@Component({
  selector: 'app-home',
  templateUrl: 'home.page.html',
  styleUrls: ['home.page.scss'],
})
export class HomePage {
  city: string;
  weather: any;

  constructor(private http: HttpClient) {}

  getWeather() {
    const apiKey = 'YOUR_API_KEY';
    const url = `https://api.openweathermap.org/data/2.5/weather?q=${this.city}&appid=${apiKey}&units=metric`;

    this.http.get(url).subscribe(data => {
      this.weather = data;
    });
  }
}

Integrating with a Weather API

To fetch real-time weather data, we will use the OpenWeatherMap API.

Getting an API Key

  • Sign up at OpenWeatherMap.
  • Navigate to the API section and generate an API key.

Using the API Key

Replace 'YOUR_API_KEY' in the getWeather method with your actual API key.

Advanced Features

Once the basic functionality is working, we can add more advanced features.

Adding a Loading Indicator

To improve user experience, show a loading indicator while fetching data.

import { LoadingController } from '@ionic/angular';

constructor(private http: HttpClient, private loadingController: LoadingController) {}

async getWeather() {
  const loading = await this.loadingController.create({
    message: 'Fetching weather...',
  });
  await loading.present();

  const apiKey = 'YOUR_API_KEY';
  const url = `https://api.openweathermap.org/data/2.5/weather?q=${this.city}&appid=${apiKey}&units=metric`;

  this.http.get(url).subscribe(data => {
    this.weather = data;
    loading.dismiss();
  }, error => {
    loading.dismiss();
    console.error('Error fetching weather data', error);
  });
}

Navigating to Details Page

Pass data to the details page and display more detailed weather information.

Modifying Home Page

<ion-button expand="full" (click)="goToDetails()">View Details</ion-button>
import { NavController } from '@ionic/angular';

constructor(private http: HttpClient, private loadingController: LoadingController, private navCtrl: NavController) {}

goToDetails() {
  this.navCtrl.navigateForward('/details', {
    queryParams: {
      weather: JSON.stringify(this.weather)
    }
  });
}

Designing Details Page

Open src/app/details/details.page.html and add the following code:

<ion-header>
  <ion-toolbar>
    <ion-title>Weather Details</ion-title>
  </ion-toolbar>
</ion-header>

<ion-content>
  <ion-card *ngIf="weather">
    <ion-card-header>
      <ion-card-title>{{ weather.name }}</ion-card-title>
    </ion-card-header>
    <ion-card-content>
      <p>Temperature: {{ weather.main.temp }}°C</p>
      <p>Condition: {{ weather.weather[0].description }}</p>
      <p>Humidity: {{ weather.main.humidity }}%</p>
      <p>Wind Speed: {{ weather.wind.speed }} m/s</p>
    </ion-card-content>
  </ion-card>
</ion-content>

Adding Logic to Details Page

Open src/app/details/details.page.ts and add the following code:

import { Component, OnInit } from '@angular/core';
import { ActivatedRoute } from '@angular/router';

@Component({
  selector: 'app-details',
  templateUrl: './details.page.html',
  styleUrls: ['./details.page.scss'],
})
export class DetailsPage implements OnInit {
  weather: any;

  constructor(private route: ActivatedRoute) {}

  ngOnInit() {
    this.route.queryParams.subscribe(params => {
      if (params && params.weather) {
        this.weather = JSON.parse(params.weather);
      }
    });
  }
}

Conclusion

In this course, we have built a weather app using Ionic from scratch. We covered:

  • Setting up the Ionic environment.
  • Creating and designing pages.
  • Fetching data from an API.
  • Adding advanced features like loading indicators and navigation.

By the end of this course, you should have a solid understanding of how to build and enhance a weather app using Ionic. Keep experimenting with new features and APIs to further improve your app. Happy coding!
