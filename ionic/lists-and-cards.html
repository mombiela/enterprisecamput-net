Introduction

In this section, we will explore how to create and manage lists and cards in Ionic. Lists and cards are fundamental components for displaying collections of data in a structured and visually appealing manner. By the end of this section, you will be able to create dynamic lists and cards, customize their appearance, and handle user interactions.

Lists in Ionic

Key Concepts

  • IonList: The main container for a list.
  • IonItem: Represents each item within a list.
  • IonLabel: Used to display text within an item.
  • IonAvatar: Used to display an avatar or image within an item.
  • IonIcon: Used to display icons within an item.

Basic List Example

Here is a basic example of a list in Ionic:

<ion-list>
  <ion-item>
    <ion-label>Item 1</ion-label>
  </ion-item>
  <ion-item>
    <ion-label>Item 2</ion-label>
  </ion-item>
  <ion-item>
    <ion-label>Item 3</ion-label>
  </ion-item>
</ion-list>

In this example:

  • <ion-list> is the container for the list.
  • <ion-item> represents each item in the list.
  • <ion-label> is used to display the text for each item.

List with Avatars and Icons

You can enhance your list items by adding avatars and icons:

<ion-list>
  <ion-item>
    <ion-avatar slot="start">
      <img src="avatar1.png" />
    </ion-avatar>
    <ion-label>Item 1</ion-label>
    <ion-icon name="star" slot="end"></ion-icon>
  </ion-item>
  <ion-item>
    <ion-avatar slot="start">
      <img src="avatar2.png" />
    </ion-avatar>
    <ion-label>Item 2</ion-label>
    <ion-icon name="heart" slot="end"></ion-icon>
  </ion-item>
</ion-list>

In this example:

  • <ion-avatar> is used to display an image at the start of each item.
  • <ion-icon> is used to display an icon at the end of each item.

Cards in Ionic

Key Concepts

  • IonCard: The main container for a card.
  • IonCardHeader: Used to define the header section of a card.
  • IonCardTitle: Used to define the title within the card header.
  • IonCardSubtitle: Used to define the subtitle within the card header.
  • IonCardContent: Used to define the main content of the card.

Basic Card Example

Here is a basic example of a card in Ionic:

<ion-card>
  <ion-card-header>
    <ion-card-title>Card Title</ion-card-title>
    <ion-card-subtitle>Card Subtitle</ion-card-subtitle>
  </ion-card-header>
  <ion-card-content>
    This is the content of the card.
  </ion-card-content>
</ion-card>

In this example:

  • <ion-card> is the container for the card.
  • <ion-card-header> contains the title and subtitle.
  • <ion-card-content> contains the main content of the card.

Card with Images and Buttons

You can enhance your cards by adding images and buttons:

<ion-card>
  <img src="card-image.png" />
  <ion-card-header>
    <ion-card-title>Card Title</ion-card-title>
    <ion-card-subtitle>Card Subtitle</ion-card-subtitle>
  </ion-card-header>
  <ion-card-content>
    This is the content of the card.
    <ion-button expand="full">Button</ion-button>
  </ion-card-content>
</ion-card>

In this example:

  • An image is added at the top of the card.
  • A button is added within the card content.

Conclusion

Lists and cards are essential components in Ionic for displaying data in a structured and visually appealing way. By understanding how to use IonList, IonItem, IonCard, and their related components, you can create dynamic and interactive user interfaces. Practice creating lists and cards with different configurations to become proficient in using these components in your Ionic applications.
