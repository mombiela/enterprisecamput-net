Overview

In this section, we will explore the fundamentals of navigation in Ionic applications. Navigation is a crucial aspect of any mobile application, as it defines how users move between different pages or views within the app. We will cover the basic concepts, types of navigation, and provide practical examples to help you understand how to implement navigation in your Ionic projects.

Key Concepts

  • Navigation Controller: Manages the navigation stack and history.
  • Pages: Individual views or screens in the application.
  • Routing: Defines the paths and components associated with each route.
  • NavController: A service that provides methods to navigate between pages.

Types of Navigation

Ionic supports several types of navigation, including:

  1. Push Navigation: Adding a new page to the navigation stack.
  2. Pop Navigation: Removing the current page from the navigation stack.
  3. Root Navigation: Setting a new root page, clearing the navigation stack.
  4. Tab Navigation: Using tabs to switch between different views.

Setting Up Navigation

Installing Ionic CLI

Before we start, ensure you have the Ionic CLI installed. If not, you can install it using npm:

npm install -g @ionic/cli

Creating a New Ionic Project

Create a new Ionic project using the following command:

ionic start myApp blank

Navigate to the project directory:

cd myApp

Adding Pages

To add a new page, use the Ionic CLI:

ionic generate page Home
ionic generate page About

Configuring Routes

Open src/app/app-routing.module.ts and configure the routes:

import { NgModule } from '@angular/core';
import { PreloadAllModules, RouterModule, Routes } from '@angular/router';

const routes: Routes = [
  {
    path: '',
    redirectTo: 'home',
    pathMatch: 'full'
  },
  {
    path: 'home',
    loadChildren: () => import('./home/home.module').then(m => m.HomePageModule)
  },
  {
    path: 'about',
    loadChildren: () => import('./about/about.module').then(m => m.AboutPageModule)
  }
];

@NgModule({
  imports: [
    RouterModule.forRoot(routes, { preloadingStrategy: PreloadAllModules })
  ],
  exports: [RouterModule]
})
export class AppRoutingModule { }

Using NavController for Navigation

To navigate between pages, you can use the NavController service. Here is an example of navigating from the Home page to the About page:

Home Page (home.page.ts)

import { Component } from '@angular/core';
import { NavController } from '@ionic/angular';

@Component({
  selector: 'app-home',
  templateUrl: 'home.page.html',
  styleUrls: ['home.page.scss'],
})
export class HomePage {
  constructor(private navCtrl: NavController) {}

  navigateToAbout() {
    this.navCtrl.navigateForward('/about');
  }
}

Home Page Template (home.page.html)

<ion-header>
  <ion-toolbar>
    <ion-title>
      Home
    </ion-title>
  </ion-toolbar>
</ion-header>

<ion-content>
  <ion-button (click)="navigateToAbout()">Go to About</ion-button>
</ion-content>
