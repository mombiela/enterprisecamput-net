Introduction

PL/SQL (Procedural Language/Structured Query Language) is Oracle Corporation's procedural extension for SQL and the Oracle relational database. It combines the data manipulation power of SQL with the processing power of procedural languages. In this section, we will learn how to write a simple "Hello World" program in PL/SQL.

Key Concepts

  • PL/SQL Block Structure: PL/SQL programs are composed of blocks, which can be anonymous or named.
  • Anonymous Block: A PL/SQL block that is not stored in the database and is executed immediately.
  • DBMS_OUTPUT Package: A package that enables you to display output from PL/SQL blocks, subprograms, packages, and triggers.

PL/SQL Block Structure

A PL/SQL block consists of three sections:

  1. Declaration Section: Optional. Used to declare variables, constants, cursors, etc.
  2. Execution Section: Mandatory. Contains the code that performs the desired operations.
  3. Exception Handling Section: Optional. Used to handle runtime errors.

Basic Structure of a PL/SQL Block

DECLARE
   -- Declaration section (optional)
BEGIN
   -- Execution section (mandatory)
EXCEPTION
   -- Exception handling section (optional)
END;
/

Writing "Hello World" in PL/SQL

To write a "Hello World" program in PL/SQL, we will use the DBMS_OUTPUT.PUT_LINE procedure to print the message.

Example Code

BEGIN
   DBMS_OUTPUT.PUT_LINE('Hello, World!');
END;
/

Explanation

  • BEGIN: Marks the beginning of the execution section.
  • DBMS_OUTPUT.PUT_LINE: A procedure that outputs a line of text.
  • 'Hello, World!': The string to be printed.
  • END;: Marks the end of the PL/SQL block.
  • /: Executes the PL/SQL block in SQL*Plus or SQL Developer.

Enabling Output in SQL*Plus or SQL Developer

Before running the PL/SQL block, you need to enable the server output to see the printed message.

Enabling Server Output

SET SERVEROUTPUT ON;

Complete Example with Server Output

SET SERVEROUTPUT ON;

BEGIN
   DBMS_OUTPUT.PUT_LINE('Hello, World!');
END;
/

Conclusion

In this section, we learned how to write a simple "Hello World" program in PL/SQL. We covered the basic structure of a PL/SQL block and how to use the DBMS_OUTPUT.PUT_LINE procedure to print output. This foundational knowledge will help you as you progress to more complex PL/SQL programming tasks.
