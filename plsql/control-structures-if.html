Introduction

In PL/SQL, control structures are used to dictate the flow of execution within a program. One of the most fundamental control structures is the IF statement, which allows you to execute certain sections of code based on specific conditions.

Key Concepts

  • IF Statement: Executes a block of code if a specified condition is true.
  • ELSIF Statement: Provides additional conditions if the initial IF condition is false.
  • ELSE Statement: Executes a block of code if none of the preceding conditions are true.
  • Boolean Expressions: Conditions used within IF statements that evaluate to TRUE, FALSE, or NULL.

Basic IF Statement

The simplest form of an IF statement checks a single condition and executes a block of code if the condition is true.

Syntax

IF condition THEN
    -- statements to execute if condition is true
END IF;

Example

DECLARE
    v_salary NUMBER := 5000;
BEGIN
    IF v_salary > 3000 THEN
        DBMS_OUTPUT.PUT_LINE('Salary is greater than 3000');
    END IF;
END;

In this example, the message "Salary is greater than 3000" will be printed because the condition v_salary > 3000 is true.

IF-ELSE Statement

An IF-ELSE statement allows you to execute one block of code if the condition is true and another block if the condition is false.

Syntax

IF condition THEN
    -- statements to execute if condition is true
ELSE
    -- statements to execute if condition is false
END IF;

Example

DECLARE
    v_salary NUMBER := 2000;
BEGIN
    IF v_salary > 3000 THEN
        DBMS_OUTPUT.PUT_LINE('Salary is greater than 3000');
    ELSE
        DBMS_OUTPUT.PUT_LINE('Salary is 3000 or less');
    END IF;
END;

In this example, the message "Salary is 3000 or less" will be printed because the condition v_salary > 3000 is false.

IF-ELSIF-ELSE Statement

The IF-ELSIF-ELSE statement allows you to check multiple conditions sequentially. If one condition is true, its corresponding block of code is executed, and the rest are skipped.

Syntax

IF condition1 THEN
    -- statements to execute if condition1 is true
ELSIF condition2 THEN
    -- statements to execute if condition2 is true
ELSE
    -- statements to execute if none of the above conditions are true
END IF;

Example

DECLARE
    v_salary NUMBER := 4000;
BEGIN
    IF v_salary > 5000 THEN
        DBMS_OUTPUT.PUT_LINE('Salary is greater than 5000');
    ELSIF v_salary > 3000 THEN
        DBMS_OUTPUT.PUT_LINE('Salary is greater than 3000 but less than or equal to 5000');
    ELSE
        DBMS_OUTPUT.PUT_LINE('Salary is 3000 or less');
    END IF;
END;

In this example, the message "Salary is greater than 3000 but less than or equal to 5000" will be printed because the condition v_salary > 3000 is true.

Nested IF Statements

You can nest IF statements within each other to create more complex conditional logic.

Example

DECLARE
    v_salary NUMBER := 4500;
    v_bonus NUMBER := 500;
BEGIN
    IF v_salary > 3000 THEN
        IF v_bonus > 0 THEN
            DBMS_OUTPUT.PUT_LINE('Salary is greater than 3000 and bonus is positive');
        ELSE
            DBMS_OUTPUT.PUT_LINE('Salary is greater than 3000 but no bonus');
        END IF;
    ELSE
        DBMS_OUTPUT.PUT_LINE('Salary is 3000 or less');
    END IF;
END;

In this example, the message "Salary is greater than 3000 and bonus is positive" will be printed because both conditions v_salary > 3000 and v_bonus > 0 are true.

Conclusion

IF statements are a fundamental part of PL/SQL programming, allowing you to control the flow of execution based on conditions. By understanding and using IF, IF-ELSE, IF-ELSIF-ELSE, and nested IF statements, you can create more dynamic and responsive PL/SQL programs. Practice writing these statements with different conditions to become proficient in controlling the logic of your PL/SQL code.
