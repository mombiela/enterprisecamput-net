Introduction

In this section, we will present the project that we will develop throughout the Fundamentals of Programming course. This project will serve as a practical application of the concepts we will learn, allowing you to consolidate your programming knowledge and skills.

Project Objectives

  • Apply basic programming concepts: Variables, control structures, functions, etc.
  • Develop problem-solving skills: Analyze and break down complex problems into manageable parts.
  • Promote good coding practices: Write clean, commented, and well-structured code.
  • Introduce tools and development environments: Get familiar with IDEs, version control systems, etc.

Project Description

The project will consist of developing a task management application. This application will allow users to:

  • Create tasks: Add new tasks with details such as title, description, and due date.
  • List tasks: View all created tasks, with options to filter and sort.
  • Update tasks: Modify the details of an existing task.
  • Delete tasks: Delete tasks that are no longer needed.

Project Requirements

Basic Functionalities

  • User Interface: A simple interface that allows interaction with the mentioned functionalities.
  • Data Persistence: Store tasks in a file or database so they persist between sessions.
  • Data Validation: Ensure that the data entered by the user is valid and consistent.

Technologies and Tools

  • Programming Language: Python
  • Development Environment: Visual Studio Code (VSCode)
  • Version Control: Git and GitHub

Project Structure

Modules and Components

  • main.py: Entry point of the application.
  • task_manager.py: Module containing the logic for managing tasks.
  • ui.py: Module handling the user interface.
  • storage.py: Module responsible for data persistence.

Code Example

Below is an example of what the basic structure of task_manager.py might look like:

# task_manager.py

tasks = []

def add_task(title, description, due_date):
    task = {
        'title': title,
        'description': description,
        'due_date': due_date,
        'completed': False
    }
    tasks.append(task)

def list_tasks():
    for task in tasks:
        print(f"Title: {task['title']}, Due Date: {task['due_date']}, Completed: {task['completed']}")

def update_task(index, title=None, description=None, due_date=None, completed=None):
    if index < len(tasks):
        if title:
            tasks[index]['title'] = title
        if description:
            tasks[index]['description'] = description
        if due_date:
            tasks[index]['due_date'] = due_date
        if completed is not None:
            tasks[index]['completed'] = completed

def delete_task(index):
    if index < len(tasks):
        tasks.pop(index)

Practical Exercise

  1. Create a Task: Implement the add_task function in task_manager.py.
  2. List Tasks: Implement the list_tasks function and test by listing the created tasks.
  3. Update a Task: Implement the update_task function and test by updating an existing task.
  4. Delete a Task: Implement the delete_task function and test by deleting a task.
