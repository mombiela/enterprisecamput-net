Introduction to Arrays

Arrays are one of the most fundamental data structures in PHP. They allow you to store multiple values in a single variable, making it easier to manage and manipulate collections of data.

Key Concepts

  • Indexed Arrays: Arrays with a numeric index.
  • Associative Arrays: Arrays with named keys.
  • Multidimensional Arrays: Arrays containing one or more arrays.

Creating Arrays

Indexed Arrays

Indexed arrays use numeric keys to store and access values.

// Creating an indexed array
$fruits = array("Apple", "Banana", "Cherry");

// Accessing elements
echo $fruits[0]; // Outputs: Apple

Associative Arrays

Associative arrays use named keys to store and access values.

// Creating an associative array
$ages = array("Peter" => 35, "Ben" => 37, "Joe" => 43);

// Accessing elements
echo $ages["Peter"]; // Outputs: 35

Multidimensional Arrays

Multidimensional arrays contain one or more arrays.

// Creating a multidimensional array
$contacts = array(
    "John" => array("phone" => "123-4567", "email" => "john@example.com"),
    "Jane" => array("phone" => "987-6543", "email" => "jane@example.com")
);

// Accessing elements
echo $contacts["John"]["phone"]; // Outputs: 123-4567

Array Functions

PHP provides a wide range of built-in functions to work with arrays.

Common Array Functions

  • count(): Counts the number of elements in an array.
  • array_push(): Adds one or more elements to the end of an array.
  • array_pop(): Removes the last element of an array.
  • array_merge(): Merges one or more arrays.
  • array_keys(): Returns all the keys of an array.
  • array_values(): Returns all the values of an array.
// Example usage of array functions
$numbers = array(1, 2, 3);
array_push($numbers, 4); // Adds 4 to the end of the array
print_r($numbers); // Outputs: Array ( [0] => 1 [1] => 2 [2] => 3 [3] => 4 )

$last = array_pop($numbers); // Removes the last element
echo $last; // Outputs: 4

$merged = array_merge($numbers, array(5, 6));
print_r($merged); // Outputs: Array ( [0] => 1 [1] => 2 [2] => 3 [3] => 5 [4] => 6 )

Iterating Over Arrays

Using `foreach` Loop

The foreach loop is the most common way to iterate over arrays.

// Iterating over an indexed array
$fruits = array("Apple", "Banana", "Cherry");
foreach ($fruits as $fruit) {
    echo $fruit . "\n";
}

// Iterating over an associative array
$ages = array("Peter" => 35, "Ben" => 37, "Joe" => 43);
foreach ($ages as $name => $age) {
    echo "$name is $age years old.\n";
}

Using `for` Loop

You can also use a for loop to iterate over indexed arrays.

$fruits = array("Apple", "Banana", "Cherry");
for ($i = 0; $i < count($fruits); $i++) {
    echo $fruits[$i] . "\n";
}

Advanced Array Techniques

Sorting Arrays

PHP provides several functions to sort arrays.

  • sort(): Sorts indexed arrays in ascending order.
  • rsort(): Sorts indexed arrays in descending order.
  • asort(): Sorts associative arrays in ascending order, according to the value.
  • ksort(): Sorts associative arrays in ascending order, according to the key.
// Sorting an indexed array
$numbers = array(4, 2, 8, 6);
sort($numbers);
print_r($numbers); // Outputs: Array ( [0] => 2 [1] => 4 [2] => 6 [3] => 8 )

// Sorting an associative array by value
$ages = array("Peter" => 35, "Ben" => 37, "Joe" => 43);
asort($ages);
print_r($ages); // Outputs: Array ( [Peter] => 35 [Ben] => 37 [Joe] => 43 )

Filtering Arrays

You can filter arrays using the array_filter() function.

// Filtering an array
$numbers = array(1, 2, 3, 4, 5, 6);
$even = array_filter($numbers, function($num) {
    return $num % 2 == 0;
});
print_r($even); // Outputs: Array ( [1] => 2 [3] => 4 [5] => 6 )

Conclusion

Arrays are a powerful and versatile data structure in PHP. They allow you to store and manipulate collections of data efficiently. By understanding how to create, access, and manipulate arrays, you can handle a wide range of programming tasks more effectively. Practice using arrays and their associated functions to become proficient in managing data in your PHP applications.
