Introduction to PHP

PHP (Hypertext Preprocessor) is a popular server-side scripting language designed for web development but also used as a general-purpose programming language. This section will cover the fundamental syntax and basic constructs of PHP to get you started with writing PHP scripts.

Basic PHP Syntax

  • PHP Tags: PHP code is embedded within HTML using PHP tags. 
    <?php
// PHP code goes here
?>
  • Echo Statement: Used to output text or variables. 
    <?php
echo "Hello, World!";
?>

Variables and Data Types

  • Variables: Declared using the $ symbol followed by the variable name. 
    <?php
$greeting = "Hello, World!";
echo $greeting;
?>
  • Data Types:
    • String
    • Integer
    • Float (Double)
    • Boolean
    • Array
    • Object
    • NULL

Example of Different Data Types

<?php
$string = "Hello, World!";
$integer = 42;
$float = 3.14;
$boolean = true;
$array = array("apple", "banana", "cherry");
$object = (object) ['property' => 'value'];
$null = NULL;

echo $string;
echo $integer;
echo $float;
echo $boolean;
print_r($array);
echo $object->property;
echo $null;
?>

Operators

  • Arithmetic Operators: +, -, *, /, % 
    <?php
$sum = 5 + 3; // 8
$difference = 5 - 3; // 2
$product = 5 * 3; // 15
$quotient = 5 / 3; // 1.6667
$remainder = 5 % 3; // 2
?>
  • Assignment Operators: =, +=, -=, *=, /=, %= 
    <?php
$x = 10;
$x += 5; // 15
$x -= 3; // 12
$x *= 2; // 24
$x /= 4; // 6
$x %= 5; // 1
?>
  • Comparison Operators: ==, ===, !=, !==, >, <, >=, <= 
    <?php
$a = 5;
$b = 10;
var_dump($a == $b); // false
var_dump($a === $b); // false
var_dump($a != $b); // true
var_dump($a !== $b); // true
var_dump($a > $b); // false
var_dump($a < $b); // true
var_dump($a >= $b); // false
var_dump($a <= $b); // true
?>

Control Structures

  • If-Else Statement: Conditional execution of code blocks. 
    <?php
$age = 20;
if ($age >= 18) {
    echo "You are an adult.";
} else {
    echo "You are a minor.";
}
?>
  • Switch Statement: Selects one of many blocks of code to be executed. 
    <?php
$day = "Monday";
switch ($day) {
    case "Monday":
        echo "Start of the work week.";
        break;
    case "Friday":
        echo "End of the work week.";
        break;
    default:
        echo "Midweek days.";
}
?>

Loops

  • While Loop: Repeats a block of code as long as the specified condition is true. 
    <?php
$i = 0;
while ($i < 5) {
    echo $i;
    $i++;
}
?>
  • For Loop: Loops through a block of code a specified number of times. 
    <?php
for ($i = 0; $i < 5; $i++) {
    echo $i;
}
?>
  • Foreach Loop: Loops through each key/value pair in an array. 
    <?php
$fruits = array("apple", "banana", "cherry");
foreach ($fruits as $fruit) {
    echo $fruit;
}
?>

Functions

  • Defining Functions: Functions are blocks of code that can be repeatedly called. 
    <?php
function greet($name) {
    return "Hello, " . $name;
}
echo greet("Alice");
?>

Conclusion

In this section, we covered the basic syntax and constructs of PHP, including variables, data types, operators, control structures, loops, and functions. These fundamentals form the building blocks for more advanced PHP programming. Practice these concepts with the provided examples to solidify your understanding before moving on to more complex topics.
