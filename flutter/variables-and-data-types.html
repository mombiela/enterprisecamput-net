Introduction

In this section, we will cover the fundamental concepts of variables and data types in Flutter. Understanding these basics is crucial for building robust and efficient Flutter applications. We will start with the basics of variables and then move on to different data types available in Dart, the programming language used in Flutter.

Variables

Variables are used to store data that can be referenced and manipulated in a program. In Dart, variables are declared using the var, final, or const keywords.

Declaring Variables

  • var: The type of the variable is inferred by the compiler.
  • final: A variable whose value cannot be changed once it is set.
  • const: A compile-time constant.
void main() {
  var name = 'Flutter'; // String type inferred
  final age = 5; // int type inferred, cannot be changed
  const pi = 3.14; // double type, compile-time constant

  print(name); // Output: Flutter
  print(age); // Output: 5
  print(pi); // Output: 3.14
}

Data Types

Dart supports various data types, which can be broadly categorized into the following:

  • Numbers
  • Strings
  • Booleans
  • Lists
  • Maps

Numbers

Dart has two types of numbers: int and double.

void main() {
  int age = 30;
  double height = 5.9;

  print(age); // Output: 30
  print(height); // Output: 5.9
}

Strings

Strings are a sequence of characters. They can be created using single or double quotes.

void main() {
  String greeting = 'Hello, Flutter!';
  String name = "John Doe";

  print(greeting); // Output: Hello, Flutter!
  print(name); // Output: John Doe
}

Booleans

Booleans represent true or false values.

void main() {
  bool isFlutterAwesome = true;
  bool isLearningFun = true;

  print(isFlutterAwesome); // Output: true
  print(isLearningFun); // Output: true
}

Lists

Lists are ordered collections of objects. They are similar to arrays in other programming languages.

void main() {
  List<String> fruits = ['Apple', 'Banana', 'Mango'];
  
  print(fruits); // Output: [Apple, Banana, Mango]
  print(fruits[0]); // Output: Apple
}

Maps

Maps are collections of key-value pairs. Each key in a map should be unique.

void main() {
  Map<String, String> capitals = {
    'USA': 'Washington, D.C.',
    'India': 'New Delhi',
    'Japan': 'Tokyo'
  };

  print(capitals); // Output: {USA: Washington, D.C., India: New Delhi, Japan: Tokyo}
  print(capitals['India']); // Output: New Delhi
}

Summary

In this section, we covered the basics of variables and data types in Flutter. We learned how to declare variables using var, final, and const. We also explored different data types available in Dart, including numbers, strings, booleans, lists, and maps. Understanding these fundamentals is essential for building any Flutter application. In the next section, we will dive deeper into more advanced topics and how to effectively use these data types in real-world applications.
