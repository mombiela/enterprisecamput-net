Introduction

Collision detection is an essential component in video game physics. It allows determining when two or more objects in a virtual environment touch or interact in some way. This process is crucial for gameplay and realistic physics in video games.

Key Concepts

  • Collision: Occurs when two objects occupy the same space at a given moment.
  • Bounding Box: A rectangular box that encloses an object, used to simplify collision detection.
  • Bounding Sphere: A sphere that encloses an object, another simplification for collision detection.
  • Separating Axis Theorem (SAT): A method to determine if two convex polygons are colliding.
  • Broad Phase and Narrow Phase: Two stages in collision detection. The broad phase reduces the number of possible collisions, and the narrow phase performs more precise detection.

Types of Collisions

AABB (Axis-Aligned Bounding Box) Collisions

AABB collisions are one of the simplest and fastest forms of collision detection. They use rectangular boxes aligned with the coordinate system axes.

class AABB:
    def __init__(self, x, y, width, height):
        self.x = x
        self.y = y
        self.width = width
        self.height = height

def check_collision(a, b):
    return (a.x < b.x + b.width and
            a.x + a.width > b.x and
            a.y < b.y + b.height and
            a.y + a.height > b.y)

# Example usage
a = AABB(0, 0, 10, 10)
b = AABB(5, 5, 10, 10)
print(check_collision(a, b))  # Returns True

Sphere Collisions

Sphere collisions are useful when objects have a more circular shape. Detection is based on the distance between the centers of the spheres.

import math

class Sphere:
    def __init__(self, x, y, radius):
        self.x = x
        self.y = y
        self.radius = radius

def check_collision(s1, s2):
    distance = math.sqrt((s1.x - s2.x)**2 + (s1.y - s2.y)**2)
    return distance < (s1.radius + s2.radius)

# Example usage
s1 = Sphere(0, 0, 5)
s2 = Sphere(7, 0, 5)
print(check_collision(s1, s2))  # Returns True

Separating Axis Theorem (SAT)

The SAT is more complex but very effective for convex polygons. It is based on projecting the vertices of the polygons onto an axis and checking for separation.

# Simplified example of SAT in pseudocode
def sat_collision(poly1, poly2):
    for axis in poly1.get_axes() + poly2.get_axes():
        projection1 = poly1.project(axis)
        projection2 = poly2.project(axis)
        if not projection1.overlaps(projection2):
            return False
    return True

# poly1 and poly2 are objects that have get_axes() and project(axis) methods

Broad Phase and Narrow Phase

Broad Phase

The broad phase reduces the number of possible collisions using methods like sweep and prune or spatial grids.

Narrow Phase

The narrow phase performs more precise collision detection between the pairs of objects that passed the broad phase.

Conclusion

Collision detection is fundamental for video game physics, providing the basis for realistic interactions between objects. From simple methods like AABB and sphere collisions to more advanced techniques like SAT, each method has its advantages and disadvantages. Understanding and applying these concepts is crucial for any video game developer.
