Parameter Passing in C
Recursion ►

Introduction

Parameter passing is a fundamental concept in C programming that allows functions to receive input values and return results. Understanding how parameters are passed to functions is crucial for writing efficient and effective C code.

Types of Parameter Passing

In C, there are two primary ways to pass parameters to functions:

  1. Pass by Value
  2. Pass by Reference

Pass by Value

When parameters are passed by value, a copy of the actual parameter's value is made in memory. This means that changes made to the parameter inside the function do not affect the original value.

Example:

#include <stdio.h>

void addTen(int num) {
    num = num + 10;
    printf("Value inside function: %d\n", num);
}

int main() {
    int number = 5;
    addTen(number);
    printf("Value outside function: %d\n", number);
    return 0;
}

Explanation:

  • number is passed to addTen by value.
  • Inside addTen, num is a copy of number.
  • Changes to num do not affect number.

Pass by Reference

When parameters are passed by reference, the function receives a reference (or pointer) to the actual parameter. This allows the function to modify the original value.

Example:

#include <stdio.h>

void addTen(int *num) {
    *num = *num + 10;
    printf("Value inside function: %d\n", *num);
}

int main() {
    int number = 5;
    addTen(&number);
    printf("Value outside function: %d\n", number);
    return 0;
}

Explanation:

  • number is passed to addTen by reference using its address (&number).
  • Inside addTen, num is a pointer to number.
  • Changes to *num affect number.

Comparison of Pass by Value and Pass by Reference

| Feature | Pass by Value | Pass by Reference | |------------------------|-----------------------------------|-------------------------------------| | Parameter Type | Copy of the actual value | Address of the actual value | | Memory Usage | More (due to copying) | Less (no copying, just referencing) | | Effect on Original | No effect | Can modify the original value | | Use Case | When original data should remain unchanged | When original data needs to be modified |

Practical Examples

Swapping Values

Pass by Value (Incorrect):

#include <stdio.h>

void swap(int a, int b) {
    int temp = a;
    a = b;
    b = temp;
}

int main() {
    int x = 10, y = 20;
    swap(x, y);
    printf("x: %d, y: %d\n", x, y); // Output will be x: 10, y: 20
    return 0;
}

Pass by Reference (Correct):

#include <stdio.h>

void swap(int *a, int *b) {
    int temp = *a;
    *a = *b;
    *b = temp;
}

int main() {
    int x = 10, y = 20;
    swap(&x, &y);
    printf("x: %d, y: %d\n", x, y); // Output will be x: 20, y: 10
    return 0;
}

Conclusion

Understanding parameter passing is essential for effective C programming. By mastering both pass by value and pass by reference, you can control how data is manipulated within your functions, leading to more efficient and predictable code. Remember:

  • Use pass by value when you do not want the function to modify the original data.
  • Use pass by reference when you need the function to modify the original data.
