Introduction to Linked Lists

Linked lists are a fundamental data structure in computer science, providing a way to store a collection of elements. Unlike arrays, linked lists do not require contiguous memory locations, making them flexible and efficient for certain operations.

Key Concepts

  • Node: The basic unit of a linked list, containing data and a reference (or pointer) to the next node.
  • Head: The first node in a linked list.
  • Tail: The last node in a linked list, which points to NULL.
  • NULL: A special value indicating the end of the list.

Types of Linked Lists

Singly Linked List

A singly linked list is the simplest type of linked list, where each node points to the next node in the sequence.

Doubly Linked List

A doubly linked list allows traversal in both directions, as each node contains pointers to both the next and the previous nodes.

Circular Linked List

In a circular linked list, the last node points back to the first node, forming a circular structure.

Implementing a Singly Linked List in C

Defining the Node Structure

#include <stdio.h>
#include <stdlib.h>

// Define the node structure
struct Node {
    int data;
    struct Node* next;
};
  • data: Stores the value of the node.
  • next: Pointer to the next node in the list.

Creating a New Node

struct Node* createNode(int data) {
    struct Node* newNode = (struct Node*)malloc(sizeof(struct Node));
    newNode->data = data;
    newNode->next = NULL;
    return newNode;
}
  • malloc: Allocates memory for the new node.
  • data: Initializes the node's data.
  • next: Sets the next pointer to NULL.

Inserting a Node at the Beginning

void insertAtBeginning(struct Node** head, int data) {
    struct Node* newNode = createNode(data);
    newNode->next = *head;
    *head = newNode;
}
  • head: Double pointer to the head node to allow modification.
  • newNode->next: Points to the current head.
  • *head: Updates the head to the new node.

Traversing the List

void traverseList(struct Node* head) {
    struct Node* current = head;
    while (current != NULL) {
        printf("%d -> ", current->data);
        current = current->next;
    }
    printf("NULL\n");
}
  • current: Pointer used to traverse the list.
  • printf: Prints each node's data followed by an arrow.

Deleting a Node

void deleteNode(struct Node** head, int key) {
    struct Node* temp = *head;
    struct Node* prev = NULL;

    // If head node itself holds the key
    if (temp != NULL && temp->data == key) {
        *head = temp->next;
        free(temp);
        return;
    }

    // Search for the key
    while (temp != NULL && temp->data != key) {
        prev = temp;
        temp = temp->next;
    }

    // If key was not present in the list
    if (temp == NULL) return;

    // Unlink the node and free memory
    prev->next = temp->next;
    free(temp);
}
  • temp: Temporary pointer to traverse the list.
  • prev: Pointer to the previous node to update the link.
  • free: Deallocates memory of the deleted node.

Advanced Operations on Linked Lists

Reversing a Linked List

void reverseList(struct Node** head) {
    struct Node* prev = NULL;
    struct Node* current = *head;
    struct Node* next = NULL;

    while (current != NULL) {
        next = current->next;
        current->next = prev;
        prev = current;
        current = next;
    }
    *head = prev;
}
  • prev: Tracks the previous node.
  • current: Tracks the current node.
  • next: Temporarily stores the next node.

Detecting a Cycle in a Linked List

int detectCycle(struct Node* head) {
    struct Node* slow = head;
    struct Node* fast = head;

    while (fast != NULL && fast->next != NULL) {
        slow = slow->next;
        fast = fast->next->next;

        if (slow == fast) {
            return 1; // Cycle detected
        }
    }
    return 0; // No cycle
}
  • slow: Moves one step at a time.
  • fast: Moves two steps at a time.
  • Cycle detection: If slow and fast meet, a cycle exists.

Conclusion

Linked lists are a versatile data structure that can be used to efficiently manage collections of data. Understanding the basic operations, such as insertion, deletion, traversal, and advanced operations like reversing and cycle detection, is crucial for leveraging linked lists in various applications. By mastering these concepts, programmers can enhance their problem-solving skills and optimize their code for better performance.
