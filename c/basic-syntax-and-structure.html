Introduction

Understanding the basic syntax and structure of C is crucial for writing efficient and error-free programs. This section will cover the fundamental elements of C programming, including the structure of a C program, data types, variables, and basic input/output operations.

Structure of a C Program

A C program typically consists of the following parts:

  • Preprocessor Commands
  • Functions
  • Variables
  • Statements & Expressions
  • Comments

Example of a Simple C Program

#include <stdio.h> // Preprocessor Command

// Main Function
int main() {
    // Variable Declaration
    int number;

    // Print Statement
    printf("Enter a number: ");

    // Input Statement
    scanf("%d", &number);

    // Output Statement
    printf("You entered: %d\n", number);

    return 0; // Return Statement
}

Explanation:

  • #include <stdio.h>: This is a preprocessor command that tells the compiler to include the standard input-output library before compiling the program.
  • int main() { ... }: This is the main function where the execution of the program begins.
  • int number;: This declares an integer variable named number.
  • printf("Enter a number: ");: This prints the message "Enter a number:" to the console.
  • scanf("%d", &number);: This reads an integer input from the user and stores it in the variable number.
  • printf("You entered: %d\n", number);: This prints the message "You entered:" followed by the value of number.
  • return 0;: This statement terminates the main function and returns the value 0 to the calling process.

Data Types and Variables

In C, data types specify the type of data that a variable can hold. Common data types include:

  • int: Integer type
  • float: Floating-point type
  • char: Character type
  • double: Double-precision floating-point type

Declaring Variables

Variables must be declared before they are used. The syntax for declaring a variable is:

data_type variable_name;

Example:

int age;
float salary;
char grade;
double distance;

Basic Input/Output Operations

C provides standard functions for input and output operations, primarily through the stdio.h library.

Input using `scanf`

The scanf function is used to read formatted input from the standard input (keyboard).

int number;
scanf("%d", &number); // Reads an integer and stores it in the variable 'number'

Output using `printf`

The printf function is used to print formatted output to the standard output (console).

int number = 10;
printf("The number is: %d\n", number); // Prints "The number is: 10"

Comments

Comments are used to explain code and are ignored by the compiler. There are two types of comments in C:

  • Single-line comments: Start with //
  • Multi-line comments: Enclosed within /* ... */

Example:

// This is a single-line comment

/*
This is a
multi-line comment
*/

Summary

In this section, we covered the basic syntax and structure of a C program. We learned about the essential components such as preprocessor commands, functions, variables, and statements. We also explored basic input/output operations using scanf and printf, and the use of comments to make the code more readable. Understanding these fundamentals is the first step towards becoming proficient in C programming.
