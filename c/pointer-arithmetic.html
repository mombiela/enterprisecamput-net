Pointer Arithmetic
Pointers and Arrays ►

Pointer arithmetic is a powerful feature in C that allows you to perform operations on pointers to navigate through memory. Understanding pointer arithmetic is crucial for working with arrays, dynamic memory, and various data structures.

Key Concepts

  • Pointers and Memory Addresses: Pointers store memory addresses of variables. Pointer arithmetic involves manipulating these addresses.
  • Pointer Types: The type of a pointer determines the size of the steps taken during arithmetic operations.
  • Increment and Decrement: You can increment or decrement pointers to move to the next or previous memory location.
  • Addition and Subtraction: You can add or subtract integers to/from pointers to navigate through memory.

Increment and Decrement

When you increment or decrement a pointer, it moves to the next or previous memory location based on the type it points to.

Example: Incrementing a Pointer

#include <stdio.h>

int main() {
    int arr[3] = {10, 20, 30};
    int *ptr = arr; // ptr points to the first element of arr

    printf("Initial pointer address: %p\n", ptr);
    printf("Initial value: %d\n", *ptr);

    ptr++; // Move to the next integer in the array

    printf("Pointer address after increment: %p\n", ptr);
    printf("Value after increment: %d\n", *ptr);

    return 0;
}

Explanation:

  • ptr initially points to arr[0].
  • After ptr++, it points to arr[1].

Addition and Subtraction

You can add or subtract integers to/from pointers to move multiple steps at once.

Example: Pointer Addition

#include <stdio.h>

int main() {
    int arr[3] = {10, 20, 30};
    int *ptr = arr; // ptr points to the first element of arr

    printf("Initial pointer address: %p\n", ptr);
    printf("Initial value: %d\n", *ptr);

    ptr += 2; // Move two integers ahead

    printf("Pointer address after addition: %p\n", ptr);
    printf("Value after addition: %d\n", *ptr);

    return 0;
}

Explanation:

  • ptr initially points to arr[0].
  • After ptr += 2, it points to arr[2].

Pointer Subtraction

Pointer subtraction can be used to find the distance between two pointers.

Example: Pointer Subtraction

#include <stdio.h>

int main() {
    int arr[3] = {10, 20, 30};
    int *ptr1 = &arr[0];
    int *ptr2 = &arr[2];

    printf("Pointer 1 address: %p\n", ptr1);
    printf("Pointer 2 address: %p\n", ptr2);

    int distance = ptr2 - ptr1; // Calculate the distance between ptr2 and ptr1

    printf("Distance between pointers: %d\n", distance);

    return 0;
}

Explanation:

  • ptr1 points to arr[0].
  • ptr2 points to arr[2].
  • The distance between ptr2 and ptr1 is 2.

Pointer Arithmetic with Different Types

Pointer arithmetic behaves differently with different data types because the size of the type affects the step size.

Example: Pointer Arithmetic with `char` and `int`

#include <stdio.h>

int main() {
    char charArr[3] = {'a', 'b', 'c'};
    int intArr[3] = {10, 20, 30};

    char *charPtr = charArr;
    int *intPtr = intArr;

    printf("Initial char pointer address: %p\n", charPtr);
    printf("Initial int pointer address: %p\n", intPtr);

    charPtr++;
    intPtr++;

    printf("Char pointer address after increment: %p\n", charPtr);
    printf("Int pointer address after increment: %p\n", intPtr);

    return 0;
}

Explanation:

  • charPtr++ moves by 1 byte (size of char).
  • intPtr++ moves by 4 bytes (size of int).

Summary

Pointer arithmetic is a fundamental concept in C that allows you to navigate through memory efficiently. By understanding how to increment, decrement, add, and subtract pointers, you can manipulate arrays and other data structures more effectively. Remember that the type of the pointer determines the size of the steps taken during arithmetic operations. Practice these concepts with different data types to gain a deeper understanding.
