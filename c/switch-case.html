Introduction to Switch Case

The switch statement in C is a control statement that allows a variable to be tested for equality against a list of values. Each value is called a case, and the variable being switched on is checked for each case.

Key Concepts

  • Switch Statement: A control statement that tests a variable against multiple cases.
  • Case: A possible value for the variable being tested.
  • Break Statement: Used to terminate a case in the switch statement.
  • Default Case: An optional case that executes if none of the specified cases match the variable.

Syntax of Switch Case

The basic syntax of a switch case statement in C is as follows:

switch (expression) {
    case constant1:
        // code to be executed if expression equals constant1
        break;
    case constant2:
        // code to be executed if expression equals constant2
        break;
    // you can have any number of case statements
    default:
        // code to be executed if expression doesn't match any case
}

Example

Here is a simple example to demonstrate the use of the switch case statement:

#include <stdio.h>

int main() {
    int day = 3;

    switch (day) {
        case 1:
            printf("Monday\n");
            break;
        case 2:
            printf("Tuesday\n");
            break;
        case 3:
            printf("Wednesday\n");
            break;
        case 4:
            printf("Thursday\n");
            break;
        case 5:
            printf("Friday\n");
            break;
        case 6:
            printf("Saturday\n");
            break;
        case 7:
            printf("Sunday\n");
            break;
        default:
            printf("Invalid day\n");
    }

    return 0;
}

Explanation

  • The switch statement evaluates the expression day.
  • It then compares day to each case.
  • When day matches a case (in this example, day is 3, so it matches case 3), the corresponding block of code is executed.
  • The break statement prevents the execution from falling through to the next case.
  • If no case matches, the default case is executed.

Advanced Usage

Nested Switch Case

You can nest switch statements within each other. This is useful for more complex decision-making processes.

#include <stdio.h>

int main() {
    char grade = 'B';

    switch (grade) {
        case 'A':
        case 'B':
            printf("Good job!\n");
            switch (grade) {
                case 'A':
                    printf("Excellent!\n");
                    break;
                case 'B':
                    printf("Well done!\n");
                    break;
            }
            break;
        case 'C':
            printf("You passed.\n");
            break;
        default:
            printf("Better luck next time.\n");
    }

    return 0;
}

Fall-Through Behavior

In C, if you omit the break statement, the program continues to execute the next case. This is known as fall-through behavior.

#include <stdio.h>

int main() {
    int num = 2;

    switch (num) {
        case 1:
            printf("One\n");
        case 2:
            printf("Two\n");
        case 3:
            printf("Three\n");
        default:
            printf("Default\n");
    }

    return 0;
}

Explanation

  • In this example, since there is no break statement after case 2, the program will continue to execute case 3 and the default case, resulting in the output: 
    Two
Three
Default

Summary

  • The switch statement is a powerful tool for handling multiple conditions based on the value of a single variable.
  • Each case represents a possible value for the variable.
  • The break statement is crucial to prevent fall-through behavior.
  • The default case is optional but useful for handling unexpected values.
  • Advanced usage includes nested switch statements and intentional fall-through for specific logic.

By understanding and utilizing the switch case statement, you can write more readable and maintainable code for complex decision-making scenarios.
