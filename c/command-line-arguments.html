Introduction

Command line arguments are a way to provide input to a program from the command line. They allow users to specify parameters or options when running a program, making it more flexible and dynamic.

Key Concepts

  • Command Line Interface (CLI): A text-based interface used to interact with software and operating systems.
  • Arguments: Values provided to the program at runtime via the command line.
  • argc (Argument Count): An integer representing the number of command line arguments.
  • argv (Argument Vector): An array of character pointers listing all the arguments.

Understanding Command Line Arguments

In C, command line arguments are handled using two parameters of the main function:

int main(int argc, char *argv[]) {
    // Your code here
}
  • argc: This parameter (argument count) holds the number of arguments passed to the program, including the program's name.
  • argv: This parameter (argument vector) is an array of strings (character pointers) representing the arguments.

Example: Basic Usage

Let's look at a simple example to understand how command line arguments work:

#include <stdio.h>

int main(int argc, char *argv[]) {
    printf("Number of arguments: %d\n", argc);
    for (int i = 0; i < argc; i++) {
        printf("Argument %d: %s\n", i, argv[i]);
    }
    return 0;
}

Explanation

  • The program prints the number of arguments (argc).
  • It then iterates through the argv array and prints each argument.

Running the Program

To run the program with command line arguments, compile it and execute it from the terminal:

gcc main.c -o main
./main arg1 arg2 arg3

Output

Number of arguments: 4
Argument 0: ./main
Argument 1: arg1
Argument 2: arg2
Argument 3: arg3

Advanced Usage

Parsing Arguments

In more complex programs, you might need to parse and validate the command line arguments. Here's an example that expects two numerical arguments and calculates their sum:

#include <stdio.h>
#include <stdlib.h>

int main(int argc, char *argv[]) {
    if (argc != 3) {
        printf("Usage: %s <num1> <num2>\n", argv[0]);
        return 1;
    }

    int num1 = atoi(argv[1]);
    int num2 = atoi(argv[2]);
    int sum = num1 + num2;

    printf("Sum: %d\n", sum);
    return 0;
}

Explanation

  • The program checks if the number of arguments is exactly 3 (including the program name).
  • It converts the arguments from strings to integers using atoi.
  • It calculates the sum and prints it.

Running the Program

gcc main.c -o main
./main 5 10

Output

Sum: 15

Common Pitfalls

  • Incorrect Argument Count: Always validate the number of arguments to avoid unexpected behavior.
  • Invalid Data Types: Ensure that the arguments are of the expected type (e.g., converting strings to integers).

Conclusion

Command line arguments are a powerful feature in C that allow you to make your programs more flexible and interactive. By understanding how to use argc and argv, you can create programs that accept user input directly from the command line, enhancing their usability and functionality. Practice by writing programs that take different types of input and perform various operations based on those inputs.
