What is Programming?

Programming is the process of creating a set of instructions that tell a computer how to perform a task. These instructions are written in a programming language. C is one of the most widely used programming languages due to its efficiency and control over system resources.

Why Learn C?

  • Foundation for Other Languages: Many modern languages like C++, Java, and Python have their roots in C.
  • System-Level Programming: C is used for system programming, including operating systems and embedded systems.
  • Performance: C provides low-level access to memory and system processes, making it highly efficient.

Basic Concepts in C

Variables and Data Types

Variables are used to store data that can be manipulated by the program. Each variable in C must be declared with a specific data type.

  • int: Integer numbers
  • float: Floating-point numbers
  • char: Single characters
#include <stdio.h>

int main() {
    int age = 25;
    float height = 5.9;
    char initial = 'A';

    printf("Age: %d\n", age);
    printf("Height: %.1f\n", height);
    printf("Initial: %c\n", initial);

    return 0;
}

Operators

Operators are symbols that perform operations on variables and values.

  • Arithmetic Operators: +, -, *, /, %
  • Relational Operators: ==, !=, >, <, >=, <=
  • Logical Operators: &&, ||, !
#include <stdio.h>

int main() {
    int a = 10, b = 20;
    int sum = a + b;
    int isEqual = (a == b);

    printf("Sum: %d\n", sum);
    printf("Are they equal? %d\n", isEqual);

    return 0;
}

Control Structures

Control structures are used to control the flow of execution in a program.

  • if-else: Conditional statements
  • for: Loop with a known number of iterations
  • while: Loop with an unknown number of iterations
  • do-while: Loop that executes at least once
#include <stdio.h>

int main() {
    int number = 10;

    // if-else statement
    if (number > 0) {
        printf("Number is positive\n");
    } else {
        printf("Number is not positive\n");
    }

    // for loop
    for (int i = 0; i < 5; i++) {
        printf("i: %d\n", i);
    }

    // while loop
    int count = 0;
    while (count < 3) {
        printf("Count: %d\n", count);
        count++;
    }

    // do-while loop
    int j = 0;
    do {
        printf("j: %d\n", j);
        j++;
    } while (j < 2);

    return 0;
}

Functions

Functions are blocks of code that perform a specific task and can be reused.

  • Declaration: Specifies the function's name, return type, and parameters.
  • Definition: Contains the actual code to be executed.
  • Call: Executes the function.
#include <stdio.h>

// Function declaration
void greet();

// Main function
int main() {
    greet(); // Function call
    return 0;
}

// Function definition
void greet() {
    printf("Hello, World!\n");
}

Conclusion

In this introductory topic, we've covered the basics of programming in C, including variables, data types, operators, control structures, and functions. These fundamental concepts are the building blocks for more advanced topics in C programming. Understanding these basics will provide a strong foundation for your journey into more complex and powerful programming techniques.
