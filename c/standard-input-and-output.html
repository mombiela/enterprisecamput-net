Standard Input and Output in C
String Functions ►

Introduction

Standard Input and Output (I/O) are fundamental concepts in C programming. They allow programs to interact with users and other programs by reading input and writing output. This section will cover the basics of standard I/O, including functions like printf, scanf, getchar, and putchar.

Basic Output with `printf`

The printf function is used to output text to the console. It is part of the standard I/O library, which is included by adding #include <stdio.h> at the beginning of your program.

Syntax

#include <stdio.h>

int main() {
    printf("Hello, World!\n");
    return 0;
}
  • #include <stdio.h>: Includes the standard I/O library.
  • printf("Hello, World!\n");: Prints "Hello, World!" followed by a newline character.

Format Specifiers

printf uses format specifiers to print different types of data:

  • %d: Integer
  • %f: Floating-point number
  • %c: Character
  • %s: String

Example

#include <stdio.h>

int main() {
    int age = 25;
    float height = 5.9;
    char initial = 'A';
    char name[] = "Alice";

    printf("Name: %s\n", name);
    printf("Initial: %c\n", initial);
    printf("Age: %d\n", age);
    printf("Height: %.1f\n", height);

    return 0;
}

Basic Input with `scanf`

The scanf function is used to read formatted input from the standard input (usually the keyboard).

Syntax

#include <stdio.h>

int main() {
    int age;
    printf("Enter your age: ");
    scanf("%d", &age);
    printf("Your age is %d\n", age);
    return 0;
}
  • scanf("%d", &age);: Reads an integer from the user and stores it in the variable age.

Format Specifiers

scanf uses the same format specifiers as printf to read different types of data:

  • %d: Integer
  • %f: Floating-point number
  • %c: Character
  • %s: String

Example

#include <stdio.h>

int main() {
    int age;
    float height;
    char initial;
    char name[50];

    printf("Enter your name: ");
    scanf("%s", name);
    printf("Enter your initial: ");
    scanf(" %c", &initial); // Note the space before %c to consume any leftover newline character
    printf("Enter your age: ");
    scanf("%d", &age);
    printf("Enter your height: ");
    scanf("%f", &height);

    printf("Name: %s\n", name);
    printf("Initial: %c\n", initial);
    printf("Age: %d\n", age);
    printf("Height: %.2f\n", height);

    return 0;
}

Character Input and Output

For single character input and output, getchar and putchar functions are used.

`getchar` Example

#include <stdio.h>

int main() {
    char ch;
    printf("Enter a character: ");
    ch = getchar();
    printf("You entered: %c\n", ch);
    return 0;
}

`putchar` Example

#include <stdio.h>

int main() {
    char ch = 'A';
    printf("The character is: ");
    putchar(ch);
    putchar('\n');
    return 0;
}

Advanced I/O Functions

For more advanced I/O operations, C provides functions like fgets and fputs for string input and output.

`fgets` Example

#include <stdio.h>

int main() {
    char str[50];
    printf("Enter a string: ");
    fgets(str, sizeof(str), stdin);
    printf("You entered: %s", str);
    return 0;
}

`fputs` Example

#include <stdio.h>

int main() {
    char str[] = "Hello, World!";
    printf("The string is: ");
    fputs(str, stdout);
    putchar('\n');
    return 0;
}

Summary

  • printf: Used for formatted output.
  • scanf: Used for formatted input.
  • getchar: Reads a single character from standard input.
  • putchar: Writes a single character to standard output.
  • fgets: Reads a string from standard input.
  • fputs: Writes a string to standard output.

Understanding these basic I/O functions is crucial for interacting with users and other programs. Practice using these functions to become proficient in handling input and output in C.
