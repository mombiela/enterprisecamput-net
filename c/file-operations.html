File Operations in C
Reading and Writing Files ►

Introduction

File operations are an essential part of programming in C. They allow you to read from and write to files, enabling data persistence and manipulation. This section will cover the basics of file operations, including opening, reading, writing, and closing files.

File Handling Functions

C provides a set of standard library functions for file handling. Here are the key functions you will use:

  • fopen(): Opens a file.
  • fclose(): Closes a file.
  • fread(): Reads data from a file.
  • fwrite(): Writes data to a file.
  • fgets(): Reads a string from a file.
  • fputs(): Writes a string to a file.
  • fprintf(): Writes formatted data to a file.
  • fscanf(): Reads formatted data from a file.

Opening and Closing Files

To perform any operation on a file, you first need to open it using the fopen() function. After completing the operations, you should close the file using the fclose() function.

Example: Opening and Closing a File

#include <stdio.h>

int main() {
    FILE *file;
    file = fopen("example.txt", "r"); // Open file for reading

    if (file == NULL) {
        printf("Error opening file.\n");
        return 1;
    }

    // Perform file operations here

    fclose(file); // Close the file
    return 0;
}

Reading from a File

There are multiple ways to read data from a file in C. The most common functions are fread(), fgets(), and fscanf().

Example: Reading a File Using `fgets()`

#include <stdio.h>

int main() {
    FILE *file;
    char buffer[100];

    file = fopen("example.txt", "r");
    if (file == NULL) {
        printf("Error opening file.\n");
        return 1;
    }

    while (fgets(buffer, 100, file) != NULL) {
        printf("%s", buffer);
    }

    fclose(file);
    return 0;
}

Writing to a File

To write data to a file, you can use functions like fwrite(), fputs(), and fprintf().

Example: Writing to a File Using `fprintf()`

#include <stdio.h>

int main() {
    FILE *file;
    file = fopen("example.txt", "w"); // Open file for writing

    if (file == NULL) {
        printf("Error opening file.\n");
        return 1;
    }

    fprintf(file, "Hello, World!\n");
    fprintf(file, "Writing to a file in C.\n");

    fclose(file);
    return 0;
}

File Modes

When opening a file, you need to specify the mode in which you want to open it. Here are some common file modes:

| Mode | Description | |------|----------------------------------| | "r" | Open for reading. | | "w" | Open for writing (creates file if it doesn't exist, truncates if it does). | | "a" | Open for appending (creates file if it doesn't exist). | | "r+" | Open for reading and writing. | | "w+" | Open for reading and writing (creates file if it doesn't exist, truncates if it does). | | "a+" | Open for reading and appending (creates file if it doesn't exist). |

Error Handling

It's crucial to handle errors when performing file operations to ensure your program can gracefully handle unexpected situations.

Example: Error Handling in File Operations

#include <stdio.h>

int main() {
    FILE *file;
    file = fopen("example.txt", "r");

    if (file == NULL) {
        perror("Error opening file");
        return 1;
    }

    // Perform file operations here

    if (fclose(file) != 0) {
        perror("Error closing file");
        return 1;
    }

    return 0;
}

Conclusion

File operations in C are fundamental for managing data persistence and manipulation. By understanding how to open, read, write, and close files, you can effectively handle data in your programs. Always remember to handle errors gracefully to make your programs robust and reliable.
