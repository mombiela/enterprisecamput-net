Introduction

The "Hello World" program is traditionally the first program written by beginners learning a new programming language. It serves as a simple introduction to the syntax and structure of the language. In this section, we will cover how to write, compile, and run a "Hello World" program in C.

Writing the Program

To write a "Hello World" program in C, you need to understand the basic structure of a C program. Here is the code for a simple "Hello World" program:

#include <stdio.h>

int main() {
    printf("Hello, World!\n");
    return 0;
}

Explanation

  • #include <stdio.h>: This line includes the Standard Input Output library, which is necessary for using the printf function.
  • int main() { ... }: This is the main function where the execution of the program begins.
  • printf("Hello, World!\n");: This line prints the string "Hello, World!" to the console. The \n is a newline character that moves the cursor to the next line.
  • return 0;: This line terminates the main function and returns 0 to the operating system, indicating that the program ended successfully.

Compiling the Program

To run a C program, you need to compile it first. Here are the steps to compile and run the "Hello World" program:

  1. Save the Program: Save the code in a file with a .c extension, for example, hello.c.

  2. Open Terminal/Command Prompt: Navigate to the directory where you saved the file.

  3. Compile the Program: Use a C compiler like gcc to compile the program. Run the following command:

    gcc hello.c -o hello

    This command compiles hello.c and creates an executable file named hello.

  4. Run the Program: Execute the compiled program by running:

    ./hello

Example

$ gcc hello.c -o hello
$ ./hello
Hello, World!

Key Concepts

  • Preprocessor Directives: Lines beginning with # are preprocessor directives. #include <stdio.h> tells the compiler to include the Standard Input Output library.
  • Main Function: The entry point of a C program. Every C program must have a main function.
  • Standard Output: printf is used to print text to the console.
  • Return Statement: return 0; indicates that the program has executed successfully.

Common Errors

  • Missing Semicolon: Every statement in C must end with a semicolon (;). Forgetting this will result in a compilation error.
  • Case Sensitivity: C is case-sensitive. main is different from Main.
  • File Extension: Ensure your file is saved with a .c extension.

Conclusion

The "Hello World" program is a fundamental starting point for learning C programming. It introduces the basic structure of a C program, including preprocessor directives, the main function, and standard output. By understanding and practicing this simple program, you lay the groundwork for more complex C programming concepts.
