Introduction to Strings in C

In C, strings are arrays of characters terminated by a null character ('\0'). Understanding how to manipulate strings is crucial for many programming tasks, from simple text processing to complex data parsing.

Key Concepts

  • String Declaration: Strings are declared as arrays of characters.
  • Null Terminator: The end of a string is marked by a null character ('\0').
  • String Initialization: Strings can be initialized at the time of declaration.

Example

char str1[] = "Hello, World!";
char str2[20] = "Hello, C!";
char str3[20]; // Uninitialized string

Standard String Functions

The C Standard Library provides a set of functions for string manipulation, declared in the <string.h> header file.

Commonly Used String Functions

  • strlen(): Returns the length of a string.
  • strcpy(): Copies one string to another.
  • strncpy(): Copies a specified number of characters from one string to another.
  • strcat(): Concatenates two strings.
  • strncat(): Concatenates a specified number of characters from one string to another.
  • strcmp(): Compares two strings.
  • strncmp(): Compares a specified number of characters from two strings.
  • strchr(): Finds the first occurrence of a character in a string.
  • strstr(): Finds the first occurrence of a substring in a string.

Example Usage

#include <stdio.h>
#include <string.h>

int main() {
    char str1[20] = "Hello";
    char str2[20] = "World";
    char str3[40];

    // String Length
    printf("Length of str1: %lu\n", strlen(str1));

    // String Copy
    strcpy(str3, str1);
    printf("str3 after strcpy: %s\n", str3);

    // String Concatenation
    strcat(str3, " ");
    strcat(str3, str2);
    printf("str3 after strcat: %s\n", str3);

    // String Comparison
    if (strcmp(str1, str2) == 0) {
        printf("str1 and str2 are equal\n");
    } else {
        printf("str1 and str2 are not equal\n");
    }

    return 0;
}

Advanced String Functions

For more complex string manipulations, C provides additional functions.

Memory Management Functions

  • strdup(): Duplicates a string by allocating memory dynamically.
  • strtok(): Tokenizes a string based on delimiters.

Example Usage

#include <stdio.h>
#include <string.h>
#include <stdlib.h>

int main() {
    char str[] = "Hello, World!";
    char *token;
    char *dup_str;

    // String Duplication
    dup_str = strdup(str);
    if (dup_str != NULL) {
        printf("Duplicated string: %s\n", dup_str);
        free(dup_str); // Free the allocated memory
    }

    // String Tokenization
    token = strtok(str, ", ");
    while (token != NULL) {
        printf("Token: %s\n", token);
        token = strtok(NULL, ", ");
    }

    return 0;
}

Summary

String functions in C are essential tools for handling and manipulating text. From basic operations like copying and concatenation to more advanced tasks like tokenization and dynamic memory management, mastering these functions will significantly enhance your programming capabilities. Always remember to include the <string.h> header file and manage memory carefully when dealing with dynamic strings.
