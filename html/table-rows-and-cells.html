Table Rows and Cells
Table Attributes ►

In this section, we will explore how to create and manipulate table rows and cells in HTML. Tables are a fundamental part of web development, allowing you to organize data in a structured format. We will cover the basic tags used to create tables, rows, and cells, and provide examples and exercises to solidify your understanding.

Key Concepts

  • Table (<table>): The container element for a table.
  • Table Row (<tr>): Defines a row in a table.
  • Table Header (<th>): Defines a header cell in a table.
  • Table Data (<td>): Defines a standard cell in a table.

Creating a Basic Table

To create a table, you need to use the <table> element. Inside this element, you can define rows using the <tr> element. Each row can contain header cells (<th>) and data cells (<td>).

Example

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Basic Table</title>
</head>
<body>
    <table border="1">
        <tr>
            <th>Header 1</th>
            <th>Header 2</th>
            <th>Header 3</th>
        </tr>
        <tr>
            <td>Data 1</td>
            <td>Data 2</td>
            <td>Data 3</td>
        </tr>
        <tr>
            <td>Data 4</td>
            <td>Data 5</td>
            <td>Data 6</td>
        </tr>
    </table>
</body>
</html>

Explanation

  • <table border="1">: Creates a table with a border.
  • <tr>: Defines a table row.
  • <th>: Defines a header cell, which is bold and centered by default.
  • <td>: Defines a standard cell.

Adding More Rows and Cells

You can add as many rows and cells as needed. Each row (<tr>) can contain multiple header (<th>) or data (<td>) cells.

Example

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Extended Table</title>
</head>
<body>
    <table border="1">
        <tr>
            <th>Header 1</th>
            <th>Header 2</th>
            <th>Header 3</th>
        </tr>
        <tr>
            <td>Data 1</td>
            <td>Data 2</td>
            <td>Data 3</td>
        </tr>
        <tr>
            <td>Data 4</td>
            <td>Data 5</td>
            <td>Data 6</td>
        </tr>
        <tr>
            <td>Data 7</td>
            <td>Data 8</td>
            <td>Data 9</td>
        </tr>
    </table>
</body>
</html>

Merging Cells

You can merge cells horizontally or vertically using the colspan and rowspan attributes.

Example

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Merged Cells</title>
</head>
<body>
    <table border="1">
        <tr>
            <th>Header 1</th>
            <th>Header 2</th>
            <th>Header 3</th>
        </tr>
        <tr>
            <td colspan="2">Merged Cell 1</td>
            <td>Data 3</td>
        </tr>
        <tr>
            <td>Data 4</td>
            <td rowspan="2">Merged Cell 2</td>
            <td>Data 6</td>
        </tr>
        <tr>
            <td>Data 7</td>
            <td>Data 9</td>
        </tr>
    </table>
</body>
</html>

Explanation

  • colspan="2": Merges two cells horizontally.
  • rowspan="2": Merges two cells vertically.

Styling Tables

Tables can be styled using CSS to improve their appearance.

Example

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Styled Table</title>
    <style>
        table {
            width: 100%;
            border-collapse: collapse;
        }
        th, td {
            border: 1px solid black;
            padding: 8px;
            text-align: left;
        }
        th {
            background-color: #f2f2f2;
        }
    </style>
</head>
<body>
    <table>
        <tr>
            <th>Header 1</th>
            <th>Header 2</th>
            <th>Header 3</th>
        </tr>
        <tr>
            <td>Data 1</td>
            <td>Data 2</td>
            <td>Data 3</td>
        </tr>
        <tr>
            <td>Data 4</td>
            <td>Data 5</td>
            <td>Data 6</td>
        </tr>
    </table>
</body>
</html>

Explanation

  • border-collapse: collapse;: Collapses the borders into a single border.
  • padding: 8px;: Adds padding inside the cells.
  • background-color: #f2f2f2;: Sets the background color of header cells.

Conclusion

In this section, we covered the basics of creating tables in HTML, including how to define rows and cells, merge cells, and apply basic styling. Tables are a powerful tool for organizing data, and understanding how to use them effectively is essential for any web developer. Practice creating your own tables and experiment with different styles to become more comfortable with these concepts.
