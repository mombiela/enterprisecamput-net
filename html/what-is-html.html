Introduction to HTML

HTML (HyperText Markup Language) is the standard language used to create and design documents on the World Wide Web. It structures web content and allows browsers to display text, images, and other elements in a readable format.

Key Concepts of HTML

  • Elements: The building blocks of HTML, defined by tags.
  • Tags: Keywords surrounded by angle brackets, e.g., <tagname>.
  • Attributes: Provide additional information about elements, usually in the form of name/value pairs.
  • Document Object Model (DOM): A tree-like structure representing the HTML document.

Basic Structure of an HTML Document

An HTML document has a specific structure that includes several key elements:

<!DOCTYPE html>
<html>
<head>
    <title>Page Title</title>
</head>
<body>
    <h1>This is a Heading</h1>
    <p>This is a paragraph.</p>
</body>
</html>

Explanation of the Basic Structure

  • <!DOCTYPE html>: Declares the document type and version of HTML.
  • <html>: The root element of an HTML page.
  • <head>: Contains meta-information about the HTML document, such as the title and links to stylesheets.
  • <title>: Sets the title of the web page, displayed in the browser's title bar or tab.
  • <body>: Contains the content of the HTML document, such as text, images, and other media.

Common HTML Elements

Here are some commonly used HTML elements:

  • Headings: Defined with <h1> to <h6> tags.
  • Paragraphs: Defined with the <p> tag.
  • Links: Defined with the <a> tag.
  • Images: Defined with the <img> tag.
  • Lists: Ordered lists (<ol>) and unordered lists (<ul>).

Examples of Common HTML Elements

Headings

<h1>This is a Heading 1</h1>
<h2>This is a Heading 2</h2>

Paragraphs

<p>This is a paragraph of text.</p>

Links

<a href="https://www.example.com">This is a link</a>

Images

<img src="image.jpg" alt="Description of image">

Lists

<ul>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
</ul>

<ol>
    <li>First item</li>
    <li>Second item</li>
    <li>Third item</li>
</ol>

Attributes in HTML

Attributes provide additional information about HTML elements. They are always included in the opening tag and usually come in name/value pairs like name="value".

Example of Attributes

<a href="https://www.example.com" target="_blank">Visit Example</a>
<img src="image.jpg" alt="Description of image" width="500" height="600">
  • href: Specifies the URL of the page the link goes to.
  • target: Specifies where to open the linked document.
  • src: Specifies the path to the image.
  • alt: Provides alternative text for the image.
  • width and height: Specify the dimensions of the image.

Conclusion

HTML is the foundational language for creating web pages. Understanding its basic structure, elements, and attributes is essential for anyone looking to build or design websites. As you progress through this course, you will learn more advanced HTML concepts and how to integrate them with CSS and JavaScript to create dynamic and visually appealing web pages.
