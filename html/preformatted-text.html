Preformatted Text in HTML

Introduction

Preformatted text in HTML is used to display text exactly as it is written in the HTML file, preserving spaces, line breaks, and other whitespace characters. This is particularly useful for displaying code snippets, poetry, or any text where the formatting is crucial.

The `
` Tag

The primary HTML element used for preformatted text is the <pre> tag.

  • Syntax: 
    <pre>
  Your preformatted text goes here.
</pre>
  • Attributes: The <pre> tag supports global attributes like id, class, style, etc.

Example

<pre>
  This is an example of preformatted text.
  Notice how the spaces and line breaks are preserved.
</pre>

When rendered in a browser, the text will appear exactly as it is written within the <pre> tags, including all spaces and line breaks.

Combining `
` with Other Tags

You can combine the <pre> tag with other HTML tags to enhance the presentation of preformatted text.

Using `` Inside `
`

The <code> tag is often used within <pre> to denote code snippets.

  • Syntax: 
    <pre>
  <code>
    // This is a code snippet
    function helloWorld() {
      console.log("Hello, World!");
    }
  </code>
</pre>

Example

<pre>
  <code>
    // This is a JavaScript function
    function greet(name) {
      console.log("Hello, " + name + "!");
    }
  </code>
</pre>

Styling Preformatted Text

You can style preformatted text using CSS to make it more readable or to match the design of your website.

Example

<style>
  pre {
    background-color: #f4f4f4;
    padding: 10px;
    border: 1px solid #ccc;
    overflow: auto;
  }
  code {
    color: #d63384;
  }
</style>

<pre>
  <code>
    // Styled code snippet
    const add = (a, b) => a + b;
  </code>
</pre>

Practical Exercises

Exercise 1: Basic Preformatted Text

Create a simple HTML file that uses the <pre> tag to display a poem with specific formatting.

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>Preformatted Text Example</title>
</head>
<body>
  <pre>
    Roses are red,
    Violets are blue,
    Sugar is sweet,
    And so are you.
  </pre>
</body>
</html>

Exercise 2: Displaying Code Snippets

Create an HTML file that uses both <pre> and <code> tags to display a Python code snippet.

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>Code Snippet Example</title>
</head>
<body>
  <pre>
    <code>
      def greet(name):
          print(f"Hello, {name}!")
    </code>
  </pre>
</body>
</html>

Exercise 3: Styling Preformatted Text

Create an HTML file that styles the preformatted text using CSS.

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>Styled Preformatted Text</title>
  <style>
    pre {
      background-color: #f0f0f0;
      padding: 15px;
      border-radius: 5px;
    }
    code {
      color: #2a9d8f;
    }
  </style>
</head>
<body>
  <pre>
    <code>
      // JavaScript function
      function add(a, b) {
        return a + b;
      }
    </code>
  </pre>
</body>
</html>

Conclusion

Preformatted text is a powerful feature in HTML that allows you to display text exactly as it is written, preserving all whitespace and line breaks. The <pre> tag is essential for this purpose, and it can be combined with other tags like <code> to display code snippets effectively. Styling preformatted text with CSS can enhance its readability and integration with your website's design. By understanding and practicing these concepts, you can effectively use preformatted text in your web projects.
