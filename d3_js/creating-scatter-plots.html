Introduction

Scatter plots are a powerful visualization tool used to display the relationship between two variables. In this section, we will learn how to create scatter plots using D3.js, a JavaScript library for producing dynamic, interactive data visualizations in web browsers.

Setting Up the Environment

Before we start creating scatter plots, ensure you have the following:

  • A basic understanding of HTML, CSS, and JavaScript.
  • D3.js library included in your project.

Including D3.js

You can include D3.js in your HTML file by adding the following script tag in the <head> section:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Scatter Plot with D3.js</title>
    <script src="https://d3js.org/d3.v7.min.js"></script>
</head>
<body>
    <!-- Your content will go here -->
</body>
</html>

Creating a Basic Scatter Plot

Let's start by creating a simple scatter plot.

HTML Structure

First, set up the basic HTML structure:

<body>
    <div id="scatter-plot"></div>
    <script src="scatter-plot.js"></script>
</body>

JavaScript Code

Next, create a scatter-plot.js file and add the following code:

// Set the dimensions and margins of the graph
const margin = {top: 20, right: 30, bottom: 40, left: 50},
      width = 460 - margin.left - margin.right,
      height = 400 - margin.top - margin.bottom;

// Append the svg object to the body of the page
const svg = d3.select("#scatter-plot")
  .append("svg")
    .attr("width", width + margin.left + margin.right)
    .attr("height", height + margin.top + margin.bottom)
  .append("g")
    .attr("transform", `translate(${margin.left},${margin.top})`);

// Sample data
const data = [
  {x: 30, y: 20}, {x: 50, y: 90}, {x: 70, y: 50},
  {x: 90, y: 120}, {x: 110, y: 80}, {x: 130, y: 150}
];

// Add X axis
const x = d3.scaleLinear()
  .domain([0, d3.max(data, d => d.x)])
  .range([ 0, width ]);
svg.append("g")
  .attr("transform", `translate(0,${height})`)
  .call(d3.axisBottom(x));

// Add Y axis
const y = d3.scaleLinear()
  .domain([0, d3.max(data, d => d.y)])
  .range([ height, 0]);
svg.append("g")
  .call(d3.axisLeft(y));

// Add dots
svg.append('g')
  .selectAll("dot")
  .data(data)
  .enter()
  .append("circle")
    .attr("cx", d => x(d.x) )
    .attr("cy", d => y(d.y) )
    .attr("r", 5)
    .style("fill", "#69b3a2");

Explanation

  • Margins and Dimensions: Define the margins and dimensions of the SVG container.
  • SVG Container: Append an SVG element to the #scatter-plot div and set its dimensions.
  • Sample Data: Use an array of objects to represent the data points.
  • Scales: Create linear scales for the X and Y axes based on the data.
  • Axes: Append the X and Y axes to the SVG.
  • Dots: Append circles to represent the data points.

Enhancing the Scatter Plot

Now that we have a basic scatter plot, let's enhance it with additional features.

Adding Labels and Titles

// Add X axis label
svg.append("text")
  .attr("text-anchor", "end")
  .attr("x", width)
  .attr("y", height + margin.top + 20)
  .text("X Axis Label");

// Add Y axis label
svg.append("text")
  .attr("text-anchor", "end")
  .attr("transform", "rotate(-90)")
  .attr("y", -margin.left + 20)
  .attr("x", -margin.top)
  .text("Y Axis Label");

// Add title
svg.append("text")
  .attr("x", width / 2)
  .attr("y", -margin.top / 2)
  .attr("text-anchor", "middle")
  .style("font-size", "16px")
  .style("text-decoration", "underline")
  .text("Scatter Plot Title");

Adding Tooltips

// Create a tooltip div that is hidden by default
const tooltip = d3.select("body").append("div")
  .style("opacity", 0)
  .attr("class", "tooltip")
  .style("background-color", "white")
  .style("border", "solid")
  .style("border-width", "1px")
  .style("border-radius", "5px")
  .style("padding", "10px");

// Add event listeners for tooltips
svg.selectAll("circle")
  .on("mouseover", (event, d) => {
    tooltip.transition()
      .duration(200)
      .style("opacity", .9);
    tooltip.html(`X: ${d.x}<br>Y: ${d.y}`)
      .style("left", (event.pageX + 5) + "px")
      .style("top", (event.pageY - 28) + "px");
  })
  .on("mouseout", () => {
    tooltip.transition()
      .duration(500)
      .style("opacity", 0);
  });

Explanation

  • Labels and Titles: Add text elements to label the axes and provide a title for the scatter plot.
  • Tooltips: Create a tooltip div and add event listeners to display the tooltip when hovering over data points.

Conclusion

In this section, we covered the basics of creating scatter plots using D3.js. We started with a simple scatter plot and enhanced it by adding labels, titles, and tooltips. Scatter plots are a versatile tool for visualizing data, and D3.js provides powerful capabilities to create interactive and dynamic visualizations.

By mastering these techniques, you can create more complex and informative scatter plots to better understand and present your data.
