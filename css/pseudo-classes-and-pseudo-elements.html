Introduction

Pseudo-classes and pseudo-elements are powerful tools in CSS that allow you to style elements based on their state or position in the document tree, without needing to add additional classes or IDs to your HTML. This section will cover the basics and advanced usage of pseudo-classes and pseudo-elements.

Pseudo-classes

Pseudo-classes are keywords added to selectors that specify a special state of the selected elements. They are used to style an element when it is in a certain state, such as when it is being hovered over by the mouse, or when it is the first child of its parent.

Common Pseudo-classes

  • :hover: Applies when the user designates an element (with some pointing device), but does not activate it.
  • :active: Applies while an element is being activated by the user.
  • :focus: Applies while an element has the focus (accepts keyboard or mouse events, or other forms of input).
  • :first-child: Applies to the first child of a specified element.
  • :last-child: Applies to the last child of a specified element.
  • :nth-child(n): Applies to the nth child of a specified element.

Example

/* Change the background color of a button when hovered */
button:hover {
  background-color: lightblue;
}

/* Change the color of the first paragraph in a container */
.container p:first-child {
  color: red;
}

/* Style every second list item */
ul li:nth-child(2n) {
  background-color: #f2f2f2;
}

Explanation

  • The button:hover selector changes the background color of a button when the mouse pointer is over it.
  • The .container p:first-child selector changes the text color of the first paragraph inside a container with the class container.
  • The ul li:nth-child(2n) selector changes the background color of every second list item in an unordered list.

Pseudo-elements

Pseudo-elements are keywords added to selectors that allow you to style specific parts of an element. They are used to style a part of an element, such as the first letter or line of a paragraph, or to insert content before or after an element's content.

Common Pseudo-elements

  • ::before: Inserts content before the content of an element.
  • ::after: Inserts content after the content of an element.
  • ::first-letter: Styles the first letter of the first line of a block-level element.
  • ::first-line: Styles the first line of a block-level element.

Example

/* Insert content before a paragraph */
p::before {
  content: "Note: ";
  font-weight: bold;
}

/* Insert content after a paragraph */
p::after {
  content: " [Read more]";
  color: blue;
}

/* Style the first letter of a paragraph */
p::first-letter {
  font-size: 2em;
  color: red;
}

/* Style the first line of a paragraph */
p::first-line {
  font-weight: bold;
}

Explanation

  • The p::before selector inserts the text "Note: " before the content of each paragraph and makes it bold.
  • The p::after selector inserts the text " [Read more]" after the content of each paragraph and colors it blue.
  • The p::first-letter selector increases the font size and changes the color of the first letter of each paragraph.
  • The p::first-line selector makes the first line of each paragraph bold.

Combining Pseudo-classes and Pseudo-elements

You can combine pseudo-classes and pseudo-elements to create more complex styles.

Example

/* Style the first letter of a hovered paragraph */
p:hover::first-letter {
  color: green;
  font-size: 3em;
}

Explanation

  • The p:hover::first-letter selector changes the color and font size of the first letter of a paragraph when the paragraph is hovered over.

Conclusion

Pseudo-classes and pseudo-elements are essential tools in CSS that allow you to apply styles based on the state or specific parts of elements. Understanding how to use them effectively can greatly enhance the interactivity and visual appeal of your web pages. Practice using these selectors in different scenarios to become proficient in their usage.
