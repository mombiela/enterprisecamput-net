Fonts and Typography
Box Model ►

Typography is a crucial aspect of web design, and CSS provides a variety of properties to control the appearance of text on your web pages. This topic will guide you from the basics to advanced techniques in CSS typography.

Basic Font Properties

Font Family

  • Definition: The font-family property specifies the font for an element.
  • Usage: You can list multiple fonts as a fallback system.
body {
    font-family: "Arial", "Helvetica", sans-serif;
}

Font Size

  • Definition: The font-size property sets the size of the text.
  • Units: Common units include px, em, rem, %, and vw.
p {
    font-size: 16px;
}

Font Weight

  • Definition: The font-weight property sets the thickness of the font.
  • Values: Common values include normal, bold, bolder, lighter, and numeric values (100-900).
h1 {
    font-weight: bold;
}

Font Style

  • Definition: The font-style property specifies the style of the font.
  • Values: normal, italic, and oblique.
em {
    font-style: italic;
}

Advanced Font Properties

Line Height

  • Definition: The line-height property sets the space between lines of text.
  • Usage: Can be set using unitless numbers, percentages, or length values.
p {
    line-height: 1.5;
}

Text Transform

  • Definition: The text-transform property controls the capitalization of text.
  • Values: none, capitalize, uppercase, and lowercase.
h2 {
    text-transform: uppercase;
}

Text Decoration

  • Definition: The text-decoration property adds decoration to text.
  • Values: none, underline, overline, line-through, and combinations.
a {
    text-decoration: underline;
}

Letter Spacing

  • Definition: The letter-spacing property controls the space between characters.
  • Usage: Can be set using length values.
h1 {
    letter-spacing: 2px;
}

Word Spacing

  • Definition: The word-spacing property sets the space between words.
  • Usage: Can be set using length values.
p {
    word-spacing: 4px;
}

Web Fonts

Using Google Fonts

  • Definition: Google Fonts is a library of free licensed font families.
  • Usage: Import fonts via a <link> tag in HTML or using @import in CSS.
<!-- HTML -->
<link href="https://fonts.googleapis.com/css2?family=Roboto:wght@400;700&display=swap" rel="stylesheet">
/* CSS */
body {
    font-family: 'Roboto', sans-serif;
}

@font-face Rule

  • Definition: The @font-face rule allows you to define custom fonts.
  • Usage: Specify the font name and the source files.
@font-face {
    font-family: 'MyCustomFont';
    src: url('mycustomfont.woff2') format('woff2'),
         url('mycustomfont.woff') format('woff');
}

body {
    font-family: 'MyCustomFont', sans-serif;
}

Responsive Typography

Fluid Typography

  • Definition: Fluid typography scales smoothly between different viewport sizes.
  • Usage: Use CSS functions like calc() and viewport units.
h1 {
    font-size: calc(1.5rem + 2vw);
}

Media Queries

  • Definition: Media queries allow you to apply styles based on the viewport size.
  • Usage: Adjust font sizes for different devices.
@media (max-width: 600px) {
    body {
        font-size: 14px;
    }
}

@media (min-width: 601px) {
    body {
        font-size: 16px;
    }
}

Conclusion

Typography in CSS is a powerful tool that can greatly enhance the readability and aesthetics of your web pages. From basic properties like font-family and font-size to advanced techniques like @font-face and responsive typography, mastering these properties will allow you to create visually appealing and accessible web designs. Practice using these properties in various combinations to see their effects and develop a keen eye for good typography.
