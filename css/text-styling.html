Introduction

Text styling is a fundamental aspect of web design that enhances the readability and visual appeal of your content. In this section, we will cover various CSS properties that allow you to style text effectively, from basic to advanced techniques.

Basic Text Styling

Font Family

  • The font-family property specifies the font for an element.
  • You can list multiple fonts as a fallback system.
p {
  font-family: Arial, Helvetica, sans-serif;
}

Font Size

  • The font-size property sets the size of the text.
  • Common units include px, em, rem, and %.
h1 {
  font-size: 24px;
}

Font Weight

  • The font-weight property sets the thickness of the text.
  • Values range from 100 (thin) to 900 (bold), or you can use keywords like normal and bold.
strong {
  font-weight: bold;
}

Font Style

  • The font-style property allows you to make text italic or oblique.
em {
  font-style: italic;
}

Text Color

  • The color property sets the color of the text.
  • You can use color names, HEX values, RGB, or HSL.
a {
  color: #3498db;
}

Intermediate Text Styling

Text Alignment

  • The text-align property aligns the text within its container.
  • Values include left, right, center, and justify.
p {
  text-align: justify;
}

Text Decoration

  • The text-decoration property adds decorations like underlines, overlines, and line-throughs.
a {
  text-decoration: underline;
}

Text Transform

  • The text-transform property controls the capitalization of text.
  • Values include uppercase, lowercase, and capitalize.
h2 {
  text-transform: uppercase;
}

Line Height

  • The line-height property sets the space between lines of text.
  • Can be set using unitless numbers, percentages, or length values.
p {
  line-height: 1.5;
}

Letter Spacing

  • The letter-spacing property adjusts the space between characters.
h1 {
  letter-spacing: 2px;
}

Word Spacing

  • The word-spacing property adjusts the space between words.
p {
  word-spacing: 4px;
}

Advanced Text Styling

Text Shadow

  • The text-shadow property adds shadow to text.
  • Syntax: text-shadow: h-shadow v-shadow blur-radius color;
h1 {
  text-shadow: 2px 2px 5px rgba(0, 0, 0, 0.3);
}

Web Fonts

  • Using @font-face to include custom fonts.
  • Example of importing a Google Font.
@import url('https://fonts.googleapis.com/css2?family=Roboto:wght@400;700&display=swap');

body {
  font-family: 'Roboto', sans-serif;
}

Advanced Text Effects

  • Using background-clip and color for gradient text.
h1 {
  background: linear-gradient(to right, #30CFD0 0%, #330867 100%);
  -webkit-background-clip: text;
  color: transparent;
}

Responsive Text

  • Using vw (viewport width) units for responsive text sizing.
h1 {
  font-size: 5vw;
}

Conclusion

Text styling in CSS ranges from basic properties like font size and color to advanced techniques like text shadows and responsive text. Mastering these properties allows you to create visually appealing and readable web pages. Practice using these properties in different combinations to see their effects and understand their behavior better.
