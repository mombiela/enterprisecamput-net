Introduction to CSS

CSS, or Cascading Style Sheets, is a stylesheet language used to describe the presentation of a document written in HTML or XML. CSS is a cornerstone technology of the World Wide Web, alongside HTML and JavaScript. It allows you to control the layout, colors, fonts, and overall visual appearance of web pages.

Key Concepts of CSS

  • Selectors: Patterns used to select the elements you want to style.
  • Properties: Aspects of the elements you want to change (e.g., color, font-size).
  • Values: Specific settings for the properties (e.g., red, 16px).
  • Rulesets: Combinations of selectors, properties, and values.

Example of a Simple CSS Ruleset

/* This is a CSS comment */
p {
  color: blue; /* This sets the text color of all <p> elements to blue */
  font-size: 16px; /* This sets the font size of all <p> elements to 16 pixels */
}

In this example:

  • p is the selector, targeting all <p> elements.
  • color and font-size are properties.
  • blue and 16px are values.

How CSS Works

CSS works by associating rules with HTML elements. These rules govern how the content of specified elements should be displayed. CSS can be applied in three different ways:

  1. Inline CSS: Directly within an HTML element using the style attribute.
  2. Internal CSS: Within a <style> tag in the <head> section of an HTML document.
  3. External CSS: In an external .css file linked to the HTML document.

Example of Inline CSS

<p style="color: blue; font-size: 16px;">This is a paragraph with inline CSS.</p>

Example of Internal CSS

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Internal CSS Example</title>
  <style>
    p {
      color: blue;
      font-size: 16px;
    }
  </style>
</head>
<body>
  <p>This is a paragraph with internal CSS.</p>
</body>
</html>

Example of External CSS

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>External CSS Example</title>
  <link rel="stylesheet" href="styles.css">
</head>
<body>
  <p>This is a paragraph with external CSS.</p>
</body>
</html>

In the styles.css file:

p {
  color: blue;
  font-size: 16px;
}

Advantages of Using CSS

  • Separation of Concerns: CSS separates content (HTML) from presentation (CSS), making it easier to maintain and update.
  • Reusability: CSS rules can be reused across multiple HTML pages.
  • Consistency: Ensures a consistent look and feel across a website.
  • Efficiency: Reduces the amount of code duplication and makes the website load faster.

Conclusion

CSS is an essential technology for web development, enabling developers to create visually appealing and user-friendly websites. By understanding the basics of CSS, including selectors, properties, and values, and how to apply CSS through inline, internal, and external methods, you can start to control the presentation of your web pages effectively. As you progress through this course, you will learn more advanced techniques and best practices to master CSS.
