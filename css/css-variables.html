Introduction to CSS Variables

CSS Variables, also known as Custom Properties, allow you to store values that you can reuse throughout your stylesheet. They are a powerful feature that can make your CSS more maintainable and flexible.

Key Concepts

  • Definition: CSS Variables are entities defined by CSS authors that contain specific values to be reused throughout a document.
  • Syntax: CSS Variables are defined using custom property notation (--property-name) and are accessed using the var() function.
  • Scope: Variables can be scoped globally (available throughout the entire document) or locally (available within a specific selector).

Defining and Using CSS Variables

Defining Variables

CSS Variables are defined within a CSS rule. They are usually defined in the :root selector for global scope.

:root {
  --main-bg-color: #3498db;
  --main-text-color: #ffffff;
  --padding: 10px;
}

Using Variables

To use a CSS Variable, you use the var() function.

body {
  background-color: var(--main-bg-color);
  color: var(--main-text-color);
  padding: var(--padding);
}

Example

Here is a simple example that demonstrates defining and using CSS Variables:

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>CSS Variables Example</title>
  <style>
    :root {
      --main-bg-color: #3498db;
      --main-text-color: #ffffff;
      --padding: 10px;
    }

    body {
      background-color: var(--main-bg-color);
      color: var(--main-text-color);
      padding: var(--padding);
    }

    .container {
      background-color: var(--main-text-color);
      color: var(--main-bg-color);
      padding: var(--padding);
    }
  </style>
</head>
<body>
  <div class="container">
    <h1>Hello, World!</h1>
    <p>This is an example of CSS Variables.</p>
  </div>
</body>
</html>

Advanced Usage of CSS Variables

Fallback Values

CSS Variables can have fallback values, which are used if the variable is not defined.

body {
  background-color: var(--main-bg-color, #2980b9); /* Fallback to #2980b9 if --main-bg-color is not defined */
}

Dynamic Updates

CSS Variables can be updated dynamically using JavaScript.

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Dynamic CSS Variables</title>
  <style>
    :root {
      --main-bg-color: #3498db;
    }

    body {
      background-color: var(--main-bg-color);
    }
  </style>
</head>
<body>
  <button onclick="changeColor()">Change Background Color</button>

  <script>
    function changeColor() {
      document.documentElement.style.setProperty('--main-bg-color', '#e74c3c');
    }
  </script>
</body>
</html>

Theming with CSS Variables

CSS Variables can be used to create themes for your application.

:root {
  --main-bg-color: #3498db;
  --main-text-color: #ffffff;
}

.dark-theme {
  --main-bg-color: #2c3e50;
  --main-text-color: #ecf0f1;
}

body {
  background-color: var(--main-bg-color);
  color: var(--main-text-color);
}

To apply the dark theme:

<body class="dark-theme">
  <!-- Content here -->
</body>

Conclusion

CSS Variables are a powerful tool for making your CSS more maintainable and flexible. They allow you to define values once and reuse them throughout your stylesheet, making it easier to manage and update your styles. By understanding and utilizing CSS Variables, you can create more dynamic and adaptable web designs.
