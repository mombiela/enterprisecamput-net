Introduction to CSS Grid Layout

CSS Grid Layout is a powerful layout system available in CSS. It allows you to create complex web layouts with ease by defining rows and columns, and placing items precisely within a grid.

Key Concepts

  • Grid Container: The element on which display: grid is applied. It's the parent of all grid items.
  • Grid Item: The children of the grid container.
  • Grid Lines: The dividing lines that make up the structure of the grid.
  • Grid Tracks: The space between two grid lines. They can be rows or columns.
  • Grid Cell: The space between four grid lines, essentially a single "box" in the grid.
  • Grid Area: A rectangular area that spans multiple grid cells.

Creating a Grid Container

To start using CSS Grid, you need to define a grid container by applying display: grid to an element.

.container {
  display: grid;
}

Defining Rows and Columns

You can define the structure of your grid using the grid-template-rows and grid-template-columns properties.

.container {
  display: grid;
  grid-template-columns: 100px 200px 100px;
  grid-template-rows: 50px 100px;
}

This creates a grid with three columns and two rows.

Placing Grid Items

You can place grid items in specific cells using the grid-column and grid-row properties.

.item1 {
  grid-column: 1 / 3; /* spans from column line 1 to 3 */
  grid-row: 1 / 2; /* spans from row line 1 to 2 */
}

Grid Gap

The grid-gap property (or column-gap and row-gap) allows you to add space between grid items.

.container {
  display: grid;
  grid-template-columns: 100px 200px 100px;
  grid-template-rows: 50px 100px;
  grid-gap: 10px;
}

Advanced Grid Properties

Grid Template Areas

You can define named grid areas and place items within these areas.

.container {
  display: grid;
  grid-template-areas:
    "header header header"
    "sidebar content content"
    "footer footer footer";
  grid-template-rows: auto;
  grid-template-columns: 1fr 2fr 1fr;
}

.header {
  grid-area: header;
}

.sidebar {
  grid-area: sidebar;
}

.content {
  grid-area: content;
}

.footer {
  grid-area: footer;
}

Implicit vs. Explicit Grids

  • Explicit Grid: Defined using grid-template-rows and grid-template-columns.
  • Implicit Grid: Created automatically by the browser when items are placed outside the explicit grid.

Auto-Placement

CSS Grid can automatically place items using the grid-auto-rows, grid-auto-columns, and grid-auto-flow properties.

.container {
  display: grid;
  grid-template-columns: 100px 200px;
  grid-auto-rows: 50px;
  grid-auto-flow: row;
}

Responsive Design with Grid

CSS Grid makes it easy to create responsive layouts. You can use media queries to change the grid structure based on the viewport size.

.container {
  display: grid;
  grid-template-columns: 1fr;
}

@media (min-width: 600px) {
  .container {
    grid-template-columns: 1fr 1fr;
  }
}

@media (min-width: 900px) {
  .container {
    grid-template-columns: 1fr 1fr 1fr;
  }
}

Conclusion

CSS Grid Layout is a versatile and powerful tool for creating complex web layouts. By understanding the key concepts and properties, you can create responsive and flexible designs with ease. Practice using grid layout in various projects to become proficient in this essential CSS feature.
