In this topic, you will learn how to add filters and controls in Google Data Studio to enhance the interactivity and customization of your reports. Filters and controls allow users to explore data more effectively, adjusting visualizations according to their specific needs.

Types of Filters and Controls

In Google Data Studio, there are several types of filters and controls you can use to allow users to interact with the data:

  • Date Filters: Allow selecting specific date ranges.
  • Dimension Filters: Allow filtering data based on specific dimensions such as categories, regions, etc.
  • Input Controls: Allow users to enter specific values to filter the data.

Adding Date Filters

Date filters are essential for any report that analyzes data over time. Here’s how to add a date filter:

  1. Select the Report: Open your report in Google Data Studio.
  2. Add Date Control:
    • Click on the "Insert" menu.
    • Select "Date Control".
    • Place the control in the desired area of the report.
  3. Configure the Date Control:
    • Click on the added date control.
    • In the properties panel, select the default date range and date format.

Example of Date Control Configuration

Default Date Range: Last 30 days
Date Format: dd/MM/yyyy

Adding Dimension Filters

Dimension filters allow users to filter data based on specific categories. Here’s how to add a dimension filter:

  1. Select the Report: Open your report in Google Data Studio.
  2. Add Dimension Control:
    • Click on the "Insert" menu.
    • Select "Dimension Control".
    • Place the control in the desired area of the report.
  3. Configure the Dimension Control:
    • Click on the added dimension control.
    • In the properties panel, select the dimension you want to filter.

Example of Dimension Control Configuration

Dimension: Product Category
Control Type: Dropdown list

Adding Input Controls

Input controls allow users to enter specific values to filter the data. Here’s how to add an input control:

  1. Select the Report: Open your report in Google Data Studio.
  2. Add Input Control:
    • Click on the "Insert" menu.
    • Select "Input Control".
    • Place the control in the desired area of the report.
  3. Configure the Input Control:
    • Click on the added input control.
    • In the properties panel, select the metric or dimension you want to filter.

Example of Input Control Configuration

Metric: Revenue
Control Type: Text box

Comparison of Types of Filters and Controls

| Filter/Control Type | Main Use | Example | |------------------------|---------------------------|--------------------------| | Date Filter | Filter by date ranges | Last 7 days, Current month | | Dimension Filter | Filter by specific categories | Product Category, Region | | Input Control | Filter by specific values | Revenue, Number of Sales |

Conclusion

Adding filters and controls in Google Data Studio is a powerful way to make your reports more interactive and useful for users. By allowing them to adjust the data according to their needs, you can provide more relevant and personalized insights. Practice adding different types of filters and controls in your reports to familiarize yourself with their configurations and capabilities.
