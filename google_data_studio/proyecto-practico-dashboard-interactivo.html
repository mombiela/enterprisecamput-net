Introduction

In this section, you will learn how to create an interactive dashboard using Google Data Studio. This practical project will allow you to apply the knowledge acquired in the course and develop key skills for data visualization.

Project Objectives

  • Create an interactive dashboard from scratch.
  • Connect and manipulate different data sources.
  • Use various visualizations to represent data.
  • Implement interactive filters and controls.
  • Customize the design and appearance of the dashboard.

Prerequisites

Before starting this project, make sure you have:

  • A Google account.
  • Access to Google Data Studio.
  • Sample data or access to a relevant data source.

Step 1: Initial Setup

Create a New Report

  1. Go to Google Data Studio.
  2. Click on "Create" and select "Report".
  3. Name your report and click "Create".

Connect Data Sources

  1. In the "Add data" panel, select the data source you want to use (e.g., Google Sheets, BigQuery, etc.).
  2. Follow the instructions to authenticate and connect your data source.
  3. Verify that the data has been imported correctly.

Step 2: Add and Configure Charts

Add a Bar Chart

  1. In the top menu, select "Add a chart" and choose "Bar chart".
  2. Drag the chart to the desired position on the canvas.
  3. Configure the dimensions and metrics in the chart properties panel.

Configuration Example

Dimension: Product Category
Metric: Total Sales

Add a Line Chart

  1. Select "Add a chart" and choose "Line chart".
  2. Drag the chart to the canvas and adjust its size.
  3. Configure the dimensions and metrics in the chart properties panel.

Configuration Example

Dimension: Date
Metric: Daily Sales

Step 3: Implement Filters and Interactive Controls

Add a Date Filter

  1. In the top menu, select "Add a control" and choose "Date selector".
  2. Drag the date selector to the canvas.
  3. Configure the date range and format in the properties panel.

Add a Category Filter

  1. Select "Add a control" and choose "Dropdown list".
  2. Drag the dropdown list to the canvas.
  3. Configure the dimension (e.g., "Product Category") in the properties panel.

Step 4: Customize the Design

Change the Theme and Style

  1. In the top menu, select "Theme and layout".
  2. Choose a predefined theme or customize the colors and fonts to your preferences.

Adjust the Canvas Layout

  1. Use the alignment and distribution tools to organize the charts and controls.
  2. Add titles and descriptions to provide context to users.

Conclusion

In this practical project, you have learned how to create an interactive dashboard in Google Data Studio. You have connected data sources, added and configured charts, implemented interactive filters and controls, and customized the design of the dashboard. This knowledge will allow you to create effective and attractive data visualizations for your professional projects.

Congratulations on completing the project!
