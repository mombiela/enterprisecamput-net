Introduction to Pipes

Pipes in Angular are a powerful feature that allows you to transform data directly in your templates. They are simple functions that accept an input value and return a transformed value. Angular provides several built-in pipes that cover common data transformation needs.

Key Built-in Pipes

Angular comes with a variety of built-in pipes that you can use out of the box. Here are some of the most commonly used ones:

  • DatePipe: Formats a date value according to locale rules.
  • UpperCasePipe: Transforms text to uppercase.
  • LowerCasePipe: Transforms text to lowercase.
  • CurrencyPipe: Transforms a number to a currency string.
  • DecimalPipe: Transforms a number into a string with a specified decimal format.
  • PercentPipe: Transforms a number into a percentage string.
  • JsonPipe: Converts a value into a JSON string.
  • SlicePipe: Creates a new array or string containing a subset (slice) of the elements.

Using Built-in Pipes

DatePipe

The DatePipe is used to format date values. You can specify the format you want the date to be displayed in.

<p>{{ today | date:'fullDate' }}</p>

In this example, today is a date object, and fullDate is a predefined format that displays the full date.

UpperCasePipe and LowerCasePipe

These pipes are used to transform text to uppercase or lowercase.

<p>{{ 'hello world' | uppercase }}</p>
<p>{{ 'HELLO WORLD' | lowercase }}</p>

CurrencyPipe

The CurrencyPipe is used to format numbers as currency.

<p>{{ price | currency:'USD' }}</p>

Here, price is a number, and 'USD' specifies the currency format.

DecimalPipe

The DecimalPipe formats numbers with a specified decimal format.

<p>{{ 3.14159 | number:'1.2-2' }}</p>

In this example, 1.2-2 specifies that the number should have at least one digit before the decimal point, and between two and two digits after the decimal point.

PercentPipe

The PercentPipe formats numbers as percentages.

<p>{{ 0.25 | percent }}</p>

JsonPipe

The JsonPipe converts a value into a JSON string, which is useful for debugging.

<p>{{ myObject | json }}</p>

SlicePipe

The SlicePipe creates a new array or string containing a subset of the elements.

<p>{{ 'Hello World' | slice:0:5 }}</p>
<p>{{ [1, 2, 3, 4, 5] | slice:1:3 }}</p>

Comparison of Common Pipes

| Pipe | Purpose | Example Usage | |---------------|----------------------------------------------|------------------------------------------| | DatePipe | Formats date values | {{ today | date:'shortDate' }} | | UpperCasePipe | Transforms text to uppercase | {{ 'hello' | uppercase }} | | LowerCasePipe | Transforms text to lowercase | {{ 'HELLO' | lowercase }} | | CurrencyPipe | Formats numbers as currency | {{ price | currency:'USD' }} | | DecimalPipe | Formats numbers with specified decimal places| {{ 3.14159 | number:'1.2-2' }} | | PercentPipe | Formats numbers as percentages | {{ 0.25 | percent }} | | JsonPipe | Converts a value into a JSON string | {{ myObject | json }} | | SlicePipe | Creates a subset of an array or string | {{ 'Hello' | slice:0:2 }} |

Conclusion

Built-in pipes in Angular provide a convenient way to transform data directly in your templates. They help keep your code clean and readable by handling common data transformation tasks. Understanding and utilizing these pipes can significantly enhance your Angular applications by making your templates more expressive and easier to maintain.
