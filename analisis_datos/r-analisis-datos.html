Introduction to R

R is a programming language and a free software environment for statistical analysis and data visualization. It is widely used by statisticians, data analysts, and data scientists due to its ability to handle, analyze, and visualize large datasets.

Advantages of Using R

  • Free and Open Source: R is completely free and its source code is available to everyone.
  • Large Community: There is a large community of users and developers who contribute packages and support.
  • Extensibility: R has a large number of packages that extend its functionality.
  • Data Visualization: R is known for its advanced data visualization capabilities.

Installation and Configuration of R

To start using R, it is necessary to install both R and RStudio, a popular integrated development environment (IDE) for R.

Installing R

  1. Visit the CRAN (Comprehensive R Archive Network) website: https://cran.r-project.org/
  2. Download and install the appropriate version for your operating system (Windows, macOS, Linux).

Installing RStudio

  1. Visit the RStudio website: https://rstudio.com/products/rstudio/download/
  2. Download and install the free version of RStudio Desktop.

Basic Concepts of R

Before diving into data analysis, it is important to familiarize yourself with some basic concepts of R.

Data Types

  • Numeric: Integer and decimal numbers.
  • Character: Text strings.
  • Factors: Categorical variables.
  • Logical: TRUE or FALSE values.

Data Structures

  • Vectors: Sequences of data of the same type.
  • Matrices: Two-dimensional arrays of data of the same type.
  • Data Frames: Data tables where each column can contain different types of data.
  • Lists: Collections of elements that can be of different types.

Code Example

# Create a numeric vector
numbers <- c(1, 2, 3, 4, 5)

# Create a data frame
data <- data.frame(
  name = c("Ana", "Luis", "Juan"),
  age = c(23, 25, 30),
  salary = c(50000, 55000, 60000)
)

# Display the data frame
print(data)

Data Import and Export

R allows importing and exporting data from and to different file formats.

Data Import

  • CSV: Comma-Separated Values
  • Excel: Microsoft Excel files
  • Databases: SQL database connections

Example of Importing a CSV File

# Read a CSV file
data <- read.csv("path/to/file.csv")

# Display the first 6 records
head(data)

Data Export

  • write.csv(): To export data to a CSV file.
  • write.xlsx(): To export data to an Excel file (requires the openxlsx package).

Example of Exporting a Data Frame to CSV

# Export the data frame to a CSV file
write.csv(data, "path/to/exported_file.csv")

Data Analysis with R

R provides a wide range of functions and packages for data analysis.

Descriptive Statistics

  • summary(): Provides a basic statistical summary.
  • mean(), median(), sd(): Calculation of mean, median, and standard deviation.

Example of Descriptive Statistics

# Calculate descriptive statistics
summary(data$salary)

# Calculate the mean of the salary
mean(data$salary)

# Calculate the standard deviation of the salary
sd(data$salary)

Data Visualization

  • ggplot2: A powerful package for creating graphics.
  • plot(): Basic function for graphics.

Example of Visualization with ggplot2

# Install and load the ggplot2 package
install.packages("ggplot2")
library(ggplot2)

# Create a scatter plot
ggplot(data, aes(x = age, y = salary)) +
  geom_point() +
  labs(title = "Relationship between Age and Salary",
       x = "Age",
       y = "Salary")

Conclusion

R is a powerful and versatile tool for data analysis. From data import and manipulation to statistical analysis and visualization, R offers a wide range of functionalities that make it indispensable for data analysis professionals. With practice and continuous use, users can take full advantage of R's capabilities to gain valuable insights from their data.
