Introduction to Lighting
Basic Animation ►

Lighting is a crucial aspect of creating realistic and visually appealing environments in Unreal Engine. This topic will guide you through the basics of lighting, different types of lights, and advanced lighting techniques to enhance your projects.

Key Concepts of Lighting

  • Light Types: Understanding the different types of lights available in Unreal Engine.
  • Light Properties: Learning about the properties and settings that can be adjusted for each light type.
  • Lighting Techniques: Exploring various techniques to achieve realistic lighting effects.
  • Performance Considerations: Balancing visual quality and performance.

Light Types

Unreal Engine provides several types of lights, each suited for different purposes:

  1. Directional Light:

    • Simulates sunlight or moonlight.
    • Casts parallel light rays.
    • Ideal for outdoor scenes.

  2. Point Light:

    • Emits light in all directions from a single point.
    • Similar to a light bulb.
    • Useful for localized light sources.

  3. Spot Light:

    • Emits light in a cone shape.
    • Similar to a flashlight or stage spotlight.
    • Great for focused lighting effects.

  4. Rect Light:

    • Emits light from a rectangular area.
    • Useful for simulating fluorescent lights or TV screens.

  5. Sky Light:

    • Captures the distant parts of the scene and applies that lighting to the whole level.
    • Ideal for ambient lighting and reflections.

Light Properties

Each light type has various properties that can be adjusted to achieve the desired effect:

  • Intensity: Controls the brightness of the light.
  • Color: Sets the color of the light.
  • Attenuation Radius: Defines the area affected by the light.
  • Cast Shadows: Determines whether the light casts shadows.
  • Source Radius/Length: Adjusts the size of the light source for softer or harder shadows.

Example: Setting Up a Point Light

// Adding a Point Light in C++
// Assuming you have a basic Actor class

#include "Components/PointLightComponent.h"

APointLightActor::APointLightActor()
{
    // Create a Point Light Component
    PointLight = CreateDefaultSubobject<UPointLightComponent>(TEXT("PointLight"));
    PointLight->Intensity = 3000.0f; // Set the intensity
    PointLight->bVisible = true; // Make it visible
    PointLight->SetupAttachment(RootComponent); // Attach to the root component
}

In Blueprints, you can add a Point Light component to an actor and adjust its properties in the Details panel.

Lighting Techniques

  1. Static Lighting:

    • Precomputed lighting.
    • Best for static objects.
    • High-quality shadows and lighting but no dynamic changes.

  2. Stationary Lighting:

    • Partially dynamic.
    • Allows for some changes in light properties at runtime.
    • Suitable for objects that move occasionally.

  3. Dynamic Lighting:

    • Fully dynamic.
    • Real-time lighting and shadows.
    • Ideal for objects that move frequently but more performance-intensive.

Example: Using Dynamic Lighting

// Enabling Dynamic Lighting for a Directional Light in C++

#include "Components/DirectionalLightComponent.h"

ADirectionalLightActor::ADirectionalLightActor()
{
    // Create a Directional Light Component
    DirectionalLight = CreateDefaultSubobject<UDirectionalLightComponent>(TEXT("DirectionalLight"));
    DirectionalLight->Mobility = EComponentMobility::Movable; // Set to Movable for dynamic lighting
    DirectionalLight->Intensity = 10.0f; // Set the intensity
    DirectionalLight->SetupAttachment(RootComponent); // Attach to the root component
}

Performance Considerations

Balancing visual quality and performance is essential:

  • Use Static Lighting for non-moving objects to save on performance.
  • Limit the number of Dynamic Lights to avoid performance hits.
  • Optimize Shadow Settings: Use lower resolution shadows for distant objects.
  • Use Light Baking: Precompute lighting for static objects to improve runtime performance.

Conclusion

Lighting in Unreal Engine is a powerful tool that can significantly enhance the visual quality of your projects. By understanding the different types of lights, their properties, and various lighting techniques, you can create realistic and immersive environments. Always consider performance implications and optimize your lighting setup to achieve the best balance between visual fidelity and performance.
