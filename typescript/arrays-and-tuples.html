Arrays and Tuples in TypeScript
Enums ►

Introduction

In this section, we will explore arrays and tuples in TypeScript. Arrays and tuples are fundamental data structures that allow you to store collections of values. Understanding how to use them effectively is crucial for any TypeScript programmer.

Arrays

Arrays are used to store multiple values in a single variable. In TypeScript, arrays can be of a specific type or a union of types.

Declaring Arrays

You can declare arrays in TypeScript using two syntaxes:

  1. Using square brackets ([]):

    let numbers: number[] = [1, 2, 3, 4, 5];
let strings: string[] = ["apple", "banana", "cherry"];

  2. Using the Array generic type:

    let numbers: Array<number> = [1, 2, 3, 4, 5];
let strings: Array<string> = ["apple", "banana", "cherry"];

Accessing Array Elements

You can access array elements using their index, starting from 0:

let fruits: string[] = ["apple", "banana", "cherry"];
console.log(fruits[0]); // Output: apple
console.log(fruits[1]); // Output: banana

Array Methods

TypeScript arrays come with a variety of built-in methods. Here are some commonly used ones:

  • push(): Adds one or more elements to the end of an array.

    let numbers: number[] = [1, 2, 3];
numbers.push(4);
console.log(numbers); // Output: [1, 2, 3, 4]

  • pop(): Removes the last element from an array.

    let numbers: number[] = [1, 2, 3];
numbers.pop();
console.log(numbers); // Output: [1, 2]

  • map(): Creates a new array with the results of calling a provided function on every element.

    let numbers: number[] = [1, 2, 3];
let doubled = numbers.map(num => num * 2);
console.log(doubled); // Output: [2, 4, 6]

  • filter(): Creates a new array with all elements that pass the test implemented by the provided function.

    let numbers: number[] = [1, 2, 3, 4, 5];
let even = numbers.filter(num => num % 2 === 0);
console.log(even); // Output: [2, 4]

Tuples

Tuples are a special type of array where the type of each element is known. They allow you to express an array with a fixed number of elements whose types are known but can be different.

Declaring Tuples

You can declare tuples by specifying the type of each element in the array:

let person: [string, number] = ["Alice", 30];

Accessing Tuple Elements

You can access tuple elements just like arrays, using their index:

let person: [string, number] = ["Alice", 30];
console.log(person[0]); // Output: Alice
console.log(person[1]); // Output: 30

Tuple Methods

Tuples support the same methods as arrays, but you need to be cautious about the types:

let person: [string, number] = ["Alice", 30];
person.push("Engineer");
console.log(person); // Output: ["Alice", 30, "Engineer"]

Note: Adding elements beyond the declared types can lead to unexpected behavior.

Comparison: Arrays vs Tuples

| Feature | Arrays | Tuples | |------------------|---------------------------------|---------------------------------| | Length | Can vary | Fixed | | Element Types | Homogeneous or Union of Types | Heterogeneous | | Use Case | Lists of similar items | Fixed structure with different types |

Conclusion

Arrays and tuples are essential data structures in TypeScript. Arrays are flexible and can hold multiple values of the same type, while tuples are fixed in length and can hold values of different types. Understanding how to use these structures effectively will help you write more robust and type-safe TypeScript code.
