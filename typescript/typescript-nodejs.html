Introduction to TypeScript with Node.js

  • Overview of TypeScript and its benefits.
  • Why use TypeScript with Node.js?
  • Setting up a TypeScript project with Node.js.

Setting Up the Environment

  • Installing Node.js and npm.
  • Installing TypeScript globally: 
    npm install -g typescript
  • Initializing a new Node.js project: 
    mkdir my-typescript-node-project
cd my-typescript-node-project
npm init -y
  • Adding TypeScript to the project: 
    npm install typescript --save-dev
  • Creating a tsconfig.json file: 
    npx tsc --init

Basic TypeScript Concepts

  • Type Annotations: 
    let message: string = "Hello, TypeScript with Node.js!";
console.log(message);
  • Interfaces: 
    interface User {
  name: string;
  age: number;
}

const user: User = {
  name: "John Doe",
  age: 30
};
console.log(user);
  • Classes: 
    class Person {
  constructor(public name: string, public age: number) {}

  greet() {
    console.log(`Hello, my name is ${this.name} and I am ${this.age} years old.`);
  }
}

const person = new Person("Jane Doe", 25);
person.greet();

Integrating TypeScript with Node.js

  • Creating a simple server with Express: 
    npm install express @types/express --save

    import express, { Request, Response } from 'express';

const app = express();
const port = 3000;

app.get('/', (req: Request, res: Response) => {
  res.send('Hello, TypeScript with Node.js!');
});

app.listen(port, () => {
  console.log(`Server is running at http://localhost:${port}`);
});
  • Compiling TypeScript to JavaScript: 
    npx tsc
  • Running the compiled JavaScript: 
    node dist/index.js

Advanced TypeScript Features

  • Generics: 
    function identity<T>(arg: T): T {
  return arg;
}

let output = identity<string>("myString");
console.log(output);
  • Decorators: 
    function log(target: any, key: string) {
  let value = target[key];

  const getter = () => {
    console.log(`Get: ${key} => ${value}`);
    return value;
  };

  const setter = (newVal: any) => {
    console.log(`Set: ${key} => ${newVal}`);
    value = newVal;
  };

  Object.defineProperty(target, key, {
    get: getter,
    set: setter,
    enumerable: true,
    configurable: true
  });
}

class Example {
  @log
  public name: string;

  constructor(name: string) {
    this.name = name;
  }
}

const example = new Example("TypeScript");
example.name = "Node.js";
console.log(example.name);

Testing TypeScript with Node.js

  • Setting up Jest for testing: 
    npm install jest ts-jest @types/jest --save-dev
  • Configuring Jest: 
    npx ts-jest config:init
  • Writing a simple test: 
    // sum.ts
export const sum = (a: number, b: number): number => {
  return a + b;
};

// sum.test.ts
import { sum } from './sum';

test('adds 1 + 2 to equal 3', () => {
  expect(sum(1, 2)).toBe(3);
});
  • Running the tests: 
    npx jest

Conclusion

  • Recap of the benefits of using TypeScript with Node.js.
  • Encouragement to explore more advanced TypeScript features and Node.js integrations.
  • Resources for further learning and community support.

By the end of this topic, you should have a solid understanding of how to set up and use TypeScript with Node.js, from basic concepts to advanced features and testing. Happy coding!
