Introduction

Data types are fundamental to any programming language, and Go is no exception. Understanding data types is crucial for writing efficient and error-free code. In this section, we will cover the basic and advanced data types in Go, their usage, and how to work with them.

Basic Data Types

Integers

Go supports both signed and unsigned integers in various sizes:

  • int and uint: Platform-dependent size (32 or 64 bits)
  • int8, int16, int32, int64: Signed integers of 8, 16, 32, and 64 bits
  • uint8, uint16, uint32, uint64: Unsigned integers of 8, 16, 32, and 64 bits
package main

import "fmt"

func main() {
    var a int = 10
    var b int8 = -128
    var c uint16 = 65535

    fmt.Println(a, b, c)
}

Floating-Point Numbers

Go supports floating-point numbers:

  • float32: 32-bit floating-point numbers
  • float64: 64-bit floating-point numbers
package main

import "fmt"

func main() {
    var a float32 = 3.14
    var b float64 = 2.718281828459045

    fmt.Println(a, b)
}

Boolean

The bool type represents a boolean value, which can be either true or false.

package main

import "fmt"

func main() {
    var isGoFun bool = true
    fmt.Println(isGoFun)
}

Strings

Strings in Go are sequences of bytes. They are immutable, meaning once created, they cannot be changed.

package main

import "fmt"

func main() {
    var greeting string = "Hello, Go!"
    fmt.Println(greeting)
}

Composite Data Types

Arrays

Arrays are fixed-size sequences of elements of the same type.

package main

import "fmt"

func main() {
    var arr [5]int = [5]int{1, 2, 3, 4, 5}
    fmt.Println(arr)
}

Slices

Slices are more flexible than arrays. They are dynamically-sized and provide more powerful and convenient operations.

package main

import "fmt"

func main() {
    var slice []int = []int{1, 2, 3, 4, 5}
    fmt.Println(slice)
}

Maps

Maps are collections of key-value pairs. Keys must be unique and of a comparable type.

package main

import "fmt"

func main() {
    var m map[string]int = map[string]int{"one": 1, "two": 2}
    fmt.Println(m)
}

Structs

Structs are collections of fields. They are used to group data together.

package main

import "fmt"

type Person struct {
    Name string
    Age  int
}

func main() {
    var p Person = Person{Name: "Alice", Age: 30}
    fmt.Println(p)
}

Advanced Data Types

Pointers

Pointers hold the memory address of a variable. They are useful for efficient data manipulation and avoiding copying large structures.

package main

import "fmt"

func main() {
    var a int = 10
    var p *int = &a

    fmt.Println("Value of a:", a)
    fmt.Println("Address of a:", p)
    fmt.Println("Value at address p:", *p)
}

Interfaces

Interfaces define a set of method signatures. Any type that implements these methods satisfies the interface.

package main

import "fmt"

type Speaker interface {
    Speak() string
}

type Dog struct{}

func (d Dog) Speak() string {
    return "Woof!"
}

func main() {
    var s Speaker = Dog{}
    fmt.Println(s.Speak())
}

Conclusion

Understanding data types in Go is essential for writing effective programs. We covered basic data types like integers, floating-point numbers, booleans, and strings, as well as composite data types like arrays, slices, maps, and structs. We also touched on advanced data types like pointers and interfaces. Mastering these concepts will help you write more efficient and robust Go code.
