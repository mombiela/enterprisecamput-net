Introduction

Pie and donut charts are popular visual tools in data visualization, especially useful for showing the proportion of different categories within a dataset. This topic will cover the basics, the differences between both types of charts, and how to create and interpret them.

Key Concepts

  • Pie Chart: Represents data in the form of a circle divided into sectors, where each sector shows a proportion of the total.
  • Donut Chart: Similar to the pie chart, but with a hole in the center, allowing for better visualization of multiple data series.

Advantages and Disadvantages

| Advantages | Disadvantages | |---------------------------|----------------------------------| | Easy to understand | Difficult to read with many categories | | Visually appealing | Does not show changes over time | | Useful for proportions | Precise comparisons are difficult |

Creating Pie Charts

Example in Python with Matplotlib

import matplotlib.pyplot as plt

# Data
labels = ['A', 'B', 'C', 'D']
sizes = [15, 30, 45, 10]
colors = ['gold', 'yellowgreen', 'lightcoral', 'lightskyblue']
explode = (0.1, 0, 0, 0)  # Highlights the first sector

# Create pie chart
plt.pie(sizes, explode=explode, labels=labels, colors=colors, autopct='%1.1f%%', shadow=True, startangle=140)
plt.axis('equal')  # Ensures the chart is a circle
plt.show()
  • labels: Labels for each sector.
  • sizes: Size of each sector.
  • colors: Colors for each sector.
  • explode: Displaces a sector to highlight it.
  • autopct: Format to show percentages.
  • shadow: Adds shadow for visual effect.
  • startangle: Starting angle of the first sector.

Creating Donut Charts

Example in Python with Matplotlib

import matplotlib.pyplot as plt

# Data
labels = ['A', 'B', 'C', 'D']
sizes = [15, 30, 45, 10]
colors = ['gold', 'yellowgreen', 'lightcoral', 'lightskyblue']
explode = (0.1, 0, 0, 0)  # Highlights the first sector

# Create pie chart
plt.pie(sizes, explode=explode, labels=labels, colors=colors, autopct='%1.1f%%', shadow=True, startangle=140)

# Add white circle in the center to create the donut effect
centre_circle = plt.Circle((0,0),0.70,fc='white')
fig = plt.gcf()
fig.gca().add_artist(centre_circle)

plt.axis('equal')  # Ensures the chart is a circle
plt.show()
  • centre_circle: White circle in the center to create the donut effect.

Practical Exercises

  1. Create a pie chart: Use your own dataset to create a pie chart that shows the proportions of different categories.
  2. Modify a pie chart: Add labels, custom colors, and highlight a specific sector.
  3. Create a donut chart: Convert your pie chart into a donut chart by adding a circle in the center.

Conclusion

Pie and donut charts are effective visual tools for showing proportions within a dataset. Although they have limitations, their simplicity and visual appeal make them useful in many situations. With practice, you will be able to create and customize these charts to communicate your data clearly and effectively.
