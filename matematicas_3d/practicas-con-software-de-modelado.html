Introduction

In this section of the course on 3D Mathematics, we will explore how to use modeling software to apply three-dimensional mathematical concepts. The goal is for students to become familiar with practical tools that allow them to visualize and manipulate objects in a three-dimensional space.

Recommended Modeling Software

For the practices in this course, we recommend using the following programs:

  • Blender: A very popular open-source software for 3D modeling, animation, and rendering.
  • Autodesk Maya: A professional tool widely used in the film and video game industry.
  • SketchUp: Ideal for beginners and widely used in architecture and design.

Installation and Configuration

Blender

  1. Download: Visit blender.org and download the latest version.
  2. Installation: Follow the installer instructions for your operating system.
  3. Initial Configuration: Open Blender and adjust the preferences according to your needs (e.g., keyboard shortcuts, interface theme).

Autodesk Maya

  1. Download: Visit autodesk.com and download the trial or educational version.
  2. Installation: Follow the installer instructions.
  3. Initial Configuration: Configure the initial preferences and ensure the necessary plugins are activated.

SketchUp

  1. Download: Visit sketchup.com and download the free or trial version.
  2. Installation: Follow the installer instructions.
  3. Initial Configuration: Adjust the preferences according to your needs.

Key Concepts

  • Vertices, Edges, and Faces: The basic components of any 3D model.
  • Transformations: Operations such as translation, rotation, and scaling.
  • Modifiers: Tools that allow you to deform and modify objects non-destructively.
  • Texturing: Applying images to the surface of a 3D model to give it a realistic appearance.

Practical Examples

Create a Cube in Blender

  1. Add a Cube:

    import bpy

bpy.ops.mesh.primitive_cube_add(size=2, enter_editmode=False, align='WORLD', location=(0, 0, 0))

  2. Scale the Cube:

    bpy.ops.transform.resize(value=(2, 2, 2))

  3. Rotate the Cube:

    bpy.ops.transform.rotate(value=1.5708, orient_axis='Z')

Create a Sphere in Maya

  1. Add a Sphere:

    polySphere -r 1 -sx 20 -sy 20;

  2. Scale the Sphere:

    scale -r 2 2 2;

  3. Rotate the Sphere:

    rotate -r 45 45 0;

Create a Cylinder in SketchUp

  1. Add a Cylinder:

    • Select the "Circle" tool and draw a circle on the plane.
    • Use the "Push/Pull" tool to extrude the circle and create a cylinder.

  2. Scale the Cylinder:

    • Select the cylinder and use the "Scale" tool to change its size.

  3. Rotate the Cylinder:

    • Select the cylinder and use the "Rotate" tool to change its orientation.

Exercises

  1. Blender: Create a scene with a cube, a sphere, and a cylinder. Apply transformations (translation, rotation, scaling) to each object.
  2. Maya: Model a simple house using basic primitives (cubes, pyramids, etc.). Apply textures to the surfaces.
  3. SketchUp: Design a simple piece of furniture (e.g., a table) and apply materials to give it a realistic look.

Conclusion

Using 3D modeling software is essential for visualizing and applying three-dimensional mathematical concepts. Through these practices, students will not only learn to handle modeling tools but also better understand how objects are represented and manipulated in a three-dimensional space.
