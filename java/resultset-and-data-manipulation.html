ResultSet and Data Manipulation in Java
Transactions ►

Introduction

In this section, we will explore how to work with ResultSet in Java to manipulate data retrieved from a database. We will cover the basics of ResultSet, how to navigate through it, and how to update data using it. This topic is crucial for anyone looking to build robust Java applications that interact with databases.

Understanding ResultSet

A ResultSet is a table of data representing the result of a database query. It is generated by executing a statement that queries the database.

Key Concepts

  • ResultSet Interface: The ResultSet interface provides methods for retrieving and manipulating the data in the result set.
  • Cursor: The ResultSet maintains a cursor pointing to its current row of data.
  • Types of ResultSet:
    • TYPE_FORWARD_ONLY: The cursor can only move forward.
    • TYPE_SCROLL_INSENSITIVE: The cursor can move both forward and backward, but the result set is not sensitive to changes made by others.
    • TYPE_SCROLL_SENSITIVE: The cursor can move both forward and backward, and the result set is sensitive to changes made by others.

Creating a ResultSet

To create a ResultSet, you need to execute a query using a Statement or PreparedStatement.

Connection conn = DriverManager.getConnection("jdbc:mysql://localhost:3306/mydatabase", "user", "password");
Statement stmt = conn.createStatement();
ResultSet rs = stmt.executeQuery("SELECT * FROM employees");

Navigating ResultSet

You can navigate through the rows of a ResultSet using various methods.

Key Methods

  • next(): Moves the cursor to the next row.
  • previous(): Moves the cursor to the previous row (only for scrollable ResultSet).
  • first(): Moves the cursor to the first row.
  • last(): Moves the cursor to the last row.
  • absolute(int row): Moves the cursor to the specified row number.

Example

while (rs.next()) {
    int id = rs.getInt("id");
    String name = rs.getString("name");
    System.out.println("ID: " + id + ", Name: " + name);
}

Retrieving Data from ResultSet

You can retrieve data from the current row of a ResultSet using getter methods.

Key Methods

  • getInt(String columnLabel): Retrieves an integer value.
  • getString(String columnLabel): Retrieves a string value.
  • getDouble(String columnLabel): Retrieves a double value.
  • getDate(String columnLabel): Retrieves a date value.

Example

if (rs.next()) {
    int id = rs.getInt("id");
    String name = rs.getString("name");
    double salary = rs.getDouble("salary");
    Date hireDate = rs.getDate("hire_date");
    System.out.println("ID: " + id + ", Name: " + name + ", Salary: " + salary + ", Hire Date: " + hireDate);
}

Updating Data with ResultSet

A ResultSet can also be used to update data in the database.

Key Concepts

  • Concurrency Types:
    • CONCUR_READ_ONLY: The ResultSet cannot be updated.
    • CONCUR_UPDATABLE: The ResultSet can be updated.

Example

To create an updatable ResultSet, you need to specify the concurrency type when creating the Statement.

Statement stmt = conn.createStatement(ResultSet.TYPE_SCROLL_INSENSITIVE, ResultSet.CONCUR_UPDATABLE);
ResultSet rs = stmt.executeQuery("SELECT * FROM employees");

// Move to the row you want to update
if (rs.next()) {
    rs.updateString("name", "John Doe");
    rs.updateRow();
}

Inserting Data

You can also insert new rows into the database using ResultSet.

rs.moveToInsertRow();
rs.updateInt("id", 101);
rs.updateString("name", "Jane Doe");
rs.updateDouble("salary", 75000);
rs.updateDate("hire_date", java.sql.Date.valueOf("2023-01-01"));
rs.insertRow();

Deleting Data

To delete a row from the database using ResultSet:

if (rs.next()) {
    rs.deleteRow();
}

Conclusion

In this section, we have covered the basics of working with ResultSet in Java. We explored how to create, navigate, retrieve, update, insert, and delete data using ResultSet. Understanding these concepts is essential for effective data manipulation in Java applications that interact with databases. Practice these techniques to become proficient in handling database operations in Java.
