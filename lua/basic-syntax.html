Introduction to Lua

Lua is a lightweight, high-level, multi-paradigm programming language designed primarily for embedded use in applications. It is known for its simplicity and flexibility, making it a popular choice for game development, scripting, and rapid prototyping.

Key Concepts

  • Case Sensitivity: Lua is case-sensitive, meaning Variable and variable are considered different identifiers.
  • Comments: Lua supports both single-line and multi-line comments.
  • Whitespace: Whitespace is generally ignored in Lua, but it is used to separate tokens.

Comments

  • Single-line comments: Use -- to start a single-line comment.
  • Multi-line comments: Use --[[ to start and ]] to end a multi-line comment.
-- This is a single-line comment

--[[
This is a
multi-line comment
]]

Variables and Data Types

  • Variable Declaration: Variables in Lua are dynamically typed and can be declared without specifying a type.
  • Data Types: Lua supports several basic data types, including nil, boolean, number, string, function, userdata, thread, and table.
local myNumber = 10       -- number
local myString = "Hello"  -- string
local myBoolean = true    -- boolean
local myNil = nil         -- nil

Operators

Lua supports a variety of operators for arithmetic, relational, logical, and other operations.

Arithmetic Operators

  • + (addition)
  • - (subtraction)
  • * (multiplication)
  • / (division)
  • % (modulus)
  • ^ (exponentiation)
local a = 5
local b = 2
print(a + b)  -- Output: 7
print(a - b)  -- Output: 3
print(a * b)  -- Output: 10
print(a / b)  -- Output: 2.5
print(a % b)  -- Output: 1
print(a ^ b)  -- Output: 25

Relational Operators

  • == (equal to)
  • ~= (not equal to)
  • < (less than)
  • > (greater than)
  • <= (less than or equal to)
  • >= (greater than or equal to)
local x = 10
local y = 20
print(x == y)  -- Output: false
print(x ~= y)  -- Output: true
print(x < y)   -- Output: true
print(x > y)   -- Output: false
print(x <= y)  -- Output: true
print(x >= y)  -- Output: false

Logical Operators

  • and
  • or
  • not
local a = true
local b = false
print(a and b)  -- Output: false
print(a or b)   -- Output: true
print(not a)    -- Output: false

Control Structures

Lua provides several control structures for managing the flow of the program.

If Statements

local age = 18
if age >= 18 then
    print("You are an adult.")
else
    print("You are a minor.")
end

While Loops

local count = 1
while count <= 5 do
    print(count)
    count = count + 1
end

For Loops

for i = 1, 5 do
    print(i)
end

Repeat-Until Loops

local count = 1
repeat
    print(count)
    count = count + 1
until count > 5

Functions

Functions in Lua are first-class values and can be stored in variables, passed as arguments, and returned from other functions.

Defining a Function

function greet(name)
    return "Hello, " .. name
end

print(greet("World"))  -- Output: Hello, World

Anonymous Functions

local add = function(a, b)
    return a + b
end

print(add(2, 3))  -- Output: 5

Tables

Tables are the only data structure in Lua and can be used as arrays, dictionaries, or even objects.

Creating a Table

local person = {
    name = "John",
    age = 30
}

print(person.name)  -- Output: John
print(person.age)   -- Output: 30

Iterating Over a Table

local fruits = {"apple", "banana", "cherry"}

for index, value in ipairs(fruits) do
    print(index, value)
end

Conclusion

Understanding the basic syntax and structure of Lua is crucial for writing effective and efficient code. This includes knowing how to declare variables, use operators, control the flow of the program, define functions, and work with tables. With these fundamentals, you are well-equipped to explore more advanced features and applications of Lua.
