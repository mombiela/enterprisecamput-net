Basic Input and Output in MUMPS (M)
Introduction to Globals ►

Introduction

In this section, we will cover the basics of input and output operations in MUMPS (M). Understanding how to handle input and output is crucial for interacting with users and managing data within your programs.

Key Concepts

  • Input: Reading data from the user or another source.
  • Output: Displaying data to the user or another destination.

Input Operations

Using the `READ` Command

The READ command is used to take input from the user.

Syntax

READ variable

Example

READ "Enter your name: ", name
WRITE "Hello, ", name, "!"

In this example:

  • The READ command prompts the user to enter their name.
  • The WRITE command then displays a greeting message using the input.

Reading Multiple Variables

You can read multiple variables in a single READ command.

Example

READ "Enter your first name: ", firstName, " Enter your last name: ", lastName
WRITE "Hello, ", firstName, " ", lastName, "!"

In this example:

  • The user is prompted to enter their first and last names.
  • The WRITE command displays a greeting message using both inputs.

Output Operations

Using the `WRITE` Command

The WRITE command is used to display data to the user.

Syntax

WRITE expression

Example

SET greeting = "Hello, World!"
WRITE greeting

In this example:

  • The SET command assigns the string "Hello, World!" to the variable greeting.
  • The WRITE command displays the value of greeting.

Formatting Output

You can format the output using special characters and expressions.

Example

SET age = 25
WRITE "You are ", age, " years old."

In this example:

  • The WRITE command displays a formatted message that includes the value of age.

Combining Input and Output

Example Program

Here is a simple program that combines input and output operations:

READ "Enter your age: ", age
IF age<18 WRITE "You are a minor."
ELSE  WRITE "You are an adult."

In this example:

  • The READ command prompts the user to enter their age.
  • The IF statement checks if the age is less than 18.
  • Depending on the condition, the WRITE command displays an appropriate message.

Conclusion

In this section, we have covered the basics of input and output operations in MUMPS. You learned how to use the READ command to take input from the user and the WRITE command to display output. We also explored how to format output and combine input and output operations in a simple program. Mastering these basics will enable you to create interactive MUMPS programs that can effectively communicate with users.
