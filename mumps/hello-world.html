Introduction

MUMPS (Massachusetts General Hospital Utility Multi-Programming System), also known as M, is a programming language with a rich history in the healthcare industry. It is known for its built-in database capabilities and is still used in various medical and financial systems. In this section, we will cover the basics of writing a simple "Hello World" program in MUMPS.

Key Concepts

  • MUMPS Syntax: Understanding the basic syntax of MUMPS.
  • Writing a Program: How to write and execute a simple MUMPS program.
  • Output: Displaying output to the console.

Writing a Simple "Hello World" Program

MUMPS Syntax

MUMPS has a unique syntax that may seem unusual to those familiar with more modern programming languages. Here are some key points:

  • Commands: MUMPS commands are often abbreviated (e.g., WRITE can be abbreviated as W).
  • Line Continuation: Lines can be continued using a space or a dot (.).
  • Variables: Variables are typeless and do not require declaration.

Example Code

Let's write a simple "Hello World" program in MUMPS.

HELLO ; This is a label
    WRITE "Hello, World!",!
    QUIT

Explanation

  • Label: HELLO is a label that marks the beginning of the program. Labels are optional but useful for organization.
  • WRITE Command: The WRITE (or W) command outputs text to the console. In this case, it writes "Hello, World!".
  • New Line: The ! at the end of the string moves the cursor to the next line.
  • QUIT Command: The QUIT command ends the execution of the program.

Running the Program

To run the program, you need to:

  1. Save the code in a file with a .m extension, for example, hello.m.
  2. Open your MUMPS environment (e.g., GT.M or Cache).
  3. Load and execute the program using the following commands:
DO ^hello

Conclusion

Writing a "Hello World" program in MUMPS is a straightforward way to get acquainted with the language's syntax and basic commands. Understanding how to write and execute simple programs is the first step in mastering MUMPS. In the next sections, we will delve deeper into more complex features and functionalities of the language.
