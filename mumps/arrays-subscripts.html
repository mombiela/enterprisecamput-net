Introduction to Arrays

Arrays in MUMPS (also known as M) are a fundamental data structure used to store and manipulate collections of data. Unlike arrays in many other programming languages, MUMPS arrays are associative, meaning they are indexed by keys that can be strings or numbers.

Key Concepts

  • Associative Arrays: Arrays indexed by keys, which can be strings or numbers.
  • Subscripts: The keys used to index arrays.
  • Global Arrays: Arrays that are stored on disk and persist between sessions.
  • Local Arrays: Arrays that exist only during the execution of a program.

Working with Local Arrays

Local arrays are temporary and exist only during the execution of a program. They are useful for storing data that does not need to persist.

Creating and Accessing Local Arrays

To create and access local arrays, use the following syntax:

SET array("key1") = "value1"
SET array("key2") = "value2"
WRITE array("key1"), !
WRITE array("key2"), !

Example

SET fruits("apple") = 10
SET fruits("banana") = 20
WRITE "Apple count: ", fruits("apple"), !
WRITE "Banana count: ", fruits("banana"), !

Iterating Over Local Arrays

You can iterate over local arrays using the FOR loop:

SET fruits("apple") = 10
SET fruits("banana") = 20
SET fruits("cherry") = 30

SET key = ""
FOR  SET key=$ORDER(fruits(key)) QUIT:key=""  DO
. WRITE key, ": ", fruits(key), !

Working with Global Arrays

Global arrays are stored on disk and persist between sessions. They are identified by a caret (^) prefix.

Creating and Accessing Global Arrays

To create and access global arrays, use the following syntax:

SET ^globalArray("key1") = "value1"
SET ^globalArray("key2") = "value2"
WRITE ^globalArray("key1"), !
WRITE ^globalArray("key2"), !

Example

SET ^inventory("apple") = 100
SET ^inventory("banana") = 200
WRITE "Apple inventory: ", ^inventory("apple"), !
WRITE "Banana inventory: ", ^inventory("banana"), !

Iterating Over Global Arrays

You can iterate over global arrays similarly to local arrays:

SET ^inventory("apple") = 100
SET ^inventory("banana") = 200
SET ^inventory("cherry") = 300

SET key = ""
FOR  SET key=$ORDER(^inventory(key)) QUIT:key=""  DO
. WRITE key, ": ", ^inventory(key), !

Advanced Topics

Multi-dimensional Arrays

MUMPS supports multi-dimensional arrays, which are arrays of arrays.

Example

SET matrix(1,1) = 10
SET matrix(1,2) = 20
SET matrix(2,1) = 30
SET matrix(2,2) = 40

WRITE "Matrix(1,1): ", matrix(1,1), !
WRITE "Matrix(1,2): ", matrix(1,2), !
WRITE "Matrix(2,1): ", matrix(2,1), !
WRITE "Matrix(2,2): ", matrix(2,2), !

Dynamic Subscripts

Subscripts in MUMPS can be dynamically generated and used to access array elements.

Example

SET i = 1
SET j = 2
SET matrix(i,j) = 50
WRITE "Matrix(", i, ",", j, "): ", matrix(i,j), !

Conclusion

Arrays and subscripts are powerful features in MUMPS that allow for flexible and efficient data storage and manipulation. Understanding how to work with both local and global arrays, as well as advanced concepts like multi-dimensional arrays and dynamic subscripts, is essential for effective MUMPS programming. Practice creating, accessing, and iterating over arrays to become proficient in using these structures.
