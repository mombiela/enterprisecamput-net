String Manipulation in MUMPS (M)
File Handling ►

String manipulation is a fundamental skill in MUMPS (M), a language known for its powerful string handling capabilities. This topic will guide you from basic to advanced string manipulation techniques in MUMPS.

Introduction to Strings in MUMPS

  • Definition: In MUMPS, a string is a sequence of characters.
  • String Variables: Strings are stored in variables and can be manipulated using various built-in functions.

Basic String Operations

Assigning Strings

SET str="Hello, World!"
WRITE str
  • Explanation: The SET command assigns the string "Hello, World!" to the variable str. The WRITE command outputs the value of str.

Concatenation

SET str1="Hello"
SET str2="World"
SET str3=str1_" "_str2
WRITE str3
  • Explanation: The _ operator concatenates str1, a space, and str2 into str3.

String Length

Using the $LENGTH Function

SET str="Hello, World!"
SET len=$LENGTH(str)
WRITE len
  • Explanation: The $LENGTH function returns the length of the string str.

Substring Extraction

Using the $EXTRACT Function

SET str="Hello, World!"
SET sub=$EXTRACT(str,1,5)
WRITE sub
  • Explanation: The $EXTRACT function extracts a substring from str starting at position 1 and ending at position 5.

String Searching

Using the $FIND Function

SET str="Hello, World!"
SET pos=$FIND(str,"World")
WRITE pos
  • Explanation: The $FIND function returns the position immediately after the substring "World" in str.

String Replacement

Using the $TRANSLATE Function

SET str="Hello, World!"
SET newstr=$TRANSLATE(str,"World","MUMPS")
WRITE newstr
  • Explanation: The $TRANSLATE function replaces occurrences of "World" with "MUMPS" in str.

Intermediate String Operations

String Comparison

Using the $ORDER Function

SET str1="Apple"
SET str2="Banana"
IF str1]str2 WRITE "str1 is greater"
ELSE  WRITE "str2 is greater"
  • Explanation: The ] operator compares two strings lexicographically.

String Splitting

Using the $PIECE Function

SET str="apple,banana,cherry"
SET fruit1=$PIECE(str,",",1)
SET fruit2=$PIECE(str,",",2)
SET fruit3=$PIECE(str,",",3)
WRITE fruit1,!,fruit2,!,fruit3
  • Explanation: The $PIECE function splits str by the delimiter , and extracts the specified piece.

Advanced String Operations

Complex String Manipulations

Combining Functions

SET str="apple,banana,cherry"
SET newstr=$TRANSLATE($PIECE(str,",",2),"banana","grape")
WRITE newstr
  • Explanation: This example combines $PIECE and $TRANSLATE to extract "banana" and replace it with "grape".

Regular Expressions (Advanced)

  • Note: MUMPS does not natively support regular expressions, but complex string manipulations can be achieved using combinations of the functions discussed.

Conclusion

String manipulation in MUMPS is a powerful tool that allows for efficient handling and transformation of text data. From basic operations like concatenation and length measurement to more advanced techniques like complex manipulations, mastering these functions will significantly enhance your MUMPS programming skills.

  • Practice: Regularly practice these functions to become proficient.
  • Explore: Experiment with combining different functions to solve complex string manipulation tasks.

By understanding and utilizing these string manipulation techniques, you will be well-equipped to handle a wide range of text processing tasks in MUMPS.
