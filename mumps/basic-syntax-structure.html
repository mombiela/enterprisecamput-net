Introduction

MUMPS (Massachusetts General Hospital Utility Multi-Programming System), also known as M, is a programming language designed for the healthcare industry. It is known for its unique syntax and powerful database capabilities. This section will introduce you to the basic syntax and structure of MUMPS, providing a foundation for more advanced topics.

Key Concepts

  • Lines and Commands: MUMPS code is written in lines, each containing one or more commands.
  • Variables: Variables in MUMPS can be local or global.
  • Data Types: MUMPS primarily uses strings and numbers.
  • Functions: Built-in functions perform various operations.
  • Control Structures: Conditional statements and loops control the flow of the program.

Lines and Commands

In MUMPS, each line of code can contain one or more commands. Commands are separated by spaces, and each command can have arguments.

Example:

WRITE "Hello, World!",!
SET X=10
  • WRITE "Hello, World!",! prints "Hello, World!" followed by a newline.
  • SET X=10 assigns the value 10 to the variable X.

Variables

MUMPS supports two types of variables: local and global.

  • Local Variables: These are temporary and exist only during the execution of a program.
  • Global Variables: These are stored in the database and persist between program executions.

Example:

SET localVar="I am local"
SET ^globalVar="I am global"
  • localVar is a local variable.
  • ^globalVar is a global variable, indicated by the caret (^) symbol.

Data Types

MUMPS primarily uses two data types: strings and numbers. There is no need to declare the type explicitly.

Example:

SET str="Hello"
SET num=123
  • str is a string variable.
  • num is a numeric variable.

Functions

MUMPS has several built-in functions for various operations, such as string manipulation, mathematical calculations, and date/time handling.

Example:

SET length=$LENGTH("Hello") ; Returns the length of the string "Hello"
SET upper=$ZCONVERT("hello","U") ; Converts "hello" to uppercase
  • $LENGTH returns the length of a string.
  • $ZCONVERT converts a string to uppercase.

Control Structures

MUMPS provides control structures like conditional statements and loops to control the flow of the program.

Conditional Statements

IF X=10 WRITE "X is 10",!
ELSE  WRITE "X is not 10",!
  • IF checks if X is equal to 10.
  • ELSE executes if the condition is false.

Loops

SET i=1
FOR  SET i=$INCREMENT(i) QUIT:i>5  WRITE i,!
  • FOR loop increments i and prints its value until i is greater than 5.

Conclusion

Understanding the basic syntax and structure of MUMPS is crucial for writing effective programs. This section covered the essentials, including lines and commands, variables, data types, functions, and control structures. With this foundation, you are now prepared to delve deeper into more advanced topics in MUMPS programming.
