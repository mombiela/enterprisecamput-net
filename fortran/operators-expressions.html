Introduction

Operators and expressions are fundamental components in Fortran programming. They allow you to perform calculations, make decisions, and manipulate data. This section will cover the various types of operators available in Fortran, how to use them, and how to construct expressions.

Arithmetic Operators

Arithmetic operators are used to perform mathematical calculations. Fortran supports the following arithmetic operators:

  • + : Addition
  • - : Subtraction
  • * : Multiplication
  • / : Division
  • ** : Exponentiation

Examples

PROGRAM ArithmeticExample
  IMPLICIT NONE
  INTEGER :: a, b, c
  REAL :: x, y, z

  ! Integer arithmetic
  a = 10
  b = 3
  c = a + b  ! c = 13
  PRINT *, 'a + b = ', c

  c = a - b  ! c = 7
  PRINT *, 'a - b = ', c

  c = a * b  ! c = 30
  PRINT *, 'a * b = ', c

  c = a / b  ! c = 3 (integer division)
  PRINT *, 'a / b = ', c

  ! Real arithmetic
  x = 10.0
  y = 3.0
  z = x / y  ! z = 3.3333333
  PRINT *, 'x / y = ', z

  z = x ** y  ! z = 1000.0
  PRINT *, 'x ** y = ', z
END PROGRAM ArithmeticExample

Relational Operators

Relational operators are used to compare values. Fortran supports the following relational operators:

  • == : Equal to
  • /= : Not equal to
  • > : Greater than
  • < : Less than
  • >= : Greater than or equal to
  • <= : Less than or equal to

Examples

PROGRAM RelationalExample
  IMPLICIT NONE
  INTEGER :: a, b
  LOGICAL :: result

  a = 10
  b = 3

  result = (a == b)  ! result = .FALSE.
  PRINT *, 'a == b: ', result

  result = (a /= b)  ! result = .TRUE.
  PRINT *, 'a /= b: ', result

  result = (a > b)  ! result = .TRUE.
  PRINT *, 'a > b: ', result

  result = (a < b)  ! result = .FALSE.
  PRINT *, 'a < b: ', result

  result = (a >= b)  ! result = .TRUE.
  PRINT *, 'a >= b: ', result

  result = (a <= b)  ! result = .FALSE.
  PRINT *, 'a <= b: ', result
END PROGRAM RelationalExample

Logical Operators

Logical operators are used to perform logical operations. Fortran supports the following logical operators:

  • .AND. : Logical AND
  • .OR. : Logical OR
  • .NOT. : Logical NOT

Examples

PROGRAM LogicalExample
  IMPLICIT NONE
  LOGICAL :: p, q, result

  p = .TRUE.
  q = .FALSE.

  result = p .AND. q  ! result = .FALSE.
  PRINT *, 'p .AND. q: ', result

  result = p .OR. q  ! result = .TRUE.
  PRINT *, 'p .OR. q: ', result

  result = .NOT. p  ! result = .FALSE.
  PRINT *, '.NOT. p: ', result
END PROGRAM LogicalExample

Assignment Operators

Assignment operators are used to assign values to variables. The basic assignment operator in Fortran is =.

Examples

PROGRAM AssignmentExample
  IMPLICIT NONE
  INTEGER :: a
  REAL :: x

  a = 5  ! Assign 5 to a
  PRINT *, 'a = ', a

  x = 3.14  ! Assign 3.14 to x
  PRINT *, 'x = ', x
END PROGRAM AssignmentExample

Combined Operators

Fortran also supports combined operators for arithmetic operations and assignment. These include:

  • += : Addition assignment
  • -= : Subtraction assignment
  • *= : Multiplication assignment
  • /= : Division assignment

Examples

PROGRAM CombinedOperatorsExample
  IMPLICIT NONE
  INTEGER :: a

  a = 10
  a += 5  ! Equivalent to a = a + 5
  PRINT *, 'a += 5: ', a  ! a = 15

  a -= 3  ! Equivalent to a = a - 3
  PRINT *, 'a -= 3: ', a  ! a = 12

  a *= 2  ! Equivalent to a = a * 2
  PRINT *, 'a *= 2: ', a  ! a = 24

  a /= 4  ! Equivalent to a = a / 4
  PRINT *, 'a /= 4: ', a  ! a = 6
END PROGRAM CombinedOperatorsExample

Operator Precedence

Operator precedence determines the order in which operations are performed in an expression. The following table summarizes the precedence of operators in Fortran, from highest to lowest:

| Operator | Description | |----------------|-----------------------| | ** | Exponentiation | | *, / | Multiplication, Division | | +, - | Addition, Subtraction | | .NOT. | Logical NOT | | .AND. | Logical AND | | .OR. | Logical OR | | ==, /=, >, <, >=, <= | Relational Operators |

Examples

PROGRAM PrecedenceExample
  IMPLICIT NONE
  INTEGER :: a, b, c, result

  a = 2
  b = 3
  c = 4

  result = a + b * c  ! Multiplication before addition
  PRINT *, 'a + b * c = ', result  ! result = 14

  result = (a + b) * c  ! Parentheses change the order
  PRINT *, '(a + b) * c = ', result  ! result = 20

  result = a ** b + c  ! Exponentiation before addition
  PRINT *, 'a ** b + c = ', result  ! result = 12
END PROGRAM PrecedenceExample

Conclusion

Understanding operators and expressions is crucial for writing effective Fortran programs. This section covered arithmetic, relational, logical, assignment, and combined operators, as well as operator precedence. By mastering these concepts, you will be able to perform complex calculations and make informed decisions in your Fortran programs. Practice using these operators in different contexts to become more comfortable with their usage and behavior.
