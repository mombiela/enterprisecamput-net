Setting Up the Development Environment

Setting up the development environment is a crucial first step in learning Fortran. This section will guide you through the process of installing the necessary tools and configuring your system to write, compile, and run Fortran programs.

1. Installing a Fortran Compiler

A Fortran compiler is essential for converting your Fortran code into executable programs. There are several popular Fortran compilers available:

  • GNU Fortran (gfortran): Part of the GNU Compiler Collection (GCC), it is open-source and widely used.
  • Intel Fortran Compiler (ifort): Known for its performance optimizations, especially on Intel hardware.
  • NAG Fortran Compiler: Known for its strong standards compliance and debugging features.

Installing gfortran

On Windows

  1. Download and Install MinGW-w64:

    • Visit the MinGW-w64 download page.
    • Choose the appropriate installer for your system (32-bit or 64-bit).
    • Follow the installation instructions, ensuring that you select the Fortran compiler during setup.

  2. Verify Installation:

    gfortran --version

    This command should display the version of gfortran installed.

On macOS

  1. Install Homebrew (if not already installed):

    /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

  2. Install gfortran:

    brew install gcc

    This will install gfortran along with other GCC compilers.

  3. Verify Installation:

    gfortran --version

On Linux

  1. Install gfortran using your package manager:

    • Debian/Ubuntu: 
      sudo apt-get update
sudo apt-get install gfortran
    • Fedora: 
      sudo dnf install gcc-gfortran

  2. Verify Installation:

    gfortran --version

2. Setting Up an Integrated Development Environment (IDE)

An IDE can significantly enhance your productivity by providing features like syntax highlighting, code completion, and debugging tools. Some popular IDEs for Fortran include:

  • Visual Studio Code (VS Code): A highly customizable and extensible editor.
  • Eclipse with Photran: A powerful IDE with a dedicated Fortran plugin.
  • Code::Blocks: A free, open-source IDE that supports Fortran.

Setting Up Visual Studio Code for Fortran

  1. Install Visual Studio Code:

    • Download and install VS Code from the official website.

  2. Install Fortran Extensions:

    • Open VS Code and go to the Extensions view by clicking the Extensions icon in the Activity Bar on the side of the window.
    • Search for "Modern Fortran" and install it.

  3. Configure the Fortran Compiler:

    • Open the Command Palette (Ctrl+Shift+P) and type "Preferences: Open Settings (JSON)".
    • Add the following configuration to specify the path to your gfortran compiler: 
      {
  "fortran.compilerPath": "/path/to/gfortran"
}

  4. Create and Run a Simple Fortran Program:

    • Create a new file with a .f90 extension (e.g., hello_world.f90).
    • Write a simple Fortran program: 
      program hello
  print *, "Hello, Fortran!"
end program hello
    • Save the file and open the terminal in VS Code.
    • Compile the program: 
      gfortran -o hello_world hello_world.f90
    • Run the executable: 
      ./hello_world

3. Configuring Environment Variables

Setting environment variables can help streamline the development process by making tools and libraries easily accessible.

Adding gfortran to the PATH

On Windows

  1. Open System Properties:

    • Right-click on "This PC" or "Computer" on the desktop or in File Explorer.
    • Select "Properties" and then "Advanced system settings".

  2. Edit Environment Variables:

    • Click on "Environment Variables".
    • Under "System variables", find the Path variable and click "Edit".
    • Add the path to the directory containing gfortran.exe (e.g., C:\MinGW\bin).

  3. Verify:

    • Open a new Command Prompt and type: 
      gfortran --version

On macOS and Linux

  1. Edit the Shell Configuration File:

    • Open ~/.bashrc (or ~/.zshrc if using Zsh) in a text editor.
    • Add the following line: 
      export PATH="/path/to/gfortran:$PATH"

  2. Apply the Changes:

    source ~/.bashrc

  3. Verify:

    gfortran --version

Conclusion

Setting up your development environment is the first step towards becoming proficient in Fortran programming. By installing a Fortran compiler, configuring an IDE, and setting environment variables, you create a robust foundation for writing, compiling, and running Fortran programs efficiently. With your environment ready, you can now focus on learning the language and developing your programming skills.
