Introduction to Loops in Fortran

Loops are fundamental control structures in programming that allow you to execute a block of code multiple times. In Fortran, there are several types of loops, each suited for different scenarios. This section will cover the basic loop constructs in Fortran, including DO loops, DO WHILE loops, and FORALL loops.

DO Loop

The DO loop is the most common loop in Fortran. It repeats a block of code a specified number of times.

Syntax

DO variable = start, end, step
    ! Code to be executed
END DO
  • variable: The loop control variable.
  • start: The initial value of the loop control variable.
  • end: The final value of the loop control variable.
  • step: The increment (optional, defaults to 1).

Example

PROGRAM DoLoopExample
    INTEGER :: i
    DO i = 1, 5
        PRINT *, "Iteration: ", i
    END DO
END PROGRAM DoLoopExample

In this example, the loop will print the iteration number from 1 to 5.

Nested DO Loops

You can nest DO loops to handle more complex iterations.

Example

PROGRAM NestedDoLoopExample
    INTEGER :: i, j
    DO i = 1, 3
        DO j = 1, 2
            PRINT *, "i = ", i, " j = ", j
        END DO
    END DO
END PROGRAM NestedDoLoopExample

This example demonstrates a nested loop where the inner loop runs twice for each iteration of the outer loop.

DO WHILE Loop

The DO WHILE loop continues to execute as long as a specified condition is true.

Syntax

DO WHILE (condition)
    ! Code to be executed
END DO

Example

PROGRAM DoWhileExample
    INTEGER :: i
    i = 1
    DO WHILE (i <= 5)
        PRINT *, "Iteration: ", i
        i = i + 1
    END DO
END PROGRAM DoWhileExample

In this example, the loop will run as long as i is less than or equal to 5.

FORALL Loop

The FORALL loop is used for array operations and can be more efficient for certain types of computations.

Syntax

FORALL (index = start:end:step, condition)
    array(index) = expression
END FORALL

Example

PROGRAM ForallExample
    INTEGER, DIMENSION(5) :: array
    FORALL (i = 1:5)
        array(i) = i * i
    END FORALL
    PRINT *, array
END PROGRAM ForallExample

This example initializes an array with the squares of its indices.

Summary

In this section, we covered the basic loop constructs in Fortran:

  • DO Loop: Repeats a block of code a specified number of times.
  • Nested DO Loops: Allows for more complex iterations.
  • DO WHILE Loop: Continues to execute as long as a condition is true.
  • FORALL Loop: Used for efficient array operations.

Understanding these loops will help you write more efficient and readable Fortran programs. Practice using these loops with different conditions and ranges to become more comfortable with their syntax and behavior.
