Introduction

Input and output (I/O) operations are fundamental in any programming language, including Fortran. They allow programs to interact with users and other systems by reading data from and writing data to various sources like the console, files, and more.

Basic Input and Output

Reading from the Console

Fortran uses the READ statement to take input from the user. The simplest form reads from the standard input (usually the keyboard).

PROGRAM ReadExample
  IMPLICIT NONE
  INTEGER :: num

  PRINT *, 'Enter a number:'
  READ *, num
  PRINT *, 'You entered:', num
END PROGRAM ReadExample
  • PRINT * is used to display a message to the console.
  • READ * reads input from the console and stores it in the variable num.

Writing to the Console

The PRINT statement is used to output data to the console.

PROGRAM PrintExample
  IMPLICIT NONE
  INTEGER :: num

  num = 10
  PRINT *, 'The number is:', num
END PROGRAM PrintExample
  • PRINT *, is followed by the message and variables to be printed.

Advanced Input and Output

Formatted Input and Output

For more control over the format of the input and output, Fortran provides formatted I/O using the READ and WRITE statements with format specifiers.

PROGRAM FormattedIO
  IMPLICIT NONE
  INTEGER :: num
  REAL :: value

  PRINT *, 'Enter an integer and a real number:'
  READ '(I5, F10.2)', num, value
  PRINT '(A, I5, A, F10.2)', 'Integer:', num, ' Real:', value
END PROGRAM FormattedIO
  • READ '(I5, F10.2)' specifies the format for reading: an integer with up to 5 digits and a real number with up to 10 digits and 2 decimal places.
  • PRINT '(A, I5, A, F10.2)' specifies the format for printing: a string, an integer with up to 5 digits, another string, and a real number with up to 10 digits and 2 decimal places.

File Input and Output

Fortran allows reading from and writing to files using the OPEN, READ, WRITE, and CLOSE statements.

PROGRAM FileIO
  IMPLICIT NONE
  INTEGER :: num
  CHARACTER(LEN=100) :: line

  ! Open a file for writing
  OPEN(UNIT=10, FILE='output.txt', STATUS='REPLACE')
  WRITE(10, *) 'This is a line of text.'
  WRITE(10, *) 12345
  CLOSE(10)

  ! Open a file for reading
  OPEN(UNIT=10, FILE='output.txt', STATUS='OLD')
  READ(10, '(A)') line
  PRINT *, 'First line:', line
  READ(10, *) num
  PRINT *, 'Second line (number):', num
  CLOSE(10)
END PROGRAM FileIO
  • OPEN opens a file with a specified unit number.
  • WRITE writes data to the file associated with the unit number.
  • READ reads data from the file associated with the unit number.
  • CLOSE closes the file associated with the unit number.

Error Handling in I/O

Fortran provides mechanisms to handle errors during I/O operations using the IOSTAT and ERR specifiers.

PROGRAM IOErrorHandling
  IMPLICIT NONE
  INTEGER :: num, io_status

  OPEN(UNIT=10, FILE='nonexistent.txt', STATUS='OLD', IOSTAT=io_status)
  IF (io_status /= 0) THEN
    PRINT *, 'Error opening file. IOSTAT =', io_status
  ELSE
    READ(10, *, IOSTAT=io_status) num
    IF (io_status /= 0) THEN
      PRINT *, 'Error reading file. IOSTAT =', io_status
    ELSE
      PRINT *, 'Number read from file:', num
    END IF
    CLOSE(10)
  END IF
END PROGRAM IOErrorHandling
  • IOSTAT captures the status of the I/O operation.
  • ERR can be used to specify a label to jump to in case of an error.

Summary

In this section, we covered the basics and advanced aspects of input and output in Fortran. We learned how to:

  • Read from and write to the console.
  • Use formatted I/O for better control over data representation.
  • Perform file I/O operations.
  • Handle errors during I/O operations.

Understanding these concepts is crucial for developing robust Fortran programs that can interact with users and other systems effectively.
