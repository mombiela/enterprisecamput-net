Introduction

Fortran, short for "Formula Translation," is one of the oldest high-level programming languages, primarily used for numerical and scientific computing. This section will guide you through writing your first Fortran program, covering the basics of syntax, structure, and execution.

Key Concepts

  • Fortran Program Structure: Understanding the basic layout of a Fortran program.
  • Variables and Data Types: Introduction to declaring and using variables.
  • Basic Input and Output: How to read from and write to the console.
  • Compiling and Running: Steps to compile and execute your Fortran code.

Fortran Program Structure

A simple Fortran program consists of the following parts:

  1. Program Name: Declares the name of the program.
  2. Implicit None: Ensures all variables must be explicitly declared.
  3. Variable Declarations: Declares the variables used in the program.
  4. Executable Statements: Contains the logic and operations.
  5. End Program: Marks the end of the program.

Example

program HelloWorld
  implicit none
  ! Variable declarations go here

  ! Executable statements
  print *, 'Hello, World!'

end program HelloWorld

Explanation

  • program HelloWorld: Defines the start of the program named "HelloWorld".
  • implicit none: Ensures that all variables must be declared explicitly, which helps prevent errors.
  • print *, 'Hello, World!': Prints the string "Hello, World!" to the console.
  • end program HelloWorld: Marks the end of the program.

Variables and Data Types

Fortran supports several data types, including integers, real numbers, and characters.

Example

program VariablesExample
  implicit none
  integer :: age
  real :: height
  character(len=20) :: name

  ! Assigning values to variables
  age = 25
  height = 5.9
  name = 'John Doe'

  ! Printing variables
  print *, 'Name:', name
  print *, 'Age:', age
  print *, 'Height:', height

end program VariablesExample

Explanation

  • integer :: age: Declares an integer variable named age.
  • real :: height: Declares a real (floating-point) variable named height.
  • character(len=20) :: name: Declares a character variable named name with a length of 20.
  • The print statements output the values of the variables to the console.

Basic Input and Output

Fortran uses the read and print statements for input and output operations.

Example

program InputOutputExample
  implicit none
  integer :: age
  real :: height
  character(len=20) :: name

  ! Reading input from the user
  print *, 'Enter your name:'
  read *, name
  print *, 'Enter your age:'
  read *, age
  print *, 'Enter your height:'
  read *, height

  ! Printing the input values
  print *, 'Name:', name
  print *, 'Age:', age
  print *, 'Height:', height

end program InputOutputExample

Explanation

  • read *, name: Reads a string input from the user and stores it in the variable name.
  • read *, age: Reads an integer input from the user and stores it in the variable age.
  • read *, height: Reads a real number input from the user and stores it in the variable height.

Compiling and Running

To compile and run a Fortran program, you typically use a Fortran compiler like gfortran.

Steps

  1. Write the code: Save your Fortran code in a file with a .f90 extension (e.g., HelloWorld.f90).
  2. Compile the code: Use the command gfortran HelloWorld.f90 -o HelloWorld to compile the code.
  3. Run the executable: Execute the compiled program with ./HelloWorld.

Example

$ gfortran HelloWorld.f90 -o HelloWorld
$ ./HelloWorld
Hello, World!

Conclusion

In this section, you learned how to write a basic Fortran program, declare variables, perform input and output operations, and compile and run your code. These foundational skills are essential as you progress to more complex Fortran programming tasks. Practice writing simple programs to reinforce these concepts and build your confidence in using Fortran.
