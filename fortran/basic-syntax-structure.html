Introduction to Fortran

Fortran, short for "Formula Translation," is one of the oldest high-level programming languages, primarily used for numerical and scientific computing. Understanding its basic syntax and structure is crucial for writing efficient and error-free programs.

Key Concepts

  • Program Structure: Fortran programs are structured into units such as programs, subroutines, and functions.
  • Comments: Lines that are not executed, used for documentation.
  • Variables and Data Types: Fortran supports various data types including integers, real numbers, and characters.
  • Operators: Arithmetic, relational, and logical operators.
  • Control Structures: Conditional statements and loops.
  • Input/Output: Reading from and writing to files or the console.

Program Structure

A basic Fortran program consists of a program name, declarations, executable statements, and an end statement.

program HelloWorld
  implicit none
  print *, 'Hello, World!'
end program HelloWorld
  • program HelloWorld: Declares the program name.
  • implicit none: Ensures all variables must be explicitly declared.
  • print *, 'Hello, World!': Prints text to the console.
  • end program HelloWorld: Marks the end of the program.

Comments

Comments are used to explain code and are ignored during execution. In Fortran, comments start with an exclamation mark (!).

! This is a comment
print *, 'Hello, World!'  ! This prints Hello, World!

Variables and Data Types

Fortran supports several data types. Variables must be declared before use.

Common Data Types

  • Integer: Whole numbers.
  • Real: Floating-point numbers.
  • Character: Strings of text.
  • Logical: Boolean values (true or false).
program VariablesExample
  implicit none
  integer :: i
  real :: x
  character(len=20) :: name
  logical :: flag

  i = 10
  x = 3.14
  name = 'Fortran'
  flag = .true.

  print *, 'Integer:', i
  print *, 'Real:', x
  print *, 'Character:', name
  print *, 'Logical:', flag
end program VariablesExample

Operators

Fortran includes various operators for performing arithmetic, relational, and logical operations.

Arithmetic Operators

  • + Addition
  • - Subtraction
  • * Multiplication
  • / Division
  • ** Exponentiation

Relational Operators

  • == Equal to
  • /= Not equal to
  • > Greater than
  • < Less than
  • >= Greater than or equal to
  • <= Less than or equal to

Logical Operators

  • .and. Logical AND
  • .or. Logical OR
  • .not. Logical NOT
program OperatorsExample
  implicit none
  integer :: a, b
  logical :: result

  a = 5
  b = 3

  print *, 'Addition:', a + b
  print *, 'Multiplication:', a * b
  print *, 'Exponentiation:', a ** b

  result = (a > b) .and. (a /= 0)
  print *, 'Logical AND result:', result
end program OperatorsExample

Control Structures

Control structures allow for conditional execution and looping.

Conditional Statements

  • if statement
  • else if statement
  • else statement
program ConditionalExample
  implicit none
  integer :: num

  num = 10

  if (num > 0) then
    print *, 'Number is positive'
  else if (num == 0) then
    print *, 'Number is zero'
  else
    print *, 'Number is negative'
  end if
end program ConditionalExample

Loops

  • do loop
  • do while loop
program LoopExample
  implicit none
  integer :: i

  do i = 1, 5
    print *, 'Iteration:', i
  end do

  i = 1
  do while (i <= 5)
    print *, 'While loop iteration:', i
    i = i + 1
  end do
end program LoopExample

Input/Output

Fortran provides simple ways to handle input and output operations.

Reading Input

program InputExample
  implicit none
  integer :: num

  print *, 'Enter a number:'
  read *, num
  print *, 'You entered:', num
end program InputExample

Writing Output

program OutputExample
  implicit none
  integer :: num

  num = 42
  print *, 'The answer is:', num
end program OutputExample

Conclusion

Understanding the basic syntax and structure of Fortran is essential for writing effective programs. This includes knowing how to structure a program, declare variables, use operators, control the flow of the program, and handle input and output. With these fundamentals, you can start exploring more advanced features and capabilities of Fortran.
