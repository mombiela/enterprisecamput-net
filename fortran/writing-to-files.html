Writing to Files in Fortran
File Positioning ►

Writing to files is a fundamental skill in Fortran programming, allowing you to save data for later use, share results, or log information. This topic will guide you through the basics to advanced techniques of writing to files in Fortran.

Basic File Writing

Opening a File

To write to a file, you first need to open it using the OPEN statement. This statement associates a unit number with a file.

PROGRAM WriteToFile
  INTEGER :: unit_number
  unit_number = 10
  OPEN(UNIT=unit_number, FILE='output.txt', STATUS='NEW')
END PROGRAM WriteToFile
  • UNIT: A unique identifier for the file.
  • FILE: The name of the file.
  • STATUS: Specifies the file status (NEW, OLD, REPLACE, etc.).

Writing Data

Use the WRITE statement to write data to the file.

PROGRAM WriteToFile
  INTEGER :: unit_number
  unit_number = 10
  OPEN(UNIT=unit_number, FILE='output.txt', STATUS='NEW')
  WRITE(unit_number, *) 'Hello, Fortran!'
  CLOSE(unit_number)
END PROGRAM WriteToFile
  • WRITE(unit_number, *): Writes to the file associated with unit_number.
  • CLOSE(unit_number): Closes the file.

Example: Writing Multiple Lines

PROGRAM WriteMultipleLines
  INTEGER :: unit_number
  unit_number = 10
  OPEN(UNIT=unit_number, FILE='output.txt', STATUS='NEW')
  WRITE(unit_number, *) 'Line 1: Hello, Fortran!'
  WRITE(unit_number, *) 'Line 2: Writing to files is easy.'
  CLOSE(unit_number)
END PROGRAM WriteMultipleLines

Intermediate File Writing

Formatting Output

You can format the output using format specifiers.

PROGRAM WriteFormatted
  INTEGER :: unit_number
  unit_number = 10
  OPEN(UNIT=unit_number, FILE='output.txt', STATUS='NEW')
  WRITE(unit_number, '(A, I3)') 'Value: ', 42
  CLOSE(unit_number)
END PROGRAM WriteFormatted
  • '(A, I3)': Format specifier where A is for string and I3 is for an integer with a width of 3.

Writing Arrays

You can also write arrays to files.

PROGRAM WriteArray
  INTEGER :: unit_number
  INTEGER, DIMENSION(5) :: array = (/1, 2, 3, 4, 5/)
  unit_number = 10
  OPEN(UNIT=unit_number, FILE='output.txt', STATUS='NEW')
  WRITE(unit_number, '(5I3)') array
  CLOSE(unit_number)
END PROGRAM WriteArray
  • '(5I3)': Writes 5 integers, each with a width of 3.

Advanced File Writing

Handling Errors

Use the IOSTAT parameter to handle errors during file operations.

PROGRAM WriteWithErrorHandling
  INTEGER :: unit_number, io_status
  unit_number = 10
  OPEN(UNIT=unit_number, FILE='output.txt', STATUS='NEW', IOSTAT=io_status)
  IF (io_status /= 0) THEN
    PRINT *, 'Error opening file!'
    STOP
  END IF
  WRITE(unit_number, *) 'Hello, Fortran!'
  CLOSE(unit_number)
END PROGRAM WriteWithErrorHandling
  • IOSTAT=io_status: Stores the status of the I/O operation.

Appending to Files

To append data to an existing file, use the STATUS='OLD' and POSITION='APPEND' options.

PROGRAM AppendToFile
  INTEGER :: unit_number
  unit_number = 10
  OPEN(UNIT=unit_number, FILE='output.txt', STATUS='OLD', POSITION='APPEND')
  WRITE(unit_number, *) 'Appending this line.'
  CLOSE(unit_number)
END PROGRAM AppendToFile

Writing Complex Data Structures

For more complex data structures, such as derived types, you need to write each component individually.

MODULE DataModule
  TYPE :: Person
    CHARACTER(LEN=20) :: name
    INTEGER :: age
  END TYPE Person
END MODULE DataModule

PROGRAM WriteComplexData
  USE DataModule
  INTEGER :: unit_number
  TYPE(Person) :: person
  person = Person('John Doe', 30)
  unit_number = 10
  OPEN(UNIT=unit_number, FILE='output.txt', STATUS='NEW')
  WRITE(unit_number, '(A, I3)') person%name, person%age
  CLOSE(unit_number)
END PROGRAM WriteComplexData

Conclusion

Writing to files in Fortran involves several steps: opening the file, writing data, and closing the file. You can format the output, handle errors, append to files, and write complex data structures. Mastering these techniques will enable you to effectively manage data output in your Fortran programs.
