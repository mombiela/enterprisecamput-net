String Handling in Fortran
Array and String Operations ►

String handling is a crucial aspect of programming in Fortran, especially when dealing with text data. This topic will cover the basics of string handling, manipulation techniques, and advanced string operations in Fortran.

Introduction to Strings in Fortran

In Fortran, strings are sequences of characters. They are typically used to store and manipulate text data.

Declaring Strings

  • Strings in Fortran are declared using the CHARACTER keyword.
  • The length of the string can be specified.
CHARACTER(LEN=20) :: str
str = 'Hello, Fortran!'

Basic String Operations

  • Assignment: Assigning a value to a string variable.
  • Concatenation: Combining two or more strings.
  • Substring: Extracting a part of a string.
CHARACTER(LEN=20) :: str1, str2, result
str1 = 'Hello'
str2 = 'World'
result = str1 // ' ' // str2  ! Concatenation
PRINT *, result  ! Output: Hello World

result = str1(1:4)  ! Substring
PRINT *, result  ! Output: Hell

String Manipulation Techniques

String Length

  • The LEN function returns the length of a string.
CHARACTER(LEN=20) :: str
str = 'Fortran'
PRINT *, LEN(str)  ! Output: 20 (including trailing spaces)
PRINT *, LEN_TRIM(str)  ! Output: 7 (excluding trailing spaces)

Trimming Strings

  • The TRIM function removes trailing spaces from a string.
CHARACTER(LEN=20) :: str
str = 'Fortran    '
PRINT *, TRIM(str)  ! Output: 'Fortran'

String Comparison

  • Strings can be compared using relational operators (==, /=, <, >, <=, >=).
CHARACTER(LEN=10) :: str1, str2
str1 = 'abc'
str2 = 'def'
IF (str1 < str2) THEN
    PRINT *, 'str1 is less than str2'
ENDIF

Advanced String Operations

Intrinsic String Functions

Fortran provides several intrinsic functions for advanced string operations.

  • INDEX: Finds the position of a substring within a string.
  • ADJUSTL: Left-justifies a string.
  • ADJUSTR: Right-justifies a string.
CHARACTER(LEN=20) :: str
str = 'Hello, Fortran!'
PRINT *, INDEX(str, 'Fortran')  ! Output: 8

str = '   Fortran'
PRINT *, ADJUSTL(str)  ! Output: 'Fortran   '
PRINT *, ADJUSTR(str)  ! Output: '   Fortran'

Dynamic String Allocation

  • Fortran 2003 introduced the ability to allocate strings dynamically.
CHARACTER(:), ALLOCATABLE :: dyn_str
ALLOCATE(CHARACTER(LEN=30) :: dyn_str)
dyn_str = 'Dynamic String Allocation'
PRINT *, dyn_str
DEALLOCATE(dyn_str)

Exercises

  1. Basic String Operations: Declare two strings, concatenate them, and print the result.
  2. Substring Extraction: Extract and print a substring from a given string.
  3. String Comparison: Write a program to compare two strings and print which one is greater.
  4. Advanced Functions: Use the INDEX function to find the position of a substring within a string.

Conclusion

String handling in Fortran involves various operations such as assignment, concatenation, and substring extraction. Understanding these basics, along with advanced techniques like dynamic allocation and intrinsic functions, is essential for effective text data manipulation in Fortran. Practice these concepts through the provided exercises to solidify your understanding.
