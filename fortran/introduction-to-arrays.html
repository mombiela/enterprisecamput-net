Overview

Arrays are a fundamental data structure in Fortran, allowing you to store and manipulate collections of data efficiently. This section will introduce you to the basics of arrays, including their declaration, initialization, and common operations.

What is an Array?

  • An array is a collection of elements, all of the same type, stored in contiguous memory locations.
  • Arrays can be one-dimensional (like a list) or multi-dimensional (like a matrix).

Declaring Arrays

To declare an array in Fortran, you specify the type of elements and the dimensions of the array.

One-Dimensional Arrays

INTEGER :: array1D(10)
REAL :: array1D_real(5)
  • INTEGER :: array1D(10) declares an integer array with 10 elements.
  • REAL :: array1D_real(5) declares a real number array with 5 elements.

Multi-Dimensional Arrays

INTEGER :: array2D(3, 3)
REAL :: array3D(2, 2, 2)
  • INTEGER :: array2D(3, 3) declares a 3x3 integer array.
  • REAL :: array3D(2, 2, 2) declares a 2x2x2 real number array.

Initializing Arrays

Arrays can be initialized at the time of declaration or later in the code.

Initialization at Declaration

INTEGER :: array1D(5) = (/1, 2, 3, 4, 5/)
REAL :: array2D(2, 2) = RESHAPE((/1.0, 2.0, 3.0, 4.0/), (/2, 2/))
  • INTEGER :: array1D(5) = (/1, 2, 3, 4, 5/) initializes a 1D array with values 1 to 5.
  • REAL :: array2D(2, 2) = RESHAPE((/1.0, 2.0, 3.0, 4.0/), (/2, 2/)) initializes a 2x2 array with values 1.0, 2.0, 3.0, and 4.0.

Initialization Later in the Code

INTEGER :: array1D(5)
array1D = (/1, 2, 3, 4, 5/)

REAL :: array2D(2, 2)
array2D = RESHAPE((/1.0, 2.0, 3.0, 4.0/), (/2, 2/))

Accessing Array Elements

You can access and modify array elements using their indices.

Accessing Elements

INTEGER :: value
value = array1D(3)  ! Access the third element of array1D

Modifying Elements

array1D(3) = 10  ! Set the third element of array1D to 10

Common Array Operations

  • Array Traversal: Looping through array elements.
  • Array Slicing: Accessing subarrays.
  • Array Intrinsic Functions: Built-in functions for array manipulation.

Array Traversal

INTEGER :: i
DO i = 1, 5
    PRINT *, array1D(i)
END DO

Array Slicing

INTEGER, DIMENSION(5) :: subArray
subArray = array1D(2:4)  ! Extract elements 2 to 4

Array Intrinsic Functions

REAL :: sum
sum = SUM(array1D_real)  ! Calculate the sum of elements in array1D_real

Conclusion

Arrays are a powerful tool in Fortran for handling collections of data. Understanding how to declare, initialize, and manipulate arrays is essential for efficient programming. Practice these concepts with the provided examples and exercises to solidify your understanding.

Exercises

  1. Declare a one-dimensional integer array with 7 elements and initialize it with values from 1 to 7.
  2. Create a 3x3 real array and initialize it with values from 1.0 to 9.0.
  3. Write a loop to print all elements of the 3x3 array.
  4. Use array slicing to extract a subarray from the one-dimensional array created in exercise 1.
  5. Calculate the sum of elements in the 3x3 array using an intrinsic function.
