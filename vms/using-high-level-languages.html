Introduction

OpenVMS is a versatile operating system that supports various high-level programming languages, including C and Fortran. This section will guide you through the basics to advanced usage of these languages on OpenVMS, providing clear explanations, examples, and exercises.

Getting Started with High-Level Languages on OpenVMS

Setting Up Your Environment

  • Install the necessary compilers: Ensure that the C and Fortran compilers are installed on your OpenVMS system.
  • Set up environment variables: Configure your environment to use the compilers effectively. 
    $ SET DEFAULT SYS$SYSTEM
$ DEFINE CC$COMPILER CC
$ DEFINE FORTRAN$COMPILER FORTRAN

Writing Your First Program

C Example

  1. Create a simple C program: 
    #include <stdio.h>

int main() {
    printf("Hello, OpenVMS!\n");
    return 0;
}
  2. Save the file as hello.c.

Fortran Example

  1. Create a simple Fortran program: 
    PROGRAM HELLO
PRINT *, 'Hello, OpenVMS!'
END PROGRAM HELLO
  2. Save the file as hello.f90.

Compiling and Running Programs

  • Compile the C program: 
    $ CC hello.c
$ LINK hello
$ RUN hello
  • Compile the Fortran program: 
    $ FORTRAN hello.f90
$ LINK hello
$ RUN hello

Intermediate Concepts

Handling Input and Output

C Example

  • Reading input from the user: 
    #include <stdio.h>

int main() {
    char name[50];
    printf("Enter your name: ");
    scanf("%s", name);
    printf("Hello, %s!\n", name);
    return 0;
}

Fortran Example

  • Reading input from the user: 
    PROGRAM GREET
CHARACTER(LEN=50) :: name
PRINT *, 'Enter your name:'
READ *, name
PRINT *, 'Hello, ', TRIM(name), '!'
END PROGRAM GREET

File Operations

C Example

  • Reading from and writing to a file: 
    #include <stdio.h>

int main() {
    FILE *file;
    file = fopen("example.txt", "w");
    fprintf(file, "Hello, OpenVMS!\n");
    fclose(file);

    file = fopen("example.txt", "r");
    char line[100];
    fgets(line, sizeof(line), file);
    printf("%s", line);
    fclose(file);
    return 0;
}

Fortran Example

  • Reading from and writing to a file: 
    PROGRAM FILE_IO
INTEGER :: unit = 10
CHARACTER(LEN=100) :: line

OPEN(UNIT=unit, FILE='example.txt', STATUS='NEW')
WRITE(unit, *) 'Hello, OpenVMS!'
CLOSE(unit)

OPEN(UNIT=unit, FILE='example.txt', STATUS='OLD')
READ(unit, *) line
PRINT *, TRIM(line)
CLOSE(unit)
END PROGRAM FILE_IO

Advanced Topics

Interfacing C and Fortran

  • Calling Fortran from C: 
    extern void fortran_subroutine();

int main() {
    fortran_subroutine();
    return 0;
}
  • Fortran subroutine: 
    SUBROUTINE FORTRAN_SUBROUTINE
PRINT *, 'Called from C!'
END SUBROUTINE FORTRAN_SUBROUTINE

Performance Optimization

  • Compiler optimizations: Use compiler flags to optimize performance.
    • C: CC /OPTIMIZE=LEVEL=4 hello.c
    • Fortran: FORTRAN /OPTIMIZE=LEVEL=4 hello.f90

Debugging Techniques

  • Using the OpenVMS Debugger:
    • Start the debugger: 
      $ DEBUG hello
    • Set breakpoints and step through code: 
      DBG> SET BREAK main
DBG> GO
DBG> STEP

Conclusion

In this section, you have learned how to use high-level languages like C and Fortran on OpenVMS, from setting up your environment to writing, compiling, and running programs. You have also explored intermediate and advanced topics, including file operations, interfacing between languages, performance optimization, and debugging techniques. By practicing these concepts, you will become proficient in developing robust applications on the OpenVMS platform.
