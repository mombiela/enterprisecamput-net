Overview

OpenVMS (Open Virtual Memory System) is a multi-user, multiprocessing virtual memory-based operating system designed for use in time-sharing, batch processing, and transaction processing. This course will guide you through the basics of OpenVMS application development, from beginner to advanced levels.

Getting Started with OpenVMS

What is OpenVMS?

  • Multi-user and multiprocessing: Supports multiple users and processes simultaneously.
  • Virtual memory: Efficient memory management.
  • High availability: Designed for mission-critical applications.
  • Security: Robust security features.

Setting Up Your Environment

  • Accessing OpenVMS: Using terminal emulators or SSH.
  • Basic Commands: 
    $ SHOW SYSTEM
$ DIRECTORY
$ TYPE filename.txt

Basic File Operations

  • Creating Files: 
    $ CREATE filename.txt
  • Editing Files: Using editors like EDT or EVE. 
    $ EDIT/TPU filename.txt
  • Deleting Files: 
    $ DELETE filename.txt

Programming in OpenVMS

Supported Languages

  • C/C++
  • Fortran
  • Pascal
  • COBOL
  • BASIC

Writing Your First Program in C

  • Creating a Source File: 
    $ CREATE hello.c
  • Sample C Program: 
    #include <stdio.h>

int main() {
    printf("Hello, OpenVMS!\n");
    return 0;
}
  • Compiling and Running: 
    $ CC hello.c
$ LINK hello
$ RUN hello

Advanced Topics

System Services

  • Overview: System services are callable routines provided by OpenVMS for various system-level operations.
  • Example: 
    #include <lib$routines.h>
#include <starlet.h>

int main() {
    unsigned long status;
    status = lib$put_output(&"Hello, OpenVMS System Service!\n");
    return status;
}

Interprocess Communication

  • Mailboxes: Used for communication between processes. 
    $ CREATE/MAILBOX MBX1
$ WRITE MBX1 "Message to another process"

Using RMS (Record Management Services)

  • Overview: RMS is used for file handling and record management.
  • Example: 
    #include <rms.h>

struct FAB fab;
struct RAB rab;
char record[100];

int main() {
    fab = cc$rms_fab;
    fab.fab$l_fna = "example.dat";
    fab.fab$b_fns = strlen(fab.fab$l_fna);
    fab.fab$b_fac = FAB$M_BIO | FAB$M_GET;
    fab.fab$b_shr = FAB$M_SHRGET | FAB$M_SHRPUT;

    rab = cc$rms_rab;
    rab.rab$l_fab = &fab;
    rab.rab$l_rop = RAB$M_RAH | RAB$M_RRL;
    rab.rab$l_ubf = record;
    rab.rab$w_usz = sizeof(record);

    sys$open(&fab);
    sys$connect(&rab);
    sys$get(&rab);
    printf("Record: %s\n", record);
    sys$disconnect(&rab);
    sys$close(&fab);

    return 0;
}

Conclusion

OpenVMS is a powerful and versatile operating system suitable for various types of applications. This course has introduced you to the basics of OpenVMS application development, from setting up your environment to writing and running programs. As you progress, you will explore more advanced topics such as system services, interprocess communication, and RMS. With practice and exploration, you will become proficient in developing robust applications on OpenVMS.
