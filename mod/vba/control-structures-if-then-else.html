Introduction

Control structures are fundamental in programming as they allow you to dictate the flow of your program based on certain conditions. In VBA (Visual Basic for Applications), the If...Then...Else statement is a powerful tool for decision-making. This section will guide you from the basics to advanced usage of If...Then...Else in VBA.

Basic If...Then Statement

The simplest form of control structure is the If...Then statement. It allows you to execute a block of code only if a specified condition is true.

Syntax

If condition Then
    ' Code to execute if condition is true
End If

Example

Sub CheckValue()
    Dim value As Integer
    value = 10
    
    If value > 5 Then
        MsgBox "Value is greater than 5"
    End If
End Sub

In this example, the message box will display "Value is greater than 5" because the condition value > 5 is true.

If...Then...Else Statement

When you need to execute one block of code if a condition is true and another block if it is false, you use the If...Then...Else statement.

Syntax

If condition Then
    ' Code to execute if condition is true
Else
    ' Code to execute if condition is false
End If

Example

Sub CheckValue()
    Dim value As Integer
    value = 3
    
    If value > 5 Then
        MsgBox "Value is greater than 5"
    Else
        MsgBox "Value is 5 or less"
    End If
End Sub

In this example, the message box will display "Value is 5 or less" because the condition value > 5 is false.

If...Then...ElseIf...Else Statement

For multiple conditions, you can use the ElseIf keyword to check additional conditions.

Syntax

If condition1 Then
    ' Code to execute if condition1 is true
ElseIf condition2 Then
    ' Code to execute if condition2 is true
Else
    ' Code to execute if none of the above conditions are true
End If

Example

Sub CheckValue()
    Dim value As Integer
    value = 7
    
    If value > 10 Then
        MsgBox "Value is greater than 10"
    ElseIf value > 5 Then
        MsgBox "Value is greater than 5 but 10 or less"
    Else
        MsgBox "Value is 5 or less"
    End If
End Sub

In this example, the message box will display "Value is greater than 5 but 10 or less" because the condition value > 5 is true.

Nested If Statements

You can nest If...Then...Else statements within each other to handle more complex decision-making.

Example

Sub CheckValue()
    Dim value As Integer
    value = 12
    
    If value > 10 Then
        If value > 15 Then
            MsgBox "Value is greater than 15"
        Else
            MsgBox "Value is greater than 10 but 15 or less"
        End If
    Else
        MsgBox "Value is 10 or less"
    End If
End Sub

In this example, the message box will display "Value is greater than 10 but 15 or less" because the outer condition value > 10 is true, but the inner condition value > 15 is false.

Advanced Usage

Using Logical Operators

You can combine multiple conditions using logical operators such as And, Or, and Not.

Example

Sub CheckValue()
    Dim value As Integer
    value = 8
    
    If value > 5 And value < 10 Then
        MsgBox "Value is between 5 and 10"
    Else
        MsgBox "Value is not between 5 and 10"
    End If
End Sub

In this example, the message box will display "Value is between 5 and 10" because both conditions value > 5 and value < 10 are true.

Conclusion

The If...Then...Else statement is a versatile control structure in VBA that allows you to make decisions based on conditions. By understanding and utilizing its various forms, from simple If...Then statements to complex nested and logical condition checks, you can control the flow of your VBA programs effectively. Practice with different scenarios to become proficient in using these control structures.
