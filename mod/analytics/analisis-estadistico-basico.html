Introduction

Basic statistical analysis is fundamental for any professional in the field of analytics. This topic covers the essential concepts and basic techniques that will allow you to interpret and analyze data effectively.

Key Concepts

  • Population and Sample:

    • Population: Complete set of elements that you want to study.
    • Sample: Subset of the population used for analysis.

  • Variables:

    • Qualitative: Describe categories or attributes (e.g., color, gender).
    • Quantitative: Describe numerical quantities (e.g., height, weight).

  • Measures of Central Tendency:

    • Mean: Arithmetic average of a data set.
    • Median: Central value of an ordered data set.
    • Mode: Value that appears most frequently in a data set.

  • Measures of Dispersion:

    • Range: Difference between the maximum and minimum values.
    • Variance: Measure of data dispersion relative to the mean.
    • Standard Deviation: Square root of the variance, indicates how much data deviates from the mean.

Practical Examples

Example 1: Calculation of Mean, Median, and Mode

Suppose we have the following data set representing the ages of a group of people: [23, 29, 20, 32, 23, 21, 23, 25, 29].

  • Mean:

    ages = [23, 29, 20, 32, 23, 21, 23, 25, 29]
mean = sum(ages) / len(ages)
print("Mean:", mean)

    Result: Mean: 25.0

  • Median:

    ages.sort()
n = len(ages)
median = ages[n//2] if n % 2 != 0 else (ages[n//2 - 1] + ages[n//2]) / 2
print("Median:", median)

    Result: Median: 23

  • Mode:

    from collections import Counter
mode = Counter(ages).most_common(1)[0][0]
print("Mode:", mode)

    Result: Mode: 23

Example 2: Calculation of Variance and Standard Deviation

Continuing with the same data set:

  • Variance:

    mean = sum(ages) / len(ages)
variance = sum((x - mean) ** 2 for x in ages) / len(ages)
print("Variance:", variance)

    Result: Variance: 14.0

  • Standard Deviation:

    standard_deviation = variance ** 0.5
print("Standard Deviation:", standard_deviation)

    Result: Standard Deviation: 3.74

Comparison of Measures of Central Tendency and Dispersion

| Measure | Description | Formula | |-----------------------|--------------------------------------------------|----------------------------------------------| | Mean | Arithmetic average | \(\bar{x} = \frac{\sum{x_i}}{n}\) | | Median | Central value | - | | Mode | Most frequent value | - | | Range | Difference between the maximum and minimum values| \(R = x_{max} - x_{min}\) | | Variance | Dispersion relative to the mean | \(\sigma^2 = \frac{\sum{(x_i - \bar{x})^2}}{n}\) | | Standard Deviation | Square root of the variance | \(\sigma = \sqrt{\sigma^2}\) |

Conclusion

Basic statistical analysis provides the fundamental tools to interpret and analyze data effectively. Understanding and applying concepts such as mean, median, mode, variance, and standard deviation is crucial for any professional in the field of analytics. Practicing these concepts with practical examples and code will help you consolidate your understanding and skills in statistical analysis.
