Introduction

Mathematical functions are essential in programming for performing various calculations. In C, the standard library provides a wide range of mathematical functions that can be used to perform operations such as trigonometry, exponentiation, and logarithms.

Key Concepts

  • Standard Library: The math.h header file contains the declarations for mathematical functions.
  • Function Prototypes: Understanding the prototypes of these functions is crucial for their correct usage.
  • Return Types: Most mathematical functions return double values.
  • Error Handling: Some functions may return special values like NaN (Not a Number) or INF (Infinity) in case of errors.

Common Mathematical Functions

Basic Arithmetic Functions

  • fabs(double x): Returns the absolute value of x.
  • fmod(double x, double y): Returns the remainder of x divided by y.
#include <stdio.h>
#include <math.h>

int main() {
    double x = -5.3;
    double y = 2.0;

    printf("Absolute value of %.2f is %.2f\n", x, fabs(x));
    printf("Remainder of %.2f divided by %.2f is %.2f\n", x, y, fmod(x, y));

    return 0;
}

Exponential and Logarithmic Functions

  • exp(double x): Returns e raised to the power of x.
  • log(double x): Returns the natural logarithm (base e) of x.
  • log10(double x): Returns the base-10 logarithm of x.
#include <stdio.h>
#include <math.h>

int main() {
    double x = 2.0;

    printf("e raised to the power of %.2f is %.2f\n", x, exp(x));
    printf("Natural logarithm of %.2f is %.2f\n", x, log(x));
    printf("Base-10 logarithm of %.2f is %.2f\n", x, log10(x));

    return 0;
}

Power Functions

  • pow(double base, double exponent): Returns base raised to the power of exponent.
  • sqrt(double x): Returns the square root of x.
#include <stdio.h>
#include <math.h>

int main() {
    double base = 3.0;
    double exponent = 4.0;

    printf("%.2f raised to the power of %.2f is %.2f\n", base, exponent, pow(base, exponent));
    printf("Square root of %.2f is %.2f\n", base, sqrt(base));

    return 0;
}

Trigonometric Functions

  • sin(double x): Returns the sine of x (x in radians).
  • cos(double x): Returns the cosine of x (x in radians).
  • tan(double x): Returns the tangent of x (x in radians).
#include <stdio.h>
#include <math.h>

int main() {
    double angle = M_PI / 4; // 45 degrees in radians

    printf("Sine of %.2f radians is %.2f\n", angle, sin(angle));
    printf("Cosine of %.2f radians is %.2f\n", angle, cos(angle));
    printf("Tangent of %.2f radians is %.2f\n", angle, tan(angle));

    return 0;
}

Hyperbolic Functions

  • sinh(double x): Returns the hyperbolic sine of x.
  • cosh(double x): Returns the hyperbolic cosine of x.
  • tanh(double x): Returns the hyperbolic tangent of x.
#include <stdio.h>
#include <math.h>

int main() {
    double x = 1.0;

    printf("Hyperbolic sine of %.2f is %.2f\n", x, sinh(x));
    printf("Hyperbolic cosine of %.2f is %.2f\n", x, cosh(x));
    printf("Hyperbolic tangent of %.2f is %.2f\n", x, tanh(x));

    return 0;
}

Error Handling

  • Domain Errors: Occur when input values are outside the domain over which the function is defined.
  • Range Errors: Occur when the result of a function cannot be represented within the range of the return type.

Example of Error Handling

#include <stdio.h>
#include <math.h>
#include <errno.h>

int main() {
    errno = 0; // Reset global error indicator
    double result = sqrt(-1.0);

    if (errno == EDOM) {
        printf("Domain error occurred\n");
    } else {
        printf("Square root is %.2f\n", result);
    }

    return 0;
}

Conclusion

Mathematical functions in C are powerful tools for performing a wide range of calculations. By understanding the standard library functions, their prototypes, and how to handle errors, you can effectively utilize these functions in your programs. Practice using these functions with different inputs to gain a deeper understanding and become proficient in their usage.
