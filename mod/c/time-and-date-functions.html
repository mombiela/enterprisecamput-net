Time and Date Functions in C
Code Readability ►

Introduction

Time and date functions are essential in many programming tasks, such as logging events, measuring time intervals, and scheduling tasks. In C, the <time.h> library provides a variety of functions to work with dates and times.

Key Concepts

  • Epoch Time: The number of seconds that have elapsed since 00:00:00 UTC on 1 January 1970.
  • time_t: A data type used to represent time.
  • struct tm: A structure used to hold a date and time.

Basic Time Functions

Getting the Current Time

To get the current time, you can use the time() function, which returns the current time as a time_t value.

#include <stdio.h>
#include <time.h>

int main() {
    time_t current_time;
    current_time = time(NULL);
    printf("Current time: %ld\n", current_time);
    return 0;
}
  • time(NULL) returns the current time.
  • current_time holds the number of seconds since the Epoch.

Converting time_t to Human-Readable Format

To convert time_t to a human-readable format, use the ctime() function.

#include <stdio.h>
#include <time.h>

int main() {
    time_t current_time;
    current_time = time(NULL);
    printf("Current time: %s", ctime(&current_time));
    return 0;
}
  • ctime(&current_time) converts time_t to a string representing the local time.

Working with struct tm

Breaking Down time_t into Components

The localtime() function converts time_t to a struct tm representing the local time.

#include <stdio.h>
#include <time.h>

int main() {
    time_t current_time;
    struct tm *local_time;
    
    current_time = time(NULL);
    local_time = localtime(&current_time);
    
    printf("Year: %d\n", local_time->tm_year + 1900);
    printf("Month: %d\n", local_time->tm_mon + 1);
    printf("Day: %d\n", local_time->tm_mday);
    printf("Hour: %d\n", local_time->tm_hour);
    printf("Minute: %d\n", local_time->tm_min);
    printf("Second: %d\n", local_time->tm_sec);
    
    return 0;
}
  • localtime(&current_time) returns a pointer to a struct tm.
  • tm_year is the number of years since 1900.
  • tm_mon is the number of months since January (0-11).

Formatting Time and Date

The strftime() function formats the time and date according to a specified format.

#include <stdio.h>
#include <time.h>

int main() {
    time_t current_time;
    struct tm *local_time;
    char buffer[80];
    
    current_time = time(NULL);
    local_time = localtime(&current_time);
    
    strftime(buffer, 80, "Today is %A, %B %d, %Y", local_time);
    printf("%s\n", buffer);
    
    return 0;
}
  • strftime(buffer, 80, "format", local_time) formats the date and time.
  • Common format specifiers:
    • %A: Full weekday name.
    • %B: Full month name.
    • %d: Day of the month.
    • %Y: Year with century.

Advanced Time Functions

Measuring Time Intervals

To measure time intervals, use the clock() function, which returns the processor time consumed by the program.

#include <stdio.h>
#include <time.h>

int main() {
    clock_t start, end;
    double cpu_time_used;
    
    start = clock();
    // Code to measure
    for (int i = 0; i < 1000000; i++);
    end = clock();
    
    cpu_time_used = ((double) (end - start)) / CLOCKS_PER_SEC;
    printf("Time used: %f seconds\n", cpu_time_used);
    
    return 0;
}
  • clock() returns the number of clock ticks since the program started.
  • CLOCKS_PER_SEC is the number of clock ticks per second.

Manipulating Time

The mktime() function converts a struct tm back to time_t.

#include <stdio.h>
#include <time.h>

int main() {
    struct tm time_info = {0};
    time_t future_time;
    
    time_info.tm_year = 2023 - 1900; // Year since 1900
    time_info.tm_mon = 11 - 1; // Month (0-11)
    time_info.tm_mday = 25; // Day of the month
    
    future_time = mktime(&time_info);
    printf("Future time: %s", ctime(&future_time));
    
    return 0;
}
  • mktime(&time_info) converts struct tm to time_t.

Summary

  • The <time.h> library provides essential functions for working with time and date in C.
  • time_t and struct tm are key data types for handling time.
  • Functions like time(), ctime(), localtime(), and strftime() help in getting and formatting the current time.
  • Advanced functions like clock() and mktime() are useful for measuring time intervals and manipulating time.

Understanding these functions will enable you to handle time and date effectively in your C programs.
