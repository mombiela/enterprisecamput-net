Introduction

Stacks and queues are fundamental data structures in computer science. They are used to store and manage data in a particular order. Understanding these structures is crucial for solving various computational problems efficiently.

Stacks

Key Concepts

  • LIFO (Last In, First Out): The last element added to the stack is the first one to be removed.
  • Push: Adding an element to the top of the stack.
  • Pop: Removing the top element from the stack.
  • Peek/Top: Viewing the top element without removing it.

Stack Operations in C

Structure Definition

#define MAX 100

typedef struct {
    int data[MAX];
    int top;
} Stack;

Initialize Stack

void initialize(Stack *s) {
    s->top = -1;
}

Push Operation

void push(Stack *s, int value) {
    if (s->top == MAX - 1) {
        printf("Stack overflow\n");
        return;
    }
    s->data[++(s->top)] = value;
}

Pop Operation

int pop(Stack *s) {
    if (s->top == -1) {
        printf("Stack underflow\n");
        return -1; // Return an error value
    }
    return s->data[(s->top)--];
}

Peek Operation

int peek(Stack *s) {
    if (s->top == -1) {
        printf("Stack is empty\n");
        return -1; // Return an error value
    }
    return s->data[s->top];
}

Example Usage

#include <stdio.h>

int main() {
    Stack s;
    initialize(&s);

    push(&s, 10);
    push(&s, 20);
    push(&s, 30);

    printf("Top element is %d\n", peek(&s));
    printf("Popped element is %d\n", pop(&s));
    printf("Top element is %d\n", peek(&s));

    return 0;
}

Queues

Key Concepts

  • FIFO (First In, First Out): The first element added to the queue is the first one to be removed.
  • Enqueue: Adding an element to the end of the queue.
  • Dequeue: Removing the front element from the queue.
  • Front: Viewing the front element without removing it.
  • Rear: Viewing the rear element without removing it.

Queue Operations in C

Structure Definition

#define MAX 100

typedef struct {
    int data[MAX];
    int front;
    int rear;
} Queue;

Initialize Queue

void initialize(Queue *q) {
    q->front = -1;
    q->rear = -1;
}

Enqueue Operation

void enqueue(Queue *q, int value) {
    if (q->rear == MAX - 1) {
        printf("Queue overflow\n");
        return;
    }
    if (q->front == -1) q->front = 0;
    q->data[++(q->rear)] = value;
}

Dequeue Operation

int dequeue(Queue *q) {
    if (q->front == -1 || q->front > q->rear) {
        printf("Queue underflow\n");
        return -1; // Return an error value
    }
    return q->data[(q->front)++];
}

Front Operation

int front(Queue *q) {
    if (q->front == -1 || q->front > q->rear) {
        printf("Queue is empty\n");
        return -1; // Return an error value
    }
    return q->data[q->front];
}

Example Usage

#include <stdio.h>

int main() {
    Queue q;
    initialize(&q);

    enqueue(&q, 10);
    enqueue(&q, 20);
    enqueue(&q, 30);

    printf("Front element is %d\n", front(&q));
    printf("Dequeued element is %d\n", dequeue(&q));
    printf("Front element is %d\n", front(&q));

    return 0;
}

Comparison of Stacks and Queues

| Feature | Stack (LIFO) | Queue (FIFO) | |---------------|---------------------|---------------------| | Insertion | Push (top) | Enqueue (rear) | | Removal | Pop (top) | Dequeue (front) | | Access | Peek/Top (top) | Front (front) |

Conclusion

Stacks and queues are essential data structures that help manage data in specific orders. Stacks follow the LIFO principle, making them suitable for tasks like reversing data and backtracking algorithms. Queues follow the FIFO principle, making them ideal for scheduling tasks and managing buffers. Understanding how to implement and use these structures in C is fundamental for efficient programming and problem-solving.
