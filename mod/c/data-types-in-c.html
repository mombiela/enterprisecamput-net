Introduction

Data types in C are fundamental to understanding how to store, manipulate, and use data in your programs. This section will cover the basic and advanced data types in C, their usage, and how to choose the appropriate data type for your needs.

Basic Data Types

C provides several basic data types that are used to define variables and the type of data they can hold.

Integer Types

  • int: Used to store integers (whole numbers).
  • short: A shorter version of int, typically using less memory.
  • long: A longer version of int, typically using more memory.
  • long long: An even longer version of int for very large numbers.
int a = 10;
short b = 20;
long c = 30000;
long long d = 4000000000;

Floating-Point Types

  • float: Used to store single-precision floating-point numbers.
  • double: Used to store double-precision floating-point numbers.
  • long double: Used for extended precision floating-point numbers.
float e = 3.14f;
double f = 3.141592653589793;
long double g = 3.14159265358979323846L;

Character Type

  • char: Used to store single characters. It can also be used to store small integers.
char h = 'A';
char i = 65; // ASCII value of 'A'

Boolean Type

  • _Bool: Used to store boolean values (true or false).
_Bool j = 1; // true
_Bool k = 0; // false

Derived Data Types

Derived data types are built from the basic data types and include arrays, pointers, structures, and unions.

Arrays

Arrays are collections of elements of the same type.

int arr[5] = {1, 2, 3, 4, 5};
char str[] = "Hello";

Pointers

Pointers store the memory address of another variable.

int l = 10;
int *ptr = &l;

Structures

Structures are used to group different types of variables under a single name.

struct Person {
    char name[50];
    int age;
};

struct Person person1;
person1.age = 30;

Unions

Unions are similar to structures but use the same memory location for all its members.

union Data {
    int intVal;
    float floatVal;
    char charVal;
};

union Data data;
data.intVal = 10;

Enumerated Types

Enumerated types (enums) are user-defined types that consist of named integer constants.

enum Weekday {Sunday, Monday, Tuesday, Wednesday, Thursday, Friday, Saturday};
enum Weekday today = Wednesday;

Type Qualifiers

Type qualifiers add special properties to variables.

  • const: Indicates that the value of the variable cannot be changed.
  • volatile: Indicates that the value of the variable may change at any time without any action being taken by the code.
  • restrict: A pointer qualifier that indicates that the pointer is the only reference to that memory location.
const int m = 100;
volatile int n = 200;
int * restrict p = &n;

Summary

Understanding data types in C is crucial for efficient programming. Basic data types include integers, floating-point numbers, characters, and booleans. Derived data types include arrays, pointers, structures, and unions, while enumerated types allow for user-defined named constants. Type qualifiers add additional properties to variables, enhancing control over how data is used and stored. By mastering these concepts, you can write more efficient and effective C programs.
