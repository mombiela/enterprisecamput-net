Introduction

In C programming, variables and constants are fundamental concepts that you need to understand to write effective and efficient code. This section will cover what variables and constants are, how to declare and use them, and the differences between them.

Variables

What is a Variable?

  • A variable is a storage location in memory with a specific type that determines what kind of data it can hold.
  • Variables have names (identifiers) that are used to access their stored values.

Declaring Variables

  • To declare a variable, you need to specify its type followed by its name.
  • Syntax: type variable_name;
int age;
float salary;
char initial;

Initializing Variables

  • You can initialize a variable at the time of declaration.
  • Syntax: type variable_name = value;
int age = 25;
float salary = 50000.50;
char initial = 'A';

Using Variables

  • Variables can be used in expressions and statements.
  • Example:
#include <stdio.h>

int main() {
    int age = 25;
    float salary = 50000.50;
    char initial = 'A';

    printf("Age: %d\n", age);
    printf("Salary: %.2f\n", salary);
    printf("Initial: %c\n", initial);

    return 0;
}

Constants

What is a Constant?

  • A constant is a value that cannot be altered by the program during its execution.
  • Constants are used to define fixed values that do not change.

Declaring Constants

  • In C, you can declare constants using the const keyword or #define preprocessor directive.

Using const Keyword

  • Syntax: const type constant_name = value;
const int DAYS_IN_WEEK = 7;
const float PI = 3.14159;
const char NEWLINE = '\n';

Using #define Preprocessor Directive

  • Syntax: #define CONSTANT_NAME value
#define DAYS_IN_WEEK 7
#define PI 3.14159
#define NEWLINE '\n'

Using Constants

  • Constants can be used in expressions and statements just like variables.
  • Example:
#include <stdio.h>

#define PI 3.14159

int main() {
    const int DAYS_IN_WEEK = 7;

    printf("Days in a week: %d\n", DAYS_IN_WEEK);
    printf("Value of PI: %.5f\n", PI);

    return 0;
}

Differences Between Variables and Constants

| Feature | Variables | Constants | |-----------------|---------------------------------|-----------------------------------| | Mutability | Can change during program execution | Cannot change once defined | | Declaration | type variable_name; | const type constant_name = value; or #define CONSTANT_NAME value | | Usage | Used to store data that may change | Used to store fixed values |

Conclusion

Understanding variables and constants is crucial for managing data in your C programs. Variables allow you to store and manipulate data that can change, while constants provide a way to define values that remain the same throughout the program. By mastering these concepts, you will be able to write more robust and maintainable code.
