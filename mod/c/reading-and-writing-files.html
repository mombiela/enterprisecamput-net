Introduction

In C programming, handling files is a crucial skill for reading data from files and writing data to files. This topic will cover the basics of file operations, including opening, reading, writing, and closing files. By the end of this section, you will be able to perform basic file I/O operations in C.

File Operations in C

Key Concepts

  • File Pointer: A pointer of type FILE used to keep track of the file being accessed.
  • Opening a File: Using fopen() to open a file.
  • Closing a File: Using fclose() to close a file.
  • Reading from a File: Using functions like fgetc(), fgets(), and fread().
  • Writing to a File: Using functions like fputc(), fputs(), and fwrite().

Opening and Closing Files

To work with files in C, you need to open them first. This is done using the fopen() function, which returns a file pointer. After performing the necessary operations, you should close the file using fclose().

Syntax

FILE *fopen(const char *filename, const char *mode);
int fclose(FILE *stream);

Modes for fopen()

| Mode | Description | |------|-------------| | "r" | Open for reading. The file must exist. | | "w" | Open for writing. Creates a new file if it doesn't exist or truncates the file if it does. | | "a" | Open for appending. Data is added to the end of the file. | | "r+" | Open for both reading and writing. The file must exist. | | "w+" | Open for both reading and writing. Creates a new file if it doesn't exist or truncates the file if it does. | | "a+" | Open for both reading and appending. Creates a new file if it doesn't exist. |

Example

#include <stdio.h>

int main() {
    FILE *file = fopen("example.txt", "w");
    if (file == NULL) {
        printf("Error opening file!\n");
        return 1;
    }
    // Perform file operations
    fclose(file);
    return 0;
}

Reading from a File

There are several functions to read data from a file, each suitable for different types of data and use cases.

fgetc()

Reads a single character from the file.

int fgetc(FILE *stream);

fgets()

Reads a string from the file.

char *fgets(char *str, int n, FILE *stream);

fread()

Reads a block of data from the file.

size_t fread(void *ptr, size_t size, size_t nmemb, FILE *stream);

Example

#include <stdio.h>

int main() {
    FILE *file = fopen("example.txt", "r");
    if (file == NULL) {
        printf("Error opening file!\n");
        return 1;
    }
    
    char ch;
    while ((ch = fgetc(file)) != EOF) {
        putchar(ch);
    }
    
    fclose(file);
    return 0;
}

Writing to a File

Similar to reading, there are several functions to write data to a file.

fputc()

Writes a single character to the file.

int fputc(int char, FILE *stream);

fputs()

Writes a string to the file.

int fputs(const char *str, FILE *stream);

fwrite()

Writes a block of data to the file.

size_t fwrite(const void *ptr, size_t size, size_t nmemb, FILE *stream);

Example

#include <stdio.h>

int main() {
    FILE *file = fopen("example.txt", "w");
    if (file == NULL) {
        printf("Error opening file!\n");
        return 1;
    }
    
    fputs("Hello, World!\n", file);
    
    fclose(file);
    return 0;
}

Conclusion

Understanding how to read from and write to files in C is essential for many programming tasks. This topic covered the basics of file operations, including opening, reading, writing, and closing files. By practicing these concepts, you will be able to handle file I/O operations effectively in your C programs.
