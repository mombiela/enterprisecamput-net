Introduction

String handling is a fundamental aspect of programming in C. Strings in C are arrays of characters terminated by a null character ('\0'). This topic will cover the basics of string handling, including declaration, initialization, standard library functions, and advanced string manipulation techniques.

1. Basics of Strings in C

1.1 Declaration and Initialization

  • Strings are declared as arrays of characters.
  • They must be terminated with a null character ('\0').
char str1[20]; // Declaration of a string with a maximum size of 20 characters
char str2[] = "Hello, World!"; // Initialization with a string literal

1.2 Input and Output of Strings

  • Use scanf and printf for basic input and output.
  • gets and puts can also be used, but gets is unsafe and should be avoided.
char name[50];
printf("Enter your name: ");
scanf("%s", name); // Note: scanf stops reading at the first whitespace
printf("Hello, %s!\n", name);

2. Standard Library Functions for Strings

2.1 `strlen` - String Length

  • Returns the length of the string (excluding the null character).
#include <stdio.h>
#include <string.h>

char str[] = "Hello";
int length = strlen(str); // length = 5

2.2 `strcpy` - String Copy

  • Copies one string to another.
#include <stdio.h>
#include <string.h>

char src[] = "Hello";
char dest[20];
strcpy(dest, src); // dest now contains "Hello"

2.3 `strcat` - String Concatenation

  • Appends one string to the end of another.
#include <stdio.h>
#include <string.h>

char str1[20] = "Hello";
char str2[] = " World";
strcat(str1, str2); // str1 now contains "Hello World"

2.4 `strcmp` - String Comparison

  • Compares two strings lexicographically.
#include <stdio.h>
#include <string.h>

char str1[] = "Hello";
char str2[] = "World";
int result = strcmp(str1, str2); // result < 0 because "Hello" is less than "World"

2.5 `strchr` and `strrchr` - Character Search

  • strchr finds the first occurrence of a character.
  • strrchr finds the last occurrence of a character.
#include <stdio.h>
#include <string.h>

char str[] = "Hello, World!";
char *first = strchr(str, 'o'); // first points to the first 'o' in "Hello, World!"
char *last = strrchr(str, 'o'); // last points to the last 'o' in "Hello, World!"

3. Advanced String Manipulation

3.1 Dynamic Strings

  • Use pointers and dynamic memory allocation for strings whose size is not known at compile time.
#include <stdio.h>
#include <stdlib.h>
#include <string.h>

char *dynamicStr = (char *)malloc(50 * sizeof(char));
if (dynamicStr != NULL) {
    strcpy(dynamicStr, "Dynamic String");
    printf("%s\n", dynamicStr);
    free(dynamicStr); // Always free dynamically allocated memory
}

3.2 Tokenizing Strings

  • Use strtok to split a string into tokens based on delimiters.
#include <stdio.h>
#include <string.h>

char str[] = "Hello, World! Welcome to C programming.";
char *token = strtok(str, " ,.!");

while (token != NULL) {
    printf("%s\n", token);
    token = strtok(NULL, " ,.!"); // Continue tokenizing the rest of the string
}

Conclusion

String handling in C involves understanding how strings are represented as arrays of characters and using various standard library functions to manipulate them. From basic operations like copying and concatenation to more advanced techniques like dynamic allocation and tokenization, mastering string handling is essential for effective C programming. Always remember to handle strings carefully to avoid common pitfalls such as buffer overflows and memory leaks.
