What are Structures in C?

Structures in C are user-defined data types that allow you to combine data items of different kinds. Structures are used to represent a record, for example, a student record might contain a name, age, and marks.

Key Concepts

  • Definition: A structure is defined using the struct keyword.
  • Members: The variables declared inside a structure are called members.
  • Accessing Members: Members of a structure are accessed using the dot (.) operator.
  • Initialization: Structures can be initialized at the time of declaration.

Defining a Structure

To define a structure, you use the struct keyword followed by the structure name and the structure members enclosed in curly braces {}.

struct Student {
    char name[50];
    int age;
    float marks;
};

In this example:

  • Student is the name of the structure.
  • name, age, and marks are members of the structure.

Declaring Structure Variables

After defining a structure, you can declare variables of that type.

struct Student student1, student2;

Here, student1 and student2 are variables of type struct Student.

Accessing Structure Members

You can access the members of a structure using the dot operator (.).

student1.age = 20;
student1.marks = 85.5;
strcpy(student1.name, "John Doe");

In this example:

  • student1.age sets the age of student1 to 20.
  • student1.marks sets the marks of student1 to 85.5.
  • strcpy(student1.name, "John Doe") copies the string "John Doe" into the name member of student1.

Initializing Structures

Structures can be initialized at the time of declaration.

struct Student student1 = {"John Doe", 20, 85.5};

This initializes student1 with the name "John Doe", age 20, and marks 85.5.

Example Program

Here is a complete example program that demonstrates the use of structures.

#include <stdio.h>
#include <string.h>

// Define the structure
struct Student {
    char name[50];
    int age;
    float marks;
};

int main() {
    // Declare and initialize structure variables
    struct Student student1 = {"John Doe", 20, 85.5};
    struct Student student2;

    // Assign values to student2
    strcpy(student2.name, "Jane Smith");
    student2.age = 22;
    student2.marks = 90.0;

    // Print student1 details
    printf("Student 1:\n");
    printf("Name: %s\n", student1.name);
    printf("Age: %d\n", student1.age);
    printf("Marks: %.2f\n", student1.marks);

    // Print student2 details
    printf("Student 2:\n");
    printf("Name: %s\n", student2.name);
    printf("Age: %d\n", student2.age);
    printf("Marks: %.2f\n", student2.marks);

    return 0;
}

Conclusion

Structures in C provide a way to group related data items of different types into a single unit. This is particularly useful for representing complex data structures like records. By understanding how to define, declare, initialize, and access structures, you can manage and manipulate related data more effectively in your programs.
