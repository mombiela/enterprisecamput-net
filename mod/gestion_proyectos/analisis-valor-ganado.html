Earned Value Analysis (EVA) is a project management technique that allows for the objective measurement of project performance and progress. This technique combines scope, schedule, and cost measurements to provide an integrated view of project performance.

Key Concepts of Earned Value Analysis

  • Planned Value (PV): Also known as BCWS (Budgeted Cost of Work Scheduled), it is the value of the work that was expected to be completed at a specific point in the project.
  • Earned Value (EV): Also known as BCWP (Budgeted Cost of Work Performed), it is the value of the work actually completed to date.
  • Actual Cost (AC): Also known as ACWP (Actual Cost of Work Performed), it is the actual cost incurred for the work performed to date.

Performance Indicators

  • Cost Variance (CV): Measures the difference between earned value and actual cost.

    • Formula: CV = EV - AC
    • Interpretation: A positive CV indicates that the project is under budget, while a negative CV indicates that it is over budget.

  • Schedule Variance (SV): Measures the difference between earned value and planned value.

    • Formula: SV = EV - PV
    • Interpretation: A positive SV indicates that the project is ahead of schedule, while a negative SV indicates that it is behind schedule.

  • Cost Performance Index (CPI): Measures the cost efficiency of the project.

    • Formula: CPI = EV / AC
    • Interpretation: A CPI greater than 1 indicates that the project is using resources efficiently, while a CPI less than 1 indicates inefficiency.

  • Schedule Performance Index (SPI): Measures the schedule efficiency of the project.

    • Formula: SPI = EV / PV
    • Interpretation: An SPI greater than 1 indicates that the project is ahead of schedule, while an SPI less than 1 indicates a delay.

Practical Example

Suppose we are managing a project with the following data:

  • Planned Value (PV) to date: $100,000
  • Earned Value (EV) to date: $90,000
  • Actual Cost (AC) to date: $85,000

Calculation of Indicators

  • CV = EV - AC

    CV = $90,000 - $85,000 = $5,000

    Interpretation: The project is $5,000 under budget.

  • SV = EV - PV

    SV = $90,000 - $100,000 = -$10,000

    Interpretation: The project is $10,000 behind in terms of planned value.

  • CPI = EV / AC

    CPI = $90,000 / $85,000 ≈ 1.06

    Interpretation: The project is using resources efficiently.

  • SPI = EV / PV

    SPI = $90,000 / $100,000 = 0.9

    Interpretation: The project is behind schedule.

Conclusion

Earned Value Analysis is a powerful tool for project management as it provides a clear and objective view of project performance in terms of cost and schedule. By using key indicators such as CV, SV, CPI, and SPI, project managers can make informed decisions to keep the project on track and ensure its success.
