Introduction

File and directory operations are fundamental tasks in Bash scripting. This section will cover the basics of creating, deleting, moving, and copying files and directories, as well as more advanced topics like file permissions and symbolic links.

Basic File Operations

Creating Files

  • touch: Create an empty file or update the timestamp of an existing file.

    touch filename.txt

Viewing File Contents

  • cat: Concatenate and display file content.

    cat filename.txt

  • less: View file content one page at a time.

    less filename.txt

  • head: Display the first few lines of a file.

    head filename.txt

  • tail: Display the last few lines of a file.

    tail filename.txt

Deleting Files

  • rm: Remove files.

    rm filename.txt

Basic Directory Operations

Creating Directories

  • mkdir: Create a new directory.

    mkdir new_directory

Navigating Directories

  • cd: Change the current directory.

    cd new_directory

  • pwd: Print the current working directory.

    pwd

Listing Directory Contents

  • ls: List files and directories.

    ls

  • ls -l: List files and directories with detailed information.

    ls -l

Deleting Directories

  • rmdir: Remove an empty directory.

    rmdir empty_directory

  • rm -r: Remove a directory and its contents recursively.

    rm -r directory_name

Advanced File Operations

Copying Files

  • cp: Copy files from one location to another.

    cp source_file.txt destination_file.txt

Moving and Renaming Files

  • mv: Move or rename files.

    mv old_name.txt new_name.txt

File Permissions

  • chmod: Change file permissions.

    chmod 755 filename.txt

  • chown: Change file ownership.

    chown user:group filename.txt

Symbolic Links

  • ln -s: Create a symbolic link to a file.

    ln -s target_file.txt link_name.txt

Advanced Directory Operations

Copying Directories

  • cp -r: Copy directories recursively.

    cp -r source_directory destination_directory

Finding Files and Directories

  • find: Search for files and directories.

    find /path/to/search -name "filename.txt"

Archiving and Compressing

  • tar: Archive multiple files into a single file.

    tar -cvf archive.tar file1.txt file2.txt

  • gzip: Compress files.

    gzip filename.txt

  • tar with gzip: Create a compressed archive.

    tar -czvf archive.tar.gz file1.txt file2.txt

Conclusion

Understanding file and directory operations in Bash is crucial for effective scripting and system management. From basic tasks like creating and deleting files to advanced operations like setting permissions and creating symbolic links, mastering these commands will significantly enhance your productivity and capability in a Unix-like environment. Practice these commands regularly to become proficient in managing files and directories through Bash.
