Introduction

In this section, we will cover the fundamental commands for navigating the file system using the Bash command line. Understanding these commands is essential for efficiently managing files and directories in a Unix-like operating system.

Key Concepts

  • Current Directory: The directory you are currently working in.
  • Home Directory: The default directory for your user account.
  • Absolute Path: The full path from the root directory to a specific file or directory.
  • Relative Path: The path relative to the current directory.

Navigating Directories

Viewing the Current Directory

To see the directory you are currently in, use the pwd (print working directory) command.

pwd

Changing Directories

To change your current directory, use the cd (change directory) command.

  • To move to a specific directory:

    cd /path/to/directory

  • To move to your home directory:

    cd ~

  • To move up one directory level:

    cd ..

  • To move to the previous directory:

    cd -

Listing Directory Contents

To list the files and directories in your current directory, use the ls command.

  • Basic listing:

    ls

  • Detailed listing:

    ls -l

  • Including hidden files:

    ls -a

  • Combining options:

    ls -la

Creating and Removing Directories

To create and remove directories, use the mkdir and rmdir commands, respectively.

  • Creating a directory:

    mkdir new_directory

  • Removing an empty directory:

    rmdir new_directory

Examples and Exercises

Example 1: Navigating to a Directory

  1. Open your terminal.
  2. Use pwd to check your current directory.
  3. Change to the /tmp directory: 
    cd /tmp
  4. Verify your new location with pwd.

Example 2: Listing Files

  1. List the contents of your home directory: 
    ls ~
  2. List all files, including hidden ones, in the /etc directory: 
    ls -a /etc

Exercise 1: Directory Navigation Practice

  1. Navigate to your home directory.
  2. Create a new directory called practice.
  3. Change into the practice directory.
  4. Create another directory inside practice called subdir.
  5. List the contents of the practice directory to verify subdir was created.
  6. Remove the subdir directory.
  7. Navigate back to your home directory.

Conclusion

Mastering basic command line navigation is a crucial skill for any programmer working in a Unix-like environment. By understanding and practicing these commands, you will be able to efficiently manage your files and directories, making your workflow more productive. Continue practicing these commands until you feel comfortable navigating the file system without hesitation.
