Introduction

Loops are fundamental constructs in programming that allow you to execute a block of code multiple times. In Bash scripting, there are three primary types of loops: for, while, and until. This section will cover each type of loop, provide examples, and include exercises to reinforce your understanding.

For Loop

The for loop iterates over a list of items and executes a block of code for each item.

Syntax

for item in list
do
    # Commands to execute
done

Example

#!/bin/bash

# List of items
for fruit in apple banana cherry
do
    echo "I like $fruit"
done

Explanation:

  • The loop iterates over the list apple banana cherry.
  • For each item in the list, it prints "I like [fruit]".

Exercise

Write a script that prints numbers from 1 to 5 using a for loop.

While Loop

The while loop executes a block of code as long as a specified condition is true.

Syntax

while [ condition ]
do
    # Commands to execute
done

Example

#!/bin/bash

count=1

while [ $count -le 5 ]
do
    echo "Count is $count"
    ((count++))
done

Explanation:

  • The loop continues as long as count is less than or equal to 5.
  • It prints the current value of count and then increments count by 1.

Exercise

Write a script that prints even numbers from 2 to 10 using a while loop.

Until Loop

The until loop is similar to the while loop but executes a block of code as long as a specified condition is false.

Syntax

until [ condition ]
do
    # Commands to execute
done

Example

#!/bin/bash

count=1

until [ $count -gt 5 ]
do
    echo "Count is $count"
    ((count++))
done

Explanation:

  • The loop continues until count is greater than 5.
  • It prints the current value of count and then increments count by 1.

Exercise

Write a script that prints numbers from 5 to 1 using an until loop.

Comparison of Loops

Loop Type Condition Check Use Case
for Iterates over a list of items When you know the number of iterations
while Executes as long as the condition is true When the number of iterations is not known beforehand
until Executes as long as the condition is false Similar to while, but with the condition inverted

Conclusion

Loops are powerful tools in Bash scripting that allow you to automate repetitive tasks. The for loop is ideal for iterating over a predefined list, while the while and until loops are better suited for situations where the number of iterations depends on dynamic conditions. Practice using these loops with the provided exercises to solidify your understanding.
