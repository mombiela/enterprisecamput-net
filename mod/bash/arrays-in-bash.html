Introduction to Arrays

Arrays in Bash are a powerful feature that allows you to store multiple values in a single variable. This can be particularly useful for managing lists of data, such as filenames, user inputs, or configuration options.

Creating Arrays

  • Indexed Arrays: These arrays use numeric indices to access their elements.
  • Associative Arrays: These arrays use strings as indices.

Indexed Arrays

To create an indexed array, you can use the following syntax:

# Method 1: Using parentheses
fruits=("Apple" "Banana" "Cherry")

# Method 2: Using individual assignments
fruits[0]="Apple"
fruits[1]="Banana"
fruits[2]="Cherry"

Associative Arrays

To create an associative array, you need to declare it first and then assign values:

# Declare an associative array
declare -A colors

# Assign values
colors[apple]="red"
colors[banana]="yellow"
colors[cherry]="red"

Accessing Array Elements

  • Indexed Arrays: Access elements using their numeric index.
  • Associative Arrays: Access elements using their string key.

Indexed Arrays

# Accessing elements
echo ${fruits[0]}  # Output: Apple
echo ${fruits[1]}  # Output: Banana
echo ${fruits[2]}  # Output: Cherry

Associative Arrays

# Accessing elements
echo ${colors[apple]}  # Output: red
echo ${colors[banana]} # Output: yellow
echo ${colors[cherry]} # Output: red

Modifying Arrays

  • Adding Elements: You can add elements by assigning values to new indices or keys.
  • Removing Elements: Use the unset command to remove elements.

Adding Elements

# Indexed Array
fruits[3]="Date"

# Associative Array
colors[date]="brown"

Removing Elements

# Indexed Array
unset fruits[1]  # Removes "Banana"

# Associative Array
unset colors[banana]  # Removes the key "banana"

Looping Through Arrays

You can use loops to iterate through arrays and process their elements.

Indexed Arrays

# Loop through indexed array
for fruit in "${fruits[@]}"; do
  echo $fruit
done

Associative Arrays

# Loop through associative array
for key in "${!colors[@]}"; do
  echo "$key is ${colors[$key]}"
done

Array Length

You can get the length of an array using the ${#array[@]} syntax.

Indexed Arrays

# Get length of indexed array
echo ${#fruits[@]}  # Output: 4

Associative Arrays

# Get length of associative array
echo ${#colors[@]}  # Output: 3

Summary

Arrays in Bash provide a flexible way to handle multiple values within a single variable. They can be indexed or associative, allowing for both numeric and string-based indexing. Key operations include creating, accessing, modifying, and looping through arrays. Understanding these concepts is essential for efficient Bash scripting and data management.

By mastering arrays, you can write more efficient and readable Bash scripts, making your code more powerful and versatile.
