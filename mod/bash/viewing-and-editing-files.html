Introduction

In this section, we will cover how to view and edit files using Bash. These skills are essential for managing and manipulating files directly from the command line. We will start with basic commands and move towards more advanced techniques.

Viewing Files

`cat` Command

The cat command is used to concatenate and display the content of files.

  • Syntax: cat [options] filename
  • Example: 
    cat example.txt
    This command will display the contents of example.txt.

`less` Command

The less command allows you to view the content of a file one screen at a time, which is useful for large files.

  • Syntax: less filename
  • Example: 
    less example.txt
    Use the arrow keys to navigate through the file. Press q to quit.

`more` Command

Similar to less, the more command also lets you view files one screen at a time but with fewer features.

  • Syntax: more filename
  • Example: 
    more example.txt
    Use the spacebar to move to the next page and q to quit.

`head` and `tail` Commands

The head and tail commands are used to display the beginning and end of a file, respectively.

  • Syntax:
    • head -n number_of_lines filename
    • tail -n number_of_lines filename
  • Example: 
    head -n 10 example.txt
tail -n 10 example.txt
    These commands will display the first and last 10 lines of example.txt.

Editing Files

`nano` Editor

nano is a simple, user-friendly text editor.

  • Syntax: nano filename
  • Example: 
    nano example.txt
    Use the keyboard to navigate and edit the file. Press Ctrl + X to exit, Y to save changes, and N to discard changes.

`vim` Editor

vim is a powerful text editor with a steeper learning curve but offers more features.

  • Syntax: vim filename
  • Example: 
    vim example.txt
    • Press i to enter insert mode and start editing.
    • Press Esc to exit insert mode.
    • Type :wq to save and quit, or :q! to quit without saving.

`sed` Command

sed is a stream editor used for parsing and transforming text in files.

  • Syntax: sed 's/old_text/new_text/g' filename
  • Example: 
    sed 's/hello/world/g' example.txt
    This command replaces all instances of "hello" with "world" in example.txt.

`awk` Command

awk is a powerful programming language for pattern scanning and processing.

  • Syntax: awk 'pattern {action}' filename
  • Example: 
    awk '/search_pattern/ {print $0}' example.txt
    This command prints lines in example.txt that match search_pattern.

Conclusion

In this section, we covered various commands and tools for viewing and editing files in Bash. From basic commands like cat and less to more advanced tools like vim, sed, and awk, you now have a comprehensive set of tools to manage files directly from the command line. Practice these commands to become proficient in file manipulation using Bash.
