Introduction

Updating documents in MongoDB is a fundamental skill for managing and manipulating data. This section will guide you through the basics of updating documents, various update operators, and advanced techniques for more complex updates.

Basic Updates

MongoDB provides several methods to update documents in a collection. The most commonly used methods are updateOne(), updateMany(), and replaceOne().

`updateOne()`

The updateOne() method updates a single document that matches a specified filter.

db.collection.updateOne(
  { _id: 1 }, // Filter
  { $set: { name: "John Doe" } } // Update operation
);
  • Filter: Specifies the criteria to match documents.
  • Update operation: Defines the modifications to apply.

`updateMany()`

The updateMany() method updates all documents that match a specified filter.

db.collection.updateMany(
  { status: "active" }, // Filter
  { $set: { status: "inactive" } } // Update operation
);

`replaceOne()`

The replaceOne() method replaces a single document that matches a specified filter with a new document.

db.collection.replaceOne(
  { _id: 1 }, // Filter
  { name: "Jane Doe", age: 30 } // New document
);

Update Operators

MongoDB provides a variety of update operators to modify documents. Here are some of the most commonly used operators:

  • $set: Sets the value of a field.
  • $unset: Removes a field from a document.
  • $inc: Increments the value of a field by a specified amount.
  • $push: Adds an element to an array.
  • $pull: Removes an element from an array.

Examples

$set

db.collection.updateOne(
  { _id: 1 },
  { $set: { age: 25 } }
);

$unset

db.collection.updateOne(
  { _id: 1 },
  { $unset: { age: "" } }
);

$inc

db.collection.updateOne(
  { _id: 1 },
  { $inc: { age: 1 } }
);

$push

db.collection.updateOne(
  { _id: 1 },
  { $push: { hobbies: "reading" } }
);

$pull

db.collection.updateOne(
  { _id: 1 },
  { $pull: { hobbies: "reading" } }
);

Upserts

An upsert operation updates a document if it exists, or inserts a new document if it does not. This is achieved by setting the upsert option to true.

db.collection.updateOne(
  { _id: 2 },
  { $set: { name: "Alice" } },
  { upsert: true }
);

Array Updates

Updating arrays in MongoDB can be more complex. Here are some common operations:

Updating Specific Array Elements

To update a specific element in an array, use the positional $ operator.

db.collection.updateOne(
  { _id: 1, "grades.grade": 85 },
  { $set: { "grades.$.grade": 90 } }
);

Adding Elements to Arrays

Use the $addToSet operator to add an element to an array only if it does not already exist.

db.collection.updateOne(
  { _id: 1 },
  { $addToSet: { hobbies: "swimming" } }
);

Conclusion

Updating documents in MongoDB is a powerful feature that allows you to modify your data efficiently. By understanding the basic update methods, various update operators, and advanced techniques like upserts and array updates, you can manage your MongoDB collections effectively. Practice these concepts with different scenarios to become proficient in updating documents in MongoDB.
