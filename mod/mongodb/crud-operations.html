Introduction

CRUD stands for Create, Read, Update, and Delete. These are the four basic operations that can be performed on a database. In this section, we will explore how to perform CRUD operations using MongoDB, a popular NoSQL database.

Prerequisites

Before diving into CRUD operations, ensure you have:

  • MongoDB installed on your machine.
  • MongoDB Compass or any other MongoDB client.
  • Basic understanding of MongoDB collections and documents.

Create Operations

Creating data in MongoDB involves inserting documents into a collection.

Insert a Single Document

To insert a single document into a collection, use the insertOne method.

db.collectionName.insertOne({
  name: "John Doe",
  age: 29,
  email: "john.doe@example.com"
});

Insert Multiple Documents

To insert multiple documents at once, use the insertMany method.

db.collectionName.insertMany([
  { name: "Jane Doe", age: 25, email: "jane.doe@example.com" },
  { name: "Alice Smith", age: 30, email: "alice.smith@example.com" }
]);

Read Operations

Reading data from MongoDB involves querying the database to retrieve documents.

Find a Single Document

To find a single document that matches a query, use the findOne method.

db.collectionName.findOne({ name: "John Doe" });

Find Multiple Documents

To find multiple documents that match a query, use the find method.

db.collectionName.find({ age: { $gt: 25 } });

Projection

To limit the fields returned by a query, use projection.

db.collectionName.find(
  { age: { $gt: 25 } },
  { name: 1, email: 1, _id: 0 }
);

Update Operations

Updating data in MongoDB involves modifying existing documents.

Update a Single Document

To update a single document, use the updateOne method.

db.collectionName.updateOne(
  { name: "John Doe" },
  { $set: { age: 30 } }
);

Update Multiple Documents

To update multiple documents, use the updateMany method.

db.collectionName.updateMany(
  { age: { $lt: 30 } },
  { $set: { status: "young" } }
);

Replace a Document

To replace an entire document, use the replaceOne method.

db.collectionName.replaceOne(
  { name: "John Doe" },
  { name: "John Doe", age: 30, email: "john.doe@newdomain.com" }
);

Delete Operations

Deleting data in MongoDB involves removing documents from a collection.

Delete a Single Document

To delete a single document, use the deleteOne method.

db.collectionName.deleteOne({ name: "John Doe" });

Delete Multiple Documents

To delete multiple documents, use the deleteMany method.

db.collectionName.deleteMany({ age: { $lt: 25 } });

Conclusion

In this section, we covered the basic CRUD operations in MongoDB:

  • Create: Inserting single or multiple documents.
  • Read: Querying single or multiple documents with optional projections.
  • Update: Modifying single or multiple documents, or replacing entire documents.
  • Delete: Removing single or multiple documents.

Understanding these operations is fundamental for working with MongoDB, as they form the basis of interacting with the database. Practice these operations to become proficient in managing MongoDB collections and documents.
