Introduction

In this section, we will guide you through the process of setting up your first MongoDB database. MongoDB is a popular NoSQL database known for its flexibility and scalability. By the end of this section, you will have a working MongoDB database and understand the basics of interacting with it.

Prerequisites

Before we begin, ensure you have the following:

  • A computer with internet access
  • Basic knowledge of command-line interface (CLI)
  • MongoDB installed on your machine (we will cover installation briefly)

Installing MongoDB

On Windows

  1. Download MongoDB: Visit the MongoDB Download Center and download the installer for Windows.
  2. Run the Installer: Follow the installation prompts. Ensure you select the option to install MongoDB as a service.
  3. Verify Installation: Open Command Prompt and type: 
    mongod --version
    You should see the version information if MongoDB is installed correctly.

On macOS

  1. Install Homebrew: If you don't have Homebrew installed, open Terminal and run: 
    /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
  2. Install MongoDB: Use Homebrew to install MongoDB: 
    brew tap mongodb/brew
brew install mongodb-community
  3. Start MongoDB: Start the MongoDB service: 
    brew services start mongodb/brew/mongodb-community
  4. Verify Installation: Check the installation by running: 
    mongod --version

On Linux

  1. Import the Public Key: Open Terminal and run: 
    wget -qO - https://www.mongodb.org/static/pgp/server-4.4.asc | sudo apt-key add -
  2. Create a List File: Create the /etc/apt/sources.list.d/mongodb-org-4.4.list file for Ubuntu: 
    echo "deb [ arch=amd64,arm64 ] https://repo.mongodb.org/apt/ubuntu focal/mongodb-org/4.4 multiverse" | sudo tee /etc/apt/sources.list.d/mongodb-org-4.4.list
  3. Install MongoDB: Update the package list and install MongoDB: 
    sudo apt-get update
sudo apt-get install -y mongodb-org
  4. Start MongoDB: Start the MongoDB service: 
    sudo systemctl start mongod
  5. Verify Installation: Check the installation by running: 
    mongod --version

Creating Your First Database

Starting the MongoDB Shell

To interact with MongoDB, you will use the MongoDB shell. Open your terminal or command prompt and type:

mongo

This command starts the MongoDB shell, where you can run commands to interact with your database.

Creating a Database

In MongoDB, databases are created implicitly when you first store data in them. To create a database, switch to a new database context:

use myFirstDatabase

This command switches to myFirstDatabase. If it doesn't exist, MongoDB will create it when you first store data.

Inserting Data

To insert data into your new database, use the insertOne method:

db.myCollection.insertOne({ name: "John Doe", age: 30, profession: "Software Engineer" })

This command creates a collection named myCollection and inserts a document into it.

Querying Data

To retrieve data from your collection, use the find method:

db.myCollection.find()

This command returns all documents in the myCollection collection.

Updating Data

To update data in your collection, use the updateOne method:

db.myCollection.updateOne({ name: "John Doe" }, { $set: { age: 31 } })

This command updates the age field of the document where name is "John Doe".

Deleting Data

To delete data from your collection, use the deleteOne method:

db.myCollection.deleteOne({ name: "John Doe" })

This command deletes the document where name is "John Doe".

Summary

In this section, you learned how to set up MongoDB on different operating systems, create your first database, and perform basic CRUD (Create, Read, Update, Delete) operations. These foundational skills are essential for working with MongoDB and will serve as the basis for more advanced topics in this course.
