Introduction

MongoDB is a popular NoSQL database known for its flexibility and scalability. This section will guide you through the process of installing MongoDB on different operating systems. By the end of this section, you will have MongoDB up and running on your machine.

Prerequisites

Before you begin, ensure you have the following:

  • A computer with an internet connection.
  • Basic knowledge of using the command line or terminal.

Installing MongoDB on Windows

Step 1: Download MongoDB

  1. Visit the MongoDB Download Center.
  2. Select the version suitable for your Windows operating system.
  3. Click on the download button to get the .msi installer.

Step 2: Run the Installer

  1. Locate the downloaded .msi file and double-click to run it.
  2. Follow the installation prompts:
    • Choose "Complete" setup type.
    • Ensure "Install MongoDB as a Service" is checked.
    • Optionally, you can choose to install MongoDB Compass, a graphical interface for MongoDB.

Step 3: Verify Installation

  1. Open Command Prompt.
  2. Type mongo --version and press Enter.
  3. You should see the MongoDB version information, confirming the installation.
C:\> mongo --version
MongoDB shell version v4.4.6

Installing MongoDB on macOS

Step 1: Install Homebrew

If you don't have Homebrew installed, open Terminal and run the following command:

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

Step 2: Install MongoDB

  1. Open Terminal.
  2. Run the following commands to tap the MongoDB formula and install it:
brew tap mongodb/brew
brew install mongodb-community@4.4

Step 3: Start MongoDB

  1. To start MongoDB, run:
brew services start mongodb/brew/mongodb-community
  1. Verify the installation by running:
mongo --version

You should see the MongoDB version information.

Installing MongoDB on Linux

Step 1: Import the Public Key

  1. Open Terminal.
  2. Run the following command to import the MongoDB public GPG key:
wget -qO - https://www.mongodb.org/static/pgp/server-4.4.asc | sudo apt-key add -

Step 2: Create a List File for MongoDB

  1. Create the list file using the following command:
echo "deb [ arch=amd64,arm64 ] https://repo.mongodb.org/apt/ubuntu focal/mongodb-org/4.4 multiverse" | sudo tee /etc/apt/sources.list.d/mongodb-org-4.4.list

Step 3: Install MongoDB Packages

  1. Update the package list and install MongoDB:
sudo apt-get update
sudo apt-get install -y mongodb-org

Step 4: Start MongoDB

  1. Start MongoDB using the following command:
sudo systemctl start mongod
  1. Verify the installation by running:
mongo --version

You should see the MongoDB version information.

Conclusion

In this section, you have learned how to install MongoDB on Windows, macOS, and Linux. Ensuring that MongoDB is correctly installed is the first step towards utilizing its powerful features for database management. In the next sections, we will explore how to configure and use MongoDB for various applications.
