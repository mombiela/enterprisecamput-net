Introduction to x86 Assembly

  • Overview of Assembly Language
  • Importance of Learning Assembly
  • Basic Concepts: Registers, Memory, and Instructions

Code Example: Hello World in x86 Assembly

section .data
    hello db 'Hello, World!', 0

section .text
    global _start

_start:
    ; Write the string to stdout
    mov eax, 4          ; syscall number for sys_write
    mov ebx, 1          ; file descriptor 1 is stdout
    mov ecx, hello      ; pointer to the string
    mov edx, 13         ; length of the string
    int 0x80            ; call kernel

    ; Exit the program
    mov eax, 1          ; syscall number for sys_exit
    xor ebx, ebx        ; exit code 0
    int 0x80            ; call kernel
  • Explanation:
    • section .data: Defines a data section where variables are declared.
    • section .text: Defines a code section where the executable code resides.
    • global _start: Entry point for the program.
    • mov instruction: Moves data from one place to another.
    • int 0x80: Interrupt to call the kernel.

Basic x86 Assembly Instructions

  • Data Movement Instructions: mov, push, pop
  • Arithmetic Instructions: add, sub, mul, div
  • Logical Instructions: and, or, xor, not
  • Control Flow Instructions: jmp, je, jne, call, ret

Code Example: Basic Arithmetic Operations

section .text
    global _start

_start:
    mov eax, 5          ; Load 5 into eax
    add eax, 3          ; Add 3 to eax (eax = 8)
    sub eax, 2          ; Subtract 2 from eax (eax = 6)
    ; Exit the program
    mov eax, 1          ; syscall number for sys_exit
    xor ebx, ebx        ; exit code 0
    int 0x80            ; call kernel
  • Explanation:
    • add eax, 3: Adds 3 to the value in the eax register.
    • sub eax, 2: Subtracts 2 from the value in the eax register.

Code Example: Using Pointers

section .data
    num1 dd 10          ; Define a double word with value 10
    num2 dd 20          ; Define a double word with value 20

section .text
    global _start

_start:
    mov eax, [num1]     ; Load the value of num1 into eax
    add eax, [num2]     ; Add the value of num2 to eax (eax = 30)
    ; Exit the program
    mov eax, 1          ; syscall number for sys_exit
    xor ebx, ebx        ; exit code 0
    int 0x80            ; call kernel
  • Explanation:
    • [num1]: Accesses the value stored at the memory location labeled num1.
    • [num2]: Accesses the value stored at the memory location labeled num2.

Control Flow and Procedures

  • Conditional Jumps: je, jne, jg, jl
  • Loop Constructs: loop, loopz, loopnz
  • Defining and Calling Procedures

Code Example: Simple Loop

section .data
    counter dd 5        ; Define a counter with initial value 5

section .text
    global _start

_start:
    mov ecx, [counter]  ; Load counter value into ecx

loop_start:
    dec ecx             ; Decrement ecx by 1
    cmp ecx, 0          ; Compare ecx with 0
    jne loop_start      ; Jump to loop_start if ecx is not zero

    ; Exit the program
    mov eax, 1          ; syscall number for sys_exit
    xor ebx, ebx        ; exit code 0
    int 0x80            ; call kernel
  • Explanation:
    • dec ecx: Decrements the value in the ecx register by 1.
    • cmp ecx, 0: Compares the value in the ecx register with 0.
    • jne loop_start: Jumps to loop_start if the comparison is not equal.

Code Example: Inline Assembly in C

#include <stdio.h>

int main() {
    int a = 5, b = 3, result;

    __asm__ (
        "movl %1, %%eax;"
        "addl %2, %%eax;"
        "movl %%eax, %0;"
        : "=r" (result)
        : "r" (a), "r" (b)
        : "%eax"
    );

    printf("Result: %d\n", result);
    return 0;
}
  • Explanation:
    • __asm__: Inline assembly block in C.
    • "movl %1, %%eax;": Moves the value of a into the eax register.
    • "addl %2, %%eax;": Adds the value of b to the eax register.
    • "movl %%eax, %0;": Moves the result from the eax register to result.
