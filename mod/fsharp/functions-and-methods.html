Introduction

Functions and methods are fundamental building blocks in F#. They allow you to encapsulate logic, promote code reuse, and improve readability. This section will cover the basics of defining and using functions, as well as more advanced topics such as higher-order functions, recursion, and methods in object-oriented programming.

Basic Functions

Functions in F# are first-class citizens, meaning they can be passed as arguments, returned from other functions, and assigned to variables.

Defining a Function

A simple function can be defined using the let keyword.

let add x y = x + y
  • let: Keyword used to define a function.
  • add: Name of the function.
  • x and y: Parameters of the function.
  • x + y: Body of the function.

Calling a Function

You can call a function by using its name followed by arguments.

let result = add 3 5
printfn "The result is %d" result

Function with Multiple Lines

If a function has multiple lines, use indentation to define the scope.

let greet name =
    let message = "Hello, " + name
    printfn "%s" message

greet "Alice"

Higher-Order Functions

Higher-order functions are functions that take other functions as parameters or return functions as results.

Example: `map` Function

The List.map function applies a given function to each element of a list.

let square x = x * x
let numbers = [1; 2; 3; 4]
let squaredNumbers = List.map square numbers
printfn "%A" squaredNumbers

Recursion

Recursion is a technique where a function calls itself to solve a problem.

Example: Factorial Function

A classic example of recursion is the factorial function.

let rec factorial n =
    if n <= 1 then 1
    else n * factorial (n - 1)

printfn "Factorial of 5 is %d" (factorial 5)
  • rec: Keyword used to define a recursive function.
  • factorial: Name of the function.
  • n: Parameter of the function.
  • if n <= 1 then 1: Base case.
  • else n * factorial (n - 1): Recursive case.

Methods in Object-Oriented Programming

F# supports object-oriented programming, and methods are functions defined within a class or object.

Defining a Class with Methods

Here is an example of a simple class with methods.

type Person(name: string, age: int) =
    member this.Name = name
    member this.Age = age
    member this.Greet() = printfn "Hello, my name is %s and I am %d years old." this.Name this.Age

let alice = Person("Alice", 30)
alice.Greet()
  • type: Keyword used to define a class.
  • Person: Name of the class.
  • name and age: Constructor parameters.
  • member: Keyword used to define a method or property.
  • this: Reference to the current instance of the class.

Summary

In this section, we covered the basics of functions and methods in F#. We started with simple function definitions and calls, moved on to higher-order functions and recursion, and concluded with methods in object-oriented programming. Understanding these concepts is crucial for writing efficient and reusable code in F#. Practice these concepts with various examples to solidify your understanding.
