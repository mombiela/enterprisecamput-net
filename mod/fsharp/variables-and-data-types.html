Introduction

In this section, we will explore the fundamental concepts of variables and data types in F#. Understanding these basics is crucial for writing effective and efficient F# programs. We will cover:

  • Declaring variables
  • Immutable and mutable variables
  • Basic data types
  • Type inference
  • Custom data types

Declaring Variables

In F#, variables are declared using the let keyword. By default, variables in F# are immutable, meaning their values cannot be changed once assigned.

Example:

let x = 10
let y = "Hello, F#"

In the above example:

  • x is an integer variable with a value of 10.
  • y is a string variable with a value of "Hello, F#".

Immutable and Mutable Variables

Immutable Variables

As mentioned, variables declared with let are immutable.

Example:

let a = 5
// a <- 10  // This will cause an error because 'a' is immutable

Mutable Variables

If you need a variable whose value can change, you can use the mutable keyword.

Example:

let mutable b = 5
b <- 10  // This is allowed because 'b' is mutable

Basic Data Types

F# supports several basic data types. Here are some of the most commonly used ones:

| Data Type | Description | Example | |-----------|-----------------------|------------------------| | int | Integer | let num = 42 | | float | Floating-point number | let pi = 3.14 | | bool | Boolean | let isTrue = true | | string | String | let greeting = "Hi" | | char | Character | let letter = 'A' |

Example:

let age: int = 25
let price: float = 19.99
let isActive: bool = true
let name: string = "Alice"
let initial: char = 'A'

Type Inference

F# has a powerful type inference system that can often deduce the type of a variable from its value, so you don't always need to explicitly specify the type.

Example:

let inferredInt = 100  // F# infers this as int
let inferredString = "Type Inference"  // F# infers this as string

Custom Data Types

F# allows you to define your own data types using type keyword. You can create record types, union types, and more.

Record Types

Record types are used to group related values.

Example:

type Person = {
    Name: string
    Age: int
}

let person1 = { Name = "John"; Age = 30 }

Union Types

Union types are used to represent a value that can be one of several different types.

Example:

type Shape =
    | Circle of float
    | Rectangle of float * float

let circle = Circle 5.0
let rectangle = Rectangle (4.0, 6.0)

Conclusion

In this section, we covered the basics of variables and data types in F#. We learned how to declare variables, the difference between immutable and mutable variables, and explored basic and custom data types. Understanding these concepts is essential for writing robust F# programs. In the next section, we will delve deeper into functions and how to use them effectively in F#.
