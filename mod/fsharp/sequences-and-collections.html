Introduction

In F#, sequences and collections are fundamental data structures that allow you to store and manipulate groups of data. This section will cover the basics of sequences, lists, arrays, and other collection types, providing you with the knowledge to effectively use them in your programs.

Lists

Lists are immutable collections of elements of the same type. They are one of the most commonly used collection types in F#.

Creating Lists

You can create lists using square brackets and separating elements with semicolons.

let numbers = [1; 2; 3; 4; 5]
let fruits = ["apple"; "banana"; "cherry"]

Basic Operations

  • Accessing Elements: Use the List.item function or the . operator.
  • Adding Elements: Use the :: operator to prepend elements.
  • Concatenating Lists: Use the @ operator.
let firstElement = List.item 0 numbers
let newList = 0 :: numbers
let combinedList = numbers @ [6; 7; 8]

Common Functions

  • List.map: Applies a function to each element.
  • List.filter: Filters elements based on a predicate.
  • List.fold: Aggregates elements using a function.
let squaredNumbers = List.map (fun x -> x * x) numbers
let evenNumbers = List.filter (fun x -> x % 2 = 0) numbers
let sum = List.fold (+) 0 numbers

Arrays

Arrays are mutable collections of elements of the same type, allowing for efficient random access and updates.

Creating Arrays

You can create arrays using square brackets and separating elements with semicolons, or using the Array.init function.

let numbersArray = [|1; 2; 3; 4; 5|]
let initializedArray = Array.init 5 (fun i -> i * 2)

Basic Operations

  • Accessing Elements: Use the . operator with an index.
  • Updating Elements: Use the <- operator with an index.
let firstElementArray = numbersArray.[0]
numbersArray.[0] <- 10

Common Functions

  • Array.map: Applies a function to each element.
  • Array.filter: Filters elements based on a predicate.
  • Array.fold: Aggregates elements using a function.
let squaredNumbersArray = Array.map (fun x -> x * x) numbersArray
let evenNumbersArray = Array.filter (fun x -> x % 2 = 0) numbersArray
let sumArray = Array.fold (+) 0 numbersArray

Sequences

Sequences are lazy collections that are evaluated on demand, making them suitable for working with large or infinite data sets.

Creating Sequences

You can create sequences using the seq keyword or using sequence expressions.

let numbersSeq = seq {1 .. 5}
let infiniteSeq = Seq.initInfinite (fun i -> i * 2)

Basic Operations

  • Accessing Elements: Use the Seq.item function.
  • Converting to Other Collections: Use functions like Seq.toList or Seq.toArray.
let firstElementSeq = Seq.item 0 numbersSeq
let numbersList = Seq.toList numbersSeq
let numbersArrayFromSeq = Seq.toArray numbersSeq

Common Functions

  • Seq.map: Applies a function to each element.
  • Seq.filter: Filters elements based on a predicate.
  • Seq.fold: Aggregates elements using a function.
let squaredNumbersSeq = Seq.map (fun x -> x * x) numbersSeq
let evenNumbersSeq = Seq.filter (fun x -> x % 2 = 0) numbersSeq
let sumSeq = Seq.fold (+) 0 numbersSeq

Other Collections

F# also provides other collection types such as sets and maps, which offer unique properties and use cases.

Sets

Sets are collections of unique elements.

let numbersSet = Set.ofList [1; 2; 3; 4; 5; 5]
let isMember = Set.contains 3 numbersSet

Maps

Maps are collections of key-value pairs.

let numbersMap = Map.ofList [(1, "one"); (2, "two"); (3, "three")]
let value = Map.find 2 numbersMap

Conclusion

In this section, we explored various sequences and collections in F#, including lists, arrays, sequences, sets, and maps. Understanding these data structures and their operations is crucial for effective programming in F#. Practice using these collections in different scenarios to become proficient in managing and manipulating data in your F# applications.
