CRUD Operations in C#
Entity Framework ►

CRUD stands for Create, Read, Update, and Delete. These are the four basic operations for managing data in a database. In this section, we will explore how to perform CRUD operations in C# using various approaches, including ADO.NET and Entity Framework.

Introduction to CRUD Operations

  • Create: Adding new records to the database.
  • Read: Retrieving existing records from the database.
  • Update: Modifying existing records in the database.
  • Delete: Removing records from the database.

Setting Up the Environment

Before diving into CRUD operations, ensure you have the following setup:

  • Visual Studio installed.
  • SQL Server or any other database management system.
  • A basic understanding of C# syntax.

Using ADO.NET for CRUD Operations

Create Operation

To insert a new record into the database:

using System;
using System.Data.SqlClient;

class Program
{
    static void Main()
    {
        string connectionString = "your_connection_string_here";
        using (SqlConnection connection = new SqlConnection(connectionString))
        {
            string query = "INSERT INTO Students (Name, Age) VALUES (@Name, @Age)";
            SqlCommand command = new SqlCommand(query, connection);
            command.Parameters.AddWithValue("@Name", "John Doe");
            command.Parameters.AddWithValue("@Age", 21);

            connection.Open();
            int result = command.ExecuteNonQuery();

            // Check if the insert was successful
            if (result > 0)
                Console.WriteLine("Record inserted successfully.");
            else
                Console.WriteLine("Error inserting record.");
        }
    }
}

Read Operation

To retrieve records from the database:

using System;
using System.Data.SqlClient;

class Program
{
    static void Main()
    {
        string connectionString = "your_connection_string_here";
        using (SqlConnection connection = new SqlConnection(connectionString))
        {
            string query = "SELECT * FROM Students";
            SqlCommand command = new SqlCommand(query, connection);

            connection.Open();
            SqlDataReader reader = command.ExecuteReader();

            while (reader.Read())
            {
                Console.WriteLine($"ID: {reader["ID"]}, Name: {reader["Name"]}, Age: {reader["Age"]}");
            }
        }
    }
}

Update Operation

To update an existing record in the database:

using System;
using System.Data.SqlClient;

class Program
{
    static void Main()
    {
        string connectionString = "your_connection_string_here";
        using (SqlConnection connection = new SqlConnection(connectionString))
        {
            string query = "UPDATE Students SET Age = @Age WHERE Name = @Name";
            SqlCommand command = new SqlCommand(query, connection);
            command.Parameters.AddWithValue("@Name", "John Doe");
            command.Parameters.AddWithValue("@Age", 22);

            connection.Open();
            int result = command.ExecuteNonQuery();

            // Check if the update was successful
            if (result > 0)
                Console.WriteLine("Record updated successfully.");
            else
                Console.WriteLine("Error updating record.");
        }
    }
}

Delete Operation

To delete a record from the database:

using System;
using System.Data.SqlClient;

class Program
{
    static void Main()
    {
        string connectionString = "your_connection_string_here";
        using (SqlConnection connection = new SqlConnection(connectionString))
        {
            string query = "DELETE FROM Students WHERE Name = @Name";
            SqlCommand command = new SqlCommand(query, connection);
            command.Parameters.AddWithValue("@Name", "John Doe");

            connection.Open();
            int result = command.ExecuteNonQuery();

            // Check if the delete was successful
            if (result > 0)
                Console.WriteLine("Record deleted successfully.");
            else
                Console.WriteLine("Error deleting record.");
        }
    }
}

Using Entity Framework for CRUD Operations

Entity Framework (EF) is an ORM (Object-Relational Mapper) that simplifies data access in .NET applications.

Setting Up Entity Framework

  1. Install Entity Framework via NuGet Package Manager:

    Install-Package EntityFramework

  2. Create a model class and a context class:

using System.Data.Entity;

public class Student
{
    public int ID { get; set; }
    public string Name { get; set; }
    public int Age { get; set; }
}

public class SchoolContext : DbContext
{
    public DbSet<Student> Students { get; set; }
}

Create Operation

To insert a new record using Entity Framework:

using System;

class Program
{
    static void Main()
    {
        using (var context = new SchoolContext())
        {
            var student = new Student { Name = "Jane Doe", Age = 20 };
            context.Students.Add(student);
            context.SaveChanges();
            Console.WriteLine("Record inserted successfully.");
        }
    }
}

Read Operation

To retrieve records using Entity Framework:

using System;
using System.Linq;

class Program
{
    static void Main()
    {
        using (var context = new SchoolContext())
        {
            var students = context.Students.ToList();
            foreach (var student in students)
            {
                Console.WriteLine($"ID: {student.ID}, Name: {student.Name}, Age: {student.Age}");
            }
        }
    }
}

Update Operation

To update an existing record using Entity Framework:

using System;
using System.Linq;

class Program
{
    static void Main()
    {
        using (var context = new SchoolContext())
        {
            var student = context.Students.FirstOrDefault(s => s.Name == "Jane Doe");
            if (student != null)
            {
                student.Age = 21;
                context.SaveChanges();
                Console.WriteLine("Record updated successfully.");
            }
        }
    }
}

Delete Operation

To delete a record using Entity Framework:

using System;
using System.Linq;

class Program
{
    static void Main()
    {
        using (var context = new SchoolContext())
        {
            var student = context.Students.FirstOrDefault(s => s.Name == "Jane Doe");
            if (student != null)
            {
                context.Students.Remove(student);
                context.SaveChanges();
                Console.WriteLine("Record deleted successfully.");
            }
        }
    }
}

Conclusion

In this section, we covered the basics of CRUD operations in C# using both ADO.NET and Entity Framework. We learned how to create, read, update, and delete records in a database. Understanding these operations is fundamental for any application that interacts with a database. As you progress, you can explore more advanced topics such as transaction management, stored procedures, and performance optimization.
