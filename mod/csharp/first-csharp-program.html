Introduction

Welcome to your journey into C# programming! In this section, we will guide you through writing your first C# program. By the end of this lesson, you will understand the basic structure of a C# program and how to run it.

Setting Up Your Environment

Before we start coding, you need to set up your development environment. Follow these steps:

  1. Install Visual Studio: Download and install Visual Studio Community Edition from the official website.
  2. Create a New Project:
    • Open Visual Studio.
    • Select "Create a new project".
    • Choose "Console App (.NET Core)".
    • Click "Next", name your project, and click "Create".

Understanding the Basic Structure

A C# program consists of several key components. Here is a simple example of a C# program:

using System;

namespace HelloWorld
{
    class Program
    {
        static void Main(string[] args)
        {
            Console.WriteLine("Hello, World!");
        }
    }
}

Key Components Explained

  • using System;: This line imports the System namespace, which contains fundamental classes for basic operations.
  • namespace HelloWorld: Namespaces are used to organize code and prevent naming conflicts.
  • class Program: A class is a blueprint for creating objects. The Program class contains the code that will be executed.
  • static void Main(string[] args): The Main method is the entry point of a C# program. It is where the program starts execution.
  • Console.WriteLine("Hello, World!");: This line prints "Hello, World!" to the console.

Writing and Running Your First Program

Let's write and run your first C# program step-by-step:

  1. Open Visual Studio and create a new Console App project as described earlier.
  2. Replace the default code in Program.cs with the following:
using System;

namespace HelloWorld
{
    class Program
    {
        static void Main(string[] args)
        {
            Console.WriteLine("Hello, World!");
        }
    }
}
  1. Run the Program:
    • Click the "Start" button (or press F5).
    • You should see a console window displaying "Hello, World!".

Exercises

To reinforce your understanding, try the following exercises:

  1. Modify the Message: Change the message in Console.WriteLine to print your name instead of "Hello, World!".
  2. Add Another Line: Add another Console.WriteLine statement to print a second message.
  3. Experiment with Comments: Add comments to your code to describe what each part does. Use // for single-line comments and /* */ for multi-line comments.

Example:

using System;

namespace HelloWorld
{
    class Program
    {
        static void Main(string[] args)
        {
            // This prints a greeting message to the console
            Console.WriteLine("Hello, [Your Name]!");
            
            // This prints another message
            Console.WriteLine("Welcome to C# programming!");
        }
    }
}

Conclusion

Congratulations! You've written and run your first C# program. You now understand the basic structure of a C# application and how to execute it. This foundational knowledge will be crucial as we dive deeper into more advanced topics. Keep experimenting with the code and try out the exercises to solidify your understanding. Happy coding!
