Introduction

Methods are fundamental building blocks in C#. They allow you to encapsulate code into reusable blocks, making your programs more modular and easier to maintain. In this section, we will cover how to define and call methods, from basic to advanced concepts.

Defining Methods

Basic Method Definition

A method in C# is defined within a class and consists of a signature and a body. The signature includes the access modifier, return type, method name, and parameters (if any).

public class Program
{
    // Method definition
    public void SayHello()
    {
        Console.WriteLine("Hello, World!");
    }
}
  • Access Modifier: Determines the visibility of the method (e.g., public, private).
  • Return Type: Specifies the type of value the method returns (e.g., void if no value is returned).
  • Method Name: A unique identifier for the method.
  • Parameters: Optional; used to pass data into the method.

Method with Parameters

Methods can take parameters to perform operations on input data.

public class Program
{
    public void GreetUser(string name)
    {
        Console.WriteLine($"Hello, {name}!");
    }
}
  • Parameters: Defined within the parentheses and separated by commas if there are multiple.

Method with Return Value

Methods can return values to the caller.

public class Program
{
    public int Add(int a, int b)
    {
        return a + b;
    }
}
  • Return Statement: Used to return a value from the method.

Calling Methods

Calling a Method without Parameters

To call a method, you use the method name followed by parentheses.

public class Program
{
    public void SayHello()
    {
        Console.WriteLine("Hello, World!");
    }

    public static void Main()
    {
        Program program = new Program();
        program.SayHello(); // Method call
    }
}

Calling a Method with Parameters

When calling a method with parameters, you pass the arguments within the parentheses.

public class Program
{
    public void GreetUser(string name)
    {
        Console.WriteLine($"Hello, {name}!");
    }

    public static void Main()
    {
        Program program = new Program();
        program.GreetUser("Alice"); // Method call with argument
    }
}

Calling a Method with Return Value

When calling a method that returns a value, you can store the result in a variable.

public class Program
{
    public int Add(int a, int b)
    {
        return a + b;
    }

    public static void Main()
    {
        Program program = new Program();
        int result = program.Add(5, 3); // Method call with return value
        Console.WriteLine(result); // Output: 8
    }
}

Advanced Concepts

Method Overloading

Method overloading allows you to define multiple methods with the same name but different parameters.

public class Program
{
    public void Display(int number)
    {
        Console.WriteLine($"Number: {number}");
    }

    public void Display(string text)
    {
        Console.WriteLine($"Text: {text}");
    }

    public static void Main()
    {
        Program program = new Program();
        program.Display(10); // Calls the first Display method
        program.Display("Hello"); // Calls the second Display method
    }
}

Optional Parameters

C# allows you to define methods with optional parameters, providing default values if arguments are not supplied.

public class Program
{
    public void DisplayMessage(string message = "Default Message")
    {
        Console.WriteLine(message);
    }

    public static void Main()
    {
        Program program = new Program();
        program.DisplayMessage(); // Output: Default Message
        program.DisplayMessage("Custom Message"); // Output: Custom Message
    }
}

Named Arguments

Named arguments allow you to specify the parameter name when calling a method, improving readability.

public class Program
{
    public void DisplayInfo(string name, int age)
    {
        Console.WriteLine($"Name: {name}, Age: {age}");
    }

    public static void Main()
    {
        Program program = new Program();
        program.DisplayInfo(name: "Alice", age: 30); // Named arguments
    }
}

Summary

In this section, we covered the basics of defining and calling methods in C#. We explored:

  • Basic method definitions and calls.
  • Methods with parameters and return values.
  • Advanced concepts like method overloading, optional parameters, and named arguments.

Understanding how to define and call methods is crucial for writing clean, efficient, and maintainable code. Practice these concepts to become proficient in method usage in C#.
