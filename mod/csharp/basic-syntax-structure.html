Introduction

C# is a modern, object-oriented programming language developed by Microsoft. It is widely used for developing desktop applications, web applications, and games. Understanding the basic syntax and structure of C# is essential for writing efficient and error-free code.

Key Concepts

1. Hello World Program

The best way to start learning any programming language is by writing a simple "Hello, World!" program. This program will introduce you to the basic structure of a C# program.

using System;

namespace HelloWorld
{
    class Program
    {
        static void Main(string[] args)
        {
            Console.WriteLine("Hello, World!");
        }
    }
}
  • using System;: This line imports the System namespace, which contains fundamental classes for basic input and output operations.
  • namespace HelloWorld: Namespaces are used to organize code and prevent naming conflicts.
  • class Program: Classes are the building blocks of C# programs.
  • static void Main(string[] args): The Main method is the entry point of a C# application. It is where the program starts execution.
  • Console.WriteLine("Hello, World!");: This line prints "Hello, World!" to the console.

2. Variables and Data Types

Variables are used to store data, and data types specify the type of data that can be stored in a variable.

Common Data Types

| Data Type | Description | Example | |-----------|-----------------------------|---------------| | int | Integer numbers | 1, 2, 3 | | double | Floating-point numbers | 1.23, 4.56 | | char | Single character | 'a', 'b' | | string | Sequence of characters | "Hello" | | bool | Boolean values (true/false) | true, false |

Example

int age = 25;
double salary = 50000.50;
char grade = 'A';
string name = "John";
bool isEmployed = true;

3. Operators

Operators are used to perform operations on variables and values.

Common Operators

  • Arithmetic Operators: +, -, *, /, %
  • Comparison Operators: ==, !=, >, <, >=, <=
  • Logical Operators: &&, ||, !

Example

int a = 10;
int b = 20;
int sum = a + b; // 30
bool isEqual = (a == b); // false
bool isGreater = (a > b); // false
bool logicalAnd = (a < b && b > 15); // true

4. Control Structures

Control structures are used to control the flow of execution in a program.

If-Else Statement

int number = 10;
if (number > 0)
{
    Console.WriteLine("Positive number");
}
else
{
    Console.WriteLine("Non-positive number");
}

Switch Statement

int day = 3;
switch (day)
{
    case 1:
        Console.WriteLine("Monday");
        break;
    case 2:
        Console.WriteLine("Tuesday");
        break;
    case 3:
        Console.WriteLine("Wednesday");
        break;
    default:
        Console.WriteLine("Other day");
        break;
}

Loops

  • For Loop
for (int i = 0; i < 5; i++)
{
    Console.WriteLine(i);
}
  • While Loop
int i = 0;
while (i < 5)
{
    Console.WriteLine(i);
    i++;
}
  • Do-While Loop
int i = 0;
do
{
    Console.WriteLine(i);
    i++;
} while (i < 5);

5. Methods

Methods are blocks of code that perform a specific task and can be called when needed.

Example

class Program
{
    static void Main(string[] args)
    {
        Greet("John");
    }

    static void Greet(string name)
    {
        Console.WriteLine("Hello, " + name);
    }
}

Conclusion

Understanding the basic syntax and structure of C# is crucial for any beginner. This includes knowing how to write a simple program, declare variables, use operators, control the flow of the program with control structures, and define methods. Mastery of these basics will provide a strong foundation for more advanced topics in C#.
