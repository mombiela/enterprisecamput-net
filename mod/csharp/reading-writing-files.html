Reading and Writing Files in C#
File Streams ►

Introduction

Reading and writing files is a fundamental skill in programming. In C#, the System.IO namespace provides classes and methods to handle file operations. This topic will cover the basics of reading from and writing to files, as well as more advanced techniques.

Basics of File Handling

Key Concepts

  • File: Represents a file in the file system.
  • Stream: A sequence of bytes that can be read from or written to.
  • FileStream: Provides a stream for file operations.
  • StreamReader: Reads characters from a byte stream.
  • StreamWriter: Writes characters to a byte stream.

Reading Files

Using File.ReadAllText

The simplest way to read a file is by using the File.ReadAllText method, which reads all the text from a file into a single string.

using System;
using System.IO;

class Program
{
    static void Main()
    {
        string path = "example.txt";
        string content = File.ReadAllText(path);
        Console.WriteLine(content);
    }
}

Using StreamReader

For more control, you can use the StreamReader class.

using System;
using System.IO;

class Program
{
    static void Main()
    {
        string path = "example.txt";
        using (StreamReader sr = new StreamReader(path))
        {
            string line;
            while ((line = sr.ReadLine()) != null)
            {
                Console.WriteLine(line);
            }
        }
    }
}

Writing Files

Using File.WriteAllText

The simplest way to write to a file is by using the File.WriteAllText method, which writes a string to a file, creating the file if it doesn't exist.

using System;
using System.IO;

class Program
{
    static void Main()
    {
        string path = "example.txt";
        string content = "Hello, World!";
        File.WriteAllText(path, content);
    }
}

Using StreamWriter

For more control, you can use the StreamWriter class.

using System;
using System.IO;

class Program
{
    static void Main()
    {
        string path = "example.txt";
        using (StreamWriter sw = new StreamWriter(path))
        {
            sw.WriteLine("Hello, World!");
            sw.WriteLine("This is a new line.");
        }
    }
}

Advanced File Handling

Key Concepts

  • FileMode: Specifies how the operating system should open a file.
  • FileAccess: Defines the level of access permitted to a file.
  • FileShare: Determines how the file will be shared by processes.

Using `FileStream`

For advanced scenarios, you can use the FileStream class, which provides more control over file operations.

using System;
using System.IO;

class Program
{
    static void Main()
    {
        string path = "example.txt";
        using (FileStream fs = new FileStream(path, FileMode.OpenOrCreate, FileAccess.ReadWrite))
        {
            byte[] info = new System.Text.UTF8Encoding(true).GetBytes("Hello, World!");
            fs.Write(info, 0, info.Length);
        }
    }
}

Asynchronous File Operations

C# also supports asynchronous file operations, which can improve performance in I/O-bound applications.

Asynchronous Reading

using System;
using System.IO;
using System.Threading.Tasks;

class Program
{
    static async Task Main()
    {
        string path = "example.txt";
        string content = await File.ReadAllTextAsync(path);
        Console.WriteLine(content);
    }
}

Asynchronous Writing

using System;
using System.IO;
using System.Threading.Tasks;

class Program
{
    static async Task Main()
    {
        string path = "example.txt";
        string content = "Hello, World!";
        await File.WriteAllTextAsync(path, content);
    }
}

Summary

In this topic, we covered the basics and advanced techniques for reading and writing files in C#. We explored simple methods like File.ReadAllText and File.WriteAllText, as well as more advanced classes like StreamReader, StreamWriter, and FileStream. Additionally, we touched on asynchronous file operations to handle I/O-bound tasks efficiently. Mastering these techniques will enable you to handle file operations effectively in your C# applications.
