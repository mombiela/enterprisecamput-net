Overview

Databases are essential for storing, retrieving, and managing data in applications. This section will introduce you to databases, how to interact with them using C#, and the basics of SQL (Structured Query Language).

Key Concepts

  • Database: An organized collection of data, generally stored and accessed electronically from a computer system.
  • SQL: A standard language for accessing and manipulating databases.
  • RDBMS: Relational Database Management System, a type of database management system that stores data in tables and allows for relationships between tables.
  • CRUD Operations: The four basic operations for managing data - Create, Read, Update, Delete.

Setting Up

Before diving into code, ensure you have the following:

  • SQL Server: Install SQL Server or use a cloud-based service like Azure SQL Database.
  • SQL Server Management Studio (SSMS): A tool to manage SQL Server databases.
  • Visual Studio: IDE for writing C# code.

Connecting to a Database

To interact with a database in C#, you need to establish a connection using the SqlConnection class.

Example

using System;
using System.Data.SqlClient;

class Program
{
    static void Main()
    {
        string connectionString = "Server=your_server;Database=your_database;User Id=your_username;Password=your_password;";
        
        using (SqlConnection connection = new SqlConnection(connectionString))
        {
            try
            {
                connection.Open();
                Console.WriteLine("Connection successful!");
            }
            catch (Exception ex)
            {
                Console.WriteLine("Connection failed: " + ex.Message);
            }
        }
    }
}

Explanation:

  • connectionString: Contains the details required to connect to the database.
  • SqlConnection: Represents a connection to a SQL Server database.
  • Open(): Opens the connection to the database.
  • try-catch: Handles any exceptions that occur during the connection attempt.

CRUD Operations

Create

To insert data into a database, use the INSERT INTO SQL statement.

Example

using System;
using System.Data.SqlClient;

class Program
{
    static void Main()
    {
        string connectionString = "Server=your_server;Database=your_database;User Id=your_username;Password=your_password;";
        string query = "INSERT INTO Users (Name, Age) VALUES (@Name, @Age)";

        using (SqlConnection connection = new SqlConnection(connectionString))
        {
            SqlCommand command = new SqlCommand(query, connection);
            command.Parameters.AddWithValue("@Name", "John Doe");
            command.Parameters.AddWithValue("@Age", 30);

            try
            {
                connection.Open();
                int result = command.ExecuteNonQuery();
                Console.WriteLine("Rows affected: " + result);
            }
            catch (Exception ex)
            {
                Console.WriteLine("Error: " + ex.Message);
            }
        }
    }
}

Explanation:

  • SqlCommand: Represents a SQL statement to execute against a SQL Server database.
  • Parameters: Used to prevent SQL injection by parameterizing the query.

Read

To retrieve data, use the SELECT SQL statement.

Example

using System;
using System.Data.SqlClient;

class Program
{
    static void Main()
    {
        string connectionString = "Server=your_server;Database=your_database;User Id=your_username;Password=your_password;";
        string query = "SELECT Name, Age FROM Users";

        using (SqlConnection connection = new SqlConnection(connectionString))
        {
            SqlCommand command = new SqlCommand(query, connection);

            try
            {
                connection.Open();
                SqlDataReader reader = command.ExecuteReader();
                while (reader.Read())
                {
                    Console.WriteLine($"Name: {reader["Name"]}, Age: {reader["Age"]}");
                }
            }
            catch (Exception ex)
            {
                Console.WriteLine("Error: " + ex.Message);
            }
        }
    }
}

Explanation:

  • SqlDataReader: Provides a way of reading a forward-only stream of rows from a SQL Server database.

Update

To modify existing data, use the UPDATE SQL statement.

Example

using System;
using System.Data.SqlClient;

class Program
{
    static void Main()
    {
        string connectionString = "Server=your_server;Database=your_database;User Id=your_username;Password=your_password;";
        string query = "UPDATE Users SET Age = @Age WHERE Name = @Name";

        using (SqlConnection connection = new SqlConnection(connectionString))
        {
            SqlCommand command = new SqlCommand(query, connection);
            command.Parameters.AddWithValue("@Name", "John Doe");
            command.Parameters.AddWithValue("@Age", 31);

            try
            {
                connection.Open();
                int result = command.ExecuteNonQuery();
                Console.WriteLine("Rows affected: " + result);
            }
            catch (Exception ex)
            {
                Console.WriteLine("Error: " + ex.Message);
            }
        }
    }
}

Explanation:

  • UPDATE: Modifies existing records in a table.

Delete

To remove data, use the DELETE SQL statement.

Example

using System;
using System.Data.SqlClient;

class Program
{
    static void Main()
    {
        string connectionString = "Server=your_server;Database=your_database;User Id=your_username;Password=your_password;";
        string query = "DELETE FROM Users WHERE Name = @Name";

        using (SqlConnection connection = new SqlConnection(connectionString))
        {
            SqlCommand command = new SqlCommand(query, connection);
            command.Parameters.AddWithValue("@Name", "John Doe");

            try
            {
                connection.Open();
                int result = command.ExecuteNonQuery();
                Console.WriteLine("Rows affected: " + result);
            }
            catch (Exception ex)
            {
                Console.WriteLine("Error: " + ex.Message);
            }
        }
    }
}

Explanation:

  • DELETE: Removes records from a table.

Conclusion

In this section, you learned the basics of databases, how to connect to a database using C#, and how to perform CRUD operations. These are fundamental skills for any developer working with data-driven applications. Practice these examples and try to create your own database interactions to solidify your understanding.
