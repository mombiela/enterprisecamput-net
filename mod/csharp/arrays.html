Introduction to Arrays

Arrays are a fundamental data structure in C# that allow you to store multiple values of the same type in a single variable. They are useful for managing collections of data efficiently.

Key Concepts

  • Definition: An array is a collection of variables that are accessed with an index number.
  • Fixed Size: Arrays have a fixed size that is defined when they are created.
  • Zero-based Indexing: The first element of an array is at index 0.

Creating Arrays

To create an array in C#, you need to specify the type of elements it will hold and its size.

// Declaring and initializing an array of integers with 5 elements
int[] numbers = new int[5];

// Declaring and initializing an array with specific values
int[] predefinedNumbers = { 1, 2, 3, 4, 5 };

Accessing Array Elements

You can access elements in an array using their index.

int firstNumber = numbers[0]; // Accessing the first element
numbers[1] = 10; // Setting the second element to 10

Iterating Over Arrays

You can use loops to iterate over arrays.

// Using a for loop
for (int i = 0; i < numbers.Length; i++)
{
    Console.WriteLine(numbers[i]);
}

// Using a foreach loop
foreach (int number in predefinedNumbers)
{
    Console.WriteLine(number);
}

Advanced Array Concepts

Multi-dimensional Arrays

C# supports multi-dimensional arrays, such as 2D arrays.

// Declaring a 2D array
int[,] matrix = new int[3, 3];

// Initializing a 2D array
int[,] predefinedMatrix = {
    { 1, 2, 3 },
    { 4, 5, 6 },
    { 7, 8, 9 }
};

// Accessing elements in a 2D array
int element = matrix[1, 2]; // Accessing the element at row 1, column 2

Jagged Arrays

Jagged arrays are arrays of arrays, where each "inner" array can have a different length.

// Declaring a jagged array
int[][] jaggedArray = new int[3][];

// Initializing the jagged array
jaggedArray[0] = new int[] { 1, 2 };
jaggedArray[1] = new int[] { 3, 4, 5 };
jaggedArray[2] = new int[] { 6, 7, 8, 9 };

// Accessing elements in a jagged array
int jaggedElement = jaggedArray[1][2]; // Accessing the third element of the second array

Array Methods

C# provides several built-in methods to work with arrays, such as Array.Sort, Array.Reverse, and Array.Copy.

int[] numbers = { 5, 3, 8, 1, 2 };

// Sorting the array
Array.Sort(numbers);

// Reversing the array
Array.Reverse(numbers);

// Copying the array
int[] copiedNumbers = new int[numbers.Length];
Array.Copy(numbers, copiedNumbers, numbers.Length);

Conclusion

Arrays are a powerful and essential part of C# programming. They allow you to store and manage collections of data efficiently. Understanding how to create, access, and manipulate arrays is crucial for any C# programmer. As you advance, you will encounter more complex array structures like multi-dimensional and jagged arrays, which offer even more flexibility for handling data.
