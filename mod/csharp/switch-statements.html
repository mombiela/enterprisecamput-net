Introduction

Switch statements in C# provide a way to execute different blocks of code based on the value of a variable or expression. They are an alternative to using multiple if-else statements and can make your code cleaner and more readable.

Basic Syntax

The basic syntax of a switch statement in C# is as follows:

switch (expression)
{
    case value1:
        // Code to execute if expression == value1
        break;
    case value2:
        // Code to execute if expression == value2
        break;
    // Additional cases...
    default:
        // Code to execute if none of the above cases match
        break;
}

Example

Here's a simple example to illustrate the use of a switch statement:

int day = 3;

switch (day)
{
    case 1:
        Console.WriteLine("Monday");
        break;
    case 2:
        Console.WriteLine("Tuesday");
        break;
    case 3:
        Console.WriteLine("Wednesday");
        break;
    case 4:
        Console.WriteLine("Thursday");
        break;
    case 5:
        Console.WriteLine("Friday");
        break;
    case 6:
        Console.WriteLine("Saturday");
        break;
    case 7:
        Console.WriteLine("Sunday");
        break;
    default:
        Console.WriteLine("Invalid day");
        break;
}

Key Concepts

  • Expression: The value or variable being evaluated.
  • Case: Each possible value of the expression that you want to handle.
  • Break: Terminates the current case block.
  • Default: Optional; executes if none of the cases match the expression.

Advanced Usage

Pattern Matching

C# 7.0 introduced pattern matching in switch statements, allowing for more complex conditions.

object obj = "Hello";

switch (obj)
{
    case int i:
        Console.WriteLine($"Integer: {i}");
        break;
    case string s:
        Console.WriteLine($"String: {s}");
        break;
    default:
        Console.WriteLine("Unknown type");
        break;
}

Switch Expressions

C# 8.0 introduced switch expressions, which provide a more concise syntax.

int number = 3;
string result = number switch
{
    1 => "One",
    2 => "Two",
    3 => "Three",
    _ => "Unknown"
};

Console.WriteLine(result); // Output: Three

Common Pitfalls

  • Forgetting the break statement: This can lead to fall-through behavior, where multiple cases are executed.
  • Not handling all possible values: Always include a default case to handle unexpected values.

Exercises

  1. Basic Switch Statement: Write a switch statement that prints the name of a month based on an integer input (1-12).
  2. Pattern Matching: Create a switch statement that handles different types of objects (e.g., int, string, bool) and prints a message for each type.
  3. Switch Expression: Use a switch expression to convert a grade (A, B, C, D, F) to a corresponding description (Excellent, Good, Average, Poor, Fail).

Conclusion

Switch statements are a powerful tool in C# for handling multiple conditions in a clean and readable way. By understanding the basic syntax and advanced features like pattern matching and switch expressions, you can write more efficient and maintainable code. Always remember to handle all possible cases and use the default case to cover unexpected values.
