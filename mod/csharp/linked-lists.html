Introduction to Linked Lists

Linked lists are a fundamental data structure in computer science, providing a way to store a collection of elements in a linear order. Unlike arrays, linked lists do not require contiguous memory allocation, making them more flexible for certain types of operations.

Key Concepts

  • Node: The basic unit of a linked list, containing data and a reference (or pointer) to the next node.
  • Head: The first node in a linked list.
  • Tail: The last node in a linked list, which points to null.
  • Singly Linked List: A linked list where each node points to the next node.
  • Doubly Linked List: A linked list where each node points to both the next and the previous node.

Singly Linked List

Structure of a Node

In C#, a node in a singly linked list can be represented as a class with two properties: one for the data and one for the reference to the next node.

public class Node
{
    public int Data { get; set; }
    public Node Next { get; set; }

    public Node(int data)
    {
        Data = data;
        Next = null;
    }
}

Creating a Singly Linked List

To create a singly linked list, we need a class that manages the nodes and provides methods to manipulate the list.

public class SinglyLinkedList
{
    private Node head;

    public SinglyLinkedList()
    {
        head = null;
    }

    // Method to add a node at the end
    public void AddLast(int data)
    {
        Node newNode = new Node(data);
        if (head == null)
        {
            head = newNode;
        }
        else
        {
            Node current = head;
            while (current.Next != null)
            {
                current = current.Next;
            }
            current.Next = newNode;
        }
    }

    // Method to display the list
    public void Display()
    {
        Node current = head;
        while (current != null)
        {
            Console.Write(current.Data + " -> ");
            current = current.Next;
        }
        Console.WriteLine("null");
    }
}

Example Usage

class Program
{
    static void Main(string[] args)
    {
        SinglyLinkedList list = new SinglyLinkedList();
        list.AddLast(1);
        list.AddLast(2);
        list.AddLast(3);
        list.Display(); // Output: 1 -> 2 -> 3 -> null
    }
}

Doubly Linked List

Structure of a Node

A node in a doubly linked list contains references to both the next and the previous nodes.

public class DoublyNode
{
    public int Data { get; set; }
    public DoublyNode Next { get; set; }
    public DoublyNode Previous { get; set; }

    public DoublyNode(int data)
    {
        Data = data;
        Next = null;
        Previous = null;
    }
}

Creating a Doubly Linked List

A doubly linked list class manages the nodes and provides methods to manipulate the list.

public class DoublyLinkedList
{
    private DoublyNode head;

    public DoublyLinkedList()
    {
        head = null;
    }

    // Method to add a node at the end
    public void AddLast(int data)
    {
        DoublyNode newNode = new DoublyNode(data);
        if (head == null)
        {
            head = newNode;
        }
        else
        {
            DoublyNode current = head;
            while (current.Next != null)
            {
                current = current.Next;
            }
            current.Next = newNode;
            newNode.Previous = current;
        }
    }

    // Method to display the list
    public void Display()
    {
        DoublyNode current = head;
        while (current != null)
        {
            Console.Write(current.Data + " <-> ");
            current = current.Next;
        }
        Console.WriteLine("null");
    }
}

Example Usage

class Program
{
    static void Main(string[] args)
    {
        DoublyLinkedList list = new DoublyLinkedList();
        list.AddLast(1);
        list.AddLast(2);
        list.AddLast(3);
        list.Display(); // Output: 1 <-> 2 <-> 3 <-> null
    }
}

Advanced Operations

Insertion at the Beginning

Adding a node at the beginning of the list requires updating the head and the next reference of the new node.

public void AddFirst(int data)
{
    Node newNode = new Node(data);
    newNode.Next = head;
    head = newNode;
}

Deletion of a Node

Deleting a node involves updating the references of the previous and next nodes to bypass the node to be deleted.

public void Delete(int data)
{
    if (head == null) return;

    if (head.Data == data)
    {
        head = head.Next;
        return;
    }

    Node current = head;
    while (current.Next != null && current.Next.Data != data)
    {
        current = current.Next;
    }

    if (current.Next != null)
    {
        current.Next = current.Next.Next;
    }
}

Reversing a Linked List

Reversing a linked list involves changing the direction of the pointers.

public void Reverse()
{
    Node previous = null;
    Node current = head;
    Node next = null;

    while (current != null)
    {
        next = current.Next;
        current.Next = previous;
        previous = current;
        current = next;
    }

    head = previous;
}

Conclusion

Linked lists are a versatile data structure that can be used to efficiently manage collections of data. Understanding how to implement and manipulate linked lists in C# is a valuable skill for any programmer. By mastering both singly and doubly linked lists, as well as advanced operations like insertion, deletion, and reversal, you can handle a wide range of programming challenges.
