Type Conversion in C#
Operators ►

Type conversion is an essential concept in C# programming, allowing developers to convert data from one type to another. This topic covers the basics of type conversion, including implicit and explicit conversions, and dives into more advanced concepts such as custom conversions and the use of conversion methods.

Implicit Conversions

Implicit conversions are automatically handled by the C# compiler. These conversions are safe and do not result in data loss.

  • Numeric Conversions: Converting smaller numeric types to larger numeric types.
  • Reference Conversions: Converting derived classes to base classes.

Examples of Implicit Conversions

int intValue = 123;
double doubleValue = intValue; // Implicit conversion from int to double

DerivedClass derived = new DerivedClass();
BaseClass baseClass = derived; // Implicit conversion from derived to base class

Explicit Conversions

Explicit conversions, or casting, require the programmer to specify the conversion. These conversions can potentially result in data loss or runtime exceptions.

  • Numeric Conversions: Converting larger numeric types to smaller numeric types.
  • Reference Conversions: Converting base classes to derived classes (requires a cast).

Examples of Explicit Conversions

double doubleValue = 123.45;
int intValue = (int)doubleValue; // Explicit conversion from double to int

BaseClass baseClass = new DerivedClass();
DerivedClass derived = (DerivedClass)baseClass; // Explicit conversion from base to derived class

Conversion Methods

C# provides several methods for converting between types, which are part of the System.Convert class.

  • ToInt32(): Converts a value to a 32-bit integer.
  • ToDouble(): Converts a value to a double-precision floating-point number.
  • ToString(): Converts a value to its string representation.

Examples of Conversion Methods

string stringValue = "123";
int intValue = Convert.ToInt32(stringValue); // Converts string to int

double doubleValue = Convert.ToDouble(stringValue); // Converts string to double

int number = 456;
string numberString = Convert.ToString(number); // Converts int to string

Custom Conversions

Custom conversions can be defined by overloading the implicit and explicit operators in a class. This allows for more control over how objects of that class are converted to and from other types.

Example of Custom Conversions

public class CustomType
{
    public int Value { get; set; }

    // Implicit conversion from CustomType to int
    public static implicit operator int(CustomType custom)
    {
        return custom.Value;
    }

    // Explicit conversion from int to CustomType
    public static explicit operator CustomType(int value)
    {
        return new CustomType { Value = value };
    }
}

// Usage
CustomType custom = new CustomType { Value = 123 };
int intValue = custom; // Implicit conversion
CustomType newCustom = (CustomType)intValue; // Explicit conversion

Summary

Type conversion in C# is a powerful feature that allows for flexible data manipulation. Understanding the differences between implicit and explicit conversions, as well as how to use conversion methods and define custom conversions, is crucial for writing robust and efficient C# code. Always be mindful of potential data loss and runtime exceptions when performing explicit conversions.
