Hands-On Labs and Exercises

  • Creating a Virtual Machine

    az vm create 
        --resource-group myResourceGroup 
        --name myVM 
        --image UbuntuLTS 
        --admin-username azureuser 
        --generate-ssh-keys
    • Explanation: This command creates a new virtual machine in Azure using the Azure CLI. It specifies the resource group, VM name, image type, admin username, and generates SSH keys for secure access.

  • Setting Up a Web App

    az webapp up --name myUniqueAppName --resource-group myResourceGroup --plan myAppServicePlan
    • Explanation: This command deploys a web app to Azure App Service. It specifies the app name, resource group, and the app service plan.

  • Configuring a Virtual Network

    az network vnet create 
        --name myVnet 
        --resource-group myResourceGroup 
        --subnet-name mySubnet
    • Explanation: This command creates a virtual network with a specified subnet. It helps in setting up the network infrastructure for your Azure resources.
