Setting Up HDInsight Cluster

# Example: Creating an HDInsight cluster using Azure CLI
az hdinsight create \
    --name myHDInsightCluster \
    --resource-group myResourceGroup \
    --cluster-type spark \
    --location eastus \
    --version 4.0 \
    --component-version spark=2.4 \
    --headnode-size Standard_D3_V2 \
    --workernode-size Standard_D3_V2 \
    --workernode-count 3

Working with Data in HDInsight

  • Loading data into HDInsight
  • Processing data with Apache Spark
  • Example: Word count program in PySpark
# Example: Word count in PySpark
from pyspark import SparkContext

sc = SparkContext("local", "WordCount")
text_file = sc.textFile("hdfs:///path/to/input.txt")
counts = text_file.flatMap(lambda line: line.split(" ")) \
                  .map(lambda word: (word, 1)) \
                  .reduceByKey(lambda a, b: a + b)
counts.saveAsTextFile("hdfs:///path/to/output")

  • Prerequisites
  • Step-by-step guide to creating a Data Lake Storage account
  • Configuration options
# Example: Creating an Azure Data Lake Storage account using Azure CLI
az storage account create \
    --name mydatalakestorage \
    --resource-group myResourceGroup \
    --location eastus \
    --sku Standard_LRS \
    --kind StorageV2 \
    --hierarchical-namespace true

  • Uploading and managing data

    --account-name mydatalakestorage \
    --file-system myfilesystem \
    --path /path/to/uploadedfile.txt \
    --source /local/path/to/file.txt
