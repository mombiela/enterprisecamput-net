Overview

Power BI Mobile allows users to access and interact with their Power BI reports and dashboards on mobile devices. This section will cover the basics of Power BI Mobile, its features, and how to get started with it.

Key Concepts

  • Power BI Mobile App: A mobile application available for iOS, Android, and Windows devices that allows users to view and interact with Power BI reports and dashboards.
  • Mobile Reports and Dashboards: Optimized versions of Power BI reports and dashboards designed for mobile viewing.
  • Interactivity: Features that allow users to interact with data, such as filters, drill-downs, and touch gestures.

Getting Started with Power BI Mobile

Installing the Power BI Mobile App

  1. iOS: Download from the Apple App Store.
  2. Android: Download from the Google Play Store.
  3. Windows: Download from the Microsoft Store.

Logging In

  • Open the Power BI Mobile app.
  • Sign in with your Power BI account credentials.

Navigating the App

  • Home Screen: Displays recent reports and dashboards.
  • Favorites: Access your favorite reports and dashboards quickly.
  • Shared with Me: View reports and dashboards shared by others.
  • Workspaces: Access different workspaces you are a part of.

Viewing and Interacting with Reports

Viewing Reports

  • Open a report from the home screen or workspace.
  • Swipe left or right to navigate between report pages.

Interacting with Data

  • Filters: Tap on filter icons to apply filters.
  • Drill-Down: Tap on data points to drill down into more detailed data.
  • Touch Gestures: Use pinch-to-zoom and swipe gestures to navigate and explore data.

Example: Applying a Filter

1. Open a report.
2. Tap the filter icon on the top right corner.
3. Select the filter criteria (e.g., Date Range, Product Category).
4. The report will update to reflect the selected filter.

Customizing Mobile Reports

Creating Mobile-Optimized Reports

  • Use Power BI Desktop to create mobile-optimized layouts.
  • Switch to the mobile layout view in Power BI Desktop.
  • Arrange visuals to fit the mobile screen.

Example: Creating a Mobile Layout

1. Open your report in Power BI Desktop.
2. Click on the "View" tab.
3. Select "Phone Layout".
4. Drag and drop visuals to arrange them for mobile viewing.
5. Save and publish the report.

Notifications and Alerts

  • Data Alerts: Set up alerts to get notified when data changes.
  • Push Notifications: Receive notifications on your mobile device for important updates.

Setting Up a Data Alert

1. Open a dashboard.
2. Tap on a tile you want to set an alert for.
3. Tap the ellipsis (...) and select "Manage alerts".
4. Set the conditions for the alert (e.g., when sales exceed a certain amount).
5. Save the alert.

Conclusion

Power BI Mobile provides a convenient way to access and interact with your Power BI reports and dashboards on the go. By understanding how to navigate the app, view and interact with reports, and customize mobile layouts, you can make the most of Power BI's mobile capabilities. Whether you are checking data on the fly or setting up alerts for critical metrics, Power BI Mobile ensures you stay connected with your data anytime, anywhere.
