Overview

Theming in Ionic allows you to customize the look and feel of your application to match your brand or design preferences. This section will guide you through the basics of theming in Ionic, from understanding the core concepts to implementing custom themes.

Key Concepts

  • CSS Variables: Ionic uses CSS variables (custom properties) to define theme colors and other styles.
  • Global Styles: Styles that apply to the entire application.
  • Component Styles: Styles that apply to specific components.
  • Dark Mode: A theme that uses darker colors for better readability in low-light environments.

CSS Variables

CSS variables are a powerful way to manage and apply styles across your Ionic application. They allow you to define reusable values that can be applied to various CSS properties.

Example

:root {
  --ion-color-primary: #3880ff;
  --ion-color-secondary: #0cd1e8;
  --ion-color-tertiary: #7044ff;
}

In this example, --ion-color-primary, --ion-color-secondary, and --ion-color-tertiary are CSS variables that define the primary, secondary, and tertiary colors of your theme.

Global Styles

Global styles are defined in the src/theme/variables.scss file. This is where you can set the default styles for your entire application.

Example

// src/theme/variables.scss
:root {
  --ion-font-family: 'Open Sans', sans-serif;
  --ion-color-primary: #3880ff;
  --ion-color-secondary: #0cd1e8;
  --ion-color-tertiary: #7044ff;
}

In this example, we set the default font family and colors for the entire application.

Component Styles

Component styles are scoped to individual components. You can define these styles in the component's stylesheet.

Example

// src/pages/home/home.page.scss
.page-home {
  ion-content {
    --background: #f0f0f0;
  }

  .custom-button {
    --background: var(--ion-color-primary);
    --color: white;
  }
}

In this example, we set a custom background color for the ion-content and a custom style for a button within the home page.

Dark Mode

Ionic supports dark mode, which can be enabled by setting a CSS class on the body element.

Example

<!-- index.html -->
<body class="dark-theme">
  <app-root></app-root>
</body>

Dark Mode Variables

You can define dark mode-specific variables in your variables.scss file.

// src/theme/variables.scss
body.dark-theme {
  --ion-background-color: #222428;
  --ion-text-color: #ffffff;
}

In this example, we set the background and text colors for dark mode.

Summary

Theming in Ionic is a powerful way to customize the appearance of your application. By using CSS variables, global styles, component styles, and dark mode, you can create a cohesive and visually appealing user experience. Remember to:

  • Utilize CSS variables for reusable and maintainable styles.
  • Define global styles in variables.scss for application-wide consistency.
  • Scope styles to individual components when necessary.
  • Implement dark mode to enhance readability in low-light environments.

With these tools and techniques, you can create a unique and polished look for your Ionic applications.
