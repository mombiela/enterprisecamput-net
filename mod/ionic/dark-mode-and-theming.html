Dark Mode and Theming in Ionic
Using Sass with Ionic ►

Introduction

In this section, we will explore how to implement dark mode and theming in Ionic applications. Theming allows you to customize the look and feel of your app, while dark mode provides a user-friendly experience in low-light environments.

Key Concepts

  • Theming: Customizing the appearance of your app using CSS variables.
  • Dark Mode: A theme that uses dark colors to reduce eye strain in low-light conditions.
  • CSS Variables: Variables defined in CSS that can be reused throughout your stylesheets.

Setting Up Theming

Default Theme

Ionic uses a default theme that you can customize. The default theme is defined in the variables.scss file.

// src/theme/variables.scss

:root {
  --ion-color-primary: #3880ff;
  --ion-color-secondary: #0cd1e8;
  --ion-color-tertiary: #7044ff;
  --ion-color-success: #10dc60;
  --ion-color-warning: #ffce00;
  --ion-color-danger: #f04141;
  --ion-color-dark: #222428;
  --ion-color-medium: #989aa2;
  --ion-color-light: #f4f5f8;
}

Customizing the Theme

You can customize the theme by changing the values of these CSS variables.

// src/theme/variables.scss

:root {
  --ion-color-primary: #ff5733; // Changed primary color to a custom orange
  --ion-color-secondary: #33ff57; // Changed secondary color to a custom green
  // Add more customizations as needed
}

Implementing Dark Mode

Detecting Dark Mode Preference

You can use the prefers-color-scheme media query to detect if the user prefers dark mode.

// src/theme/variables.scss

@media (prefers-color-scheme: dark) {
  :root {
    --ion-color-primary: #222428;
    --ion-color-secondary: #f4f5f8;
    --ion-color-tertiary: #7044ff;
    --ion-color-success: #10dc60;
    --ion-color-warning: #ffce00;
    --ion-color-danger: #f04141;
    --ion-color-dark: #f4f5f8;
    --ion-color-medium: #989aa2;
    --ion-color-light: #222428;
  }
}

Toggling Dark Mode Manually

You can also allow users to toggle dark mode manually. This involves adding a toggle switch in your app and updating the theme accordingly.

HTML

<!-- src/app/app.component.html -->

<ion-header>
  <ion-toolbar>
    <ion-title>
      Dark Mode Example
    </ion-title>
    <ion-buttons slot="end">
      <ion-button (click)="toggleDarkMode()">
        <ion-icon slot="icon-only" name="moon"></ion-icon>
      </ion-button>
    </ion-buttons>
  </ion-toolbar>
</ion-header>

TypeScript

// src/app/app.component.ts

import { Component } from '@angular/core';

@Component({
  selector: 'app-root',
  templateUrl: 'app.component.html',
  styleUrls: ['app.component.scss'],
})
export class AppComponent {
  constructor() {}

  toggleDarkMode() {
    document.body.classList.toggle('dark', !document.body.classList.contains('dark'));
  }
}

CSS

// src/theme/variables.scss

body.dark {
  --ion-color-primary: #222428;
  --ion-color-secondary: #f4f5f8;
  --ion-color-tertiary: #7044ff;
  --ion-color-success: #10dc60;
  --ion-color-warning: #ffce00;
  --ion-color-danger: #f04141;
  --ion-color-dark: #f4f5f8;
  --ion-color-medium: #989aa2;
  --ion-color-light: #222428;
}

Advanced Theming Techniques

Dynamic Theming

You can create a more dynamic theming system by using JavaScript to update CSS variables at runtime.

TypeScript

// src/app/app.component.ts

import { Component } from '@angular/core';

@Component({
  selector: 'app-root',
  templateUrl: 'app.component.html',
  styleUrls: ['app.component.scss'],
})
export class AppComponent {
  constructor() {}

  setTheme(theme: 'light' | 'dark') {
    const root = document.documentElement;
    if (theme === 'dark') {
      root.style.setProperty('--ion-color-primary', '#222428');
      root.style.setProperty('--ion-color-secondary', '#f4f5f8');
      // Set other dark theme variables
    } else {
      root.style.setProperty('--ion-color-primary', '#3880ff');
      root.style.setProperty('--ion-color-secondary', '#0cd1e8');
      // Set other light theme variables
    }
  }
}

Using CSS Variables in Components

You can also use CSS variables directly in your components to create more granular theming.

HTML

<!-- src/app/home/home.page.html -->

<ion-content>
  <div class="custom-box">
    Themed Box
  </div>
</ion-content>

CSS

// src/app/home/home.page.scss

.custom-box {
  background-color: var(--ion-color-primary);
  color: var(--ion-color-light);
  padding: 16px;
  border-radius: 8px;
}

Conclusion

In this section, we covered the basics of theming and implementing dark mode in Ionic applications. We learned how to customize the default theme, detect dark mode preferences, toggle dark mode manually, and apply advanced theming techniques. By mastering these concepts, you can create visually appealing and user-friendly applications that cater to a wide range of user preferences.
