Introduction

In this section, we will guide you through the process of creating your first Ionic app. Ionic is a popular framework for building cross-platform mobile applications using web technologies like HTML, CSS, and JavaScript. By the end of this section, you will have a basic Ionic app running on your local machine.

Prerequisites

Before we start, ensure you have the following installed on your computer:

  • Node.js and npm (Node Package Manager)
  • Ionic CLI (Command Line Interface)
  • A code editor (e.g., Visual Studio Code)

Setting Up Your Environment

  1. Install Node.js and npm: Download and install Node.js from nodejs.org. This will also install npm.
  2. Install Ionic CLI: Open your terminal or command prompt and run the following command: 
    npm install -g @ionic/cli
    This command installs the Ionic CLI globally on your system.

Creating a New Ionic Project

  1. Open your terminal and navigate to the directory where you want to create your project.

  2. Run the following command to create a new Ionic project:

    ionic start myFirstApp blank
    • myFirstApp is the name of your project.
    • blank is the starter template. You can choose other templates like tabs or sidemenu.

  3. Navigate into your project directory:

    cd myFirstApp

Running Your Ionic App

  1. Serve the app locally by running: 
    ionic serve
    This command will compile your app and open it in your default web browser. You should see a blank Ionic app running.

Understanding the Project Structure

Here is a brief overview of the key files and directories in your new Ionic project:

  • src/: Contains the source code of your app.
    • app/: Contains the main app module and root component.
    • assets/: Contains static assets like images and fonts.
    • environments/: Contains environment-specific configuration files.
    • theme/: Contains global styles and variables.
    • index.html: The main HTML file.
    • main.ts: The main entry point for the app.
  • www/: Contains the compiled output of your app.
  • ionic.config.json: Configuration file for the Ionic CLI.

Adding a Simple Component

Let's add a simple component to your app to display a welcome message.

  1. Generate a new component:

    ionic generate component Welcome

  2. Modify the component:

    • Open src/app/welcome/welcome.component.html and add the following code: 
      <div>
  <h1>Welcome to My First Ionic App!</h1>
</div>
    • Open src/app/welcome/welcome.component.ts and ensure it looks like this: 
      import { Component } from '@angular/core';

@Component({
  selector: 'app-welcome',
  templateUrl: './welcome.component.html',
  styleUrls: ['./welcome.component.scss']
})
export class WelcomeComponent {}

  3. Use the component in your app:

    • Open src/app/app.component.html and add the following line: 
      <app-welcome></app-welcome>

  4. Serve the app again:

    ionic serve

    You should now see the welcome message displayed in your app.

Conclusion

Congratulations! You have successfully created your first Ionic app and added a simple component. In this section, you learned how to set up your development environment, create a new Ionic project, understand the project structure, and add a basic component. This foundational knowledge will help you as you progress through more advanced topics in Ionic development.
