Project 1: Sentiment Analysis on Social Media

Project Description

The objective of this project is to analyze the sentiments expressed in social media posts about a specific topic. We will use natural language processing (NLP) techniques and Big Data tools to collect, process, and analyze the data.

Project Steps

  1. Data Collection:

    • Use the Twitter API to collect tweets related to a specific topic.
    • Store the data in a distributed storage system like Hadoop HDFS or Amazon S3.

  2. Data Processing:

    • Clean and preprocess the data using Apache Spark.
    • Tokenize and normalize the text to facilitate analysis.

  3. Sentiment Analysis:

    • Apply a pre-trained sentiment analysis model (e.g., VADER or TextBlob).
    • Classify the tweets into sentiment categories (positive, negative, neutral).

  4. Results Visualization:

    • Use visualization tools like Tableau or Matplotlib to present the analysis results.

Code Example

from pyspark.sql import SparkSession
from textblob import TextBlob

# Create a Spark session
spark = SparkSession.builder.appName("TwitterSentimentAnalysis").getOrCreate()

# Load tweet data from HDFS
tweets_df = spark.read.json("hdfs://path/to/tweets.json")

# Function to analyze the sentiment of a tweet
def analyze_sentiment(tweet):
    analysis = TextBlob(tweet)
    if analysis.sentiment.polarity > 0:
        return 'positive'
    elif analysis.sentiment.polarity == 0:
        return 'neutral'
    else:
        return 'negative'

# Apply the sentiment analysis function to the tweets
tweets_df = tweets_df.withColumn("sentiment", analyze_sentiment(tweets_df.text))

# Show the results
tweets_df.show()

Project 2: Product Recommender

Project Description

In this project, we will develop a product recommendation system using collaborative filtering techniques. The objective is to recommend products to users based on their past preferences and behaviors.

Project Steps

  1. Data Collection:

    • Obtain user transaction data from an online store.
    • Store the data in a distributed storage system.

  2. Data Processing:

    • Clean and preprocess the data using Apache Spark.
    • Create a user-product matrix representing the interactions.

  3. Building the Recommendation Model:

    • Use Spark's MLlib library to build a collaborative filtering model.
    • Train the model with the transaction data.

  4. Generating Recommendations:

    • Use the trained model to generate personalized recommendations for each user.
    • Evaluate the model's accuracy using metrics like RMSE.

Code Example

from pyspark.sql import SparkSession
from pyspark.ml.recommendation import ALS

# Create a Spark session
spark = SparkSession.builder.appName("ProductRecommender").getOrCreate()

# Load transaction data from HDFS
transactions_df = spark.read.csv("hdfs://path/to/transactions.csv", header=True, inferSchema=True)

# Create a user-product matrix
user_product_df = transactions_df.select("user_id", "product_id", "rating")

# Build the recommendation model using ALS
als = ALS(userCol="user_id", itemCol="product_id", ratingCol="rating", coldStartStrategy="drop")
model = als.fit(user_product_df)

# Generate recommendations for all users
user_recommendations = model.recommendForAllUsers(10)

# Show the recommendations
user_recommendations.show()
