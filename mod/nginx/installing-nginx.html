Introduction

NGINX is a high-performance HTTP server, reverse proxy, and an IMAP/POP3 proxy server. This section will guide you through the process of installing NGINX on various operating systems. By the end of this section, you will have a running NGINX server on your machine.

Prerequisites

Before installing NGINX, ensure you have the following:

  • A machine running a Unix-based OS (Linux, macOS) or Windows.
  • Basic command-line knowledge.
  • Administrative (root) access to your machine.

Installing NGINX on Linux

Using Package Manager (Ubuntu/Debian)

  1. Update the package list: 
    sudo apt update
  2. Install NGINX: 
    sudo apt install nginx
  3. Start NGINX: 
    sudo systemctl start nginx
  4. Enable NGINX to start on boot: 
    sudo systemctl enable nginx
  5. Verify the installation: Open your web browser and navigate to http://localhost. You should see the NGINX welcome page.

Using Package Manager (CentOS/RHEL)

  1. Install EPEL repository: 
    sudo yum install epel-release
  2. Install NGINX: 
    sudo yum install nginx
  3. Start NGINX: 
    sudo systemctl start nginx
  4. Enable NGINX to start on boot: 
    sudo systemctl enable nginx
  5. Verify the installation: Open your web browser and navigate to http://localhost. You should see the NGINX welcome page.

Installing NGINX on macOS

Using Homebrew

  1. Install Homebrew (if not already installed): 
    /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
  2. Install NGINX: 
    brew install nginx
  3. Start NGINX: 
    sudo nginx
  4. Verify the installation: Open your web browser and navigate to http://localhost:8080. You should see the NGINX welcome page.

Installing NGINX on Windows

Using NGINX Binary

  1. Download the NGINX binary: Go to the NGINX download page and download the stable version for Windows.
  2. Extract the downloaded file: Extract the contents of the zip file to a directory of your choice (e.g., C:\nginx).
  3. Run NGINX: Open Command Prompt and navigate to the NGINX directory: 
    cd C:\nginx
    Start NGINX: 
    start nginx
  4. Verify the installation: Open your web browser and navigate to http://localhost. You should see the NGINX welcome page.

Common Post-Installation Steps

Checking NGINX Status

  • Linux: 
    sudo systemctl status nginx
  • macOS: 
    sudo nginx -t
  • Windows: Open Command Prompt and navigate to the NGINX directory, then run: 
    nginx -t

Stopping and Restarting NGINX

  • Linux: 
    sudo systemctl stop nginx
sudo systemctl restart nginx
  • macOS: 
    sudo nginx -s stop
sudo nginx -s reload
  • Windows: Open Command Prompt and navigate to the NGINX directory, then run: 
    nginx -s stop
nginx -s reload

Conclusion

In this section, you have learned how to install NGINX on various operating systems including Linux, macOS, and Windows. You have also learned how to start, stop, and verify the status of your NGINX server. With NGINX installed, you are now ready to configure and use it for your web server needs.
