Introduction to Loops

Loops are fundamental constructs in programming that allow you to execute a block of code multiple times. Java supports several types of loops, each with its own use case.

Key Concepts

  • Iteration: Repeated execution of a set of statements.
  • Loop Control Statements: Keywords that manage the flow of loops (break, continue).

Types of Loops

1. `for` Loop

The for loop is used when the number of iterations is known beforehand.

Syntax

for (initialization; condition; update) {
    // Code to be executed
}

Example

for (int i = 0; i < 5; i++) {
    System.out.println("Iteration: " + i);
}
  • Initialization: int i = 0 sets the starting point.
  • Condition: i < 5 checks if the loop should continue.
  • Update: i++ increments the loop counter.

2. `while` Loop

The while loop is used when the number of iterations is not known and depends on a condition.

Syntax

while (condition) {
    // Code to be executed
}

Example

int i = 0;
while (i < 5) {
    System.out.println("Iteration: " + i);
    i++;
}
  • Condition: i < 5 checks if the loop should continue.

3. `do-while` Loop

The do-while loop is similar to the while loop but guarantees at least one iteration.

Syntax

do {
    // Code to be executed
} while (condition);

Example

int i = 0;
do {
    System.out.println("Iteration: " + i);
    i++;
} while (i < 5);
  • Condition: i < 5 checks if the loop should continue after the first iteration.

4. Enhanced `for` Loop (for-each)

The enhanced for loop is used to iterate over arrays or collections.

Syntax

for (type element : array) {
    // Code to be executed
}

Example

int[] numbers = {1, 2, 3, 4, 5};
for (int number : numbers) {
    System.out.println("Number: " + number);
}
  • Type: int specifies the type of elements in the array.
  • Element: number is the variable that holds each element during iteration.

Loop Control Statements

`break` Statement

The break statement exits the loop immediately.

Example

for (int i = 0; i < 10; i++) {
    if (i == 5) {
        break;
    }
    System.out.println("Iteration: " + i);
}
  • Output: Iterations from 0 to 4.

`continue` Statement

The continue statement skips the current iteration and proceeds to the next one.

Example

for (int i = 0; i < 10; i++) {
    if (i % 2 == 0) {
        continue;
    }
    System.out.println("Odd number: " + i);
}
  • Output: Odd numbers from 1 to 9.

Summary

Loops are essential for performing repetitive tasks efficiently. Java provides several types of loops (for, while, do-while, and enhanced for loop) to handle different scenarios. Understanding how to use loop control statements like break and continue can help manage the flow of loops effectively. Practice with these constructs will build a strong foundation for more advanced programming concepts.
