Introduction

Welcome to your journey into Java programming! In this section, we will guide you through writing your very first Java program. By the end of this lesson, you will understand the basic structure of a Java program and how to compile and run it.

Setting Up Your Environment

Before we start coding, you need to set up your development environment.

  • Install Java Development Kit (JDK): Download and install the latest version of the JDK from the official Oracle website.
  • Install an Integrated Development Environment (IDE): We recommend using IntelliJ IDEA, Eclipse, or NetBeans. These tools provide a user-friendly interface for writing, compiling, and running Java programs.

Writing Your First Java Program

Let's write a simple Java program that prints "Hello, World!" to the console.

Step-by-Step Guide

  1. Create a New Java File: Open your IDE and create a new Java project. Inside the project, create a new Java file named HelloWorld.java.

  2. Write the Code: Enter the following code into your HelloWorld.java file:

    public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

  3. Explanation of the Code:

    • public class HelloWorld: This line declares a public class named HelloWorld. In Java, every application must have at least one class definition.
    • public static void main(String[] args): This is the main method. It's the entry point of any Java application. The Java Virtual Machine (JVM) calls this method to start the program.
    • System.out.println("Hello, World!");: This line prints the string "Hello, World!" to the console. System.out is a standard output stream, and println is a method that prints a line of text.

Compiling and Running Your Program

To see your program in action, you need to compile and run it.

Compiling the Program

  • Using the Command Line:
    1. Open your command line interface (CLI).
    2. Navigate to the directory where your HelloWorld.java file is located.
    3. Compile the program by typing: 
      javac HelloWorld.java
    4. If there are no errors, this command will produce a file named HelloWorld.class.

Running the Program

  • Using the Command Line:
    1. In the same directory, run the compiled program by typing: 
      java HelloWorld
    2. You should see the output: 
      Hello, World!

Common Errors and Troubleshooting

Here are some common errors you might encounter and how to fix them:

  • Class Not Found: Ensure that you are in the correct directory and that the class name matches the file name.
  • Syntax Errors: Double-check your code for typos or missing semicolons.
  • JDK Not Installed: Make sure the JDK is properly installed and the javac and java commands are available in your system's PATH.

Conclusion

Congratulations! You've written, compiled, and run your first Java program. This simple "Hello, World!" example introduces you to the basic structure of a Java application and the process of compiling and running Java code. As you progress through this course, you'll build on these fundamentals to create more complex and powerful Java applications. Keep practicing, and happy coding!
