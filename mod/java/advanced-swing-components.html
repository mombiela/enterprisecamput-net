Introduction

In this section, we will delve into advanced Swing components in Java. Swing is a part of Java Foundation Classes (JFC) used to create window-based applications. While basic components like JButton, JLabel, and JTextField are commonly used, Swing also offers more advanced components for creating sophisticated user interfaces.

Key Advanced Swing Components

  • JTable
  • JTree
  • JTabbedPane
  • JToolBar
  • JInternalFrame

JTable

JTable is used to display and edit regular two-dimensional tables of cells.

Example:

import javax.swing.*;
import javax.swing.table.DefaultTableModel;

public class JTableExample {
    public static void main(String[] args) {
        JFrame frame = new JFrame("JTable Example");
        frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
        frame.setSize(400, 300);

        String[] columns = {"ID", "Name", "Age"};
        Object[][] data = {
            {1, "John Doe", 25},
            {2, "Jane Smith", 30},
            {3, "Kathy Green", 22}
        };

        DefaultTableModel model = new DefaultTableModel(data, columns);
        JTable table = new JTable(model);
        JScrollPane scrollPane = new JScrollPane(table);

        frame.add(scrollPane);
        frame.setVisible(true);
    }
}
  • Explanation: This example creates a simple table with three columns: ID, Name, and Age. The DefaultTableModel is used to manage the data.

JTree

JTree is used to display hierarchical data in a tree structure.

Example:

import javax.swing.*;
import javax.swing.tree.DefaultMutableTreeNode;

public class JTreeExample {
    public static void main(String[] args) {
        JFrame frame = new JFrame("JTree Example");
        frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
        frame.setSize(400, 300);

        DefaultMutableTreeNode root = new DefaultMutableTreeNode("Root");
        DefaultMutableTreeNode child1 = new DefaultMutableTreeNode("Child 1");
        DefaultMutableTreeNode child2 = new DefaultMutableTreeNode("Child 2");

        root.add(child1);
        root.add(child2);

        JTree tree = new JTree(root);
        JScrollPane scrollPane = new JScrollPane(tree);

        frame.add(scrollPane);
        frame.setVisible(true);
    }
}
  • Explanation: This example creates a simple tree with a root node and two child nodes. DefaultMutableTreeNode is used to create the nodes.

JTabbedPane

JTabbedPane is used to create a tabbed pane interface, where multiple tabs can be added.

Example:

import javax.swing.*;

public class JTabbedPaneExample {
    public static void main(String[] args) {
        JFrame frame = new JFrame("JTabbedPane Example");
        frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
        frame.setSize(400, 300);

        JTabbedPane tabbedPane = new JTabbedPane();

        JPanel panel1 = new JPanel();
        panel1.add(new JLabel("This is Tab 1"));
        tabbedPane.addTab("Tab 1", panel1);

        JPanel panel2 = new JPanel();
        panel2.add(new JLabel("This is Tab 2"));
        tabbedPane.addTab("Tab 2", panel2);

        frame.add(tabbedPane);
        frame.setVisible(true);
    }
}
  • Explanation: This example creates a tabbed pane with two tabs, each containing a simple label.

JToolBar

JToolBar is used to create a toolbar that can contain buttons, labels, and other components.

Example:

import javax.swing.*;

public class JToolBarExample {
    public static void main(String[] args) {
        JFrame frame = new JFrame("JToolBar Example");
        frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
        frame.setSize(400, 300);

        JToolBar toolBar = new JToolBar();
        JButton button1 = new JButton("Button 1");
        JButton button2 = new JButton("Button 2");

        toolBar.add(button1);
        toolBar.add(button2);

        frame.add(toolBar, "North");
        frame.setVisible(true);
    }
}
  • Explanation: This example creates a toolbar with two buttons and adds it to the top (North) of the frame.

JInternalFrame

JInternalFrame is used to create internal frames (sub-windows) within a main application window.

Example:

import javax.swing.*;

public class JInternalFrameExample {
    public static void main(String[] args) {
        JFrame frame = new JFrame("JInternalFrame Example");
        frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
        frame.setSize(600, 400);

        JDesktopPane desktopPane = new JDesktopPane();

        JInternalFrame internalFrame = new JInternalFrame("Internal Frame", true, true, true, true);
        internalFrame.setSize(200, 150);
        internalFrame.setVisible(true);

        desktopPane.add(internalFrame);
        frame.add(desktopPane);
        frame.setVisible(true);
    }
}
  • Explanation: This example creates an internal frame within a desktop pane. The internal frame can be moved, resized, closed, and maximized.

Conclusion

Advanced Swing components like JTable, JTree, JTabbedPane, JToolBar, and JInternalFrame provide powerful tools for creating complex and feature-rich Java applications. Understanding and utilizing these components can significantly enhance the functionality and user experience of your applications. Practice using these components in different scenarios to become proficient in building advanced Swing-based user interfaces.
