Introduction to Swing

Swing is a part of Java Foundation Classes (JFC) used to create window-based applications. It is built on top of the Abstract Window Toolkit (AWT) and provides a richer set of GUI components.

  • Lightweight Components: Unlike AWT, Swing components are lightweight, meaning they are written entirely in Java and do not rely on native code.
  • Pluggable Look and Feel: Swing allows you to change the appearance of your application without altering the underlying code.
  • MVC Architecture: Swing follows the Model-View-Controller architecture, which separates data (model), user interface (view), and control logic (controller).

Setting Up a Swing Application

To create a basic Swing application, you need to follow these steps:

  1. Import Swing Packages: Import the necessary Swing and AWT packages.
  2. Create a JFrame: The main window of your application.
  3. Add Components: Add various Swing components like buttons, labels, text fields, etc.
  4. Set Layout: Define the layout manager to arrange components.
  5. Display the Frame: Make the frame visible.

Example: A Simple Swing Application

import javax.swing.JFrame;
import javax.swing.JLabel;

public class HelloWorldSwing {
    public static void main(String[] args) {
        // Create a new JFrame
        JFrame frame = new JFrame("Hello World Swing");
        
        // Set the default close operation
        frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
        
        // Add a JLabel to the frame
        JLabel label = new JLabel("Hello, World!");
        frame.getContentPane().add(label);
        
        // Set the frame size
        frame.setSize(300, 100);
        
        // Make the frame visible
        frame.setVisible(true);
    }
}
  • Explanation:
    • JFrame: The main window.
    • JLabel: A simple text label.
    • setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE): Ensures the application exits when the window is closed.
    • setSize(300, 100): Sets the size of the window.
    • setVisible(true): Makes the window visible.

Common Swing Components

Swing provides a variety of components to build a user interface.

  • JButton: A button that can trigger an action.
  • JLabel: A display area for a short text string or an image.
  • JTextField: A single-line text field for user input.
  • JTextArea: A multi-line area to display/edit text.
  • JCheckBox: A box that can be checked or unchecked.
  • JRadioButton: A button in a group where only one can be selected.
  • JComboBox: A drop-down list of items.

Example: Adding More Components

import javax.swing.*;

public class SwingComponentsExample {
    public static void main(String[] args) {
        JFrame frame = new JFrame("Swing Components Example");
        frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
        frame.setSize(400, 200);
        
        // Create components
        JButton button = new JButton("Click Me");
        JLabel label = new JLabel("Label:");
        JTextField textField = new JTextField(20);
        JCheckBox checkBox = new JCheckBox("Check Me");
        JRadioButton radioButton = new JRadioButton("Select Me");
        JComboBox<String> comboBox = new JComboBox<>(new String[]{"Option 1", "Option 2", "Option 3"});
        
        // Add components to the frame
        JPanel panel = new JPanel();
        panel.add(label);
        panel.add(textField);
        panel.add(button);
        panel.add(checkBox);
        panel.add(radioButton);
        panel.add(comboBox);
        
        frame.getContentPane().add(panel);
        frame.setVisible(true);
    }
}
  • Explanation:
    • JPanel: A container to group components.
    • JButton, JLabel, JTextField, JCheckBox, JRadioButton, JComboBox: Various Swing components added to the panel.

Event Handling in Swing

Swing uses the delegation event model to handle events. This involves three components:

  • Event Source: The component that generates the event (e.g., a button).
  • Event Object: Encapsulates the event information.
  • Event Listener: An interface that receives and processes the event.

Example: Handling Button Clicks

import javax.swing.*;
import java.awt.event.ActionEvent;
import java.awt.event.ActionListener;

public class ButtonClickExample {
    public static void main(String[] args) {
        JFrame frame = new JFrame("Button Click Example");
        frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
        frame.setSize(300, 200);
        
        JButton button = new JButton("Click Me");
        JLabel label = new JLabel("Button not clicked");
        
        // Add ActionListener to the button
        button.addActionListener(new ActionListener() {
            @Override
            public void actionPerformed(ActionEvent e) {
                label.setText("Button clicked!");
            }
        });
        
        JPanel panel = new JPanel();
        panel.add(button);
        panel.add(label);
        
        frame.getContentPane().add(panel);
        frame.setVisible(true);
    }
}
  • Explanation:
    • ActionListener: An interface to handle action events.
    • actionPerformed(ActionEvent e): Method called when the button is clicked.
    • label.setText("Button clicked!"): Updates the label text when the button is clicked.

Layout Managers

Layout managers are used to arrange components in a container. Some common layout managers include:

  • FlowLayout: Arranges components in a left-to-right flow.
  • BorderLayout: Arranges components in five regions: North, South, East, West, and Center.
  • GridLayout: Arranges components in a grid of cells.
  • BoxLayout: Arranges components either vertically or horizontally.

Example: Using BorderLayout

import javax.swing.*;
import java.awt.*;

public class BorderLayoutExample {
    public static void main(String[] args) {
        JFrame frame = new JFrame("BorderLayout Example");
        frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
        frame.setSize(400, 300);
        
        JButton northButton = new JButton("North");
        JButton southButton = new JButton("South");
        JButton eastButton = new JButton("East");
        JButton westButton = new JButton("West");
        JButton centerButton = new JButton("Center");
        
        frame.setLayout(new BorderLayout());
        frame.add(northButton, BorderLayout.NORTH);
        frame.add(southButton, BorderLayout.SOUTH);
        frame.add(eastButton, BorderLayout.EAST);
        frame.add(westButton, BorderLayout.WEST);
        frame.add(centerButton, BorderLayout.CENTER);
        
        frame.setVisible(true);
    }
}
  • Explanation:
    • BorderLayout: Divides the container into five regions.
    • frame.add(component, BorderLayout.REGION): Adds components to specific regions.

Conclusion

Swing is a powerful toolkit for creating GUI applications in Java. By understanding the basics of Swing components, event handling, and layout managers, you can build interactive and user-friendly applications. As you progress, you can explore more advanced features and customizations to enhance your applications further.
