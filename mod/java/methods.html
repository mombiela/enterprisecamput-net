Introduction to Methods

Methods are blocks of code that perform a specific task and can be called upon whenever needed. They help in organizing code, making it reusable, and improving readability.

Key Concepts

  • Definition: A method is a collection of statements grouped together to perform an operation.
  • Syntax: The basic syntax for defining a method in Java is: 
    returnType methodName(parameters) {
    // method body
}
  • Return Type: Specifies the type of value the method returns. Use void if the method does not return a value.
  • Method Name: A unique identifier for the method.
  • Parameters: Inputs to the method, enclosed in parentheses. If no inputs are needed, leave the parentheses empty.
  • Method Body: The block of code that defines what the method does.

Example

public class Main {
    // Method to add two numbers
    public int add(int a, int b) {
        return a + b;
    }

    public static void main(String[] args) {
        Main obj = new Main();
        int result = obj.add(5, 3);
        System.out.println("Sum: " + result); // Output: Sum: 8
    }
}

In this example:

  • add is a method that takes two integers as parameters and returns their sum.
  • main method is the entry point of the program where the add method is called.

Types of Methods

Instance Methods

Instance methods belong to an instance of a class. They require an object of the class to be created before they can be called.

public class Calculator {
    public int multiply(int a, int b) {
        return a * b;
    }

    public static void main(String[] args) {
        Calculator calc = new Calculator();
        int product = calc.multiply(4, 5);
        System.out.println("Product: " + product); // Output: Product: 20
    }
}

Static Methods

Static methods belong to the class rather than any instance. They can be called without creating an object of the class.

public class MathUtils {
    public static int square(int num) {
        return num * num;
    }

    public static void main(String[] args) {
        int result = MathUtils.square(6);
        System.out.println("Square: " + result); // Output: Square: 36
    }
}

Method Overloading

Method overloading allows a class to have more than one method with the same name, as long as their parameter lists are different.

public class Printer {
    public void print(String s) {
        System.out.println(s);
    }

    public void print(int i) {
        System.out.println(i);
    }

    public static void main(String[] args) {
        Printer printer = new Printer();
        printer.print("Hello, World!"); // Output: Hello, World!
        printer.print(100);             // Output: 100
    }
}

Advanced Method Concepts

Recursion

Recursion is a technique where a method calls itself to solve a problem.

public class Factorial {
    public int factorial(int n) {
        if (n == 0) {
            return 1;
        } else {
            return n * factorial(n - 1);
        }
    }

    public static void main(String[] args) {
        Factorial fact = new Factorial();
        int result = fact.factorial(5);
        System.out.println("Factorial: " + result); // Output: Factorial: 120
    }
}

Pass-by-Value

Java uses pass-by-value for passing arguments to methods. This means that a copy of the argument is passed to the method.

public class PassByValue {
    public void modifyValue(int num) {
        num = 10;
    }

    public static void main(String[] args) {
        int x = 5;
        PassByValue obj = new PassByValue();
        obj.modifyValue(x);
        System.out.println("Value of x: " + x); // Output: Value of x: 5
    }
}

Conclusion

Methods are fundamental building blocks in Java programming. They help in organizing code, making it reusable, and improving readability. Understanding different types of methods, method overloading, recursion, and the concept of pass-by-value is crucial for writing efficient and maintainable Java programs. Practice writing and calling methods to become proficient in using them effectively.
