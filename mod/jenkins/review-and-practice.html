Practical Exercises

Exercise 1: Basic Job Creation

Create a simple Jenkins job that prints "Hello, Jenkins!" to the console.

pipeline {
    agent any
    stages {
        stage('Hello') {
            steps {
                echo 'Hello, Jenkins!'
            }
        }
    }
}

Exercise 2: Building a Java Application

Set up a Jenkins pipeline to build a Java application using Maven.

pipeline {
    agent any
    tools {
        maven 'Maven 3.6.3' // Ensure Maven is installed and configured in Jenkins
    }
    stages {
        stage('Checkout') {
            steps {
                git 'https://github.com/your-repo/your-java-app.git'
            }
        }
        stage('Build') {
            steps {
                sh 'mvn clean install'
            }
        }
        stage('Test') {
            steps {
                sh 'mvn test'
            }
        }
    }
}

Exercise 3: Using Docker with Jenkins

Create a Jenkins pipeline that builds a Docker image and pushes it to Docker Hub.

pipeline {
    agent any
    environment {
        DOCKERHUB_CREDENTIALS = credentials('dockerhub-credentials-id')
    }
    stages {
        stage('Checkout') {
            steps {
                git 'https://github.com/your-repo/your-docker-app.git'
            }
        }
        stage('Build Docker Image') {
            steps {
                script {
                    dockerImage = docker.build("your-dockerhub-username/your-docker-app:${env.BUILD_ID}")
                }
            }
        }
        stage('Push Docker Image') {
            steps {
                script {
                    docker.withRegistry('https://index.docker.io/v1/', 'DOCKERHUB_CREDENTIALS') {
                        dockerImage.push()
                    }
                }
            }
        }
    }
}

Exercise 4: Setting Up a CI/CD Pipeline

Create a complete CI/CD pipeline that includes code checkout, build, test, and deployment stages.

pipeline {
    agent any
    tools {
        maven 'Maven 3.6.3'
    }
    stages {
        stage('Checkout') {
            steps {
                git 'https://github.com/your-repo/your-java-app.git'
            }
        }
        stage('Build') {
            steps {
                sh 'mvn clean install'
            }
        }
        stage('Test') {
            steps {
                sh 'mvn test'
            }
        }
        stage('Deploy') {
            steps {
                echo 'Deploying application...'
                // Add your deployment steps here
            }
        }
    }
}

Summary

In this section, we reviewed the essential concepts of Jenkins, including job creation, pipelines, plugins, and CI/CD practices. The practical exercises provided hands-on experience with creating Jenkins jobs and pipelines, integrating with version control systems, using Docker, and setting up a complete CI/CD pipeline. By completing these exercises, you should now have a solid understanding of how to use Jenkins to automate your software development processes effectively.
