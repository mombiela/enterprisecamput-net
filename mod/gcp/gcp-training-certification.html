Introduction to Google Cloud Platform (GCP)

Google Cloud Platform (GCP) is a suite of cloud computing services offered by Google. This course will take you from a beginner to an advanced level, covering all essential aspects of GCP.

What is GCP?

  • A collection of cloud services including computing, data storage, data analytics, and machine learning.
  • Provides infrastructure as a service (IaaS), platform as a service (PaaS), and serverless computing environments.

Key Concepts

  • Regions and Zones: Physical locations where resources are hosted.
  • Projects: Organizational units for managing resources.
  • Billing: Understanding how GCP charges for services.
  • IAM (Identity and Access Management): Managing permissions and access.

Core GCP Services

Compute Engine

  • Virtual Machines (VMs) on GCP.
  • Creating and managing VMs.
# Create a VM instance
gcloud compute instances create my-vm --zone=us-central1-a --machine-type=e2-medium

App Engine

  • Platform as a Service (PaaS) for building scalable web applications.
  • Deploying a simple web application.
# app.yaml configuration for Python App Engine
runtime: python38

# main.py for a simple web app
from flask import Flask
app = Flask(__name__)

@app.route('/')
def hello():
    return 'Hello, World!'

if __name__ == '__main__':
    app.run(host='0.0.0.0', port=8080)

Kubernetes Engine

  • Managed Kubernetes service.
  • Deploying a containerized application.
# Create a Kubernetes cluster
gcloud container clusters create my-cluster --zone us-central1-a

# Deploy an application
kubectl create deployment hello-server --image=gcr.io/google-samples/hello-app:1.0

# Expose the application
kubectl expose deployment hello-server --type=LoadBalancer --port 80 --target-port 8080

Data Storage and Databases

Cloud Storage

  • Object storage for unstructured data.
  • Creating and managing storage buckets.
# Create a storage bucket
gsutil mb gs://my-bucket

# Upload a file to the bucket
gsutil cp my-file.txt gs://my-bucket/

Cloud SQL

  • Managed relational database service.
  • Setting up a MySQL instance.
# Create a Cloud SQL instance
gcloud sql instances create my-instance --database-version=MYSQL_5_7 --tier=db-n1-standard-1 --region=us-central1

# Connect to the instance
gcloud sql connect my-instance --user=root
