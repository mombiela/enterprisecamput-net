Introduction

Flask is a micro web framework written in Python. It is designed to be simple and easy to use, making it an excellent choice for beginners who want to get started with web development. In this section, we will guide you through creating your first Flask app.

Setting Up Your Environment

Before we start coding, we need to set up our development environment.

Installing Flask

To install Flask, you need to have Python and pip (Python's package installer) installed on your system. You can install Flask using pip with the following command:

pip install Flask

Creating a Project Directory

Create a directory for your Flask project. This will help you keep your project files organized.

mkdir my_flask_app
cd my_flask_app

Writing Your First Flask App

Now that we have our environment set up, let's write our first Flask app.

Creating the Application File

Create a new Python file named app.py in your project directory. This file will contain the code for your Flask application.

# app.py
from flask import Flask

app = Flask(__name__)

@app.route('/')
def home():
    return "Hello, Flask!"

if __name__ == '__main__':
    app.run(debug=True)

Explanation of the Code

  • Importing Flask: We start by importing the Flask class from the flask module.
  • Creating an Instance of Flask: We create an instance of the Flask class. This instance will be our WSGI application.
  • Defining a Route: We use the @app.route('/') decorator to define a route for the root URL (/). When a user visits this URL, the home function will be called.
  • Returning a Response: The home function returns a simple string "Hello, Flask!" which will be displayed in the user's web browser.
  • Running the Application: Finally, we check if the script is run directly (not imported as a module) and call app.run(debug=True) to start the development server. The debug=True argument enables debug mode, which provides helpful error messages and auto-reloads the server when code changes.

Running Your Flask App

To run your Flask app, navigate to your project directory and execute the following command:

python app.py

You should see output indicating that the server is running. Open your web browser and go to http://127.0.0.1:5000/. You should see the message "Hello, Flask!".

Summary

In this section, we covered the basics of creating a simple Flask application. We set up our development environment, wrote a basic Flask app, and ran it on a local server. This foundational knowledge will serve as the building block for more advanced Flask topics.
