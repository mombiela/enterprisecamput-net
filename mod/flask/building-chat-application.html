Building a Chat Application with Flask
Security Best Practices ►

Introduction

In this topic, we will guide you through building a chat application using Flask, a popular micro web framework for Python. This course will cover everything from setting up your development environment to deploying your chat application. By the end of this course, you will have a solid understanding of how to build and deploy a real-time chat application.

Prerequisites

Before starting this course, you should have:

  • Basic knowledge of Python programming.
  • Familiarity with HTML, CSS, and JavaScript.
  • Basic understanding of web development concepts.

Setting Up the Development Environment

To start building your chat application, you need to set up your development environment.

Installing Flask

First, ensure you have Python installed. Then, install Flask using pip:

pip install Flask

Creating the Project Structure

Create a new directory for your project and set up the following structure:

chat-app/
│
├── app.py
├── templates/
│   └── index.html
├── static/
│   ├── css/
│   │   └── styles.css
│   └── js/
│       └── main.js
└── requirements.txt

Building the Flask Application

Setting Up the Flask Server

Create the app.py file and set up a basic Flask server:

from flask import Flask, render_template

app = Flask(__name__)

@app.route('/')
def index():
    return render_template('index.html')

if __name__ == '__main__':
    app.run(debug=True)

Creating the HTML Template

Create the index.html file in the templates directory:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Chat Application</title>
    <link rel="stylesheet" href="{{ url_for('static', filename='css/styles.css') }}">
</head>
<body>
    <div id="chat-container">
        <div id="messages"></div>
        <input type="text" id="message-input" placeholder="Type a message...">
        <button id="send-button">Send</button>
    </div>
    <script src="{{ url_for('static', filename='js/main.js') }}"></script>
</body>
</html>

Adding Styles

Create the styles.css file in the static/css directory:

#chat-container {
    width: 300px;

Implementing Real-Time Communication

To enable real-time communication, we will use Flask-SocketIO.

Installing Flask-SocketIO

Install Flask-SocketIO using pip:

pip install flask-socketio

Setting Up Flask-SocketIO

Update your app.py to include Flask-SocketIO:

from flask import Flask, render_template
from flask_socketio import SocketIO, send

app = Flask(__name__)
socketio = SocketIO(app)

@app.route('/')
def index():
    return render_template('index.html')

@socketio.on('message')
def handle_message(msg):
    send(msg, broadcast=True)

if __name__ == '__main__':
    socketio.run(app, debug=True)

Updating the JavaScript

Create the main.js file in the static/js directory:

document.addEventListener('DOMContentLoaded', () => {
    const socket = io();

    const messageInput = document.getElementById('message-input');
    const sendButton = document.getElementById('send-button');
    const messages = document.getElementById('messages');

    sendButton.addEventListener('click', () => {
        const message = messageInput.value;
        socket.send(message);
        messageInput.value = '';
    });

    socket.on('message', (msg) => {
        const messageElement = document.createElement('div');
        messageElement.textContent = msg;
        messages.appendChild(messageElement);
    });
});

Enhancing the Chat Application

Adding Usernames

To make the chat more interactive, we can add usernames.

Updating the HTML Template

Update the index.html file to include a username input:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Chat Application</title>
    <link rel="stylesheet" href="{{ url_for('static', filename='css/styles.css') }}">
</head>
<body>
    <div id="chat-container">
        <input type="text" id="username-input" placeholder="Enter your username">
        <div id="messages"></div>
        <input type="text" id="message-input" placeholder="Type a message...">
        <button id="send-button">Send</button>
    </div>
    <script src="{{ url_for('static', filename='js/main.js') }}"></script>
</body>
</html>

Updating the JavaScript

Update the main.js file to include the username:

document.addEventListener('DOMContentLoaded', () => {
    const socket = io();

    const usernameInput = document.getElementById('username-input');
    const messageInput = document.getElementById('message-input');
    const sendButton = document.getElementById('send-button');
    const messages = document.getElementById('messages');

    sendButton.addEventListener('click', () => {
        const message = messageInput.value;
        const username = usernameInput.value;
        socket.send({ username, message });
        messageInput.value = '';
    });

    socket.on('message', (data) => {
        const messageElement = document.createElement('div');
        messageElement.textContent = `${data.username}: ${data.message}`;
        messages.appendChild(messageElement);
    });
});

Updating the Flask Server

Update the app.py to handle the new message format:

from flask import Flask, render_template
from flask_socketio import SocketIO, send

app = Flask(__name__)
socketio = SocketIO(app)

@app.route('/')
def index():
    return render_template('index.html')

@socketio.on('message')
def handle_message(data):
    send(data, broadcast=True)

if __name__ == '__main__':
    socketio.run(app, debug=True)

Deploying the Chat Application

Preparing for Deployment

To deploy your application, you need to:

  • Freeze your dependencies.
  • Configure your application for production.

Freezing Dependencies

Create a requirements.txt file:

pip freeze > requirements.txt

Configuring for Production

Update your app.py to disable debug mode for production:

if __name__ == '__main__':
    socketio.run(app)

Deploying to Heroku

  1. Install the Heroku CLI.
  2. Create a Procfile in your project directory:
web: gunicorn -k eventlet -w 1 app:app
  1. Initialize a git repository and commit your code:
git init
git add .
git commit -m "Initial commit"
  1. Create a new Heroku application and push your code:
heroku create
git push heroku master

Conclusion

In this topic, we have covered how to build a real-time chat application using Flask and Flask-SocketIO. We started with setting up the development environment, building the basic Flask application, implementing real-time communication, enhancing the chat application with usernames, and finally deploying the application to Heroku. By following these steps, you should now have a functional chat application and a deeper understanding of Flask and real-time web applications.
