Conditional Rendering in Vue.js
List Rendering ►

Conditional rendering is a fundamental concept in Vue.js that allows you to dynamically control the visibility and rendering of elements in the DOM based on certain conditions. This topic will cover the basics to advanced techniques of conditional rendering in Vue.js.

Basics of Conditional Rendering

v-if Directive

The v-if directive is used to conditionally render a block. The block will only be rendered if the directive's expression returns a truthy value.

<template>
  <div v-if="isVisible">
    This content is visible if isVisible is true.
  </div>
</template>

<script>
export default {
  data() {
    return {
      isVisible: true
    };
  }
};
</script>

v-else Directive

The v-else directive can be used to provide an "else" block for v-if.

<template>
  <div>
    <div v-if="isVisible">
      This content is visible if isVisible is true.
    </div>
    <div v-else>
      This content is visible if isVisible is false.
    </div>
  </div>
</template>

<script>
export default {
  data() {
    return {
      isVisible: false
    };
  }
};
</script>

v-else-if Directive

The v-else-if directive can be used to chain multiple conditions.

<template>
  <div>
    <div v-if="status === 'success'">
      Operation was successful!
    </div>
    <div v-else-if="status === 'error'">
      There was an error.
    </div>
    <div v-else>
      Status is unknown.
    </div>
  </div>
</template>

<script>
export default {
  data() {
    return {
      status: 'error'
    };
  }
};
</script>

Advanced Conditional Rendering

v-show Directive

The v-show directive toggles the visibility of an element by changing its display CSS property. Unlike v-if, the element is always rendered and remains in the DOM.

<template>
  <div v-show="isVisible">
    This content is always in the DOM but only visible if isVisible is true.
  </div>
</template>

<script>
export default {
  data() {
    return {
      isVisible: true
    };
  }
};
</script>

v-if vs v-show

| Feature | v-if | v-show | |---------------|-------------------------------|-------------------------------| | Rendering | Conditionally renders/removes | Always renders, toggles display | | Performance | Higher cost for toggling | Lower cost for toggling | | Use Case | Rarely toggled elements | Frequently toggled elements |

Conditional Groups

You can use a <template> element with v-if, v-else-if, and v-else to conditionally render multiple elements.

<template>
  <template v-if="status === 'success'">
    <p>Operation was successful!</p>
    <button @click="retry">Retry</button>
  </template>
  <template v-else-if="status === 'error'">
    <p>There was an error.</p>
    <button @click="retry">Retry</button>
  </template>
  <template v-else>
    <p>Status is unknown.</p>
  </template>
</template>

<script>
export default {
  data() {
    return {
      status: 'unknown'
    };
  },
  methods: {
    retry() {
      // Retry logic here
    }
  }
};
</script>

Conclusion

Conditional rendering in Vue.js provides powerful tools to control the visibility and rendering of elements based on dynamic conditions. By understanding and utilizing v-if, v-else, v-else-if, and v-show, you can create more interactive and responsive user interfaces. Remember to choose between v-if and v-show based on the performance implications and the frequency of toggling the element's visibility.
