What is Axios?

Axios is a popular JavaScript library used to make HTTP requests from the browser and Node.js. It is promise-based, which allows for cleaner and more manageable asynchronous code. In the context of Vue.js, Axios is commonly used to fetch data from APIs and handle HTTP requests in a Vue application.

Key Features of Axios

  • Promise-based: Simplifies asynchronous code with async/await or .then() and .catch().
  • Interceptors: Allows you to intercept requests or responses before they are handled.
  • Automatic JSON Data Transformation: Automatically transforms JSON data for you.
  • Error Handling: Provides a simple way to handle errors.
  • Request Cancellation: Supports canceling requests using the CancelToken API.

Installing Axios

To use Axios in a Vue.js project, you first need to install it. You can do this using npm or yarn.

npm install axios
# or
yarn add axios

Using Axios in a Vue Component

Basic GET Request

Here's how you can make a basic GET request to fetch data from an API and display it in a Vue component.

<template>
  <div>
    <h1>Fetched Data</h1>
    <ul>
      <li v-for="item in items" :key="item.id">{{ item.name }}</li>
    </ul>
  </div>
</template>

<script>
import axios from 'axios';

export default {
  data() {
    return {
      items: []
    };
  },
  created() {
    axios.get('https://api.example.com/items')
      .then(response => {
        this.items = response.data;
      })
      .catch(error => {
        console.error('There was an error!', error);
      });
  }
};
</script>

POST Request

To send data to an API, you can use a POST request. Here's an example of how to do that in a Vue component.

<template>
  <div>
    <h1>Add New Item</h1>
    <form @submit.prevent="addItem">
      <input v-model="newItem" placeholder="Enter item name" />
      <button type="submit">Add Item</button>
    </form>
  </div>
</template>

<script>
import axios from 'axios';

export default {
  data() {
    return {
      newItem: ''
    };
  },
  methods: {
    addItem() {
      axios.post('https://api.example.com/items', { name: this.newItem })
        .then(response => {
          console.log('Item added:', response.data);
          this.newItem = ''; // Clear the input field
        })
        .catch(error => {
          console.error('There was an error!', error);
        });
    }
  }
};
</script>

Handling Errors

Error handling in Axios is straightforward. You can use .catch() to handle errors in your requests.

axios.get('https://api.example.com/items')
  .then(response => {
    this.items = response.data;
  })
  .catch(error => {
    console.error('There was an error!', error);
  });

Using Axios Interceptors

Interceptors allow you to run your code or modify the request/response before it is handled by then or catch.

Request Interceptor

axios.interceptors.request.use(config => {
  // Do something before request is sent
  console.log('Request Interceptor:', config);
  return config;
}, error => {
  // Do something with request error
  return Promise.reject(error);
});

Response Interceptor

axios.interceptors.response.use(response => {
  // Do something with response data
  console.log('Response Interceptor:', response);
  return response;
}, error => {
  // Do something with response error
  return Promise.reject(error);
});

Canceling Requests

You can cancel an Axios request using the CancelToken API.

const CancelToken = axios.CancelToken;
let cancel;

axios.get('https://api.example.com/items', {
  cancelToken: new CancelToken(function executor(c) {
    cancel = c;
  })
}).catch(thrown => {
  if (axios.isCancel(thrown)) {
    console.log('Request canceled', thrown.message);
  } else {
    // handle error
  }
});

// Cancel the request
cancel('Operation canceled by the user.');

Conclusion

Axios is a powerful and flexible library for handling HTTP requests in Vue.js applications. It simplifies the process of making asynchronous requests, handling errors, and managing request/response transformations. By understanding and utilizing Axios, you can efficiently interact with APIs and enhance the functionality of your Vue.js applications.
