Introduction

Control structures are fundamental in programming as they allow you to control the flow of execution based on conditions and loops. In this section, we will explore the control structures in MUMPS (M), including IF, ELSE, FOR, and WHILE.

IF and ELSE Statements

The IF statement in MUMPS is used to execute a block of code only if a specified condition is true. The ELSE statement can be used to execute an alternative block of code if the condition is false.

Syntax

IF condition DO
. ; code to execute if condition is true
ELSE  DO
. ; code to execute if condition is false

Example

SET age=20
IF age<18 DO
. WRITE "You are a minor.",!
ELSE  DO
. WRITE "You are an adult.",!
  • Explanation: In this example, the program checks if the age is less than 18. If true, it prints "You are a minor." Otherwise, it prints "You are an adult."

FOR Loop

The FOR loop in MUMPS is used to repeat a block of code a specific number of times.

Syntax

FOR variable=start:step:end DO
. ; code to execute in each iteration

Example

FOR i=1:1:5 DO
. WRITE "Iteration: ",i,!
  • Explanation: This loop will iterate from 1 to 5, incrementing i by 1 each time, and print the current iteration number.

WHILE Loop

The WHILE loop in MUMPS is used to repeat a block of code as long as a specified condition is true.

Syntax

WHILE condition DO
. ; code to execute while condition is true

Example

SET count=1
WHILE count<=5 DO
. WRITE "Count: ",count,!
. SET count=count+1
  • Explanation: This loop will continue to execute as long as count is less than or equal to 5. It prints the current count and then increments count by 1.

Combining Control Structures

You can combine IF, ELSE, FOR, and WHILE to create more complex control flows.

Example

SET num=1
FOR i=1:1:10 DO
. IF num#2=0 DO
. . WRITE "Even number: ",num,!
. ELSE  DO
. . WRITE "Odd number: ",num,!
. SET num=num+1
  • Explanation: This example uses a FOR loop to iterate 10 times. Inside the loop, it uses an IF statement to check if num is even or odd and prints the appropriate message. The num is then incremented by 1.

Conclusion

Understanding and effectively using control structures like IF, ELSE, FOR, and WHILE is crucial for controlling the flow of your MUMPS programs. These constructs allow you to make decisions, repeat tasks, and create complex logic in your code. Practice using these structures in different scenarios to become proficient in MUMPS programming.
