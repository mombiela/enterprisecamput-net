Introduction

In this section, we will explore the various operators and expressions used in MUMPS (M). Understanding these fundamental concepts is crucial for writing effective and efficient MUMPS code. We will cover arithmetic, logical, and relational operators, as well as how to construct and evaluate expressions.

Arithmetic Operators

Arithmetic operators are used to perform mathematical calculations. The common arithmetic operators in MUMPS include:

  • + : Addition
  • - : Subtraction
  • * : Multiplication
  • / : Division
  • # : Integer Division
  • ** : Exponentiation

Examples

SET A=10, B=3
WRITE "Addition: ", A+B, !
WRITE "Subtraction: ", A-B, !
WRITE "Multiplication: ", A*B, !
WRITE "Division: ", A/B, !
WRITE "Integer Division: ", A#B, !
WRITE "Exponentiation: ", A**B, !

Output:

Addition: 13
Subtraction: 7
Multiplication: 30
Division: 3.333333
Integer Division: 3
Exponentiation: 1000

Logical Operators

Logical operators are used to perform logical operations. The common logical operators in MUMPS include:

  • & : Logical AND
  • ! : Logical OR
  • ' : Logical NOT

Examples

SET X=1, Y=0
WRITE "Logical AND: ", X&Y, !
WRITE "Logical OR: ", X!Y, !
WRITE "Logical NOT: ", 'X, !

Output:

Logical AND: 0
Logical OR: 1
Logical NOT: 0

Relational Operators

Relational operators are used to compare two values. The common relational operators in MUMPS include:

  • = : Equal to
  • `'= : Not equal to
  • < : Less than
  • > : Greater than
  • <= : Less than or equal to
  • >= : Greater than or equal to

Examples

SET A=5, B=10
WRITE "Equal to: ", A=B, !
WRITE "Not equal to: ", A'=B, !
WRITE "Less than: ", A<B, !
WRITE "Greater than: ", A>B, !
WRITE "Less than or equal to: ", A<=B, !
WRITE "Greater than or equal to: ", A>=B, !

Output:

Equal to: 0
Not equal to: 1
Less than: 1
Greater than: 0
Less than or equal to: 1
Greater than or equal to: 0

Combining Operators in Expressions

Operators can be combined to form complex expressions. The order of operations (precedence) in MUMPS follows standard mathematical rules:

  1. Parentheses ()
  2. Exponentiation **
  3. Multiplication * and Division /
  4. Addition + and Subtraction -
  5. Relational operators
  6. Logical operators

Examples

SET A=5, B=10, C=2
WRITE "Expression 1: ", (A+B)*C, !
WRITE "Expression 2: ", A+B*C, !
WRITE "Expression 3: ", A>B&(B>C), !
WRITE "Expression 4: ", (A<B)!('C), !

Output:

Expression 1: 30
Expression 2: 25
Expression 3: 0
Expression 4: 1

Conclusion

Understanding and effectively using operators and expressions is essential for programming in MUMPS. This section covered the basic arithmetic, logical, and relational operators, and how to combine them to form complex expressions. Practice using these operators in different scenarios to become proficient in writing MUMPS code.
