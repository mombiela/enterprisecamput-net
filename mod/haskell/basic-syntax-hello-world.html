Introduction

Welcome to the first module of our Haskell course! In this section, we will cover the basic syntax of Haskell and guide you through writing your first "Hello World" program. By the end of this section, you will have a foundational understanding of Haskell's syntax and how to run a simple program.

Key Concepts

  • Haskell Basics: Understanding the structure and syntax of Haskell code.
  • Comments: How to write comments in Haskell.
  • Hello World Program: Writing and running your first Haskell program.

Haskell Basics

Haskell is a purely functional programming language with a strong static type system. Here are some fundamental aspects of Haskell's syntax:

  • Modules: Haskell code is organized into modules.
  • Functions: Functions are first-class citizens in Haskell.
  • Types: Haskell has a strong, static type system.
  • Expressions: Everything in Haskell is an expression.

Example: Basic Structure

-- This is a single-line comment

{-
This is a multi-line comment
-}

module Main where

-- Define a function
add :: Int -> Int -> Int
add x y = x + y

-- Main function
main :: IO ()
main = do
    let result = add 5 3
    putStrLn ("The result is " ++ show result)

Explanation

  • Comments: Single-line comments start with --, and multi-line comments are enclosed in {- -}.
  • Module Declaration: module Main where declares a module named Main.
  • Function Definition: add :: Int -> Int -> Int defines a function add that takes two integers and returns an integer.
  • Main Function: main :: IO () is the entry point of the program. The do notation is used for sequencing IO actions.

Writing Your First "Hello World" Program

Let's write a simple "Hello World" program in Haskell.

Example: Hello World

module Main where

main :: IO ()
main = putStrLn "Hello, World!"

Explanation

  • Module Declaration: module Main where declares the main module.
  • Main Function: main :: IO () is the main function that performs IO actions.
  • Output: putStrLn "Hello, World!" prints "Hello, World!" to the console.

Running the Program

To run your Haskell program, follow these steps:

  1. Save the code in a file named Main.hs.
  2. Open a terminal and navigate to the directory containing Main.hs.
  3. Compile the program using the GHC compiler: 
    ghc -o hello Main.hs
  4. Run the compiled program: 
    ./hello

Conclusion

In this section, we covered the basic syntax of Haskell and wrote a simple "Hello World" program. You learned about Haskell's module system, function definitions, and how to perform basic IO operations. This foundational knowledge will be essential as we move on to more advanced topics in Haskell. Keep practicing, and soon you'll be comfortable with Haskell's syntax and functional programming paradigm!
