Introduction

In this section, we will embark on a journey to build a complete Haskell application from scratch. This project will help you consolidate your Haskell knowledge and give you practical experience in developing a real-world application.

Project Overview

  • Objective: Build a simple command-line to-do list application.
  • Features:
    • Add a new task.
    • List all tasks.
    • Mark a task as completed.
    • Delete a task.
  • Tools: GHC (Glasgow Haskell Compiler), Stack (Haskell build tool).

Setting Up the Environment

Before we start coding, let's set up our development environment.

Installing GHC and Stack

  1. GHC: The Glasgow Haskell Compiler is the most widely used Haskell compiler.
  2. Stack: A cross-platform program for developing Haskell projects.

Installation Steps

  • Windows: 
    choco install haskell-stack
  • macOS: 
    brew install haskell-stack
  • Linux: 
    curl -sSL https://get.haskellstack.org/ | sh

Creating a New Project

Use Stack to create a new Haskell project:

stack new todo-app simple
cd todo-app
stack setup

Designing the Application

Let's outline the structure of our application.

Data Model

We'll define a Task data type to represent each to-do item.

data Task = Task {
    taskId :: Int,
    taskDescription :: String,
    taskCompleted :: Bool
} deriving (Show, Read)

Main Functions

  • Add Task: Add a new task to the list.
  • List Tasks: Display all tasks.
  • Complete Task: Mark a task as completed.
  • Delete Task: Remove a task from the list.

Implementing the Application

Main Module

Create a Main.hs file in the src directory.

Importing Necessary Modules

import System.IO
import Data.List
import Control.Monad

Main Function

main :: IO ()
main = do
    putStrLn "Welcome to the Haskell To-Do List Application!"
    mainMenu

Main Menu

mainMenu :: IO ()
mainMenu = do
    putStrLn "1. Add Task"
    putStrLn "2. List Tasks"
    putStrLn "3. Complete Task"
    putStrLn "4. Delete Task"
    putStrLn "5. Exit"
    putStr "Choose an option: "
    hFlush stdout
    option <- getLine
    handleOption option

Handling User Input

handleOption :: String -> IO ()
handleOption option = case option of
    "1" -> addTask
    "2" -> listTasks
    "3" -> completeTask
    "4" -> deleteTask
    "5" -> putStrLn "Goodbye!"
    _   -> putStrLn "Invalid option!" >> mainMenu

Adding a Task

addTask :: IO ()
addTask = do
    putStr "Enter task description: "
    hFlush stdout
    description <- getLine
    -- Here you would normally save the task to a file or database
    putStrLn $ "Task added: " ++ description
    mainMenu

Listing Tasks

listTasks :: IO ()

    -- Here you would normally read the tasks from a file or database

Completing a Task

completeTask :: IO ()
completeTask = do
    putStr "Enter task ID to complete: "
    hFlush stdout
    taskId <- getLine
    -- Here you would normally update the task in a file or database
    putStrLn $ "Task " ++ taskId ++ " marked as completed."

deleteTask = do
    putStr "Enter task ID to delete: "

    taskId <- getLine
    -- Here you would normally delete the task from a file or database
    putStrLn $ "Task " ++ taskId ++ " deleted."
