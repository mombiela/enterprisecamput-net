Introduction

Haskell, a statically typed, purely functional programming language, offers a variety of basic data types that are essential for any Haskell programmer to understand. This section will cover the fundamental data types in Haskell, providing clear explanations, examples, and exercises to solidify your understanding.

Key Data Types

Integers

  • Int: A fixed-precision integer type.
  • Integer: An arbitrary-precision integer type.
-- Example of Int
let x :: Int
x = 42

-- Example of Integer
let y :: Integer
y = 12345678901234567890

Floating-Point Numbers

  • Float: A single-precision floating-point number.
  • Double: A double-precision floating-point number.
-- Example of Float
let a :: Float
a = 3.14

-- Example of Double
let b :: Double
b = 2.718281828459045

Characters and Strings

  • Char: Represents a single character.
  • String: A list of characters, essentially [Char].
-- Example of Char
let c :: Char
c = 'H'

-- Example of String
let s :: String
s = "Hello, Haskell!"

Boolean

  • Bool: Represents a boolean value, which can be either True or False.
-- Example of Bool
let isHaskellFun :: Bool
isHaskellFun = True

Tuples

  • Tuples are fixed-size collections of elements, which can be of different types.
-- Example of a tuple
let person :: (String, Int)
person = ("Alice", 30)

Lists

  • Lists are collections of elements of the same type.
-- Example of a list
let numbers :: [Int]
numbers = [1, 2, 3, 4, 5]

Comparison of Data Types

| Data Type | Description | Example | |-----------|-------------|---------| | Int | Fixed-precision integer | 42 | | Integer | Arbitrary-precision integer | 12345678901234567890 | | Float | Single-precision floating-point | 3.14 | | Double | Double-precision floating-point | 2.718281828459045 | | Char | Single character | 'H' | | String | List of characters | "Hello, Haskell!" | | Bool | Boolean value | True | | Tuple | Fixed-size collection of elements | ("Alice", 30) | | List | Collection of elements of the same type | [1, 2, 3, 4, 5] |

Exercises

  1. Define Variables:

    • Define an Int variable with the value 10.
    • Define a Double variable with the value 3.14159.
    • Define a String variable with the value "Haskell".

  2. Create Tuples and Lists:

    • Create a tuple representing a book with a title (String) and the number of pages (Int).
    • Create a list of Bool values representing the results of some conditions: [True, False, True].

  3. Manipulate Data:

    • Write a function that takes a list of integers and returns the sum of the elements.
    • Write a function that takes a tuple of two integers and returns their product.
-- Exercise 1
let myInt :: Int
myInt = 10

let myDouble :: Double
myDouble = 3.14159

let myString :: String
myString = "Haskell"

-- Exercise 2
let book :: (String, Int)
book = ("Learn You a Haskell", 400)

let results :: [Bool]
results = [True, False, True]

-- Exercise 3
sumList :: [Int] -> Int
sumList xs = sum xs

productTuple :: (Int, Int) -> Int
productTuple (a, b) = a * b

Conclusion

Understanding Haskell's basic data types is crucial for writing effective and efficient code. This section has introduced you to the primary data types, including integers, floating-point numbers, characters, strings, booleans, tuples, and lists. By practicing the provided exercises, you will gain a solid foundation in handling these data types in Haskell.
