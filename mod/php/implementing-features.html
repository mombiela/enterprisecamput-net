Introduction

In this section, we will explore how to implement various features in PHP, ranging from basic to advanced levels. By the end of this section, you will have a solid understanding of how to create functional and dynamic PHP applications.

Basic Features

1. Handling Forms

Handling forms is a fundamental aspect of web development. PHP makes it easy to collect, validate, and process form data.

Example: Simple Contact Form

<!DOCTYPE html>
<html>
<head>
    <title>Contact Form</title>
</head>
<body>
    <form method="post" action="process_form.php">
        Name: <input type="text" name="name"><br>
        Email: <input type="email" name="email"><br>
        Message: <textarea name="message"></textarea><br>
        <input type="submit" value="Submit">
    </form>
</body>
</html>

Processing the Form Data

<?php
// process_form.php
if ($_SERVER["REQUEST_METHOD"] == "POST") {
    $name = htmlspecialchars($_POST['name']);
    $email = htmlspecialchars($_POST['email']);
    $message = htmlspecialchars($_POST['message']);

    // Basic validation
    if (!empty($name) && !empty($email) && !empty($message)) {
        echo "Thank you, $name. We have received your message.";
    } else {
        echo "All fields are required.";
    }
}
?>

2. File Uploads

Uploading files is another common feature in web applications. PHP provides built-in functions to handle file uploads securely.

Example: File Upload Form

<!DOCTYPE html>
<html>
<head>
    <title>File Upload</title>
</head>
<body>
    <form method="post" enctype="multipart/form-data" action="upload.php">
        Select file: <input type="file" name="fileToUpload"><br>
        <input type="submit" value="Upload File">
    </form>
</body>
</html>

Processing the File Upload

<?php
// upload.php
$target_dir = "uploads/";
$target_file = $target_dir . basename($_FILES["fileToUpload"]["name"]);
$uploadOk = 1;
$fileType = strtolower(pathinfo($target_file, PATHINFO_EXTENSION));

// Check if file already exists
if (file_exists($target_file)) {
    echo "Sorry, file already exists.";
    $uploadOk = 0;
}

// Check file size
if ($_FILES["fileToUpload"]["size"] > 500000) {
    echo "Sorry, your file is too large.";
    $uploadOk = 0;
}

// Allow certain file formats
if ($fileType != "jpg" && $fileType != "png" && $fileType != "jpeg" && $fileType != "gif") {
    echo "Sorry, only JPG, JPEG, PNG & GIF files are allowed.";
    $uploadOk = 0;
}

// Check if $uploadOk is set to 0 by an error
if ($uploadOk == 0) {
    echo "Sorry, your file was not uploaded.";
} else {
    if (move_uploaded_file($_FILES["fileToUpload"]["tmp_name"], $target_file)) {
        echo "The file ". htmlspecialchars(basename($_FILES["fileToUpload"]["name"])). " has been uploaded.";
    } else {
        echo "Sorry, there was an error uploading your file.";
    }
}
?>

Intermediate Features

1. User Authentication

User authentication is crucial for securing web applications. PHP can be used to create a simple login system.

Example: Login Form

<!DOCTYPE html>
<html>
<head>
    <title>Login</title>
</head>
<body>
    <form method="post" action="login.php">
        Username: <input type="text" name="username"><br>
        Password: <input type="password" name="password"><br>
        <input type="submit" value="Login">
    </form>
</body>
</html>

Processing the Login

<?php
// login.php
session_start();
$users = [
    'admin' => 'password123',
    'user' => 'userpass'
];

if ($_SERVER["REQUEST_METHOD"] == "POST") {
    $username = $_POST['username'];
    $password = $_POST['password'];

    if (isset($users[$username]) && $users[$username] == $password) {
        $_SESSION['username'] = $username;
        echo "Welcome, $username!";
    } else {
        echo "Invalid username or password.";
    }
}
?>

2. Database Interaction

Interacting with a database is essential for dynamic web applications. PHP provides several ways to connect and interact with databases, such as MySQL.

Example: Connecting to a MySQL Database

<?php
$servername = "localhost";
$username = "username";
$password = "password";
$dbname = "myDB";

// Create connection
$conn = new mysqli($servername, $username, $password, $dbname);

// Check connection
if ($conn->connect_error) {
    die("Connection failed: " . $conn->connect_error);
}
echo "Connected successfully";
?>

Example: Inserting Data into a Database

<?php
// Assuming connection is established as shown above
$sql = "INSERT INTO Users (username, email, password) VALUES ('john_doe', 'john@example.com', 'securepassword')";

if ($conn->query($sql) === TRUE) {
    echo "New record created successfully";
} else {
    echo "Error: " . $sql . "<br>" . $conn->error;
}

$conn->close();
?>

Advanced Features

1. Implementing RESTful APIs

RESTful APIs allow different applications to communicate with each other. PHP can be used to create and consume RESTful APIs.

Example: Simple RESTful API

<?php
header("Content-Type: application/json");

$method = $_SERVER['REQUEST_METHOD'];

switch ($method) {
    case 'GET':
        echo json_encode(["message" => "GET request received"]);
        break;
    case 'POST':
        echo json_encode(["message" => "POST request received"]);
        break;
    default:
        echo json_encode(["message" => "Request method not supported"]);
        break;
}
?>

2. Using Frameworks

PHP frameworks like Laravel and Symfony can significantly speed up development and enforce best practices.

Example: Basic Laravel Route

// routes/web.php
Route::get('/', function () {
    return view('welcome');
});

Example: Basic Controller in Laravel

// app/Http/Controllers/ExampleController.php
namespace App\Http\Controllers;

use Illuminate\Http\Request;

class ExampleController extends Controller
{
    public function index()
    {
        return view('example');
    }
}

Conclusion

Implementing features in PHP ranges from handling simple forms to creating complex RESTful APIs and using frameworks. By mastering these features, you can build robust and dynamic web applications. Practice each example and experiment with variations to deepen your understanding.
