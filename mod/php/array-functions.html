Introduction to Arrays in PHP

Arrays are one of the most powerful data types in PHP. They allow you to store multiple values in a single variable, which can be accessed using indices or keys.

Key Concepts

  • Indexed Arrays: Arrays with numeric indices.
  • Associative Arrays: Arrays with named keys.
  • Multidimensional Arrays: Arrays containing one or more arrays.

Example

// Indexed Array
$fruits = array("Apple", "Banana", "Cherry");

// Associative Array
$ages = array("Peter" => 35, "Ben" => 37, "Joe" => 43);

// Multidimensional Array
$contacts = array(
    array("name" => "Peter", "email" => "peter@example.com"),
    array("name" => "Ben", "email" => "ben@example.com")
);

Basic Array Functions

`count()`

The count() function returns the number of elements in an array.

$fruits = array("Apple", "Banana", "Cherry");
echo count($fruits); // Output: 3

`array_push()`

The array_push() function adds one or more elements to the end of an array.

$fruits = array("Apple", "Banana");
array_push($fruits, "Cherry", "Date");
print_r($fruits); // Output: Array ( [0] => Apple [1] => Banana [2] => Cherry [3] => Date )

`array_pop()`

The array_pop() function removes the last element from an array.

$fruits = array("Apple", "Banana", "Cherry");
array_pop($fruits);
print_r($fruits); // Output: Array ( [0] => Apple [1] => Banana )

Intermediate Array Functions

`array_merge()`

The array_merge() function merges one or more arrays into one array.

$array1 = array("color" => "red", 2, 4);
$array2 = array("a", "b", "color" => "green", "shape" => "trapezoid", 4);
$result = array_merge($array1, $array2);
print_r($result);
// Output: Array ( [color] => green [0] => 2 [1] => 4 [2] => a [3] => b [shape] => trapezoid [4] => 4 )

`array_diff()`

The array_diff() function compares arrays and returns the differences.

$array1 = array("a" => "green", "red", "blue", "red");
$array2 = array("b" => "green", "yellow", "red");
$result = array_diff($array1, $array2);
print_r($result); // Output: Array ( [1] => blue )

`array_keys()`

The array_keys() function returns all the keys of an array.

$array = array("color" => "blue", "size" => "medium", "shape" => "circle");
$keys = array_keys($array);
print_r($keys); // Output: Array ( [0] => color [1] => size [2] => shape )

Advanced Array Functions

`array_map()`

The array_map() function applies a callback to the elements of the given arrays.

function square($n) {
    return $n * $n;
}
$numbers = array(1, 2, 3, 4, 5);
$squaredNumbers = array_map("square", $numbers);
print_r($squaredNumbers); // Output: Array ( [0] => 1 [1] => 4 [2] => 9 [3] => 16 [4] => 25 )

`array_filter()`

The array_filter() function filters elements of an array using a callback function.

function isEven($n) {
    return $n % 2 == 0;
}
$numbers = array(1, 2, 3, 4, 5, 6);
$evenNumbers = array_filter($numbers, "isEven");
print_r($evenNumbers); // Output: Array ( [1] => 2 [3] => 4 [5] => 6 )

`array_reduce()`

The array_reduce() function reduces an array to a single value using a callback function.

function sum($carry, $item) {
    $carry += $item;
    return $carry;
}
$numbers = array(1, 2, 3, 4, 5);
$total = array_reduce($numbers, "sum");
echo $total; // Output: 15

Conclusion

PHP provides a rich set of functions to manipulate arrays, from basic operations like counting elements and adding/removing items, to more advanced operations like mapping, filtering, and reducing arrays. Understanding these functions will significantly enhance your ability to work with arrays effectively in PHP. Practice using these functions with different types of arrays to become proficient in handling array data structures.
