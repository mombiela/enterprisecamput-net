Introduction to Laravel

Laravel is a powerful and flexible PHP framework designed to make web development easier and more enjoyable. It follows the Model-View-Controller (MVC) architectural pattern and provides a rich set of features for building modern web applications.

Key Concepts

  • MVC Architecture: Separates the application logic into three main components: Model, View, and Controller.
  • Routing: Defines how the application responds to various HTTP requests.
  • Eloquent ORM: Provides a simple and elegant way to interact with the database.
  • Blade Templating Engine: A powerful templating engine for creating dynamic views.
  • Artisan CLI: A command-line interface that provides numerous helpful commands for application development.

Setting Up Laravel

To get started with Laravel, you need to set up your development environment.

Prerequisites

  • PHP >= 7.3
  • Composer (Dependency Manager for PHP)
  • A web server like Apache or Nginx
  • A database like MySQL or SQLite

Installation Steps

  1. Install Composer: If you haven't installed Composer yet, download and install it from getcomposer.org.
  2. Create a New Laravel Project: 
    composer create-project --prefer-dist laravel/laravel myLaravelApp
  3. Navigate to the Project Directory: 
    cd myLaravelApp
  4. Serve the Application: 
    php artisan serve
    This command will start a local development server at http://localhost:8000.

Laravel Directory Structure

Understanding the directory structure is crucial for efficient development.

  • app/: Contains the core code of your application.
  • bootstrap/: Contains the application bootstrapping script.
  • config/: Contains configuration files.
  • database/: Contains database migrations, model factories, and seeds.
  • public/: The entry point for the application, contains the index.php file.
  • resources/: Contains views, raw assets (CSS, JS), and language files.
  • routes/: Contains all route definitions.
  • storage/: Contains compiled Blade templates, file-based sessions, file caches, and other files generated by the framework.
  • tests/: Contains automated tests.
  • vendor/: Contains Composer dependencies.

Basic Routing

Routing in Laravel is straightforward and powerful. Routes are defined in the routes/web.php file.

Example Route

Route::get('/', function () {
    return view('welcome');
});

This route listens for GET requests to the root URL (/) and returns the welcome view.

Route Parameters

Route::get('/user/{id}', function ($id) {
    return 'User '.$id;
});

This route captures a parameter ({id}) from the URL and passes it to the closure.

Controllers

Controllers are responsible for handling incoming requests and returning responses.

Creating a Controller

Use the Artisan CLI to create a controller:

php artisan make:controller UserController

This command creates a new controller file in the app/Http/Controllers directory.

Example Controller Method

namespace App\Http\Controllers;

use Illuminate\Http\Request;

class UserController extends Controller
{
    public function show($id)
    {
        return 'User '.$id;
    }
}

Define a route that uses this controller method:

Route::get('/user/{id}', [UserController::class, 'show']);

Eloquent ORM

Eloquent provides an elegant syntax for interacting with the database.

Defining a Model

Create a model using Artisan:

php artisan make:model User

This command creates a new model file in the app/Models directory.

Basic Eloquent Usage

use App\Models\User;

// Retrieve all users
$users = User::all();

// Retrieve a user by ID
$user = User::find(1);

// Create a new user
$user = new User;
$user->name = 'John Doe';
$user->email = 'john@example.com';
$user->save();

Blade Templating Engine

Blade is Laravel's powerful templating engine.

Basic Blade Syntax

Create a Blade view file in resources/views/welcome.blade.php:

<!DOCTYPE html>
<html>
<head>
    <title>Laravel</title>
</head>
<body>
    <h1>Hello, {{ $name }}</h1>
</body>
</html>

Pass data to the view from a route or controller:

Route::get('/', function () {
    return view('welcome', ['name' => 'John Doe']);
});

Conclusion

Laravel is a robust and flexible framework that simplifies PHP web development. By understanding its core concepts, setting up the environment, and exploring its features like routing, controllers, Eloquent ORM, and Blade templating, you can build powerful web applications efficiently. Continue exploring Laravel's extensive documentation and community resources to deepen your knowledge and skills.
