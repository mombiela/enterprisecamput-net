Introduction

Operators are special characters or symbols in PHP that are used to perform operations on variables and values. They are essential for manipulating data and controlling the flow of a program. This section will cover the different types of operators available in PHP, providing clear explanations, examples, and exercises to help you understand their usage.

Types of Operators

Arithmetic Operators

Arithmetic operators are used to perform common mathematical operations.

  • Addition (+): Adds two operands.
  • Subtraction (-): Subtracts the second operand from the first.
  • Multiplication (*): Multiplies two operands.
  • Division (/): Divides the first operand by the second.
  • Modulus (%): Returns the remainder of the division of the first operand by the second.
  • Exponentiation (**): Raises the first operand to the power of the second operand.
<?php
$a = 10;
$b = 3;

echo $a + $b; // Output: 13
echo $a - $b; // Output: 7
echo $a * $b; // Output: 30
echo $a / $b; // Output: 3.3333333333333
echo $a % $b; // Output: 1
echo $a ** $b; // Output: 1000
?>

Assignment Operators

Assignment operators are used to assign values to variables.

  • Basic Assignment (=): Assigns the value of the right operand to the left operand.
  • Addition Assignment (+=): Adds the right operand to the left operand and assigns the result to the left operand.
  • Subtraction Assignment (-=): Subtracts the right operand from the left operand and assigns the result to the left operand.
  • Multiplication Assignment (*=): Multiplies the left operand by the right operand and assigns the result to the left operand.
  • Division Assignment (/=): Divides the left operand by the right operand and assigns the result to the left operand.
  • Modulus Assignment (%=): Takes the modulus using two operands and assigns the result to the left operand.
<?php
$a = 10;
$b = 5;

$a += $b; // $a is now 15
$a -= $b; // $a is now 10
$a *= $b; // $a is now 50
$a /= $b; // $a is now 10
$a %= $b; // $a is now 0
?>

Comparison Operators

Comparison operators are used to compare two values.

  • Equal (==): Returns true if both operands are equal.
  • Identical (===): Returns true if both operands are equal and of the same type.
  • Not Equal (!= or <>): Returns true if operands are not equal.
  • Not Identical (!==): Returns true if operands are not equal or not of the same type.
  • Greater Than (>): Returns true if the left operand is greater than the right operand.
  • Less Than (<): Returns true if the left operand is less than the right operand.
  • Greater Than or Equal To (>=): Returns true if the left operand is greater than or equal to the right operand.
  • Less Than or Equal To (<=): Returns true if the left operand is less than or equal to the right operand.
<?php
$a = 10;
$b = 20;

var_dump($a == $b); // Output: bool(false)
var_dump($a === $b); // Output: bool(false)
var_dump($a != $b); // Output: bool(true)
var_dump($a !== $b); // Output: bool(true)
var_dump($a > $b); // Output: bool(false)
var_dump($a < $b); // Output: bool(true)
var_dump($a >= $b); // Output: bool(false)
var_dump($a <= $b); // Output: bool(true)
?>

Logical Operators

Logical operators are used to combine conditional statements.

  • And (&& or and): Returns true if both operands are true.
  • Or (|| or or): Returns true if at least one of the operands is true.
  • Not (!): Returns true if the operand is false.
<?php
$a = true;
$b = false;

var_dump($a && $b); // Output: bool(false)
var_dump($a || $b); // Output: bool(true)
var_dump(!$a); // Output: bool(false)
?>

Increment/Decrement Operators

Increment and decrement operators are used to increase or decrease a variable's value by one.

  • Increment (++): Increases the variable's value by one.
  • Decrement (--): Decreases the variable's value by one.
<?php
$a = 10;

echo ++$a; // Output: 11 (Pre-increment)
echo $a++; // Output: 11 (Post-increment)
echo $a;   // Output: 12

echo --$a; // Output: 11 (Pre-decrement)
echo $a--; // Output: 11 (Post-decrement)
echo $a;   // Output: 10
?>

String Operators

String operators are used to manipulate string values.

  • Concatenation (.): Concatenates two strings.
  • Concatenation Assignment (.=): Appends the right operand to the left operand and assigns the result to the left operand.
<?php
$a = "Hello";
$b = " World";

echo $a . $b; // Output: Hello World

$a .= $b;
echo $a; // Output: Hello World
?>

Array Operators

Array operators are used to compare arrays.

  • Union (+): Merges two arrays.
  • Equality (==): Returns true if two arrays have the same key/value pairs.
  • Identity (===): Returns true if two arrays have the same key/value pairs in the same order and of the same types.
  • Inequality (!= or <>): Returns true if two arrays are not equal.
  • Non-identity (!==): Returns true if two arrays are not identical.
<?php
$a = array("a" => "apple", "b" => "banana");
$b = array("c" => "cherry", "d" => "date");

$c = $a + $b; // Union of $a and $b
var_dump($c); // Output: array(4) { ["a"]=> string(5) "apple" ["b"]=> string(6) "banana" ["c"]=> string(6) "cherry" ["d"]=> string(4) "date" }

var_dump($a == $b); // Output: bool(false)
var_dump($a === $b); // Output: bool(false)
var_dump($a != $b); // Output: bool(true)
var_dump($a !== $b); // Output: bool(true)
?>

Conclusion

Operators are fundamental in PHP for performing various operations on data. Understanding how to use arithmetic, assignment, comparison, logical, increment/decrement, string, and array operators will significantly enhance your ability to write efficient and effective PHP code. Practice using these operators in different scenarios to become more comfortable with their syntax and functionality.
