Introduction to SQL Language

The Structured Query Language (SQL) is a standard language used to manage and manipulate relational databases. SQL allows you to perform various operations such as creating databases and tables, inserting data, updating and deleting records, and querying stored information.

Key Concepts

  • Database: An organized set of data stored and managed electronically.
  • Table: A structure within a database that organizes data into rows and columns.
  • Record: A row in a table that represents a single data entry.
  • Field: A column in a table that represents a specific attribute of the data.

Basic SQL Operations

Creating Databases and Tables

To start working with SQL, we first need to create a database and then define the tables that will store our data.

-- Create a database
CREATE DATABASE MyDatabase;

-- Select the database
USE MyDatabase;

-- Create a table
CREATE TABLE Employees (
    ID INT PRIMARY KEY,
    FirstName VARCHAR(50),
    LastName VARCHAR(50),
    Age INT,
    Department VARCHAR(50)
);

Inserting Data

Once we have our tables, we can insert data into them.

-- Insert a record into the Employees table
INSERT INTO Employees (ID, FirstName, LastName, Age, Department)
VALUES (1, 'John', 'Doe', 30, 'Sales');

Querying Data

Querying data is one of the most common operations in SQL. We use the SELECT clause to retrieve information from one or more tables.

-- Query all records from the Employees table
SELECT * FROM Employees;

-- Query specific records
SELECT FirstName, LastName FROM Employees WHERE Age > 25;

Updating Data

We can update existing data in a table using the UPDATE clause.

-- Update an employee's department
UPDATE Employees
SET Department = 'Marketing'
WHERE ID = 1;

Deleting Data

To delete records from a table, we use the DELETE clause.

-- Delete a record from the Employees table
DELETE FROM Employees WHERE ID = 1;

Functions and Operators in SQL

Aggregate Functions

SQL provides several aggregate functions that allow you to perform calculations on a set of values.

  • COUNT: Counts the number of rows.
  • SUM: Sums the values of a column.
  • AVG: Calculates the average of the values of a column.
  • MAX: Finds the maximum value.
  • MIN: Finds the minimum value.
-- Count the number of employees
SELECT COUNT(*) FROM Employees;

-- Calculate the average age of employees
SELECT AVG(Age) FROM Employees;

Comparison Operators

Comparison operators are used to compare two values.

  • =: Equal to
  • <>: Not equal to
  • >: Greater than
  • <: Less than
  • >=: Greater than or equal to
  • <=: Less than or equal to
-- Select employees older than 25 years
SELECT * FROM Employees WHERE Age > 25;

Conclusion

The SQL language is a powerful and essential tool for any professional working with databases. Through this topic, we have covered the basic SQL operations, including creating databases and tables, inserting, querying, updating, and deleting data. Additionally, we have explored aggregate functions and comparison operators that allow for more complex queries and obtaining valuable information from our databases. Practicing these concepts is fundamental to mastering SQL and effectively applying it in data management.
