Introduction

In this section, we will focus on practicing and reinforcing the SQL concepts learned in previous lessons. The exercises are designed to cover a variety of common queries and operations in databases, from the most basic to the most advanced.

Basic Exercises

Data Selection

  • Objective: Practice selecting data from a table.
  • Exercise: Select all records from the employees table.
SELECT * FROM employees;
  • Explanation: This query selects all columns and rows from the employees table.

Data Filtering

  • Objective: Learn to filter data using the WHERE clause.
  • Exercise: Select all employees whose salary is greater than 50,000.
SELECT * FROM employees
WHERE salary > 50000;
  • Explanation: The WHERE clause is used to filter records that meet a specific condition.

Intermediate Exercises

Data Sorting

  • Objective: Practice sorting results.
  • Exercise: Select all employees and sort them by last name in ascending order.
SELECT * FROM employees
ORDER BY last_name ASC;
  • Explanation: The ORDER BY clause is used to sort the results of a query in ascending (ASC) or descending (DESC) order.

Aggregation Functions

  • Objective: Use aggregation functions to summarize data.
  • Exercise: Calculate the average salary of all employees.
SELECT AVG(salary) AS average_salary
FROM employees;
  • Explanation: The AVG function calculates the average of a numeric column. AS is used to rename the resulting column.

Advanced Exercises

Joins

  • Objective: Practice combining data from multiple tables.
  • Exercise: Select the names of employees and the name of their departments.
SELECT employees.first_name, employees.last_name, departments.department_name
FROM employees
JOIN departments ON employees.department_id = departments.department_id;
  • Explanation: The JOIN clause is used to combine rows from two or more tables, based on a related column between them.

Subqueries

  • Objective: Use subqueries to perform more complex queries.
  • Exercise: Select the names of employees who have a salary higher than the company's average salary.
SELECT first_name, last_name
FROM employees
WHERE salary > (SELECT AVG(salary) FROM employees);
  • Explanation: The subquery (SELECT AVG(salary) FROM employees) calculates the average salary, and the outer query selects employees whose salary is higher than this average.

Conclusion

These exercises cover a variety of SQL operations and queries that are fundamental for working with databases. Practicing these exercises will help consolidate your knowledge and skills in SQL, preparing you to handle more complex tasks in the future. Remember that constant practice is key to mastering any query language.
