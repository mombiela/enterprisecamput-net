Control Structures in Perl
Loops ►

Control structures are fundamental in any programming language, and Perl is no exception. They allow you to control the flow of your program, making decisions, looping through data, and handling exceptions. This topic will cover the essential control structures in Perl, from basic to advanced levels.

Conditional Statements

Conditional statements allow you to execute code based on certain conditions.

if Statement

The if statement is used to execute a block of code if a specified condition is true.

my $age = 18;

if ($age >= 18) {
    print "You are an adult.\n";
}

if-else Statement

The if-else statement provides an alternative block of code to execute if the condition is false.

my $age = 16;

if ($age >= 18) {
    print "You are an adult.\n";
} else {
    print "You are a minor.\n";
}

if-elsif-else Statement

The if-elsif-else statement allows for multiple conditions to be checked in sequence.

my $age = 20;

if ($age < 13) {
    print "You are a child.\n";
} elsif ($age < 18) {
    print "You are a teenager.\n";
} else {
    print "You are an adult.\n";
}

Looping Structures

Looping structures allow you to execute a block of code multiple times.

while Loop

The while loop executes a block of code as long as the condition is true.

my $count = 0;

while ($count < 5) {
    print "Count is $count\n";
    $count++;
}

for Loop

The for loop is used to iterate over a range of values.

for (my $i = 0; $i < 5; $i++) {
    print "Iteration $i\n";
}

foreach Loop

The foreach loop is used to iterate over a list of values.

my @colors = ('red', 'blue', 'green');

foreach my $color (@colors) {
    print "Color: $color\n";
}

until Loop

The until loop is the opposite of the while loop; it executes a block of code as long as the condition is false.

my $count = 0;

until ($count == 5) {
    print "Count is $count\n";
    $count++;
}

Control Flow Modifiers

Control flow modifiers can be used to alter the behavior of loops and conditional statements.

last

The last statement is used to exit a loop immediately.

for (my $i = 0; $i < 10; $i++) {
    if ($i == 5) {
        last;
    }
    print "Iteration $i\n";
}

next

The next statement is used to skip the rest of the current iteration and move to the next iteration.

for (my $i = 0; $i < 10; $i++) {
    if ($i % 2 == 0) {
        next;
    }
    print "Odd number: $i\n";
}

redo

The redo statement restarts the current iteration without evaluating the loop condition again.

my $count = 0;

while ($count < 5) {
    $count++;
    if ($count == 3) {
        redo;
    }
    print "Count is $count\n";
}

Advanced Control Structures

given-when

The given-when structure is similar to the switch-case statement found in other languages.

use feature 'switch';

my $fruit = 'apple';

given ($fruit) {
    when ('apple') {
        print "It's an apple.\n";
    }
    when ('banana') {
        print "It's a banana.\n";
    }
    default {
        print "Unknown fruit.\n";
    }
}

Conclusion

Control structures are essential for directing the flow of your Perl programs. From basic conditional statements like if and if-else to more complex looping structures like while, for, and foreach, understanding these constructs will allow you to write more efficient and readable code. Additionally, control flow modifiers like last, next, and redo provide further control over your loops, while advanced structures like given-when offer more expressive ways to handle multiple conditions. Practice these constructs to become proficient in managing the flow of your Perl programs.
