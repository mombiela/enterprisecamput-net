Introduction

Perl, a highly capable and feature-rich programming language, comes with a vast array of built-in functions. These functions are designed to perform a wide range of tasks, from basic string manipulation to complex data processing. Understanding these built-in functions is crucial for writing efficient and effective Perl code.

Basic Built-in Functions

Print and Say

  • print: Outputs a string or a list of strings to the standard output.
  • say: Similar to print, but automatically appends a newline at the end of the output (available in Perl 5.10 and later).
print "Hello, World!";
say "Hello, World!";  # Outputs "Hello, World!" followed by a newline

Scalar Functions

  • chomp: Removes the newline character from the end of a string.
  • length: Returns the length of a string.
my $string = "Hello, World!\n";
chomp($string);  # $string is now "Hello, World!"
my $length = length($string);  # $length is 12

Numeric Functions

  • int: Returns the integer portion of a number.
  • rand: Generates a random number between 0 and the specified value.
my $number = 3.14;
my $integer = int($number);  # $integer is 3
my $random = rand(10);  # $random is a number between 0 and 10

Intermediate Built-in Functions

List Functions

  • push: Adds elements to the end of an array.
  • pop: Removes the last element from an array.
  • shift: Removes the first element from an array.
  • unshift: Adds elements to the beginning of an array.
my @array = (1, 2, 3);
push(@array, 4);  # @array is now (1, 2, 3, 4)
pop(@array);  # @array is now (1, 2, 3)
shift(@array);  # @array is now (2, 3)
unshift(@array, 1);  # @array is now (1, 2, 3)

String Functions

  • substr: Extracts a substring from a string.
  • index: Finds the position of a substring within a string.
  • split: Splits a string into a list of strings based on a delimiter.
my $string = "Hello, World!";
my $substring = substr($string, 0, 5);  # $substring is "Hello"
my $position = index($string, "World");  # $position is 7
my @words = split(/, /, $string);  # @words is ("Hello", "World!")

Hash Functions

  • keys: Returns a list of keys from a hash.
  • values: Returns a list of values from a hash.
  • each: Iterates over key-value pairs in a hash.
my %hash = (foo => 1, bar => 2);
my @keys = keys %hash;  # @keys is ("foo", "bar")
my @values = values %hash;  # @values is (1, 2)
while (my ($key, $value) = each %hash) {
    print "$key: $value\n";
}

Advanced Built-in Functions

File Handling Functions

  • open: Opens a file for reading, writing, or appending.
  • close: Closes an open file handle.
  • read: Reads data from a file handle.
  • write: Writes data to a file handle.
open(my $fh, '<', 'file.txt') or die "Cannot open file: $!";
while (my $line = <$fh>) {
    print $line;
}
close($fh);

System Interaction Functions

  • system: Executes an external command and waits for it to finish.
  • exec: Replaces the current process with an external command.
system("ls -l");  # Executes the 'ls -l' command
exec("perl script.pl");  # Replaces the current process with 'perl script.pl'

Regular Expression Functions

  • m//: Matches a string against a regular expression.
  • s///: Substitutes a part of a string that matches a regular expression.
  • tr///: Transliterates characters in a string.
my $string = "Hello, World!";
if ($string =~ m/World/) {
    print "Match found!\n";
}
$string =~ s/World/Perl/;  # $string is now "Hello, Perl!"
$string =~ tr/A-Z/a-z/;  # $string is now "hello, perl!"

Conclusion

Perl's built-in functions are powerful tools that can simplify many programming tasks. From basic input/output operations to complex data manipulation, these functions provide a solid foundation for writing efficient Perl code. By mastering these built-in functions, you can significantly enhance your Perl programming skills and tackle a wide range of problems with ease.
