Introduction

Welcome to your first Perl script! Perl is a versatile and powerful scripting language used for a variety of tasks, from text processing to web development. In this section, we'll guide you through writing and running your first Perl script. By the end of this lesson, you'll have a basic understanding of Perl syntax and how to execute Perl scripts.

Setting Up Your Environment

Before we start writing our first Perl script, ensure you have Perl installed on your system. You can check if Perl is installed by running the following command in your terminal or command prompt:

perl -v

If Perl is installed, you will see the version information. If not, you can download and install Perl from perl.org.

Writing Your First Perl Script

Let's dive into writing a simple Perl script. We'll create a script that prints "Hello, World!" to the console.

Step-by-Step Guide

  1. Create a New File: Open your favorite text editor and create a new file named hello_world.pl.

  2. Write the Script: Add the following code to your hello_world.pl file:

    #!/usr/bin/perl
use strict;
use warnings;

print "Hello, World!\n";
    • #!/usr/bin/perl: This is the shebang line that tells the system to use the Perl interpreter to run the script.
    • use strict; and use warnings;: These lines enable strict and warnings modes, which help catch potential errors and enforce good coding practices.
    • print "Hello, World!\n";: This line prints "Hello, World!" followed by a newline character to the console.

  3. Save the File: Save the file and close the text editor.

Running Your Perl Script

Now that you've written your first Perl script, it's time to run it.

  1. Open Terminal or Command Prompt: Navigate to the directory where you saved hello_world.pl.

  2. Run the Script: Execute the script by typing the following command:

    perl hello_world.pl

    You should see the output:

    Hello, World!

Key Concepts

  • Shebang Line (#!/usr/bin/perl):
    • Tells the operating system to use the Perl interpreter to execute the script.
  • Strict and Warnings Modes:
    • use strict;: Enforces strict variable declaration rules.
    • use warnings;: Provides warnings for potential issues in the code.
  • Print Function:
    • print "Hello, World!\n";: Outputs text to the console.

Common Errors and Troubleshooting

  • Permission Denied:
    • If you encounter a "Permission denied" error, ensure the script has execute permissions. You can add execute permissions with the following command: 
      chmod +x hello_world.pl
  • Command Not Found:
    • If you see a "command not found" error, ensure Perl is installed and the shebang line points to the correct Perl interpreter path.

Conclusion

Congratulations! You've written and executed your first Perl script. This simple "Hello, World!" program introduces you to the basic structure of a Perl script, including the shebang line, strict and warnings modes, and the print function. As you progress through this course, you'll build on these fundamentals to create more complex and powerful Perl scripts. Happy coding!
