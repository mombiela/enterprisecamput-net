Introduction to Matrices

Matrices are fundamental to MATLAB, which stands for "Matrix Laboratory." Understanding how to create, manipulate, and perform operations on matrices is essential for any MATLAB programmer.

Creating Matrices

  • Row Vector: A 1xN matrix. 
    row_vector = [1, 2, 3, 4];
  • Column Vector: An Nx1 matrix. 
    column_vector = [1; 2; 3; 4];
  • 2D Matrix: An MxN matrix. 
    matrix_2d = [1, 2, 3; 4, 5, 6; 7, 8, 9];

Basic Matrix Operations

  • Addition and Subtraction: Element-wise operations. 
    A = [1, 2; 3, 4];
B = [5, 6; 7, 8];
C = A + B; % Addition
D = A - B; % Subtraction
  • Scalar Multiplication: Multiplying each element by a scalar. 
    scalar = 2;
E = scalar * A;

Matrix Multiplication

Matrix multiplication follows the rules of linear algebra.

  • Element-wise Multiplication: Use the .* operator. 
    F = A .* B;
  • Matrix Product: Use the * operator. 
    G = A * B;

Transpose of a Matrix

  • Transpose: Flipping a matrix over its diagonal. 
    H = A';

Advanced Matrix Operations

Inverse and Determinant

  • Inverse: Only for square matrices. 
    I = inv(A);
  • Determinant: A scalar value that is a function of the entries of a square matrix. 
    det_A = det(A);

Eigenvalues and Eigenvectors

  • Eigenvalues and Eigenvectors: Important in many applications like stability analysis. 
    [V, D] = eig(A);
    • V contains the eigenvectors.
    • D is a diagonal matrix with eigenvalues.

Solving Linear Systems

  • Linear System: Ax = b 
    A = [1, 2; 3, 4];
b = [5; 6];
x = A \ b; % Solves for x

Special Matrices

  • Identity Matrix: A square matrix with ones on the diagonal and zeros elsewhere. 
    I = eye(3); % 3x3 Identity matrix
  • Zeros Matrix: A matrix filled with zeros. 
    Z = zeros(3, 2); % 3x2 matrix of zeros
  • Ones Matrix: A matrix filled with ones. 
    O = ones(2, 3); % 2x3 matrix of ones

Practical Examples

Example 1: Matrix Addition and Multiplication

A = [1, 2; 3, 4];
B = [5, 6; 7, 8];
C = A + B; % Matrix addition
D = A * B; % Matrix multiplication

Example 2: Solving a Linear System

A = [2, -1; -3, 4];
b = [1; 7];
x = A \ b; % Solving Ax = b

Example 3: Eigenvalues and Eigenvectors

A = [4, -2; 1, 1];
[V, D] = eig(A);

Summary

Understanding matrix operations in MATLAB is crucial for performing a wide range of mathematical and engineering tasks. From basic operations like addition and multiplication to advanced concepts like eigenvalues and solving linear systems, mastering these skills will significantly enhance your ability to use MATLAB effectively. Practice these operations with different matrices to become proficient in handling matrix-related problems.
