Cell Arrays in MATLAB
Structures ►

Cell arrays are a versatile data type in MATLAB that allow you to store arrays of different sizes and types. This makes them particularly useful for handling heterogeneous data. In this section, we will cover the basics of cell arrays, how to create and manipulate them, and some advanced techniques.

Introduction to Cell Arrays

  • Definition: A cell array is a data type with indexed data containers called cells, where each cell can contain any type of data.
  • Use Cases: Useful for storing a collection of different types of data, such as strings, numbers, and other arrays.

Creating Cell Arrays

Using the `cell` Function

The cell function creates an empty cell array of specified size.

C = cell(3, 2); % Creates a 3x2 cell array

Direct Assignment

You can directly assign values to cells using curly braces {}.

C{1, 1} = 42;          % Assigns a number
C{1, 2} = 'Hello';     % Assigns a string
C{2, 1} = [1, 2, 3];   % Assigns a numeric array
C{2, 2} = {1, 'a', 3}; % Assigns a cell array

Combining Different Data Types

C = {42, 'Hello'; [1, 2, 3], {1, 'a', 3}};

Accessing Data in Cell Arrays

  • Using Curly Braces {}: To access the content of a cell.
  • Using Parentheses (): To access the cell itself.

Examples

% Accessing the content
value = C{1, 1}; % Returns 42

% Accessing the cell
cellContent = C(1, 1); % Returns {42}

Manipulating Cell Arrays

Adding Elements

You can add elements to a cell array by assigning a value to a new cell.

C{3, 1} = 'New Element';

Removing Elements

You can remove elements by assigning an empty array [] to the cell.

C{1, 1} = [];

Resizing Cell Arrays

Cell arrays can be resized dynamically by assigning values outside the current dimensions.

C{4, 3} = 'Another Element'; % Expands the cell array to 4x3

Advanced Techniques

Cell Functions

MATLAB provides several functions to work with cell arrays:

  • cellfun: Apply a function to each cell.
  • celldisp: Display the contents of a cell array.
  • cell2mat: Convert a cell array of matrices into a single matrix.

Example: Using `cellfun`

C = {1, 2, 3; 4, 5, 6};
result = cellfun(@(x) x^2, C); % Squares each element in the cell array

Example: Using `cell2mat`

C = {1, 2; 3, 4};
matrix = cell2mat(C); % Converts to a 2x2 matrix

Summary

Cell arrays in MATLAB are a powerful tool for managing heterogeneous data. They allow you to store different types and sizes of data in a single container. Understanding how to create, access, and manipulate cell arrays is essential for advanced data handling in MATLAB. By mastering cell arrays, you can write more flexible and efficient MATLAB code.
