Introduction to Structures

Structures in MATLAB are data types that group related data using data containers called fields. Each field can contain any type of data, including arrays, strings, or even other structures. This makes structures a versatile way to manage complex data.

Key Concepts

  • Field Names: Identifiers for the data stored in a structure.
  • Dot Notation: Used to access the data in a structure's fields.
  • Nested Structures: Structures that contain other structures as fields.

Creating Structures

To create a structure, you can use the struct function or direct assignment using dot notation.

Using the `struct` Function

student = struct('Name', 'John Doe', 'Age', 21, 'GPA', 3.5);

In this example:

  • Name, Age, and GPA are field names.
  • 'John Doe', 21, and 3.5 are the corresponding values.

Using Dot Notation

student.Name = 'John Doe';
student.Age = 21;
student.GPA = 3.5;

This method achieves the same result as using the struct function.

Accessing and Modifying Data in Structures

You can access and modify the data in a structure using dot notation.

Accessing Data

name = student.Name;  % Retrieves 'John Doe'
age = student.Age;    % Retrieves 21
gpa = student.GPA;    % Retrieves 3.5

Modifying Data

student.GPA = 3.8;  % Updates GPA to 3.8

Nested Structures

Structures can contain other structures, allowing for more complex data organization.

Creating Nested Structures

student.Address = struct('Street', '123 Main St', 'City', 'Springfield', 'Zip', '12345');

Accessing Data in Nested Structures

city = student.Address.City;  % Retrieves 'Springfield'

Arrays of Structures

You can create arrays of structures to manage multiple records.

Creating an Array of Structures

students(1) = struct('Name', 'John Doe', 'Age', 21, 'GPA', 3.5);
students(2) = struct('Name', 'Jane Smith', 'Age', 22, 'GPA', 3.7);

Accessing Data in an Array of Structures

name1 = students(1).Name;  % Retrieves 'John Doe'
gpa2 = students(2).GPA;    % Retrieves 3.7

Advanced Operations with Structures

MATLAB provides functions to manipulate structures more efficiently.

Fieldnames and Structfun

  • fieldnames: Returns a cell array of field names.
  • structfun: Applies a function to each field of a structure.

Example

fields = fieldnames(student);  % Returns {'Name', 'Age', 'GPA', 'Address'}
gpaSum = structfun(@(x) sum([students.GPA]), students);  % Sums up all GPAs

Summary

Structures in MATLAB are powerful tools for organizing and managing complex data. They allow for flexible data storage with named fields, support for nested structures, and the ability to create arrays of structures. Understanding how to create, access, and manipulate structures is essential for efficient data management in MATLAB.
