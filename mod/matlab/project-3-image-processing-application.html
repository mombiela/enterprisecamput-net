Introduction

In this project, you will learn how to develop an image processing application using MATLAB. This project will cover various image processing techniques from basic to advanced levels, including reading and displaying images, image enhancement, filtering, and edge detection.

Objectives

  • Understand the basics of image processing.
  • Learn how to read, display, and save images in MATLAB.
  • Apply image enhancement techniques.
  • Implement filtering methods.
  • Perform edge detection on images.

Prerequisites

  • Basic understanding of MATLAB syntax and functions.
  • Familiarity with matrix operations.

Reading and Displaying Images

Key Concepts

  • Reading images using imread.
  • Displaying images using imshow.
  • Understanding image data types and color spaces.

Example

% Read an image from a file
image = imread('example.jpg');

% Display the image
imshow(image);

% Get image information
info = imfinfo('example.jpg');
disp(info);

Explanation

  • imread('example.jpg') reads the image file and stores it in the variable image.
  • imshow(image) displays the image in a figure window.
  • imfinfo('example.jpg') retrieves information about the image file, such as its dimensions, color type, and file format.

Image Enhancement

Key Concepts

  • Adjusting image brightness and contrast.
  • Histogram equalization.
  • Image sharpening.

Example

% Read an image
image = imread('example.jpg');

% Convert to grayscale
grayImage = rgb2gray(image);

% Adjust brightness
brightImage = imadjust(grayImage, [], [], 1.2);

% Histogram equalization
equalizedImage = histeq(grayImage);

% Display results
subplot(1, 3, 1), imshow(grayImage), title('Original Grayscale Image');
subplot(1, 3, 2), imshow(brightImage), title('Brightness Adjusted Image');
subplot(1, 3, 3), imshow(equalizedImage), title('Histogram Equalized Image');

Explanation

  • rgb2gray(image) converts the RGB image to grayscale.
  • imadjust(grayImage, [], [], 1.2) adjusts the brightness of the grayscale image.
  • histeq(grayImage) performs histogram equalization to enhance the contrast of the image.
  • subplot and imshow functions are used to display multiple images in a single figure window.

Filtering

Key Concepts

  • Smoothing filters (e.g., Gaussian filter).
  • Sharpening filters (e.g., Laplacian filter).
  • Median filtering.

Example

% Read an image
image = imread('example.jpg');

% Convert to grayscale
grayImage = rgb2gray(image);

% Apply Gaussian filter
gaussianFiltered = imgaussfilt(grayImage, 2);

% Apply Laplacian filter
laplacianFiltered = imfilter(grayImage, fspecial('laplacian'));

% Apply Median filter
medianFiltered = medfilt2(grayImage);

% Display results
subplot(1, 3, 1), imshow(gaussianFiltered), title('Gaussian Filtered Image');
subplot(1, 3, 2), imshow(laplacianFiltered), title('Laplacian Filtered Image');
subplot(1, 3, 3), imshow(medianFiltered), title('Median Filtered Image');

Explanation

  • imgaussfilt(grayImage, 2) applies a Gaussian filter with a standard deviation of 2 to the grayscale image.
  • imfilter(grayImage, fspecial('laplacian')) applies a Laplacian filter to the grayscale image.
  • medfilt2(grayImage) applies a median filter to the grayscale image.

Edge Detection

Key Concepts

  • Sobel operator.
  • Canny edge detector.

Example

% Read an image
image = imread('example.jpg');

% Convert to grayscale
grayImage = rgb2gray(image);

% Apply Sobel edge detection
sobelEdges = edge(grayImage, 'sobel');

% Apply Canny edge detection
cannyEdges = edge(grayImage, 'canny');

% Display results
subplot(1, 2, 1), imshow(sobelEdges), title('Sobel Edge Detection');
subplot(1, 2, 2), imshow(cannyEdges), title('Canny Edge Detection');

Explanation

  • edge(grayImage, 'sobel') detects edges in the grayscale image using the Sobel operator.
  • edge(grayImage, 'canny') detects edges in the grayscale image using the Canny edge detector.

Conclusion

In this project, you have learned how to develop an image processing application in MATLAB. You have covered essential techniques such as reading and displaying images, image enhancement, filtering, and edge detection. These skills are fundamental for more advanced image processing tasks and applications. Continue practicing and exploring additional MATLAB functions to further enhance your image processing capabilities.
