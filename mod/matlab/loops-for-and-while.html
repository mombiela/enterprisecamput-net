Introduction

Loops are fundamental constructs in programming that allow you to execute a block of code multiple times. In MATLAB, the two primary types of loops are for loops and while loops. This section will guide you through the basics of these loops, provide examples, and offer exercises to practice.

for Loops

Basic Syntax

A for loop in MATLAB repeats a group of statements a fixed, predetermined number of times. The syntax is as follows:

for index = startValue:endValue
    % Statements to execute
end

Example

Let's start with a simple example where we print numbers from 1 to 5:

for i = 1:5
    disp(i)
end

Explanation

  • i = 1:5 initializes the loop variable i to range from 1 to 5.
  • disp(i) displays the current value of i in each iteration.

Nested for Loops

You can also nest for loops to handle more complex tasks. For example, creating a multiplication table:

for i = 1:5
    for j = 1:5
        fprintf('%d * %d = %d\n', i, j, i*j);
    end
end

Explanation

  • The outer loop runs from 1 to 5.
  • The inner loop also runs from 1 to 5.
  • fprintf prints the product of i and j.

while Loops

Basic Syntax

A while loop in MATLAB repeats a group of statements an indefinite number of times as long as the condition remains true. The syntax is:

while condition
    % Statements to execute
end

Example

Let's look at an example where we print numbers from 1 to 5 using a while loop:

i = 1;
while i <= 5
    disp(i)
    i = i + 1;
end

Explanation

  • i = 1 initializes the loop variable i.
  • while i <= 5 checks if i is less than or equal to 5.
  • disp(i) displays the current value of i.
  • i = i + 1 increments i by 1 in each iteration.

Infinite Loops

Be cautious with while loops as they can create infinite loops if the condition never becomes false. For example:

i = 1;
while true
    disp(i)
    i = i + 1;
    if i > 5
        break;
    end
end

Explanation

  • while true creates an infinite loop.
  • if i > 5 checks if i is greater than 5.
  • break exits the loop when i is greater than 5.

Comparison of for and while Loops

| Feature | for Loop | while Loop | |----------------|-----------------------------------|------------------------------------| | Use Case | Known number of iterations | Unknown number of iterations | | Initialization | Inside the loop definition | Outside the loop | | Condition | Defined by the range | Defined by a logical condition | | Common Pitfall | None | Risk of infinite loops |

Exercises

Exercise 1: Sum of Numbers

Write a for loop to calculate the sum of numbers from 1 to 10.

sum = 0;
for i = 1:10
    sum = sum + i;
end
disp(sum)

Exercise 2: Factorial Calculation

Write a while loop to calculate the factorial of a given number n.

n = 5; % You can change this value to test with different numbers
factorial = 1;
i = 1;
while i <= n
    factorial = factorial * i;
    i = i + 1;
end
disp(factorial)

Conclusion

Understanding and using loops effectively is crucial for efficient programming in MATLAB. for loops are ideal when the number of iterations is known beforehand, while while loops are better suited for situations where the number of iterations depends on a condition. Practice using both types of loops to become proficient in their application.
