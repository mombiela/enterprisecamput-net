Introduction

Plotting is a fundamental aspect of data visualization in MATLAB. This section will guide you through the basics of creating plots, customizing them, and understanding the different types of plots available in MATLAB. By the end of this section, you will be able to create and customize simple plots and understand how to use MATLAB's plotting functions effectively.

Creating Simple Plots

To create a basic plot in MATLAB, you can use the plot function. This function takes in data points and plots them on a 2D graph.

Example: Basic Line Plot

% Define the data points
x = 0:0.1:10;
y = sin(x);

% Create the plot
plot(x, y);

% Add labels and title
xlabel('X-axis');
ylabel('Y-axis');
title('Basic Line Plot');
  • x = 0:0.1:10; creates a vector of x values from 0 to 10 with a step of 0.1.
  • y = sin(x); computes the sine of each x value.
  • plot(x, y); creates the plot.
  • xlabel, ylabel, and title add labels and a title to the plot.

Customizing Plots

MATLAB provides various options to customize plots, including changing line styles, colors, markers, and adding grid lines.

Example: Customizing Line Style and Color

% Define the data points
x = 0:0.1:10;
y = cos(x);

% Create the plot with custom line style and color
plot(x, y, '--r', 'LineWidth', 2);

% Add labels and title
xlabel('X-axis');
ylabel('Y-axis');
title('Customized Line Plot');
grid on; % Add grid lines
  • '--r' specifies a dashed red line.
  • 'LineWidth', 2 sets the line width to 2.
  • grid on; adds grid lines to the plot.

Multiple Plots in One Figure

You can plot multiple datasets on the same figure using the hold on and hold off commands.

Example: Multiple Plots

% Define the data points
x = 0:0.1:10;
y1 = sin(x);
y2 = cos(x);

% Create the first plot
plot(x, y1, '-b', 'LineWidth', 2);
hold on; % Keep the current plot

% Create the second plot
plot(x, y2, '--r', 'LineWidth', 2);

% Add labels and title
xlabel('X-axis');
ylabel('Y-axis');
title('Multiple Plots');
legend('sin(x)', 'cos(x)'); % Add a legend
hold off; % Release the plot
  • hold on; allows multiple plots to be drawn on the same figure.
  • legend adds a legend to differentiate between the plots.

Subplots

Subplots allow you to create multiple plots in a single figure, arranged in a grid.

Example: Subplots

% Define the data points
x = 0:0.1:10;
y1 = sin(x);
y2 = cos(x);

% Create a 2x1 grid of subplots
subplot(2, 1, 1); % First subplot
plot(x, y1, '-b');
title('Sine Function');

subplot(2, 1, 2); % Second subplot
plot(x, y2, '--r');
title('Cosine Function');
  • subplot(2, 1, 1); creates the first subplot in a 2x1 grid.
  • subplot(2, 1, 2); creates the second subplot.

Advanced Plotting Techniques

MATLAB offers advanced plotting techniques such as 3D plots, contour plots, and surface plots.

Example: 3D Plot

% Define the data points
[x, y] = meshgrid(-5:0.5:5, -5:0.5:5);
z = sin(sqrt(x.^2 + y.^2));

% Create the 3D plot
surf(x, y, z);

% Add labels and title
xlabel('X-axis');
ylabel('Y-axis');
zlabel('Z-axis');
title('3D Surface Plot');
  • meshgrid generates a grid of x and y values.
  • surf creates a 3D surface plot.
  • zlabel adds a label to the z-axis.

Conclusion

In this section, you learned the basics of plotting in MATLAB, including creating simple plots, customizing them, plotting multiple datasets, using subplots, and creating advanced 3D plots. Mastering these basics will enable you to effectively visualize data and enhance your MATLAB programming skills.
