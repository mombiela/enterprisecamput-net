Introduction to Machine Learning Toolbox

The Machine Learning Toolbox in MATLAB provides tools and functions to perform various machine learning tasks. This section will cover the basics of the toolbox, including installation and an overview of its capabilities.

  • Installation: Ensure the Machine Learning Toolbox is installed via MATLAB's Add-On Explorer.
  • Capabilities: Data preprocessing, classification, regression, clustering, and model evaluation.

Data Preprocessing

Data preprocessing is a crucial step in machine learning. This section will cover techniques to prepare data for analysis.

  • Data Import: Use readtable, readmatrix, and other functions to import data.
  • Data Cleaning: Handle missing values, outliers, and data normalization.
  • Feature Engineering: Create new features from existing data.

Example: Data Import and Cleaning

% Import data
data = readtable('data.csv');

% Display first few rows of the table
head(data)

% Handle missing values by filling with mean
data = fillmissing(data, 'constant', mean(data, 'omitnan'));

% Normalize data
data = normalize(data);

Classification

Classification is the process of predicting the class of given data points. This section will cover various classification algorithms available in MATLAB.

  • Algorithms: Decision Trees, Support Vector Machines (SVM), k-Nearest Neighbors (k-NN), and Neural Networks.
  • Training: Use fitctree, fitcsvm, fitcknn, and fitcnet for training models.
  • Evaluation: Use confusion matrices and cross-validation to evaluate models.

Example: Training a Decision Tree

% Load sample data
load fisheriris

% Split data into training and test sets
cv = cvpartition(species, 'HoldOut', 0.3);
trainData = meas(training(cv), :);
trainLabels = species(training(cv));
testData = meas(test(cv), :);
testLabels = species(test(cv));

% Train a decision tree classifier
treeModel = fitctree(trainData, trainLabels);

% Predict on test data
predictions = predict(treeModel, testData);

% Evaluate the model
confMat = confusionmat(testLabels, predictions);
disp('Confusion Matrix:');
disp(confMat);

Regression

Regression analysis is used to predict a continuous output variable based on one or more input features. This section will cover various regression techniques.

  • Algorithms: Linear Regression, Polynomial Regression, Support Vector Regression (SVR).
  • Training: Use fitlm, fitrsvm, and other functions for training regression models.
  • Evaluation: Use metrics like Mean Squared Error (MSE) and R-squared.

Example: Linear Regression

% Load sample data
load carsmall

% Prepare data
X = [Weight, Horsepower];
y = MPG;

% Train a linear regression model
lm = fitlm(X, y);

% Predict on new data
newData = [3500, 200]; % Example new data
predictedMPG = predict(lm, newData);

% Display the result
disp(['Predicted MPG: ', num2str(predictedMPG)]);

Clustering

Clustering is the task of grouping a set of objects in such a way that objects in the same group are more similar to each other than to those in other groups.

  • Algorithms: k-Means, Hierarchical Clustering, Gaussian Mixture Models (GMM).
  • Training: Use kmeans, linkage, and fitgmdist for clustering.
  • Evaluation: Use silhouette plots and cluster validity indices.

Example: k-Means Clustering

% Load sample data
load fisheriris

% Perform k-means clustering
numClusters = 3;
[idx, C] = kmeans(meas, numClusters);

% Plot the clusters
figure;
gscatter(meas(:,1), meas(:,2), idx);
hold on;
plot(C(:,1), C(:,2), 'kx', 'MarkerSize', 15, 'LineWidth', 3);
title('k-Means Clustering');
xlabel('Sepal Length');
ylabel('Sepal Width');
hold off;

Model Evaluation and Hyperparameter Tuning

Evaluating and tuning machine learning models is essential for improving their performance. This section will cover various evaluation metrics and techniques for hyperparameter tuning.

  • Evaluation Metrics: Accuracy, Precision, Recall, F1-Score, MSE, R-squared.
  • Cross-Validation: Use crossval and cvpartition for cross-validation.
  • Hyperparameter Tuning: Use bayesopt, fitctree with hyperparameter options.

Example: Cross-Validation and Hyperparameter Tuning

% Load sample data
load fisheriris

% Define the model and hyperparameters
treeTemplate = templateTree('MaxNumSplits', 1:10);

% Perform cross-validation
cvModel = fitcensemble(meas, species, 'Method', 'Bag', 'Learners', treeTemplate, 'CrossVal', 'on');

% Display cross-validation results
cvLoss = kfoldLoss(cvModel);
disp(['Cross-Validation Loss: ', num2str(cvLoss)]);

Conclusion

The Machine Learning Toolbox in MATLAB provides a comprehensive set of tools for data preprocessing, classification, regression, clustering, and model evaluation. By understanding and utilizing these tools, you can effectively build and evaluate machine learning models for various applications. Practice with the provided examples and explore further to deepen your understanding and proficiency in machine learning with MATLAB.
