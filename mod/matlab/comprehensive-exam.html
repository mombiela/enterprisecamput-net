Overview

The comprehensive exam is designed to evaluate your understanding and proficiency in MATLAB, covering topics from beginner to advanced levels. This exam will test your knowledge on various aspects of MATLAB, including basic syntax, data manipulation, advanced functions, and application development.

Exam Structure

The exam is divided into three sections:

  1. Basic Level
  2. Intermediate Level
  3. Advanced Level

Each section will contain multiple-choice questions, short answer questions, and practical coding exercises.

Basic Level

Key Concepts

  • MATLAB Environment
    • Command Window
    • Workspace
    • Editor
  • Basic Syntax
    • Variables and Data Types
    • Arithmetic Operations
    • Basic Functions
  • Control Structures
    • If-Else Statements
    • For Loops
    • While Loops

Example Questions

  1. Multiple Choice:

    • What is the command to clear all variables from the workspace?
      • a) clear all
      • b) clc
      • c) clearvars
      • d) reset

  2. Short Answer:

    • Explain the difference between = and == in MATLAB.

  3. Coding Exercise:

    • Write a script that calculates the factorial of a given number using a for loop.
% Factorial Calculation
n = 5; % Example number
factorial = 1;
for i = 1:n
    factorial = factorial * i;
end
disp(['Factorial of ', num2str(n), ' is ', num2str(factorial)]);

    • Write a script to plot the sine and cosine functions on the same graph with different colors and add a legend.
% Plotting Sine and Cosine
x = linspace(0, 2*pi, 100);
y1 = sin(x);
y2 = cos(x);

figure;
plot(x, y1, 'r', 'DisplayName', 'Sine');
hold on;
plot(x, y2, 'b', 'DisplayName', 'Cosine');
legend show;
title('Sine and Cosine Functions');
xlabel('x');
ylabel('y');
hold off;

  3. Coding Exercise:

    • Create a simple GUI that allows the user to input a number and displays its square.
% Simple GUI for Squaring a Number
function simple_gui
    fig = uifigure('Position', [100 100 300 200]);
    lbl = uilabel(fig, 'Position', [100 150 100 22], 'Text', 'Enter a number:');
    edt = uieditfield(fig, 'numeric', 'Position', [100 120 100 22]);
    btn = uibutton(fig, 'Position', [100 80 100 22], 'Text', 'Calculate', ...
        'ButtonPushedFcn', @(btn,event) calculateSquare(edt.Value));
    lblResult = uilabel(fig, 'Position', [100 50 100 22], 'Text', '');
    
    function calculateSquare(num)
        result = num^2;
        lblResult.Text = ['Square: ', num2str(result)];
    end
end
