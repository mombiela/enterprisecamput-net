Exporting Data
Working with Different File Formats ►

Exporting data is a crucial skill in MATLAB, enabling you to save your work and share results with others. This topic will guide you through various methods to export data, from basic to advanced techniques.

1. Basic Data Export

1.1 Exporting to Text Files

Text files are a simple and widely used format for exporting data.

  • save Function: Saves workspace variables to a file.
  • dlmwrite Function: Writes matrix data to a text file with a specified delimiter.
  • fprintf Function: Writes formatted data to a text file.

Example: Using save

% Create some data
A = rand(5);
B = magic(5);

% Save data to a .mat file
save('myData.mat', 'A', 'B');

Example: Using dlmwrite

% Create some data
C = rand(5);

% Save data to a text file with comma delimiter
dlmwrite('myData.txt', C, 'delimiter', ',');

Example: Using fprintf

% Create some data
D = rand(5);

% Open a file for writing
fileID = fopen('myDataFormatted.txt', 'w');

% Write data with formatted output
fprintf(fileID, '%6.2f %6.2f %6.2f %6.2f %6.2f\n', D');

% Close the file
fclose(fileID);

1.2 Exporting to Excel Files

Excel files are useful for data analysis and sharing with non-programmers.

  • xlswrite Function: Writes data to an Excel file.

Example: Using xlswrite

% Create some data
E = rand(5);

% Save data to an Excel file
xlswrite('myData.xlsx', E);

2. Intermediate Data Export

2.1 Exporting to CSV Files

CSV (Comma-Separated Values) files are a common format for data exchange.

  • csvwrite Function: Writes matrix data to a CSV file.
  • writematrix Function: Writes matrix data to a file, supporting various formats.

Example: Using csvwrite

% Create some data
F = rand(5);

% Save data to a CSV file
csvwrite('myData.csv', F);

Example: Using writematrix

% Create some data
G = rand(5);

% Save data to a CSV file
writematrix(G, 'myData.csv');

2.2 Exporting to Other Spreadsheet Formats

MATLAB supports various spreadsheet formats beyond Excel.

  • writetable Function: Writes table data to a file, supporting different formats like Excel, CSV, and text.

Example: Using writetable

% Create a table
T = table(rand(5,1), rand(5,1), 'VariableNames', {'Column1', 'Column2'});

% Save table data to an Excel file
writetable(T, 'myTableData.xlsx');

% Save table data to a CSV file
writetable(T, 'myTableData.csv');

3. Advanced Data Export

3.1 Exporting to Custom Formats

For specialized applications, you may need to export data in custom formats.

  • fwrite Function: Writes binary data to a file.
  • jsonencode Function: Encodes MATLAB data into JSON format.
  • xmlwrite Function: Writes data to an XML file.

Example: Using fwrite

% Create some data
H = rand(5);

% Open a file for writing
fileID = fopen('myData.bin', 'w');

% Write binary data
fwrite(fileID, H, 'double');

% Close the file
fclose(fileID);

Example: Using jsonencode

% Create a structure
dataStruct = struct('Field1', rand(5,1), 'Field2', magic(5));

% Encode structure to JSON
jsonData = jsonencode(dataStruct);

% Write JSON data to a file
fid = fopen('myData.json', 'w');
fwrite(fid, jsonData, 'char');
fclose(fid);

Example: Using xmlwrite

% Create an XML document
docNode = com.mathworks.xml.XMLUtils.createDocument('root_element');
docRootNode = docNode.getDocumentElement;
docRootNode.appendChild(docNode.createElement('child_element'));

% Write XML data to a file
xmlwrite('myData.xml', docNode);

Conclusion

Exporting data in MATLAB is a versatile process that can be tailored to various needs, from simple text files to complex custom formats. By mastering these techniques, you can ensure your data is saved, shared, and utilized effectively in different environments.
