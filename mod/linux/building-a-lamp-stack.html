Introduction

A LAMP stack is a popular open-source web platform used to run dynamic websites and servers. LAMP stands for Linux, Apache, MySQL, and PHP. This section will guide you through the process of setting up a LAMP stack from scratch on a Linux server.

Prerequisites

Before we begin, ensure you have:

  • A Linux server (Ubuntu/Debian-based distributions are recommended for this tutorial).
  • Basic knowledge of the Linux command line.
  • Root or sudo access to the server.

Step 1: Update Your System

First, update your package index and upgrade your system packages to the latest versions.

sudo apt update
sudo apt upgrade -y

Step 2: Install Apache

Apache is the web server component of the LAMP stack.

Installing Apache

sudo apt install apache2 -y

Starting and Enabling Apache

Start the Apache service and enable it to start on boot.

sudo systemctl start apache2
sudo systemctl enable apache2

Verifying Apache Installation

Open your web browser and navigate to your server's IP address. You should see the Apache2 Ubuntu Default Page.

http://your_server_ip

Step 3: Install MySQL

MySQL is the database management system used in the LAMP stack.

Installing MySQL

sudo apt install mysql-server -y

Securing MySQL Installation

Run the security script to set up MySQL security options.

sudo mysql_secure_installation

Follow the prompts to set a root password and secure your MySQL installation.

Step 4: Install PHP

PHP is the scripting language used to create dynamic web pages.

Installing PHP

sudo apt install php libapache2-mod-php php-mysql -y

Testing PHP

Create a PHP info file to test your PHP installation.

sudo nano /var/www/html/info.php

Add the following content to the file:

<?php
phpinfo();
?>

Save and close the file. Then, navigate to http://your_server_ip/info.php in your web browser. You should see a page displaying PHP information.

Step 5: Configure Apache to Use PHP

By default, Apache will serve index.html files before index.php files. To change this, edit the dir.conf file.

sudo nano /etc/apache2/mods-enabled/dir.conf

Move the index.php directive to the beginning of the list.

<IfModule mod_dir.c>
    DirectoryIndex index.php index.html index.cgi index.pl index.xhtml index.htm
</IfModule>

Save and close the file, then restart Apache.

sudo systemctl restart apache2

Step 6: Testing the LAMP Stack

To ensure everything is working correctly, create a simple PHP script to connect to the MySQL database.

Creating a Test Database and User

Log in to MySQL and create a test database and user.

sudo mysql -u root -p

Inside the MySQL shell, run the following commands:

CREATE DATABASE testdb;
CREATE USER 'testuser'@'localhost' IDENTIFIED BY 'password';
GRANT ALL PRIVILEGES ON testdb.* TO 'testuser'@'localhost';
FLUSH PRIVILEGES;
EXIT;

Creating a PHP Script to Test Database Connection

Create a new PHP file to test the database connection.

sudo nano /var/www/html/testdb.php

Add the following content to the file:

<?php
$servername = "localhost";
$username = "testuser";
$password = "password";
$dbname = "testdb";

// Create connection
$conn = new mysqli($servername, $username, $password, $dbname);

// Check connection
if ($conn->connect_error) {
    die("Connection failed: " . $conn->connect_error);
}
echo "Connected successfully";
?>

Save and close the file. Navigate to http://your_server_ip/testdb.php in your web browser. You should see "Connected successfully".

Conclusion

Congratulations! You have successfully set up a LAMP stack on your Linux server. This stack provides a robust environment for developing and deploying dynamic web applications. You can now proceed to develop your web applications using PHP and MySQL, served by Apache on your Linux server.
