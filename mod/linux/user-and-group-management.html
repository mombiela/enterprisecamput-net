Introduction

User and group management is a fundamental aspect of Linux system administration. It involves creating, modifying, and deleting user accounts and groups, as well as managing permissions and access controls.

Key Concepts

  • Users: Individual accounts that can log into the system.
  • Groups: Collections of users that share common permissions.
  • UID (User ID): A unique identifier assigned to each user.
  • GID (Group ID): A unique identifier assigned to each group.
  • Home Directory: The default directory assigned to a user.
  • Shell: The command-line interface assigned to a user.

Managing Users

Creating a User

To create a new user, use the useradd command:

sudo useradd -m -s /bin/bash username
  • -m: Creates a home directory for the user.
  • -s: Specifies the user's default shell.

Setting a Password

Set a password for the new user using the passwd command:

sudo passwd username

Modifying a User

To modify an existing user, use the usermod command:

sudo usermod -s /bin/zsh username
  • -s: Changes the user's default shell.

Deleting a User

To delete a user, use the userdel command:

sudo userdel -r username
  • -r: Removes the user's home directory and mail spool.

Managing Groups

Creating a Group

To create a new group, use the groupadd command:

sudo groupadd groupname

Adding a User to a Group

To add a user to a group, use the usermod command with the -aG option:

sudo usermod -aG groupname username
  • -aG: Appends the user to the specified group.

Removing a User from a Group

To remove a user from a group, use the gpasswd command:

sudo gpasswd -d username groupname

Deleting a Group

To delete a group, use the groupdel command:

sudo groupdel groupname

Viewing User and Group Information

Listing Users

To list all users, view the /etc/passwd file:

cat /etc/passwd

Listing Groups

To list all groups, view the /etc/group file:

cat /etc/group

Checking Group Membership

To check which groups a user belongs to, use the groups command:

groups username

Practical Examples

Example: Creating a New User and Adding to a Group

  1. Create a new user: 
    sudo useradd -m -s /bin/bash alice
  2. Set a password for the new user: 
    sudo passwd alice
  3. Create a new group: 
    sudo groupadd developers
  4. Add the new user to the group: 
    sudo usermod -aG developers alice

Example: Modifying User Information

  1. Change the user's shell: 
    sudo usermod -s /bin/zsh alice

Example: Deleting a User and Group

  1. Delete the user and their home directory: 
    sudo userdel -r alice
  2. Delete the group: 
    sudo groupdel developers

Summary

User and group management in Linux is essential for maintaining system security and organization. By understanding how to create, modify, and delete users and groups, as well as manage their permissions, you can effectively control access to system resources. Practice these commands to become proficient in managing users and groups on a Linux system.
