Introduction to Functions

Functions are fundamental building blocks in Dart programming. They allow you to encapsulate a block of code that performs a specific task, which can then be reused throughout your program.

Key Concepts

  • Function Definition: How to define a function.
  • Function Invocation: How to call a function.
  • Parameters and Arguments: How to pass data to functions.
  • Return Values: How functions can return data.
  • Anonymous Functions: Functions without a name.
  • Arrow Functions: A concise way to write functions.
  • Higher-Order Functions: Functions that take other functions as parameters or return them.

Function Definition and Invocation

To define a function in Dart, you use the void keyword if the function does not return a value, or specify a return type if it does.

// Function that does not return a value
void greet() {
  print('Hello, World!');
}

// Function that returns a value
int add(int a, int b) {
  return a + b;
}

To call a function, simply use its name followed by parentheses.

void main() {
  greet(); // Output: Hello, World!
  int sum = add(3, 4);
  print(sum); // Output: 7
}

Parameters and Arguments

Functions can take parameters, which are specified in the function definition. When calling the function, you provide arguments.

void printMessage(String message) {
  print(message);
}

void main() {
  printMessage('Hello, Dart!'); // Output: Hello, Dart!
}

Optional Parameters

Dart supports optional parameters, which can be either positional or named.

Positional Optional Parameters

Use square brackets [] to define positional optional parameters.

void greet([String name = 'Guest']) {
  print('Hello, $name!');
}

void main() {
  greet(); // Output: Hello, Guest!
  greet('Alice'); // Output: Hello, Alice!
}

Named Optional Parameters

Use curly braces {} to define named optional parameters.

void greet({String name = 'Guest'}) {
  print('Hello, $name!');
}

void main() {
  greet(); // Output: Hello, Guest!
  greet(name: 'Alice'); // Output: Hello, Alice!
}

Return Values

Functions can return values using the return keyword.

double multiply(double a, double b) {
  return a * b;
}

void main() {
  double result = multiply(2.5, 4.0);
  print(result); // Output: 10.0
}

Anonymous Functions

Anonymous functions, also known as lambda or inline functions, are functions without a name. They are often used as arguments to higher-order functions.

void main() {
  var list = ['apple', 'banana', 'cherry'];
  list.forEach((item) {
    print(item);
  });
}

Arrow Functions

Arrow functions provide a concise syntax for functions that contain a single expression.

int square(int num) => num * num;

void main() {
  print(square(4)); // Output: 16
}

Higher-Order Functions

Higher-order functions are functions that take other functions as parameters or return them.

void printElement(int element) {
  print(element);
}

void main() {
  var list = [1, 2, 3, 4];
  list.forEach(printElement); // Output: 1 2 3 4
}

Conclusion

Functions in Dart are versatile and powerful tools that help you organize and reuse code efficiently. From basic function definitions to advanced concepts like higher-order functions, understanding how to work with functions is crucial for effective Dart programming. Practice defining, invoking, and utilizing different types of functions to become proficient in Dart.
