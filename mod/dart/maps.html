Introduction to Maps

Maps are a collection of key-value pairs where each key is unique. They are similar to dictionaries in other programming languages. Maps are useful when you need to associate values with keys and retrieve them efficiently.

  • Key Concepts:
    • Key: A unique identifier for a value.
    • Value: The data associated with a key.
    • Map: A collection of key-value pairs.

Creating Maps

You can create maps in Dart using the Map class or the map literal syntax.

Using Map Class

void main() {
  Map<String, int> ages = Map();
  ages['Alice'] = 25;
  ages['Bob'] = 30;
  print(ages); // Output: {Alice: 25, Bob: 30}
}

Using Map Literals

void main() {
  var ages = {
    'Alice': 25,
    'Bob': 30,
  };
  print(ages); // Output: {Alice: 25, Bob: 30}
}

Accessing Map Elements

You can access map elements using their keys.

void main() {
  var ages = {
    'Alice': 25,
    'Bob': 30,
  };
  print(ages['Alice']); // Output: 25
}

Modifying Maps

You can add, update, or remove elements from a map.

Adding Elements

void main() {
  var ages = {
    'Alice': 25,
  };
  ages['Bob'] = 30;
  print(ages); // Output: {Alice: 25, Bob: 30}
}

Updating Elements

void main() {
  var ages = {
    'Alice': 25,
    'Bob': 30,
  };
  ages['Alice'] = 26;
  print(ages); // Output: {Alice: 26, Bob: 30}
}

Removing Elements

void main() {
  var ages = {
    'Alice': 25,
    'Bob': 30,
  };
  ages.remove('Bob');
  print(ages); // Output: {Alice: 25}
}

Iterating Over Maps

You can iterate over the keys, values, or key-value pairs in a map.

Iterating Over Keys

void main() {
  var ages = {
    'Alice': 25,
    'Bob': 30,
  };
  for (var key in ages.keys) {
    print(key); // Output: Alice Bob
  }
}

Iterating Over Values

void main() {
  var ages = {
    'Alice': 25,
    'Bob': 30,
  };
  for (var value in ages.values) {
    print(value); // Output: 25 30
  }
}

Iterating Over Key-Value Pairs

void main() {
  var ages = {
    'Alice': 25,
    'Bob': 30,
  };
  for (var entry in ages.entries) {
    print('${entry.key}: ${entry.value}'); // Output: Alice: 25 Bob: 30
  }
}

Advanced Map Operations

Checking for Key or Value

You can check if a map contains a specific key or value.

void main() {
  var ages = {
    'Alice': 25,
    'Bob': 30,
  };
  print(ages.containsKey('Alice')); // Output: true
  print(ages.containsValue(30)); // Output: true
}

Merging Maps

You can merge two maps using the addAll method.

void main() {
  var ages1 = {
    'Alice': 25,
  };
  var ages2 = {
    'Bob': 30,
  };
  ages1.addAll(ages2);
  print(ages1); // Output: {Alice: 25, Bob: 30}
}

Conclusion

Maps are a powerful and flexible way to manage key-value pairs in Dart. They allow for efficient data retrieval and manipulation. Understanding how to create, access, modify, and iterate over maps is essential for effective Dart programming. By mastering these concepts, you can handle complex data structures and improve the performance of your applications.
