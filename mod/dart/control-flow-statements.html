Control Flow Statements in Dart
Functions ►

Control flow statements are fundamental in programming as they allow you to dictate the order in which statements are executed. In Dart, control flow statements include conditionals, loops, and branching statements. This section will cover these concepts from a beginner to an advanced level.

Conditionals

Conditionals are used to execute code based on certain conditions.

if and else Statements

The if statement allows you to execute a block of code if a condition is true. The else statement can be used to execute a block of code if the condition is false.

void main() {
  int number = 10;

  if (number > 0) {
    print('The number is positive.');
  } else {
    print('The number is not positive.');
  }
}

else if Statement

The else if statement allows you to check multiple conditions.

void main() {
  int number = 0;

  if (number > 0) {
    print('The number is positive.');
  } else if (number == 0) {
    print('The number is zero.');
  } else {
    print('The number is negative.');
  }
}

Ternary Operator

The ternary operator is a shorthand for if-else statements.

void main() {
  int number = 10;
  String result = number > 0 ? 'Positive' : 'Not positive';
  print(result);
}

Switch Case

The switch statement allows you to execute one block of code among many options.

void main() {
  String grade = 'A';

  switch (grade) {
    case 'A':
      print('Excellent');
      break;
    case 'B':
      print('Good');
      break;
    case 'C':
      print('Fair');
      break;
    case 'D':
      print('Poor');
      break;
    default:
      print('Fail');
  }
}

Loops

Loops are used to execute a block of code multiple times.

for Loop

The for loop is used when you know in advance how many times you want to execute a statement or a block of statements.

void main() {
  for (int i = 0; i < 5; i++) {
    print('Iteration $i');
  }
}

while Loop

The while loop is used when you want to execute a block of code as long as a condition is true.

void main() {
  int i = 0;

  while (i < 5) {
    print('Iteration $i');
    i++;
  }
}

do-while Loop

The do-while loop is similar to the while loop, but it guarantees that the block of code will be executed at least once.

void main() {
  int i = 0;

  do {
    print('Iteration $i');
    i++;
  } while (i < 5);
}

Branching Statements

Branching statements allow you to change the flow of execution.

break Statement

The break statement is used to exit a loop or a switch statement.

void main() {
  for (int i = 0; i < 5; i++) {
    if (i == 3) {
      break;
    }
    print('Iteration $i');
  }
}

continue Statement

The continue statement skips the current iteration of a loop and proceeds to the next iteration.

void main() {
  for (int i = 0; i < 5; i++) {
    if (i == 3) {
      continue;
    }
    print('Iteration $i');
  }
}

Conclusion

Control flow statements are essential for directing the execution of your code. Understanding how to use conditionals, loops, and branching statements will allow you to write more complex and efficient Dart programs. Practice these concepts with various examples to become proficient in controlling the flow of your Dart applications.
