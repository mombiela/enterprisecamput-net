Introduction

Flutter is an open-source UI software development toolkit created by Google. It is used to develop cross-platform applications for Android, iOS, Linux, Mac, Windows, Google Fuchsia, and the web from a single codebase. In this section, we will guide you through setting up Flutter on your machine.

Prerequisites

Before setting up Flutter, ensure you have the following:

  • A computer running macOS, Windows, or Linux.
  • Basic understanding of Dart programming language.
  • An IDE (Integrated Development Environment) such as Android Studio, IntelliJ IDEA, or Visual Studio Code.

Step-by-Step Setup

1. Download Flutter SDK

  • Visit the Flutter official website.
  • Choose your operating system (Windows, macOS, or Linux).
  • Download the Flutter SDK zip file.

2. Extract the Flutter SDK

  • Extract the downloaded zip file to a desired location on your system.
  • Windows: Extract to C:\src\flutter (or any other location).
  • macOS/Linux: Extract to your home directory or any other location.

3. Update Your Path

To run Flutter commands in the terminal, you need to add Flutter to your system PATH.

Windows

  1. Open the Start Search, type in "env", and select "Edit the system environment variables".
  2. In the System Properties window, click on the "Environment Variables" button.
  3. Under "System variables", find the Path variable, select it, and click "Edit".
  4. Click "New" and add the path to the flutter\bin directory (e.g., C:\src\flutter\bin).
  5. Click OK to save the changes.

macOS/Linux

  1. Open a terminal.
  2. Open your bash profile in a text editor: 
    nano ~/.bash_profile
  3. Add the following line to the file: 
    export PATH="$PATH:`pwd`/flutter/bin"
  4. Save the file and run: 
    source ~/.bash_profile

4. Verify Installation

Open a terminal and run the following command to verify that Flutter is installed correctly:

flutter doctor

This command checks your environment and displays a report of the status of your Flutter installation. The output should look something like this:

Doctor summary (to see all details, run flutter doctor -v):
[✓] Flutter (Channel stable, 2.5.3, on macOS 11.6 20G165 darwin-x64, locale en-US)
[✓] Android toolchain - develop for Android devices (Android SDK version 31.0.0)
[✓] Xcode - develop for iOS and macOS (Xcode 13.1)
[✓] Chrome - develop for the web
[✓] Android Studio (version 2020.3)
[✓] VS Code (version 1.61.2)
[✓] Connected device (2 available)

Ensure all checks are marked with a green checkmark. If there are any issues, follow the provided instructions to resolve them.

5. Set Up an Editor

Flutter supports various editors. Here, we'll cover setting up Visual Studio Code and Android Studio.

Visual Studio Code

  1. Install Visual Studio Code from here.
  2. Open Visual Studio Code.
  3. Go to the Extensions view by clicking the Extensions icon in the Activity Bar on the side of the window or by pressing Ctrl+Shift+X.
  4. Search for "Flutter" and install the Flutter extension by Dart Code.

Android Studio

  1. Download and install Android Studio from here.
  2. Open Android Studio.
  3. Go to File > Settings > Plugins.
  4. Search for "Flutter" and click "Install".
  5. Follow the prompts to install the Dart plugin as well.

6. Create a New Flutter Project

Once your editor is set up, you can create a new Flutter project.

Visual Studio Code

  1. Open Visual Studio Code.
  2. Press Ctrl+Shift+P to open the command palette.
  3. Type Flutter: New Project and select it.
  4. Follow the prompts to name your project and choose a location to save it.

Android Studio

  1. Open Android Studio.
  2. Click on Start a new Flutter project.
  3. Select Flutter Application and click Next.
  4. Follow the prompts to configure your new Flutter project.

7. Run Your First Flutter App

To run your Flutter app, follow these steps:

Visual Studio Code

  1. Open the main Dart file (lib/main.dart).
  2. Press F5 to start debugging.
  3. Select your target device (e.g., an Android emulator or a connected device).

Android Studio

  1. Open the main Dart file (lib/main.dart).
  2. Click the green play button in the toolbar.
  3. Select your target device (e.g., an Android emulator or a connected device).

Conclusion

Setting up Flutter involves downloading the SDK, updating your system PATH, verifying the installation, setting up an editor, creating a new project, and running your first app. By following these steps, you should now have a fully functional Flutter development environment. This setup will allow you to build and run Flutter applications seamlessly.
