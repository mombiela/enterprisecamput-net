Introduction

Dart is a versatile programming language that can be used for both client-side and server-side development. One of its powerful features is the ability to interact with HTML, making it an excellent choice for web development. In this section, we will explore how to use Dart with HTML, from basic concepts to advanced techniques.

Setting Up Dart for Web Development

Before we dive into using Dart with HTML, let's set up our environment.

  • Install Dart SDK: Ensure you have the Dart SDK installed on your machine.
  • Install Stagehand: A Dart project scaffolding tool. 
    pub global activate stagehand
  • Create a new Dart web project: 
    stagehand web-simple
cd <your_project_name>
pub get

Basic HTML Integration

Let's start with a simple example of integrating Dart with HTML.

HTML Structure

Create an index.html file with the following content:

<!DOCTYPE html>
<html>
<head>
  <title>Dart with HTML</title>
  <script defer src="main.dart.js"></script>
</head>
<body>
  <h1 id="title">Hello, Dart!</h1>
</body>
</html>

Dart Code

Create a main.dart file with the following content:

import 'dart:html';

void main() {
  querySelector('#title')?.text = 'Hello, Dart and HTML!';
}

Explanation

  • HTML: The HTML file includes a script tag that points to the compiled Dart code (main.dart.js).
  • Dart: The Dart code uses querySelector to select the HTML element with the id title and changes its text content.

Interacting with HTML Elements

Dart provides a rich set of APIs to interact with HTML elements.

Selecting Elements

  • By ID: 
    var element = querySelector('#elementId');
  • By Class: 
    var elements = querySelectorAll('.className');

Modifying Elements

  • Changing Text: 
    querySelector('#elementId')?.text = 'New Text';
  • Changing Attributes: 
    querySelector('#elementId')?.setAttribute('attributeName', 'value');

Example

import 'dart:html';

void main() {
  var button = querySelector('#myButton');
  button?.onClick.listen((event) {
    querySelector('#output')?.text = 'Button clicked!';
  });
}

HTML

<!DOCTYPE html>
<html>
<head>
  <title>Dart with HTML</title>
  <script defer src="main.dart.js"></script>
</head>
<body>
  <button id="myButton">Click Me</button>
  <div id="output"></div>
</body>
</html>

Event Handling

Handling events in Dart is straightforward and similar to JavaScript.

Example: Click Event

import 'dart:html';

void main() {
  var button = querySelector('#myButton');
  button?.onClick.listen((event) {
    querySelector('#output')?.text = 'Button clicked!';
  });
}

Explanation

  • Event Listener: The onClick.listen method attaches an event listener to the button, which updates the text of the output div when the button is clicked.

Advanced Techniques

Using Dart with Forms

Handling forms is a common task in web development.

Example: Form Submission

import 'dart:html';

void main() {
  var form = querySelector('#myForm');
  form?.onSubmit.listen((event) {
    event.preventDefault(); // Prevent the default form submission
    var input = (querySelector('#name') as InputElement).value;
    querySelector('#output')?.text = 'Hello, $input!';
  });
}

HTML

<!DOCTYPE html>
<html>
<head>
  <title>Dart with HTML</title>
  <script defer src="main.dart.js"></script>
</head>
<body>
  <form id="myForm">
    <input type="text" id="name" placeholder="Enter your name">
    <button type="submit">Submit</button>
  </form>
  <div id="output"></div>
</body>
</html>

Explanation

  • Form Handling: The onSubmit.listen method attaches an event listener to the form, which prevents the default form submission and updates the output div with the input value.

Conclusion

Using Dart with HTML allows you to create dynamic and interactive web applications. By understanding how to select and manipulate HTML elements, handle events, and work with forms, you can leverage Dart's capabilities to build robust web applications. As you progress, you can explore more advanced features and libraries to enhance your Dart web development skills.
