Introduction

Welcome to your first Dart programming lesson! Dart is a client-optimized language for fast apps on any platform. In this section, you will learn how to write, run, and understand a basic Dart program. By the end of this lesson, you will have a solid foundation to build upon as you progress through the course.

Setting Up Your Environment

Before writing your first Dart program, you need to set up your development environment. Follow these steps:

  1. Install Dart SDK: Download and install the Dart SDK from the official Dart website.
  2. Choose an IDE: You can use any text editor, but IDEs like Visual Studio Code or IntelliJ IDEA offer great support for Dart.
  3. Install Dart Plugin: If you are using an IDE, make sure to install the Dart plugin for syntax highlighting and other features.

Writing Your First Dart Program

Let's write a simple Dart program that prints "Hello, World!" to the console.

Code Example

Create a new file named main.dart and add the following code:

void main() {
  print('Hello, World!');
}

Explanation

  • void main() { ... }: This is the main function, the entry point of every Dart application. The void keyword indicates that the function does not return a value.
  • print('Hello, World!');: This line prints the string 'Hello, World!' to the console. The print function is a built-in function in Dart used for output.

Running Your Dart Program

To run your Dart program, open a terminal or command prompt and navigate to the directory where your main.dart file is located. Then, execute the following command:

dart run main.dart

You should see the output:

Hello, World!

Key Concepts

  • Main Function: The starting point of a Dart program.
  • Print Function: Used to output text to the console.
  • Comments: Use // for single-line comments and /* ... */ for multi-line comments.

Example with Comments

// This is a single-line comment

  // Print a greeting to the console
  print('Hello, World!');

Summary

In this lesson, you have learned how to:

  • Set up your Dart development environment.
  • Write a simple Dart program.
  • Understand the structure of a basic Dart program.
  • Run your Dart program and see the output.

Congratulations on writing your first Dart program! This foundational knowledge will be crucial as you move on to more complex topics. Keep practicing, and soon you'll be building more advanced applications with Dart.
