Introduction to Classes and Objects

In Dart, classes are the blueprint for creating objects. An object is an instance of a class, and it can have properties (variables) and methods (functions) that define its behavior.

Key Concepts

  • Class: A template for creating objects.
  • Object: An instance of a class.
  • Constructor: A special method used to initialize objects.
  • Properties: Variables that belong to an object.
  • Methods: Functions that belong to an object.

Defining a Class

To define a class in Dart, use the class keyword followed by the class name and a pair of curly braces {}.

class Animal {
  // Properties
  String name;
  int age;

  // Constructor
  Animal(this.name, this.age);

  // Method
  void displayInfo() {
    print('Name: $name, Age: $age');
  }
}

Explanation

  • Properties: name and age are properties of the Animal class.
  • Constructor: Animal(this.name, this.age); initializes the properties.
  • Method: displayInfo() is a method that prints the object's information.

Creating Objects

To create an object, use the new keyword followed by the class name and the required parameters.

void main() {
  // Creating an object
  Animal dog = new Animal('Buddy', 3);

  // Accessing properties
  print(dog.name); // Output: Buddy

  // Calling methods
  dog.displayInfo(); // Output: Name: Buddy, Age: 3
}

Explanation

  • Object Creation: Animal dog = new Animal('Buddy', 3); creates an object named dog.
  • Accessing Properties: dog.name accesses the name property.
  • Calling Methods: dog.displayInfo() calls the displayInfo method.

Advanced Concepts

Inheritance

Inheritance allows a class to inherit properties and methods from another class.

class Dog extends Animal {
  String breed;

  Dog(String name, int age, this.breed) : super(name, age);

  void displayBreed() {
    print('Breed: $breed');
  }
}

Explanation

  • Inheritance: class Dog extends Animal means Dog inherits from Animal.
  • Constructor: Dog(String name, int age, this.breed) : super(name, age); calls the parent class constructor.

Polymorphism

Polymorphism allows methods to do different things based on the object it is acting upon.

void main() {
  Animal myDog = new Dog('Buddy', 3, 'Golden Retriever');
  myDog.displayInfo(); // Output: Name: Buddy, Age: 3
  // myDog.displayBreed(); // This will cause an error
}

Explanation

  • Polymorphism: Animal myDog = new Dog('Buddy', 3, 'Golden Retriever'); creates a Dog object but treats it as an Animal.

Encapsulation

Encapsulation restricts direct access to some of an object's components.

class Cat {
  String _name; // Private property

  Cat(this._name);

  String get name => _name; // Getter

  set name(String name) {
    _name = name;
  }
}

Explanation

  • Private Property: _name is private to the Cat class.
  • Getter and Setter: get name and set name provide controlled access to _name.

Summary

  • Classes are blueprints for creating objects.
  • Objects are instances of classes with properties and methods.
  • Constructors initialize objects.
  • Inheritance allows a class to inherit from another class.
  • Polymorphism enables methods to act differently based on the object.
  • Encapsulation restricts direct access to object properties.

Understanding these concepts is crucial for mastering object-oriented programming in Dart. Practice creating classes, objects, and using inheritance to build a solid foundation.
