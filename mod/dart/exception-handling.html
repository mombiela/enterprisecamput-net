Introduction to Exception Handling

Exception handling is a crucial aspect of robust software development. It allows developers to manage and respond to unexpected events or errors that occur during the execution of a program. In Dart, exception handling is performed using try, catch, and finally blocks.

Key Concepts

  • Exception: An event that disrupts the normal flow of a program.
  • try block: Contains code that might throw an exception.
  • catch block: Handles the exception.
  • finally block: Executes code regardless of whether an exception was thrown or not.
  • throw: Used to raise an exception.

Basic Exception Handling

In Dart, you can handle exceptions using try and catch blocks.

Example

void main() {
  try {
    int result = 12 ~/ 0; // This will throw an exception
    print(result);
  } catch (e) {
    print('An error occurred: $e');
  }
}
  • Explanation:
    • The try block contains code that might throw an exception.
    • The catch block captures the exception and prints an error message.

Catching Specific Exceptions

You can catch specific types of exceptions to handle them differently.

Example

void main() {
  try {
    int result = 12 ~/ 0; // This will throw an IntegerDivisionByZeroException
    print(result);
  } on IntegerDivisionByZeroException {
    print('Cannot divide by zero!');
  } catch (e) {
    print('An error occurred: $e');
  }
}
  • Explanation:
    • The on keyword is used to catch specific exceptions.
    • If the exception is of type IntegerDivisionByZeroException, a specific message is printed.
    • The catch block handles any other types of exceptions.

Using finally Block

The finally block contains code that will always execute, regardless of whether an exception was thrown or not.

Example

void main() {
  try {
    int result = 12 ~/ 0;
    print(result);
  } catch (e) {
    print('An error occurred: $e');
  } finally {
    print('This code always runs.');
  }
}
  • Explanation:
    • The finally block executes after the try and catch blocks, ensuring that the code within it always runs.

Throwing Exceptions

You can throw exceptions using the throw keyword.

Example

void checkAge(int age) {
  if (age < 18) {
    throw Exception('Age must be 18 or older.');
  }
}

void main() {
  try {
    checkAge(16);
  } catch (e) {
    print('An error occurred: $e');
  }
}
  • Explanation:
    • The checkAge function throws an exception if the age is less than 18.
    • The try block calls checkAge, and the catch block handles the exception.

Custom Exceptions

You can create custom exceptions by extending the Exception class.

Example

class AgeException implements Exception {
  String cause;
  AgeException(this.cause);
}

void checkAge(int age) {
  if (age < 18) {
    throw AgeException('Age must be 18 or older.');
  }
}

void main() {
  try {
    checkAge(16);
  } catch (e) {
    print('An error occurred: $e');
  }
}
  • Explanation:
    • The AgeException class implements the Exception interface.
    • The checkAge function throws an AgeException if the age is less than 18.
    • The catch block handles the custom exception.

Summary

Exception handling in Dart is a powerful feature that helps manage errors and unexpected events in your code. By using try, catch, finally, and throw, you can ensure that your programs are robust and can handle various error conditions gracefully. Additionally, creating custom exceptions allows for more specific error handling tailored to your application's needs.
