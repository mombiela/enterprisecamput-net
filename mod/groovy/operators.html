Introduction

Operators are special symbols or keywords that are used to perform operations on operands. In Groovy, operators are similar to those in Java, but with some enhancements and additional features. This section will cover the various types of operators available in Groovy, from basic arithmetic to more advanced operators.

Arithmetic Operators

Arithmetic operators are used to perform basic mathematical operations.

  • Addition (+)
  • Subtraction (-)
  • Multiplication (*)
  • Division (/)
  • Modulus (%)

Examples

def a = 10
def b = 5

println a + b  // Output: 15
println a - b  // Output: 5
println a * b  // Output: 50
println a / b  // Output: 2
println a % b  // Output: 0

Relational Operators

Relational operators are used to compare two values.

  • Equal to (==)
  • Not equal to (!=)
  • Greater than (>)
  • Less than (<)
  • Greater than or equal to (>=)
  • Less than or equal to (<=)

Examples

def a = 10
def b = 5

println a == b  // Output: false
println a != b  // Output: true
println a > b   // Output: true
println a < b   // Output: false
println a >= b  // Output: true
println a <= b  // Output: false

Logical Operators

Logical operators are used to combine multiple boolean expressions.

  • AND (&&)
  • OR (||)
  • NOT (!)

Examples

def a = true
def b = false

println a && b  // Output: false
println a || b  // Output: true
println !a      // Output: false

Bitwise Operators

Bitwise operators are used to perform operations on individual bits of integer values.

  • AND (&)
  • OR (|)
  • XOR (^)
  • NOT (~)
  • Left shift (<<)
  • Right shift (>>)
  • Unsigned right shift (>>>)

Examples

def a = 5  // Binary: 0101
def b = 3  // Binary: 0011

println a & b  // Output: 1  (Binary: 0001)
println a | b  // Output: 7  (Binary: 0111)
println a ^ b  // Output: 6  (Binary: 0110)
println ~a     // Output: -6 (Binary: 1010, two's complement)
println a << 1 // Output: 10 (Binary: 1010)
println a >> 1 // Output: 2  (Binary: 0010)
println a >>> 1 // Output: 2 (Binary: 0010)

Assignment Operators

Assignment operators are used to assign values to variables.

  • Simple assignment (=)
  • Addition assignment (+=)
  • Subtraction assignment (-=)
  • Multiplication assignment (*=)
  • Division assignment (/=)
  • Modulus assignment (%=)

Examples

def a = 10

a += 5  // Equivalent to a = a + 5
println a  // Output: 15

a -= 3  // Equivalent to a = a - 3
println a  // Output: 12

a *= 2  // Equivalent to a = a * 2
println a  // Output: 24

a /= 4  // Equivalent to a = a / 4
println a  // Output: 6

a %= 5  // Equivalent to a = a % 5
println a  // Output: 1

Conclusion

Operators are fundamental to performing various operations in Groovy. Understanding how to use arithmetic, relational, logical, bitwise, and assignment operators will enable you to write more efficient and effective code. Practice using these operators in different scenarios to become more proficient in Groovy programming.
