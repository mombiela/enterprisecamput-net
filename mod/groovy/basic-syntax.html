Introduction

Groovy is an object-oriented programming language for the Java platform. It is dynamically compiled to Java bytecode and integrates seamlessly with Java. Understanding the basic syntax of Groovy is essential for writing effective and efficient code. This section will cover the fundamental elements of Groovy syntax, including variables, data types, operators, and control structures.

Variables and Data Types

Groovy is dynamically typed, meaning you don't have to explicitly declare the type of a variable. However, you can still use static typing if you prefer.

Variable Declaration

  • Dynamic Typing: You can declare a variable without specifying its type.
  • Static Typing: You can declare a variable with a specific type.
// Dynamic Typing
def name = "Groovy"
def age = 25

// Static Typing
String language = "Groovy"
int year = 2023

Data Types

Groovy supports various data types, including:

  • Numbers: Integer, Float, Double, BigDecimal
  • Strings: Single-quoted, Double-quoted, Triple-quoted
  • Booleans: true, false
  • Collections: Lists, Maps, Ranges
// Numbers
int num = 10
float pi = 3.14
BigDecimal bigNum = 12345678901234567890.12345678901234567890

// Strings
String singleQuote = 'Hello'
String doubleQuote = "World"
String tripleQuote = """This is a
multi-line string"""

// Booleans
boolean isGroovy = true

// Collections
List<Integer> numbers = [1, 2, 3, 4, 5]
Map<String, Integer> ageMap = ['Alice': 30, 'Bob': 25]
Range<Integer> range = 1..5

Operators

Groovy supports a wide range of operators, including arithmetic, relational, logical, and bitwise operators.

Arithmetic Operators

  • Addition (+)
  • Subtraction (-)
  • Multiplication (*)
  • Division (/)
  • Modulus (%)
int a = 10
int b = 5

println a + b  // 15
println a - b  // 5
println a * b  // 50
println a / b  // 2
println a % b  // 0

Relational Operators

  • Equal to (==)
  • Not equal to (!=)
  • Greater than (>)
  • Less than (<)
  • Greater than or equal to (>=)
  • Less than or equal to (<=)
int x = 10
int y = 20

println x == y  // false
println x != y  // true
println x > y   // false
println x < y   // true
println x >= y  // false
println x <= y  // true

Logical Operators

  • AND (&&)
  • OR (||)
  • NOT (!)
boolean isTrue = true
boolean isFalse = false

println isTrue && isFalse  // false
println isTrue || isFalse  // true
println !isTrue            // false

Control Structures

Groovy provides several control structures for managing the flow of the program, including conditional statements and loops.

Conditional Statements

  • if-else: Executes a block of code if a condition is true, otherwise executes another block of code.
int score = 85

if (score >= 90) {
    println "A"
} else if (score >= 80) {
    println "B"
} else {
    println "C"
}

Loops

  • for loop: Iterates over a range, list, or map.
  • while loop: Repeats a block of code while a condition is true.
// for loop
for (int i = 1; i <= 5; i++) {
    println i
}

// while loop
int count = 1
while (count <= 5) {
    println count
    count++
}

Conclusion

Understanding the basic syntax of Groovy is crucial for writing effective code. This section covered variable declaration, data types, operators, and control structures. With these fundamentals, you can start writing simple Groovy programs and gradually move on to more complex tasks. In the next sections, we will delve deeper into more advanced features and capabilities of Groovy.
