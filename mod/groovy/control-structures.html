Control Structures in Groovy
Lists ►

Control structures are fundamental to programming as they allow you to dictate the flow of your program. In Groovy, control structures are similar to those in Java but with some enhancements and syntactic sugar that make them more concise and expressive.

Conditional Statements

`if` and `else`

The if and else statements are used to execute code blocks based on a condition.

def age = 18

if (age >= 18) {
    println "You are an adult."
} else {
    println "You are a minor."
}
  • Key Concepts:
    • The condition inside the if statement must evaluate to a boolean.
    • The else block is optional and executes if the if condition is false.

`else if`

The else if statement allows you to check multiple conditions.

def score = 85

if (score >= 90) {
    println "Grade: A"
} else if (score >= 80) {
    println "Grade: B"
} else if (score >= 70) {
    println "Grade: C"
} else {
    println "Grade: F"
}
  • Key Concepts:
    • Multiple else if blocks can be chained together.
    • The first condition that evaluates to true will execute its block.

Switch Statement

The switch statement in Groovy is more powerful than in Java, allowing for more flexible case values.

def fruit = "Apple"

switch (fruit) {
    case "Apple":
        println "It's an apple."
        break
    case "Banana":
        println "It's a banana."
        break
    default:
        println "Unknown fruit."
}
  • Key Concepts:
    • The switch statement evaluates the expression and matches it against the case values.
    • The break statement is used to exit the switch block.
    • The default case is optional and executes if no other case matches.

Looping Structures

`for` Loop

The for loop in Groovy can iterate over a range, list, or any iterable object.

for (i in 1..5) {
    println "Iteration: $i"
}
  • Key Concepts:
    • The in keyword is used to iterate over a range or collection.
    • The loop variable (i in this case) takes on each value in the range or collection.

`while` Loop

The while loop continues to execute as long as its condition is true.

def count = 1

while (count <= 5) {
    println "Count: $count"
    count++
}
  • Key Concepts:
    • The condition is evaluated before each iteration.
    • If the condition is false initially, the loop body will not execute.

`do-while` Loop

The do-while loop is similar to the while loop but guarantees that the loop body will execute at least once.

def count = 1

do {
    println "Count: $count"
    count++
} while (count <= 5)
  • Key Concepts:
    • The loop body executes before the condition is checked.
    • Useful when the loop body needs to execute at least once regardless of the condition.

Enhanced Looping with Groovy

`each` Method

Groovy provides the each method for iterating over collections in a more expressive way.

def list = [1, 2, 3, 4, 5]

list.each { item ->
    println "Item: $item"
}
  • Key Concepts:
    • The each method takes a closure that is executed for each item in the collection.
    • The loop variable (item in this case) is defined within the closure.

`collect` Method

The collect method transforms each item in a collection and returns a new collection.

def list = [1, 2, 3, 4, 5]
def squaredList = list.collect { it * it }

println squaredList  // Output: [1, 4, 9, 16, 25]
  • Key Concepts:
    • The collect method takes a closure that transforms each item.
    • The result is a new collection with the transformed items.

Conclusion

Control structures in Groovy provide powerful and flexible ways to control the flow of your program. From basic conditional statements and loops to enhanced methods like each and collect, Groovy offers a rich set of tools to make your code more expressive and concise. Understanding these control structures is essential for writing effective Groovy programs.
