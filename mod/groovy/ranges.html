Ranges in Groovy
Closures ►

Ranges are a powerful and versatile feature in Groovy, allowing you to create sequences of values with ease. They are commonly used for iteration, slicing collections, and more. This topic will cover the basics of ranges, their usage, and advanced features.

What is a Range?

A range in Groovy is a sequence of values defined by a start and an end value. Ranges can be inclusive or exclusive, and they can be used with numbers, characters, and even custom objects.

Key Concepts

  • Inclusive Ranges: Include both the start and end values.
  • Exclusive Ranges: Exclude the end value.
  • Step Values: Define the increment between values in the range.
  • Reverse Ranges: Define ranges in descending order.

Creating Ranges

Inclusive Ranges

An inclusive range includes both the start and end values. You can create an inclusive range using the .. operator.

def inclusiveRange = 1..5
println inclusiveRange // Output: [1, 2, 3, 4, 5]

Exclusive Ranges

An exclusive range excludes the end value. You can create an exclusive range using the ..< operator.

def exclusiveRange = 1..<5
println exclusiveRange // Output: [1, 2, 3, 4]

Character Ranges

Ranges can also be created with characters.

def charRange = 'a'..'e'
println charRange // Output: [a, b, c, d, e]

Reverse Ranges

Ranges can be defined in reverse order.

def reverseRange = 5..1
println reverseRange // Output: [5, 4, 3, 2, 1]

Using Ranges

Iteration

Ranges are often used in loops for iteration.

for (i in 1..5) {
    println i
}
// Output: 
// 1
// 2
// 3
// 4
// 5

Slicing Collections

Ranges can be used to slice lists and arrays.

def list = [10, 20, 30, 40, 50]
def sublist = list[1..3]
println sublist // Output: [20, 30, 40]

Step Values

You can define the step value for a range using the step method.

def steppedRange = 1..10
println steppedRange.step(2) // Output: [1, 3, 5, 7, 9]

Advanced Features

Custom Object Ranges

You can create ranges with custom objects by implementing the next and previous methods.

class CustomRange {
    int value

    CustomRange next() {
        return new CustomRange(value: value + 1)
    }

    CustomRange previous() {
        return new CustomRange(value: value - 1)
    }

    String toString() {
        return value.toString()
    }
}

def customRange = new CustomRange(value: 1)..new CustomRange(value: 5)
println customRange // Output: [1, 2, 3, 4, 5]

Checking Membership

You can check if a value is within a range using the contains method or the in keyword.

def range = 1..5
println range.contains(3) // Output: true
println 3 in range        // Output: true
println 6 in range        // Output: false

Conclusion

Ranges in Groovy are a powerful tool for creating sequences of values, iterating, and slicing collections. They can be inclusive or exclusive, and they support custom objects and step values. Understanding how to use ranges effectively can greatly enhance your Groovy programming skills. Practice creating and using ranges in different contexts to become more comfortable with this feature.
