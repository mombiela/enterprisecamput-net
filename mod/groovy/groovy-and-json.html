Introduction to JSON

  • JSON (JavaScript Object Notation) is a lightweight data interchange format.
  • It is easy for humans to read and write and easy for machines to parse and generate.
  • JSON is often used for transmitting data in web applications.

Example of JSON

{
  "name": "John Doe",
  "age": 30,
  "isStudent": false,
  "courses": ["Math", "Science", "History"]
}

Working with JSON in Groovy

Groovy provides built-in support for JSON through the groovy.json package. This package includes classes like JsonSlurper and JsonOutput to parse and generate JSON data.

Parsing JSON with JsonSlurper

  • JsonSlurper is used to parse JSON strings into Groovy objects.

Example: Parsing JSON

import groovy.json.JsonSlurper

def jsonString = '''
{
  "name": "John Doe",
  "age": 30,
  "isStudent": false,
  "courses": ["Math", "Science", "History"]
}
'''

def jsonSlurper = new JsonSlurper()
def jsonObject = jsonSlurper.parseText(jsonString)

println jsonObject.name  // Output: John Doe
println jsonObject.age   // Output: 30
println jsonObject.isStudent  // Output: false
println jsonObject.courses  // Output: [Math, Science, History]

Generating JSON with JsonOutput

  • JsonOutput is used to convert Groovy objects into JSON strings.

Example: Generating JSON

import groovy.json.JsonOutput

def data = [
  name: "Jane Doe",
  age: 25,
  isStudent: true,
  courses: ["English", "Art", "Physics"]
]

def jsonString = JsonOutput.toJson(data)
println jsonString

Pretty Printing JSON

  • JsonOutput.prettyPrint can be used to format JSON strings for better readability.

Example: Pretty Printing JSON

import groovy.json.JsonOutput

def data = [
  name: "Jane Doe",
  age: 25,
  isStudent: true,
  courses: ["English", "Art", "Physics"]
]

def jsonString = JsonOutput.toJson(data)
def prettyJsonString = JsonOutput.prettyPrint(jsonString)
println prettyJsonString

Handling Complex JSON Structures

  • Groovy can handle nested and complex JSON structures with ease.

Example: Parsing Complex JSON

import groovy.json.JsonSlurper

def jsonString = '''
{
  "person": {
    "name": "John Doe",
    "age": 30,
    "address": {
      "street": "123 Main St",
      "city": "Springfield",
      "zip": "12345"
    },
    "courses": ["Math", "Science", "History"]
  }
}
'''

def jsonSlurper = new JsonSlurper()
def jsonObject = jsonSlurper.parseText(jsonString)

println jsonObject.person.name  // Output: John Doe
println jsonObject.person.address.city  // Output: Springfield
println jsonObject.person.courses  // Output: [Math, Science, History]

Summary

  • JSON is a widely-used format for data interchange.
  • Groovy provides robust support for JSON through the groovy.json package.
  • JsonSlurper is used for parsing JSON strings into Groovy objects.
  • JsonOutput is used for generating JSON strings from Groovy objects.
  • Groovy can handle both simple and complex JSON structures efficiently.

By mastering these tools, you can effectively work with JSON data in your Groovy applications, making data interchange seamless and efficient.
