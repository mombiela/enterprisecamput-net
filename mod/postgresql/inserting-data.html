In this section, we will cover how to insert data into PostgreSQL tables. This is a fundamental skill for any database user, as it allows you to populate your tables with the necessary data for your applications. We will start with basic insertions and move towards more advanced techniques.

Basic Insertions

The most straightforward way to insert data into a PostgreSQL table is using the INSERT INTO statement.

Syntax

INSERT INTO table_name (column1, column2, column3, ...)
VALUES (value1, value2, value3, ...);

Example

Let's assume we have a table named students with the following structure:

CREATE TABLE students (
    id SERIAL PRIMARY KEY,
    name VARCHAR(100),
    age INT,
    grade VARCHAR(10)
);

To insert a new student into this table, you would use the following SQL statement:

INSERT INTO students (name, age, grade)
VALUES ('John Doe', 20, 'A');

Explanation

  • INSERT INTO students: Specifies the table where the data will be inserted.
  • (name, age, grade): Lists the columns that will receive the data.
  • VALUES ('John Doe', 20, 'A'): Provides the actual data to be inserted into the specified columns.

Inserting Multiple Rows

You can insert multiple rows in a single INSERT INTO statement by separating each row's values with a comma.

Example

INSERT INTO students (name, age, grade)
VALUES 
    ('Jane Smith', 22, 'B'),
    ('Emily Johnson', 21, 'A'),
    ('Michael Brown', 23, 'C');

Explanation

  • Each set of values within the VALUES clause corresponds to a new row in the students table.

Inserting Data from Another Table

You can also insert data into a table by selecting data from another table. This is useful for copying data or transforming data from one table to another.

Syntax

INSERT INTO target_table (column1, column2, ...)
SELECT column1, column2, ...
FROM source_table
WHERE condition;

Example

Assume we have another table new_students with the same structure as students. To copy all data from new_students to students, you would use:

INSERT INTO students (name, age, grade)
SELECT name, age, grade
FROM new_students;

Explanation

  • SELECT name, age, grade FROM new_students: Selects the data from the new_students table.
  • INSERT INTO students (name, age, grade): Inserts the selected data into the students table.

Using Default Values

If a column has a default value, you can omit it in the INSERT INTO statement. PostgreSQL will automatically use the default value for that column.

Example

Assume the grade column in the students table has a default value of 'B'. You can insert a new student without specifying the grade:

INSERT INTO students (name, age)
VALUES ('Alice Green', 19);

Explanation

  • The grade column will automatically be set to 'B' for the new row.

Inserting JSON Data

PostgreSQL supports JSON data types, allowing you to insert JSON data directly into a table.

Example

Assume we have a table student_info with a JSON column:

CREATE TABLE student_info (
    id SERIAL PRIMARY KEY,
    info JSON
);

To insert JSON data into this table:

INSERT INTO student_info (info)
VALUES ('{"name": "Tom White", "age": 24, "grade": "A"}');

Explanation

  • The info column stores the JSON object as a single value.

Conclusion

Inserting data into PostgreSQL tables is a fundamental operation that can be performed in various ways depending on your needs. Whether you are inserting single rows, multiple rows, copying data from another table, or working with JSON data, PostgreSQL provides flexible and powerful tools to manage your data efficiently. Understanding these techniques will help you effectively populate and manage your databases.
