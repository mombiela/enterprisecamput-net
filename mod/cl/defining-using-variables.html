Introduction

In this section, we will cover the basics of defining and using variables in Control Language (CL). Variables are essential for storing data that can be manipulated and used throughout your CL programs. We will start with basic concepts and gradually move to more advanced topics.

What is a Variable?

  • A variable is a storage location identified by a name (or identifier) that holds data which can be modified during program execution.
  • Variables in CL can store different types of data, such as numeric values, strings, and more.

Defining Variables

To define a variable in CL, you use the DCL (Declare) command. Here is the basic syntax:

DCL VAR(&VariableName) TYPE(*DataType) LEN(length)

Example:

DCL VAR(&NAME) TYPE(*CHAR) LEN(20)
DCL VAR(&AGE) TYPE(*DEC) LEN(3 0)
  • &NAME is a character variable with a length of 20.
  • &AGE is a decimal variable with a length of 3 digits and 0 decimal places.

Key Concepts:

  • VAR: Specifies the name of the variable.
  • TYPE: Specifies the type of data the variable will hold. Common types include:
    • *CHAR: Character string.
    • *DEC: Decimal number.
    • *INT: Integer.
  • LEN: Specifies the length of the variable. For decimal types, it includes the total number of digits and the number of decimal places.

Assigning Values to Variables

Once a variable is defined, you can assign values to it using the CHGVAR (Change Variable) command.

Example:

CHGVAR VAR(&NAME) VALUE('John Doe')
CHGVAR VAR(&AGE) VALUE(30)
  • &NAME is assigned the value 'John Doe'.
  • &AGE is assigned the value 30.

Key Concepts:

  • CHGVAR: Command used to change the value of a variable.
  • VALUE: Specifies the new value to be assigned to the variable.

Using Variables in Expressions

Variables can be used in expressions to perform calculations or concatenate strings.

Example:

DCL VAR(&SALARY) TYPE(*DEC) LEN(7 2)
DCL VAR(&BONUS) TYPE(*DEC) LEN(7 2)
DCL VAR(&TOTAL) TYPE(*DEC) LEN(7 2)

CHGVAR VAR(&SALARY) VALUE(50000.00)
CHGVAR VAR(&BONUS) VALUE(5000.00)
CHGVAR VAR(&TOTAL) VALUE(&SALARY + &BONUS)
  • &TOTAL is calculated by adding &SALARY and &BONUS.

Conditional Statements with Variables

Variables are often used in conditional statements to control the flow of the program.

Example:

DCL VAR(&AGE) TYPE(*DEC) LEN(3 0)

CHGVAR VAR(&AGE) VALUE(20)

IF COND(&AGE *LT 18) THEN(DO)
    /* Code for under 18 */
    SNDPGMMSG MSG('Under 18')
ENDDO
ELSE DO
    /* Code for 18 and over */
    SNDPGMMSG MSG('18 or older')
ENDDO
  • The IF statement checks if &AGE is less than 18 and executes the corresponding block of code.

Advanced Variable Usage

Arrays

Arrays are used to store multiple values in a single variable.

Example:

DCL VAR(&NAMES) TYPE(*CHAR) LEN(20) DIM(5)

CHGVAR VAR(&NAMES(1)) VALUE('Alice')
CHGVAR VAR(&NAMES(2)) VALUE('Bob')
CHGVAR VAR(&NAMES(3)) VALUE('Charlie')
CHGVAR VAR(&NAMES(4)) VALUE('Diana')
CHGVAR VAR(&NAMES(5)) VALUE('Eve')
  • &NAMES is an array of 5 character strings, each with a length of 20.

Data Structures

Data structures allow you to group related variables together.

Example:

DCL VAR(&EMPLOYEE) TYPE(*CHAR) LEN(50) 
DCL VAR(&EMPID) TYPE(*CHAR) LEN(10) STG(*DEFINED) DEFVAR(&EMPLOYEE 1)
DCL VAR(&EMPNAME) TYPE(*CHAR) LEN(40) STG(*DEFINED) DEFVAR(&EMPLOYEE 11)

CHGVAR VAR(&EMPID) VALUE('E12345')
CHGVAR VAR(&EMPNAME) VALUE('John Smith')
  • &EMPLOYEE is a data structure containing &EMPID and &EMPNAME.

Conclusion

In this section, we covered the basics of defining and using variables in CL, including:

  • Declaring variables with the DCL command.
  • Assigning values using the CHGVAR command.
  • Using variables in expressions and conditional statements.
  • Advanced usage with arrays and data structures.

Understanding these concepts is crucial for writing effective CL programs. Practice by creating and manipulating variables in your own CL scripts to reinforce these concepts.
