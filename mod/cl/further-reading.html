Books and Publications

To deepen your understanding of Control Language (CL), consider exploring the following books and publications:

  • "IBM i Control Language (CL): The Basics" by IBM Redbooks

    • This book provides a comprehensive introduction to CL, covering fundamental concepts and basic commands.

  • "Advanced Control Language Programming for IBM i" by Bryan Meyers

    • This publication dives into advanced topics and techniques, making it ideal for those looking to master CL.

  • "IBM i Control Language Programming" by Ted Holt

    • A practical guide that includes numerous examples and exercises to help you apply what you've learned.

Example Code Snippet

Here is a simple example of a CL program that demonstrates basic syntax and structure:

PGM        /* Start of the CL program */
DCL        VAR(&MSG) TYPE(*CHAR) LEN(50) /* Declare a variable */
CHGVAR     VAR(&MSG) VALUE('Hello, World!') /* Change the variable's value */
SNDPGMMSG  MSG(&MSG) /* Send a program message */
ENDPGM     /* End of the CL program */
  • Explanation:
    • PGM and ENDPGM mark the beginning and end of the CL program.
    • DCL is used to declare a variable named &MSG of type character with a length of 50.
    • CHGVAR changes the value of &MSG to 'Hello, World!'.
    • SNDPGMMSG sends the message stored in &MSG to the program message queue.

Remember to experiment with code, participate in community forums, and stay updated with the latest developments in IBM i programming. Happy coding!
