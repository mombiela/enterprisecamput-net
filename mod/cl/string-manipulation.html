String Manipulation in CL (Control Language)
Conditional Statements ►

String manipulation is a fundamental aspect of programming in Control Language (CL). This topic will guide you from basic to advanced string manipulation techniques in CL, providing clear explanations, examples, and exercises to solidify your understanding.

Introduction to Strings in CL

  • Definition: In CL, a string is a sequence of characters.
  • Usage: Strings are used for storing and manipulating text.

Basic String Operations

Declaring Strings

To declare a string in CL, you use the DCL command.

DCL VAR(&MYSTRING) TYPE(*CHAR) LEN(50)
  • VAR(&MYSTRING): Declares a variable named MYSTRING.
  • TYPE(*CHAR): Specifies that the variable is a character string.
  • LEN(50): Sets the length of the string to 50 characters.

Assigning Values to Strings

You can assign values to strings using the CHGVAR command.

CHGVAR VAR(&MYSTRING) VALUE('Hello, World!')
  • VAR(&MYSTRING): The variable to which the value is assigned.
  • VALUE('Hello, World!'): The string value to assign.

String Concatenation

Concatenation is the process of joining two or more strings.

DCL VAR(&FIRSTNAME) TYPE(*CHAR) LEN(20)
DCL VAR(&LASTNAME) TYPE(*CHAR) LEN(20)
DCL VAR(&FULLNAME) TYPE(*CHAR) LEN(40)

CHGVAR VAR(&FIRSTNAME) VALUE('John')
CHGVAR VAR(&LASTNAME) VALUE('Doe')
CHGVAR VAR(&FULLNAME) VALUE(&FIRSTNAME *CAT ' ' *CAT &LASTNAME)
  • *CAT: Concatenates strings with a space in between.

Extracting Substrings

To extract a substring, use the %SST built-in function.

DCL VAR(&MYSTRING) TYPE(*CHAR) LEN(50)
DCL VAR(&SUBSTRING) TYPE(*CHAR) LEN(5)

CHGVAR VAR(&MYSTRING) VALUE('Hello, World!')
CHGVAR VAR(&SUBSTRING) VALUE(%SST(&MYSTRING 1 5))
  • %SST(&MYSTRING 1 5): Extracts a substring starting at position 1 with a length of 5 characters.

Intermediate String Operations

String Length

To determine the length of a string, use the %LEN built-in function.

DCL VAR(&MYSTRING) TYPE(*CHAR) LEN(50)
DCL VAR(&LENGTH) TYPE(*DEC) LEN(5 0)

CHGVAR VAR(&MYSTRING) VALUE('Hello, World!')
CHGVAR VAR(&LENGTH) VALUE(%LEN(&MYSTRING))
  • %LEN(&MYSTRING): Returns the length of the string.

Trimming Strings

Trimming removes leading or trailing spaces from a string.

DCL VAR(&MYSTRING) TYPE(*CHAR) LEN(50)
DCL VAR(&TRIMMED) TYPE(*CHAR) LEN(50)

CHGVAR VAR(&MYSTRING) VALUE('   Hello, World!   ')
CHGVAR VAR(&TRIMMED) VALUE(%TRIM(&MYSTRING))
  • %TRIM(&MYSTRING): Removes both leading and trailing spaces.

Replacing Substrings

To replace a substring within a string, use the %REPLACE built-in function.

DCL VAR(&MYSTRING) TYPE(*CHAR) LEN(50)
DCL VAR(&NEWSTRING) TYPE(*CHAR) LEN(50)

CHGVAR VAR(&MYSTRING) VALUE('Hello, World!')
CHGVAR VAR(&NEWSTRING) VALUE(%REPLACE('Universe' &MYSTRING 8 5))
  • %REPLACE('Universe' &MYSTRING 8 5): Replaces 5 characters starting at position 8 with 'Universe'.

Advanced String Operations

Pattern Matching

Pattern matching can be done using the POS function to find the position of a substring.

DCL VAR(&MYSTRING) TYPE(*CHAR) LEN(50)
DCL VAR(&POSITION) TYPE(*DEC) LEN(5 0)

CHGVAR VAR(&MYSTRING) VALUE('Hello, World!')
CHGVAR VAR(&POSITION) VALUE(%SCAN('World' &MYSTRING))
  • %SCAN('World' &MYSTRING): Returns the starting position of 'World' in MYSTRING.

Complex String Manipulations

Combining multiple string functions for complex operations.

DCL VAR(&MYSTRING) TYPE(*CHAR) LEN(50)
DCL VAR(&RESULT) TYPE(*CHAR) LEN(50)

CHGVAR VAR(&MYSTRING) VALUE('   Hello, World!   ')
CHGVAR VAR(&RESULT) VALUE(%TRIM(%REPLACE('Universe' %SST(&MYSTRING 4 12) 8 5)))
  • Combines TRIM, REPLACE, and SST to manipulate the string.

Summary

In this topic, we covered:

  • Basic string operations: declaration, assignment, concatenation, and extraction.
  • Intermediate operations: determining length, trimming, and replacing substrings.
  • Advanced operations: pattern matching and combining multiple functions for complex manipulations.

Understanding these string manipulation techniques in CL will enhance your ability to handle text data efficiently in your programs. Practice these concepts with the provided examples and exercises to gain proficiency.
