Introduction

System Management Commands in Control Language (CL) are essential for managing and controlling the IBM i operating system. This topic will guide you from the basics to advanced usage of these commands, providing you with the knowledge to effectively manage system resources, users, and jobs.

Basic System Management Commands

Display Commands

  • DSPJOB (Display Job): Displays information about a job.
  • DSPUSRPRF (Display User Profile): Displays information about a user profile.
  • DSPMSG (Display Message): Displays messages from a message queue.

Example: Displaying Job Information

DSPJOB JOB(123456/USER/JOBNAME)

This command displays detailed information about the job with the specified job number, user, and job name.

Work Commands

  • WRKACTJOB (Work with Active Jobs): Displays a list of active jobs.
  • WRKSYSSTS (Work with System Status): Displays the current status of the system.
  • WRKUSRPRF (Work with User Profiles): Displays and allows management of user profiles.

Example: Working with Active Jobs

WRKACTJOB

This command opens a screen displaying all active jobs, allowing you to manage them.

Intermediate System Management Commands

Control Commands

  • ENDJOB (End Job): Ends a specified job.
  • ENDSYS (End System): Ends the system in a controlled manner.
  • ENDSBS (End Subsystem): Ends a specified subsystem.

Example: Ending a Job

ENDJOB JOB(123456/USER/JOBNAME) OPTION(*IMMED)

This command immediately ends the specified job.

User Management Commands

  • CRTUSRPRF (Create User Profile): Creates a new user profile.
  • CHGUSRPRF (Change User Profile): Changes attributes of an existing user profile.
  • DLTUSRPRF (Delete User Profile): Deletes a user profile.

Example: Creating a User Profile

CRTUSRPRF USRPRF(NEWUSER) PASSWORD('password') USRCLS(*USER)

This command creates a new user profile named NEWUSER with the specified password and user class.

Advanced System Management Commands

Backup and Restore Commands

  • SAVLIB (Save Library): Saves a library to a specified device.
  • RSTLIB (Restore Library): Restores a library from a specified device.
  • SAVSYS (Save System): Saves the entire system.

Example: Saving a Library

SAVLIB LIB(MYLIB) DEV(TAP01)

This command saves the library MYLIB to the tape device TAP01.

Performance Management Commands

  • WRKSYSACT (Work with System Activity): Displays and manages system activity.
  • WRKDSKSTS (Work with Disk Status): Displays and manages disk status.
  • WRKSHRPOOL (Work with Shared Pools): Manages shared memory pools.

Example: Working with System Activity

WRKSYSACT

This command opens a screen displaying system activity, allowing you to monitor and manage performance.

Summary

System Management Commands in CL are powerful tools for managing the IBM i operating system. From displaying job information to managing user profiles and system performance, these commands provide comprehensive control over system operations. By mastering these commands, you can ensure efficient and effective system management.

Key Takeaways

  • Basic commands like DSPJOB and WRKACTJOB are essential for monitoring system status.
  • Intermediate commands such as ENDJOB and CRTUSRPRF allow for more direct control over jobs and user profiles.
  • Advanced commands like SAVLIB and WRKSYSACT are crucial for system backup, restore, and performance management.

By practicing these commands and understanding their applications, you will be well-equipped to manage an IBM i system effectively.
