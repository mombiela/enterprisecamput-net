Introduction

Operators in PowerShell are special characters or keywords that are used to perform operations on variables and values. They are essential for manipulating data and controlling the flow of scripts. This section will cover the various types of operators available in PowerShell, from basic arithmetic to advanced comparison and logical operators.

Arithmetic Operators

Arithmetic operators are used to perform mathematical calculations.

  • + : Addition
  • - : Subtraction
  • * : Multiplication
  • / : Division
  • % : Modulus (remainder)

Examples

# Addition
$sum = 5 + 3
Write-Output $sum  # Output: 8

# Subtraction
$difference = 10 - 4
Write-Output $difference  # Output: 6

# Multiplication
$product = 7 * 6
Write-Output $product  # Output: 42

# Division
$quotient = 20 / 4
Write-Output $quotient  # Output: 5

# Modulus
$remainder = 10 % 3
Write-Output $remainder  # Output: 1

Assignment Operators

Assignment operators are used to assign values to variables.

  • = : Simple assignment
  • += : Add and assign
  • -= : Subtract and assign
  • *= : Multiply and assign
  • /= : Divide and assign
  • %= : Modulus and assign

Examples

# Simple assignment
$a = 5

# Add and assign
$a += 3  # Equivalent to $a = $a + 3
Write-Output $a  # Output: 8

# Subtract and assign
$a -= 2  # Equivalent to $a = $a - 2
Write-Output $a  # Output: 6

# Multiply and assign
$a *= 4  # Equivalent to $a = $a * 4
Write-Output $a  # Output: 24

# Divide and assign
$a /= 3  # Equivalent to $a = $a / 3
Write-Output $a  # Output: 8

# Modulus and assign
$a %= 5  # Equivalent to $a = $a % 5
Write-Output $a  # Output: 3

Comparison Operators

Comparison operators are used to compare values and return a boolean result ($true or $false).

  • -eq : Equal to
  • -ne : Not equal to
  • -gt : Greater than
  • -ge : Greater than or equal to
  • -lt : Less than
  • -le : Less than or equal to

Examples

# Equal to
Write-Output (5 -eq 5)  # Output: True

# Not equal to
Write-Output (5 -ne 3)  # Output: True

# Greater than
Write-Output (7 -gt 4)  # Output: True

# Greater than or equal to
Write-Output (7 -ge 7)  # Output: True

# Less than
Write-Output (3 -lt 5)  # Output: True

# Less than or equal to
Write-Output (3 -le 3)  # Output: True

Logical Operators

Logical operators are used to combine multiple conditions.

  • -and : Logical AND
  • -or : Logical OR
  • -not : Logical NOT

Examples

# Logical AND
Write-Output ((5 -eq 5) -and (3 -lt 4))  # Output: True

# Logical OR
Write-Output ((5 -eq 5) -or (3 -gt 4))  # Output: True

# Logical NOT
Write-Output (-not (5 -eq 3))  # Output: True

Special Operators

PowerShell also includes some special operators for specific tasks.

  • .. : Range operator
  • -contains : Contains operator
  • -match : Regular expression match
  • -replace : Replace operator

Examples

# Range operator
$range = 1..5
Write-Output $range  # Output: 1 2 3 4 5

# Contains operator
$array = @(1, 2, 3, 4, 5)
Write-Output ($array -contains 3)  # Output: True

# Regular expression match
$string = "PowerShell"
Write-Output ($string -match "Power")  # Output: True

# Replace operator
$string = "Hello World"
$newString = $string -replace "World", "PowerShell"
Write-Output $newString  # Output: Hello PowerShell

Conclusion

Operators in PowerShell are fundamental tools that enable you to perform a wide range of operations, from simple arithmetic to complex logical evaluations. Understanding how to use these operators effectively will greatly enhance your ability to write powerful and efficient PowerShell scripts. Practice using these operators in different scenarios to become more proficient in PowerShell scripting.
