Color Grading in Blender
Adding Effects ►

Color grading is an essential part of the post-production process in 3D rendering and animation. It involves adjusting the colors and tones of your rendered images or animations to achieve a desired look or mood. This topic will guide you through the basics to advanced techniques of color grading in Blender.

Introduction to Color Grading

Color grading can dramatically change the feel of your project. It can make your scenes look more cinematic, highlight important elements, or create a specific atmosphere.

Key Concepts

  • Color Correction: Adjusting colors to make them look natural and consistent.
  • Color Grading: Stylizing colors to achieve a specific artistic effect.
  • LUTs (Look-Up Tables): Predefined color grading presets that can be applied to your renders.
  • Scopes: Tools like histograms, waveforms, and vectorscopes that help analyze color information.

Basic Color Grading Techniques

Using the Color Balance Node

The Color Balance node in Blender allows you to adjust the lift, gamma, and gain of your image, which correspond to shadows, midtones, and highlights, respectively.

import bpy

# Create a new node tree for compositing
bpy.context.scene.use_nodes = True
tree = bpy.context.scene.node_tree

# Clear default nodes
for node in tree.nodes:
    tree.nodes.remove(node)

# Add Render Layers node
render_layers = tree.nodes.new(type='CompositorNodeRLayers')

# Add Color Balance node
color_balance = tree.nodes.new(type='CompositorNodeColorBalance')

# Add Composite node
composite = tree.nodes.new(type='CompositorNodeComposite')

# Link nodes
tree.links.new(render_layers.outputs['Image'], color_balance.inputs['Image'])
tree.links.new(color_balance.outputs['Image'], composite.inputs['Image'])

# Adjust Color Balance settings
color_balance.correction_method = 'LIFT_GAMMA_GAIN'
color_balance.lift = [0.8, 0.8, 0.8]
color_balance.gamma = [1.2, 1.2, 1.2]
color_balance.gain = [1.1, 1.1, 1.1]

Using the RGB Curves Node

The RGB Curves node provides more precise control over the color channels and overall brightness.

import bpy

# Add RGB Curves node
rgb_curves = tree.nodes.new(type='CompositorNodeCurveRGB')

# Link nodes
tree.links.new(render_layers.outputs['Image'], rgb_curves.inputs['Image'])
tree.links.new(rgb_curves.outputs['Image'], composite.inputs['Image'])

# Adjust RGB Curves settings
curve = rgb_curves.mapping.curves[3]  # Master curve
curve.points.new(0.3, 0.2)
curve.points.new(0.7, 0.8)

Intermediate Color Grading Techniques

Using LUTs (Look-Up Tables)

LUTs can be used to apply complex color grading presets to your renders.

import bpy

# Add a LUT node
lut_node = tree.nodes.new(type='CompositorNodeLUT')

# Link nodes
tree.links.new(render_layers.outputs['Image'], lut_node.inputs['Image'])
tree.links.new(lut_node.outputs['Image'], composite.inputs['Image'])

# Load LUT file
lut_node.lut.filepath = '/path/to/your/lut.cube'

Using Scopes for Analysis

Scopes help you analyze the color information in your image to make more informed adjustments.

  • Histogram: Shows the distribution of pixel brightness.
  • Waveform: Displays the brightness levels across the image.
  • Vectorscope: Visualizes the color information.

Advanced Color Grading Techniques

Combining Multiple Nodes

For advanced color grading, you can combine multiple nodes to achieve complex effects.

import bpy

# Add multiple nodes
hue_saturation = tree.nodes.new(type='CompositorNodeHueSat')
brightness_contrast = tree.nodes.new(type='CompositorNodeBrightContrast')

# Link nodes
tree.links.new(render_layers.outputs['Image'], hue_saturation.inputs['Image'])
tree.links.new(hue_saturation.outputs['Image'], brightness_contrast.inputs['Image'])
tree.links.new(brightness_contrast.outputs['Image'], composite.inputs['Image'])

# Adjust settings
hue_saturation.color_saturation = 1.2
brightness_contrast.inputs['Bright'].default_value = 0.1
brightness_contrast.inputs['Contrast'].default_value = 1.5

Conclusion

Color grading is a powerful tool in Blender that can significantly enhance the visual quality of your projects. By understanding and utilizing the various nodes and techniques available, you can achieve professional-level results. Start with basic adjustments using the Color Balance and RGB Curves nodes, explore LUTs for quick presets, and dive into advanced techniques by combining nodes and using scopes for precise control. Happy grading!
