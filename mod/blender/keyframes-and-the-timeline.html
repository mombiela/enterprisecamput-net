Introduction

In this section, we will explore the fundamental concepts of keyframes and the timeline in Blender. These tools are essential for creating animations, allowing you to control the motion and transformation of objects over time.

Keyframes

Keyframes are markers that define the start and end points of any transition. They are used to specify the values of an object's properties at a particular point in time.

Key Concepts

  • Keyframe: A marker that records the value of a property at a specific frame.
  • Interpolation: The process of calculating intermediate values between keyframes.
  • Property: Any attribute of an object that can be animated, such as location, rotation, scale, or material properties.

Adding Keyframes

To add a keyframe in Blender:

  1. Select the object you want to animate.
  2. Go to the frame where you want to add the keyframe.
  3. Press I to bring up the Insert Keyframe Menu.
  4. Choose the property you want to keyframe (e.g., Location, Rotation, Scale).
# Example: Adding a keyframe to the location of an object
import bpy

# Select the object
obj = bpy.context.object

# Set the frame to 1
bpy.context.scene.frame_set(1)

# Set the location
obj.location = (1, 1, 1)

# Insert a keyframe for the location
obj.keyframe_insert(data_path="location", frame=1)

Editing Keyframes

To edit keyframes:

  1. Open the Graph Editor or Dope Sheet.
  2. Select the keyframe you want to edit.
  3. Move, scale, or delete keyframes as needed.

The Timeline

The timeline is a horizontal bar that represents the sequence of frames in your animation. It allows you to navigate through frames and manage keyframes.

Key Concepts

  • Frame: A single point in the timeline.
  • Playback Controls: Buttons to play, pause, and stop the animation.
  • Current Frame Indicator: Shows the current frame number.

Navigating the Timeline

  • Scrubbing: Click and drag the current frame indicator to move through frames.
  • Jump to Frame: Press Shift + Left Arrow to go to the start, Shift + Right Arrow to go to the end.

Managing Keyframes in the Timeline

  • Add Keyframe: Press I at the desired frame.
  • Delete Keyframe: Select the keyframe and press X.
  • Move Keyframe: Click and drag the keyframe to a new frame.

Example: Simple Animation

Let's create a simple animation where a cube moves from one point to another.

  1. Setup: Open Blender and add a cube.
  2. First Keyframe:
    • Go to frame 1.
    • Move the cube to the starting position.
    • Press I and select Location to insert a keyframe.
  3. Second Keyframe:
    • Go to frame 50.
    • Move the cube to the ending position.
    • Press I and select Location to insert a keyframe.
# Example: Simple animation script
import bpy

# Select the object
obj = bpy.context.object

# First keyframe at frame 1
bpy.context.scene.frame_set(1)
obj.location = (0, 0, 0)
obj.keyframe_insert(data_path="location", frame=1)

# Second keyframe at frame 50
bpy.context.scene.frame_set(50)
obj.location = (5, 5, 5)
obj.keyframe_insert(data_path="location", frame=50)

Summary

Keyframes and the timeline are crucial for creating animations in Blender. Keyframes mark the values of properties at specific frames, while the timeline helps you navigate and manage these keyframes. By mastering these tools, you can create complex animations with smooth transitions and precise control over your objects' movements.
