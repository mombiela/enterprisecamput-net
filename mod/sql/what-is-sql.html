Introduction to SQL

SQL, or Structured Query Language, is a standard programming language specifically designed for managing and manipulating relational databases. It is used to perform various operations on data stored in a database, such as querying, updating, inserting, and deleting data.

Key Concepts of SQL

  • Database: A structured collection of data stored electronically in a computer system.
  • Table: A collection of related data entries and it consists of columns and rows.
  • Query: A request for data or information from a database table or combination of tables.
  • Schema: The structure that defines the organization of data in a database, including tables, columns, and relationships.
  • Primary Key: A unique identifier for a record in a table.
  • Foreign Key: A field in one table that uniquely identifies a row of another table.

Basic SQL Commands

SQL commands are categorized into several types, including Data Query Language (DQL), Data Definition Language (DDL), Data Manipulation Language (DML), and Data Control Language (DCL).

Data Query Language (DQL)

  • SELECT: Used to retrieve data from a database.
SELECT column1, column2
FROM table_name;

Example:

SELECT first_name, last_name
FROM employees;

This query retrieves the first and last names of all employees from the employees table.

Data Definition Language (DDL)

  • CREATE: Used to create a new table or database.
CREATE TABLE table_name (
    column1 datatype,
    column2 datatype,
    ...
);

Example:

CREATE TABLE employees (
    employee_id INT PRIMARY KEY,
    first_name VARCHAR(50),
    last_name VARCHAR(50),
    hire_date DATE
);

This command creates a new table named employees with specified columns.

  • ALTER: Used to modify an existing database object, such as a table.
ALTER TABLE table_name
ADD column_name datatype;

Example:

ALTER TABLE employees
ADD email VARCHAR(100);

This command adds a new column email to the employees table.

  • DROP: Used to delete a table or database.
DROP TABLE table_name;

Example:

DROP TABLE employees;

This command deletes the employees table from the database.

Data Manipulation Language (DML)

  • INSERT: Used to insert new data into a table.
INSERT INTO table_name (column1, column2, ...)
VALUES (value1, value2, ...);

Example:

INSERT INTO employees (employee_id, first_name, last_name, hire_date)
VALUES (1, 'John', 'Doe', '2023-01-01');

This command inserts a new employee record into the employees table.

  • UPDATE: Used to update existing data within a table.
UPDATE table_name
SET column1 = value1, column2 = value2, ...
WHERE condition;

Example:

UPDATE employees
SET email = 'john.doe@example.com'
WHERE employee_id = 1;

This command updates the email address of the employee with employee_id 1.

  • DELETE: Used to delete data from a table.
DELETE FROM table_name
WHERE condition;

Example:

DELETE FROM employees
WHERE employee_id = 1;

This command deletes the employee record with employee_id 1 from the employees table.

Data Control Language (DCL)

  • GRANT: Used to give users access privileges to the database.
GRANT SELECT, INSERT ON table_name TO 'username';

Example:

GRANT SELECT, INSERT ON employees TO 'john_doe';

This command grants SELECT and INSERT privileges on the employees table to the user john_doe.

  • REVOKE: Used to take back permissions granted to users.
REVOKE SELECT, INSERT ON table_name FROM 'username';

Example:

REVOKE SELECT, INSERT ON employees FROM 'john_doe';

This command revokes SELECT and INSERT privileges on the employees table from the user john_doe.

Conclusion

SQL is an essential tool for anyone working with relational databases. It allows users to efficiently manage and manipulate data through a variety of commands. Understanding the basic concepts and commands of SQL is the first step towards mastering database management and becoming proficient in data handling tasks. As you progress through this course, you will learn more advanced SQL techniques and best practices to enhance your database skills.
