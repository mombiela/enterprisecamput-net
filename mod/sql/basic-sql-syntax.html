Introduction

Structured Query Language (SQL) is the standard language for managing and manipulating relational databases. Understanding basic SQL syntax is crucial for performing operations such as querying data, updating records, and managing database structures.

Key Concepts

  • SQL Statements: Commands used to communicate with the database.
  • Clauses: Components of SQL statements that specify conditions or actions.
  • Keywords: Reserved words with special meaning in SQL.
  • Identifiers: Names given to database objects like tables and columns.
  • Operators: Symbols used to specify conditions in SQL statements.

SQL Statements

SQL statements are the building blocks of SQL. Here are some of the most common SQL statements:

SELECT Statement

The SELECT statement is used to query data from a database.

SELECT column1, column2
FROM table_name;
  • SELECT: Specifies the columns to retrieve.
  • FROM: Specifies the table to query.

WHERE Clause

The WHERE clause is used to filter records.

SELECT column1, column2
FROM table_name
WHERE condition;
  • WHERE: Specifies the condition to filter records.

INSERT INTO Statement

The INSERT INTO statement is used to add new records to a table.

INSERT INTO table_name (column1, column2)
VALUES (value1, value2);
  • INSERT INTO: Specifies the table and columns to insert data into.
  • VALUES: Specifies the values to insert.

UPDATE Statement

The UPDATE statement is used to modify existing records.

UPDATE table_name
SET column1 = value1, column2 = value2
WHERE condition;
  • UPDATE: Specifies the table to update.
  • SET: Specifies the columns and new values.
  • WHERE: Specifies the condition to identify records to update.

DELETE Statement

The DELETE statement is used to remove records from a table.

DELETE FROM table_name
WHERE condition;
  • DELETE FROM: Specifies the table to delete records from.
  • WHERE: Specifies the condition to identify records to delete.

Examples and Exercises

Example 1: Selecting Data

Retrieve the names and ages of all employees from the employees table.

SELECT name, age
FROM employees;

Exercise 1: Selecting Data

Write an SQL query to retrieve the title and author from the books table.

Example 2: Filtering Data

Retrieve the names of employees who are older than 30.

SELECT name
FROM employees
WHERE age > 30;

Exercise 2: Filtering Data

Write an SQL query to retrieve the title of books published after the year 2000 from the books table.

Example 3: Inserting Data

Add a new employee to the employees table.

INSERT INTO employees (name, age, department)
VALUES ('John Doe', 28, 'Marketing');

Exercise 3: Inserting Data

Write an SQL query to add a new book to the books table with the columns title, author, and year.

Example 4: Updating Data

Update the age of an employee named 'John Doe' to 29.

UPDATE employees
SET age = 29
WHERE name = 'John Doe';

Exercise 4: Updating Data

Write an SQL query to change the year of a book titled 'SQL Basics' to 2021 in the books table.

Example 5: Deleting Data

Remove an employee named 'John Doe' from the employees table.

DELETE FROM employees
WHERE name = 'John Doe';

Exercise 5: Deleting Data

Write an SQL query to delete a book titled 'Old Book' from the books table.

Conclusion

Understanding basic SQL syntax is essential for interacting with relational databases. By mastering the SELECT, INSERT INTO, UPDATE, and DELETE statements, you can perform a wide range of data manipulation tasks. Practice these statements and their clauses to build a strong foundation in SQL.
