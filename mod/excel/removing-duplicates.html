Removing Duplicates in Excel
Logical Functions (IF, AND, OR) ►

Removing duplicates is a common task in Excel, especially when dealing with large datasets. This topic will guide you through various methods to identify and remove duplicate entries, from beginner to advanced levels.

Beginner Level

Using the 'Remove Duplicates' Feature

Excel provides a built-in feature to remove duplicates easily.

  1. Select the Data Range:

    • Click and drag to select the range of cells you want to check for duplicates.
    • Alternatively, click any cell within your data range and Excel will automatically select the entire range.

  2. Navigate to the 'Data' Tab:

    • Go to the 'Data' tab on the Ribbon.
    • Click on the 'Remove Duplicates' button in the 'Data Tools' group.

  3. Choose Columns:

    • A dialog box will appear. Here, you can choose which columns to check for duplicates.
    • By default, all columns are selected. Uncheck any columns you do not want to include in the duplicate check.

  4. Remove Duplicates:

    • Click 'OK'. Excel will remove the duplicate rows and show a message indicating how many duplicates were removed.

Example

Suppose you have the following data:

Name Age City
John 25 New York
Jane 30 Los Angeles
John 25 New York
Mike 22 Chicago

After using the 'Remove Duplicates' feature on all columns, the data will be:

Name Age City
John 25 New York
Jane 30 Los Angeles
Mike 22 Chicago

Intermediate Level

Using Conditional Formatting to Highlight Duplicates

Conditional Formatting can be used to highlight duplicates without removing them, allowing you to review them first.

  1. Select the Data Range:

    • Highlight the range of cells you want to check for duplicates.

  2. Navigate to the 'Home' Tab:

    • Go to the 'Home' tab on the Ribbon.
    • Click on 'Conditional Formatting' in the 'Styles' group.

  3. Highlight Duplicates:

    • Choose 'Highlight Cells Rules' > 'Duplicate Values...'.
    • A dialog box will appear. You can choose the formatting style for the duplicates.
    • Click 'OK' to apply the formatting.

Example

Using the same data:

Name Age City
John 25 New York
Jane 30 Los Angeles
John 25 New York
Mike 22 Chicago

After applying Conditional Formatting, the duplicate rows will be highlighted.

Advanced Level

Using Formulas to Identify Duplicates

For more control, you can use Excel formulas to identify duplicates.

  1. Using the COUNTIF Function:

    • The COUNTIF function can count the number of times a value appears in a range.

  2. Create a Helper Column:

    • Add a new column next to your data to use as a helper column.
    • In the first cell of the helper column, enter the formula: 
      =COUNTIF($A$2:$A$5, A2) > 1
    • Drag the fill handle to copy the formula down the column.

  3. Filter or Sort by Helper Column:

    • You can now filter or sort your data based on the helper column to see which rows are duplicates.

Example

Using the same data:

Name Age City Duplicate
John 25 New York TRUE
Jane 30 Los Angeles FALSE
John 25 New York TRUE
Mike 22 Chicago FALSE

The helper column shows TRUE for duplicates.

Conclusion

Removing duplicates in Excel can be done using various methods, from simple built-in features to more advanced techniques involving formulas. Understanding these methods allows you to clean and manage your data effectively, ensuring accuracy and reliability in your datasets. Whether you are a beginner or an advanced user, mastering these techniques will enhance your data handling skills in Excel.
