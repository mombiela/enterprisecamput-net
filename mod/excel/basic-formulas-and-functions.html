Introduction

Excel is a powerful tool for data analysis and manipulation, and understanding its basic formulas and functions is essential for any user. This section will cover the foundational formulas and functions that every Excel user should know, from simple arithmetic operations to more complex functions.

Arithmetic Operations

Excel allows you to perform basic arithmetic operations directly within cells.

  • Addition (+): Adds two or more numbers.
  • Subtraction (-): Subtracts one number from another.
  • Multiplication (*): Multiplies two or more numbers.
  • Division (/): Divides one number by another.

Examples

=5 + 3      # Result: 8
=10 - 4     # Result: 6
=7 * 2      # Result: 14
=20 / 4     # Result: 5

Basic Functions

Excel functions are predefined formulas that perform calculations using specific values, called arguments, in a particular order.

SUM

Adds all the numbers in a range of cells.

=SUM(A1:A5)  # Adds all numbers from cell A1 to A5

AVERAGE

Calculates the average of a group of numbers.

=AVERAGE(B1:B10)  # Calculates the average of numbers from cell B1 to B10

MIN and MAX

Finds the smallest and largest number in a range of cells, respectively.

=MIN(C1:C10)  # Finds the smallest number in the range C1 to C10
=MAX(D1:D10)  # Finds the largest number in the range D1 to D10

COUNT

Counts the number of cells that contain numbers.

=COUNT(E1:E10)  # Counts the number of cells with numbers in the range E1 to E10

Logical Functions

Logical functions allow you to perform comparisons and return results based on whether a condition is true or false.

IF

Checks whether a condition is met, and returns one value if true and another value if false.

=IF(A1 > 10, "Greater", "Lesser")  # Returns "Greater" if A1 is more than 10, otherwise "Lesser"

AND, OR

Combines multiple conditions.

  • AND: Returns TRUE if all conditions are true.
  • OR: Returns TRUE if any condition is true.
=AND(A1 > 10, B1 < 5)  # Returns TRUE if A1 is more than 10 and B1 is less than 5
=OR(A1 > 10, B1 < 5)   # Returns TRUE if either A1 is more than 10 or B1 is less than 5

Text Functions

Text functions allow you to manipulate text strings.

CONCATENATE

Joins two or more text strings into one string.

=CONCATENATE("Hello", " ", "World")  # Result: "Hello World"

LEFT, RIGHT, MID

Extracts a substring from a text string.

  • LEFT: Extracts a given number of characters from the left.
  • RIGHT: Extracts a given number of characters from the right.
  • MID: Extracts characters from the middle of a text string.
=LEFT("Excel", 2)   # Result: "Ex"
=RIGHT("Excel", 2)  # Result: "el"
=MID("Excel", 2, 3) # Result: "xce"

Date and Time Functions

Date and time functions allow you to work with dates and times.

TODAY and NOW

Returns the current date and time.

=TODAY()  # Returns the current date
=NOW()    # Returns the current date and time

DATE, YEAR, MONTH, DAY

Creates a date or extracts parts of a date.

=DATE(2023, 10, 5)  # Creates the date October 5, 2023
=YEAR(TODAY())      # Extracts the year from the current date
=MONTH(TODAY())     # Extracts the month from the current date
=DAY(TODAY())       # Extracts the day from the current date

Lookup and Reference Functions

Lookup and reference functions help you find and reference data within a table.

VLOOKUP

Searches for a value in the first column of a table and returns a value in the same row from a specified column.

=VLOOKUP(A1, B1:D10, 2, FALSE)  # Looks for the value in A1 within the range B1:D10 and returns the value in the second column

HLOOKUP

Searches for a value in the first row of a table and returns a value in the same column from a specified row.

=HLOOKUP(A1, B1:D10, 2, FALSE)  # Looks for the value in A1 within the range B1:D10 and returns the value in the second row

Conclusion

Understanding and mastering these basic formulas and functions in Excel is crucial for efficiently managing and analyzing data. These foundational skills will serve as the building blocks for more advanced Excel techniques. Practice using these functions and formulas to become proficient and confident in your Excel abilities.
