Introduction to What-If Analysis

What-If Analysis in Excel allows you to experiment with data to see how changes affect outcomes. This is particularly useful for decision-making and forecasting. The three primary tools for What-If Analysis in Excel are:

  • Goal Seek
  • Data Tables
  • Scenario Manager

Goal Seek

Goal Seek is a simple tool that allows you to find the input value needed to achieve a specific goal. It works by adjusting one variable to reach a target value.

How to Use Goal Seek

  1. Set up your data: Ensure you have a formula that calculates the result you are interested in.
  2. Open Goal Seek: Go to the Data tab, click on What-If Analysis, and select Goal Seek.
  3. Configure Goal Seek:
    • Set Cell: The cell containing the formula.
    • To Value: The desired result you want to achieve.
    • By Changing Cell: The cell that Excel will change to achieve the desired result.

Example

Suppose you want to find out how much you need to sell to achieve a revenue of $10,000.

| A | B | |-----------|---------| | Units Sold| 100 | | Price | $50 | | Revenue | =A2*B2 |

To find the required units sold to achieve $10,000 in revenue:

  1. Set Cell: B3 (Revenue)
  2. To Value: 10000
  3. By Changing Cell: A2 (Units Sold)

Data Tables

Data Tables allow you to see how changes in one or two variables affect the results of a formula. There are two types of Data Tables:

  • One-Variable Data Table
  • Two-Variable Data Table

One-Variable Data Table

A One-Variable Data Table shows how changing one variable impacts the result.

How to Create a One-Variable Data Table

  1. Set up your data: Create a formula that depends on one variable.
  2. Create a table structure: List the variable values in a column or row.
  3. Insert Data Table:
    • Select the range including the formula and variable values.
    • Go to the Data tab, click on What-If Analysis, and select Data Table.
    • For a column of variable values, enter the cell reference for the variable in the Column Input Cell.

Example

Calculate the revenue for different units sold.

| A | B | C | |-----------|---------|---------| | Units Sold| Revenue | | | 50 | | | | 100 | | | | 150 | | | | 200 | | | | Formula | =A2*50 | |

  1. Select the range A1:B5.
  2. Go to Data > What-If Analysis > Data Table.
  3. Enter A2 as the Column Input Cell.

Two-Variable Data Table

A Two-Variable Data Table shows how changing two variables simultaneously affects the result.

How to Create a Two-Variable Data Table

  1. Set up your data: Create a formula that depends on two variables.
  2. Create a table structure: List one variable's values in a row and the other in a column.
  3. Insert Data Table:
    • Select the range including the formula and variable values.
    • Go to the Data tab, click on What-If Analysis, and select Data Table.
    • Enter the cell references for the row and column input cells.

Example

Calculate the revenue for different units sold and prices.

| A | B | C | D | E | |-----------|---------|---------|---------|---------| | Units Sold| 40 | 50 | 60 | 70 | | 100 | | | | | | 150 | | | | | | 200 | | | | | | Formula | =A2*B1 | | | |

  1. Select the range A1:E4.
  2. Go to Data > What-If Analysis > Data Table.
  3. Enter A2 as the Row Input Cell and B1 as the Column Input Cell.

Scenario Manager

Scenario Manager allows you to create and save different sets of input values, called scenarios, and switch between them to see how they affect your results.

How to Use Scenario Manager

  1. Set up your data: Identify the cells that will change in each scenario.
  2. Open Scenario Manager: Go to the Data tab, click on What-If Analysis, and select Scenario Manager.
  3. Create Scenarios:
    • Click Add to create a new scenario.
    • Name the scenario and select the changing cells.
    • Enter the values for each scenario.
  4. View Scenarios: Select a scenario and click Show to see the results.

Example

Suppose you have different sales strategies with varying units sold and prices.

| A | B | C | |-----------|---------|---------| | Units Sold| 100 | | | Price | $50 | | | Revenue | =A2*B2 | |

  1. Go to Data > What-If Analysis > Scenario Manager.
  2. Click Add to create a scenario for Strategy 1:
    • Name: Strategy 1
    • Changing Cells: A2, B2
    • Values: 100, 50
  3. Add another scenario for Strategy 2:
    • Name: Strategy 2
    • Changing Cells: A2, B2
    • Values: 150, 45
  4. Select a scenario and click Show to see the impact on revenue.

Conclusion

What-If Analysis in Excel is a powerful tool for exploring different scenarios and making informed decisions. By using Goal Seek, Data Tables, and Scenario Manager, you can analyze how changes in variables affect your results and plan accordingly. Practice using these tools with your own data to become proficient in What-If Analysis.
