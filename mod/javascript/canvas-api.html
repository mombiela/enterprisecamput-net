The Canvas API is a powerful feature in HTML5 that allows you to draw graphics using JavaScript. This topic will cover the basics of the Canvas API, progressing to more advanced techniques.

Introduction to Canvas API

  • What is the Canvas API?

    • A JavaScript API for drawing 2D graphics.
    • Allows for dynamic, scriptable rendering of shapes and images.

  • Setting Up the Canvas

    • HTML <canvas> element.
    • Accessing the canvas context in JavaScript.

Example: Basic Canvas Setup

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Canvas Example</title>
</head>
<body>
    <canvas id="myCanvas" width="500" height="500" style="border:1px solid #000000;"></canvas>
    <script>
        const canvas = document.getElementById('myCanvas');
        const ctx = canvas.getContext('2d');
    </script>
</body>
</html>

Drawing Shapes

  • Rectangles
    • fillRect(x, y, width, height)
    • strokeRect(x, y, width, height)
    • clearRect(x, y, width, height)

Example: Drawing Rectangles

ctx.fillStyle = 'green';
ctx.fillRect(10, 10, 100, 100);

ctx.strokeStyle = 'blue';
ctx.strokeRect(130, 10, 100, 100);

ctx.clearRect(50, 50, 60, 60);
  • Paths
    • beginPath()
    • moveTo(x, y)
    • lineTo(x, y)
    • closePath()
    • stroke()
    • fill()

Example: Drawing Paths

ctx.beginPath();
ctx.moveTo(50, 50);
ctx.lineTo(200, 50);
ctx.lineTo(200, 200);
ctx.closePath();
ctx.stroke();

Working with Colors and Styles

  • Fill and Stroke Styles
    • fillStyle
    • strokeStyle
    • globalAlpha

Example: Using Colors and Styles

ctx.fillStyle = 'rgba(255, 0, 0, 0.5)';
ctx.fillRect(10, 10, 100, 100);

ctx.strokeStyle = 'rgba(0, 0, 255, 0.5)';
ctx.strokeRect(130, 10, 100, 100);

Drawing Text

  • Text Methods
    • fillText(text, x, y)
    • strokeText(text, x, y)
    • font
    • textAlign
    • textBaseline

Example: Drawing Text

ctx.font = '30px Arial';
ctx.fillText('Hello Canvas', 10, 50);

ctx.strokeText('Hello Canvas', 10, 100);

Transformations

  • Scaling

    • scale(x, y)

  • Rotating

    • rotate(angle)

  • Translating

    • translate(x, y)

Example: Applying Transformations

ctx.translate(100, 100);
ctx.rotate(Math.PI / 4);
ctx.fillStyle = 'purple';
ctx.fillRect(0, 0, 100, 100);

Advanced Techniques

  • Gradients
    • createLinearGradient(x0, y0, x1, y1)
    • createRadialGradient(x0, y0, r0, x1, y1, r1)

Example: Using Gradients

const gradient = ctx.createLinearGradient(0, 0, 200, 0);
gradient.addColorStop(0, 'red');
gradient.addColorStop(1, 'blue');

ctx.fillStyle = gradient;
ctx.fillRect(10, 10, 200, 100);
  • Patterns
    • createPattern(image, repetition)

Example: Using Patterns

const img = new Image();
img.src = 'path/to/image.jpg';
img.onload = function() {
    const pattern = ctx.createPattern(img, 'repeat');
    ctx.fillStyle = pattern;
    ctx.fillRect(0, 0, 500, 500);
};

Animations

  • Basic Animation Loop
    • requestAnimationFrame(callback)

Example: Basic Animation

let x = 0;
function draw() {
    ctx.clearRect(0, 0, canvas.width, canvas.height);
    ctx.fillRect(x, 50, 50, 50);
    x += 1;
    requestAnimationFrame(draw);
}
draw();

Conclusion

The Canvas API is a versatile tool for creating dynamic graphics in web applications. By understanding the basics of drawing shapes, working with colors, applying transformations, and creating animations, you can build complex and interactive visualizations. Practice these techniques to become proficient in using the Canvas API for your projects.
