Introduction

JavaScript is a versatile and powerful programming language primarily used for web development. This section will cover the fundamental syntax and basic concepts that are essential for anyone starting with JavaScript.

Key Concepts

Variables

Variables are used to store data that can be referenced and manipulated in a program. JavaScript uses three keywords to declare variables:

  • var
  • let
  • const

Example:

var name = "John";
let age = 30;
const isStudent = true;

Data Types

JavaScript supports several data types, including:

  • String: Textual data
  • Number: Numeric data
  • Boolean: True or false values
  • Array: Ordered list of values
  • Object: Collection of key-value pairs
  • Null: Represents the intentional absence of any object value
  • Undefined: Variable that has been declared but not assigned a value

Example:

let message = "Hello, World!"; // String
let count = 42; // Number
let isActive = false; // Boolean
let fruits = ["Apple", "Banana", "Cherry"]; // Array
let person = { firstName: "John", lastName: "Doe" }; // Object
let emptyValue = null; // Null
let notAssigned; // Undefined

Operators

Operators are used to perform operations on variables and values. Common operators include:

  • Arithmetic Operators: +, -, *, /, %
  • Assignment Operators: =, +=, -=, *=, /=
  • Comparison Operators: ==, ===, !=, !==, >, <, >=, <=
  • Logical Operators: &&, ||, !

Example:

let a = 10;
let b = 20;
let sum = a + b; // Arithmetic
a += 5; // Assignment
let isEqual = (a == b); // Comparison
let isBothTrue = (a > 5 && b < 30); // Logical

Control Structures

Control structures are used to dictate the flow of the program. Common control structures include:

  • Conditional Statements: if, else if, else
  • Switch Statements: switch
  • Loops: for, while, do...while

Example:

// Conditional Statement
if (age > 18) {
    console.log("Adult");
} else {
    console.log("Minor");
}

// Switch Statement
let day = 3;
switch (day) {
    case 1:
        console.log("Monday");
        break;
    case 2:
        console.log("Tuesday");
        break;
    case 3:
        console.log("Wednesday");
        break;
    default:
        console.log("Invalid day");
}

// For Loop
for (let i = 0; i < 5; i++) {
    console.log(i);
}

// While Loop
let count = 0;
while (count < 5) {
    console.log(count);
    count++;
}

// Do...While Loop
let num = 0;
do {
    console.log(num);
    num++;
} while (num < 5);

Functions

Functions are blocks of code designed to perform a particular task. They are executed when they are called.

Example:

// Function Declaration
function greet(name) {
    return "Hello, " + name + "!";
}

// Function Expression
const greet = function(name) {
    return "Hello, " + name + "!";
};

// Arrow Function (ES6)
const greet = (name) => {
    return "Hello, " + name + "!";
};

// Calling a Function
console.log(greet("Alice"));

Conclusion

Understanding the basics of JavaScript syntax is crucial for building a strong foundation in programming. This section covered variables, data types, operators, control structures, and functions. Mastering these concepts will enable you to write simple yet effective JavaScript code and prepare you for more advanced topics. Keep practicing and experimenting with these basics to become more comfortable with JavaScript.
