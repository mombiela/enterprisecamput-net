Introduction

In statistics, random variables and probability distributions are fundamental concepts that allow us to model and analyze uncertain phenomena. This topic covers the basic concepts, types of random variables, and the most common probability distributions.

Random Variables

A random variable is a function that assigns a numerical value to each possible outcome of a random experiment.

Types of Random Variables

  • Discrete Random Variable: Takes a finite or countable number of values. Example: Number of heads when flipping a coin three times.
  • Continuous Random Variable: Takes an infinite number of values within an interval. Example: Waiting time in a queue.

Example of Discrete Random Variable

Suppose we roll a fair die. The random variable \(X\) represents the outcome of the die roll.

  • Possible values of \(X\): {1, 2, 3, 4, 5, 6}

Example of Continuous Random Variable

Consider the time \(T\) that a customer waits in a queue. This variable can take any positive value.

  • Possible values of \(T\): {0, 0.1, 0.2, ..., ∞}

Probability Distributions

A probability distribution describes how the values of a random variable are distributed.

Probability Distribution for Discrete Variables

  • Probability Mass Function (PMF): Assigns a probability to each possible value of the discrete random variable.

    Example: Rolling a fair die. \[ P(X = x) = \frac{1}{6} \quad \text{for } x \in {1, 2, 3, 4, 5, 6} \]

Probability Distribution for Continuous Variables

  • Probability Density Function (PDF): Describes the probability that a continuous random variable takes a value within an interval.

    Example: Normal distribution. \[ f(x) = \frac{1}{\sqrt{2\pi\sigma^2}} e^{-\frac{(x-\mu)^2}{2\sigma^2}} \] where \(\mu\) is the mean and \(\sigma\) is the standard deviation.

Common Probability Distributions

Binomial Distribution

  • Description: Models the number of successes in a series of independent trials with a constant probability of success.
  • Parameters: \(n\) (number of trials), \(p\) (probability of success).
  • PMF: \[ P(X = k) = \binom{n}{k} p^k (1-p)^{n-k} \] where \(\binom{n}{k}\) is the binomial coefficient.

Normal Distribution

  • Description: Models natural phenomena and is symmetric around the mean.
  • Parameters: \(\mu\) (mean), \(\sigma\) (standard deviation).
  • PDF: \[ f(x) = \frac{1}{\sqrt{2\pi\sigma^2}} e^{-\frac{(x-\mu)^2}{2\sigma^2}} \]

Poisson Distribution

  • Description: Models the number of events that occur in an interval of time or space.
  • Parameters: \(\lambda\) (rate of events).
  • PMF: \[ P(X = k) = \frac{\lambda^k e^{-\lambda}}{k!} \]

Code Examples

Example in Python: Binomial Distribution

import numpy as np
import matplotlib.pyplot as plt
from scipy.stats import binom

# Parameters
n = 10  # number of trials
p = 0.5  # probability of success

# Possible values of X
x = np.arange(0, n+1)

# PMF
pmf = binom.pmf(x, n, p)

# Plot
plt.bar(x, pmf)
plt.xlabel('Number of successes')
plt.ylabel('Probability')
plt.title('Binomial Distribution (n=10, p=0.5)')
plt.show()

Example in Python: Normal Distribution

import numpy as np
import matplotlib.pyplot as plt
from scipy.stats import norm

# Parameters
mu = 0  # mean
sigma = 1  # standard deviation

# Possible values of X
x = np.linspace(mu - 3*sigma, mu + 3*sigma, 100)

# PDF
pdf = norm.pdf(x, mu, sigma)

# Plot
plt.plot(x, pdf)
plt.xlabel('Value')
plt.ylabel('Probability Density')
plt.title('Normal Distribution (μ=0, σ=1)')
plt.show()

Conclusion

Random variables and probability distributions are essential tools in statistics for modeling and analyzing data. Understanding these concepts allows professionals to make informed decisions based on data and effectively handle uncertainty. In this topic, we have covered the types of random variables, the most common probability distributions, and provided practical examples to illustrate these concepts.
