Introduction

Deviation analysis is a fundamental tool in strategic thinking that allows organizations to identify and understand the differences between expected results and actual results. This analysis is crucial for informed decision-making and for the implementation of corrective strategies.

Key Concepts

  • Deviation: Difference between the expected result and the actual result.
  • Objective: Goal or result that is expected to be achieved.
  • Actual Result: Result that has been obtained in practice.
  • Cause of Deviation: Factors that have contributed to the difference between the objective and the actual result.

Steps to Perform a Deviation Analysis

  1. Define Objectives:

    • Establish clear and measurable goals.
    • Example: Increase sales by 10% in the next quarter.

  2. Measure Actual Results:

    • Collect accurate and updated data.
    • Example: Sales increased by 7% in the last quarter.

  3. Calculate the Deviation:

    • Compare actual results with objectives.
    • Formula: Deviation = Actual Result - Objective
    • Example: Deviation = 7% - 10% = -3%

  4. Identify the Causes of the Deviation:

    • Analyze the factors that have influenced the deviation.
    • Example: Supply chain issues, changes in market demand, etc.

  5. Develop Corrective Strategies:

    • Propose actions to correct the deviations.
    • Example: Improve supply chain efficiency, adjust marketing strategies, etc.

Practical Example

Suppose a software company has set a goal to reduce the development time of a new product by 20%. After a quarter, it is observed that the development time has only been reduced by 15%.

Step 1: Define Objectives

Objective: Reduce development time by 20%.

Step 2: Measure Actual Results

Actual Result: Reduction in development time by 15%.

Step 3: Calculate the Deviation

Deviation = 15% - 20% = -5%

Step 4: Identify the Causes of the Deviation

  • Cause 1: Lack of trained personnel.
  • Cause 2: Unforeseen technical problems.
  • Cause 3: Delays in project approval.

Step 5: Develop Corrective Strategies

  • Strategy 1: Hire and train new personnel.
  • Strategy 2: Implement a contingency plan for technical problems.
  • Strategy 3: Improve the project approval process.

Conclusion

Deviation analysis is an essential tool in strategic thinking that allows organizations to identify and correct the differences between objectives and actual results. By following a structured process, companies can develop effective strategies to achieve their goals and improve their overall performance.
