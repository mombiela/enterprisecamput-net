Introduction

Tracking and control in project management are crucial to ensure that the project stays on the right path and meets the established objectives. In this section, we will explore practical exercises that will help you apply tracking and control techniques in your projects.

Key Concepts

  • Project Tracking: The process of collecting, measuring, and distributing information about the project's performance.
  • Project Control: The process of comparing actual performance with planned performance and taking corrective actions when necessary.
  • Performance Indicators: Metrics used to evaluate the project's progress, such as Earned Value (EV), Actual Cost (AC), and Planned Value (PV).
  • Tracking Tools: Software and techniques used to monitor the project's progress, such as Gantt charts, control charts, and dashboards.

Exercise 1: Tracking Progress with Gantt Charts

Objective

Learn to use a Gantt chart to track the progress of a project.

Instructions

  1. Create a Gantt Chart:

    • Use a tool like Microsoft Project, Trello, or any project management software you prefer.
    • Define the project's tasks and their durations.
    • Assign resources to each task.

  2. Update Progress:

    • Mark completed tasks.
    • Update the percentage of progress of ongoing tasks.

  3. Analyze the Chart:

    • Identify delayed tasks.
    • Evaluate the impact of delays on the overall project schedule.

Example

Task             | Duration | Start Date   | End Date    | Progress
-----------------|----------|--------------|-------------|---------
Planning         | 5 days   | 01/10/2023   | 05/10/2023  | 100%
Development      | 10 days  | 06/10/2023   | 15/10/2023  | 50%
Testing          | 5 days   | 16/10/2023   | 20/10/2023  | 0%
Implementation   | 3 days   | 21/10/2023   | 23/10/2023  | 0%

Exercise 2: Using Performance Indicators

Objective

Calculate and analyze the project's performance indicators.

Instructions

  1. Collect Data:

    • Planned Value (PV): $10,000
    • Actual Cost (AC): $8,000
    • Earned Value (EV): $9,000

  2. Calculate Indicators:

    • CPI (Cost Performance Index): EV / AC
    • SPI (Schedule Performance Index): EV / PV

  3. Interpret Results:

    • A CPI > 1 indicates that the project is under budget.
    • An SPI > 1 indicates that the project is ahead of schedule.

Example

CPI = EV / AC = 9000 / 8000 = 1.125
SPI = EV / PV = 9000 / 10000 = 0.9

Interpretation:

  • The project is under budget (CPI > 1).
  • The project is behind schedule (SPI < 1).

Exercise 3: Implementing Corrective Actions

Objective

Develop a plan of corrective actions to address the problems identified in the project's tracking.

Instructions

  1. Identify Problems:

    • Delayed tasks.
    • Budget deviations.

  2. Develop Corrective Actions:

    • Reassign resources to critical tasks.
    • Adjust the schedule.
    • Review the budget and reallocate funds if necessary.

  3. Implement and Monitor:

    • Apply the corrective actions.
    • Monitor the impact of these actions on the project's progress.

Example

Problem: Delay in the Development task.
Corrective Action: Assign additional resources to speed up development.
Monitoring: Review progress daily and adjust as necessary.

Conclusion

Tracking and control are essential components of project management. Through these exercises, you have learned to use tools like Gantt charts, performance indicators, and corrective actions to keep your project on the right track. Practicing these techniques will allow you to manage projects more effectively and efficiently.
