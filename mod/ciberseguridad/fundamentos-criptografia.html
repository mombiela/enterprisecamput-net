Introduction to Cryptography

Cryptography is a fundamental discipline in the field of cybersecurity that deals with protecting information through mathematical techniques. Its main objective is to ensure the confidentiality, integrity, and authenticity of data.

Key Concepts

  • Confidentiality: Ensures that information is only accessible to authorized persons.
  • Integrity: Ensures that information has not been altered in an unauthorized manner.
  • Authenticity: Verifies the identity of the parties involved in the communication.
  • Non-repudiation: Prevents a party from denying having performed an action.

Types of Cryptography

Symmetric Cryptography

In symmetric cryptography, the same key is used to both encrypt and decrypt the information.

  • Advantages:
    • Fast and efficient.
    • Lower resource consumption.
  • Disadvantages:
    • Secure key distribution is a challenge.
    • Not scalable for large systems.

Example of Symmetric Cryptography

from Crypto.Cipher import AES
from Crypto.Random import get_random_bytes

# Generate a 16-byte key
key = get_random_bytes(16)

# Create an AES cipher object
cipher = AES.new(key, AES.MODE_EAX)

# Message to encrypt
data = b"Secret message"

# Encrypt the message
nonce = cipher.nonce
ciphertext, tag = cipher.encrypt_and_digest(data)

print("Encrypted message:", ciphertext)

Asymmetric Cryptography

Asymmetric cryptography uses a pair of keys: a public key to encrypt and a private key to decrypt.

  • Advantages:
    • Facilitates key distribution.
    • Scalable for large systems.
  • Disadvantages:
    • Slower and consumes more resources.
    • Complexity in key management.

Example of Asymmetric Cryptography

from Crypto.PublicKey import RSA
from Crypto.Cipher import PKCS1_OAEP

# Generate an RSA key pair
key = RSA.generate(2048)
public_key = key.publickey()

# Create an RSA cipher object
cipher_rsa = PKCS1_OAEP.new(public_key)

# Message to encrypt
data = b"Secret message"

# Encrypt the message
ciphertext = cipher_rsa.encrypt(data)

print("Encrypted message:", ciphertext)

Common Cryptographic Algorithms

| Algorithm | Type | Main Use | |----------------|-------------|---------------------| | AES | Symmetric | Data encryption | | DES | Symmetric | Data encryption | | RSA | Asymmetric | Encryption and signatures | | ECC | Asymmetric | Encryption and signatures | | SHA-256 | Hash | Data integrity |

Applications of Cryptography

  • Data encryption: Protects stored and transmitted information.
  • Digital signatures: Verifies the authenticity and integrity of messages.
  • Digital certificates: Authenticates the identity of entities in a network.
  • Secure protocols: SSL/TLS for secure web communications.

Conclusion

Cryptography is an essential tool in cybersecurity that allows protecting information against unauthorized access and alterations. Understanding the fundamentals of cryptography, including the types of cryptography and common algorithms, is crucial for any IT professional who wishes to ensure the integrity and confidentiality of data in a digital environment.
