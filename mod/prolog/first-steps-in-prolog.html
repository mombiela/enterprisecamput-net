Introduction to Prolog

Prolog (Programming in Logic) is a high-level programming language associated with artificial intelligence and computational linguistics. Prolog is based on formal logic and allows for declarative programming, where you describe the problem rather than the solution.

Key Concepts

  • Facts: Basic assertions about the world.
  • Rules: Logical statements that define relationships between facts.
  • Queries: Questions asked to the Prolog system to retrieve information based on facts and rules.

Writing Facts

Facts are the simplest form of Prolog statements. They declare information that is unconditionally true.

Example

parent(tom, bob).
parent(bob, alice).

In this example:

  • parent(tom, bob). states that Tom is a parent of Bob.
  • parent(bob, alice). states that Bob is a parent of Alice.

Writing Rules

Rules define relationships between facts using logical implications. They typically consist of a head and a body, separated by :-.

Example

grandparent(X, Y) :- parent(X, Z), parent(Z, Y).

This rule states that X is a grandparent of Y if X is a parent of Z and Z is a parent of Y.

Making Queries

Queries are used to ask questions about the information stored in the Prolog database.

Example

?- parent(tom, bob).

This query asks if Tom is a parent of Bob. Prolog will respond with true.

Example with Variables

?- parent(X, alice).

This query asks who is a parent of Alice. Prolog will respond with X = bob.

Practical Examples

Example 1: Family Relationships

% Facts
parent(tom, bob).
parent(bob, alice).
parent(alice, john).

% Rules
grandparent(X, Y) :- parent(X, Z), parent(Z, Y).
ancestor(X, Y) :- parent(X, Y).
ancestor(X, Y) :- parent(X, Z), ancestor(Z, Y).

% Queries
?- grandparent(tom, alice).
% Expected output: true

?- ancestor(tom, john).
% Expected output: true

Example 2: Simple Database

% Facts
book('The Hobbit', tolkien).
book('1984', orwell).
book('Brave New World', huxley).

% Queries
?- book('1984', Author).
% Expected output: Author = orwell.

?- book(Title, tolkien).
% Expected output: Title = 'The Hobbit'.

Summary

In this section, we covered the basics of Prolog, including:

  • Facts: Simple assertions about the world.
  • Rules: Logical statements that define relationships.
  • Queries: Questions to retrieve information.

By understanding these fundamental concepts, you can start writing simple Prolog programs and querying information. In the next sections, we will delve deeper into more advanced topics and complex problem-solving using Prolog.
