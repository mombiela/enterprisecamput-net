Introduction

In this section of the course, you will learn to use formulas and calculated fields in Google Data Studio. These elements are essential for transforming and analyzing data effectively, allowing you to create more dynamic and personalized reports.

Key Concepts

  • Calculated Fields: These are fields created using formulas to perform calculations or transformations on existing data.
  • Formulas: Mathematical or logical expressions used to manipulate data within calculated fields.

Creating Calculated Fields

To create a calculated field in Google Data Studio, follow these steps:

  1. Open your report in Google Data Studio.
  2. Select the data source to which you want to add a calculated field.
  3. Click on "Add a field" in the upper right corner.
  4. Enter a name for the new field.
  5. Write the formula you want to use.
  6. Click on "Create field."

Example of a Calculated Field

Suppose you have a data source with the following columns: Sales, Costs, and Date. You want to create a calculated field to obtain the profit (Profit = Sales - Costs).

  1. Field name: Profit
  2. Formula: Sales - Costs
Profit = Sales - Costs

Types of Formulas

Mathematical Formulas

  • Addition: Sales + Taxes
  • Subtraction: Sales - Costs
  • Multiplication: Price * Quantity
  • Division: Revenue / Number of Sales

Logical Formulas

  • IF: IF(Condition, Value if true, Value if false)

Example:

IF(Sales > 1000, "High", "Low")

Text Formulas

  • CONCAT: CONCAT(Field1, Field2)
  • LOWER: LOWER(Field)
  • UPPER: UPPER(Field)

Example:

CONCAT(LOWER(Name), "@company.com")

Date Formulas

  • YEAR: YEAR(Date)
  • MONTH: MONTH(Date)
  • DAY: DAY(Date)

Example:

YEAR(Date)

Practical Examples

Example 1: Calculate the Profit Margin

Suppose you have the columns Revenue and Costs. You want to calculate the profit margin as a percentage.

  1. Field name: Profit Margin
  2. Formula: ((Revenue - Costs) / Revenue) * 100
Profit_Margin = ((Revenue - Costs) / Revenue) * 100

Example 2: Classify Sales

You want to classify sales as "High", "Medium", and "Low" based on their value.

  1. Field name: Sales Classification
  2. Formula:
IF(Sales > 1000, "High", IF(Sales > 500, "Medium", "Low"))

Comparison of Formulas and Calculated Fields

| Type of Formula | Example | Description | |-----------------|---------|-------------| | Mathematical | Sales - Costs | Performs arithmetic operations. | | Logical | IF(Sales > 1000, "High", "Low") | Evaluates conditions and returns values based on the result. | | Text | CONCAT(Name, Surname) | Manipulates text strings. | | Date | YEAR(Date) | Extracts specific parts of a date. |

Conclusion

The use of formulas and calculated fields in Google Data Studio is an essential skill for any professional working with data. These tools allow you to transform and analyze data more effectively, creating more dynamic and personalized reports. Practice creating your own calculated fields and experiment with different formulas to see how they can enhance your reports.
