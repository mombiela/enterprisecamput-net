Introduction

In this section of the course on Google Data Studio, you will learn how to connect and use various external data sources. Google Data Studio allows you to integrate data from multiple origins, making it easier to create comprehensive and customized reports and dashboards.

Compatible Data Sources

Google Data Studio is compatible with a wide variety of data sources. Some of the most common include:

  • Google Sheets
  • Google Analytics
  • BigQuery
  • MySQL
  • PostgreSQL
  • CSV Uploads
  • Facebook Ads
  • YouTube Analytics
  • Salesforce

Connecting Google Sheets

Google Sheets is one of the most popular data sources due to its simplicity and accessibility. To connect Google Sheets to Google Data Studio, follow these steps:

  1. Open Google Data Studio: Log in to your Google Data Studio account.
  2. Create a Data Source: Click the "Create" button and select "Data source."
  3. Select Google Sheets: From the list of connectors, select "Google Sheets."
  4. Authorize Access: Authorize Google Data Studio to access your Google Sheets.
  5. Select the File and Sheet: Choose the Google Sheets file and the specific sheet you want to connect.
  6. Configure the Fields: Review and configure the data fields as needed.
# Example of field configuration
{
  "fieldName": "Date",
  "fieldType": "DATE"
},
{
  "fieldName": "Sales",
  "fieldType": "NUMBER"
}

Connecting BigQuery

BigQuery is a powerful data analysis tool from Google Cloud. To connect BigQuery to Google Data Studio:

  1. Open Google Data Studio: Log in to your Google Data Studio account.
  2. Create a Data Source: Click the "Create" button and select "Data source."
  3. Select BigQuery: From the list of connectors, select "BigQuery."
  4. Authorize Access: Authorize Google Data Studio to access your BigQuery projects.
  5. Select the Project and Dataset: Choose the project and dataset you want to connect.
  6. Configure the Fields: Review and configure the data fields as needed.
# Example of SQL query in BigQuery
SELECT
  date,
  sales
FROM
  `my_project.my_dataset.my_table`
WHERE
  date BETWEEN '2023-01-01' AND '2023-12-31'

Connecting MySQL

To connect a MySQL database to Google Data Studio:

  1. Open Google Data Studio: Log in to your Google Data Studio account.
  2. Create a Data Source: Click the "Create" button and select "Data source."
  3. Select MySQL: From the list of connectors, select "MySQL."
  4. Configure the Connection: Enter the connection details, such as the server address, port, username, and password.
  5. Select the Database and Table: Choose the database and table you want to connect.
  6. Configure the Fields: Review and configure the data fields as needed.
# Example of SQL query in MySQL
SELECT
  date,
  sales
FROM
  my_table
WHERE
  date BETWEEN '2023-01-01' AND '2023-12-31'

Conclusion

Connecting various data sources to Google Data Studio allows you to create more comprehensive and customized reports and dashboards. Whether you use Google Sheets, BigQuery, MySQL, or other sources, the connection process is relatively straightforward and gives you the flexibility to work with the data you need.
