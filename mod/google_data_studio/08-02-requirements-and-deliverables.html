Requirements and Deliverables
Next ►

In this section, we will outline the requirements and deliverables for the final project of the Google Data Studio course. This project is designed to consolidate your learning and demonstrate your ability to create comprehensive, interactive reports using Google Data Studio.

Project Requirements

To successfully complete the final project, you must meet the following requirements:

  1. Data Source Integration:

    • Connect at least two different data sources to your report. These can include Google Sheets, Google Analytics, BigQuery, or any other compatible data source.
    • Ensure that the data sources are properly managed and updated.

  2. Report Creation:

    • Create a new report from scratch.
    • The report should include at least five pages, each focusing on different aspects of the data.

  3. Charts and Visualizations:

    • Include a variety of charts and visualizations, such as bar charts, line charts, pie charts, and tables.
    • Each page should have at least two different types of visualizations.

  4. Customization:

    • Apply themes and styles to ensure a consistent and professional look throughout the report.
    • Customize charts and tables to enhance readability and visual appeal.

  5. Interactivity:

    • Add filters and interactive controls to allow users to explore the data dynamically.
    • Include at least one date range control and one filter control.

  6. Advanced Features:

    • Use calculated fields to create new metrics or dimensions.
    • Create segments or custom metrics to provide deeper insights into the data.
    • Integrate with other Google tools, such as Google Ads or Google Search Console, if applicable.

  7. Collaboration and Sharing:

    • Share the report with at least one other user for collaboration.
    • Ensure that the report is accessible and can be viewed by others.

  8. Documentation:

    • Provide a brief documentation (1-2 pages) explaining the data sources, the purpose of the report, and the key insights derived from the data.
    • Include screenshots of the report pages and visualizations.

Deliverables

Upon completion of the final project, you are required to submit the following deliverables:

  1. Google Data Studio Report:

    • A link to the final Google Data Studio report, ensuring that it is accessible for review.
    • The report should meet all the requirements outlined above.

  2. Documentation:

    • A PDF or Word document containing the project documentation.
    • The documentation should include:
      • An introduction explaining the purpose of the report.
      • A description of the data sources used.
      • An overview of the report structure and the key insights from each page.
      • Screenshots of the report pages and visualizations.

  3. Collaboration Evidence:

    • A screenshot or any other evidence showing that the report has been shared with at least one other user for collaboration.

Submission Guidelines

  • Ensure that all deliverables are complete and meet the specified requirements.
  • Submit the Google Data Studio report link and the documentation file through the designated submission platform or email.
  • Double-check that the report link is accessible and that all visualizations and interactive elements are functioning correctly.

Evaluation Criteria

Your final project will be evaluated based on the following criteria:

  1. Completeness:

    • All required elements are present and correctly implemented.

  2. Accuracy:

    • Data is accurately represented and visualizations are correctly configured.

  3. Design and Usability:

    • The report is visually appealing, well-organized, and easy to navigate.
    • Interactive elements enhance the user experience.

  4. Insightfulness:

    • The report provides meaningful insights and effectively communicates the data story.

  5. Documentation:

    • The documentation is clear, concise, and provides a comprehensive overview of the project.

Conclusion

Completing the final project will demonstrate your proficiency in using Google Data Studio to create interactive and insightful reports. This project will serve as a valuable portfolio piece, showcasing your ability to manage data sources, create visualizations, and customize reports to meet specific needs. Good luck, and we look forward to seeing your impressive reports!
