Example 2: Answering a Question on Stack Overflow

**Question**: How can I create a stacked bar chart in Google Data Studio?

**Answer**:
To create a stacked bar chart in Google Data Studio, follow these steps:

1. Open your report in Google Data Studio.
2. Click on "Add a chart" and select "Bar chart".
3. In the chart configuration panel, select the dimensions and metrics you want to visualize.
4. In the "Style" section, enable the "Stacked bars" option.

Here is an example of how the configuration code would look:

{ "chartType": "BAR", "dimensions": ["Category"], "metrics": ["Sales", "Profit"], "stacked": true }

I hope this helps.
