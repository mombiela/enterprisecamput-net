Introduction to Conditionals

Conditionals are fundamental in programming as they allow you to execute different code based on certain conditions. In Kotlin, conditionals are primarily handled using if, else if, and else statements, as well as the when expression.

If-Else Statements

The if statement is the most basic form of conditional. It allows you to execute a block of code only if a specified condition is true.

Syntax

if (condition) {
    // code to execute if condition is true
} else {
    // code to execute if condition is false
}

Example

val number = 10

if (number > 0) {
    println("The number is positive")
} else {
    println("The number is not positive")
}

Explanation

  • if (number > 0): Checks if number is greater than 0.
  • println("The number is positive"): Executes if the condition is true.
  • else: Executes if the condition is false.
  • println("The number is not positive"): Executes if the condition is false.

Else If Statements

When you have multiple conditions to check, you can use else if.

Syntax

if (condition1) {
    // code to execute if condition1 is true
} else if (condition2) {
    // code to execute if condition2 is true
} else {
    // code to execute if none of the above conditions are true
}

Example

val number = 0

if (number > 0) {
    println("The number is positive")
} else if (number < 0) {
    println("The number is negative")
} else {
    println("The number is zero")
}

Explanation

  • else if (number < 0): Checks if number is less than 0.
  • println("The number is negative"): Executes if the second condition is true.
  • else: Executes if none of the conditions are true.
  • println("The number is zero"): Executes if none of the conditions are true.

When Expression

The when expression in Kotlin is a powerful alternative to the if-else if-else chain. It is similar to the switch statement in other languages.

Syntax

when (variable) {
    value1 -> {
        // code to execute if variable == value1
    }
    value2 -> {
        // code to execute if variable == value2
    }
    else -> {
        // code to execute if none of the above conditions are true
    }
}

Example

val dayOfWeek = 3

val dayName = when (dayOfWeek) {
    1 -> "Monday"
    2 -> "Tuesday"
    3 -> "Wednesday"
    4 -> "Thursday"
    5 -> "Friday"
    6 -> "Saturday"
    7 -> "Sunday"
    else -> "Invalid day"
}

println(dayName)

Explanation

  • when (dayOfWeek): Evaluates the value of dayOfWeek.
  • 1 -> "Monday": Executes if dayOfWeek is 1.
  • else -> "Invalid day": Executes if none of the conditions match.

Advanced Usage of When

The when expression can also be used without an argument, acting like an advanced if-else chain.

Example

val number = -10

val result = when {
    number > 0 -> "Positive"
    number < 0 -> "Negative"
    else -> "Zero"
}

println(result)

Explanation

  • when {}: No argument is provided.
  • number > 0 -> "Positive": Executes if number is greater than 0.
  • number < 0 -> "Negative": Executes if number is less than 0.
  • else -> "Zero": Executes if none of the conditions match.

Conclusion

Conditionals are essential for controlling the flow of your Kotlin programs. The if-else statements are straightforward and useful for simple conditions, while the when expression offers a more readable and flexible way to handle multiple conditions. Understanding and using these constructs effectively will allow you to write more dynamic and responsive code.
