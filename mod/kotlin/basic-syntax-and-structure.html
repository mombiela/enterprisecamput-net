Introduction to Kotlin

Kotlin is a modern, statically-typed programming language that runs on the Java Virtual Machine (JVM) and can also be compiled to JavaScript or native code. It is designed to be fully interoperable with Java and offers a more concise syntax, making it a popular choice for Android development and other applications.

Key Concepts

  • Variables and Constants
  • Data Types
  • Control Flow
  • Functions
  • Classes and Objects

Variables and Constants

In Kotlin, you can declare variables using var and constants using val.

Variables

var mutableVariable: Int = 10
mutableVariable = 20 // This is allowed
  • var is used to declare a mutable variable, meaning its value can be changed.

Constants

val immutableVariable: Int = 10
immutableVariable = 20 // This will cause a compilation error
  • val is used to declare a read-only variable, meaning its value cannot be changed once assigned.

Data Types

Kotlin has a rich set of built-in data types.

Basic Data Types

  • Numbers: Int, Long, Float, Double
  • Characters: Char
  • Booleans: Boolean
  • Strings: String

Example

val anInt: Int = 42
val aDouble: Double = 3.14
val aChar: Char = 'A'
val aBoolean: Boolean = true
val aString: String = "Hello, Kotlin!"

Control Flow

Kotlin provides several ways to control the flow of your program.

If-Else

val max = if (a > b) a else b
  • The if statement can be used as an expression.

When

val result = when (x) {
    1 -> "One"
    2 -> "Two"
    else -> "Unknown"
}
  • when is a more powerful version of the switch statement in other languages.

Loops

For Loop

for (i in 1..10) {
    println(i)
}
  • The for loop iterates over a range.

While Loop

var i = 0
while (i < 10) {
    println(i)
    i++
}
  • The while loop continues as long as the condition is true.

Functions

Functions in Kotlin are first-class citizens and can be declared in various ways.

Basic Function

fun add(a: Int, b: Int): Int {
    return a + b
}
  • Functions are declared using the fun keyword.

Single-Expression Function

fun add(a: Int, b: Int): Int = a + b
  • Functions with a single expression can be written more concisely.

Default Arguments

fun greet(name: String = "World") {
    println("Hello, $name!")
}
  • Functions can have default parameter values.

Classes and Objects

Kotlin supports object-oriented programming with classes and objects.

Basic Class

class Person(val name: String, var age: Int)
  • A class can have properties declared in the primary constructor.

Creating an Object

val person = Person("John", 30)
println(person.name) // John
println(person.age)  // 30
  • Objects are created using the new keyword.

Methods in Classes

class Person(val name: String, var age: Int) {
    fun greet() {
        println("Hello, my name is $name and I am $age years old.")
    }
}
  • Methods can be defined inside classes.

Conclusion

Understanding the basic syntax and structure of Kotlin is crucial for writing effective and efficient code. This includes knowing how to declare variables and constants, understanding data types, controlling the flow of your program, defining functions, and creating classes and objects. Mastery of these fundamental concepts will provide a strong foundation for more advanced Kotlin programming topics.
