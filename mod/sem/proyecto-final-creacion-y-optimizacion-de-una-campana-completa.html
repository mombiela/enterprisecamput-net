Introducción

En este proyecto final, aplicarás todo lo aprendido sobre SEM (Marketing en Motores de Búsqueda) para crear y optimizar una campaña completa. Este proyecto te permitirá consolidar tus conocimientos y adquirir experiencia práctica en la gestión de campañas SEM.

Objetivos del Proyecto

  • Crear una campaña SEM desde cero.
  • Optimizar la campaña para mejorar su rendimiento.
  • Analizar los resultados y realizar ajustes basados en datos.

Paso 1: Investigación de Palabras Clave

La investigación de palabras clave es el primer paso crucial en la creación de una campaña SEM efectiva.

  • Herramientas de Investigación:

    • Google Keyword Planner
    • SEMrush
    • Ahrefs

  • Tipos de Palabras Clave:

    • Palabras clave de cola corta: Tienen un alto volumen de búsqueda pero también alta competencia.
    • Palabras clave de cola larga: Tienen un menor volumen de búsqueda pero son más específicas y tienen menor competencia.
# Ejemplo de uso de la API de Google Keyword Planner para obtener ideas de palabras clave
from googleads import adwords

def obtener_ideas_palabras_clave(client, palabras_clave):
    targeting_idea_service = client.GetService('TargetingIdeaService', version='v201809')
    selector = {
        'ideaType': 'KEYWORD',
        'requestType': 'IDEAS',
        'requestedAttributeTypes': ['KEYWORD_TEXT', 'SEARCH_VOLUME'],
        'searchParameters': [
            {
                'xsi_type': 'RelatedToQuerySearchParameter',
                'queries': palabras_clave
            }
        ],
        'paging': {
            'startIndex': 0,
            'numberResults': 10
        }
    }
    page = targeting_idea_service.get(selector)
    for result in page['entries']:
        data = result['data']
        print(f"Palabra clave: {data['KEYWORD_TEXT']['value']}, Volumen de búsqueda: {data['SEARCH_VOLUME']['value']}")

# Ejemplo de uso
client = adwords.AdWordsClient.LoadFromStorage()
obtener_ideas_palabras_clave(client, ['marketing digital'])

Paso 2: Creación de la Campaña

Una vez que tienes tus palabras clave, el siguiente paso es crear la campaña en Google Ads.

  • Configuración de la Campaña:
    • Objetivo de la campaña: Ventas, leads, tráfico del sitio web, etc.
    • Presupuesto diario: Define cuánto estás dispuesto a gastar por día.
    • Segmentación geográfica: Selecciona las ubicaciones donde quieres que se muestren tus anuncios.
    • Segmentación por dispositivo: Decide si quieres mostrar tus anuncios en móviles, computadoras, o ambos.
# Ejemplo de creación de una campaña en Google Ads
from googleads import adwords

def crear_campaña(client, nombre_campaña, presupuesto_diario, ubicaciones):
    campaign_service = client.GetService('CampaignService', version='v201809')
    operations = [{
        'operator': 'ADD',
        'operand': {
            'name': nombre_campaña,
            'status': 'PAUSED',
            'advertisingChannelType': 'SEARCH',
            'biddingStrategyConfiguration': {
                'biddingStrategyType': 'MANUAL_CPC'
            },
            'budget': {
                'budgetId': presupuesto_diario
            },
            'settings': [
                {
                    'xsi_type': 'GeoTargetTypeSetting',
                    'positiveGeoTargetType': 'DONT_CARE',
                    'negativeGeoTargetType': 'DONT_CARE'
                }
            ],
            'networkSetting': {
                'targetGoogleSearch': True,
                'targetSearchNetwork': True,
                'targetContentNetwork': False,
                'targetPartnerSearchNetwork': False
            },
            'startDate': '20230101',
            'endDate': '20231231',
            'adServingOptimizationStatus': 'ROTATE'
        }
    }]
    campaign_service.mutate(operations)

# Ejemplo de uso
client = adwords.AdWordsClient.LoadFromStorage()
crear_campaña(client, 'Mi Primera Campaña', 5000000, ['ES'])

Paso 3: Creación de Grupos de Anuncios y Anuncios

Dentro de tu campaña, crearás grupos de anuncios que contengan anuncios específicos y palabras clave relacionadas.

  • Grupos de Anuncios:

    • Nombre del grupo de anuncios: Debe ser descriptivo y reflejar el tema del grupo.
    • Palabras clave: Asigna las palabras clave relevantes a cada grupo de anuncios.

  • Anuncios:

    • Título: Atrae la atención del usuario.
    • Descripción: Explica claramente lo que ofreces.
    • URL visible: La URL que se muestra en el anuncio.
    • URL de destino: La URL a la que se dirige el usuario al hacer clic.
# Ejemplo de creación de un grupo de anuncios y anuncios en Google Ads
from googleads import adwords

def crear_grupo_anuncios(client, campaign_id, nombre_grupo, palabras_clave, anuncios):
    ad_group_service = client.GetService('AdGroupService', version='v201809')
    operations = [{
        'operator': 'ADD',
        'operand': {
            'campaignId': campaign_id,
            'name': nombre_grupo,
            'status': 'ENABLED',
            'biddingStrategyConfiguration': {
                'bids': [{
                    'xsi_type': 'CpcBid',
                    'bid': {
                        'microAmount': 1000000
                    }
                }]
            }
        }
    }]
    ad_group_service.mutate(operations)

    ad_group_criterion_service = client.GetService('AdGroupCriterionService', version='v201809')
    for palabra_clave in palabras_clave:
        operations = [{
            'operator': 'ADD',
            'operand': {
                'xsi_type': 'BiddableAdGroupCriterion',
                'adGroupId': ad_group_id,
                'criterion': {
                    'xsi_type': 'Keyword',
                    'text': palabra_clave,
                    'matchType': 'BROAD'
                }
            }
        }]
        ad_group_criterion_service.mutate(operations)

    ad_group_ad_service = client.GetService('AdGroupAdService', version='v201809')
    for anuncio in anuncios:
        operations = [{
            'operator': 'ADD',
            'operand': {
                'adGroupId': ad_group_id,
                'ad': {
                    'xsi_type': 'ExpandedTextAd',
                    'headlinePart1': anuncio['titulo1'],
                    'headlinePart2': anuncio['titulo2'],
                    'description': anuncio['descripcion'],
                    'finalUrls': [anuncio['url_destino']]
                },
                'status': 'ENABLED'
            }
        }]
        ad_group_ad_service.mutate(operations)

# Ejemplo de uso
client = adwords.AdWordsClient.LoadFromStorage()
crear_grupo_anuncios(client, '1234567890', 'Grupo de Anuncios 1', ['palabra clave 1', 'palabra clave 2'], [{'titulo1': 'Título 1', 'titulo2': 'Título 2', 'descripcion': 'Descripción del anuncio', 'url_destino': 'http://www.ejemplo.com'}])

Paso 4: Optimización de la Campaña

La optimización es un proceso continuo que implica ajustar y mejorar la campaña para obtener mejores resultados.

  • Ajustes de Oferta:

    • Por dispositivo: Aumenta o disminuye las ofertas según el rendimiento en diferentes dispositivos.
    • Por ubicación: Ajusta las ofertas según el rendimiento en diferentes ubicaciones geográficas.

  • Análisis de Rendimiento:

    • CTR (Click-Through Rate): Porcentaje de clics en relación con las impresiones.
    • CPC (Cost-Per-Click): Costo promedio por clic.
    • Conversiones: Número de acciones deseadas realizadas por los usuarios.
# Ejemplo de ajuste de oferta por dispositivo en Google Ads
from googleads import adwords

def ajustar_oferta_dispositivo(client, campaign_id, dispositivo, ajuste_porcentaje):
    campaign_criterion_service = client.GetService('CampaignCriterionService', version='v201809')
    operations = [{
        'operator': 'SET',
        'operand': {
            'campaignId': campaign_id,
            'criterion': {
                'xsi_type': 'Platform',
                'id': dispositivo
            },
            'bidModifier': ajuste_porcentaje
        }
    }]
    campaign_criterion_service.mutate(operations)

# Ejemplo de uso
client = adwords.AdWordsClient.LoadFromStorage()
ajustar_oferta_dispositivo(client, '1234567890', '30001', 1.2)  # 30001 es el ID para móviles
