Introduction

Elasticsearch is a powerful search and analytics engine that is widely used for various applications, including log and event data analysis, full-text search, and more. This section will guide you through the process of installing Elasticsearch on different operating systems.

Prerequisites

Before installing Elasticsearch, ensure you have the following prerequisites:

  • Java Development Kit (JDK) 8 or higher installed.
  • Sufficient system resources (RAM, CPU, disk space).

Installing Elasticsearch on Different Operating Systems

Installing on Windows

  1. Download Elasticsearch:

    • Visit the Elasticsearch download page.
    • Download the Windows ZIP file.

  2. Extract the ZIP file:

    • Extract the downloaded ZIP file to a directory of your choice, e.g., C:\elasticsearch.

  3. Run Elasticsearch:

    • Open a Command Prompt window.
    • Navigate to the bin directory inside the extracted folder.
    • Run the following command: 
      elasticsearch.bat
    • Elasticsearch will start, and you can access it by navigating to http://localhost:9200 in your web browser.

Installing on macOS

  1. Download Elasticsearch:

    • Visit the Elasticsearch download page.
    • Download the macOS TAR file.

  2. Extract the TAR file:

    • Open a Terminal window.
    • Navigate to the directory where the TAR file was downloaded.
    • Run the following command to extract the file: 
      tar -xzf elasticsearch-<version>.tar.gz

  3. Run Elasticsearch:

    • Navigate to the bin directory inside the extracted folder.
    • Run the following command: 
      ./elasticsearch
    • Elasticsearch will start, and you can access it by navigating to http://localhost:9200 in your web browser.

Installing on Linux

  1. Download Elasticsearch:

    • Visit the Elasticsearch download page.
    • Download the Linux TAR file.

  2. Extract the TAR file:

    • Open a Terminal window.
    • Navigate to the directory where the TAR file was downloaded.
    • Run the following command to extract the file: 
      tar -xzf elasticsearch-<version>.tar.gz

  3. Run Elasticsearch:

    • Navigate to the bin directory inside the extracted folder.
    • Run the following command: 
      ./elasticsearch
    • Elasticsearch will start, and you can access it by navigating to http://localhost:9200 in your web browser.

Installing via Package Managers

Using Homebrew on macOS

  1. Install Homebrew:

    • If you don't have Homebrew installed, install it by running the following command in Terminal: 
      /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

  2. Install Elasticsearch:

    • Run the following command to install Elasticsearch: 
      brew tap elastic/tap
brew install elastic/tap/elasticsearch-full

  3. Run Elasticsearch:

    • Start Elasticsearch by running: 
      brew services start elastic/tap/elasticsearch-full

Using APT on Debian-based Linux

  1. Install APT Transport HTTPS:

    • Run the following command: 
      sudo apt-get install apt-transport-https

  2. Add Elasticsearch GPG Key:

    • Run the following command: 
      wget -qO - https://artifacts.elastic.co/GPG-KEY-elasticsearch | sudo apt-key add -

  3. Add Elasticsearch Repository:

    • Run the following command: 
      echo "deb https://artifacts.elastic.co/packages/7.x/apt stable main" | sudo tee -a /etc/apt/sources.list.d/elastic-7.x.list

  4. Install Elasticsearch:

    • Update the package list and install Elasticsearch: 
      sudo apt-get update
sudo apt-get install elasticsearch

  5. Run Elasticsearch:

    • Start Elasticsearch by running: 
      sudo systemctl start elasticsearch

Verifying the Installation

After installing Elasticsearch, verify that it is running correctly by navigating to http://localhost:9200 in your web browser. You should see a JSON response similar to the following:

{
  "name" : "your-node-name",
  "cluster_name" : "elasticsearch",
  "cluster_uuid" : "your-cluster-uuid",
  "version" : {
    "number" : "7.x.x",
    "build_flavor" : "default",
    "build_type" : "tar",
    "build_hash" : "your-build-hash",
    "build_date" : "your-build-date",
    "build_snapshot" : false,
    "lucene_version" : "8.x.x",
    "minimum_wire_compatibility_version" : "6.x.x",
    "minimum_index_compatibility_version" : "6.x.x"
  },
  "tagline" : "You Know, for Search"
}

Conclusion

In this section, we covered the steps to install Elasticsearch on various operating systems, including Windows, macOS, and Linux. We also discussed how to verify the installation. With Elasticsearch up and running, you are now ready to start exploring its powerful search and analytics capabilities.
