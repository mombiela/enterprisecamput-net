Introduction to Firebase Hosting
Deploying a Web App ►

Firebase Hosting is a fully managed hosting service for static and dynamic content as well as microservices. It provides fast and secure hosting for web applications, static websites, and serverless backend services.

What is Firebase Hosting?

Firebase Hosting is part of the Firebase suite of tools and services provided by Google. It allows developers to deploy web applications and static content to a global content delivery network (CDN) with a single command.

Key Features of Firebase Hosting

  • Fast and Secure: Firebase Hosting serves your content over a secure connection (HTTPS) and uses a global CDN to ensure fast delivery.
  • Custom Domains: You can connect your custom domain to Firebase Hosting.
  • Automatic SSL: Firebase Hosting automatically provisions SSL certificates for your custom domains.
  • Versioning: Every deployment is versioned, allowing you to roll back to previous versions if needed.
  • Integration with Firebase Services: Seamlessly integrates with other Firebase services like Firestore, Authentication, and Cloud Functions.

Setting Up Firebase Hosting

Prerequisites

Before you start, ensure you have the following:

  • A Google account.
  • Node.js and npm installed on your machine.
  • Firebase CLI installed.

Step-by-Step Guide

  1. Install Firebase CLI:

    npm install -g firebase-tools

  2. Login to Firebase:

    firebase login

  3. Initialize Firebase in Your Project: Navigate to your project directory and run:

    firebase init
    • Select "Hosting" from the list of Firebase features.
    • Choose an existing Firebase project or create a new one.
    • Specify the public directory (e.g., public).
    • Configure as a single-page app if applicable.

  4. Deploy Your Site:

    firebase deploy

Example: Deploying a Simple HTML Site

  1. Create a Project Directory:

    mkdir my-firebase-site
cd my-firebase-site

  2. Initialize Firebase:

    firebase init

  3. Create an index.html File:

    <!DOCTYPE html>
<html>
<head>
    <title>My Firebase Site</title>
</head>
<body>
    <h1>Welcome to My Firebase Site!</h1>
</body>
</html>

  4. Deploy the Site:

    firebase deploy

Managing Your Firebase Hosting

Custom Domains

To add a custom domain:

  1. Go to the Firebase Console.
  2. Select your project.
  3. Navigate to the Hosting section.
  4. Click "Add Custom Domain" and follow the instructions.

Rollback to a Previous Version

Firebase Hosting keeps track of your deployment history. To rollback:

  1. Go to the Firebase Console.
  2. Select your project.
  3. Navigate to the Hosting section.
  4. Click on the "View" button next to the version you want to rollback to.
  5. Click "Rollback" to deploy the selected version.

Conclusion

Firebase Hosting is a powerful tool for deploying and managing web applications. It offers a range of features that make it easy to deploy, manage, and secure your web content. By following the steps outlined in this guide, you can quickly get started with Firebase Hosting and take advantage of its robust capabilities.
