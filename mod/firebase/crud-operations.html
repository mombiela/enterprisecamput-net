Introduction to Firebase

Firebase is a platform developed by Google for creating mobile and web applications. It provides a variety of tools and services to help developers build high-quality apps, including a real-time NoSQL database, authentication, analytics, and more.

What are CRUD Operations?

CRUD stands for Create, Read, Update, and Delete. These are the four basic operations that can be performed on data in a database. Understanding how to perform these operations is fundamental to working with any database, including Firebase.

Setting Up Firebase

Before we dive into CRUD operations, let's set up Firebase for your project.

  1. Create a Firebase Project:

    • Go to the Firebase Console.
    • Click on "Add project" and follow the instructions to create a new project.

  2. Add Firebase to Your App:

    • For web apps, add the Firebase SDK to your project.
    • For Android/iOS apps, follow the specific instructions provided in the Firebase Console.

  3. Initialize Firebase in Your App:

    // Import the Firebase modules that you need in your app.
import { initializeApp } from "firebase/app";
import { getFirestore } from "firebase/firestore";

// TODO: Replace the following with your app's Firebase project configuration
const firebaseConfig = {
  apiKey: "YOUR_API_KEY",
  authDomain: "YOUR_AUTH_DOMAIN",
  projectId: "YOUR_PROJECT_ID",
  storageBucket: "YOUR_STORAGE_BUCKET",
  messagingSenderId: "YOUR_MESSAGING_SENDER_ID",
  appId: "YOUR_APP_ID"
};

// Initialize Firebase
const app = initializeApp(firebaseConfig);
const db = getFirestore(app);

Create Operation

Creating data in Firebase involves adding new documents to a collection.

Example: Adding a New Document

import { collection, addDoc } from "firebase/firestore"; 

async function addUser() {
  try {
    const docRef = await addDoc(collection(db, "users"), {
      first: "Ada",
      last: "Lovelace",
      born: 1815
    });
    console.log("Document written with ID: ", docRef.id);
  } catch (e) {
    console.error("Error adding document: ", e);
  }
}

addUser();
  • Explanation:
    • collection(db, "users"): Specifies the collection where the document will be added.
    • addDoc: Adds a new document with the specified data.

Read Operation

Reading data from Firebase involves retrieving documents from a collection.

Example: Getting All Documents in a Collection

import { collection, getDocs } from "firebase/firestore"; 

async function getUsers() {
  const querySnapshot = await getDocs(collection(db, "users"));
  querySnapshot.forEach((doc) => {
    console.log(`${doc.id} => ${doc.data()}`);
  });
}

getUsers();
  • Explanation:
    • getDocs(collection(db, "users")): Retrieves all documents from the "users" collection.
    • querySnapshot.forEach: Iterates over each document in the collection.

Update Operation

Updating data in Firebase involves modifying an existing document.

Example: Updating a Document

import { doc, updateDoc } from "firebase/firestore"; 

async function updateUser(userId) {
  const userRef = doc(db, "users", userId);

  // Set the "capital" field of the city 'DC'
  await updateDoc(userRef, {
    last: "Byron",
    born: 1812
  });
}

updateUser("USER_DOCUMENT_ID");
  • Explanation:
    • doc(db, "users", userId): Specifies the document to be updated.
    • updateDoc: Updates the specified fields of the document.

Delete Operation

Deleting data in Firebase involves removing a document from a collection.

Example: Deleting a Document

import { doc, deleteDoc } from "firebase/firestore"; 

async function deleteUser(userId) {
  await deleteDoc(doc(db, "users", userId));
}

deleteUser("USER_DOCUMENT_ID");
  • Explanation:
    • doc(db, "users", userId): Specifies the document to be deleted.
    • deleteDoc: Deletes the specified document.

Conclusion

CRUD operations are fundamental to managing data in any database, including Firebase. By mastering these operations, you can effectively create, read, update, and delete data in your Firebase applications. Practice these operations to become proficient in handling Firebase Firestore, and explore more advanced features as you grow more comfortable with the basics.
