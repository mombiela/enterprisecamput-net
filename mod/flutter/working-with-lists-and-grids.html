Introduction

In this section, we will explore how to work with lists and grids in Flutter. Lists and grids are essential components for displaying collections of data in a structured and visually appealing manner. By the end of this section, you will be able to create dynamic lists and grids, customize their appearance, and handle user interactions.

Lists in Flutter

Basic ListView

The ListView widget is the most commonly used widget for displaying a list of items in Flutter. It can be used to create both static and dynamic lists.

Static ListView

A static ListView is useful when you have a fixed number of items to display.

import 'package:flutter/material.dart';

void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        appBar: AppBar(title: Text('Static ListView')),
        body: ListView(
          children: <Widget>[
            ListTile(
              leading: Icon(Icons.map),
              title: Text('Map'),
            ),
            ListTile(
              leading: Icon(Icons.photo_album),
              title: Text('Album'),
            ),
            ListTile(
              leading: Icon(Icons.phone),
              title: Text('Phone'),
            ),
          ],
        ),
      ),
    );
  }
}

Dynamic ListView

A dynamic ListView is useful when the number of items is not fixed and can change during runtime.

import 'package:flutter/material.dart';

void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  final List<String> items = List<String>.generate(100, (i) => "Item $i");

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        appBar: AppBar(title: Text('Dynamic ListView')),
        body: ListView.builder(
          itemCount: items.length,
          itemBuilder: (context, index) {
            return ListTile(
              title: Text('${items[index]}'),
            );
          },
        ),
      ),
    );
  }
}

Customizing List Items

You can customize the appearance of list items using various widgets and properties.

ListTile(
  leading: Icon(Icons.map),
  title: Text('Map'),
  subtitle: Text('Navigate the world'),
  trailing: Icon(Icons.navigate_next),
  onTap: () {
    // Handle tap event
  },
);

Grids in Flutter

Basic GridView

The GridView widget is used to display items in a grid layout. It can be used to create both static and dynamic grids.

Static GridView

A static GridView is useful when you have a fixed number of items to display.

import 'package:flutter/material.dart';

void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        appBar: AppBar(title: Text('Static GridView')),
        body: GridView.count(
          crossAxisCount: 2,
          children: <Widget>[
            Container(
              color: Colors.red,
              child: Center(child: Text('Item 1')),
            ),
            Container(
              color: Colors.blue,
              child: Center(child: Text('Item 2')),
            ),
            Container(
              color: Colors.green,
              child: Center(child: Text('Item 3')),
            ),
            Container(
              color: Colors.yellow,
              child: Center(child: Text('Item 4')),
            ),
          ],
        ),
      ),
    );
  }
}

Dynamic GridView

A dynamic GridView is useful when the number of items is not fixed and can change during runtime.

import 'package:flutter/material.dart';

void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  final List<String> items = List<String>.generate(100, (i) => "Item $i");

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        appBar: AppBar(title: Text('Dynamic GridView')),
        body: GridView.builder(
          gridDelegate: SliverGridDelegateWithFixedCrossAxisCount(
            crossAxisCount: 2,
          ),
          itemCount: items.length,
          itemBuilder: (context, index) {
            return Card(
              child: Center(child: Text('${items[index]}')),
            );
          },
        ),
      ),
    );
  }
}

Customizing Grid Items

You can customize the appearance of grid items using various widgets and properties.

Card(
  child: Column(
    mainAxisAlignment: MainAxisAlignment.center,
    children: <Widget>[
      Icon(Icons.map, size: 50),
      Text('Map'),
    ],
  ),
);

Conclusion

In this section, we covered the basics of working with lists and grids in Flutter. We learned how to create both static and dynamic lists and grids, and how to customize their appearance. Lists and grids are powerful tools for displaying data in a structured and visually appealing way, and mastering them will greatly enhance your Flutter development skills.
