Control Flow Statements in Flutter
Functions and Methods ►

Control flow statements are fundamental in any programming language, including Dart, which is used in Flutter. These statements allow you to control the execution flow of your code, making it possible to perform different actions based on conditions, repeat actions, and handle exceptions.

Conditional Statements

Conditional statements allow you to execute certain pieces of code based on whether a condition is true or false.

if and else

The if statement is used to execute a block of code if a condition is true. The else statement can be used to execute a block of code if the condition is false.

void main() {
  int number = 10;

  if (number > 0) {
    print('The number is positive.');
  } else {
    print('The number is not positive.');
  }
}

else if

The else if statement allows you to check multiple conditions.

void main() {
  int number = 0;

  if (number > 0) {
    print('The number is positive.');
  } else if (number < 0) {
    print('The number is negative.');
  } else {
    print('The number is zero.');
  }
}

Switch Statement

The switch statement allows you to execute one block of code among many based on the value of a variable.

void main() {
  String grade = 'B';

  switch (grade) {
    case 'A':
      print('Excellent!');
      break;
    case 'B':
      print('Good!');
      break;
    case 'C':
      print('Fair!');
      break;
    case 'D':
      print('Poor!');
      break;
    default:
      print('Invalid grade.');
  }
}

Looping Statements

Looping statements allow you to execute a block of code multiple times.

for Loop

The for loop is used to run a block of code a specific number of times.

void main() {
  for (int i = 0; i < 5; i++) {
    print('Iteration $i');
  }
}

while Loop

The while loop is used to run a block of code as long as a condition is true.

void main() {
  int i = 0;

  while (i < 5) {
    print('Iteration $i');
    i++;
  }
}

do-while Loop

The do-while loop is similar to the while loop, but it guarantees that the block of code will be executed at least once.

void main() {
  int i = 0;

  do {
    print('Iteration $i');
    i++;
  } while (i < 5);
}

Exception Handling

Exception handling allows you to manage errors and other exceptional events in your code.

try and catch

The try block lets you test a block of code for errors, while the catch block lets you handle the error.

void main() {
  try {
    int result = 12 ~/ 0;
    print(result);
  } catch (e) {
    print('An error occurred: $e');
  }
}

finally

The finally block lets you execute code, regardless of whether an exception was thrown or not.

void main() {
  try {
    int result = 12 ~/ 0;
    print(result);
  } catch (e) {
    print('An error occurred: $e');
  } finally {
    print('This code runs no matter what.');
  }
}

Conclusion

Control flow statements are essential for managing the execution of your code. By understanding and using these statements effectively, you can create more dynamic and responsive Flutter applications. Practice using these statements in different scenarios to become proficient in controlling the flow of your Dart code.
