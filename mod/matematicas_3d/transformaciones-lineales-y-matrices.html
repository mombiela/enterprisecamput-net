Introduction

Linear transformations and matrices are fundamental concepts in the study of 3D mathematics. These tools allow for the manipulation and transformation of vectors and points in three-dimensional space, which is essential in fields such as computer graphics, physics, and engineering.

Key Concepts

  • Linear Transformation: A function that takes a vector and transforms it into another vector, maintaining the operations of addition and scalar multiplication.
  • Matrix: A compact representation of a linear transformation. Matrices allow for complex calculations to be performed efficiently.
  • Vector: An entity that has magnitude and direction in three-dimensional space.

Linear Transformations

A linear transformation in 3D can be represented by a 3x3 matrix. If \( T \) is a linear transformation and \( \mathbf{v} \) is a vector in 3D, then:

\[ T(\mathbf{v}) = A \mathbf{v} \]

where \( A \) is a 3x3 matrix and \( \mathbf{v} \) is a 3x1 column vector.

Example of a Linear Transformation

Suppose we have a transformation matrix \( A \) and a vector \( \mathbf{v} \):

\[ A = \begin{pmatrix} 2 & 0 & 0 \ 0 & 2 & 0 \ 0 & 0 & 2 \end{pmatrix}, \quad \mathbf{v} = \begin{pmatrix} 1 \ 1 \ 1 \end{pmatrix} \]

The transformation \( T(\mathbf{v}) \) is calculated as:

\[ T(\mathbf{v}) = A \mathbf{v} = \begin{pmatrix} 2 & 0 & 0 \ 0 & 2 & 0 \ 0 & 0 & 2 \end{pmatrix} \begin{pmatrix} 1 \ 1 \ 1 \end{pmatrix} = \begin{pmatrix} 2 \ 2 \ 2 \end{pmatrix} \]

Types of Transformations

  • Rotation: Rotates a vector around an axis.
  • Scaling: Changes the magnitude of a vector.
  • Translation: Moves a vector to a new position (note: translation is not a linear transformation, but it can be represented using 4x4 matrices in homogeneous space).

Example of Rotation

To rotate a vector around the \( z \) axis by an angle \( \theta \):

\[ R_z(\theta) = \begin{pmatrix} \cos \theta & -\sin \theta & 0 \ \sin \theta & \cos \theta & 0 \ 0 & 0 & 1 \end{pmatrix} \]

If \( \theta = 90^\circ \) and \( \mathbf{v} = \begin{pmatrix} 1 \ 0 \ 0 \end{pmatrix} \):

\[ R_z(90^\circ) = \begin{pmatrix} 0 & -1 & 0 \ 1 & 0 & 0 \ 0 & 0 & 1 \end{pmatrix}, \quad \mathbf{v} = \begin{pmatrix} 1 \ 0 \ 0 \end{pmatrix} \]

\[ T(\mathbf{v}) = R_z(90^\circ) \mathbf{v} = \begin{pmatrix} 0 & -1 & 0 \ 1 & 0 & 0 \ 0 & 0 & 1 \end{pmatrix} \begin{pmatrix} 1 \ 0 \ 0 \end{pmatrix} = \begin{pmatrix} 0 \ 1 \ 0 \end{pmatrix} \]

Practical Exercises

  1. Scaling Vectors:

    • Given the scaling matrix \( S = \begin{pmatrix} 3 & 0 & 0 \ 0 & 3 & 0 \ 0 & 0 & 3 \end{pmatrix} \) and the vector \( \mathbf{v} = \begin{pmatrix} 2 \ 2 \ 2 \end{pmatrix} \), calculate \( T(\mathbf{v}) \).

  2. Rotating Vectors:

    • Rotate the vector \( \mathbf{v} = \begin{pmatrix} 1 \ 1 \ 0 \end{pmatrix} \) around the \( y \) axis by an angle of \( 45^\circ \).

  3. Composition of Transformations:

    • Apply a \( 90^\circ \) rotation around the \( x \) axis followed by a scaling by a factor of 2 to the vector \( \mathbf{v} = \begin{pmatrix} 1 \ 0 \ 0 \end{pmatrix} \).

Conclusion

Linear transformations and matrices are powerful tools for manipulating vectors in three-dimensional space. Understanding how to apply and combine these transformations is essential for solving complex problems in various disciplines. Practicing with examples and exercises will help consolidate these concepts and develop practical skills in handling 3D transformations.
