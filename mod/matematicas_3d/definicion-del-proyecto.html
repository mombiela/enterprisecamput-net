Introduction

In this course on 3D Mathematics, we will focus on the application of three-dimensional mathematical concepts in various fields such as graphic programming, physical simulation, and engineering. The main objective is to provide a solid understanding of 3D mathematics and how it can be applied in real projects.

Project Objectives

  • Understand the basic concepts of 3D mathematics: vectors, matrices, transformations, and their applications.
  • Apply these concepts in graphic programming: using languages like Python and libraries like OpenGL.
  • Develop skills to solve three-dimensional problems: through practical exercises and projects.

Project Contents

Basic Concepts

  • 3D Vectors: Definition, operations (addition, subtraction, dot product, and cross product).
  • 3D Matrices: Types of matrices, operations, and transformations.
  • Transformations: Translation, rotation, and scaling.

Practical Applications

  • Graphic Programming: Using libraries like OpenGL to create 3D graphics.
  • Physical Simulation: Application of 3D mathematics in the simulation of movements and forces.
  • Engineering: Use of 3D mathematics in the design and analysis of structures.

Code Examples

Example 1: Vector Operations in Python

import numpy as np

# Definition of vectors
vector_a = np.array([1, 2, 3])
vector_b = np.array([4, 5, 6])

# Vector addition
sum = vector_a + vector_b
print("Sum:", sum)

# Dot product
dot_product = np.dot(vector_a, vector_b)
print("Dot Product:", dot_product)

# Cross product
cross_product = np.cross(vector_a, vector_b)
print("Cross Product:", cross_product)

Example 2: Transformations with Matrices in Python

import numpy as np

# Rotation matrix around the Z axis
theta = np.radians(45)
rotation_matrix = np.array([
    [np.cos(theta), -np.sin(theta), 0],
    [np.sin(theta), np.cos(theta), 0],
    [0, 0, 1]
])

# Vector to transform
vector = np.array([1, 0, 0])

# Applying the transformation
transformed_vector = np.dot(rotation_matrix, vector)
print("Transformed Vector:", transformed_vector)
