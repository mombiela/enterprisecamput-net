Introduction

In this section, we will explore the fundamental concepts of vectors and vector spaces, essential for understanding 3D mathematics. These concepts are the foundation for many applications in physics, engineering, computer science, and other disciplines.

Vectors

A vector is an entity that has magnitude and direction. In the context of 3D mathematics, vectors are represented in a three-dimensional space.

Representation of Vectors

  • Notation: A vector in 3D can be represented as \(\mathbf{v} = \begin{pmatrix} v_x \ v_y \ v_z \end{pmatrix}\), where \(v_x\), \(v_y\), and \(v_z\) are the components of the vector in the \(x\), \(y\), and \(z\) directions respectively.
  • Example: \(\mathbf{v} = \begin{pmatrix} 3 \ -2 \ 5 \end{pmatrix}\)

Vector Operations

  • Vector Addition: The sum of two vectors \(\mathbf{u} = \begin{pmatrix} u_x \ u_y \ u_z \end{pmatrix}\) and \(\mathbf{v} = \begin{pmatrix} v_x \ v_y \ v_z \end{pmatrix}\) is defined as: \[ \mathbf{u} + \mathbf{v} = \begin{pmatrix} u_x + v_x \ u_y + v_y \ u_z + v_z \end{pmatrix} \]
  • Dot Product: The dot product of two vectors \(\mathbf{u}\) and \(\mathbf{v}\) is defined as: \[ \mathbf{u} \cdot \mathbf{v} = u_x v_x + u_y v_y + u_z v_z \]
  • Cross Product: The cross product of two vectors \(\mathbf{u}\) and \(\mathbf{v}\) is defined as: \[ \mathbf{u} \times \mathbf{v} = \begin{vmatrix} \mathbf{i} & \mathbf{j} & \mathbf{k} \ u_x & u_y & u_z \ v_x & v_y & v_z \end{vmatrix} \]

Code Example

import numpy as np

# Define vectors
u = np.array([1, 2, 3])
v = np.array([4, 5, 6])

# Vector addition
sum = u + v
print("Vector addition:", sum)

# Dot product
dot_product = np.dot(u, v)
print("Dot product:", dot_product)

# Cross product
cross_product = np.cross(u, v)
print("Cross product:", cross_product)

Vector Spaces

A vector space is a set of vectors that can be added together and multiplied by scalars, satisfying certain properties.

Definition and Properties

  • Definition: A vector space \(V\) over a field \(F\) is a set of elements (vectors) along with two operations (addition and scalar multiplication) that satisfy the following properties:
    • Closure under addition: If \(\mathbf{u}, \mathbf{v} \in V\), then \(\mathbf{u} + \mathbf{v} \in V\).
    • Closure under scalar multiplication: If \(\mathbf{u} \in V\) and \(c \in F\), then \(c\mathbf{u} \in V\).
    • Associativity of addition: \((\mathbf{u} + \mathbf{v}) + \mathbf{w} = \mathbf{u} + (\mathbf{v} + \mathbf{w})\) for all \(\mathbf{u}, \mathbf{v}, \mathbf{w} \in V\).
    • Commutativity of addition: \(\mathbf{u} + \mathbf{v} = \mathbf{v} + \mathbf{u}\) for all \(\mathbf{u}, \mathbf{v} \in V\).
    • Additive identity: There exists a vector \(\mathbf{0} \in V\) such that \(\mathbf{u} + \mathbf{0} = \mathbf{u}\) for all \(\mathbf{u} \in V\).
    • Additive inverse: For each \(\mathbf{u} \in V\), there exists a vector \(-\mathbf{u} \in V\) such that \(\mathbf{u} + (-\mathbf{u}) = \mathbf{0}\).
    • Distributivity of scalar multiplication: \(c(\mathbf{u} + \mathbf{v}) = c\mathbf{u} + c\mathbf{v}\) and \((c + d)\mathbf{u} = c\mathbf{u} + d\mathbf{u}\) for all \(c, d \in F\) and \(\mathbf{u}, \mathbf{v} \in V\).
    • Compatibility of scalar multiplication: \((cd)\mathbf{u} = c(d\mathbf{u})\) for all \(c, d \in F\) and \(\mathbf{u} \in V\).
    • Multiplicative identity: \(1\mathbf{u} = \mathbf{u}\) for all \(\mathbf{u} \in V\).

Example of a Vector Space

  • Vector Space \(\mathbb{R}^3\): The set of all vectors of the form \(\begin{pmatrix} x \ y \ z \end{pmatrix}\) where \(x, y, z \in \mathbb{R}\) is a vector space over the field of real numbers \(\mathbb{R}\).

Code Example

# Verify properties of a vector space in Python

# Define vectors
u = np.array([1, 2, 3])
v = np.array([4, 5, 6])
w = np.array([7, 8, 9])
c = 2
d = 3

# Closure under addition
closure_addition = np.all((u + v) in np.array([u, v, w]))
print("Closure under addition:", closure_addition)

# Closure under scalar multiplication
closure_scalar = np.all((c * u) in np.array([u, v, w]))
print("Closure under scalar multiplication:", closure_scalar)

# Associativity of addition
associativity_addition = np.all((u + (v + w)) == ((u + v) + w))
print("Associativity of addition:", associativity_addition)

# Commutativity of addition
commutativity_addition = np.all((u + v) == (v + u))
print("Commutativity of addition:", commutativity_addition)
