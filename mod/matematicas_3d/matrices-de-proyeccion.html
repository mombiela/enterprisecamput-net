Introduction

In the realm of 3D mathematics, projection matrices are fundamental tools for transforming coordinates in a three-dimensional space to a two-dimensional space. This process is essential in computer graphics, where it is necessary to project 3D scenes onto a 2D screen. In this section, we will explore the basic concepts of projection matrices, their types, and how they are implemented.

Key Concepts

  • Projection: The process of transforming points in a three-dimensional space to a two-dimensional space.
  • Projection Matrix: A matrix used to perform the projection transformation.
  • Orthographic Projection: A projection that maintains the relative dimensions of objects.
  • Perspective Projection: A projection that simulates the way the human eye perceives the world, where distant objects appear smaller.

Types of Projection

Orthographic Projection

Orthographic projection is a technique where objects are projected onto a surface without considering depth. This means that the dimensions of the objects do not change with distance.

Orthographic Projection Matrix

The orthographic projection matrix is defined as follows:

\[ P_{ortho} = \begin{bmatrix} \frac{2}{r-l} & 0 & 0 & -\frac{r+l}{r-l}
0 & \frac{2}{t-b} & 0 & -\frac{t+b}{t-b}
0 & 0 & \frac{-2}{f-n} & -\frac{f+n}{f-n}
0 & 0 & 0 & 1 \end{bmatrix} \]

Where:

  • \( r \) and \( l \) are the right and left limits of the view volume.
  • \( t \) and \( b \) are the top and bottom limits of the view volume.
  • \( f \) and \( n \) are the far and near clipping planes.

Perspective Projection

Perspective projection simulates how the human eye perceives the world, making distant objects appear smaller.

Perspective Projection Matrix

The perspective projection matrix is defined as follows:

\[ P_{persp} = \begin{bmatrix} \frac{2n}{r-l} & 0 & \frac{r+l}{r-l} & 0
0 & \frac{2n}{t-b} & \frac{t+b}{t-b} & 0
0 & 0 & -\frac{f+n}{f-n} & -\frac{2fn}{f-n}
0 & 0 & -1 & 0 \end{bmatrix} \]

Where:

  • \( n \) is the distance to the near plane.
  • \( f \) is the distance to the far plane.
  • \( r \), \( l \), \( t \), and \( b \) are the limits of the view volume.

Code Examples

Orthographic Projection in Python

import numpy as np

def ortho_projection_matrix(l, r, b, t, n, f):
    return np.array([
        [2/(r-l), 0, 0, -(r+l)/(r-l)],
        [0, 2/(t-b), 0, -(t+b)/(t-b)],
        [0, 0, -2/(f-n), -(f+n)/(f-n)],
        [0, 0, 0, 1]
    ])

# Example usage
l, r, b, t, n, f = -1, 1, -1, 1, 1, 10
P_ortho = ortho_projection_matrix(l, r, b, t, n, f)
print(P_ortho)

Perspective Projection in Python

import numpy as np

def perspective_projection_matrix(l, r, b, t, n, f):
    return np.array([
        [2*n/(r-l), 0, (r+l)/(r-l), 0],
        [0, 2*n/(t-b), (t+b)/(t-b), 0],
        [0, 0, -(f+n)/(f-n), -2*f*n/(f-n)],
        [0, 0, -1, 0]
    ])

# Example usage
l, r, b, t, n, f = -1, 1, -1, 1, 1, 10
P_persp = perspective_projection_matrix(l, r, b, t, n, f)
print(P_persp)

Understanding the differences between orthographic projection and perspective projection, as well as their respective matrices, is crucial for working with 3D graphics.
