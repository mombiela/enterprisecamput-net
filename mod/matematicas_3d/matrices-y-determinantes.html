Introduction

In this section of the 3D Mathematics course, we will explore the concepts of matrices and determinants. These are fundamental in various applications, from solving systems of linear equations to transforming coordinates in 3D graphics.

Matrices

A matrix is a rectangular arrangement of numbers in rows and columns. Matrices are used to represent and manipulate data in many areas of mathematics and engineering.

Key Concepts

  • Matrix Definition: A matrix is an array of numbers arranged in rows and columns. It is generally denoted by an uppercase letter, such as \( A \).
  • Matrix Dimensions: A matrix with \( m \) rows and \( n \) columns is called an \( m \times n \) matrix.
  • Matrix Elements: The numbers in a matrix are called elements and are denoted by \( a_{ij} \), where \( i \) is the row number and \( j \) is the column number.

Matrix Operations

  • Matrix Addition: Two matrices can be added if they have the same dimensions. The addition is performed element by element.
  • Matrix Multiplication: The product of two matrices \( A \) and \( B \) is possible if the number of columns of \( A \) is equal to the number of rows of \( B \). The element \( c_{ij} \) of the resulting matrix \( C \) is calculated as:

\[ c_{ij} = \sum_{k=1}^{n} a_{ik} b_{kj} \]

Code Example

import numpy as np

# Matrix definition
A = np.array([[1, 2], [3, 4]])
B = np.array([[5, 6], [7, 8]])

# Matrix addition
C = A + B
print("Sum of A and B:\n", C)

# Matrix multiplication
D = np.dot(A, B)
print("Product of A and B:\n", D)

Determinants

The determinant is a scalar value associated with a square matrix. It is useful in solving systems of linear equations, calculating matrix inverses, and characterizing certain geometric properties.

Key Concepts

  • Determinant Definition: The determinant of a square matrix \( A \) is denoted as \( \det(A) \) or \( |A| \).
  • Determinant Properties:
    • If a row or column of a matrix is zero, its determinant is zero.
    • If two rows or columns are identical, its determinant is zero.
    • The determinant of a triangular matrix (upper or lower) is the product of the elements of the main diagonal.

Determinant Calculation

For a \( 2 \times 2 \) matrix:

\[ \det \begin{pmatrix} a & b \ c & d \end{pmatrix} = ad - bc \]

For a \( 3 \times 3 \) matrix:

\[ \det \begin{pmatrix} a & b & c \ d & e & f \ g & h & i \end{pmatrix} = a(ei - fh) - b(di - fg) + c(dh - eg) \]

Code Example

import numpy as np

# Matrix definition
A = np.array([[1, 2], [3, 4]])

# Determinant calculation
det_A = np.linalg.det(A)
