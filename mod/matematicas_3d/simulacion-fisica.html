Introduction to Physical Simulation

Physical simulation is a technique used to model and analyze the behavior of physical systems in a virtual environment. In the context of 3D mathematics, physical simulation allows representing and predicting how objects interact in a three-dimensional space under various forces and conditions.

Key Concepts

  • Classical Mechanics: Study of the motion of bodies under the action of forces.
  • Dynamics: Analysis of forces and their effect on motion.
  • Kinematics: Description of motion without considering the forces that cause it.
  • Forces: Interactions that change the state of motion of an object.
  • Collisions: Interactions between objects that result in a change of speed and direction.
  • Gravity: Force of attraction between two masses.
  • Friction: Force that resists the relative motion of two surfaces in contact.

Modeling Forces

Gravitational Force

Gravitational force is one of the most common forces in physical simulation. It can be modeled using the formula:

\[ F = m \cdot g \]

Where:

  • \( F \) is the gravitational force.
  • \( m \) is the mass of the object.
  • \( g \) is the acceleration due to gravity (approximately \( 9.81 , m/s^2 \) on Earth).

Friction Force

Friction can be modeled as follows:

\[ F_f = \mu \cdot N \]

Where:

  • \( F_f \) is the friction force.
  • \( \mu \) is the coefficient of friction.
  • \( N \) is the normal force (perpendicular to the contact surface).

Code Implementation

Below is an example of how to implement a basic simulation of an object falling under the influence of gravity using Python:

import numpy as np
import matplotlib.pyplot as plt

# Initial parameters
mass = 1.0  # kg
gravity = 9.81  # m/s^2
total_time = 10  # seconds
delta_t = 0.01  # time interval

# Variable initialization
position = 0.0  # initial position
velocity = 0.0  # initial velocity
time = np.arange(0, total_time, delta_t)
positions = []

# Simulation
for t in time:
    force = mass * gravity
    acceleration = force / mass
    velocity += acceleration * delta_t
    position += velocity * delta_t
    positions.append(position)

# Visualization
plt.plot(time, positions)
plt.xlabel('Time (s)')
plt.ylabel('Position (m)')
plt.title('Free fall under the influence of gravity')
plt.show()

Code Explanation

  1. Initial Parameters: We define the mass of the object, the acceleration due to gravity, the total simulation time, and the time interval.
  2. Variable Initialization: We set the initial position and velocity of the object.
  3. Simulation: We use a loop to calculate the position of the object at each time interval, updating the velocity and position at each iteration.
  4. Visualization: We use matplotlib to plot the position of the object as a function of time.

Collisions

Collisions are important events in physical simulation. They can be elastic or inelastic:

  • Elastic Collision: Conserves both kinetic energy and momentum.
  • Inelastic Collision: Conserves momentum but not kinetic energy.

Example of Elastic Collision

def elastic_collision(m1, v1, m2, v2):
    # Velocities after the collision
    v1_final = (v1 * (m1 - m2) + 2 * m2 * v2) / (m1 + m2)
    v2_final = (v2 * (m2 - m1) + 2 * m1 * v1) / (m1 + m2)
    return v1_final, v2_final

# Example parameters
m1, v1 = 1.0, 2.0  # mass and velocity of the first object
m2, v2 = 1.0, -1.0  # mass and velocity of the second object

v1_final, v2_final = elastic_collision(m1, v1, m2, v2)
print(f"Final velocity of the first object: {v1_final} m/s")
print(f"Final velocity of the second object: {v2_final} m/s")
