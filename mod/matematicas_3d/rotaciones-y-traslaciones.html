Introduction

In the study of 3D mathematics, rotations and translations are fundamental transformations that allow manipulating objects in three-dimensional space. These transformations are essential in fields such as computer graphics, robotics, and physical simulation.

Translations

Translation is a transformation that shifts each point of an object a constant distance in a specific direction.

  • Definition: A translation in 3D is defined by a translation vector \(\mathbf{t} = (t_x, t_y, t_z)\).
  • Formula: If a point \(\mathbf{P} = (x, y, z)\) is translated by \(\mathbf{t}\), the new point \(\mathbf{P'}\) will be: \[ \mathbf{P'} = \mathbf{P} + \mathbf{t} = (x + t_x, y + t_y, z + t_z) \]

Translation Example

Suppose we have a point \(\mathbf{P} = (1, 2, 3)\) and we want to translate it by the vector \(\mathbf{t} = (4, -2, 1)\).

\[ \mathbf{P'} = (1 + 4, 2 - 2, 3 + 1) = (5, 0, 4) \]

In Python code, this can be done as follows:

import numpy as np

# Original point
P = np.array([1, 2, 3])

# Translation vector
t = np.array([4, -2, 1])

# New point after translation
P_prime = P + t

print("Original point:", P)
print("Translation vector:", t)
print("New point:", P_prime)

Rotations

Rotation is a transformation that rotates an object around a fixed axis.

  • Definition: A rotation in 3D is defined by a rotation angle \(\theta\) and a rotation axis \(\mathbf{u} = (u_x, u_y, u_z)\).
  • Rotation Matrix: The rotation matrix \(R\) for an angle \(\theta\) around the axis \(\mathbf{u}\) can be expressed using Rodrigues' formula: \[ R = I + \sin(\theta)K + (1 - \cos(\theta))K^2 \] where \(I\) is the identity matrix and \(K\) is the cross-product matrix of the vector \(\mathbf{u}\).

Rotation Example

Suppose we want to rotate a point \(\mathbf{P} = (1, 0, 0)\) 90 degrees (\(\pi/2\) radians) around the z-axis.

The rotation matrix \(R_z\) for an angle \(\theta\) around the z-axis is:

\[ R_z = \begin{bmatrix} \cos(\theta) & -\sin(\theta) & 0
\sin(\theta) & \cos(\theta) & 0
0 & 0 & 1 \end{bmatrix} \]

For \(\theta = \pi/2\):

\[ R_z = \begin{bmatrix} 0 & -1 & 0
1 & 0 & 0
0 & 0 & 1 \end{bmatrix} \]

Applying this matrix to the point \(\mathbf{P}\):

\[ \mathbf{P'} = R_z \mathbf{P} = \begin{bmatrix} 0 & -1 & 0
1 & 0 & 0
0 & 0 & 1 \end{bmatrix} \begin{bmatrix} 1
0
0 \end{bmatrix} = \begin{bmatrix} 0
1
0 \end{bmatrix} \]

In Python code, this can be done as follows:

import numpy as np

# Original point
P = np.array([1, 0, 0])

# Rotation angle (90 degrees in radians)
theta = np.pi / 2

# Rotation matrix around the z-axis
R_z = np.array([
    [np.cos(theta), -np.sin(theta), 0],
    [np.sin(theta), np.cos(theta), 0],
    [0, 0, 1]
])

# New point after rotation
P_prime = R_z.dot(P)

print("Original point:", P)
print("Rotation matrix R_z:\n", R_z)
print("New point:", P_prime)

Conclusion

Rotations and translations are essential tools for manipulating objects in three-dimensional space. Understanding how to apply these transformations is crucial for any professional working in fields related to 3D mathematics. Through examples and practical exercises, one can acquire a solid understanding of these fundamental concepts.
