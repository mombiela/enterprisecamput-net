Introduction to DirectX

DirectX is a collection of APIs (Application Programming Interfaces) developed by Microsoft to handle tasks related to multimedia, especially game and video programming, on Microsoft platforms. In the context of 3D mathematics, DirectX provides essential tools for creating and manipulating three-dimensional graphics.

Key Concepts of DirectX

  • Direct3D: It is the DirectX API used to render 3D graphics. It is fundamental for creating graphical applications and games.
  • Shaders: Programs that run on the GPU to perform advanced graphics operations.
  • Buffers: Memory areas that store data the GPU needs to render graphics.
  • Transformation Matrices: Used to perform transformation operations on 3D objects, such as translation, rotation, and scaling.

Installation and Configuration of DirectX

To start using DirectX, you need to install the DirectX SDK and configure the development environment. Here is an example of how to do it in Visual Studio:

  1. Install the DirectX SDK: Download and install the DirectX SDK from the Microsoft website.
  2. Configure Visual Studio:
    • Open Visual Studio and create a new project.
    • Go to Project > Properties.
    • In the VC++ Directories section, add the paths to the DirectX SDK include and library directories.
    • In the Linker > Input section, add the necessary DirectX libraries (e.g., d3d11.lib).

Creating a Basic Window with DirectX

Below is a code example to create a basic window using DirectX:

#include <windows.h>
#include <d3d11.h>

// Global variables
HINSTANCE hInstance;
HWND hwnd;
ID3D11Device* device = nullptr;
ID3D11DeviceContext* deviceContext = nullptr;
IDXGISwapChain* swapChain = nullptr;
ID3D11RenderTargetView* renderTargetView = nullptr;

// Function prototypes
LRESULT CALLBACK WndProc(HWND, UINT, WPARAM, LPARAM);
void InitD3D(HWND hwnd);
void RenderFrame();
void CleanD3D();

int WINAPI WinMain(HINSTANCE hInstance, HINSTANCE hPrevInstance, LPSTR lpCmdLine, int nCmdShow) {
    // Register the window class
    WNDCLASSEX wc;
    ZeroMemory(&wc, sizeof(WNDCLASSEX));
    wc.cbSize = sizeof(WNDCLASSEX);
    wc.style = CS_HREDRAW | CS_VREDRAW;
    wc.lpfnWndProc = WndProc;
    wc.hInstance = hInstance;
    wc.hCursor = LoadCursor(NULL, IDC_ARROW);
    wc.lpszClassName = "WindowClass";
    RegisterClassEx(&wc);

    // Create the window
    hwnd = CreateWindowEx(NULL, "WindowClass", "DirectX Window", WS_OVERLAPPEDWINDOW, 100, 100, 800, 600, NULL, NULL, hInstance, NULL);
    ShowWindow(hwnd, nCmdShow);

    // Initialize Direct3D
    InitD3D(hwnd);

    // Message loop
    MSG msg;
    while (TRUE) {
        if (PeekMessage(&msg, NULL, 0, 0, PM_REMOVE)) {
            if (msg.message == WM_QUIT) break;
            TranslateMessage(&msg);
            DispatchMessage(&msg);
        } else {
            RenderFrame();
        }
    }

    // Clean up Direct3D
    CleanD3D();

    return msg.wParam;
}

LRESULT CALLBACK WndProc(HWND hwnd, UINT msg, WPARAM wParam, LPARAM lParam) {
    switch (msg) {
        case WM_DESTROY:
            PostQuitMessage(0);
            return 0;
    }
    return DefWindowProc(hwnd, msg, wParam, lParam);
}

void InitD3D(HWND hwnd) {
    // Create the device and device context
    D3D_FEATURE_LEVEL featureLevel;
    D3D11CreateDevice(NULL, D3D_DRIVER_TYPE_HARDWARE, NULL, 0, NULL, 0, D3D11_SDK_VERSION, &device, &featureLevel, &deviceContext);

    // Create the swap chain
    DXGI_SWAP_CHAIN_DESC scd;
    ZeroMemory(&scd, sizeof(DXGI_SWAP_CHAIN_DESC));
    scd.BufferCount = 1;
    scd.BufferDesc.Format = DXGI_FORMAT_R8G8B8A8_UNORM;
    scd.BufferUsage = DXGI_USAGE_RENDER_TARGET_OUTPUT;
    scd.OutputWindow = hwnd;
    scd.SampleDesc.Count = 4;
    scd.Windowed = TRUE;
    IDXGIFactory* factory;
    CreateDXGIFactory(__uuidof(IDXGIFactory), (void**)&factory);
    factory->CreateSwapChain(device, &scd, &swapChain);

    // Create the render target view
    ID3D11Texture2D* backBuffer;
    swapChain->GetBuffer(0, __uuidof(ID3D11Texture2D), (void**)&backBuffer);
    device->CreateRenderTargetView(backBuffer, NULL, &renderTargetView);
    backBuffer->Release();
    deviceContext->OMSetRenderTargets(1, &renderTargetView, NULL);
}

void RenderFrame() {
    // Clear the buffer
    float color[4] = {0.0f, 0.2f, 0.4f, 1.0f};
    deviceContext->ClearRenderTargetView(renderTargetView, color);

    // Present the buffer
    swapChain->Present(0, 0);
}

void CleanD3D() {
    swapChain->Release();
    renderTargetView->Release();
    deviceContext->Release();
    device->Release();
}

3D Transformations with DirectX

3D transformations are essential for manipulating objects in three-dimensional space. DirectX uses matrices to perform these transformations:

  • Translation: Moves an object from one place to another.
  • Rotation: Rotates an object around an axis.
  • Scaling: Changes the size of an object.

Code example to apply a translation transformation:

#include <DirectXMath.h>

using namespace DirectX;

XMMATRIX translationMatrix = XMMatrixTranslation(3.0f, 4.0f, 5.0f);
