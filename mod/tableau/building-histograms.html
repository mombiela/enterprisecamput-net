Introduction to Histograms

Histograms are a type of bar chart that represents the distribution of numerical data. They are useful for understanding the frequency of data points within certain ranges (bins). In Tableau, creating histograms is straightforward and can provide valuable insights into your data.

Creating a Basic Histogram

To create a histogram in Tableau, follow these steps:

  1. Connect to Data: Load your dataset into Tableau.
  2. Select the Measure: Choose the numerical field you want to analyze.
  3. Create Bins: Tableau will automatically create bins for your data.
  4. Build the Histogram: Drag the bins to the Columns shelf and the measure to the Rows shelf.

Step-by-Step Example

Let's create a histogram using a sample dataset.

  1. Connect to Data:

    • Open Tableau and connect to the "Sample - Superstore" dataset.

  2. Select the Measure:

    • Choose the Sales measure from the data pane.

  3. Create Bins:

    • Right-click on Sales and select "Create" > "Bins...".
    • In the "Create Bins" dialog box, Tableau will suggest a bin size. You can adjust this size as needed. For this example, let's use the default bin size.

  4. Build the Histogram:

    • Drag the Sales (bin) field to the Columns shelf.
    • Drag the Number of Records field to the Rows shelf.

Your worksheet should now display a histogram showing the distribution of sales.

Customizing Histograms

Customizing your histogram can make it more informative and visually appealing.

Adjusting Bin Size

  • Right-click on the Sales (bin) field in the Columns shelf.
  • Select "Edit" to change the bin size. Smaller bin sizes provide more detail, while larger bin sizes give a broader overview.

Adding Colors

  • Drag a dimension (e.g., Region) to the Color shelf to differentiate the data by categories.

Adding Labels

  • Drag the Number of Records field to the Label shelf to display the count of records on each bin.

Example of Customization

Here’s how you can customize the histogram created earlier:

  1. Adjust Bin Size:

    • Right-click on Sales (bin) and select "Edit".
    • Change the bin size to 500.

  2. Add Colors:

    • Drag the Region dimension to the Color shelf.

  3. Add Labels:

    • Drag the Number of Records field to the Label shelf.

Your histogram should now be more detailed and visually segmented by region.

Advanced Histogram Techniques

For advanced users, Tableau offers additional features to enhance histograms.

Dual-Axis Histograms

You can create dual-axis histograms to compare two distributions on the same chart.

  1. Create Two Histograms:

    • Follow the steps to create a histogram for Sales.
    • Repeat the steps to create another histogram for Profit.

  2. Combine Histograms:

    • Drag the Profit (bin) field to the Columns shelf next to Sales (bin).
    • Right-click on the second axis and select "Dual Axis".

  3. Synchronize Axes:

    • Right-click on one of the axes and select "Synchronize Axis".

Example of Dual-Axis Histogram

Here’s how you can create a dual-axis histogram comparing Sales and Profit:

  1. Create Two Histograms:

    • Create a histogram for Sales as described earlier.
    • Create another histogram for Profit following the same steps.

  2. Combine Histograms:

    • Drag the Profit (bin) field to the Columns shelf next to Sales (bin).
    • Right-click on the second axis and select "Dual Axis".

  3. Synchronize Axes:

    • Right-click on one of the axes and select "Synchronize Axis".

Your worksheet should now display a dual-axis histogram comparing the distribution of sales and profit.

Conclusion

Histograms are a powerful tool in Tableau for visualizing the distribution of numerical data. Starting from basic histograms to advanced techniques like dual-axis histograms, you can customize and enhance your visualizations to gain deeper insights. Practice creating and customizing histograms with different datasets to become proficient in this essential Tableau skill.
