Introduction

Creating calculated fields in Tableau allows you to perform computations on your data, enabling deeper insights and more dynamic visualizations. This topic will guide you through the basics of calculated fields, from simple arithmetic operations to more advanced functions.

What is a Calculated Field?

A calculated field is a new field created by applying a formula to existing data in your Tableau data source. This can be used to:

  • Perform arithmetic operations.
  • Apply conditional logic.
  • Aggregate data.
  • Manipulate strings and dates.

Creating a Simple Calculated Field

Steps to Create a Calculated Field

  1. Open Tableau and connect to your data source.
  2. Navigate to the Data pane.
  3. Right-click on the data source and select "Create Calculated Field".
  4. Enter a name for your calculated field.
  5. Enter the formula for your calculation.
  6. Click OK to create the field.

Example: Basic Arithmetic Calculation

Suppose you have a dataset with Sales and Cost fields, and you want to calculate the Profit.

[Profit] = [Sales] - [Cost]
  1. Right-click on the data source.
  2. Select "Create Calculated Field".
  3. Name the field Profit.
  4. Enter the formula [Sales] - [Cost].
  5. Click OK.

Using Functions in Calculated Fields

Tableau provides a wide range of functions that can be used in calculated fields. These include:

  • String Functions: e.g., LEFT(), RIGHT(), MID()
  • Date Functions: e.g., DATEADD(), DATEDIFF(), DATENAME()
  • Logical Functions: e.g., IF, THEN, ELSE, CASE
  • Aggregate Functions: e.g., SUM(), AVG(), MIN(), MAX()

Example: Using String Functions

Suppose you have a Full Name field and you want to extract the first name.

[First Name] = LEFT([Full Name], FIND([Full Name], ' ') - 1)
  1. Right-click on the data source.
  2. Select "Create Calculated Field".
  3. Name the field First Name.
  4. Enter the formula LEFT([Full Name], FIND([Full Name], ' ') - 1).
  5. Click OK.

Example: Using Date Functions

Suppose you want to calculate the number of days between two dates, Order Date and Ship Date.

[Days to Ship] = DATEDIFF('day', [Order Date], [Ship Date])
  1. Right-click on the data source.
  2. Select "Create Calculated Field".
  3. Name the field Days to Ship.
  4. Enter the formula DATEDIFF('day', [Order Date], [Ship Date]).
  5. Click OK.

Advanced Calculated Fields

Using Level of Detail (LOD) Expressions

LOD expressions allow you to compute values at the data source level and the visualization level, providing more control over the granularity of your calculations.

Example: Fixed LOD Expression

Suppose you want to calculate the average sales per region, regardless of the level of detail in the view.

[Avg Sales per Region] = { FIXED [Region] : AVG([Sales]) }
  1. Right-click on the data source.
  2. Select "Create Calculated Field".
  3. Name the field Avg Sales per Region.
  4. Enter the formula { FIXED [Region] : AVG([Sales]) }.
  5. Click OK.

Using Table Calculations

Table calculations are transformations you apply to the values in a visualization. They are computed based on the data in the view.

Example: Running Total

Suppose you want to calculate a running total of sales.

  1. Drag Sales to the Rows shelf.
  2. Right-click on the Sales pill.
  3. Select "Quick Table Calculation" > "Running Total".

Conclusion

Creating calculated fields in Tableau is a powerful way to enhance your data analysis capabilities. By understanding and utilizing basic arithmetic operations, functions, LOD expressions, and table calculations, you can derive more meaningful insights from your data. Practice creating different types of calculated fields to become proficient in transforming and analyzing your data in Tableau.
