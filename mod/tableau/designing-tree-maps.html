Introduction to Tree Maps

Tree maps are a powerful visualization tool in Tableau that allows you to display hierarchical data using nested rectangles. Each rectangle represents a category, and its size and color can be used to convey additional information.

Key Concepts

  • Hierarchical Data: Data that is organized into a tree-like structure with parent-child relationships.
  • Rectangles: Each category or sub-category is represented by a rectangle.
  • Size and Color: The size of the rectangle can represent one measure (e.g., sales), while the color can represent another measure (e.g., profit).

Creating Your First Tree Map

To create a basic tree map in Tableau, follow these steps:

  1. Connect to Data: Open Tableau and connect to your data source.
  2. Drag and Drop: Drag a dimension to the Rows shelf and a measure to the Columns shelf.
  3. Change to Tree Map: Click on the "Show Me" panel and select the Tree Map option.

Example

Suppose you have a dataset of sales data with the following columns: Category, Sub-Category, Sales, and Profit.

Category    Sub-Category    Sales    Profit
Furniture   Chairs          5000     200
Furniture   Tables          3000     150
Office      Paper           4000     100
Office      Binders         2000     50

To create a tree map:

  1. Drag Category to the Rows shelf.
  2. Drag Sales to the Columns shelf.
  3. Click on "Show Me" and select the Tree Map option.

Customizing Tree Maps

Once you have created a basic tree map, you can customize it to better suit your needs.

Adjusting Size and Color

  • Size: Drag a measure (e.g., Sales) to the Size shelf to adjust the size of the rectangles.
  • Color: Drag a measure (e.g., Profit) to the Color shelf to adjust the color of the rectangles.

Adding Labels

  • Drag the dimension (e.g., Sub-Category) to the Label shelf to add labels to each rectangle.

Example

To enhance the previous example:

  1. Drag Sales to the Size shelf.
  2. Drag Profit to the Color shelf.
  3. Drag Sub-Category to the Label shelf.

Advanced Tree Map Techniques

For advanced users, Tableau offers several techniques to enhance tree maps further.

Using Calculated Fields

You can create calculated fields to add more complex logic to your tree maps.

Example

Create a calculated field to categorize profit margins:

IF [Profit] > 100 THEN "High"
ELSE "Low"
END

Drag this calculated field to the Color shelf to color-code the rectangles based on profit margins.

Using Hierarchies

Tableau allows you to create hierarchies to better organize your data.

Example

  1. Right-click on Category and select "Hierarchy" > "Create Hierarchy".
  2. Add Sub-Category to the hierarchy.
  3. Use this hierarchy in your tree map to display nested rectangles.

Conclusion

Tree maps are a versatile and powerful way to visualize hierarchical data in Tableau. By understanding the basics and exploring advanced techniques, you can create insightful and visually appealing tree maps that effectively communicate your data's story. Practice creating and customizing tree maps with your datasets to become proficient in this visualization technique.
