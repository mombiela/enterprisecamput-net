Introduction

Pie charts are a popular way to visualize data proportions and distributions. In this section, we will cover how to create and customize pie charts in Tableau, from basic to advanced techniques.

Getting Started with Pie Charts

Creating a Basic Pie Chart

  1. Connect to Data Source:

    • Open Tableau and connect to your data source (e.g., Excel, SQL database).

  2. Load Data:

    • Drag the relevant data fields into the workspace.

  3. Build the Pie Chart:

    • Drag a dimension (e.g., Category) to the Columns shelf.
    • Drag a measure (e.g., Sales) to the Rows shelf.
    • Click on the Show Me panel and select the Pie Chart icon.

  4. Adjust the Pie Chart:

    • Drag the dimension (e.g., Category) to the Color shelf.
    • Drag the measure (e.g., Sales) to the Angle shelf.
    • Optionally, drag the measure to the Label shelf to display values.
# Example Data
Category | Sales
---------|------
A        | 100
B        | 200
C        | 300

# Steps in Tableau
1. Drag 'Category' to Columns.
2. Drag 'Sales' to Rows.
3. Select Pie Chart from Show Me.
4. Drag 'Category' to Color.
5. Drag 'Sales' to Angle.
6. Drag 'Sales' to Label (optional).

Customizing Pie Charts

Adding Labels and Tooltips

  • Labels:

    • Drag the measure (e.g., Sales) to the Label shelf to display the values on the pie slices.
    • Customize the label format by clicking on the Label shelf and adjusting the settings.

  • Tooltips:

    • Customize tooltips by clicking on the Tooltip shelf and editing the text to include relevant information.

Formatting and Styling

  • Colors:

    • Click on the Color shelf to change the color palette or assign specific colors to categories.

  • Borders:

    • Add borders to pie slices by clicking on the Color shelf and adjusting the border settings.

  • Size:

    • Adjust the size of the pie chart by dragging the Size shelf slider.

Example of Customization

# Customizing Labels and Tooltips
1. Drag 'Sales' to Label.
2. Click on Label shelf and format the text.
3. Click on Tooltip shelf and edit the tooltip text.

# Formatting and Styling
1. Click on Color shelf to change colors.
2. Adjust border settings in Color shelf.
3. Use Size shelf slider to adjust pie size.

Advanced Techniques

Creating a Donut Chart

  1. Create a Dual-Axis Pie Chart:

    • Duplicate the pie chart by dragging the measure (e.g., Sales) to the Rows shelf again.
    • Right-click on the second measure and select Dual-Axis.

  2. Synchronize Axes:

    • Right-click on one of the axes and select Synchronize Axis.

  3. Convert to Donut:

    • Click on one of the pie charts and reduce its size to create a hole in the middle.
    • Adjust the size of the second pie chart to fit around the first.

Example of a Donut Chart

# Steps to Create a Donut Chart
1. Duplicate 'Sales' on Rows shelf.
2. Right-click second 'Sales' and select Dual-Axis.
3. Right-click axis and select Synchronize Axis.
4. Adjust size of first pie chart to create hole.
5. Adjust size of second pie chart to fit around the first.

Using Calculated Fields for Dynamic Slices

  • Create Calculated Fields:
    • Use calculated fields to create dynamic slices based on conditions or formulas.
    • Example: Create a calculated field to segment sales into different ranges.
# Example Calculated Field
IF [Sales] > 200 THEN "High"
ELSEIF [Sales] > 100 THEN "Medium"
ELSE "Low"
END
  • Apply Calculated Field:
    • Drag the calculated field to the Color or Label shelf to visualize the segments.

Conclusion

Pie charts are a versatile tool in Tableau for visualizing data proportions. Starting from basic pie charts, you can customize and enhance them with labels, tooltips, and advanced techniques like donut charts and calculated fields. By mastering these skills, you can create insightful and visually appealing pie charts to effectively communicate your data insights.
