Introduction

Scatter plots are a powerful visualization tool used to display the relationship between two continuous variables. In this section, we will cover the basics of creating scatter plots in Tableau, progressing from beginner to advanced techniques.

Getting Started with Scatter Plots

What is a Scatter Plot?

  • A scatter plot displays data points on a two-dimensional graph.
  • Each point represents the values of two variables.
  • Useful for identifying correlations, trends, and outliers.

Basic Steps to Create a Scatter Plot

  1. Connect to Data Source: Open Tableau and connect to your data source.
  2. Drag and Drop Fields: Drag one continuous variable to the Columns shelf and another to the Rows shelf.
  3. Select Scatter Plot: Tableau will automatically generate a scatter plot if the data is appropriate.

Example: Creating a Basic Scatter Plot

Let's create a scatter plot using a sample dataset.

  1. Connect to Sample Data: Use Tableau's built-in "Sample - Superstore" dataset.
  2. Drag Fields:
    • Drag Sales to the Columns shelf.
    • Drag Profit to the Rows shelf.
  3. View Scatter Plot: Tableau will generate a scatter plot showing the relationship between Sales and Profit.
Columns: Sales
Rows: Profit

Customizing Scatter Plots

Adding Details with Marks

  • Color: Drag a categorical variable (e.g., Category) to the Color shelf to differentiate data points by color.
  • Size: Drag a measure (e.g., Quantity) to the Size shelf to vary the size of the points.
  • Label: Drag a field (e.g., Region) to the Label shelf to add labels to the points.

Example: Customizing the Scatter Plot

  1. Color by Category: Drag Category to the Color shelf.
  2. Size by Quantity: Drag Quantity to the Size shelf.
  3. Label by Region: Drag Region to the Label shelf.
Columns: Sales
Rows: Profit
Color: Category

Adding Trend Lines

  • Trend lines help identify the overall direction of the data points.
  • To add a trend line, right-click on the scatter plot, select Trend Lines, and choose Show Trend Lines.

Example: Adding a Trend Line

  1. Right-click on Scatter Plot: Select Trend Lines.
  2. Show Trend Lines: Choose Show Trend Lines.

Advanced Techniques

Using Calculated Fields

  • Create calculated fields to derive new metrics or dimensions.
  • Example: Calculate the profit ratio.
Profit Ratio = [Profit] / [Sales]
  1. Create Calculated Field: Go to Analysis > Create Calculated Field.
  2. Enter Formula: Name it Profit Ratio and enter the formula [Profit] / [Sales].
  3. Use in Scatter Plot: Drag Profit Ratio to the Columns or Rows shelf.

Dual-Axis Scatter Plots

  • Combine two scatter plots on the same graph using dual axes.
  • Useful for comparing two different relationships simultaneously.

Example: Creating a Dual-Axis Scatter Plot

  1. Drag Fields:
    • Drag Sales to the Columns shelf.
    • Drag Profit to the Rows shelf.
  2. Add Second Measure: Drag Quantity to the opposite axis (e.g., right-click on the axis and select Dual-Axis).
  3. Synchronize Axes: Right-click on one of the axes and select Synchronize Axis.

Conclusion

Scatter plots are a versatile tool in Tableau for visualizing relationships between variables. Starting from basic scatter plots, you can enhance your visualizations with colors, sizes, labels, trend lines, calculated fields, and dual axes. Mastering these techniques will enable you to create insightful and compelling data visualizations.
