Introduction to Table Calculations

Table calculations in Tableau are transformations you can apply to the values in a visualization. They are computed based on the data in the view and can be used to create complex calculations without needing to alter the underlying data source.

Key Concepts

  • Table Calculation: A transformation applied to the values in the view.
  • Scope: The subset of data used to compute the calculation.
  • Direction: The order in which the calculation is applied.

Types of Table Calculations

Tableau provides several built-in table calculations that can be used to perform common tasks.

Common Table Calculations

  • Running Total: Computes a cumulative total.
  • Percent of Total: Calculates the percentage that each value contributes to the total.
  • Difference: Computes the difference between values.
  • Percent Difference: Calculates the percentage difference between values.
  • Rank: Assigns a rank to each value.

Example: Running Total

To create a running total, follow these steps:

  1. Drag a measure to the Rows shelf.
  2. Right-click the measure and select "Quick Table Calculation" > "Running Total".

Custom Table Calculations

In addition to built-in calculations, you can create custom table calculations using calculated fields.

Creating a Custom Table Calculation

  1. Right-click in the Data pane and select "Create Calculated Field".
  2. Enter a name for the calculation.
  3. Enter a formula using table calculation functions.

Example: Custom Moving Average

To create a 3-month moving average:

  1. Create a calculated field with the following formula:
WINDOW_AVG(SUM([Sales]), -2, 0)

This formula calculates the average of the current month and the two preceding months.

Advanced Table Calculations

Advanced table calculations involve more complex transformations and often require a deeper understanding of the data and the calculation functions.

Nested Table Calculations

Nested table calculations involve using one table calculation as part of another.

Example: Nested Table Calculation

  1. Create a calculated field for the running total:
RUNNING_SUM(SUM([Sales]))
  1. Create another calculated field for the percent of total based on the running total:
SUM([Sales]) / RUNNING_SUM(SUM([Sales]))

Conclusion

Table calculations in Tableau are powerful tools for transforming and analyzing data directly within your visualizations. By understanding and utilizing both built-in and custom table calculations, you can perform complex analyses without altering your underlying data source. Practice with different types of table calculations to become proficient in using them to gain deeper insights from your data.
