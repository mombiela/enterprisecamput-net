Scheduling Extracts
Using Tableau Online ►

In this section, we will cover the concept of scheduling extracts in Tableau, which is essential for automating data refreshes and ensuring that your visualizations are always up-to-date. We will start with the basics and gradually move to more advanced topics.

Introduction to Extracts

Extracts are snapshots of data optimized for aggregation and loaded into system memory to be quickly recalled for visualization. They can significantly improve performance, especially with large datasets.

  • Benefits of Extracts:
    • Improved performance.
    • Offline access to data.
    • Reduced load on the original data source.

Creating Extracts

Before scheduling extracts, you need to create them. Hereâ€™s how you can create an extract in Tableau:

  1. Connect to your data source.
  2. Select the data you want to extract.
  3. Click on the Data menu and select Extract Data.
  4. Configure the extract options (e.g., filters, aggregation).
  5. Click Extract.
# Example: Creating an Extract
1. Open Tableau and connect to your data source.
2. Go to the Data menu and select 'Extract Data'.
3. Choose the fields and filters you want to include in the extract.
4. Click 'Extract' to create the extract file.

Scheduling Extracts in Tableau Desktop

Scheduling extracts in Tableau Desktop involves setting up a schedule to refresh the data at specified intervals.

  • Steps to Schedule Extracts:
    1. Create an extract as described above.
    2. Publish the workbook to Tableau Server or Tableau Online.
    3. Configure the extract refresh schedule during the publishing process.
# Example: Scheduling an Extract in Tableau Desktop
1. Create and save your extract.
2. Go to 'Server' -> 'Publish Workbook'.
3. In the 'Publish Workbook' dialog, select 'Scheduling & Authentication'.
4. Choose an existing schedule or create a new one.
5. Publish the workbook.

Scheduling Extracts in Tableau Server

Tableau Server provides more robust options for scheduling and managing extracts.

  • Steps to Schedule Extracts:
    1. Navigate to the Data Sources tab on Tableau Server.
    2. Select the data source you want to schedule.
    3. Click on the Extract Refreshes tab.
    4. Click on New Extract Refresh.
    5. Configure the schedule (e.g., frequency, time).
# Example: Scheduling an Extract in Tableau Server
1. Log in to Tableau Server and go to the 'Data Sources' tab.
2. Select the data source you want to schedule.
3. Click on 'Extract Refreshes' and then 'New Extract Refresh'.
4. Set the frequency (e.g., daily, weekly) and time for the refresh.
5. Save the schedule.

Advanced Scheduling Options

For advanced users, Tableau provides additional options such as incremental refreshes and conditional refreshes.

  • Incremental Refresh: Only new or updated data is added to the extract.
  • Conditional Refresh: Refreshes are triggered based on specific conditions or events.
# Example: Incremental Refresh
1. While creating the extract, select 'Incremental refresh'.
2. Specify the column to track changes (e.g., a timestamp column).
3. Schedule the extract as usual.

# Example: Conditional Refresh
1. Use Tableau's REST API to set up conditional refreshes.
2. Write a script to trigger the refresh based on specific conditions.
3. Schedule the script to run at desired intervals.

Conclusion

Scheduling extracts in Tableau is a powerful feature that ensures your data visualizations are always up-to-date without manual intervention. By understanding how to create, schedule, and manage extracts, you can significantly improve the efficiency and reliability of your Tableau dashboards.

  • Key Takeaways:
    • Extracts improve performance and provide offline access.
    • Scheduling extracts automates data refreshes.
    • Tableau Server offers more advanced scheduling options.

By mastering these concepts, you can ensure that your Tableau visualizations are both performant and accurate, providing real-time insights to your stakeholders.
