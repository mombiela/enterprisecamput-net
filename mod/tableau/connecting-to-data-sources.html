Connecting to Data Sources
Basic Data Preparation ►

Connecting to data sources is a fundamental skill in Tableau, enabling you to visualize and analyze data from various origins. This topic will guide you through the process of connecting to different data sources, from basic to advanced levels.

Introduction to Data Connections

Before diving into the specifics, it's essential to understand what data connections are and why they are crucial in Tableau.

  • Data Connections: The links between Tableau and the data sources you want to analyze.
  • Importance: Allows you to pull data into Tableau for visualization and analysis.

Types of Data Sources

Tableau supports a wide range of data sources. Here are some common types:

  • File-Based: Excel, CSV, JSON, PDF.
  • Database Servers: MySQL, PostgreSQL, SQL Server, Oracle.
  • Cloud-Based: Google Sheets, Salesforce, Amazon Redshift.
  • Web Data Connectors: APIs, custom connectors.

Connecting to File-Based Data Sources

Excel

  1. Open Tableau.
  2. Click on "Microsoft Excel" under the "Connect" pane.
  3. Select the Excel file you want to connect to.
  4. Drag and drop sheets from the left pane to the canvas.
# Example: Connecting to an Excel file
1. Open Tableau.
2. Click on "Microsoft Excel".
3. Select "SalesData.xlsx".
4. Drag "Sheet1" to the canvas.

CSV

  1. Open Tableau.
  2. Click on "Text File" under the "Connect" pane.
  3. Select the CSV file you want to connect to.
  4. Drag and drop the file to the canvas.
# Example: Connecting to a CSV file
1. Open Tableau.
2. Click on "Text File".
3. Select "SalesData.csv".
4. Drag the file to the canvas.

Connecting to Database Servers

MySQL

  1. Open Tableau.
  2. Click on "MySQL" under the "Connect" pane.
  3. Enter the server details (Server, Port, Database, Username, Password).
  4. Click "Sign In".
# Example: Connecting to a MySQL database
1. Open Tableau.
2. Click on "MySQL".
3. Enter the server details:
   - Server: `localhost`
   - Port: `3306`
   - Database: `sales_db`
   - Username: `root`
   - Password: `password`
4. Click "Sign In".

PostgreSQL

  1. Open Tableau.
  2. Click on "PostgreSQL" under the "Connect" pane.
  3. Enter the server details (Server, Port, Database, Username, Password).
  4. Click "Sign In".
# Example: Connecting to a PostgreSQL database
1. Open Tableau.
2. Click on "PostgreSQL".
3. Enter the server details:
   - Server: `localhost`
   - Port: `5432`
   - Database: `sales_db`
   - Username: `postgres`
   - Password: `password`
4. Click "Sign In".

Connecting to Cloud-Based Data Sources

Google Sheets

  1. Open Tableau.
  2. Click on "Google Sheets" under the "Connect" pane.
  3. Sign in to your Google account.
  4. Select the Google Sheet you want to connect to.
  5. Drag and drop sheets from the left pane to the canvas.
# Example: Connecting to Google Sheets
1. Open Tableau.
2. Click on "Google Sheets".
3. Sign in to your Google account.
4. Select "SalesData" Google Sheet.
5. Drag "Sheet1" to the canvas.

Salesforce

  1. Open Tableau.
  2. Click on "Salesforce" under the "Connect" pane.
  3. Sign in to your Salesforce account.
  4. Select the data you want to connect to.
  5. Drag and drop objects from the left pane to the canvas.
# Example: Connecting to Salesforce
1. Open Tableau.
2. Click on "Salesforce".
3. Sign in to your Salesforce account.
4. Select the data you want to connect to.
5. Drag the objects to the canvas.

Advanced Data Connections

Web Data Connectors

  1. Open Tableau.
  2. Click on "Web Data Connector" under the "Connect" pane.
  3. Enter the URL of the web data connector.
  4. Follow the prompts to connect to the data.
# Example: Connecting to a Web Data Connector
1. Open Tableau.
2. Click on "Web Data Connector".
3. Enter the URL: `https://example.com/webdataconnector`.
4. Follow the prompts to connect to the data.

Custom SQL

  1. Open Tableau.
  2. Connect to a database (e.g., MySQL, PostgreSQL).
  3. Click on "New Custom SQL".
  4. Enter your SQL query.
-- Example: Custom SQL Query
SELECT 
    customer_id,
    customer_name,
    total_sales
FROM 
    sales_data
WHERE 
    total_sales > 1000;

Conclusion

Connecting to data sources in Tableau is a critical step in the data visualization process. Whether you are working with file-based data, databases, cloud services, or custom web connectors, Tableau provides a robust set of tools to help you bring your data into the platform. Mastering these connections will enable you to leverage the full power of Tableau for your data analysis needs.
