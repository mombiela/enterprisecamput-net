Creating Parameters in Tableau

Introduction

Parameters in Tableau are dynamic values that can replace constant values in calculations, filters, and reference lines. They allow users to create more interactive and flexible visualizations. This section will guide you through the process of creating and using parameters, from basic to advanced levels.

What are Parameters?

  • Definition: Parameters are dynamic input controls that allow users to change the values used in calculations, filters, and other elements of a Tableau workbook.
  • Use Cases:
    • Allowing users to select a value from a list.
    • Enabling dynamic calculations.
    • Creating what-if analysis scenarios.

Creating Basic Parameters

Step-by-Step Guide

  1. Open Tableau and connect to your data source.
  2. Right-click in the Data pane and select Create Parameter.
  3. Name your parameter appropriately.
  4. Set the data type (e.g., Integer, Float, String, Date).
  5. Specify the allowable values:
    • All: Any value within the data type.
    • List: Specific values you define.
    • Range: A range of values with a minimum and maximum.

Example

Parameter Name: Sales Threshold
Data Type: Float
Allowable Values: Range
Minimum: 0
Maximum: 100000
Step Size: 1000

Using Parameters in Calculations

  1. Create a calculated field using the parameter.
  2. Example Calculation: 
    IF [Sales] > [Sales Threshold] THEN 'Above Threshold' ELSE 'Below Threshold' END
  3. Drag the calculated field to the Rows or Columns shelf to visualize the data.

Intermediate Use of Parameters

Dynamic Filtering

  • Create a parameter to filter data dynamically.
  • Example: 
    Parameter Name: Region Selector
Data Type: String
Allowable Values: List
Values: East, West, Central, South
  • Create a calculated field: 
    IF [Region] = [Region Selector] THEN [Sales] END
  • Use the calculated field in your visualization to filter data based on the selected region.

Parameter Actions

  • Parameter actions allow users to change parameter values through interactions with the visualization.
  • Steps:
    1. Create a parameter.
    2. Create a parameter action by going to the Worksheet menu and selecting Actions.
    3. Configure the action to update the parameter based on user interaction.

Advanced Parameter Techniques

Cascading Parameters

  • Cascading parameters are used when the value of one parameter depends on the value of another.
  • Example:
    • Parameter 1: Country
    • Parameter 2: State (values depend on the selected country)
  • Steps:
    1. Create both parameters.
    2. Create a calculated field to filter the values of the second parameter based on the first.
    3. Use the calculated field in your visualizations.

Parameterized Calculations

  • Complex calculations can be parameterized to allow for dynamic analysis.
  • Example: 
    Parameter Name: Discount Rate
Data Type: Float
Allowable Values: Range
Minimum: 0
Maximum: 0.5
Step Size: 0.01
    • Calculation: 
      [Sales] * (1 - [Discount Rate])

Conclusion

Parameters are a powerful feature in Tableau that enhance the interactivity and flexibility of your visualizations. By understanding how to create and use parameters, you can provide users with dynamic control over the data they are analyzing. From basic parameter creation to advanced techniques like cascading parameters and parameterized calculations, mastering parameters will significantly improve your Tableau skills.
