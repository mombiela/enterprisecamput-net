Introduction

Welcome to the "Basic Scripting with C#" course for Unity. This course is designed to take you from a beginner to an advanced level in scripting with C# within the Unity environment. By the end of this course, you will be able to create interactive and dynamic games using C# scripts.

Getting Started with C# in Unity

Setting Up Your Environment

  • Install Unity Hub and Unity Editor: Ensure you have the latest version of Unity installed.
  • Create a New Project: Open Unity Hub, click on "New Project," and select the "3D" template.
  • Familiarize with the Unity Interface: Understand the Scene, Game, Hierarchy, Project, and Inspector windows.

Your First Script

  • Creating a Script:

    1. Right-click in the Project window.
    2. Select Create > C# Script.
    3. Name your script HelloWorld.

  • Editing the Script:

    using UnityEngine;

public class HelloWorld : MonoBehaviour
{
    // Start is called before the first frame update
    void Start()
    {
        Debug.Log("Hello, World!");
    }

    // Update is called once per frame
    void Update()
    {

    }
}

  • Attaching the Script to a GameObject:

    1. Create a new GameObject in the Hierarchy window.
    2. Drag and drop the HelloWorld script onto the GameObject.

Key Concepts

  • MonoBehaviour: The base class from which every Unity script derives.
  • Start() and Update() Methods:
    • Start(): Called before the first frame update.
    • Update(): Called once per frame.

Variables and Data Types

Declaring Variables

  • Syntax: 
    int score = 0;
float speed = 5.5f;
string playerName = "Player1";
bool isGameOver = false;

Common Data Types

  • int: Integer numbers.
  • float: Floating-point numbers.
  • string: Text.
  • bool: Boolean values (true/false).

Example: Using Variables in Unity

using UnityEngine;

public class Player : MonoBehaviour
{
    public int score = 0;
    public float speed = 5.5f;
    public string playerName = "Player1";
    public bool isGameOver = false;

    void Start()
    {
        Debug.Log("Player Name: " + playerName);
        Debug.Log("Initial Score: " + score);
    }

    void Update()
    {
        if (isGameOver)
        {
            Debug.Log("Game Over!");
        }
    }
}

Control Structures

Conditional Statements

  • if-else: 
    if (score >= 100)
{
    Debug.Log("You win!");
}
else
{
    Debug.Log("Keep trying!");
}

Loops

  • for Loop:

    for (int i = 0; i < 10; i++)
{
    Debug.Log("Count: " + i);
}

  • while Loop:

    int count = 0;
while (count < 10)
{
    Debug.Log("Count: " + count);
    count++;
}

Functions and Methods

Creating Functions

  • Syntax: 
    void PrintMessage(string message)
{
    Debug.Log(message);
}

Calling Functions

  • Example: 
    void Start()
{
    PrintMessage("Hello from a function!");
}

Unity-Specific Concepts

GameObject and Components

  • GameObject: The fundamental objects in Unity scenes.
  • Components: Scripts, physics, rendering, and other functionalities attached to GameObjects.

Transform Component

  • Position, Rotation, Scale: 
    void Update()
{
    transform.position = new Vector3(0, 1, 0);
    transform.rotation = Quaternion.Euler(0, 45, 0);
    transform.localScale = new Vector3(1, 2, 1);
}

Handling Input

  • Keyboard Input: 
    void Update()
{
    if (Input.GetKeyDown(KeyCode.Space))
    {
        Debug.Log("Space key was pressed.");
    }
}

Conclusion

In this topic, we covered the basics of scripting with C# in Unity. We started with setting up the environment and creating your first script. We then explored variables, control structures, functions, and Unity-specific concepts like GameObjects and components. By understanding these fundamentals, you are now equipped to start building more complex and interactive Unity projects. Continue practicing and experimenting with these concepts to deepen your understanding and enhance your skills. Happy coding!
