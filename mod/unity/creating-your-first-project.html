Welcome to your journey into Unity! In this section, we will guide you through the process of creating your first project in Unity. This is an essential step that will set the foundation for your future development work.

Setting Up Unity

Before we dive into creating a project, ensure you have Unity installed on your computer. You can download it from the Unity website.

Unity Hub

Unity Hub is a management tool that helps you manage multiple Unity installations and projects. Make sure you have Unity Hub installed as well.

  1. Download and Install Unity Hub: Follow the instructions on the Unity website to download and install Unity Hub.
  2. Install Unity Editor: Open Unity Hub, go to the "Installs" tab, and click "Add" to install the latest version of the Unity Editor.

Creating a New Project

Once Unity Hub and the Unity Editor are installed, you can create your first project.

  1. Open Unity Hub: Launch Unity Hub from your applications.
  2. Create a New Project:
    • Click on the "New" button.
    • Choose a template. For beginners, we recommend starting with the "3D" template.
    • Name your project (e.g., "MyFirstUnityProject").
    • Choose a location to save your project.
    • Click "Create".

Understanding the Unity Interface

When your project opens, you'll see the Unity Editor interface. Here are the key components:

  • Scene View: This is where you build and arrange your game objects.
  • Game View: This shows what your game will look like when it runs.
  • Hierarchy: This lists all the game objects in your scene.
  • Inspector: This shows properties and settings for the selected game object.
  • Project: This is where all your assets (models, scripts, textures, etc.) are stored.
  • Console: This displays messages, warnings, and errors.

Adding a Game Object

Let's add a simple game object to your scene.

  1. Create a Cube:

    • Right-click in the Hierarchy window.
    • Select 3D Object > Cube.
    • A cube will appear in your Scene view and be listed in the Hierarchy.

  2. Position the Cube:

    • Select the cube in the Hierarchy.
    • In the Inspector, set the Position to (0, 0, 0).

Writing Your First Script

Now, let's add some interactivity with a simple script.

  1. Create a Script:

    • In the Project window, right-click and select Create > C# Script.
    • Name the script MoveCube.

  2. Edit the Script:

    • Double-click the MoveCube script to open it in your code editor (e.g., Visual Studio).
    • Replace the default code with the following:
using UnityEngine;

public class MoveCube : MonoBehaviour
{
    public float speed = 5f;

    void Update()
    {
        float move = Input.GetAxis("Horizontal") * speed * Time.deltaTime;
        transform.Translate(move, 0, 0);
    }
}
  1. Attach the Script to the Cube:
    • Drag the MoveCube script from the Project window onto the cube in the Hierarchy.

Running Your Project

  1. Save Your Scene:

    • Go to File > Save Scene.
    • Name your scene (e.g., "MainScene").

  2. Play the Game:

    • Click the "Play" button at the top of the Unity Editor.
    • Use the left and right arrow keys to move the cube.

Summary

Congratulations! You've created your first Unity project, added a game object, and wrote a script to make it interactive. Here's a quick recap of what we've covered:

  • Setting up Unity and Unity Hub.
  • Creating a new project.
  • Understanding the Unity interface.
  • Adding and positioning a game object.
  • Writing and attaching a simple script.
  • Running your project.

This foundational knowledge will be crucial as you continue to explore and develop more complex projects in Unity. Happy coding!
