Introduction

In Unity, materials define how objects appear on the screen. They control the color, texture, and reflectivity of objects, among other properties. Understanding how to create and use materials is essential for making your game visually appealing.

Creating Materials

Basic Material Creation

  1. Open the Project Window: Navigate to the Project window in Unity.

  2. Create a New Material:

    • Right-click in the Project window.
    • Select Create > Material.
    • Name your new material (e.g., "MyMaterial").

  3. Assigning a Shader:

    • By default, the new material uses the Standard Shader.
    • You can change the shader by selecting the material and choosing a different shader from the Shader dropdown in the Inspector.

Setting Material Properties

  • Color: Change the base color of the material.
  • Texture: Add a texture to the material by dragging a texture file into the Albedo slot.
  • Metallic and Smoothness: Adjust these sliders to change the material’s reflectivity and smoothness.
// Example: Creating and assigning a material via script
using UnityEngine;

public class MaterialExample : MonoBehaviour
{
    void Start()
    {
        // Create a new material with a standard shader
        Material myMaterial = new Material(Shader.Find("Standard"));
        
        // Set the color to red
        myMaterial.color = Color.red;
        
        // Assign the material to the object's renderer
        GetComponent<Renderer>().material = myMaterial;
    }
}

Using Materials

Applying Materials to Objects

  1. Drag and Drop:

    • Drag the material from the Project window onto the object in the Scene or Hierarchy window.

  2. Via Inspector:

    • Select the object in the Scene or Hierarchy window.
    • In the Inspector, find the Mesh Renderer component.
    • Drag the material into the Element 0 slot under Materials.

Material Instances

  • Shared Material: Changes to the material affect all objects using it.
  • Instance Material: Changes affect only the specific instance of the material on the object.
// Example: Modifying a material instance
using UnityEngine;

public class InstanceMaterialExample : MonoBehaviour
{
    void Start()
    {
        // Get the renderer component
        Renderer renderer = GetComponent<Renderer>();
        
        // Create an instance of the material
        Material instanceMaterial = renderer.material;
        
        // Change the color of the instance material
        instanceMaterial.color = Color.blue;
    }
}

Advanced Material Techniques

Using Custom Shaders

  • Shader Graph: Unity’s visual tool for creating shaders without writing code.
  • Custom Shader Code: Write your own shaders using HLSL.

Material Property Blocks

  • Efficiently Change Material Properties: Use MaterialPropertyBlock to change properties without creating new material instances.
// Example: Using MaterialPropertyBlock
using UnityEngine;

public class MaterialPropertyBlockExample : MonoBehaviour
{
    void Start()
    {
        Renderer renderer = GetComponent<Renderer>();
        MaterialPropertyBlock block = new MaterialPropertyBlock();
        
        // Set a color property
        block.SetColor("_Color", Color.green);
        
        // Apply the property block to the renderer
        renderer.SetPropertyBlock(block);
    }
}

Conclusion

Creating and using materials in Unity is a fundamental skill for game development. By understanding how to create materials, set their properties, and apply them to objects, you can significantly enhance the visual quality of your games. Advanced techniques like custom shaders and MaterialPropertyBlocks allow for even more control and optimization. Practice these concepts to become proficient in managing materials in Unity.
