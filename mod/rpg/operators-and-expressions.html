Introduction

In RPG (Report Program Generator), operators and expressions are fundamental components that allow you to perform calculations, make decisions, and manipulate data. This section will cover the various types of operators available in RPG and how to use them effectively in expressions.

Arithmetic Operators

Arithmetic operators are used to perform mathematical calculations.

  • Addition (+): Adds two operands.
  • Subtraction (-): Subtracts the second operand from the first.
  • Multiplication (*): Multiplies two operands.
  • Division (/): Divides the first operand by the second.
  • Exponentiation (**): Raises the first operand to the power of the second.

Example

Dcl-S num1 Int(10) Inz(10);
Dcl-S num2 Int(10) Inz(5);
Dcl-S result Int(10);

result = num1 + num2; // result is 15
result = num1 - num2; // result is 5
result = num1 * num2; // result is 50
result = num1 / num2; // result is 2
result = num1 ** num2; // result is 100000

Comparison Operators

Comparison operators are used to compare two values and return a Boolean result (True or False).

  • Equal to (= or EQ)
  • Not equal to (<> or NE)
  • Greater than (> or GT)
  • Less than (< or LT)
  • Greater than or equal to (>= or GE)
  • Less than or equal to (<= or LE)

Example

Dcl-S num1 Int(10) Inz(10);
Dcl-S num2 Int(10) Inz(5);
Dcl-S isEqual Ind;

isEqual = (num1 = num2); // isEqual is *False
isEqual = (num1 <> num2); // isEqual is *True
isEqual = (num1 > num2); // isEqual is *True
isEqual = (num1 < num2); // isEqual is *False
isEqual = (num1 >= num2); // isEqual is *True
isEqual = (num1 <= num2); // isEqual is *False

Logical Operators

Logical operators are used to combine multiple conditions.

  • AND (& or AND): Returns True if both conditions are True.
  • OR (| or OR): Returns True if at least one condition is True.
  • NOT (NOT): Reverses the Boolean value of the condition.

Example

Dcl-S num1 Int(10) Inz(10);
Dcl-S num2 Int(10) Inz(5);
Dcl-S num3 Int(10) Inz(15);
Dcl-S condition Ind;

condition = (num1 > num2) AND (num1 < num3); // condition is *True
condition = (num1 > num2) OR (num1 > num3); // condition is *True
condition = NOT(num1 = num2); // condition is *True

Assignment Operators

Assignment operators are used to assign values to variables.

  • Simple assignment (=): Assigns the value of the right operand to the left operand.
  • Addition assignment (+=): Adds the right operand to the left operand and assigns the result to the left operand.
  • Subtraction assignment (-=): Subtracts the right operand from the left operand and assigns the result to the left operand.
  • Multiplication assignment (*=): Multiplies the left operand by the right operand and assigns the result to the left operand.
  • Division assignment (/=): Divides the left operand by the right operand and assigns the result to the left operand.

Example

Dcl-S num Int(10) Inz(10);

num = 5; // num is 5
num += 3; // num is 8
num -= 2; // num is 6
num *= 4; // num is 24
num /= 3; // num is 8

String Operators

String operators are used to manipulate text strings.

  • Concatenation (+): Combines two strings into one.

Example

Dcl-S firstName Char(20) Inz('John');
Dcl-S lastName Char(20) Inz('Doe');
Dcl-S fullName Char(40);

fullName = firstName + ' ' + lastName; // fullName is 'John Doe'

Summary

In this section, we covered the various types of operators available in RPG, including arithmetic, comparison, logical, assignment, and string operators. Understanding how to use these operators effectively is crucial for performing calculations, making decisions, and manipulating data in your RPG programs. Practice using these operators in different scenarios to become proficient in writing expressions in RPG.
