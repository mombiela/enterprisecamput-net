Introduction

In this final project, you will apply everything you've learned throughout the course to create a comprehensive RPG (Role-Playing Game) application. This project will help you consolidate your knowledge and skills in game development, object-oriented programming, data structures, and more.

Project Overview

The RPG application will include the following features:

  • Character creation and management
  • Inventory system
  • Combat mechanics
  • Quests and story progression
  • Save and load game functionality

Character Creation and Management

Key Concepts

  • Classes and Objects
  • Inheritance and Polymorphism
  • Encapsulation

Example

class Character:
    def __init__(self, name, health, strength, defense):
        self.name = name
        self.health = health
        self.strength = strength
        self.defense = defense

    def attack(self, target):
        damage = self.strength - target.defense
        target.health -= damage
        return damage

class Warrior(Character):
    def __init__(self, name):
        super().__init__(name, health=100, strength=20, defense=10)

class Mage(Character):
    def __init__(self, name):
        super().__init__(name, health=70, strength=30, defense=5)

# Example usage
hero = Warrior("Aragorn")
villain = Mage("Saruman")
damage_dealt = hero.attack(villain)
print(f"{hero.name} dealt {damage_dealt} damage to {villain.name}.")

Inventory System

Key Concepts

  • Lists and Dictionaries
  • Methods and Functions

Example

class Inventory:
    def __init__(self):
        self.items = {}

    def add_item(self, item_name, quantity=1):
        if item_name in self.items:
            self.items[item_name] += quantity
        else:
            self.items[item_name] = quantity

    def remove_item(self, item_name, quantity=1):
        if item_name in self.items:
            self.items[item_name] -= quantity
            if self.items[item_name] <= 0:
                del self.items[item_name]

# Example usage
inventory = Inventory()
inventory.add_item("Health Potion", 3)
inventory.add_item("Mana Potion")
inventory.remove_item("Health Potion")
print(inventory.items)

Combat Mechanics

Key Concepts

  • Conditional Statements
  • Loops

Example

def combat_round(attacker, defender):
    damage = attacker.attack(defender)
    print(f"{attacker.name} attacks {defender.name} for {damage} damage.")
    if defender.health <= 0:
        print(f"{defender.name} has been defeated!")
        return True
    return False

# Example usage
hero = Warrior("Aragorn")
villain = Mage("Saruman")
while villain.health > 0 and hero.health > 0:
    if combat_round(hero, villain):
        break
    if combat_round(villain, hero):
        break

Quests and Story Progression

Key Concepts

  • Classes and Objects
  • Lists and Dictionaries

Example

class Quest:
    def __init__(self, name, description, is_completed=False):
        self.name = name
        self.description = description
        self.is_completed = is_completed

    def complete_quest(self):
        self.is_completed = True

# Example usage
quest1 = Quest("Save the Village", "Defeat the goblins attacking the village.")
print(f"Quest: {quest1.name} - {quest1.description}")
quest1.complete_quest()
print(f"Quest Completed: {quest1.is_completed}")

Save and Load Game Functionality

Key Concepts

  • File I/O
  • Serialization and Deserialization

Example

import json

class Game:
    def __init__(self, character, inventory):
        self.character = character
        self.inventory = inventory

    def save_game(self, filename):
        game_data = {
            'character': self.character.__dict__,
            'inventory': self.inventory.items
        }
        with open(filename, 'w') as file:
            json.dump(game_data, file)

    def load_game(self, filename):
        with open(filename, 'r') as file:
            game_data = json.load(file)
            self.character.__dict__.update(game_data['character'])
            self.inventory.items = game_data['inventory']

# Example usage
hero = Warrior("Aragorn")
inventory = Inventory()
game = Game(hero, inventory)
game.save_game("savefile.json")
game.load_game("savefile.json")
