Introduction

In RPG (Report Program Generator), understanding variables and data types is fundamental. This section will cover the basics of variables, how to declare them, and the various data types available in RPG. By the end of this section, you will be able to effectively use variables and understand the different data types in RPG.

Variables

Variables are used to store data that can be manipulated and retrieved throughout a program. In RPG, variables must be declared before they can be used.

Declaring Variables

In RPG, variables are declared using the DCL-S (Declare Standalone) statement. Here is the basic syntax:

DCL-S variableName dataType;

Example

DCL-S myNumber INT(10);
DCL-S myString CHAR(50);

In this example:

  • myNumber is declared as an integer with a length of 10.
  • myString is declared as a character string with a length of 50.

Data Types

RPG supports several data types, each serving different purposes. The most commonly used data types are:

  • Character (CHAR)
  • Numeric (INT, DEC, UNS)
  • Date (DATE)
  • Time (TIME)
  • Timestamp (TIMESTAMP)

Character Data Type

Character data types are used to store alphanumeric data.

DCL-S customerName CHAR(50);

Numeric Data Types

Numeric data types are used to store numbers. There are different types of numeric data types in RPG:

  • Integer (INT): Used for whole numbers.
  • Decimal (DEC): Used for numbers with decimal points.
  • Unsigned (UNS): Used for non-negative whole numbers.
DCL-S age INT(3);
DCL-S salary DEC(7:2);
DCL-S productCount UNS(5);

Date, Time, and Timestamp Data Types

These data types are used to store date and time information.

DCL-S birthDate DATE;
DCL-S startTime TIME;
DCL-S lastUpdated TIMESTAMP;

Examples and Exercises

Example 1: Basic Variable Declaration and Initialization

DCL-S employeeName CHAR(50) INZ('John Doe');
DCL-S employeeAge INT(3) INZ(30);
DCL-S hireDate DATE INZ(D'2023-01-01');

In this example:

  • employeeName is initialized with the value 'John Doe'.
  • employeeAge is initialized with the value 30.
  • hireDate is initialized with the date '2023-01-01'.

Exercise 1: Declare and Initialize Variables

Declare and initialize the following variables:

  • productName as a character string with a length of 100 and initialize it with 'Laptop'.
  • productPrice as a decimal with 7 digits and 2 decimal places, initialize it with 999.99.
  • manufactureDate as a date, initialize it with '2022-12-15'.

Example 2: Using Variables in Expressions

DCL-S quantity INT(5) INZ(10);
DCL-S price DEC(7:2) INZ(15.75);
DCL-S totalCost DEC(9:2);

totalCost = quantity * price;

In this example:

  • quantity is multiplied by price and the result is stored in totalCost.

Exercise 2: Calculate Total Price

Declare and initialize the following variables:

  • unitPrice as a decimal with 7 digits and 2 decimal places, initialize it with 20.50.
  • quantityOrdered as an integer with 5 digits, initialize it with 25.
  • Calculate the total price and store it in a variable named totalPrice.

Summary

In this section, we covered the basics of variables and data types in RPG. We learned how to declare variables, initialize them, and use different data types such as character, numeric, date, time, and timestamp. Understanding these concepts is crucial for writing effective RPG programs. Make sure to practice the exercises to reinforce your understanding.
