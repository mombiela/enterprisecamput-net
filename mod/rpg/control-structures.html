Control Structures: IF, ELSE, and SELECT

Introduction

Control structures are fundamental in programming as they allow the flow of execution to be altered based on certain conditions. In RPG (Report Program Generator), control structures such as IF, ELSE, and SELECT are used to make decisions within the code.

IF Statement

The IF statement is used to execute a block of code only if a specified condition is true.

Syntax

IF condition;
   // Code to execute if condition is true
ENDIF;

Example

DCL-S age INT(3);
age = 20;

IF age >= 18;
   DSPLY 'You are an adult.';
ENDIF;
  • Explanation: In this example, the program checks if the variable age is greater than or equal to 18. If the condition is true, it displays the message "You are an adult."

ELSE Statement

The ELSE statement is used in conjunction with the IF statement to execute a block of code if the condition is false.

Syntax

IF condition;
   // Code to execute if condition is true
ELSE;
   // Code to execute if condition is false
ENDIF;

Example

DCL-S age INT(3);
age = 16;

IF age >= 18;
   DSPLY 'You are an adult.';
ELSE;
   DSPLY 'You are a minor.';
ENDIF;
  • Explanation: Here, the program checks if age is greater than or equal to 18. If true, it displays "You are an adult." If false, it displays "You are a minor."

ELSEIF Statement

The ELSEIF statement allows for multiple conditions to be checked in sequence.

Syntax

IF condition1;
   // Code to execute if condition1 is true
ELSEIF condition2;
   // Code to execute if condition2 is true
ELSE;
   // Code to execute if none of the conditions are true
ENDIF;

Example

DCL-S score INT(3);
score = 75;

IF score >= 90;
   DSPLY 'Grade: A';
ELSEIF score >= 80;
   DSPLY 'Grade: B';
ELSEIF score >= 70;
   DSPLY 'Grade: C';
ELSE;
   DSPLY 'Grade: F';
ENDIF;
  • Explanation: The program evaluates the score and assigns a grade based on the value. Multiple conditions are checked in sequence.

SELECT Statement

The SELECT statement is used to execute one block of code from multiple possible blocks based on the value of a variable.

Syntax

SELECT;
   WHEN condition1;
      // Code to execute if condition1 is true
   WHEN condition2;
      // Code to execute if condition2 is true
   OTHER;
      // Code to execute if none of the conditions are true
ENDSL;

Example

DCL-S day CHAR(10);
day = 'Monday';

SELECT;
   WHEN day = 'Monday';
      DSPLY 'Start of the work week.';
   WHEN day = 'Friday';
      DSPLY 'End of the work week.';
   OTHER;
      DSPLY 'Midweek day.';
ENDSL;
  • Explanation: The program checks the value of the variable day and displays a message based on the day of the week.

Comparison of IF and SELECT

| Feature | IF/ELSEIF/ELSE | SELECT/WHEN/OTHER | |------------------|--------------------------|-----------------------------| | Use Case | Simple conditions | Multiple conditions based on a single variable | | Readability | Can become complex | More readable for multiple conditions | | Flexibility | High | Limited to a single variable |

Exercises

  1. Write an RPG program that checks if a number is positive, negative, or zero using IF, ELSEIF, and ELSE statements.
  2. Create an RPG program that assigns a letter grade (A, B, C, D, F) based on a numeric score using the SELECT statement.

Conclusion

Understanding control structures like IF, ELSE, and SELECT is crucial for making decisions in your RPG programs. These structures allow you to control the flow of your program based on various conditions, making your code more dynamic and responsive. Practice using these control structures to become proficient in writing conditional logic in RPG.
