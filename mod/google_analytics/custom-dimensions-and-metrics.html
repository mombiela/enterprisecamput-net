Introduction

Custom Dimensions and Metrics in Google Analytics allow you to collect and analyze data that is specific to your business needs. They extend the default data provided by Google Analytics, enabling you to track additional information that is not automatically captured.

Key Concepts

  • Custom Dimensions: These are user-defined dimensions that you can use to collect data that Google Analytics does not automatically track. For example, you might want to track the membership level of users on your site.
  • Custom Metrics: These are user-defined metrics that allow you to quantify data that is not automatically tracked by Google Analytics. For example, you might want to track the number of times a specific button is clicked.

Setting Up Custom Dimensions and Metrics

To set up Custom Dimensions and Metrics, follow these steps:

  1. Navigate to Admin Settings:

    • Go to your Google Analytics account.
    • Select the property where you want to create Custom Dimensions and Metrics.
    • Under the property column, click on "Custom Definitions" and then "Custom Dimensions" or "Custom Metrics".

  2. Create a Custom Dimension:

    • Click on the "+ New Custom Dimension" button.
    • Enter a name for the dimension (e.g., "Membership Level").
    • Select the scope (Hit, Session, User, or Product).
    • Click "Create".

  3. Create a Custom Metric:

    • Click on the "+ New Custom Metric" button.
    • Enter a name for the metric (e.g., "Button Clicks").
    • Define the type (Integer, Currency, Time).
    • Set the minimum and maximum values.
    • Click "Create".

Implementing Custom Dimensions and Metrics in Code

To send data to Google Analytics using Custom Dimensions and Metrics, you need to modify your tracking code. Here are examples for both Universal Analytics (analytics.js) and Google Analytics 4 (gtag.js).

Universal Analytics (analytics.js)

// Setting a Custom Dimension
ga('create', 'UA-XXXXX-Y', 'auto');
ga('set', 'dimension1', 'Premium'); // Replace 'dimension1' with your Custom Dimension index and 'Premium' with the value

// Setting a Custom Metric
ga('set', 'metric1', 42); // Replace 'metric1' with your Custom Metric index and 42 with the value

ga('send', 'pageview');

Google Analytics 4 (gtag.js)

// Setting a Custom Dimension
gtag('config', 'G-XXXXXXX', {
  'custom_map': {'dimension1': 'membership_level'}
});
gtag('event', 'page_view', {'membership_level': 'Premium'}); // Replace 'membership_level' with your Custom Dimension name and 'Premium' with the value

// Setting a Custom Metric
gtag('config', 'G-XXXXXXX', {
  'custom_map': {'metric1': 'button_clicks'}
});
gtag('event', 'button_click', {'button_clicks': 42}); // Replace 'button_clicks' with your Custom Metric name and 42 with the value

Using Custom Dimensions and Metrics in Reports

Once you have set up and implemented Custom Dimensions and Metrics, you can use them in your reports:

  • Custom Reports: Create custom reports in Google Analytics to include your Custom Dimensions and Metrics.
  • Segments: Use Custom Dimensions to create segments for more detailed analysis.
  • Dashboards: Add widgets to your dashboards that display data from your Custom Dimensions and Metrics.

Example Use Cases

  • E-commerce: Track product categories, user membership levels, or the number of times a product is added to the cart.
  • Content Websites: Track author names, article categories, or the number of comments per article.
  • Applications: Track user roles, feature usage, or error occurrences.

Conclusion

Custom Dimensions and Metrics are powerful tools in Google Analytics that allow you to tailor your data collection to your specific business needs. By setting up and implementing these custom definitions, you can gain deeper insights into user behavior and make more informed decisions.

Remember to:

  • Define what additional data you need.
  • Set up Custom Dimensions and Metrics in the Google Analytics Admin.
  • Implement the necessary code changes.
  • Utilize the data in your reports for better analysis.

By mastering Custom Dimensions and Metrics, you can significantly enhance the value of your Google Analytics data.
