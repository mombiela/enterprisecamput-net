Introduction to Event Tracking

Event tracking in Google Analytics allows you to measure user interactions with your website's elements that don't automatically trigger a pageview. These interactions can include clicks on buttons, downloads, video plays, and more.

Key Concepts

  • Events: User interactions with content that can be tracked independently from a web page or a screen load.
  • Event Category: The name you supply for the group of objects you want to track.
  • Event Action: The type of interaction (e.g., 'click').
  • Event Label: Additional information about the event (e.g., the name of the button clicked).
  • Event Value: A numerical value associated with the event (e.g., the time spent watching a video).

Setting Up Basic Event Tracking

To set up event tracking, you need to modify your website's code to send event data to Google Analytics.

Using Google Tag Manager (GTM)

Google Tag Manager simplifies the process of setting up event tracking without needing to modify your website's code directly.

Steps:

  1. Create a New Tag:

    • Go to your GTM account.
    • Click on "Tags" and then "New".
    • Choose "Tag Configuration" and select "Google Analytics: Universal Analytics".
    • Set the "Track Type" to "Event".

  2. Configure Tag:

    • Fill in the Event Tracking Parameters:
      • Category: e.g., 'Button'
      • Action: e.g., 'Click'
      • Label: e.g., 'Subscribe Button'
      • Value: Optional, e.g., '1'

  3. Set Up Trigger:

    • Click on "Triggering" and then "New".
    • Choose "Click - All Elements" or a specific trigger type.
    • Configure the trigger to fire on the desired interaction (e.g., clicks on a specific button).

  4. Publish the Container:

    • Submit and publish your container to make the changes live.

Example Code for Manual Implementation

If you're not using GTM, you can manually add event tracking code to your website.

<a href="#" onclick="ga('send', 'event', 'Button', 'Click', 'Subscribe Button');">Subscribe</a>

Explanation:

  • ga('send', 'event', 'Button', 'Click', 'Subscribe Button'); sends an event to Google Analytics.
    • 'Button' is the Event Category.
    • 'Click' is the Event Action.
    • 'Subscribe Button' is the Event Label.

Advanced Event Tracking

Advanced event tracking involves more complex interactions and additional parameters.

Tracking Form Submissions

To track form submissions, you can use the following code:

<form id="contactForm" onsubmit="ga('send', 'event', 'Form', 'Submit', 'Contact Form');">
  <!-- form fields -->
  <button type="submit">Submit</button>
</form>

Tracking Video Plays

To track video plays, you can use the YouTube API or similar methods for other video platforms.

Example with YouTube API:

  1. Load the YouTube IFrame API:

    <script src="https://www.youtube.com/iframe_api"></script>

  2. Initialize the Player and Track Events:

    <div id="player"></div>
<script>
  var tag = document.createElement('script');
  tag.src = "https://www.youtube.com/iframe_api";
  var firstScriptTag = document.getElementsByTagName('script')[0];
  firstScriptTag.parentNode.insertBefore(tag, firstScriptTag);

  var player;
  function onYouTubeIframeAPIReady() {
    player = new YT.Player('player', {
      height: '390',
      width: '640',
      videoId: 'M7lc1UVf-VE',
      events: {
        'onStateChange': onPlayerStateChange
      }
    });
  }

  function onPlayerStateChange(event) {
    if (event.data == YT.PlayerState.PLAYING) {
      ga('send', 'event', 'Video', 'Play', 'Demo Video');
    }
  }
</script>

Summary

Setting up event tracking in Google Analytics allows you to measure specific user interactions on your website. Whether using Google Tag Manager for a simplified setup or manually adding tracking code, you can capture valuable data on how users engage with your content. Advanced tracking can include form submissions and video plays, providing deeper insights into user behavior. By understanding and implementing event tracking, you can make data-driven decisions to improve your website's performance and user experience.
