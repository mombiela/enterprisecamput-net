Code Examples

Below is a basic example of how to build a simple neural network using TensorFlow and Keras:

import tensorflow as tf
from tensorflow.keras.models import Sequential
from tensorflow.keras.layers import Dense

# Create a sequential model
model = Sequential()

# Add a dense layer with 10 neurons and ReLU activation function
model.add(Dense(10, activation='relu', input_shape=(784,)))

# Add an output layer with 10 neurons (for 10-class classification) and softmax activation function
model.add(Dense(10, activation='softmax'))

# Compile the model with Adam optimizer and categorical_crossentropy loss function
model.compile(optimizer='adam', loss='categorical_crossentropy', metrics=['accuracy'])

# Model summary
model.summary()

Code Explanation

  • Import libraries: TensorFlow and Keras are imported.
  • Create a sequential model: Sequential allows stacking layers linearly.
  • Add layers:
    • The first Dense layer has 10 neurons and uses the ReLU activation function. The input has 784 features (e.g., flattened 28x28 pixel images).
    • The output Dense layer has 10 neurons (one for each class in a 10-class classification problem) and uses the softmax activation function.
  • Compile the model: The optimizer, loss function, and metrics to track are specified.
  • Model summary: model.summary() shows the model structure.
