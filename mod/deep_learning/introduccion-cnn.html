What are Convolutional Neural Networks (CNN)?

Convolutional Neural Networks (CNN) are a type of deep neural network primarily used for image and visual data analysis. CNNs are especially effective for tasks such as image classification, object detection, and image segmentation.

Key Concepts

  • Convolution: Mathematical operation that combines two sets of information. In the context of CNNs, it is used to extract features from images.
  • Filters/Kernels: Small matrices that slide over the input image to perform the convolution operation.
  • Feature Maps: Results of applying filters to the input image.
  • Pooling: Technique to reduce the dimensionality of the feature maps while retaining the most relevant information.
  • Convolutional Layers: Layers where filters are applied to extract features.
  • Pooling Layers: Layers that reduce the dimensionality of the feature maps.
  • Fully Connected Layers: Final layers of the network where each neuron is connected to all neurons in the previous layer.

Architecture of a CNN

A typical CNN is composed of several layers, each with a specific purpose. Below is a table summarizing the most common layers in a CNN:

Layer Type Description
Input Layer The input image, usually of fixed size (e.g., 224x224x3).
Convolutional Layer Applies filters to extract features from the image.
Activation Layer Applies an activation function (e.g., ReLU) to introduce non-linearity.
Pooling Layer Reduces the dimensionality of the feature maps (e.g., Max Pooling).
Fully Connected Layer Connects all neurons from the previous layer to each neuron in this layer.
Output Layer Produces the final output, such as a classification probability.

Code Example

Below is an example of how to build a simple CNN using TensorFlow and Keras:

import tensorflow as tf
from tensorflow.keras import layers, models

# Define the model
model = models.Sequential()
model.add(layers.Conv2D(32, (3, 3), activation='relu', input_shape=(28, 28, 1)))
model.add(layers.MaxPooling2D((2, 2)))
model.add(layers.Conv2D(64, (3, 3), activation='relu'))
model.add(layers.MaxPooling2D((2, 2)))
model.add(layers.Conv2D(64, (3, 3), activation='relu'))

# Add fully connected layers
model.add(layers.Flatten())
model.add(layers.Dense(64, activation='relu'))
model.add(layers.Dense(10, activation='softmax'))

# Compile the model
model.compile(optimizer='adam',
              loss='sparse_categorical_crossentropy',
              metrics=['accuracy'])

# Model summary
model.summary()

Code Explanation

  1. Library Import: TensorFlow and Keras are imported.
  2. Model Definition: Sequential is used to define a sequence of layers.
    • Convolutional Layers: Three convolutional layers with different numbers of filters.
    • Pooling Layers: Two Max Pooling layers to reduce dimensionality.
    • Fully Connected Layers: A Flatten layer to flatten the feature maps and two dense layers.
  3. Model Compilation: The optimizer, loss function, and metrics are specified.
  4. Model Summary: A summary of the model is printed to verify its structure.

Conclusion
