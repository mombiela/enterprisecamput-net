Introduction

Convolutional Neural Networks (CNN) have revolutionized the field of Deep Learning, especially in image processing. In this section, we will explore some of the most popular CNN architectures that have been fundamental in advancing computer vision.

Classic Architectures

LeNet-5

  • Year of Publication: 1998
  • Authors: Yann LeCun et al.
  • Main Application: Handwritten digit recognition (MNIST).

Structure

  • Convolutional Layers: 2
  • Pooling Layers: 2
  • Fully Connected Layers: 3
import tensorflow as tf
from tensorflow.keras import layers, models

model = models.Sequential()
model.add(layers.Conv2D(6, (5, 5), activation='relu', input_shape=(32, 32, 1)))
model.add(layers.AveragePooling2D())
model.add(layers.Conv2D(16, (5, 5), activation='relu'))
model.add(layers.AveragePooling2D())
model.add(layers.Flatten())
model.add(layers.Dense(120, activation='relu'))
model.add(layers.Dense(84, activation='relu'))
model.add(layers.Dense(10, activation='softmax'))

AlexNet

  • Year of Publication: 2012
  • Authors: Alex Krizhevsky et al.
  • Main Application: Image classification on the ImageNet dataset.

Structure

  • Convolutional Layers: 5
  • Pooling Layers: 3
  • Fully Connected Layers: 3
model = models.Sequential()
model.add(layers.Conv2D(96, (11, 11), strides=(4, 4), activation='relu', input_shape=(227, 227, 3)))
model.add(layers.MaxPooling2D((3, 3), strides=(2, 2)))
model.add(layers.Conv2D(256, (5, 5), activation='relu', padding='same'))
model.add(layers.MaxPooling2D((3, 3), strides=(2, 2)))
model.add(layers.Conv2D(384, (3, 3), activation='relu', padding='same'))
model.add(layers.Conv2D(384, (3, 3), activation='relu', padding='same'))
model.add(layers.Conv2D(256, (3, 3), activation='relu', padding='same'))
model.add(layers.MaxPooling2D((3, 3), strides=(2, 2)))
model.add(layers.Flatten())
model.add(layers.Dense(4096, activation='relu'))
model.add(layers.Dense(4096, activation='relu'))
model.add(layers.Dense(1000, activation='softmax'))

VGGNet

  • Year of Publication: 2014
  • Authors: Karen Simonyan and Andrew Zisserman.
  • Main Application: Image classification on the ImageNet dataset.

Structure

  • Convolutional Layers: 16-19
  • Pooling Layers: 5
  • Fully Connected Layers: 3
model = models.Sequential()
model.add(layers.Conv2D(64, (3, 3), activation='relu', padding='same', input_shape=(224, 224, 3)))
model.add(layers.Conv2D(64, (3, 3), activation='relu', padding='same'))
model.add(layers.MaxPooling2D((2, 2), strides=(2, 2)))
# Repeat convolution and pooling blocks
model.add(layers.Flatten())
model.add(layers.Dense(4096, activation='relu'))
model.add(layers.Dense(4096, activation='relu'))
model.add(layers.Dense(1000, activation='softmax'))

Modern Architectures

ResNet

  • Year of Publication: 2015
  • Authors: Kaiming He et al.
  • Main Application: Image classification on the ImageNet dataset.

Structure

  • Convolutional Layers: 50-152
  • Pooling Layers: 1
  • Fully Connected Layers: 1
  • Residual Blocks: Introduction of skip connections.
from tensorflow.keras.applications import ResNet50

model = ResNet50(weights='imagenet')

Inception (GoogLeNet)

  • Year of Publication: 2014
  • Authors: Christian Szegedy et al.
  • Main Application: Image classification on the ImageNet dataset.

Structure

  • Convolutional Layers: Multiple
  • Pooling Layers: Multiple
  • Fully Connected Layers: 1
  • Inception Modules: Use of multiple filters of different sizes in the same layer.
from tensorflow.keras.applications import InceptionV3

model = InceptionV3(weights='imagenet')

Architecture Comparison

| Architecture | Year | Convolutional Layers | Fully Connected Layers | Key Features | |--------------|------|----------------------|------------------------|-----------------------| | LeNet-5 | 1998 | 2 | 3 | Simplicity | | AlexNet | 2012 | 5 | 3 | Use of ReLU and Dropout| | VGGNet | 2014 | 16-19 | 3 | Depth | | ResNet | 2015 | 50-152 | 1 | Skip connections | | Inception | 2014 | Multiple | 1 | Inception modules |

Conclusion

CNN architectures have evolved significantly from LeNet-5 to modern ResNet and Inception. Each of these architectures has introduced innovations that have improved the performance and generalization capability of computer vision models. Understanding these architectures is crucial for any professional who wishes to specialize in Deep Learning and computer vision.
