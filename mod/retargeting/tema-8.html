In this section of the course, we will learn how to set up retargeting campaigns on additional platforms like LinkedIn and Twitter. These platforms offer robust tools to reach your audience effectively and personally.

Retargeting Configuration on LinkedIn

LinkedIn is a powerful platform for retargeting, especially for B2B campaigns. Below are the steps to set up a retargeting campaign on LinkedIn.

Steps to Set Up Retargeting on LinkedIn

  1. Create a LinkedIn Campaign Manager Account:

    • If you don't have an account yet, sign up at LinkedIn Campaign Manager.

  2. Install the LinkedIn Insight Tag:

    • Navigate to the "Account Assets" section and select "Insight Tag".
    • Copy the Insight Tag code and paste it on all pages of your website, just before the </body> tag.
    <script type="text/javascript">

  window._linkedin_data_partner_ids.push(_linkedin_partner_id);
</script>

  3. Create a Retargeting Audience:

    • In Campaign Manager, go to "Account Assets" and select "Matched Audiences".
    • Create a new audience based on website visitors.

  4. Set Up the Retargeting Campaign:

    • Create a new campaign and select the campaign objective.
    • In the audience section, select the retargeting audience you previously created.
    • Define the budget, schedule, and creative ads.

Retargeting Configuration on Twitter

Twitter also offers retargeting capabilities that can be very effective in reaching your audience. Here are the steps to set up a retargeting campaign on Twitter.

Steps to Set Up Retargeting on Twitter

  1. Create a Twitter Ads Account:

    • If you don't have an account, sign up at Twitter Ads.

  2. Install the Twitter Pixel:

    • Go to the "Tools" section and select "Conversion tracking".
    • Create a new conversion tag and copy the pixel code.
    • Paste the pixel code on all pages of your website, just before the </body> tag.

  3. Create a Retargeting Audience:

    • In Twitter Ads, go to "Tools" and select "Audience Manager".
    • Create a new audience based on website visitors.

  4. Set Up the Retargeting Campaign:

    • Create a new campaign and select the campaign objective.
    • In the audience section, select the retargeting audience you previously created.
    • Define the budget, schedule, and creative ads.

Conclusion

Setting up retargeting campaigns on platforms like LinkedIn and Twitter can be an excellent way to reach your audience effectively. By following the steps detailed in this section, you will be able to install the necessary pixels, create retargeting audiences, and set up campaigns that maximize your return on investment. Be sure to monitor and adjust your campaigns regularly to achieve the best results.
