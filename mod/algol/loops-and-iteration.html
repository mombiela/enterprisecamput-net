Introduction

Loops and iteration are fundamental concepts in programming that allow you to execute a block of code multiple times. In ALGOL, loops are used to perform repetitive tasks efficiently. This section will cover the basics of loops, including the different types of loops available in ALGOL, and provide examples and exercises to help you understand and apply these concepts.

Types of Loops in ALGOL

For Loop

The for loop in ALGOL is used to iterate over a range of values. It is particularly useful when you know the number of iterations in advance.

Syntax

for variable := start_value step step_value until end_value do
    statement;

Example

begin
    integer i;
    for i := 1 step 1 until 10 do
        print(i);
end

In this example, the loop will print numbers from 1 to 10.

While Loop

The while loop continues to execute a block of code as long as a specified condition is true.

Syntax

while condition do
    statement;

Example

begin
    integer i;
    i := 1;
    while i <= 10 do
    begin
        print(i);
        i := i + 1;
    end
end

This loop will also print numbers from 1 to 10, similar to the for loop example.

Repeat-Until Loop

The repeat-until loop is similar to the while loop, but it checks the condition after executing the loop body. This means the loop will always execute at least once.

Syntax

repeat
    statement;
until condition;

Example

begin
    integer i;
    i := 1;
    repeat
        print(i);
        i := i + 1;
    until i > 10;
end

This loop will print numbers from 1 to 10, ensuring the loop body executes at least once.

Key Concepts

  • Initialization: Setting the initial value of the loop control variable.
  • Condition: The expression that determines whether the loop continues to execute.
  • Iteration: The process of executing the loop body repeatedly.
  • Termination: The point at which the loop stops executing.

Comparison of Loop Types

Loop Type Initialization Condition Check Execution Guarantee Use Case
For Loop Explicit Before each iteration No Known number of iterations
While Loop Explicit Before each iteration No Condition-based iteration
Repeat-Until Loop Explicit After each iteration Yes At least one execution required

Exercises

Exercise 1: For Loop

Write an ALGOL program that prints the first 15 even numbers using a for loop.

begin
    integer i;
    for i := 2 step 2 until 30 do
        print(i);
end

Exercise 2: While Loop

Write an ALGOL program that prints the numbers from 10 down to 1 using a while loop.

begin
    integer i;
    i := 10;
    while i >= 1 do
    begin
        print(i);
        i := i - 1;
    end
end

Exercise 3: Repeat-Until Loop

Write an ALGOL program that prints the numbers from 1 to 5 using a repeat-until loop.

begin
    integer i;
    i := 1;
    repeat
        print(i);
        i := i + 1;
    until i > 5;
end

Conclusion

Loops and iteration are essential tools in programming that allow you to perform repetitive tasks efficiently. In ALGOL, you have several types of loops at your disposal, including for, while, and repeat-until loops. Each type of loop has its own use cases and advantages. By understanding and practicing these loops, you can write more efficient and effective ALGOL programs.
