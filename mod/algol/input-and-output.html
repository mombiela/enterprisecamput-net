Input and Output in ALGOL

Introduction

In this section, we will explore how to handle input and output (I/O) operations in ALGOL. Understanding I/O is crucial for interacting with users and other systems, making your programs more dynamic and useful.

Basic Concepts

Input

  • Reading Data: In ALGOL, input operations are used to read data from the user or other input sources.
  • Syntax: The read statement is used to capture input.

Output

  • Displaying Data: Output operations are used to display data to the user or other output destinations.
  • Syntax: The print statement is used to display output.

Input Operations

Reading Single Values

To read a single value from the user, you can use the read statement.

begin
    integer x;
    read(x);
    print("You entered: ", x);
end
  • Explanation:
    • integer x; declares an integer variable x.
    • read(x); reads an integer value from the user and stores it in x.
    • print("You entered: ", x); displays the entered value.

Reading Multiple Values

You can read multiple values in a single read statement.

begin
    integer a, b;
    read(a, b);
    print("You entered: ", a, " and ", b);
end
  • Explanation:
    • integer a, b; declares two integer variables a and b.
    • read(a, b); reads two integer values from the user and stores them in a and b.
    • print("You entered: ", a, " and ", b); displays the entered values.

Output Operations

Printing Single Values

To print a single value, use the print statement.

begin
    integer y;
    y := 10;
    print("The value of y is: ", y);
end
  • Explanation:
    • integer y; declares an integer variable y.
    • y := 10; assigns the value 10 to y.
    • print("The value of y is: ", y); displays the value of y.

Printing Multiple Values

You can print multiple values in a single print statement.

begin
    integer m, n;
    m := 5;
    n := 15;
    print("Values are: ", m, " and ", n);
end
  • Explanation:
    • integer m, n; declares two integer variables m and n.
    • m := 5; assigns the value 5 to m.
    • n := 15; assigns the value 15 to n.
    • print("Values are: ", m, " and ", n); displays the values of m and n.

Advanced I/O Operations

Formatted Output

ALGOL allows formatted output using specific formatting options.

begin
    real r;
    r := 3.14159;
    print("The value of pi is approximately: ", r:8:4);
end
  • Explanation:
    • real r; declares a real (floating-point) variable r.
    • r := 3.14159; assigns the value 3.14159 to r.
    • print("The value of pi is approximately: ", r:8:4); displays the value of r with a total width of 8 characters and 4 decimal places.

File I/O

ALGOL supports reading from and writing to files, which is useful for handling larger datasets.

begin
    integer fileVar;
    open(fileVar, "input.txt", "read");
    while not eof(fileVar) do
    begin
        integer num;
        read(fileVar, num);
        print("Read number: ", num);
    end;
    close(fileVar);
end
  • Explanation:
    • integer fileVar; declares a file variable fileVar.
    • open(fileVar, "input.txt", "read"); opens the file input.txt for reading.
    • while not eof(fileVar) do loops until the end of the file is reached.
    • integer num; declares an integer variable num.
    • read(fileVar, num); reads an integer from the file and stores it in num.
    • print("Read number: ", num); displays the read number.
    • close(fileVar); closes the file.

Conclusion

In this section, we covered the basics and some advanced aspects of input and output operations in ALGOL. We learned how to read and print single and multiple values, format output, and handle file I/O. Mastering these operations will enable you to create more interactive and user-friendly programs.
