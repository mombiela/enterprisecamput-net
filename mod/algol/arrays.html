Introduction to Arrays

Arrays are a fundamental data structure in programming that allow you to store multiple values in a single variable. In ALGOL, arrays are used to store collections of data of the same type, which can be accessed using indices.

Key Concepts

  • Array Declaration: How to define an array in ALGOL.
  • Array Initialization: How to assign values to an array.
  • Array Access: How to retrieve and modify values in an array.
  • Multidimensional Arrays: Arrays with more than one dimension.
  • Array Operations: Common operations performed on arrays.

Array Declaration

In ALGOL, arrays are declared by specifying the type of the elements and the range of indices.

Example

integer array A[1:10];

This declares an array A of integers with indices ranging from 1 to 10.

Explanation

  • integer: Specifies that the array will store integer values.
  • array A[1:10]: Declares an array named A with indices from 1 to 10.

Array Initialization

Arrays can be initialized at the time of declaration or later in the code.

Example

integer array B[1:5] := (1, 2, 3, 4, 5);

This initializes the array B with values 1, 2, 3, 4, and 5.

Explanation

  • := (1, 2, 3, 4, 5): Assigns the values 1 through 5 to the array B.

Array Access

You can access and modify the elements of an array using their indices.

Example

integer array C[1:3];
C[1] := 10;
C[2] := 20;
C[3] := 30;
print(C[2]);  ! Outputs 20

Explanation

  • C[1] := 10;: Assigns the value 10 to the first element of the array C.
  • print(C[2]);: Prints the value of the second element of the array C, which is 20.

Multidimensional Arrays

ALGOL supports multidimensional arrays, which are arrays of arrays.

Example

integer array D[1:3, 1:3];
D[1, 1] := 1;
D[1, 2] := 2;
D[1, 3] := 3;
D[2, 1] := 4;
D[2, 2] := 5;
D[2, 3] := 6;
D[3, 1] := 7;
D[3, 2] := 8;
D[3, 3] := 9;
print(D[2, 3]);  ! Outputs 6

Explanation

  • integer array D[1:3, 1:3];: Declares a 3x3 array D of integers.
  • D[2, 3] := 6;: Assigns the value 6 to the element at the second row and third column of the array D.
  • print(D[2, 3]);: Prints the value of the element at the second row and third column, which is 6.

Array Operations

Common operations on arrays include iterating through elements, searching for values, and performing calculations.

Example: Iterating through an Array

integer array E[1:5] := (10, 20, 30, 40, 50);
integer i;

for i := 1 step 1 until 5 do
    print(E[i]);

Explanation

  • for i := 1 step 1 until 5 do: A loop that iterates from 1 to 5.
  • print(E[i]);: Prints each element of the array E.

Summary

Arrays in ALGOL are a powerful tool for managing collections of data. Understanding how to declare, initialize, access, and manipulate arrays is essential for effective programming in ALGOL. This topic covered:

  • Basic array declaration and initialization.
  • Accessing and modifying array elements.
  • Working with multidimensional arrays.
  • Performing common operations on arrays.

By mastering these concepts, you will be well-equipped to handle more complex data structures and algorithms in ALGOL.
