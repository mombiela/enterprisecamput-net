Introduction to ALGOL

ALGOL (Algorithmic Language) is a family of imperative computer programming languages originally developed in the mid-20th century. It was designed to be a universal language for expressing algorithms and has influenced many modern programming languages.

Getting Started with ALGOL

Before diving into algorithm implementation, it's essential to understand the basics of ALGOL syntax and structure.

Basic Syntax

  • Program Structure: An ALGOL program typically consists of a sequence of declarations followed by a sequence of statements.
  • Variables: Variables must be declared before they are used.
  • Control Structures: ALGOL supports standard control structures like loops and conditionals.

Example:

begin
  integer i;
  for i := 1 step 1 until 10 do
    print(i);
end

Explanation:

  • begin and end mark the start and end of the program.
  • integer i; declares a variable i of type integer.
  • The for loop iterates from 1 to 10, printing each value of i.

Basic Algorithms

Sequential Search

A simple algorithm to search for an element in an array.

Example:

begin
  integer array[1:10];
  integer i, n, key, position;
  position := 0;
  key := 5;  // Element to search for
  for i := 1 step 1 until 10 do
    if array[i] = key then
      position := i;
  if position = 0 then
    print("Element not found")
  else
    print("Element found at position: ", position);
end

Explanation:

  • An array of integers is declared.
  • The for loop iterates through the array to find the key.
  • If the key is found, its position is stored; otherwise, a message is printed.

Bubble Sort

A basic sorting algorithm that repeatedly steps through the list, compares adjacent elements, and swaps them if they are in the wrong order.

Example:

begin
  integer array[1:5];
  integer i, j, temp;
  for i := 1 step 1 until 4 do
    for j := 1 step 1 until 5 - i do
      if array[j] > array[j + 1] then
        temp := array[j];
        array[j] := array[j + 1];
        array[j + 1] := temp;
end

Explanation:

  • The outer loop runs from the first to the second-to-last element.
  • The inner loop compares adjacent elements and swaps them if necessary.
  • This process is repeated until the array is sorted.

Intermediate Algorithms

Binary Search

A more efficient search algorithm for sorted arrays.

Example:

begin
  integer array[1:10];
  integer low, high, mid, key;
  low := 1;
  high := 10;
  key := 5;  // Element to search for
  while low <= high do
    mid := (low + high) div 2;
    if array[mid] = key then
      print("Element found at position: ", mid);
      exit;
    else if array[mid] < key then
      low := mid + 1;
    else
      high := mid - 1;
  print("Element not found");
end

Explanation:

  • low and high mark the search range.
  • mid is the middle index of the current range.
  • The algorithm narrows the search range based on the comparison of key and array[mid].

Merge Sort

A divide-and-conquer algorithm that divides the array into halves, sorts them, and then merges them.

Example:

procedure mergeSort(array, left, right);
begin
  integer mid;
  if left < right then
    mid := (left + right) div 2;
    mergeSort(array, left, mid);
    mergeSort(array, mid + 1, right);
    merge(array, left, mid, right);
end

procedure merge(array, left, mid, right);
begin
  integer i, j, k;
  integer n1, n2;
  n1 := mid - left + 1;
  n2 := right - mid;
  integer L[1:n1], R[1:n2];
  for i := 1 step 1 until n1 do
    L[i] := array[left + i - 1];
  for j := 1 step 1 until n2 do
    R[j] := array[mid + j];
  i := 1; j := 1; k := left;
  while i <= n1 and j <= n2 do
    if L[i] <= R[j] then
      array[k] := L[i];
      i := i + 1;
    else
      array[k] := R[j];
      j := j + 1;
    k := k + 1;
  while i <= n1 do
    array[k] := L[i];
    i := i + 1;
    k := k + 1;
  while j <= n2 do
    array[k] := R[j];
    j := j + 1;
    k := k + 1;
end

begin
  integer array[1:10];
  mergeSort(array, 1, 10);
end

Explanation:

  • mergeSort recursively divides the array into halves.
  • merge combines two sorted halves into a single sorted array.
  • Temporary arrays L and R store the left and right halves.

Advanced Algorithms

Dynamic Programming: Fibonacci Sequence

An optimized approach to compute the Fibonacci sequence using memoization.

Example:

integer array[1:100];

procedure fibonacci(n);
begin
  if n <= 1 then
    return n;
  if array[n] != 0 then
    return array[n];
  array[n] := fibonacci(n - 1) + fibonacci(n - 2);
  return array[n];
end

begin
  integer n;
  n := 10;  // Compute the 10th Fibonacci number
  print(fibonacci(n));
end

Explanation:

  • The fibonacci procedure uses memoization to store previously computed values.
  • This reduces the time complexity from exponential to linear.

Graph Algorithms: Dijkstra's Algorithm

An algorithm to find the shortest path in a graph.

Example:

procedure dijkstra(graph, source, n);
begin
  integer dist[1:n], visited[1:n], i, j, minDist, nextNode;
  for i := 1 step 1 until n do
    dist[i] := infinity;
    visited[i] := 0;
  dist[source] := 0;
  for i := 1 step 1 until n - 1 do
    minDist := infinity;
    for j := 1 step 1 until n do
      if dist[j] < minDist and visited[j] = 0 then
        minDist := dist[j];
        nextNode := j;
    visited[nextNode] := 1;
    for j := 1 step 1 until n do
      if visited[j] = 0 and dist[nextNode] + graph[nextNode][j] < dist[j] then
        dist[j] := dist[nextNode] + graph[nextNode][j];
  for i := 1 step 1 until n do
    print("Distance from source to node ", i, ": ", dist[i]);
end

begin
  integer graph[1:5][1:5];
  dijkstra(graph, 1, 5);
end

Explanation:

  • dist array stores the shortest distance from the source to each node.
  • visited array keeps track of visited nodes.
  • The algorithm iteratively selects the unvisited node with the smallest distance and updates the distances of its neighbors.

Conclusion

We explored searching and sorting algorithms, dynamic programming, and graph algorithms. Understanding these algorithms and their implementation in ALGOL will provide a solid foundation for tackling more complex problems and optimizing solutions in your programming journey.
