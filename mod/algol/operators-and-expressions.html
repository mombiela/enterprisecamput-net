Introduction

In this section, we will explore the fundamental concepts of operators and expressions in ALGOL. Understanding these concepts is crucial for writing efficient and effective ALGOL programs. We will cover various types of operators, their precedence, and how to construct expressions.

Arithmetic Operators

Arithmetic operators are used to perform mathematical calculations. ALGOL supports the following arithmetic operators:

  • + : Addition
  • - : Subtraction
  • * : Multiplication
  • / : Division
  • := : Assignment

Examples

begin
  integer a, b, c;
  a := 10;
  b := 5;
  c := a + b;  ! c is now 15
  c := a - b;  ! c is now 5
  c := a * b;  ! c is now 50
  c := a / b;  ! c is now 2
end

Relational Operators

Relational operators are used to compare two values. The result of a relational operation is a Boolean value (true or false). ALGOL supports the following relational operators:

  • = : Equal to
  • ≠ : Not equal to
  • < : Less than
  • <= : Less than or equal to
  • > : Greater than
  • >= : Greater than or equal to

Examples

begin
  integer x, y;
  x := 10;
  y := 5;
  if x = y then
    print("x is equal to y")
  else
    print("x is not equal to y");  ! This will print "x is not equal to y"
end

Logical Operators

Logical operators are used to combine multiple Boolean expressions. ALGOL supports the following logical operators:

  • and : Logical AND
  • or : Logical OR
  • not : Logical NOT

Examples

begin
  boolean a, b, c;
  a := true;
  b := false;
  c := a and b;  ! c is now false
  c := a or b;   ! c is now true
  c := not a;    ! c is now false
end

Operator Precedence

Operator precedence determines the order in which operators are evaluated in an expression. The following table summarizes the precedence of operators in ALGOL, from highest to lowest:

| Precedence Level | Operators | |------------------|--------------------------| | 1 | not | | 2 | *, / | | 3 | +, - | | 4 | <, <=, >, >= | | 5 | =, ≠ | | 6 | and | | 7 | or | | 8 | := (assignment) |

Examples

begin
  integer a, b, c, d;
  a := 10;
  b := 5;
  c := 2;
  d := a + b * c;  ! d is now 20 because * has higher precedence than +
  d := (a + b) * c;  ! d is now 30 because parentheses change the precedence
end

Compound Expressions

Compound expressions involve multiple operators and operands. Parentheses can be used to explicitly define the order of evaluation.

Examples

begin
  integer x, y, z;
  x := 10;
  y := 5;
  z := (x + y) * (x - y);  ! z is now 75
end

Conclusion

Understanding operators and expressions is fundamental to programming in ALGOL. By mastering arithmetic, relational, and logical operators, as well as their precedence, you can write more complex and efficient programs. Practice constructing and evaluating expressions to become proficient in using these operators effectively.
