Introduction

Conditional statements are fundamental in programming as they allow the execution of code based on certain conditions. In ALGOL, conditional statements help control the flow of the program, making it possible to execute different code blocks based on specific criteria.

Key Concepts

  • Conditional Statements: Instructions that execute code only if a specified condition is true.
  • Boolean Expressions: Expressions that evaluate to either true or false.
  • If Statements: The primary conditional statement in ALGOL.
  • Else Statements: Used to execute code when the if condition is false.
  • Else If Statements: Used to check multiple conditions in sequence.

If Statements

The if statement is used to execute a block of code if a specified condition is true.

Syntax

if condition then
    statement;

Example

begin
    integer x;
    x := 10;
    if x > 5 then
        print("x is greater than 5");
end

In this example, the program checks if x is greater than 5. Since x is 10, the condition is true, and the message "x is greater than 5" is printed.

If-Else Statements

The if-else statement allows you to execute one block of code if the condition is true and another block if the condition is false.

Syntax

if condition then
    statement1
else
    statement2;

Example

begin
    integer x;
    x := 3;
    if x > 5 then
        print("x is greater than 5")
    else
        print("x is not greater than 5");
end

In this example, since x is 3, the condition x > 5 is false, so the message "x is not greater than 5" is printed.

Else If Statements

The else if statement is used to check multiple conditions in sequence.

Syntax

if condition1 then
    statement1
else if condition2 then
    statement2
else
    statement3;

Example

begin
    integer x;
    x := 7;
    if x > 10 then
        print("x is greater than 10")
    else if x > 5 then
        print("x is greater than 5 but less than or equal to 10")
    else
        print("x is 5 or less");
end

In this example, since x is 7, the first condition x > 10 is false, but the second condition x > 5 is true, so the message "x is greater than 5 but less than or equal to 10" is printed.

Nested Conditional Statements

Conditional statements can be nested within each other to handle more complex logic.

Example

begin
    integer x, y;
    x := 8;
    y := 3;
    if x > 5 then
        if y < 5 then
            print("x is greater than 5 and y is less than 5")
        else
            print("x is greater than 5 and y is 5 or more")
    else
        print("x is 5 or less");
end

In this example, the program first checks if x is greater than 5. Since x is 8, it then checks if y is less than 5. Both conditions are true, so the message "x is greater than 5 and y is less than 5" is printed.

Conclusion

Conditional statements in ALGOL are essential for controlling the flow of a program based on specific conditions. Understanding how to use if, else, and else if statements, as well as how to nest them, allows you to write more complex and dynamic programs. Practice these concepts with various conditions to become proficient in using conditional statements in ALGOL.
