Introduction

In this section of the course, we will focus on practical exercises for network configuration. These exercises are designed to provide a deep understanding of how to configure and manage networks in a professional environment. The exercises will cover basic to advanced configurations, using both graphical interfaces and command lines.

Learning Objectives

  • Understand the basic concepts of network configuration.
  • Configure network devices such as routers and switches.
  • Implement and manage subnets.
  • Configure essential network services like DHCP and DNS.
  • Perform advanced network security configurations.

Exercise 1: Basic Router Configuration

Description

In this exercise, we will configure a basic router to allow communication between two different networks.

Steps

  1. Connect the hardware:

    • Connect the router to two different networks using Ethernet cables.
    • Ensure that the router interfaces are properly connected.

  2. Access the router:

    • Use a console or a web interface to access the router.
    • Log in with the provided credentials.

  3. Configure the interfaces:

    • Assign IP addresses to the router interfaces.
    Router> enable
Router# configure terminal
Router(config)# interface GigabitEthernet0/0
Router(config-if)# ip address 192.168.1.1 255.255.255.0
Router(config-if)# no shutdown
Router(config-if)# exit
Router(config)# interface GigabitEthernet0/1
Router(config-if)# ip address 192.168.2.1 255.255.255.0
Router(config-if)# no shutdown
Router(config-if)# exit

  4. Save the configuration:

    • Ensure that the configuration is saved correctly.
    Router# copy running-config startup-config

Verification

  • Verify connectivity between the two networks using ping commands. 
    Router# ping 192.168.2.1

Exercise 2: Switch Configuration

Description

In this exercise, we will configure a switch to segment a network into multiple VLANs.

Steps

  1. Access the switch:

    • Use a console or a web interface to access the switch.
    • Log in with the provided credentials.

  2. Create VLANs:

    • Create two VLANs on the switch.
    Switch> enable
Switch# configure terminal
Switch(config)# vlan 10
Switch(config-vlan)# name Sales
Switch(config-vlan)# exit
Switch(config)# vlan 20
Switch(config-vlan)# name Engineering
Switch(config-vlan)# exit

  3. Assign ports to VLANs:

    • Assign specific ports to each VLAN.
    Switch(config)# interface range GigabitEthernet0/1 - 2
Switch(config-if-range)# switchport mode access
Switch(config-if-range)# switchport access vlan 10
Switch(config-if-range)# exit
Switch(config)# interface range GigabitEthernet0/3 - 4
Switch(config-if-range)# switchport mode access
Switch(config-if-range)# switchport access vlan 20
Switch(config-if-range)# exit

  4. Save the configuration:

    • Ensure that the configuration is saved correctly.
    Switch# copy running-config startup-config

Verification

  • Verify the VLAN configuration. 
    Switch# show vlan brief

Exercise 3: DHCP Configuration

Description

In this exercise, we will configure a DHCP server on a router to automatically assign IP addresses to devices on the network.

Steps

  1. Access the router:

    • Use a console or a web interface to access the router.
    • Log in with the provided credentials.

  2. Configure the DHCP pool:

    • Configure a pool of IP addresses for the network.
    Router> enable
Router# configure terminal
Router(config)# ip dhcp pool LAN
Router(dhcp-config)# network 192.168.1.0 255.255.255.0
Router(dhcp-config)# default-router 192.168.1.1
Router(dhcp-config)# dns-server 8.8.8.8
Router(dhcp-config)# exit

  3. Exclude IP addresses:

    • Exclude specific IP addresses from the DHCP pool.
    Router(config)# ip dhcp excluded-address 192.168.1.1 192.168.1.10

  4. Save the configuration:

    • Ensure that the configuration is saved correctly.
    Router# copy running-config startup-config

Verification

  • Verify that devices on the network receive IP addresses automatically. 
    Router# show ip dhcp binding

Conclusion

These exercises provide a solid foundation for network configuration. Through practice, students will learn to configure routers, switches, and essential network services like DHCP. The understanding and ability to perform these configurations are fundamental for any IT professional working with networks.
