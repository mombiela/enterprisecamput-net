Introduction

Welcome to your journey into HTML! In this section, we will guide you through the process of creating your very first HTML page. By the end of this lesson, you'll have a basic understanding of HTML structure and be able to create a simple web page.

Key Concepts

  • HTML (HyperText Markup Language): The standard language for creating web pages.
  • Tags: HTML uses tags to structure content. Tags are enclosed in angle brackets, e.g., <tagname>.
  • Elements: An element consists of a start tag, content, and an end tag, e.g., <p>This is a paragraph.</p>.
  • Attributes: Tags can have attributes that provide additional information about elements, e.g., <a href="https://example.com">Link</a>.

Basic Structure of an HTML Document

Every HTML document has a basic structure that includes the following elements:

<!DOCTYPE html>
<html>
<head>
    <title>My First HTML Page</title>
</head>
<body>
    <h1>Welcome to My First HTML Page</h1>
    <p>This is a paragraph of text on my first HTML page.</p>
</body>
</html>

Explanation of the Code

  • <!DOCTYPE html>: This declaration defines the document type and version of HTML. It helps the browser understand how to render the page.
  • <html>: The root element that contains all other HTML elements.
  • <head>: Contains meta-information about the document, such as the title and links to stylesheets.
  • <title>: Sets the title of the web page, which appears in the browser tab.
  • <body>: Contains the content of the HTML document, such as text, images, and links.
  • <h1>: A heading tag. There are six levels of headings, from <h1> (most important) to <h6> (least important).
  • <p>: A paragraph tag used to define a block of text.

Creating Your First HTML Page

Follow these steps to create your first HTML page:

  1. Open a Text Editor:

    • You can use any text editor, such as Notepad (Windows), TextEdit (Mac), or a code editor like Visual Studio Code or Sublime Text.

  2. Write the HTML Code:

    • Copy the basic structure provided above into your text editor.

  3. Save the File:

    • Save the file with a .html extension, e.g., index.html.

  4. Open the HTML File in a Browser:

    • Double-click the saved HTML file or open it in a web browser (Chrome, Firefox, Safari, etc.).

Example: Adding More Content

Let's add more content to our HTML page to make it more interesting.

<!DOCTYPE html>
<html>
<head>
    <title>My First HTML Page</title>
</head>
<body>
    <h1>Welcome to My First HTML Page</h1>
    <p>This is a paragraph of text on my first HTML page.</p>
    <h2>About Me</h2>
    <p>My name is John Doe, and I am learning HTML.</p>
    <ul>
        <li>I love coding.</li>
        <li>I enjoy learning new technologies.</li>
        <li>I am excited to build my first website.</li>
    </ul>
    <a href="https://example.com">Visit Example.com</a>
</body>
</html>

Explanation of the Additional Code

  • <h2>: A second-level heading.
  • <ul>: An unordered list.
  • <li>: A list item within an unordered list.
  • <a href="URL">: An anchor tag used to create hyperlinks. The href attribute specifies the URL of the page the link goes to.

Conclusion

Congratulations! You've created your first HTML page. You now understand the basic structure of an HTML document and how to add headings, paragraphs, lists, and links. Practice creating more HTML pages to become more comfortable with these concepts. In the next lessons, we will dive deeper into HTML elements and attributes to enhance your web pages further. Happy coding!
