Adding content is one of the fundamental aspects of creating web pages with HTML. This topic will guide you through the basics of adding text, images, links, and other multimedia elements to your web pages, progressing from beginner to advanced levels.

Text Content

Text is the most basic form of content you can add to a web page. HTML provides various tags to structure and format text.

Paragraphs and Headings

  • Paragraphs: Use the <p> tag to define a paragraph.
  • Headings: Use <h1> to <h6> tags to define headings, with <h1> being the highest level and <h6> the lowest.
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Text Content Example</title>
</head>
<body>
    <h1>This is an H1 Heading</h1>
    <p>This is a paragraph of text.</p>
    <h2>This is an H2 Heading</h2>
    <p>Another paragraph of text.</p>
</body>
</html>

Lists

  • Ordered Lists: Use the <ol> tag for ordered (numbered) lists.
  • Unordered Lists: Use the <ul> tag for unordered (bulleted) lists.
  • List Items: Use the <li> tag to define list items.
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Lists Example</title>
</head>
<body>
    <h2>Ordered List</h2>
    <ol>
        <li>First item</li>
        <li>Second item</li>
        <li>Third item</li>
    </ol>
    <h2>Unordered List</h2>
    <ul>
        <li>First item</li>
        <li>Second item</li>
        <li>Third item</li>
    </ul>
</body>
</html>

Images

Images can be added to your web pages using the <img> tag. The src attribute specifies the path to the image, and the alt attribute provides alternative text for screen readers.

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Images Example</title>
</head>
<body>
    <h2>Adding an Image</h2>
    <img src="path/to/image.jpg" alt="Description of the image">
</body>
</html>

Links

Links are created using the <a> tag. The href attribute specifies the URL of the page the link goes to.

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Links Example</title>
</head>
<body>
    <h2>Adding a Link</h2>
    <a href="https://www.example.com">Visit Example.com</a>
</body>
</html>

Multimedia Content

Audio

You can add audio to your web pages using the <audio> tag. The controls attribute adds play, pause, and volume controls.

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Audio Example</title>
</head>
<body>
    <h2>Adding Audio</h2>
    <audio controls>
        <source src="path/to/audio.mp3" type="audio/mpeg">
        Your browser does not support the audio element.
    </audio>
</body>
</html>

Video

The <video> tag is used to embed video content. Similar to the <audio> tag, it can include controls for user interaction.

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Video Example</title>
</head>
<body>
    <h2>Adding Video</h2>
    <video controls>
        <source src="path/to/video.mp4" type="video/mp4">
        Your browser does not support the video tag.
    </video>
</body>
</html>

Tables

Tables are used to display data in a tabular format. Use the <table> tag to create a table, <tr> for table rows, <th> for table headers, and <td> for table data cells.

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Tables Example</title>
</head>
<body>
    <h2>Adding a Table</h2>
    <table border="1">
        <tr>
            <th>Header 1</th>
            <th>Header 2</th>
        </tr>
        <tr>
            <td>Data 1</td>
            <td>Data 2</td>
        </tr>
        <tr>
            <td>Data 3</td>
            <td>Data 4</td>
        </tr>
    </table>
</body>
</html>

Conclusion

In this topic, you have learned how to add various types of content to your web pages using HTML. From basic text and lists to images, links, multimedia, and tables, these elements form the building blocks of web content. Practice using these tags to become comfortable with adding and structuring content on your web pages.
