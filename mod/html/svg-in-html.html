Introduction to SVG

Scalable Vector Graphics (SVG) is an XML-based format for creating vector images. Unlike raster images (e.g., JPEG, PNG), SVG images can be scaled to any size without losing quality, making them ideal for web graphics.

Key Concepts

  • Vector Graphics: Images defined by points, lines, curves, and shapes, which can be scaled without loss of quality.
  • XML-based: SVG files are written in XML, making them both human-readable and machine-readable.
  • Interactive and Animatable: SVG elements can be styled, animated, and made interactive using CSS and JavaScript.

Basic SVG Structure

SVG elements can be directly embedded in HTML documents using the <svg> tag.

Example: Simple SVG Circle

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>SVG Example</title>
</head>
<body>
    <svg width="100" height="100">
        <circle cx="50" cy="50" r="40" stroke="black" stroke-width="3" fill="red" />
    </svg>
</body>
</html>

Explanation

  • <svg width="100" height="100">: Defines the SVG container with a width and height of 100 units.
  • <circle cx="50" cy="50" r="40" stroke="black" stroke-width="3" fill="red" />: Draws a circle with:
    • cx and cy: Center coordinates (50, 50).
    • r: Radius (40).
    • stroke: Border color (black).
    • stroke-width: Border thickness (3 units).
    • fill: Fill color (red).

Common SVG Elements

  • <rect>: Draws rectangles.
  • <circle>: Draws circles.
  • <ellipse>: Draws ellipses.
  • <line>: Draws straight lines.
  • <polyline>: Draws a series of connected lines.
  • <polygon>: Draws a closed shape with multiple sides.
  • <path>: Draws complex shapes using a series of commands.

Example: Various SVG Elements

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>SVG Elements</title>
</head>
<body>
    <svg width="200" height="200">
        <rect x="10" y="10" width="50" height="50" stroke="black" stroke-width="2" fill="blue" />
        <circle cx="100" cy="50" r="30" stroke="black" stroke-width="2" fill="green" />
        <ellipse cx="150" cy="150" rx="30" ry="20" stroke="black" stroke-width="2" fill="yellow" />
        <line x1="10" y1="150" x2="50" y2="190" stroke="black" stroke-width="2" />
        <polyline points="60,150 80,170 100,150" stroke="black" stroke-width="2" fill="none" />
        <polygon points="110,150 130,170 150,150" stroke="black" stroke-width="2" fill="purple" />
        <path d="M 160 150 Q 180 170 200 150" stroke="black" stroke-width="2" fill="none" />
    </svg>
</body>
</html>

Styling SVG with CSS

SVG elements can be styled using CSS, just like HTML elements.

Example: Styling SVG with CSS

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Styled SVG</title>
    <style>
        .my-circle {
            fill: orange;
            stroke: blue;
            stroke-width: 4;
        }
    </style>
</head>
<body>
    <svg width="100" height="100">
        <circle cx="50" cy="50" r="40" class="my-circle" />
    </svg>
</body>
</html>

Explanation

  • The .my-circle class is defined in the <style> section.
  • The class="my-circle" attribute is added to the <circle> element to apply the styles.

Animating SVG

SVG elements can be animated using the <animate> tag or CSS animations.

Example: Simple SVG Animation

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>SVG Animation</title>
</head>
<body>
    <svg width="100" height="100">
        <circle cx="50" cy="50" r="40" stroke="black" stroke-width="3" fill="red">
            <animate attributeName="cx" from="50" to="150" dur="2s" repeatCount="indefinite" />
        </circle>
    </svg>
</body>
</html>

Explanation

  • <animate attributeName="cx" from="50" to="150" dur="2s" repeatCount="indefinite" />: Animates the cx attribute of the circle from 50 to 150 over 2 seconds, repeating indefinitely.

Conclusion

SVG is a powerful tool for creating scalable, interactive, and animated graphics in HTML. By understanding the basic structure, common elements, styling, and animation techniques, you can enhance your web projects with high-quality vector graphics.
