Introduction

Before diving into programming with R, it's essential to have the right tools installed. This section will guide you through installing R and RStudio, which are the primary software components you'll need.

Installing R

R is a free software environment for statistical computing and graphics. Follow these steps to install R on your system:

  1. Visit the CRAN Website:

    • Go to the Comprehensive R Archive Network (CRAN) at https://cran.r-project.org/.

  2. Choose Your Operating System:

    • Select your operating system (Windows, macOS, or Linux).

  3. Download R:

    • Click on the appropriate link to download the R installer for your operating system.

  4. Install R:

    • Windows:
      • Run the downloaded .exe file.
      • Follow the installation instructions, accepting the default settings.
    • macOS:
      • Open the downloaded .pkg file.
      • Follow the installation instructions.
    • Linux:
      • Open a terminal.
      • Use the package manager for your distribution (e.g., apt for Debian-based systems, yum for Red Hat-based systems) to install R. For example: 
        sudo apt-get update
sudo apt-get install r-base

Installing RStudio

RStudio is an integrated development environment (IDE) for R. It makes R programming easier by providing a user-friendly interface.

  1. Visit the RStudio Website:

    • Go to https://rstudio.com/products/rstudio/download/.

  2. Download RStudio:

    • Choose the free version of RStudio Desktop.
    • Select the installer for your operating system and download it.

  3. Install RStudio:

    • Windows:
      • Run the downloaded .exe file.
      • Follow the installation instructions.
    • macOS:
      • Open the downloaded .dmg file.
      • Drag and drop the RStudio icon into the Applications folder.
    • Linux:
      • Open a terminal.
      • Use the package manager for your distribution to install RStudio. For example, on Debian-based systems: 
        sudo apt-get install gdebi-core
sudo gdebi rstudio-x.yy.zzz-amd64.deb

Verifying the Installation

After installing both R and RStudio, it's important to verify that everything is set up correctly.

  1. Open RStudio:

    • Launch RStudio from your applications menu or desktop shortcut.

  2. Check R Version:

    • In the RStudio console, type the following command and press Enter: 
      version
    • You should see output similar to this: 
      platform       x86_64-w64-mingw32
arch           x86_64
os             mingw32
system         x86_64, mingw32
status
major          4
minor          1.2
year           2021
month          11
day            01
svn rev        80317
language       R
version.string R version 4.1.2 (2021-11-01)
nickname       Bird Hippie

  3. Run a Simple R Command:

    • Try running a simple command to ensure R is working correctly: 
      print("Hello, R!")
    • You should see the output: 
      [1] "Hello, R!"

Conclusion

By following these steps, you should have successfully installed R and RStudio on your computer. These tools will provide a robust environment for your R programming journey, from beginner to advanced levels. If you encounter any issues during installation, refer to the official documentation or seek help from the R community.
