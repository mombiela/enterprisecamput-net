Introduction to plotly

Plotly is a powerful library for creating interactive visualizations in R. It allows users to create a wide range of charts and graphs that can be embedded in web applications, reports, and dashboards.

  • Key Concepts:
    • Interactive vs. Static Visualizations
    • Overview of plotly library
    • Installation and setup
# Install and load plotly
install.packages("plotly")
library(plotly)

Basic Plotly Charts

Learn how to create basic charts using plotly, such as scatter plots, line charts, and bar charts.

  • Scatter Plot:
# Basic Scatter Plot
data <- data.frame(x = rnorm(50), y = rnorm(50))
plot_ly(data, x = ~x, y = ~y, type = 'scatter', mode = 'markers')
  • Line Chart:
# Basic Line Chart
data <- data.frame(x = 1:10, y = cumsum(rnorm(10)))
plot_ly(data, x = ~x, y = ~y, type = 'scatter', mode = 'lines')
  • Bar Chart:
# Basic Bar Chart
data <- data.frame(x = c('A', 'B', 'C'), y = c(10, 20, 30))
plot_ly(data, x = ~x, y = ~y, type = 'bar')

Customizing Visualizations

Learn how to customize your visualizations by modifying colors, adding titles, and adjusting axes.

  • Customizing Colors:
# Customizing Colors in Scatter Plot
plot_ly(data, x = ~x, y = ~y, type = 'scatter', mode = 'markers', marker = list(color = 'red'))
  • Adding Titles and Labels:
# Adding Titles and Labels
plot_ly(data, x = ~x, y = ~y, type = 'scatter', mode = 'markers') %>%
  layout(title = 'Scatter Plot Example', xaxis = list(title = 'X Axis'), yaxis = list(title = 'Y Axis'))
  • Adjusting Axes:
# Adjusting Axes
plot_ly(data, x = ~x, y = ~y, type = 'scatter', mode = 'markers') %>%
  layout(xaxis = list(range = c(-3, 3)), yaxis = list(range = c(-3, 3)))

Advanced Plotly Features

Explore advanced features such as subplots, 3D plots, and animations.

  • Subplots:
# Creating Subplots
subplot(
  plot_ly(data, x = ~x, y = ~y, type = 'scatter', mode = 'markers'),
  plot_ly(data, x = ~x, y = ~y, type = 'scatter', mode = 'lines')
)
  • 3D Plots:
# 3D Scatter Plot
data <- data.frame(x = rnorm(50), y = rnorm(50), z = rnorm(50))
plot_ly(data, x = ~x, y = ~y, z = ~z, type = 'scatter3d', mode = 'markers')
  • Animations:
# Animated Scatter Plot
data <- data.frame(x = rnorm(50), y = rnorm(50), frame = rep(1:5, each = 10))
plot_ly(data, x = ~x, y = ~y, type = 'scatter', mode = 'markers', frame = ~frame)

Integrating Plotly with Shiny

Learn how to integrate plotly visualizations into Shiny applications for interactive web apps.

  • Basic Shiny App with Plotly:
# Shiny App with Plotly
library(shiny)
library(plotly)

ui <- fluidPage(
  plotlyOutput("plot")
)

server <- function(input, output) {
  output$plot <- renderPlotly({
    plot_ly(data, x = ~x, y = ~y, type = 'scatter', mode = 'markers')
  })
}

shinyApp(ui, server)

Conclusion

Plotly is a versatile and powerful tool for creating interactive visualizations in R. By mastering the basics and exploring advanced features, you can create compelling and dynamic visualizations for a variety of applications.

  • Summary:
    • Plotly offers a wide range of chart types and customization options.
    • Advanced features like subplots, 3D plots, and animations can enhance your visualizations.
    • Integration with Shiny allows for the creation of interactive web applications.

By following this course, you will gain the skills needed to create and customize interactive visualizations using plotly in R, from basic charts to advanced interactive applications.
