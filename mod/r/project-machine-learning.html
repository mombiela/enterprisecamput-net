Introduction to Machine Learning in R

Machine Learning (ML) is a field of artificial intelligence that uses statistical techniques to give computer systems the ability to "learn" from data. In this project, we will explore how to implement machine learning algorithms using R, a powerful language for statistical computing and graphics.

Key Concepts

  • Supervised Learning: Learning from labeled data (e.g., classification, regression).
  • Unsupervised Learning: Learning from unlabeled data (e.g., clustering, association).
  • Model Evaluation: Techniques to assess the performance of ML models (e.g., cross-validation, confusion matrix).

Setting Up the Environment

Before we dive into machine learning, we need to set up our R environment.

Installing Required Packages

install.packages("caret")
install.packages("randomForest")
install.packages("e1071")
install.packages("ggplot2")

Loading the Packages

library(caret)
library(randomForest)
library(e1071)
library(ggplot2)

Data Preprocessing

Data preprocessing is a crucial step in machine learning. It involves cleaning and transforming raw data into a format suitable for modeling.

Loading and Exploring the Dataset

# Load the dataset
data(iris)
# View the first few rows
head(iris)

Handling Missing Values

# Check for missing values
sum(is.na(iris))

# Impute missing values (if any)
iris[is.na(iris)] <- median(iris, na.rm = TRUE)

Feature Scaling

# Normalize the features
normalize <- function(x) {
  return ((x - min(x)) / (max(x) - min(x)))
}

iris_norm <- as.data.frame(lapply(iris[1:4], normalize))
iris_norm$Species <- iris$Species

Supervised Learning

Classification with Random Forest

Random Forest is an ensemble learning method for classification and regression.

Training the Model

# Split the data into training and testing sets
set.seed(123)
trainIndex <- createDataPartition(iris_norm$Species, p = .8, 
                                  list = FALSE, 
                                  times = 1)
irisTrain <- iris_norm[ trainIndex,]
irisTest  <- iris_norm[-trainIndex,]

# Train the Random Forest model
model_rf <- randomForest(Species ~ ., data = irisTrain)
print(model_rf)

Making Predictions

# Predict on the test set
predictions <- predict(model_rf, irisTest)

# Evaluate the model
confusionMatrix(predictions, irisTest$Species)

Regression with Support Vector Machines (SVM)

Support Vector Machines can also be used for regression tasks.

Training the Model

# Train the SVM model
model_svm <- svm(Sepal.Length ~ ., data = irisTrain)
print(model_svm)

Making Predictions

# Predict on the test set
predictions_svm <- predict(model_svm, irisTest)

# Evaluate the model
postResample(predictions_svm, irisTest$Sepal.Length)

Unsupervised Learning

Clustering with K-Means

K-Means is a popular clustering algorithm.

Training the Model

# Determine the optimal number of clusters
wss <- (nrow(iris_norm)-1)*sum(apply(iris_norm[,1:4],2,var))
for (i in 2:15) wss[i] <- sum(kmeans(iris_norm[,1:4], centers=i)$withinss)

# Plot the elbow method
plot(1:15, wss, type="b", xlab="Number of Clusters", ylab="Within groups sum of squares")

# Train the K-Means model
set.seed(123)
kmeans_model <- kmeans(iris_norm[,1:4], centers=3)
print(kmeans_model)

Visualizing the Clusters

# Visualize the clusters
iris_norm$Cluster <- as.factor(kmeans_model$cluster)
ggplot(iris_norm, aes(Petal.Length, Petal.Width, color = Cluster)) + geom_point()

Model Evaluation

Evaluating the performance of machine learning models is essential to understand their effectiveness.

Cross-Validation

# Perform 10-fold cross-validation
control <- trainControl(method="cv", number=10)
model_cv <- train(Species ~ ., data=irisTrain, method="rf", trControl=control)
print(model_cv)

Confusion Matrix

# Confusion matrix for Random Forest model
confusionMatrix(predictions, irisTest$Species)
