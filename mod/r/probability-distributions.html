Introduction

Probability distributions are fundamental in statistics and data analysis. In R, there are numerous functions and packages available to work with various probability distributions. This section will guide you from the basics to advanced concepts of probability distributions in R.

Key Concepts

  • Probability Distribution: A function that describes the likelihood of obtaining the possible values that a random variable can take.
  • Random Variable: A variable whose values depend on outcomes of a random phenomenon.
  • Discrete vs. Continuous Distributions: Discrete distributions deal with countable outcomes, while continuous distributions deal with outcomes that can take any value within a range.

Common Probability Distributions in R

Discrete Distributions

  1. Binomial Distribution
  2. Poisson Distribution
  3. Geometric Distribution

Continuous Distributions

  1. Normal Distribution
  2. Exponential Distribution
  3. Uniform Distribution

Working with Distributions in R

Binomial Distribution

The binomial distribution models the number of successes in a fixed number of independent Bernoulli trials.

  • Functions:
    • dbinom(x, size, prob): Probability mass function.
    • pbinom(q, size, prob): Cumulative distribution function.
    • qbinom(p, size, prob): Quantile function.
    • rbinom(n, size, prob): Random generation.
# Example: Binomial Distribution
# Parameters: size = 10 trials, prob = 0.5 probability of success
size <- 10
prob <- 0.5

# Probability of exactly 5 successes
dbinom(5, size, prob)

# Cumulative probability of 5 or fewer successes
pbinom(5, size, prob)

# Quantile for 0.5 probability
qbinom(0.5, size, prob)

# Generate 10 random samples
rbinom(10, size, prob)

Normal Distribution

The normal distribution is a continuous probability distribution characterized by its bell-shaped curve.

  • Functions:
    • dnorm(x, mean, sd): Probability density function.
    • pnorm(q, mean, sd): Cumulative distribution function.
    • qnorm(p, mean, sd): Quantile function.
    • rnorm(n, mean, sd): Random generation.
# Example: Normal Distribution
# Parameters: mean = 0, sd = 1
mean <- 0
sd <- 1

# Density at x = 0
dnorm(0, mean, sd)

# Cumulative probability up to x = 1
pnorm(1, mean, sd)

# Quantile for 0.95 probability
qnorm(0.95, mean, sd)

# Generate 10 random samples
rnorm(10, mean, sd)

Poisson Distribution

The Poisson distribution models the number of events occurring within a fixed interval of time or space.

  • Functions:
    • dpois(x, lambda): Probability mass function.
    • ppois(q, lambda): Cumulative distribution function.
    • qpois(p, lambda): Quantile function.
    • rpois(n, lambda): Random generation.
# Example: Poisson Distribution
# Parameter: lambda = 3 (average rate of occurrence)
lambda <- 3

# Probability of exactly 2 events
dpois(2, lambda)

# Cumulative probability of 2 or fewer events
ppois(2, lambda)

# Quantile for 0.8 probability
qpois(0.8, lambda)

# Generate 10 random samples
rpois(10, lambda)

Advanced Topics

Fitting Distributions to Data

R provides functions to fit distributions to data, allowing for parameter estimation and goodness-of-fit testing.

  • Example: Fitting a normal distribution to data
# Generate sample data
data <- rnorm(100, mean = 5, sd = 2)

# Fit a normal distribution
fit <- fitdistr(data, "normal")

# Display the fit parameters
fit

Custom Distributions

You can define custom probability distributions using R functions.

  • Example: Custom probability density function
# Custom density function
custom_density <- function(x) {
  ifelse(x >= 0 & x <= 1, 2 * x, 0)
}

# Integrate to check if it's a valid density function
integrate(custom_density, -Inf, Inf)
