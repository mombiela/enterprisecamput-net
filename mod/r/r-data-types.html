Understanding data types is fundamental to programming in R. This section will cover the primary data types in R, how to use them, and their importance in data analysis.

Basic Data Types

R has several basic data types that are essential for any R programmer to understand. These include:

  • Numeric
  • Integer
  • Character
  • Logical
  • Complex

Numeric

Numeric data types are used to represent real numbers.

# Example of Numeric
num <- 42.5
print(num)  # Output: 42.5

Integer

Integers are whole numbers. In R, you can specify an integer by appending an L to the number.

# Example of Integer
int <- 42L
print(int)  # Output: 42

Character

Character data types are used to represent text or string values.

# Example of Character
char <- "Hello, R!"
print(char)  # Output: "Hello, R!"

Logical

Logical data types represent boolean values: TRUE or FALSE.

# Example of Logical
logi <- TRUE
print(logi)  # Output: TRUE

Complex

Complex numbers have both a real and an imaginary part.

# Example of Complex
comp <- 3 + 2i
print(comp)  # Output: 3+2i

Data Structures

R also provides several data structures to store collections of data. These include:

  • Vectors
  • Matrices
  • Arrays
  • Data Frames
  • Lists

Vectors

Vectors are the most basic data structure in R and can hold elements of the same type.

# Example of Vector
vec <- c(1, 2, 3, 4, 5)
print(vec)  # Output: 1 2 3 4 5

Matrices

Matrices are two-dimensional, homogeneous data structures.

# Example of Matrix
mat <- matrix(1:9, nrow = 3, ncol = 3)
print(mat)
# Output:
#      [,1] [,2] [,3]
# [1,]    1    4    7
# [2,]    2    5    8
# [3,]    3    6    9

Arrays

Arrays are multi-dimensional, homogeneous data structures.

# Example of Array
arr <- array(1:8, dim = c(2, 2, 2))
print(arr)
# Output:
# , , 1
#      [,1] [,2]
# [1,]    1    3
# [2,]    2    4
#
# , , 2
#      [,1] [,2]
# [1,]    5    7
# [2,]    6    8

Data Frames

Data frames are two-dimensional, heterogeneous data structures, similar to tables in a database or Excel spreadsheet.

# Example of Data Frame
df <- data.frame(
  Name = c("John", "Jane", "Doe"),
  Age = c(23, 25, 31),
  Gender = c("Male", "Female", "Male")
)
print(df)
# Output:
#   Name Age Gender
# 1 John  23   Male
# 2 Jane  25 Female
# 3  Doe  31   Male

Lists

Lists can hold elements of different types and structures.

# Example of List
lst <- list(
  Name = "John",
  Age = 23,
  Scores = c(90, 85, 88)
)
print(lst)
# Output:
# $Name
# [1] "John"
#
# $Age
# [1] 23
#
# $Scores
# [1] 90 85 88

Summary

In this section, we covered the basic data types in R, including numeric, integer, character, logical, and complex types. We also explored various data structures such as vectors, matrices, arrays, data frames, and lists. Understanding these data types and structures is crucial for effective data manipulation and analysis in R. As you progress in your R programming journey, these concepts will form the foundation for more advanced topics and techniques.
