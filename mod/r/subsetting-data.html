Subsetting Data
Merging and Joining Data ►

Subsetting data is a fundamental skill in R that allows you to extract specific elements from data structures like vectors, matrices, data frames, and lists. This topic will guide you from basic to advanced subsetting techniques.

Basic Subsetting

Vectors

Vectors are the simplest data structures in R. You can subset vectors using indices, logical vectors, or names.

  • Using Indices: Extract elements by specifying their positions.

    vec <- c(10, 20, 30, 40, 50)
vec[1]  # Extracts the first element: 10
vec[3:5]  # Extracts elements from the 3rd to the 5th: 30, 40, 50

  • Using Logical Vectors: Extract elements that meet a condition.

    vec <- c(10, 20, 30, 40, 50)
vec[vec > 25]  # Extracts elements greater than 25: 30, 40, 50

  • Using Names: Extract elements by their names.

    vec <- c(a = 10, b = 20, c = 30)
vec["b"]  # Extracts the element named 'b': 20

Matrices

Matrices are two-dimensional arrays. You can subset matrices using row and column indices.

  • Using Indices: Extract elements by specifying row and column positions. 
    mat <- matrix(1:9, nrow = 3)
mat[1, 2]  # Extracts the element in the 1st row, 2nd column: 2
mat[1, ]  # Extracts the entire 1st row: 1, 2, 3
mat[, 2]  # Extracts the entire 2nd column: 2, 5, 8

Data Frames

Data frames are similar to matrices but can contain different types of data. You can subset data frames using row and column indices, column names, or logical vectors.

  • Using Indices: Extract elements by specifying row and column positions.

    df <- data.frame(a = 1:3, b = 4:6, c = 7:9)
df[1, 2]  # Extracts the element in the 1st row, 2nd column: 4
df[1, ]  # Extracts the entire 1st row
df[, 2]  # Extracts the entire 2nd column

  • Using Column Names: Extract columns by their names.

    df <- data.frame(a = 1:3, b = 4:6, c = 7:9)
df["b"]  # Extracts the column named 'b'
df[c("a", "c")]  # Extracts columns 'a' and 'c'

  • Using Logical Vectors: Extract rows that meet a condition.

    df <- data.frame(a = 1:3, b = 4:6, c = 7:9)
df[df$a > 1, ]  # Extracts rows where column 'a' is greater than 1

Intermediate Subsetting

Lists

Lists can contain different types of data and structures. You can subset lists using indices or names.

  • Using Indices: Extract elements by specifying their positions.

    lst <- list(a = 1:3, b = "hello", c = TRUE)
lst[[1]]  # Extracts the first element: 1, 2, 3
lst[1]  # Extracts the first element as a list

  • Using Names: Extract elements by their names.

    lst <- list(a = 1:3, b = "hello", c = TRUE)
lst[["b"]]  # Extracts the element named 'b': "hello"
lst["b"]  # Extracts the element named 'b' as a list

Advanced Data Frames

Combining multiple subsetting techniques can be powerful.

  • Combining Conditions: Use multiple conditions to subset data frames.

    df <- data.frame(a = 1:5, b = 6:10, c = 11:15)
df[df$a > 2 & df$b < 9, ]  # Extracts rows where 'a' > 2 and 'b' < 9

  • Using subset Function: A more readable way to subset data frames.

    df <- data.frame(a = 1:5, b = 6:10, c = 11:15)
subset(df, a > 2 & b < 9)  # Extracts rows where 'a' > 2 and 'b' < 9

Advanced Subsetting

Using `dplyr` for Data Frames

The dplyr package provides a more readable and efficient way to subset data frames.

  • Using filter: Extract rows that meet a condition.

    library(dplyr)
df <- data.frame(a = 1:5, b = 6:10, c = 11:15)
df %>% filter(a > 2 & b < 9)  # Extracts rows where 'a' > 2 and 'b' < 9

  • Using select: Extract specific columns.

    df <- data.frame(a = 1:5, b = 6:10, c = 11:15)
df %>% select(a, c)  # Extracts columns 'a' and 'c'

  • Combining filter and select: Combine multiple operations.

    df <- data.frame(a = 1:5, b = 6:10, c = 11:15)
df %>% filter(a > 2) %>% select(a, c)  # Extracts rows where 'a' > 2 and columns 'a' and 'c'

Using `data.table` for Large Data Sets

The data.table package is optimized for large data sets and provides efficient subsetting.

  • Basic Subsetting: Extract rows and columns.

    library(data.table)
dt <- data.table(a = 1:5, b = 6:10, c = 11:15)
dt[a > 2, .(a, c)]  # Extracts rows where 'a' > 2 and columns 'a' and 'c'

  • Advanced Subsetting: Use complex conditions and operations.

    dt <- data.table(a = 1:5, b = 6:10, c = 11:15)
dt[a > 2 & b < 9, .(sum_a = sum(a), mean_c = mean(c))]  # Extracts rows where 'a' > 2 and 'b' < 9, then calculates sum of 'a' and mean of 'c'

Conclusion

Subsetting data is a crucial skill in R that allows you to manipulate and analyze data efficiently. Starting from basic techniques with vectors and matrices, you can advance to more complex operations with data frames and lists. Utilizing packages like dplyr and data.table can further enhance your ability to handle large and complex data sets. Practice these techniques to become proficient in data manipulation in R.
