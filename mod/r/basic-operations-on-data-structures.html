Introduction

In this section, we will explore basic operations on data structures in R. Understanding these operations is crucial for data manipulation and analysis. We will cover vectors, lists, matrices, data frames, and factors, providing examples and exercises to solidify your understanding.

Vectors

Vectors are the simplest type of data structure in R. They are sequences of data elements of the same basic type.

Creating Vectors

  • Use the c() function to create a vector.
# Creating a numeric vector
numeric_vector <- c(1, 2, 3, 4, 5)
print(numeric_vector)

# Creating a character vector
char_vector <- c("a", "b", "c")
print(char_vector)

Basic Operations

  • Access elements using indices.
  • Perform arithmetic operations.
# Accessing elements
first_element <- numeric_vector[1]
print(first_element)

# Arithmetic operations
sum_vector <- numeric_vector + 2
print(sum_vector)

Lists

Lists can contain elements of different types, including other lists.

Creating Lists

  • Use the list() function to create a list.
# Creating a list
my_list <- list(name = "John", age = 25, scores = c(90, 85, 88))
print(my_list)

Basic Operations

  • Access elements using $ or [[ ]].
# Accessing elements
name <- my_list$name
print(name)

# Accessing nested elements
first_score <- my_list$scores[1]
print(first_score)

Matrices

Matrices are two-dimensional, homogeneous data structures.

Creating Matrices

  • Use the matrix() function to create a matrix.
# Creating a matrix
my_matrix <- matrix(1:9, nrow = 3, ncol = 3)
print(my_matrix)

Basic Operations

  • Access elements using row and column indices.
  • Perform matrix operations.
# Accessing elements
element <- my_matrix[2, 3]
print(element)

# Matrix multiplication
result_matrix <- my_matrix %*% my_matrix
print(result_matrix)

Data Frames

Data frames are two-dimensional, heterogeneous data structures.

Creating Data Frames

  • Use the data.frame() function to create a data frame.
# Creating a data frame
my_data_frame <- data.frame(
  Name = c("John", "Jane", "Doe"),
  Age = c(25, 30, 22),
  Score = c(90, 85, 88)
)
print(my_data_frame)

Basic Operations

  • Access elements using $ or indices.
  • Add new columns.
# Accessing elements
age <- my_data_frame$Age
print(age)

# Adding a new column
my_data_frame$Gender <- c("M", "F", "M")
print(my_data_frame)

Factors

Factors are used to handle categorical data.

Creating Factors

  • Use the factor() function to create a factor.
# Creating a factor
gender <- factor(c("Male", "Female", "Female", "Male"))
print(gender)

Basic Operations

  • Access levels and change levels.
# Accessing levels
levels(gender)

# Changing levels
levels(gender) <- c("M", "F")
print(gender)

Summary

In this section, we covered the basic operations on various data structures in R, including vectors, lists, matrices, data frames, and factors. These operations form the foundation for more advanced data manipulation and analysis tasks. Practice these operations with different datasets to become proficient in handling data structures in R.
