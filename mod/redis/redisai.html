RedisAI is a Redis module designed to execute deep learning models and manage the entire lifecycle of AI models within Redis. It allows you to run models from popular deep learning frameworks such as TensorFlow, PyTorch, and ONNX directly within Redis, enabling low-latency and high-throughput AI inference.

Introduction to RedisAI

  • What is RedisAI?

    • RedisAI is a Redis module that allows you to execute and manage AI models within Redis.
    • It supports multiple deep learning frameworks, including TensorFlow, PyTorch, and ONNX.
    • RedisAI provides a seamless way to integrate AI capabilities into your Redis-based applications.

  • Why use RedisAI?

    • Low Latency: RedisAI leverages Redis's in-memory data structure to provide low-latency AI inference.
    • Scalability: RedisAI can scale horizontally with Redis, making it suitable for high-throughput applications.
    • Flexibility: Supports multiple AI frameworks, allowing you to choose the best tool for your needs.

Setting Up RedisAI

  • Prerequisites:

    • A working installation of Redis.
    • Docker (optional but recommended for ease of setup).

  • Installation Steps:

    1. Using Docker: 
      docker run -p 6379:6379 --name redisai redislabs/redisai
    2. Building from Source:
      • Clone the RedisAI repository from GitHub.
      • Follow the build instructions provided in the repository.

  • Loading the RedisAI Module:

    redis-server --loadmodule /path/to/redisai.so

Basic Operations with RedisAI

  • Loading a Model:

    • RedisAI allows you to load models into Redis and execute them.
    • Example: Loading a TensorFlow model. 
      import redis

r = redis.Redis()

with open('model.pb', 'rb') as f:
    model = f.read()

r.execute_command('AI.MODELSET', 'mymodel', 'TF', 'CPU', 'INPUTS', 'input', 'OUTPUTS', 'output', model)

  • Running Inference:

    • Example: Running inference using the loaded model. 
      import numpy as np

input_data = np.random.rand(1, 3, 224, 224).astype(np.float32).tobytes()
r.execute_command('AI.TENSORSET', 'input_tensor', 'FLOAT', 1, 3, 224, 224, 'BLOB', input_data)
r.execute_command('AI.MODELRUN', 'mymodel', 'INPUTS', 'input_tensor', 'OUTPUTS', 'output_tensor')
output = r.execute_command('AI.TENSORGET', 'output_tensor', 'BLOB')

Advanced Features of RedisAI

  • Script Management:

    • RedisAI supports executing custom scripts using the TorchScript or ONNX runtime.
    • Example: Loading and executing a TorchScript. 
      script = """
def forward(input):
    return input + 1
"""
r.execute_command('AI.SCRIPTSET', 'myscript', 'CPU', script)
r.execute_command('AI.SCRIPTRUN', 'myscript', 'forward', 'INPUTS', 'input_tensor', 'OUTPUTS', 'output_tensor')

  • Model Versioning:

    • RedisAI supports versioning of models, allowing you to manage multiple versions of the same model.
    • Example: Loading a new version of a model. 
      r.execute_command('AI.MODELSET', 'mymodel', 'TF', 'CPU', 'INPUTS', 'input', 'OUTPUTS', 'output', model, 'TAG', 'v2')

  • Monitoring and Debugging:

    • RedisAI provides commands to monitor and debug model execution.
    • Example: Checking model info. 
      redis-cli AI.MODELGET mymodel META

Use Cases and Best Practices

  • Use Cases:

    • Real-time recommendation systems.
    • Fraud detection.
    • Image and video processing.
    • Natural language processing.

  • Best Practices:

    • Optimize Models: Ensure models are optimized for inference to reduce latency.
    • Resource Management: Monitor and manage resources to avoid bottlenecks.
    • Version Control: Use model versioning to manage updates and rollbacks.

Conclusion

RedisAI extends Redis's capabilities by integrating AI model execution directly within the database. This enables low-latency, high-throughput AI applications that can scale with Redis. By understanding the basics of RedisAI, setting it up, and leveraging its advanced features, you can build powerful AI-driven applications that benefit from Redis's performance and scalability.

RedisAI is a powerful tool for developers looking to integrate AI into their Redis-based applications. With its support for multiple frameworks and advanced features, RedisAI provides a flexible and efficient way to deploy and manage AI models.
