Introduction to Redis

Redis is an open-source, in-memory data structure store used as a database, cache, and message broker. It supports various data structures such as strings, hashes, lists, sets, and more. This guide will take you from the basics of Redis to advanced implementation techniques.

Setting Up Redis

Installation

  • Linux: 
    sudo apt update
sudo apt install redis-server
  • macOS: 
    brew install redis
  • Windows: Use the Windows Subsystem for Linux (WSL) or Docker.

Starting Redis

  • Linux/macOS: 
    redis-server
  • Windows (using WSL): 
    redis-server

Connecting to Redis

  • Using Redis CLI: 
    redis-cli

Basic Redis Commands

Strings

  • Set a string value: 
    SET key "value"
  • Get a string value: 
    GET key

Lists

  • Add to a list: 
    LPUSH mylist "value1"
LPUSH mylist "value2"
  • Retrieve from a list: 
    LRANGE mylist 0 -1

Hashes

  • Set a hash field: 
    HSET myhash field1 "value1"
  • Get a hash field: 
    HGET myhash field1

Sets

  • Add to a set: 
    SADD myset "value1"
SADD myset "value2"
  • Get all members of a set: 
    SMEMBERS myset

Intermediate Redis Concepts

Expiration and TTL

  • Set a key with an expiration: 
    SETEX key 10 "value"  # Key expires in 10 seconds
  • Check TTL of a key: 
    TTL key

Transactions

  • Start a transaction: 
    MULTI
  • Execute a transaction: 
    EXEC

Pub/Sub

  • Publish a message: 
    PUBLISH channel "message"
  • Subscribe to a channel: 
    SUBSCRIBE channel

Advanced Redis Techniques

Redis Persistence

  • RDB (Redis Database Backup): Periodic snapshots of your dataset.
  • AOF (Append Only File): Logs every write operation received by the server.

Redis Clustering

  • Setting up a Redis Cluster: 
    redis-cli --cluster create 127.0.0.1:7000 127.0.0.1:7001 127.0.0.1:7002 --cluster-replicas 1

Lua Scripting

  • Running a Lua script: 
    EVAL "return redis.call('SET', KEYS[1], ARGV[1])" 1 key value

Conclusion

Redis is a powerful tool for managing in-memory data structures, offering high performance and flexibility. From basic commands to advanced clustering and scripting, mastering Redis can significantly enhance your application's performance and scalability. Practice the commands and concepts discussed in this guide to become proficient in Redis.
