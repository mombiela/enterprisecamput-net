Introduction

Trees are a fundamental data structure in computer science, used to represent hierarchies and relationships between elements. In this topic, we will explore various applications of trees in the field of computer science and how they are used to solve common problems.

Common Applications of Trees

1. Binary Search Trees (BST)

Binary Search Trees are a data structure that allows for efficient searching, insertion, and deletion of elements.

  • Properties:

    • Each node has at most two children.
    • The left child contains a value less than the parent node.
    • The right child contains a value greater than the parent node.

  • Code Example:

    class Node:
    def __init__(self, key):
        self.left = None
        self.right = None
        self.val = key

def insert(root, key):
    if root is None:
        return Node(key)
    else:
        if root.val < key:
            root.right = insert(root.right, key)
        else:
            root.left = insert(root.left, key)
    return root

# Create the tree
root = Node(50)
root = insert(root, 30)
root = insert(root, 20)
root = insert(root, 40)
root = insert(root, 70)
root = insert(root, 60)
root = insert(root, 80)

2. AVL Trees

AVL Trees are a variant of Binary Search Trees that remain balanced to ensure that search, insertion, and deletion operations are performed in logarithmic time.

  • Properties:

    • It is a binary search tree.
    • The height difference between the left and right subtrees of any node is at most 1.

  • Code Example:

    class Node:
    def __init__(self, key):
        self.key = key
        self.left = None
        self.right = None
        self.height = 1

def insert(root, key):
    if not root:
        return Node(key)
    elif key < root.key:
        root.left = insert(root.left, key)
    else:
        root.right = insert(root.right, key)

    root.height = 1 + max(get_height(root.left), get_height(root.right))

    balance = get_balance(root)

    # Rotations to balance the tree
    if balance > 1 and key < root.left.key:
        return right_rotate(root)
    if balance < -1 and key > root.right.key:
        return left_rotate(root)
    if balance > 1 and key > root.left.key:
        root.left = left_rotate(root.left)
        return right_rotate(root)
    if balance < -1 and key < root.right.key:
        root.right = right_rotate(root.right)
        return left_rotate(root)

    return root

def get_height(root):
    if not root:
        return 0
    return root.height

def get_balance(root):
    if not root:
        return 0
    return get_height(root.left) - get_height(root.right)

def right_rotate(z):
    y = z.left
    T3 = y.right
    y.right = z
    z.left = T3
    z.height = 1 + max(get_height(z.left), get_height(z.right))
    y.height = 1 + max(get_height(y.left), get_height(y.right))
    return y

def left_rotate(z):
    y = z.right
    T2 = y.left
    y.left = z
    z.right = T2
    z.height = 1 + max(get_height(z.left), get_height(z.right))
    y.height = 1 + max(get_height(y.left), get_height(y.right))
    return y

3. B-Trees

B-Trees are a self-balancing data structure that keeps data ordered and allows searches, insertions, and deletions in logarithmic time. They are widely used in database systems and file systems.

  • Properties:

    • Each node can have more than two children.
    • All leaf nodes are at the same level.
    • Internal nodes can have a variable number of children within a predefined range.

  • Code Example:

    class BTreeNode:
    def __init__(self, t, leaf=False):
        self.t = t
        self.leaf = leaf
        self.keys = []
        self.children = []

class BTree:
    def __init__(self, t):
        self.root = BTreeNode(t, True)
        self.t = t

    def insert(self, k):
        root = self.root
        if len(root.keys) == (2 * self.t) - 1:
            temp = BTreeNode(self.t)
            self.root = temp
            temp.children.insert(0, root)
            self.split_child(temp, 0)
            self.insert_non_full(temp, k)
        else:
            self.insert_non_full(root, k)

    def insert_non_full(self, x, k):
        i = len(x.keys) - 1
        if x.leaf:
            x.keys.append(None)
            while i >= 0 and k < x.keys[i]:
                x.keys[i + 1] = x.keys[i]
                i -= 1
            x.keys[i + 1] = k
        else:
            while i >= 0 and k < x.keys[i]:
                i -= 1
            i += 1
            if len(x.children[i].keys) == (2 * self.t) - 1:
                self.split_child(x, i)
                if k > x.keys[i]:
                    i += 1
            self.insert_non_full(x.children[i], k)

    def split_child(self, x, i):
        t = self.t
        y = x.children[i]
        z = BTreeNode(t, y.leaf)
        x.children.insert(i + 1, z)
        x.keys.insert(i, y.keys[t - 1])
        z.keys = y.keys[t:(2 * t) - 1]
        y.keys = y.keys[0:t - 1]
        if not y.leaf:
            z.children = y.children[t:(2 * t)]
            y.children = y.children[0:t - 1]

Conclusion

Trees are a versatile and powerful data structure with applications in various areas of computer science. From efficient searching in Binary Search Trees and AVL Trees to managing large volumes of data in B-Trees, understanding and properly using these data structures is essential for any IT professional. Through practical examples and exercises, students can develop a solid understanding of how and when to use these data structures in their projects.
