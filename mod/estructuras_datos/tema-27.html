Introduction

In this section, we will focus on practical exercises that will help you understand and apply tree concepts in data structures. Trees are a hierarchical structure used in various applications, from representing hierarchies to optimizing searches and organizing data.

Key Concepts

  • Node: Basic element of a tree that contains a value or data.
  • Root: The top node of a tree.
  • Child: Descendant node of another node.
  • Parent: Node that has one or more child nodes.
  • Leaf: Node that has no children.
  • Height: Length of the longest path from the root to a leaf.
  • Depth: Length of the path from the root to a specific node.
  • Binary Tree: Tree in which each node has at most two children.
  • Binary Search Tree (BST): Binary tree in which for each node, the values of the left subtree are smaller and those of the right subtree are larger.

Exercise 1: Creating a Binary Tree

Description

In this exercise, you will create a simple binary tree and perform basic operations such as insertion and traversal.

Example Code

class Node:
    def __init__(self, value):
        self.value = value
        self.left = None
        self.right = None

class BinaryTree:
    def __init__(self):
        self.root = None

    def insert(self, value):
        if self.root is None:
            self.root = Node(value)
        else:
            self._insert_recursive(value, self.root)

    def _insert_recursive(self, value, current_node):
        if value < current_node.value:
            if current_node.left is None:
                current_node.left = Node(value)
            else:
                self._insert_recursive(value, current_node.left)
        else:
            if current_node.right is None:
                current_node.right = Node(value)
            else:
                self._insert_recursive(value, current_node.right)

    def inorder_traversal(self, current_node):
        if current_node is not None:
            self.inorder_traversal(current_node.left)
            print(current_node.value, end=' ')
            self.inorder_traversal(current_node.right)

# Using the tree
tree = BinaryTree()
values = [7, 3, 9, 1, 5, 8, 10]
for value in values:
    tree.insert(value)

print("Inorder Traversal:")
tree.inorder_traversal(tree.root)

Explanation

  • Node Class: Defines a node with a value and pointers to its left and right children.
  • BinaryTree Class: Defines the binary tree with methods to insert values and traverse the tree in order.
  • Insert Method: Inserts a value into the tree, placing it in the correct position.
  • Inorder Traversal Method: Performs an inorder traversal of the tree, printing the values in ascending order.

Exercise 2: Searching in a Binary Search Tree (BST)

Description

In this exercise, you will implement a method to search for a value in a BST.

Example Code

class BinarySearchTree(BinaryTree):
    def search(self, value):
        return self._search_recursive(value, self.root)

    def _search_recursive(self, value, current_node):
        if current_node is None:
            return False
        elif value == current_node.value:
            return True
        elif value < current_node.value:
            return self._search_recursive(value, current_node.left)
        else:
            return self._search_recursive(value, current_node.right)

# Using the search tree
bst = BinarySearchTree()
values = [7, 3, 9, 1, 5, 8, 10]
for value in values:
    bst.insert(value)

print("Search 5:", bst.search(5))  # Should print True
print("Search 6:", bst.search(6))  # Should print False

Explanation

  • BinarySearchTree Class: Inherits from BinaryTree and adds a method to search for values.
  • Search Method: Calls a recursive method to search for a value in the tree.
  • _search_recursive Method: Performs the recursive search, comparing the searched value with the current node's value and moving left or right as appropriate.
