Introduction

Doubly linked lists are a fundamental data structure in programming and computer science. Unlike singly linked lists, where each node points only to the next node, in a doubly linked list, each node contains two pointers: one to the next node and another to the previous node. This allows bidirectional navigation through the list, which can be useful in various applications.

Key Concepts

  • Node: The basic unit of a doubly linked list. Each node contains a value and two pointers: one to the next node and another to the previous node.
  • Head: The first node of the list.
  • Tail: The last node of the list.
  • Next Pointer: A pointer that points to the next node in the list.
  • Previous Pointer: A pointer that points to the previous node in the list.

Structure of a Node

In a doubly linked list, each node has the following structure:

class Node:
    def __init__(self, value):
        self.value = value
        self.next = None  # Pointer to the next node
        self.previous = None  # Pointer to the previous node

Basic Operations

Insertion

Insertion in a doubly linked list can be done in several positions: at the beginning, at the end, or at a specific position.

Insertion at the Beginning

def insert_at_beginning(self, value):
    new_node = Node(value)
    if self.head is None:
        self.head = self.tail = new_node
    else:
        new_node.next = self.head
        self.head.previous = new_node
        self.head = new_node

Insertion at the End

def insert_at_end(self, value):
    new_node = Node(value)
    if self.tail is None:
        self.head = self.tail = new_node
    else:
        new_node.previous = self.tail
        self.tail.next = new_node
        self.tail = new_node

Deletion

Deletion can also be done in several positions: at the beginning, at the end, or at a specific position.

Deletion at the Beginning

def delete_at_beginning(self):
    if self.head is None:
        return
    if self.head == self.tail:
        self.head = self.tail = None
    else:
        self.head = self.head.next
        self.head.previous = None

Deletion at the End

def delete_at_end(self):
    if self.tail is None:
        return
    if self.head == self.tail:
        self.head = self.tail = None
    else:
        self.tail = self.tail.previous
        self.tail.next = None

Conclusion

Doubly linked lists are a versatile data structure that allows bidirectional navigation and efficient insertion and deletion operations. Although they consume more memory than singly linked lists due to the additional pointers, their flexibility and efficiency in certain operations make them a valuable option in many applications.
