Introduction

Stacks are a fundamental data structure in computer science, characterized by their LIFO (Last In, First Out) behavior. In this topic, we will explore various practical applications of stacks in software development and other areas of computer science.

Common Applications of Stacks

1. Expression Evaluation

Stacks are widely used in the evaluation of arithmetic and logical expressions, especially in postfix (RPN) and prefix notations.

  • Postfix Notation (RPN): In this notation, operators follow their operands. Stacks facilitate the evaluation of these expressions.

    Python code example:

    def evaluate_postfix(expression):
    stack = []
    for char in expression:
        if char.isdigit():
            stack.append(int(char))
        else:
            b = stack.pop()
            a = stack.pop()
            if char == '+':
                stack.append(a + b)
            elif char == '-':
                stack.append(a - b)
            elif char == '*':
                stack.append(a * b)
            elif char == '/':
                stack.append(a / b)
    return stack.pop()

expression = "231*+9-"
print(evaluate_postfix(expression))  # Output: -4

2. Handling Recursion

Stacks are fundamental for implementing recursion. The function call system uses a stack to manage function calls and returns.

  • Recursion Example: The factorial function is a classic example of recursion.

    Python code example:

    def factorial(n):
    if n == 0:
        return 1
    else:
        return n * factorial(n - 1)

print(factorial(5))  # Output: 120

3. Undo and Redo in Applications

Stacks are used in software applications to implement undo and redo functionalities.

  • Undo and Redo: When performing an action, it is saved in an undo stack. When undoing an action, it is moved to a redo stack.

    Python code example:

    class UndoRedo:
    def __init__(self):
        self.undo_stack = []
        self.redo_stack = []

    def do_action(self, action):
        self.undo_stack.append(action)
        self.redo_stack.clear()

    def undo(self):
        if self.undo_stack:
            action = self.undo_stack.pop()
            self.redo_stack.append(action)
            return action
        return None

    def redo(self):
        if self.redo_stack:
            action = self.redo_stack.pop()
            self.undo_stack.append(action)
            return action
        return None

ur = UndoRedo()
ur.do_action("Action 1")
ur.do_action("Action 2")
print(ur.undo())  # Output: Action 2
print(ur.redo())  # Output: Action 2

4. Web Page Navigation

Web browsers use stacks to manage forward and backward navigation between pages.

  • Page Navigation: When visiting a new page, it is saved in a history stack. When going back, it is moved to a future pages stack.

    Python code example:

    class BrowserHistory:
    def __init__(self):
        self.history_stack = []
        self.future_stack = []

    def visit(self, url):
        self.history_stack.append(url)
        self.future_stack.clear()

    def back(self):
        if self.history_stack:
            url = self.history_stack.pop()
            self.future_stack.append(url)
            return url
        return None

    def forward(self):
        if self.future_stack:
            url = self.future_stack.pop()
            self.history_stack.append(url)
            return url
        return None

browser = BrowserHistory()
browser.visit("page1.com")
browser.visit("page2.com")
print(browser.back())  # Output: page2.com
print(browser.forward())  # Output: page2.com

Conclusion

Stacks are a versatile and powerful data structure with numerous applications in software development and other areas of computer science. From expression evaluation to implementing undo and redo functionalities, stacks play a crucial role in the efficiency and functionality of many applications. Understanding and knowing how to implement stacks is essential for any IT professional.
