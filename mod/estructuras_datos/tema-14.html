Introduction

In this topic, we will explore the concept of a queue, a fundamental data structure in computer science. Queues are used in various applications, from task management in operating systems to the implementation of search algorithms in artificial intelligence.

Definition of Queue

A queue is a linear data structure that follows the FIFO (First In, First Out) principle, meaning the first element to enter is the first to leave. This is similar to a line in real life, where people are served in the order they arrived.

Main Characteristics

  • FIFO (First In, First Out): The first element added to the queue will be the first to be removed.
  • Basic Operations:
    • Enqueue: Add an element to the end of the queue.
    • Dequeue: Remove the first element from the queue.
    • Front: Get the first element of the queue without removing it.
    • IsEmpty: Check if the queue is empty.
    • Size: Get the number of elements in the queue.

Queue Implementation

Queues can be implemented in various ways, the most common being through arrays or linked lists. Below is a basic implementation using a linked list in Python.

Python Code Example

class Node:
    def __init__(self, value):
        self.value = value
        self.next = None

class Queue:
    def __init__(self):
        self.front = None
        self.rear = None
        self.size = 0

    def is_empty(self):
        return self.size == 0

    def enqueue(self, value):
        new_node = Node(value)
        if self.rear:
            self.rear.next = new_node
        self.rear = new_node
        if not self.front:
            self.front = new_node
        self.size += 1

    def dequeue(self):
        if self.is_empty():
            raise IndexError("Dequeue from empty queue")
        value = self.front.value
        self.front = self.front.next
        if not self.front:
            self.rear = None
        self.size -= 1
        return value

    def get_front(self):
        if self.is_empty():
            raise IndexError("Front from empty queue")
        return self.front.value

    def get_size(self):
        return self.size

# Usage example
queue = Queue()
queue.enqueue(10)
queue.enqueue(20)
print(queue.dequeue())  # Output: 10
print(queue.get_front())  # Output: 20
print(queue.get_size())  # Output: 1

Code Explanation

  • Node Class: Represents a node in the linked list, with a value and a pointer to the next node.
  • Queue Class: Implements the queue with methods for basic operations.
    • enqueue: Adds a new node to the end of the queue.
    • dequeue: Removes the node from the front of the queue and returns its value.
    • get_front: Returns the value of the node at the front of the queue without removing it.
    • is_empty: Checks if the queue is empty.
    • get_size: Returns the number of elements in the queue.

Comparison between Implementations

| Characteristic | Array | Linked List | |---------------------|-----------------|------------------------------| | Enqueue Complexity | O(1) | O(1) | | Dequeue Complexity | O(n) | O(1) | | Memory Usage | Fixed | Dynamic | | Flexibility | Lower | Higher |

Conclusion

Queues are essential data structures that allow managing elements in a specific order, following the FIFO principle. Their implementation can vary according to the specific needs of the problem, with linked lists being a flexible and efficient option for many applications. Understanding how they work and how to implement them is crucial for any IT professional.
