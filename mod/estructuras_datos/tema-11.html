Introduction to Stacks

A stack is a linear data structure that follows the LIFO (Last In, First Out) principle, which means that the last element to enter is the first to leave. Stacks are fundamental in computer science and are used in various applications, such as memory management, expression evaluation, and the implementation of backtracking algorithms.

Fundamental Operations

The basic operations that can be performed on a stack are:

  • Push: Add an element to the stack.
  • Pop: Remove the top element from the stack.
  • Peek (or Top): Get the top element of the stack without removing it.
  • isEmpty: Check if the stack is empty.
  • Size: Get the number of elements in the stack.

Push

The push operation adds a new element to the top of the stack.

class Stack:
    def __init__(self):
        self.items = []

    def push(self, item):
        self.items.append(item)

# Usage example
stack = Stack()
stack.push(1)
stack.push(2)
print(stack.items)  # Output: [1, 2]

Pop

The pop operation removes and returns the top element of the stack.

class Stack:
    def __init__(self):
        self.items = []

    def push(self, item):
        self.items.append(item)

    def pop(self):
        if not self.is_empty():
            return self.items.pop()
        else:
            return "The stack is empty"

    def is_empty(self):
        return len(self.items) == 0

# Usage example
stack = Stack()
stack.push(1)
stack.push(2)
print(stack.pop())  # Output: 2
print(stack.items)  # Output: [1]

Peek (or Top)

The peek operation returns the top element of the stack without removing it.

class Stack:
    def __init__(self):
        self.items = []

    def push(self, item):
        self.items.append(item)

    def pop(self):
        if not self.is_empty():
            return self.items.pop()
        else:
            return "The stack is empty"

    def peek(self):
        if not self.is_empty():
            return self.items[-1]
        else:
            return "The stack is empty"

    def is_empty(self):
        return len(self.items) == 0

# Usage example
stack = Stack()
stack.push(1)
stack.push(2)
print(stack.peek())  # Output: 2
print(stack.items)   # Output: [1, 2]

isEmpty

The isEmpty operation checks if the stack is empty.

class Stack:
    def __init__(self):
        self.items = []

    def is_empty(self):
        return len(self.items) == 0

# Usage example
stack = Stack()
print(stack.is_empty())  # Output: True
stack.push(1)
print(stack.is_empty())  # Output: False

Size

The size operation returns the number of elements in the stack.

class Stack:
    def __init__(self):
        self.items = []

    def size(self):
        return len(self.items)

# Usage example
stack = Stack()
stack.push(1)
stack.push(2)
print(stack.size())  # Output: 2

Comparison of Operations

| Operation | Description | Time Complexity | |-----------|--------------------------------------|----------------------| | Push | Add an element to the stack | O(1) | | Pop | Remove the top element | O(1) | | Peek | Get the top element | O(1) | | isEmpty | Check if the stack is empty | O(1) | | Size | Get the number of elements | O(1) |

Conclusion

Stacks are a fundamental data structure in computer science, and understanding their basic operations is crucial for any IT professional. The push, pop, peek, isEmpty, and size operations are essential for efficiently manipulating stacks. Practicing with code examples and exercises will help solidify these concepts and prepare for more advanced applications.
