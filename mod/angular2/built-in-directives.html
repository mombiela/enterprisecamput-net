Introduction

Angular provides a set of built-in directives that help you manage forms, lists, styles, and more. These directives are essential for building dynamic and responsive applications. In this section, we will cover the most commonly used built-in directives, their purposes, and how to use them effectively.

Structural Directives

Structural directives are responsible for altering the DOM layout by adding or removing elements. The most commonly used structural directives in Angular are *ngIf, *ngFor, and *ngSwitch.

`*ngIf`

The *ngIf directive conditionally includes a template based on the value of an expression.

Example:

<div *ngIf="isLoggedIn">
  Welcome, user!
</div>

In this example, the div element will only be rendered if isLoggedIn is true.

`*ngFor`

The *ngFor directive is used to repeat a template for each item in a list.

Example:

<ul>
  <li *ngFor="let item of items">{{ item }}</li>
</ul>

Here, an unordered list is created, and each li element represents an item in the items array.

`*ngSwitch`

The *ngSwitch directive is used to conditionally include one of several possible templates based on the value of an expression.

Example:

<div [ngSwitch]="status">
  <p *ngSwitchCase="'active'">Active</p>
  <p *ngSwitchCase="'inactive'">Inactive</p>
  <p *ngSwitchDefault>Unknown status</p>
</div>

In this example, the paragraph that matches the status value will be rendered.

Attribute Directives

Attribute directives change the appearance or behavior of an element, component, or another directive. Common attribute directives include ngClass, ngStyle, and ngModel.

`ngClass`

The ngClass directive adds and removes CSS classes on an HTML element.

Example:

<div [ngClass]="{'active': isActive, 'disabled': isDisabled}">
  This is a div with dynamic classes.
</div>

In this example, the div element will have the active class if isActive is true and the disabled class if isDisabled is true.

`ngStyle`

The ngStyle directive allows you to set inline styles dynamically.

Example:

<div [ngStyle]="{'color': textColor, 'font-size': fontSize + 'px'}">
  This is a div with dynamic styles.
</div>

Here, the div element's text color and font size are set based on the values of textColor and fontSize.

`ngModel`

The ngModel directive binds an input, select, or textarea element to a property on the scope.

Example:

<input [(ngModel)]="username" placeholder="Enter your username">
<p>Your username is: {{ username }}</p>

In this example, the input field is bound to the username property, and any changes to the input will update the username property and vice versa.

Summary

Built-in directives in Angular are powerful tools that allow you to manipulate the DOM and manage the appearance and behavior of your application dynamically. Understanding and effectively using these directives is crucial for building robust and responsive Angular applications. We covered the most commonly used structural and attribute directives, including *ngIf, *ngFor, *ngSwitch, ngClass, ngStyle, and ngModel. Practice using these directives in your projects to become more proficient in Angular development.
