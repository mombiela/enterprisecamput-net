What is TypeScript?

TypeScript is a superset of JavaScript that adds static types. It is developed and maintained by Microsoft and has gained significant popularity in the development community, especially for large-scale applications. TypeScript helps catch errors early through a type system and makes JavaScript development more robust and maintainable.

Key Features of TypeScript

  • Static Typing: Allows you to define types for variables, function parameters, and return values.
  • Type Inference: Automatically infers types when they are not explicitly defined.
  • Interfaces: Define contracts within your code.
  • Classes and Inheritance: Support for object-oriented programming.
  • Modules: Organize code into reusable components.
  • Tooling: Enhanced support in IDEs for code navigation, refactoring, and debugging.

Setting Up TypeScript

To start using TypeScript, you need to install it via npm (Node Package Manager):

npm install -g typescript

You can then compile TypeScript files to JavaScript using the tsc command:

tsc filename.ts

Basic TypeScript Syntax

Variables and Types

TypeScript allows you to define types for variables:

let isDone: boolean = false;
let decimal: number = 6;
let color: string = "blue";
let list: number[] = [1, 2, 3];

Functions

You can specify types for function parameters and return values:

function add(x: number, y: number): number {
    return x + y;
}

Interfaces

Interfaces define the structure of an object:

interface Person {
    firstName: string;
    lastName: string;
}

function greet(person: Person) {
    return "Hello, " + person.firstName + " " + person.lastName;
}

let user = { firstName: "John", lastName: "Doe" };
console.log(greet(user));

Classes

TypeScript supports classes and inheritance:

class Animal {
    name: string;
    constructor(name: string) {
        this.name = name;
    }
    move(distance: number = 0) {
        console.log(`${this.name} moved ${distance}m.`);
    }
}

class Dog extends Animal {
    bark() {
        console.log('Woof! Woof!');
    }
}

let dog = new Dog("Rex");
dog.bark();
dog.move(10);

Modules

Modules help in organizing code:

// math.ts
export function add(x: number, y: number): number {
    return x + y;
}

// app.ts
import { add } from './math';
console.log(add(5, 3));

TypeScript in Angular

Angular is built with TypeScript, and understanding TypeScript is crucial for developing Angular applications. Here are some key points where TypeScript is used in Angular:

  • Components: Define component classes with TypeScript.
  • Services: Use TypeScript to define services and their methods.
  • Dependency Injection: TypeScript helps in defining types for dependencies.
  • Routing: Define routes and their parameters using TypeScript.

Example: Angular Component with TypeScript

import { Component } from '@angular/core';

@Component({
  selector: 'app-root',
  template: `<h1>{{title}}</h1>`,
})
export class AppComponent {
  title: string = 'Hello, Angular!';
}

Conclusion

TypeScript is a powerful tool that enhances JavaScript by adding static types, interfaces, classes, and modules. It is essential for developing robust and maintainable Angular applications. By understanding the basics of TypeScript, you will be better equipped to work with Angular and take advantage of its full potential.
