Introduction

String algorithms are fundamental in computer science and have applications in various areas such as information retrieval, natural language processing, and bioinformatics. This topic covers key concepts and the most commonly used algorithms for manipulating and analyzing strings.

Key Concepts

  • Strings: Sequences of characters that can represent words, phrases, or any other type of textual data.
  • Substring: A portion of a string.
  • Pattern: A string that is searched within another string.
  • Prefix and Suffix: Substrings that start from the beginning or end at the end of a string, respectively.

Common Algorithms

Substring Search

Brute Force Algorithm

The brute force algorithm compares the pattern with all possible substrings of the main string.

def brute_force(text, pattern):
    n = len(text)
    m = len(pattern)
    for i in range(n - m + 1):
        j = 0
        while j < m and text[i + j] == pattern[j]:
            j += 1
        if j == m:
            return i
    return -1

text = "hello world"
pattern = "world"
print(brute_force(text, pattern))  # Output: 5

Knuth-Morris-Pratt (KMP) Algorithm

The KMP algorithm improves efficiency by avoiding redundant comparisons using a prefix table.

def build_kmp_table(pattern):
    m = len(pattern)
    table = [0] * m
    j = 0
    for i in range(1, m):
        if pattern[i] == pattern[j]:
            j += 1
            table[i] = j
        else:
            if j != 0:
                j = table[j - 1]
                i -= 1
            else:
                table[i] = 0
    return table

def kmp(text, pattern):
    n = len(text)
    m = len(pattern)
    table = build_kmp_table(pattern)
    i = j = 0
    while i < n:
        if pattern[j] == text[i]:
            i += 1
            j += 1
        if j == m:
            return i - j
        elif i < n and pattern[j] != text[i]:
            if j != 0:
                j = table[j - 1]
            else:
                i += 1
    return -1

text = "hello world"
pattern = "world"
print(kmp(text, pattern))  # Output: 5

Algorithm Comparison

| Algorithm | Time Complexity | Advantages | Disadvantages | |-------------------|----------------------|-------------------------------|------------------------------| | Brute Force | O(n * m) | Easy to implement | Inefficient for long strings | | Knuth-Morris-Pratt| O(n + m) | Efficient for long patterns | More complex to implement |

String Transformations

Levenshtein Algorithm

The Levenshtein algorithm calculates the minimum edit distance between two strings, i.e., the minimum number of operations needed to transform one string into another.

def levenshtein_distance(s1, s2):
    if len(s1) < len(s2):
        return levenshtein_distance(s2, s1)

    if len(s2) == 0:
        return len(s1)

    previous_row = range(len(s2) + 1)
    for i, c1 in enumerate(s1):
        current_row = [i + 1]
        for j, c2 in enumerate(s2):
            insert = previous_row[j + 1] + 1
            delete = current_row[j] + 1
            substitute = previous_row[j] + (c1 != c2)
            current_row.append(min(insert, delete, substitute))
        previous_row = current_row

    return previous_row[-1]

s1 = "kitten"
s2 = "sitting"
print(levenshtein_distance(s1, s2))  # Output: 3

Conclusion

String algorithms are powerful tools for the manipulation and analysis of textual data. From substring search to string transformation, these algorithms have practical applications in multiple fields. Understanding and knowing how to implement these algorithms is essential for any IT professional working with textual data.
