Introduction to Loops in Ada

Loops are fundamental control structures that allow you to execute a block of code multiple times. Ada provides several types of loops to handle different scenarios. Understanding how to use these loops effectively is crucial for writing efficient and readable code.

Types of Loops in Ada

Basic Loop

The basic loop in Ada is an infinite loop that continues until explicitly terminated using an exit statement.

procedure Basic_Loop is
begin
   loop
      -- Your code here
      exit when Some_Condition;
   end loop;
end Basic_Loop;
  • Key Concept: The exit when statement is used to break out of the loop when a specified condition is met.

While Loop

The while loop continues to execute as long as a specified condition is true.

procedure While_Loop is
   Counter : Integer := 0;
begin
   while Counter < 10 loop
      -- Your code here
      Counter := Counter + 1;
   end loop;
end While_Loop;
  • Key Concept: The condition is evaluated before each iteration. If the condition is false initially, the loop body may not execute at all.

For Loop

The for loop iterates over a range of values. It is commonly used for iterating a known number of times.

procedure For_Loop is
begin
   for I in 1 .. 10 loop
      -- Your code here
   end loop;
end For_Loop;
  • Key Concept: The loop variable I automatically takes on each value in the specified range.

Nested Loops

Loops can be nested within each other to handle more complex scenarios.

procedure Nested_Loops is
begin
   for I in 1 .. 5 loop
      for J in 1 .. 5 loop
         -- Your code here
      end loop;
   end loop;
end Nested_Loops;
  • Key Concept: Each loop maintains its own loop variable and scope.

Advanced Loop Concepts

Loop with Exit

Using exit statements within loops allows for more complex control flows.

procedure Loop_With_Exit is
begin
   for I in 1 .. 10 loop
      if I = 5 then
         exit;
      end if;
      -- Your code here
   end loop;
end Loop_With_Exit;
  • Key Concept: The exit statement can be used to break out of the loop based on a condition.

Loop with Continue

Ada does not have a built-in continue statement like some other languages, but similar behavior can be achieved using exit and goto.

procedure Loop_With_Continue is
begin
   for I in 1 .. 10 loop
      if I mod 2 = 0 then
         goto Continue_Loop;
      end if;
      -- Your code here
      <<Continue_Loop>>;
   end loop;
end Loop_With_Continue;
  • Key Concept: Using goto to skip the rest of the loop body and continue with the next iteration.

Exercises

  1. Basic Loop Exercise: Write a basic loop that prints numbers from 1 to 10.
  2. While Loop Exercise: Create a while loop that prints even numbers between 1 and 20.
  3. For Loop Exercise: Write a for loop that calculates the factorial of a given number.
  4. Nested Loop Exercise: Implement nested loops to print a multiplication table for numbers 1 to 5.
  5. Advanced Loop Exercise: Write a loop that skips printing numbers divisible by 3 and exits when the number is greater than 15.

Conclusion

Loops are essential constructs in Ada that allow you to perform repetitive tasks efficiently. By mastering basic loops, while loops, for loops, and nested loops, you can handle a wide range of programming scenarios. Advanced concepts like using exit and simulating continue provide additional control over loop execution. Practice these concepts with the provided exercises to solidify your understanding.
