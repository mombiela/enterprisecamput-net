Setting Up the Development Environment
Your First Ada Program ►

Setting up the development environment is a crucial first step in learning Ada. This section will guide you through the process of installing the necessary tools and configuring your system to start writing and running Ada programs.

Installing the Ada Compiler

To write and run Ada programs, you need a compiler. The most widely used Ada compiler is GNAT, which is part of the GNU Compiler Collection (GCC).

Steps to Install GNAT

  1. Download GNAT:

    • Visit the AdaCore website to download the GNAT Community Edition.
    • Choose the appropriate version for your operating system (Windows, macOS, or Linux).

  2. Install GNAT:

    • Windows: Run the installer and follow the on-screen instructions.
    • macOS: Open the downloaded .dmg file and drag the GNAT folder to your Applications directory.
    • Linux: Extract the downloaded tarball and follow the instructions in the README file.

  3. Verify Installation:

    • Open a terminal or command prompt.
    • Type gnat --version and press Enter. You should see the version information for GNAT.
$ gnat --version
GNAT 11.2.0

Setting Up an Integrated Development Environment (IDE)

While you can write Ada code in any text editor, using an Integrated Development Environment (IDE) can significantly enhance your productivity. Two popular IDEs for Ada are GNAT Studio and Visual Studio Code.

GNAT Studio

  1. Download GNAT Studio:

    • GNAT Studio is included with the GNAT Community Edition. If you have already installed GNAT, you should have GNAT Studio as well.

  2. Launch GNAT Studio:

    • Open GNAT Studio from your Applications directory or by typing gnatstudio in your terminal/command prompt.

  3. Configure GNAT Studio:

    • Set up your project by creating a new project file (.gpr).
    • Configure the project settings such as source directories and compiler options.

Visual Studio Code

  1. Install Visual Studio Code:

    • Download and install Visual Studio Code from the official website.

  2. Install Ada Extensions:

    • Open Visual Studio Code.
    • Go to the Extensions view by clicking the Extensions icon in the Activity Bar on the side of the window.
    • Search for "Ada" and install the AdaCore Ada Language Support extension.

  3. Configure Visual Studio Code:

    • Create a new workspace and add your Ada project directory.
    • Configure tasks and launch settings for building and running Ada programs.

Writing Your First Ada Program

Once your development environment is set up, you can write and run your first Ada program.

Example: Hello World

  1. Create a new file:

    • Open your IDE and create a new file named hello.adb.

  2. Write the code:

    • Enter the following code into hello.adb:
with Ada.Text_IO; use Ada.Text_IO;

procedure Hello is
begin
   Put_Line("Hello, World!");
end Hello;
  1. Compile the program:
    • Open a terminal or use the build command in your IDE.
    • Navigate to the directory containing hello.adb.
    • Run the following command to compile the program:
$ gnatmake hello.adb
  1. Run the program:
    • Execute the compiled program by typing:
$ ./hello
Hello, World!

Conclusion

Setting up the development environment for Ada involves installing the GNAT compiler, choosing and configuring an IDE, and writing your first program. By following these steps, you will be ready to start your journey into Ada programming. Remember, a well-configured environment can greatly enhance your coding experience and productivity.
