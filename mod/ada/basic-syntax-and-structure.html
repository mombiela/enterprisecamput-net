Introduction

Ada is a structured, statically typed, imperative, and object-oriented high-level programming language. It is designed for large, long-lived applications where reliability and efficiency are essential. This section will cover the basic syntax and structure of Ada programs, providing a foundation for further learning.

Key Concepts

  • Program Structure: Understanding the basic layout of an Ada program.
  • Comments: How to write comments in Ada.
  • Identifiers: Rules for naming variables, constants, and other entities.
  • Data Types: Basic data types available in Ada.
  • Control Structures: Basic control flow mechanisms like loops and conditionals.

Program Structure

An Ada program consists of one or more packages, procedures, and functions. The basic structure of an Ada program is as follows:

with Ada.Text_IO; use Ada.Text_IO;

procedure Hello is
begin
   Put_Line("Hello, World!");
end Hello;
  • with Ada.Text_IO; use Ada.Text_IO;: These lines import the Ada.Text_IO package, which provides basic input/output functionalities.
  • procedure Hello is: This line declares a procedure named Hello.
  • begin ... end Hello;: This block contains the executable statements of the procedure.

Comments

Comments in Ada are written using -- for single-line comments. There are no multi-line comment delimiters in Ada.

-- This is a single-line comment
Put_Line("Hello, World!"); -- This is also a comment

Identifiers

Identifiers in Ada are names given to various program elements like variables, constants, procedures, etc. The rules for naming identifiers are:

  • Must start with a letter.
  • Can contain letters, digits, and underscores.
  • Are case-insensitive.
My_Variable : Integer := 10;
anotherVariable : Integer := 20;

Data Types

Ada supports several basic data types, including:

  • Integer: Whole numbers.
  • Float: Floating-point numbers.
  • Boolean: True or False values.
  • Character: Single characters.
  • String: Sequences of characters.

Example:

My_Integer : Integer := 42;
My_Float : Float := 3.14;
My_Boolean : Boolean := True;
My_Character : Character := 'A';
My_String : String := "Hello, Ada!";

Control Structures

Ada provides standard control structures for decision-making and looping.

If Statements

if My_Integer > 10 then
   Put_Line("Greater than 10");
elsif My_Integer = 10 then
   Put_Line("Equal to 10");
else
   Put_Line("Less than 10");
end if;

Case Statements

case My_Integer is
   when 1 => Put_Line("One");
   when 2 => Put_Line("Two");
   when others => Put_Line("Other");
end case;

Loops

While Loop

while My_Integer < 50 loop
   My_Integer := My_Integer + 1;
end loop;

For Loop

for I in 1 .. 10 loop
   Put_Line(Integer'Image(I));
end loop;

Summary

In this section, we covered the basic syntax and structure of Ada programs. We discussed the overall program layout, how to write comments, rules for naming identifiers, basic data types, and control structures such as if statements, case statements, and loops. Understanding these fundamentals is crucial for writing and comprehending more complex Ada programs.
