Introduction to Ada

Ada is a statically typed, structured, imperative, and object-oriented high-level computer programming language. It was designed by Jean Ichbiah and his team at CII Honeywell Bull under contract to the United States Department of Defense from 1977 to 1983 to supersede the hundreds of programming languages then used by the DoD. Ada is named after Ada Lovelace, who is often credited as the first computer programmer.

Setting Up Your Ada Environment

Before writing your first Ada program, you need to set up your development environment. Here are the steps:

  1. Install GNAT (GNU NYU Ada Translator):

    • GNAT is a free, high-quality, complete compiler for Ada.
    • You can download it from AdaCore or install it via your package manager if you are using Linux.

  2. Set Up Your IDE:

    • You can use any text editor or IDE, but some popular choices include GNAT Studio, Visual Studio Code with Ada extensions, and Emacs with Ada mode.

Writing Your First Ada Program

Let's write a simple Ada program that prints "Hello, World!" to the console.

Key Concepts

  • Program Structure: Ada programs are composed of packages, procedures, and functions.
  • Main Procedure: The entry point of an Ada program is a procedure named main.

Example Code

Here is a simple "Hello, World!" program in Ada:

-- This is a comment in Ada
with Ada.Text_IO; -- This line makes the Ada.Text_IO package available

procedure Hello is
begin
   Ada.Text_IO.Put_Line("Hello, World!");
end Hello;

Explanation

  • with Ada.Text_IO;: This line includes the Ada.Text_IO package, which contains procedures for input and output operations.
  • procedure Hello is: This defines a procedure named Hello. In Ada, procedures are similar to functions but do not return a value.
  • begin ... end Hello;: This block contains the executable statements of the procedure. The begin keyword marks the start of the block, and end Hello; marks the end.
  • Ada.Text_IO.Put_Line("Hello, World!");: This line calls the Put_Line procedure from the Ada.Text_IO package to print "Hello, World!" to the console.

Compiling and Running Your Program

To compile and run your Ada program, follow these steps:

  1. Save Your Code: Save your code in a file named hello.adb.

  2. Compile: Open your terminal and navigate to the directory containing hello.adb. Run the following command:

    gnatmake hello.adb

    This will compile your program and generate an executable file named hello.

  3. Run: Execute the compiled program by running:

    ./hello

    You should see the output:

    Hello, World!

Summary

In this section, you learned how to:

  • Set up your Ada development environment.
  • Write a simple Ada program.
  • Compile and run your Ada program.

This foundational knowledge will help you as you progress through more advanced topics in Ada programming. Remember, practice is key to becoming proficient, so try modifying the "Hello, World!" program and experiment with different features of the Ada language.
