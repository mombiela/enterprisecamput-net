Introduction to Access Types

Access types in Ada, commonly known as pointers in other programming languages, are used to reference dynamically allocated objects. They provide a way to create and manipulate data structures such as linked lists, trees, and graphs.

Key Concepts

  • Access Type Declaration: Defines a type that can hold references to objects of a specified type.
  • Dynamic Allocation: Allocates memory at runtime using the new keyword.
  • Dereferencing: Accesses the value stored at the location pointed to by an access type.
  • Null Access: Represents an access type that does not point to any object.

Declaring Access Types

To declare an access type, you use the access keyword followed by the type of object it will point to.

type Int_Ptr is access Integer;

In this example, Int_Ptr is an access type that can point to an Integer.

Dynamic Allocation

Dynamic allocation in Ada is performed using the new keyword. This allocates memory for an object and returns an access value pointing to it.

declare
   Ptr : Int_Ptr := new Integer'(10);
begin
   -- Ptr now points to an Integer with value 10
end;

Dereferencing Access Types

To access the value stored at the location pointed to by an access type, you use the .all notation.

declare
   Ptr : Int_Ptr := new Integer'(10);
begin
   Put_Line(Integer'Image(Ptr.all)); -- Outputs: 10
   Ptr.all := 20;
   Put_Line(Integer'Image(Ptr.all)); -- Outputs: 20
end;

Null Access

An access type can be explicitly set to null, indicating that it does not point to any object.

declare
   Ptr : Int_Ptr := null;
begin
   if Ptr = null then
      Put_Line("Pointer is null");
   end if;
end;

Example: Linked List

Let's create a simple linked list using access types.

with Ada.Text_IO; use Ada.Text_IO;

procedure Linked_List_Example is
   type Node;
   type Node_Ptr is access Node;
   
   type Node is
      record
         Value : Integer;
         Next  : Node_Ptr;
      end record;
   
   Head : Node_Ptr := null;
   
   procedure Add_Node(Value : Integer) is
      New_Node : Node_Ptr := new Node'(Value, null);
   begin
      if Head = null then
         Head := New_Node;
      else
         New_Node.Next := Head;
         Head := New_Node;
      end if;
   end Add_Node;
   
   procedure Print_List is
      Current : Node_Ptr := Head;
   begin
      while Current /= null loop
         Put_Line(Integer'Image(Current.Value));
         Current := Current.Next;
      end loop;
   end Print_List;
   
begin
   Add_Node(10);
   Add_Node(20);
   Add_Node(30);
   
   Print_List; -- Outputs: 30 20 10
end Linked_List_Example;

Summary

Access types in Ada are powerful tools for managing dynamic memory and creating complex data structures. Key points to remember:

  • Access types are declared using the access keyword.
  • Dynamic memory allocation is done using the new keyword.
  • Dereferencing is performed using the .all notation.
  • Access types can be set to null to indicate they do not point to any object.

Understanding and using access types effectively allows you to build efficient and flexible programs in Ada.
