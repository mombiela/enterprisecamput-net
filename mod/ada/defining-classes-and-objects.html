Defining Classes and Objects in Ada
Inheritance ►

Introduction

In this section, we will explore how to define classes and objects in Ada. Ada is a statically-typed, high-level programming language that supports object-oriented programming (OOP). Understanding how to define and use classes and objects is fundamental to leveraging the full power of Ada.

Key Concepts

  • Class: In Ada, a class is defined using tagged types. Tagged types allow for inheritance and polymorphism.
  • Object: An instance of a class (tagged type) is called an object.
  • Encapsulation: Encapsulation is achieved by defining private components within a package.
  • Inheritance: Inheritance is supported through type extension.
  • Polymorphism: Polymorphism is achieved using class-wide types and dispatching operations.

Defining a Class (Tagged Type)

In Ada, a class is defined using a tagged type. Here is a simple example:

package Shape_Pkg is
    type Shape is tagged record
        X, Y : Integer;
    end record;

    procedure Move (S : in out Shape; New_X, New_Y : Integer);
end Shape_Pkg;

package body Shape_Pkg is
    procedure Move (S : in out Shape; New_X, New_Y : Integer) is
    begin
        S.X := New_X;
        S.Y := New_Y;
    end Move;
end Shape_Pkg;

Explanation

  • Package: The Shape_Pkg package encapsulates the definition of the Shape type and its operations.
  • Tagged Type: Shape is a tagged type, which means it can be extended.
  • Procedure: The Move procedure modifies the X and Y coordinates of a Shape object.

Creating and Using Objects

To create and use objects of the Shape class, you can do the following:

with Shape_Pkg;
use Shape_Pkg;

procedure Main is
    My_Shape : Shape := (X => 0, Y => 0);
begin
    Move(My_Shape, 10, 20);
    -- My_Shape now has coordinates (10, 20)
end Main;

Explanation

  • Object Creation: My_Shape is an object of type Shape initialized with coordinates (0, 0).
  • Using Procedures: The Move procedure is called to change the coordinates of My_Shape.

Inheritance and Type Extension

Ada supports inheritance through type extension. Here is an example of extending the Shape type:

package Circle_Pkg is
    type Circle is new Shape with record
        Radius : Integer;
    end record;

    procedure Resize (C : in out Circle; New_Radius : Integer);
end Circle_Pkg;

package body Circle_Pkg is
    procedure Resize (C : in out Circle; New_Radius : Integer) is
    begin
        C.Radius := New_Radius;
    end Resize;
end Circle_Pkg;

Explanation

  • Type Extension: Circle is a new type that extends Shape by adding a Radius field.
  • Procedure: The Resize procedure modifies the Radius of a Circle object.

Polymorphism

Polymorphism in Ada is achieved using class-wide types and dispatching operations. Here is an example:

procedure Draw (S : in out Shape'Class) is
begin
    -- Dispatching operation based on the actual type of S
end Draw;

procedure Main is
    My_Circle : Circle := (X => 0, Y => 0, Radius => 5);
    My_Shape : Shape'Class := My_Circle;
begin
    Draw(My_Shape);
    -- The Draw procedure will dispatch to the appropriate operation for Circle
end Main;

Explanation

  • Class-Wide Type: Shape'Class is a class-wide type that can hold any type derived from Shape.
  • Dispatching Operation: The Draw procedure will call the appropriate operation based on the actual type of S.

Conclusion

In this section, we have covered the basics of defining classes and objects in Ada. We explored:

  • How to define a class using tagged types.
  • How to create and use objects.
  • How to extend types to achieve inheritance.
  • How to use polymorphism with class-wide types and dispatching operations.

Understanding these concepts is crucial for effective object-oriented programming in Ada. Practice by defining your own classes and objects to solidify your understanding.
