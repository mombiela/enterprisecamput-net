Introduction

Operators and expressions are fundamental components of any programming language. In Ada, understanding how to use these effectively is crucial for writing clear and efficient code. This section will cover the various types of operators available in Ada, how to use them in expressions, and provide examples to illustrate their usage.

Arithmetic Operators

Arithmetic operators are used to perform mathematical operations.

  • Addition (+): Adds two operands.
  • Subtraction (-): Subtracts the second operand from the first.
  • Multiplication (*): Multiplies two operands.
  • Division (/): Divides the first operand by the second.
  • Modulus (mod): Returns the remainder of a division.
  • Exponentiation (**): Raises a number to the power of another number.

Examples

with Ada.Text_IO; use Ada.Text_IO;

procedure Arithmetic_Operators is
   A, B : Integer := 10;
   C    : Integer;
   D    : Float := 2.5;
   E    : Float;
begin
   C := A + B;  -- Addition
   Put_Line("A + B = " & Integer'Image(C));
   
   C := A - B;  -- Subtraction
   Put_Line("A - B = " & Integer'Image(C));
   
   C := A * B;  -- Multiplication
   Put_Line("A * B = " & Integer'Image(C));
   
   E := Float(A) / D;  -- Division
   Put_Line("A / D = " & Float'Image(E));
   
   C := A mod B;  -- Modulus
   Put_Line("A mod B = " & Integer'Image(C));
   
   E := D ** 2;  -- Exponentiation
   Put_Line("D ** 2 = " & Float'Image(E));
end Arithmetic_Operators;

Relational Operators

Relational operators are used to compare two values.

  • Equal to (=): Checks if two operands are equal.
  • Not equal to (/=): Checks if two operands are not equal.
  • Greater than (>): Checks if the first operand is greater than the second.
  • Less than (<): Checks if the first operand is less than the second.
  • Greater than or equal to (>=): Checks if the first operand is greater than or equal to the second.
  • Less than or equal to (<=): Checks if the first operand is less than or equal to the second.

Examples

with Ada.Text_IO; use Ada.Text_IO;

procedure Relational_Operators is
   A, B : Integer := 10;
   C    : Integer := 20;
   Result : Boolean;
begin
   Result := A = B;  -- Equal to
   Put_Line("A = B: " & Boolean'Image(Result));
   
   Result := A /= B;  -- Not equal to
   Put_Line("A /= B: " & Boolean'Image(Result));
   
   Result := C > A;  -- Greater than
   Put_Line("C > A: " & Boolean'Image(Result));
   
   Result := A < C;  -- Less than
   Put_Line("A < C: " & Boolean'Image(Result));
   
   Result := A >= B;  -- Greater than or equal to
   Put_Line("A >= B: " & Boolean'Image(Result));
   
   Result := A <= C;  -- Less than or equal to
   Put_Line("A <= C: " & Boolean'Image(Result));
end Relational_Operators;

Logical Operators

Logical operators are used to perform logical operations, often with Boolean values.

  • AND (and): Returns true if both operands are true.
  • OR (or): Returns true if at least one operand is true.
  • NOT (not): Returns the opposite of the operand.

Examples

with Ada.Text_IO; use Ada.Text_IO;

procedure Logical_Operators is
   A, B : Boolean := True;
   C    : Boolean := False;
   Result : Boolean;
begin
   Result := A and B;  -- AND
   Put_Line("A and B: " & Boolean'Image(Result));
   
   Result := A or C;  -- OR
   Put_Line("A or C: " & Boolean'Image(Result));
   
   Result := not C;  -- NOT
   Put_Line("not C: " & Boolean'Image(Result));
end Logical_Operators;

Assignment Operators

Assignment operators are used to assign values to variables.

  • Assignment (:=): Assigns the value on the right to the variable on the left.

Examples

with Ada.Text_IO; use Ada.Text_IO;

procedure Assignment_Operators is
   A : Integer;
begin
   A := 10;  -- Assignment
   Put_Line("A := 10: " & Integer'Image(A));
end Assignment_Operators;

Combined Assignment Operators

Combined assignment operators perform an operation and assign the result to a variable in one step.

  • Add and assign (+=): Adds the right operand to the left operand and assigns the result to the left operand.
  • Subtract and assign (-=): Subtracts the right operand from the left operand and assigns the result to the left operand.
  • Multiply and assign (*=): Multiplies the left operand by the right operand and assigns the result to the left operand.
  • Divide and assign (/=): Divides the left operand by the right operand and assigns the result to the left operand.

Examples

with Ada.Text_IO; use Ada.Text_IO;

procedure Combined_Assignment_Operators is
   A : Integer := 10;
begin
   A := A + 5;  -- Add and assign
   Put_Line("A += 5: " & Integer'Image(A));
   
   A := A - 3;  -- Subtract and assign
   Put_Line("A -= 3: " & Integer'Image(A));
   
   A := A * 2;  -- Multiply and assign
   Put_Line("A *= 2: " & Integer'Image(A));
   
   A := A / 2;  -- Divide and assign
   Put_Line("A /= 2: " & Integer'Image(A));
end Combined_Assignment_Operators;

Summary

In this section, we covered the various types of operators available in Ada, including arithmetic, relational, logical, assignment, and combined assignment operators. Understanding these operators and how to use them in expressions is essential for writing effective Ada programs. Practice using these operators in different contexts to become more comfortable with their syntax and functionality.
