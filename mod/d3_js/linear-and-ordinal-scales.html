Linear and Ordinal Scales

In this section, we will explore the concepts of linear and ordinal scales in D3.js. Scales are a fundamental part of D3.js, as they allow you to map data values to visual dimensions. Understanding how to use scales effectively is crucial for creating meaningful visualizations.

Linear Scales

Linear scales are used to map continuous input domains to continuous output ranges. They are particularly useful when dealing with numerical data that needs to be represented proportionally.

Key Concepts

  • Domain: The set of input values.
  • Range: The set of output values.
  • Mapping: The process of converting a value from the domain to the range.

Example

Let's create a simple linear scale that maps a domain of [0, 100] to a range of [0, 500].

// Create a linear scale
const linearScale = d3.scaleLinear()
    .domain([0, 100])
    .range([0, 500]);

// Test the scale
console.log(linearScale(0));   // Output: 0
console.log(linearScale(50));  // Output: 250
console.log(linearScale(100)); // Output: 500

Explanation

  • d3.scaleLinear(): Creates a new linear scale.
  • domain([0, 100]): Sets the input domain to range from 0 to 100.
  • range([0, 500]): Sets the output range to range from 0 to 500.

Ordinal Scales

Ordinal scales are used to map discrete input domains to discrete output ranges. They are useful when dealing with categorical data.

Key Concepts

  • Domain: The set of discrete input values.
  • Range: The set of discrete output values.
  • Mapping: The process of converting a value from the domain to the range.

Example

Let's create an ordinal scale that maps a domain of ["A", "B", "C"] to a range of [0, 100, 200].

// Create an ordinal scale
const ordinalScale = d3.scaleOrdinal()
    .domain(["A", "B", "C"])
    .range([0, 100, 200]);

// Test the scale
console.log(ordinalScale("A")); // Output: 0
console.log(ordinalScale("B")); // Output: 100
console.log(ordinalScale("C")); // Output: 200

Explanation

  • d3.scaleOrdinal(): Creates a new ordinal scale.
  • domain(["A", "B", "C"]): Sets the input domain to the discrete values "A", "B", and "C".
  • range([0, 100, 200]): Sets the output range to the discrete values 0, 100, and 200.

Comparison of Linear and Ordinal Scales

| Feature | Linear Scale | Ordinal Scale | |-----------------|------------------------------------|-------------------------------------| | Input Domain | Continuous (e.g., [0, 100]) | Discrete (e.g., ["A", "B", "C"]) | | Output Range | Continuous (e.g., [0, 500]) | Discrete (e.g., [0, 100, 200]) | | Use Case | Numerical data | Categorical data | | Mapping | Proportional | Non-proportional |

Conclusion

Understanding linear and ordinal scales is essential for creating effective visualizations in D3.js. Linear scales are ideal for numerical data, providing a proportional mapping from input to output. Ordinal scales, on the other hand, are perfect for categorical data, mapping discrete input values to discrete output values.

By mastering these scales, you can ensure that your data is accurately represented in your visualizations, making them more informative and engaging.
