Introduction

In TypeScript, functions are a fundamental building block of any application. Understanding how to define and use function types is crucial for writing robust and type-safe code. This section will cover the basics of function types, including how to define them, use them, and some advanced concepts.

Defining Function Types

In TypeScript, you can define the type of a function by specifying the types of its parameters and the type of its return value.

Basic Syntax

Here's the basic syntax for defining a function type:

type GreetFunction = (name: string) => string;

In this example:

  • GreetFunction is a type alias for a function that takes a string parameter and returns a string.

Example

Let's create a function that matches this type:

const greet: GreetFunction = (name: string) => {
    return `Hello, ${name}!`;
};

console.log(greet("Alice")); // Output: Hello, Alice!

Optional and Default Parameters

TypeScript allows you to define optional and default parameters in functions.

Optional Parameters

Optional parameters are denoted by a ? after the parameter name.

type GreetFunction = (name: string, age?: number) => string;

const greet: GreetFunction = (name, age) => {
    if (age) {
        return `Hello, ${name}. You are ${age} years old.`;
    }
    return `Hello, ${name}!`;
};

console.log(greet("Alice")); // Output: Hello, Alice!
console.log(greet("Bob", 30)); // Output: Hello, Bob. You are 30 years old.

Default Parameters

Default parameters are assigned a default value in the function signature.

type GreetFunction = (name: string, age?: number) => string;

const greet: GreetFunction = (name, age = 25) => {
    return `Hello, ${name}. You are ${age} years old.`;
};

console.log(greet("Alice")); // Output: Hello, Alice. You are 25 years old.
console.log(greet("Bob", 30)); // Output: Hello, Bob. You are 30 years old.

Function Types in Interfaces

You can also define function types within interfaces.

Example

interface MathOperations {
    add: (a: number, b: number) => number;
    subtract: (a: number, b: number) => number;
}

const operations: MathOperations = {
    add: (a, b) => a + b,
    subtract: (a, b) => a - b,
};

console.log(operations.add(5, 3)); // Output: 8
console.log(operations.subtract(5, 3)); // Output: 2

Function Overloads

Function overloads allow you to define multiple signatures for a single function.

Example

function add(a: number, b: number): number;
function add(a: string, b: string): string;
function add(a: any, b: any): any {
    return a + b;
}

console.log(add(5, 3)); // Output: 8
console.log(add("Hello, ", "world!")); // Output: Hello, world!

Conclusion

Understanding function types in TypeScript is essential for writing type-safe and maintainable code. By defining function types, using optional and default parameters, leveraging interfaces, and utilizing function overloads, you can create more robust and flexible functions. Practice these concepts with different examples to solidify your understanding and become proficient in TypeScript function types.
