Inheritance in TypeScript
Access Modifiers ►

Inheritance is a fundamental concept in object-oriented programming that allows a class to inherit properties and methods from another class. In TypeScript, inheritance helps in creating a hierarchy of classes that share a common structure and behavior, promoting code reuse and maintainability.

Key Concepts of Inheritance

  • Base Class (Parent Class): The class whose properties and methods are inherited.
  • Derived Class (Child Class): The class that inherits properties and methods from the base class.
  • extends Keyword: Used to create a derived class from a base class.
  • Method Overriding: The ability of a derived class to provide a specific implementation of a method that is already defined in its base class.
  • super Keyword: Used to call the constructor or methods of the base class from the derived class.

Creating a Base Class

Let's start by defining a simple base class called Animal.

class Animal {
    name: string;

    constructor(name: string) {
        this.name = name;
    }

    makeSound(): void {
        console.log(`${this.name} makes a sound.`);
    }
}

In this example:

  • The Animal class has a property name and a method makeSound.
  • The constructor initializes the name property.

Creating a Derived Class

Now, let's create a derived class Dog that inherits from Animal.

class Dog extends Animal {
    breed: string;

    constructor(name: string, breed: string) {
        super(name); // Call the constructor of the base class
        this.breed = breed;
    }

    makeSound(): void {
        console.log(`${this.name} barks.`);
    }
}

In this example:

  • The Dog class extends the Animal class using the extends keyword.
  • The Dog class has an additional property breed.
  • The constructor of Dog calls the constructor of Animal using super(name).
  • The makeSound method is overridden to provide a specific implementation for Dog.

Using the Derived Class

Let's create an instance of the Dog class and use its methods.

const myDog = new Dog('Buddy', 'Golden Retriever');
myDog.makeSound(); // Output: Buddy barks.
console.log(`My dog is a ${myDog.breed}.`); // Output: My dog is a Golden Retriever.

Method Overriding

Method overriding allows a derived class to provide a specific implementation of a method that is already defined in its base class. This is useful when the derived class needs to change or extend the behavior of the base class method.

class Cat extends Animal {
    constructor(name: string) {
        super(name);
    }

    makeSound(): void {
        console.log(`${this.name} meows.`);
    }
}

const myCat = new Cat('Whiskers');
myCat.makeSound(); // Output: Whiskers meows.

Summary

  • Inheritance allows a class to inherit properties and methods from another class, promoting code reuse.
  • The extends keyword is used to create a derived class from a base class.
  • The super keyword is used to call the constructor or methods of the base class.
  • Method overriding allows a derived class to provide a specific implementation of a method defined in the base class.

By understanding and using inheritance in TypeScript, you can create more structured and maintainable code, leveraging the power of object-oriented programming principles.
