Introduction to AWS Free Tier

The AWS Free Tier is a program that offers free access to a variety of AWS services for a limited time. This allows users to explore and experiment with AWS services without incurring costs. The Free Tier is divided into three categories:

  • Always Free: Services that are free indefinitely.
  • 12 Months Free: Services that are free for the first 12 months after signing up.
  • Trials: Short-term free trials for specific services.

Key Concepts

Always Free Services

These services are available for free indefinitely, making them ideal for long-term projects with minimal usage.

  • Amazon DynamoDB: 25 GB of storage, 25 Write Capacity Units, and 25 Read Capacity Units.
  • AWS Lambda: 1 million free requests per month and 400,000 GB-seconds of compute time per month.
  • Amazon SNS (Simple Notification Service): 1 million mobile push notifications and 1,000 email notifications per month.

12 Months Free Services

These services are free for the first 12 months after you sign up for AWS.

  • Amazon EC2 (Elastic Compute Cloud): 750 hours of t2.micro or t3.micro instances per month.
  • Amazon S3 (Simple Storage Service): 5 GB of standard storage.
  • Amazon RDS (Relational Database Service): 750 hours of db.t2.micro or db.t3.micro instances per month.

Trials

Short-term trials for specific services, usually lasting 30 to 60 days.

  • Amazon Redshift: 2 months free usage of dc2.large node.
  • Amazon QuickSight: 60-day free trial for up to 4 users.

Examples and Code

Example: Launching an EC2 Instance

Let's walk through launching an EC2 instance using the AWS Management Console.

  1. Sign in to AWS Management Console.
  2. Navigate to EC2 Dashboard.
  3. Click "Launch Instance".
  4. Choose an Amazon Machine Image (AMI): Select the "Amazon Linux 2 AMI".
  5. Choose an Instance Type: Select "t2.micro" which is free tier eligible.
  6. Configure Instance Details: Leave default settings.
  7. Add Storage: Leave default settings.
  8. Add Tags: Optionally add tags.
  9. Configure Security Group: Create a new security group with SSH access.
  10. Review and Launch: Review your settings and click "Launch".

Example: Using AWS Lambda

Here's a simple example of creating a Lambda function using Python.

  1. Sign in to AWS Management Console.
  2. Navigate to Lambda Dashboard.
  3. Click "Create Function".
  4. Choose "Author from scratch".
  5. Configure Function:
    • Function name: MyFirstLambda
    • Runtime: Python 3.8
  6. Click "Create Function".
  7. Add Code: 
    def lambda_handler(event, context):
    return {
        'statusCode': 200,
        'body': 'Hello from Lambda!'
    }
  8. Click "Deploy".

Summary

The AWS Free Tier is an excellent way to get started with AWS services without incurring costs. It includes a variety of services that are always free, free for the first 12 months, or available as short-term trials. By understanding and utilizing these free offerings, you can explore AWS capabilities and build your skills without financial risk. Whether you're launching an EC2 instance or creating a Lambda function, the Free Tier provides a valuable opportunity to learn and experiment.
