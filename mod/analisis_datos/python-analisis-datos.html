Introduction

Data analysis is a fundamental discipline in the information age. Python has become an essential tool for data analysts due to its simplicity and the power of its libraries. This topic will cover the basic and advanced concepts of using Python for data analysis.

Installation and Setup

Before starting with data analysis in Python, it is necessary to set up the working environment.

Installing Python

  • Download and install Python from python.org.
  • Verify the installation by running python --version in the terminal.

Installing Essential Libraries

For data analysis, we will use several popular Python libraries:

pip install numpy pandas matplotlib seaborn
  • NumPy: Fundamental library for numerical computation.
  • Pandas: Library for data manipulation and analysis.
  • Matplotlib: Library for data visualization.
  • Seaborn: Library for data visualization based on Matplotlib.

Data Manipulation with Pandas

Pandas is a powerful library for data manipulation and analysis. It provides data structures such as Series and DataFrames.

Series

A Series is a one-dimensional data structure similar to a list.

import pandas as pd

# Create a Series
data = [1, 2, 3, 4, 5]
series = pd.Series(data)
print(series)

DataFrames

A DataFrame is a two-dimensional data structure similar to a table.

# Create a DataFrame
data = {
    'Name': ['Ana', 'Luis', 'Carlos'],
    'Age': [28, 34, 29],
    'City': ['Madrid', 'Barcelona', 'Valencia']
}
df = pd.DataFrame(data)
print(df)

Basic Operations with DataFrames

  • Selecting columns:
print(df['Name'])
  • Filtering rows:
print(df[df['Age'] > 30])
  • Adding new columns:
df['Salary'] = [3000, 4000, 3500]
print(df)

Statistical Analysis with NumPy

NumPy provides support for large multidimensional arrays and high-level mathematical functions.

Creating Arrays

import numpy as np

# Create an array
array = np.array([1, 2, 3, 4, 5])
print(array)

Basic Operations

  • Sum:
print(np.sum(array))
  • Mean:
print(np.mean(array))
  • Standard deviation:
print(np.std(array))

Data Visualization

Data visualization is crucial for interpreting and communicating the results of the analysis.

Plots with Matplotlib

import matplotlib.pyplot as plt

# Create a line plot
plt.plot([1, 2, 3, 4, 5], [1, 4, 9, 16, 25])
plt.xlabel('X')
plt.ylabel('Y')
plt.title('Line Plot')
plt.show()

Plots with Seaborn

import seaborn as sns

# Create a scatter plot
sns.scatterplot(x='Age', y='Salary', data=df)
plt.show()

Practical Exercises

  2. Operations with DataFrames: Filter students with grades higher than 7.
  3. Visualization: Create a bar chart showing the number of students by grade.

Conclusion

Python offers a robust set of tools for data analysis. From data manipulation with Pandas, through statistical analysis with NumPy, to visualization with Matplotlib and Seaborn, Python facilitates the process of turning data into useful information. Practicing with these examples and exercises will help consolidate the acquired knowledge.
