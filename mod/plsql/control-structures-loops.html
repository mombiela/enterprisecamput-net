Introduction

In PL/SQL, loops are essential control structures that allow you to execute a sequence of statements multiple times. This topic will cover the different types of loops available in PL/SQL, how to use them, and provide examples and exercises to help you understand their application.

Types of Loops in PL/SQL

PL/SQL supports several types of loops:

  • Basic Loop
  • WHILE Loop
  • FOR Loop
  • Cursor FOR Loop

Basic Loop

A basic loop repeatedly executes a block of code until an EXIT statement is encountered.

Syntax

LOOP
    -- Statements to be executed
    EXIT WHEN condition;
END LOOP;

Example

DECLARE
    counter NUMBER := 1;
BEGIN
    LOOP
        DBMS_OUTPUT.PUT_LINE('Counter: ' || counter);
        counter := counter + 1;
        EXIT WHEN counter > 5;
    END LOOP;
END;
/

In this example, the loop prints the value of counter and increments it by 1 until counter exceeds 5.

WHILE Loop

A WHILE loop executes a block of code as long as a specified condition is true.

Syntax

WHILE condition LOOP
    -- Statements to be executed
END LOOP;

Example

DECLARE
    counter NUMBER := 1;
BEGIN
    WHILE counter <= 5 LOOP
        DBMS_OUTPUT.PUT_LINE('Counter: ' || counter);
        counter := counter + 1;
    END LOOP;
END;
/

This example is similar to the basic loop but uses a WHILE loop to check the condition before each iteration.

FOR Loop

A FOR loop iterates over a specified range of values.

Syntax

FOR counter IN lower_bound..upper_bound LOOP
    -- Statements to be executed
END LOOP;

Example

BEGIN
    FOR counter IN 1..5 LOOP
        DBMS_OUTPUT.PUT_LINE('Counter: ' || counter);
    END LOOP;
END;
/

In this example, the loop automatically increments the counter from 1 to 5.

Cursor FOR Loop

A cursor FOR loop iterates over the rows returned by a query.

Syntax

FOR record IN cursor_name LOOP
    -- Statements to be executed
END LOOP;

Example

DECLARE
    CURSOR emp_cursor IS
        SELECT employee_id, first_name, last_name FROM employees;
BEGIN
    FOR emp_record IN emp_cursor LOOP
        DBMS_OUTPUT.PUT_LINE('Employee ID: ' || emp_record.employee_id || 
                             ', Name: ' || emp_record.first_name || ' ' || emp_record.last_name);
    END LOOP;
END;
/

This example iterates over each row in the employees table and prints the employee's ID and name.

Comparison of Loop Types

| Loop Type | Condition Check | Use Case | |-----------------|-----------------|-----------------------------------------------| | Basic Loop | Inside loop | When the number of iterations is not known | | WHILE Loop | Before loop | When the number of iterations is not known | | FOR Loop | Before loop | When the number of iterations is known | | Cursor FOR Loop | Before loop | When iterating over rows returned by a query |

Exercises

  1. Basic Loop Exercise: Write a PL/SQL block that uses a basic loop to print the numbers from 10 to 1.
  2. WHILE Loop Exercise: Write a PL/SQL block that uses a WHILE loop to print the first 10 even numbers.
  3. FOR Loop Exercise: Write a PL/SQL block that uses a FOR loop to print the squares of numbers from 1 to 5.
  4. Cursor FOR Loop Exercise: Write a PL/SQL block that uses a cursor FOR loop to print the department names from the departments table.

Conclusion

Loops are powerful control structures in PL/SQL that allow you to execute a block of code multiple times. Understanding the different types of loops and their use cases will enable you to write more efficient and effective PL/SQL code. Practice using each type of loop with the provided exercises to solidify your understanding.
