Introduction to CSS

CSS (Cascading Style Sheets) is a language used to describe the presentation of a document written in HTML or XML. CSS controls the layout, colors, fonts, and overall visual appearance of web pages.

Key Concepts of CSS

Selectors

Selectors are used to target HTML elements that you want to style.

  • Element Selector: Targets all instances of a specific HTML element. 
    p {
  color: blue;
}
  • Class Selector: Targets elements with a specific class attribute. 
    .example {
  color: red;
}
  • ID Selector: Targets a single element with a specific id attribute. 
    #unique {
  color: green;
}
  • Attribute Selector: Targets elements with a specific attribute. 
    [type="text"] {
  border: 1px solid black;
}

Properties and Values

CSS properties are used to define the styles applied to the selected elements.

  • Color: Sets the color of the text. 
    color: blue;
  • Font-size: Sets the size of the text. 
    font-size: 16px;
  • Margin: Sets the space outside the element. 
    margin: 10px;
  • Padding: Sets the space inside the element. 
    padding: 10px;

Cascading and Specificity

CSS follows a cascading order to determine which styles are applied when multiple rules match the same element.

  • Specificity: Determines which rule is applied by giving weight to different types of selectors.
    • Inline styles have the highest specificity.
    • ID selectors have higher specificity than class selectors.
    • Class selectors have higher specificity than element selectors.
  • Inheritance: Some properties are inherited from parent elements to child elements, such as color and font-family.

Box Model

The CSS box model describes the rectangular boxes generated for elements in the document tree.

  • Content: The actual content of the element.
  • Padding: Space between the content and the border.
  • Border: The border surrounding the padding (if any).
  • Margin: Space outside the border.
div {
  width: 100px;
  height: 100px;
  padding: 10px;
  border: 5px solid black;
  margin: 20px;
}

Positioning

CSS provides several ways to position elements on the web page.

  • Static: Default positioning, elements are positioned according to the normal flow of the document.
  • Relative: Positioned relative to its normal position. 
    .relative {
  position: relative;
  top: 10px;
  left: 10px;
}
  • Absolute: Positioned relative to the nearest positioned ancestor. 
    .absolute {
  position: absolute;
  top: 50px;
  left: 50px;
}
  • Fixed: Positioned relative to the viewport, stays in the same place even when the page is scrolled. 
    .fixed {
  position: fixed;
  top: 0;
  left: 0;
}
  • Sticky: Switches between relative and fixed positioning depending on the scroll position. 
    .sticky {
  position: sticky;
  top: 0;
}

Advanced CSS Concepts

Flexbox

Flexbox is a layout model that allows for the arrangement of elements in a predictable way.

  • Container Properties:
    • display: flex;
    • flex-direction: row | column;
    • justify-content: flex-start | center | space-between;
    • align-items: flex-start | center | stretch;
  • Item Properties:
    • flex-grow: 1;
    • flex-shrink: 1;
    • flex-basis: auto;
.container {
  display: flex;
  justify-content: space-between;
}

.item {
  flex-grow: 1;
}

Grid Layout

CSS Grid Layout is a two-dimensional layout system for the web.

  • Container Properties:
    • display: grid;
    • grid-template-columns: repeat(3, 1fr);
    • grid-template-rows: auto;
    • gap: 10px;
  • Item Properties:
    • grid-column: 1 / span 2;
    • grid-row: 1 / 2;
.grid-container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 10px;
}

.grid-item {
  grid-column: 1 / span 2;
}

Media Queries

Media queries allow you to apply styles based on the characteristics of the device, such as its width.

@media (max-width: 600px) {
  .responsive {
    font-size: 12px;
  }
}

Conclusion

Understanding how CSS works is fundamental for creating visually appealing and responsive web pages. By mastering selectors, properties, the box model, positioning, Flexbox, Grid Layout, and media queries, you can control the layout and design of your web projects effectively. Practice these concepts with real-world examples to solidify your knowledge and become proficient in CSS.
