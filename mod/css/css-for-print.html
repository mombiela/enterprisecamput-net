Introduction to CSS for Print

CSS for print is a specialized area of CSS that focuses on styling web content for printed media. While most CSS is designed for screen display, print stylesheets ensure that your content looks good when printed on paper.

Key Concepts

  • Media Queries for Print: Use media queries to apply styles specifically for print.
  • Page Breaks: Control where content breaks between pages.
  • Hiding Elements: Hide elements that are unnecessary or irrelevant in print.
  • Styling for Readability: Adjust fonts, colors, and layout for better readability on paper.

Media Queries for Print

Media queries allow you to apply CSS rules based on the type of media. For print styles, you use the @media print rule.

@media print {
  body {
    font-size: 12pt;
    line-height: 1.5;
  }
}

In this example:

  • The @media print rule targets print media.
  • The body text size is set to 12pt, and the line height is increased to 1.5 for better readability.

Page Breaks

Page breaks are crucial for ensuring that content is divided logically across pages. Use the break-before, break-after, and break-inside properties.

@media print {
  h1 {
    break-before: page;
  }
  p {
    break-inside: avoid;
  }
}

In this example:

  • break-before: page ensures that each h1 starts on a new page.
  • break-inside: avoid prevents paragraphs from being split across pages.

Hiding Elements

Some elements, like navigation bars or interactive content, are unnecessary in print. Use the display: none property to hide these elements.

@media print {
  .navbar, .footer, .interactive-content {
    display: none;
  }
}

In this example:

  • The .navbar, .footer, and .interactive-content classes are hidden in the print version.

Styling for Readability

Printed content should be easy to read. Adjust fonts, colors, and layout to enhance readability.

@media print {
  body {
    font-family: serif;
    color: black;
    background: white;
  }
  a {
    color: black;
    text-decoration: none;
  }
}

In this example:

  • The body font is changed to a serif font, which is generally easier to read on paper.
  • The text color is set to black, and the background is set to white.
  • Links are styled to look like regular text, as they are not clickable in print.

Summary

CSS for print is essential for ensuring that web content is presented effectively on paper. By using media queries, controlling page breaks, hiding unnecessary elements, and styling for readability, you can create print-friendly web pages. These techniques ensure that your content maintains its integrity and readability, whether viewed on a screen or printed out.
