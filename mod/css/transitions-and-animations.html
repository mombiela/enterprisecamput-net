Transitions and Animations
Transformations ►

CSS Transitions and Animations are powerful tools that allow developers to create dynamic and engaging web experiences. This topic will cover the basics of CSS transitions and animations, progressing to more advanced techniques.

Introduction to Transitions

CSS transitions allow you to change property values smoothly (over a given duration).

Key Concepts

  • Transition Properties: transition-property, transition-duration, transition-timing-function, transition-delay.
  • Shorthand Property: transition.

Basic Example

/* HTML */
<button class="btn">Hover me!</button>

/* CSS */
.btn {
  background-color: blue;
  color: white;
  padding: 10px 20px;
  border: none;
  cursor: pointer;
  transition: background-color 0.3s ease;
}

.btn:hover {
  background-color: green;
}

In this example:

  • The transition property is used to define the transition effect.
  • When the button is hovered over, the background color changes from blue to green over 0.3 seconds.

Transition Properties Explained

  • transition-property: Specifies the CSS property to apply the transition to.
  • transition-duration: Specifies the duration of the transition.
  • transition-timing-function: Specifies the speed curve of the transition.
  • transition-delay: Specifies a delay before the transition starts.

Transition Timing Functions

Timing Function Description
ease Default. Starts slow, then fast, then ends slow.
linear Same speed from start to end.
ease-in Starts slow.
ease-out Ends slow.
ease-in-out Starts and ends slow.

Introduction to Animations

CSS animations make it possible to animate transitions from one CSS style configuration to another.

Key Concepts

  • Animation Properties: animation-name, animation-duration, animation-timing-function, animation-delay, animation-iteration-count, animation-direction, animation-fill-mode, animation-play-state.
  • Keyframes: Define the styles for the element at various points during the animation sequence.

Basic Example

/* HTML */
<div class="box"></div>

/* CSS */
.box {
  width: 100px;
  height: 100px;
  background-color: red;
  position: relative;
  animation: move 2s infinite;
}

@keyframes move {
  0% { left: 0; }
  50% { left: 200px; }
  100% { left: 0; }
}

In this example:

  • The animation property is used to define the animation.
  • The @keyframes rule specifies the animation sequence.

Animation Properties Explained

  • animation-name: Specifies the name of the @keyframes animation.
  • animation-duration: Specifies the duration of the animation.
  • animation-timing-function: Specifies the speed curve of the animation.
  • animation-delay: Specifies a delay before the animation starts.
  • animation-iteration-count: Specifies the number of times the animation should be played.
  • animation-direction: Specifies whether the animation should play in reverse on alternate cycles.
  • animation-fill-mode: Specifies how a CSS animation should apply styles to its target before and after it is executing.
  • animation-play-state: Specifies whether the animation is running or paused.

Advanced Example

/* HTML */
<div class="circle"></div>

/* CSS */
.circle {
  width: 50px;
  height: 50px;
  background-color: blue;
  border-radius: 50%;
  position: relative;
  animation: bounce 2s infinite alternate ease-in-out;
}

@keyframes bounce {
  0% { top: 0; }
  100% { top: 200px; }
}

In this example:

  • The animation property includes multiple values: bounce (name), 2s (duration), infinite (iteration count), alternate (direction), and ease-in-out (timing function).
  • The @keyframes rule defines the bounce effect.

Conclusion

CSS transitions and animations are essential for creating interactive and visually appealing web pages. By understanding and utilizing these properties, you can enhance user experience and bring your web designs to life. Practice by experimenting with different properties and keyframes to see the effects they produce.
