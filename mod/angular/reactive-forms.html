Introduction to Reactive Forms

Reactive Forms in Angular provide a model-driven approach to handling form inputs whose values change over time. This approach offers more control and flexibility compared to template-driven forms.

  • Model-driven approach: Forms are created and managed in the component class.
  • Immutable data structures: Each change to the form state returns a new state.
  • Synchronous access to the form data: Form data is available immediately.

Setting Up Reactive Forms

To use Reactive Forms, you need to import the ReactiveFormsModule in your Angular module.

import { ReactiveFormsModule } from '@angular/forms';

@NgModule({
  imports: [
    ReactiveFormsModule,
    // other imports
  ],
  // other configurations
})
export class AppModule { }

Creating a Basic Reactive Form

Step 1: Import Form Classes

Import the necessary classes from @angular/forms.

import { FormGroup, FormControl } from '@angular/forms';

Step 2: Define the Form Model

Create a form model in your component class.

export class MyComponent {
  myForm: FormGroup;

  constructor() {
    this.myForm = new FormGroup({
      name: new FormControl(''),
      email: new FormControl(''),
    });
  }
}

Step 3: Bind the Form in the Template

Bind the form model to the template using Angular directives.

<form [formGroup]="myForm" (ngSubmit)="onSubmit()">
  <label for="name">Name:</label>
  <input id="name" formControlName="name">
  
  <label for="email">Email:</label>
  <input id="email" formControlName="email">
  
  <button type="submit">Submit</button>
</form>

Step 4: Handle Form Submission

Handle the form submission in your component class.

export class MyComponent {
  myForm: FormGroup;

  constructor() {
    this.myForm = new FormGroup({
      name: new FormControl(''),
      email: new FormControl(''),
    });
  }

  onSubmit() {
    console.log(this.myForm.value);
  }
}

Advanced Form Controls

FormBuilder

FormBuilder is a service that provides convenient methods to create form controls.

import { FormBuilder, FormGroup } from '@angular/forms';

export class MyComponent {
  myForm: FormGroup;

  constructor(private fb: FormBuilder) {
    this.myForm = this.fb.group({
      name: [''],
      email: [''],
    });
  }
}

FormArray

FormArray is used to manage an array of form controls.

import { FormBuilder, FormArray } from '@angular/forms';

export class MyComponent {
  myForm: FormGroup;

  constructor(private fb: FormBuilder) {
    this.myForm = this.fb.group({
      users: this.fb.array([]),
    });
  }

  get users() {
    return this.myForm.get('users') as FormArray;
  }

  addUser() {
    this.users.push(this.fb.group({
      name: [''],
      email: [''],
    }));
  }
}

Nested FormGroups

You can nest FormGroups to create complex forms.

export class MyComponent {
  myForm: FormGroup;

  constructor(private fb: FormBuilder) {
    this.myForm = this.fb.group({
      userDetails: this.fb.group({
        name: [''],
        email: [''],
      }),
      address: this.fb.group({
        street: [''],
        city: [''],
      }),
    });
  }
}

Form Validation

Built-in Validators

Angular provides several built-in validators.

import { Validators } from '@angular/forms';

export class MyComponent {
  myForm: FormGroup;

  constructor(private fb: FormBuilder) {
    this.myForm = this.fb.group({
      name: ['', Validators.required],
      email: ['', [Validators.required, Validators.email]],
    });
  }
}

Custom Validators

You can create custom validators for more complex validation logic.

import { AbstractControl, ValidationErrors } from '@angular/forms';

export function forbiddenNameValidator(control: AbstractControl): ValidationErrors | null {
  const forbidden = /admin/.test(control.value);
  return forbidden ? { forbiddenName: { value: control.value } } : null;
}

export class MyComponent {
  myForm: FormGroup;

  constructor(private fb: FormBuilder) {
    this.myForm = this.fb.group({
      name: ['', [Validators.required, forbiddenNameValidator]],
      email: ['', [Validators.required, Validators.email]],
    });
  }
}

Dynamic Forms

Dynamic forms allow you to create forms based on runtime data.

export class MyComponent {
  myForm: FormGroup;

  constructor(private fb: FormBuilder) {
    this.myForm = this.fb.group({});
  }

  createForm(fields: any[]) {
    fields.forEach(field => {
      this.myForm.addControl(field.name, new FormControl(''));
    });
  }
}

Conclusion

Reactive Forms in Angular provide a robust and flexible way to manage form inputs and validation. By leveraging the power of FormGroup, FormControl, FormArray, and FormBuilder, you can create complex and dynamic forms that are easy to maintain and extend. Understanding these concepts will help you build more efficient and scalable Angular applications.
