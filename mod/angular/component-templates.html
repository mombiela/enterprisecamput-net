Introduction to Component Templates

Component templates in Angular define the view for a component. They are written in HTML and can include Angular-specific syntax to bind data and handle user interactions.

  • Component: A building block of Angular applications, consisting of a TypeScript class, an HTML template, and optional CSS styles.
  • Template: The HTML part of a component that defines what the user sees.

Basic Template Syntax

Angular templates use a combination of standard HTML and Angular template syntax to create dynamic views.

Interpolation

Interpolation allows you to embed expressions in your HTML.

<p>{{ title }}</p>

In this example, {{ title }} will be replaced with the value of the title property from the component's class.

Property Binding

Property binding allows you to set properties of HTML elements or directives.

<img [src]="imageUrl">

Here, [src] binds the src attribute of the img element to the imageUrl property in the component.

Event Binding

Event binding allows you to listen for and respond to user actions.

<button (click)="onClick()">Click me</button>

In this example, (click) binds the button's click event to the onClick method in the component.

Structural Directives

Structural directives alter the layout by adding or removing elements from the DOM.

*ngIf

The *ngIf directive conditionally includes a template based on the value of an expression.

<p *ngIf="isVisible">This paragraph is visible.</p>

The paragraph will only be displayed if isVisible is true.

*ngFor

The *ngFor directive repeats a template for each item in a list.

<ul>
  <li *ngFor="let item of items">{{ item }}</li>
</ul>

This will create a list item for each element in the items array.

Attribute Directives

Attribute directives change the appearance or behavior of an element.

ngClass

The ngClass directive adds and removes CSS classes.

<div [ngClass]="{ 'active': isActive }">Content</div>

The active class will be added to the div if isActive is true.

ngStyle

The ngStyle directive adds and removes inline styles.

<div [ngStyle]="{ 'color': textColor }">Styled text</div>

The color style will be set to the value of textColor.

Template Reference Variables

Template reference variables allow you to reference a DOM element within a template.

<input #inputElement type="text">
<button (click)="logValue(inputElement.value)">Log Value</button>

In this example, #inputElement is a template reference variable that refers to the input element. The logValue method can access the value of the input.

Conclusion

Component templates are a fundamental part of Angular, enabling you to create dynamic and interactive user interfaces. By understanding and utilizing interpolation, property binding, event binding, structural directives, attribute directives, and template reference variables, you can build powerful and responsive Angular applications.
