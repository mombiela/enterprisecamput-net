Control Structures in Lua
Loops ►

Control structures are fundamental in programming as they allow you to control the flow of your code. In Lua, control structures include conditionals, loops, and other constructs that help manage the execution of your code. This section will cover the basics and advanced aspects of control structures in Lua.

Conditionals

Conditionals are used to execute code based on certain conditions. Lua supports if, else, and elseif statements.

If Statement

The if statement allows you to execute a block of code if a specified condition is true.

local age = 20

if age >= 18 then
    print("You are an adult.")
end

If-Else Statement

The if-else statement provides an alternative block of code to execute if the condition is false.

local age = 16

if age >= 18 then
    print("You are an adult.")
else
    print("You are a minor.")
end

If-Elseif-Else Statement

The if-elseif-else statement allows you to check multiple conditions.

local score = 85

if score >= 90 then
    print("Grade: A")
elseif score >= 80 then
    print("Grade: B")
elseif score >= 70 then
    print("Grade: C")
else
    print("Grade: F")
end

Loops

Loops are used to execute a block of code multiple times. Lua supports while, for, and repeat-until loops.

While Loop

The while loop executes a block of code as long as a specified condition is true.

local count = 1

while count <= 5 do
    print("Count is: " .. count)
    count = count + 1
end

For Loop

The for loop is used to iterate over a range of values.

for i = 1, 5 do
    print("Iteration: " .. i)
end

Repeat-Until Loop

The repeat-until loop executes a block of code until a specified condition becomes true.

local count = 1

repeat
    print("Count is: " .. count)
    count = count + 1
until count > 5

Advanced Control Structures

Nested Loops

Loops can be nested within each other to handle more complex scenarios.

for i = 1, 3 do
    for j = 1, 2 do
        print("i: " .. i .. ", j: " .. j)
    end
end

Break Statement

The break statement is used to exit a loop prematurely.

for i = 1, 10 do
    if i == 5 then
        break
    end
    print("i: " .. i)
end

Continue-like Behavior

Lua does not have a built-in continue statement, but you can achieve similar behavior using goto.

for i = 1, 10 do
    if i % 2 == 0 then
        goto continue
    end
    print("i: " .. i)
    ::continue::
end

Summary

Control structures are essential for managing the flow of your Lua programs. By mastering conditionals, loops, and advanced control constructs, you can write more efficient and readable code. Practice using these structures in various scenarios to become proficient in Lua programming.
