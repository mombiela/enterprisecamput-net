Introduction to Loops

Loops are fundamental constructs in programming that allow you to execute a block of code multiple times. Lua provides several types of loops to handle different scenarios.

Types of Loops in Lua

`while` Loop

The while loop repeats a block of code as long as a specified condition is true.

Syntax

while condition do
    -- code to be executed
end

Example

local count = 1
while count <= 5 do
    print("Count is: " .. count)
    count = count + 1
end
  • Explanation: This loop will print the numbers 1 to 5. The loop continues as long as count is less than or equal to 5.

`for` Loop

The for loop is used for iterating over a range of numbers or through elements of a table.

Numeric for Loop

The numeric for loop iterates over a range of numbers.

Syntax

for variable = start, stop, step do
    -- code to be executed
end

Example

for i = 1, 5 do
    print("Number: " .. i)
end
  • Explanation: This loop will print the numbers 1 to 5. The loop variable i starts at 1 and increments by 1 until it reaches 5.

Generic for Loop

The generic for loop is used to iterate over elements of a table.

Syntax

for key, value in pairs(table) do
    -- code to be executed
end

Example

local fruits = {apple = "red", banana = "yellow", grape = "purple"}
for fruit, color in pairs(fruits) do
    print(fruit .. " is " .. color)
end
  • Explanation: This loop iterates over the fruits table and prints each fruit and its color.

`repeat` Loop

The repeat loop is similar to the while loop but checks the condition after executing the block of code.

Syntax

repeat
    -- code to be executed
until condition

Example

local count = 1
repeat
    print("Count is: " .. count)
    count = count + 1
until count > 5
  • Explanation: This loop will print the numbers 1 to 5. The loop continues until count is greater than 5.

Comparison of Loop Types

| Loop Type | Condition Check | Use Case | |----------------|-----------------|-----------------------------------------| | while | Before loop | When the number of iterations is unknown| | Numeric for | Before loop | When iterating over a range of numbers | | Generic for | Before loop | When iterating over elements of a table | | repeat | After loop | When the code block needs to run at least once |

Exercises

Exercise 1: Using `while` Loop

Write a while loop that prints the first 10 even numbers.

local num = 2
while num <= 20 do
    print(num)
    num = num + 2
end

Exercise 2: Using Numeric `for` Loop

Write a numeric for loop that prints the squares of numbers from 1 to 5.

for i = 1, 5 do
    print(i * i)
end

Exercise 3: Using Generic `for` Loop

Write a generic for loop that prints the keys and values of a table containing three of your favorite movies and their release years.

local movies = {Inception = 2010, Matrix = 1999, Interstellar = 2014}
for movie, year in pairs(movies) do
    print(movie .. " was released in " .. year)
end

Exercise 4: Using `repeat` Loop

Write a repeat loop that keeps asking the user to enter a number until they enter a number greater than 10.

local num
repeat
    print("Enter a number:")
    num = tonumber(io.read())
until num > 10
print("You entered a number greater than 10!")

Conclusion

Loops are essential for performing repetitive tasks efficiently. Lua provides several types of loops (while, for, and repeat) to handle different scenarios. Understanding how and when to use each type of loop is crucial for writing effective Lua programs. Practice using these loops with various examples and exercises to solidify your understanding.
