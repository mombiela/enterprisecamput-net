Inheritance in Lua
Polymorphism ►

Inheritance is a fundamental concept in object-oriented programming (OOP) that allows a class (or object) to inherit properties and methods from another class (or object). In Lua, inheritance is achieved using metatables and the __index metamethod.

Key Concepts

  • Class: A blueprint for creating objects (instances).
  • Object: An instance of a class.
  • Metatable: A table that can change the behavior of another table.
  • __index Metamethod: Used to look up values in a table when they are not found in the original table.

Creating a Basic Class

In Lua, classes are typically represented using tables. Here's how you can create a simple class:

-- Define a basic class
Person = {}
Person.__index = Person

-- Constructor
function Person:new(name, age)
    local instance = setmetatable({}, Person)
    instance.name = name
    instance.age = age
    return instance
end

-- Method to display information
function Person:display()
    print("Name: " .. self.name .. ", Age: " .. self.age)
end

-- Create an instance of Person
local person1 = Person:new("Alice", 30)
person1:display()  -- Output: Name: Alice, Age: 30

Explanation

  • Person = {}: Defines a new table to represent the class.
  • Person.__index = Person: Sets the __index metamethod to the class itself, enabling inheritance.
  • Person:new(name, age): Constructor function to create a new instance.
  • setmetatable({}, Person): Sets the metatable of the new instance to Person.
  • self.name and self.age: Instance variables.

Implementing Inheritance

To implement inheritance, we create a new class that inherits from an existing class using the __index metamethod.

-- Define a subclass
Student = setmetatable({}, {__index = Person})
Student.__index = Student

-- Constructor
function Student:new(name, age, studentID)
    local instance = Person.new(self, name, age)
    setmetatable(instance, Student)
    instance.studentID = studentID
    return instance
end

-- Method to display student information
function Student:display()
    Person.display(self)
    print("Student ID: " .. self.studentID)
end

-- Create an instance of Student
local student1 = Student:new("Bob", 22, "S12345")
student1:display()
-- Output:
-- Name: Bob, Age: 22
-- Student ID: S12345

Explanation

  • Student = setmetatable({}, {__index = Person}): Creates a new table for the subclass and sets its __index to Person.
  • Student:new(name, age, studentID): Constructor function for the subclass.
  • Person.new(self, name, age): Calls the parent class constructor.
  • setmetatable(instance, Student): Sets the metatable of the new instance to Student.
  • Person.display(self): Calls the parent class method.

Summary

Inheritance in Lua is achieved using metatables and the __index metamethod. By setting the __index of a subclass to its parent class, we can inherit properties and methods. This allows for code reuse and a clear hierarchical structure.

Key Points

  • Use tables to represent classes.
  • Use the __index metamethod to enable inheritance.
  • Call parent class constructors and methods using the self parameter.

By understanding and utilizing inheritance, you can create more complex and organized Lua programs, making your code more modular and easier to maintain.
