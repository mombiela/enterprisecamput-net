Introduction

Operators are fundamental in any programming language, and Lua is no exception. They allow you to perform various operations on data, such as arithmetic calculations, comparisons, and logical operations. This section will cover the different types of operators available in Lua, from basic arithmetic to more advanced logical and relational operators.

Arithmetic Operators

Arithmetic operators are used to perform mathematical operations.

  • + : Addition
  • - : Subtraction
  • * : Multiplication
  • / : Division
  • % : Modulus (remainder of division)
  • ^ : Exponentiation
  • - : Unary negation

Examples

local a = 10
local b = 3

print(a + b)  -- Output: 13
print(a - b)  -- Output: 7
print(a * b)  -- Output: 30
print(a / b)  -- Output: 3.3333333333333
print(a % b)  -- Output: 1
print(a ^ b)  -- Output: 1000
print(-a)     -- Output: -10

Relational Operators

Relational operators compare two values and return a boolean result (true or false).

  • == : Equal to
  • ~= : Not equal to
  • > : Greater than
  • < : Less than
  • >= : Greater than or equal to
  • <= : Less than or equal to

Examples

local x = 5
local y = 10

print(x == y)  -- Output: false
print(x ~= y)  -- Output: true
print(x > y)   -- Output: false
print(x < y)   -- Output: true
print(x >= y)  -- Output: false
print(x <= y)  -- Output: true

Logical Operators

Logical operators are used to combine or invert boolean values.

  • and : Logical AND
  • or : Logical OR
  • not : Logical NOT

Examples

local a = true
local b = false

print(a and b)  -- Output: false
print(a or b)   -- Output: true
print(not a)    -- Output: false

Concatenation Operator

The concatenation operator (..) is used to join two strings together.

Examples

local str1 = "Hello"
local str2 = "World"

print(str1 .. " " .. str2)  -- Output: Hello World

Assignment Operators

Assignment operators are used to assign values to variables. Lua supports the basic assignment operator =.

Examples

local a = 10
local b = 20

a = a + b  -- a is now 30
print(a)   -- Output: 30

Compound Assignment Operators

Lua does not have built-in compound assignment operators like += or -=, but you can achieve the same effect using the basic assignment operator.

Examples

local a = 10

a = a + 5  -- Equivalent to a += 5 in other languages
print(a)   -- Output: 15

Operator Precedence

Operator precedence determines the order in which operations are performed. Lua follows a specific order of precedence, which is summarized in the table below:

| Operator | Description | |----------------|----------------------------| | ^ | Exponentiation | | not | Logical NOT | | *, /, % | Multiplication, Division, Modulus | | +, - | Addition, Subtraction | | .. | Concatenation | | <, >, <=, >=, ==, ~= | Relational operators | | and | Logical AND | | or | Logical OR |

Examples

local result = 5 + 2 * 3  -- Multiplication is performed first
print(result)  -- Output: 11

local result2 = (5 + 2) * 3  -- Parentheses change the order of operations
print(result2)  -- Output: 21

Conclusion

Operators are essential tools in Lua that allow you to perform a wide range of operations on data. Understanding how to use arithmetic, relational, logical, and concatenation operators, as well as knowing the order of precedence, will enable you to write more efficient and effective Lua code. Practice using these operators in different scenarios to become more comfortable with their functionality.
