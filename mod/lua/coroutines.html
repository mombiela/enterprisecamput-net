Coroutines in Lua
Modules and Packages ►

Coroutines are a powerful feature in Lua that allow you to perform cooperative multitasking. They enable you to pause and resume functions, making it easier to handle asynchronous tasks, manage state, and create complex control flows.

Introduction to Coroutines

  • Definition: Coroutines are functions that can be paused and resumed, allowing for cooperative multitasking.
  • Use Cases: Useful in scenarios like asynchronous programming, state machines, and complex control flows.

Basic Concepts

  • Coroutine States: Coroutines can be in one of several states: suspended, running, dead.
  • Coroutine Functions: coroutine.create, coroutine.resume, coroutine.yield, coroutine.status.

Creating and Running Coroutines

To create a coroutine, use coroutine.create. This function takes another function as an argument and returns a coroutine object.

function myCoroutine()
    print("Hello from coroutine!")
end

co = coroutine.create(myCoroutine)
print(co)  -- Output: thread: 0x...

To start or resume a coroutine, use coroutine.resume.

coroutine.resume(co)  -- Output: Hello from coroutine!

Yielding and Resuming

Coroutines can yield execution back to the caller using coroutine.yield.

function myCoroutine()
    for i = 1, 3 do
        print("Coroutine iteration: " .. i)
        coroutine.yield()
    end
end

co = coroutine.create(myCoroutine)

coroutine.resume(co)  -- Output: Coroutine iteration: 1
coroutine.resume(co)  -- Output: Coroutine iteration: 2
coroutine.resume(co)  -- Output: Coroutine iteration: 3

Coroutine States

You can check the state of a coroutine using coroutine.status.

print(coroutine.status(co))  -- Output: suspended
coroutine.resume(co)
print(coroutine.status(co))  -- Output: dead

Advanced Usage

Passing Arguments

You can pass arguments to coroutine.resume and receive them in the coroutine function.

function myCoroutine(a, b)
    print("Coroutine received: " .. a .. ", " .. b)
end

co = coroutine.create(myCoroutine)
coroutine.resume(co, 10, 20)  -- Output: Coroutine received: 10, 20

Returning Values

Coroutines can return values using coroutine.yield.

function myCoroutine()
    return coroutine.yield(10, 20)
end

co = coroutine.create(myCoroutine)
_, a, b = coroutine.resume(co)
print(a, b)  -- Output: 10, 20

Coroutine Wrappers

You can use coroutine.wrap to simplify coroutine usage. It returns a function that resumes the coroutine each time it is called.

function myCoroutine()
    for i = 1, 3 do
        print("Coroutine iteration: " .. i)
        coroutine.yield()
    end
end

co = coroutine.wrap(myCoroutine)
co()  -- Output: Coroutine iteration: 1
co()  -- Output: Coroutine iteration: 2
co()  -- Output: Coroutine iteration: 3

Summary

  • Coroutines in Lua allow for cooperative multitasking by pausing and resuming functions.
  • Key functions include coroutine.create, coroutine.resume, coroutine.yield, and coroutine.status.
  • Coroutines can yield and resume with arguments and return values.
  • coroutine.wrap provides a simplified interface for working with coroutines.

Understanding and utilizing coroutines can greatly enhance your ability to manage complex control flows and asynchronous tasks in Lua.
