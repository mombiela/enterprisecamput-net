Introduction

Android Studio is the official Integrated Development Environment (IDE) for Android app development, based on IntelliJ IDEA. This section will guide you through the process of installing Android Studio on your computer, ensuring you have all the necessary tools to start developing Android applications.

System Requirements

Before installing Android Studio, ensure your system meets the following requirements:

  • Windows:

    • Microsoft® Windows® 7/8/10 (32- or 64-bit)
    • 4 GB RAM minimum, 8 GB RAM recommended
    • 2 GB of available disk space minimum, 4 GB recommended (500 MB for IDE + 1.5 GB for Android SDK and emulator system image)
    • 1280 x 800 minimum screen resolution

  • macOS:

    • macOS 10.10 (Yosemite) or higher
    • 4 GB RAM minimum, 8 GB RAM recommended
    • 2 GB of available disk space minimum, 4 GB recommended (500 MB for IDE + 1.5 GB for Android SDK and emulator system image)
    • 1280 x 800 minimum screen resolution

  • Linux:

    • GNOME or KDE desktop
    • Tested on Ubuntu® 14.04 LTS, Trusty Tahr (64-bit distribution capable of running 32-bit applications)
    • 4 GB RAM minimum, 8 GB RAM recommended
    • 2 GB of available disk space minimum, 4 GB recommended (500 MB for IDE + 1.5 GB for Android SDK and emulator system image)
    • 1280 x 800 minimum screen resolution

Downloading Android Studio

  1. Visit the Official Website:

    • Go to the Android Studio download page.

  2. Select Your Operating System:

    • Choose the appropriate version for your operating system (Windows, macOS, or Linux).

  3. Download the Installer:

    • Click on the download link to get the installer file.

Installing Android Studio

Windows

  1. Run the Installer:

    • Locate the downloaded .exe file and double-click to run it.

  2. Follow the Setup Wizard:

    • Click "Next" to proceed through the setup wizard.
    • Choose the installation location and components (Android Studio, Android SDK, Android Virtual Device).

  3. Complete Installation:

    • Click "Finish" once the installation is complete.

macOS

  1. Open the DMG File:

    • Locate the downloaded .dmg file and double-click to open it.

  2. Drag and Drop:

    • Drag and drop the Android Studio icon into the Applications folder.

  3. Launch Android Studio:

    • Open the Applications folder and double-click on Android Studio to launch it.

Linux

  1. Extract the Tar File:

    • Open a terminal and navigate to the directory where the downloaded .tar.gz file is located.
    • Run the following command to extract the file: 
      tar -xvf android-studio-ide-*.tar.gz

  2. Move to Installation Directory:

    • Move the extracted folder to the desired installation directory: 
      sudo mv android-studio /opt/

  3. Launch Android Studio:

    • Navigate to the bin directory and run the studio.sh script: 
      cd /opt/android-studio/bin
./studio.sh

Initial Configuration

  1. Welcome Screen:

    • When you first launch Android Studio, you will see a welcome screen. Click on "Next" to start the setup wizard.

  2. Install SDK Components:

    • The setup wizard will guide you through the process of downloading and installing the necessary SDK components. Ensure you have a stable internet connection.

  3. Configure Settings:

    • Choose the default settings unless you have specific preferences. Click "Finish" to complete the setup.

Conclusion

By following these steps, you have successfully installed Android Studio on your computer. You are now ready to start developing Android applications. In the next sections, we will explore the Android Studio interface, create your first project, and dive into more advanced features.
