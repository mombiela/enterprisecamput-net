Common UI Components

In this section, we will explore the common UI components used in Android development. Understanding these components is essential for creating intuitive and visually appealing applications. We will cover basic to advanced components, providing examples and exercises to solidify your understanding.

Basic UI Components

TextView

  • Description: Displays text to the user.
  • Attributes: text, textColor, textSize, gravity, etc.
<TextView
    android:id="@+id/textView"
    android:layout_width="wrap_content"
    android:layout_height="wrap_content"
    android:text="Hello, World!"
    android:textSize="18sp"
    android:textColor="#000000"
    android:gravity="center"/>

EditText

  • Description: Allows the user to enter and edit text.
  • Attributes: hint, inputType, text, etc.
<EditText
    android:id="@+id/editText"
    android:layout_width="match_parent"
    android:layout_height="wrap_content"
    android:hint="Enter your name"
    android:inputType="textPersonName"/>

Button

  • Description: A clickable button that performs an action.
  • Attributes: text, onClick, background, etc.
<Button
    android:id="@+id/button"
    android:layout_width="wrap_content"
    android:layout_height="wrap_content"
    android:text="Submit"
    android:onClick="onSubmit"/>

ImageView

  • Description: Displays an image.
  • Attributes: src, scaleType, contentDescription, etc.
<ImageView
    android:id="@+id/imageView"
    android:layout_width="wrap_content"
    android:layout_height="wrap_content"
    android:src="@drawable/ic_launcher_foreground"
    android:contentDescription="App Icon"/>

Intermediate UI Components

RecyclerView

  • Description: A flexible view for providing a limited window into a large data set.
  • Attributes: layoutManager, adapter, etc.
<androidx.recyclerview.widget.RecyclerView
    android:id="@+id/recyclerView"
    android:layout_width="match_parent"
    android:layout_height="match_parent"
    android:layoutManager="LinearLayoutManager"/>

CardView

  • Description: A frame layout with rounded corners and a shadow.
  • Attributes: cardCornerRadius, cardElevation, etc.
<androidx.cardview.widget.CardView
    android:id="@+id/cardView"
    android:layout_width="match_parent"
    android:layout_height="wrap_content"
    android:layout_margin="8dp"
    app:cardCornerRadius="4dp"
    app:cardElevation="2dp">

    <TextView
        android:layout_width="wrap_content"
        android:layout_height="wrap_content"
        android:text="CardView Example"/>
</androidx.cardview.widget.CardView>

Toolbar

  • Description: A generalization of ActionBar for use within application layouts.
  • Attributes: title, subtitle, navigationIcon, etc.
<com.google.android.material.appbar.Toolbar
    android:id="@+id/toolbar"
    android:layout_width="match_parent"
    android:layout_height="wrap_content"
    android:background="?attr/colorPrimary"
    android:title="My Toolbar"/>

Advanced UI Components

NavigationView

  • Description: A standard navigation menu for the app.
  • Attributes: menu, headerLayout, etc.
<com.google.android.material.navigation.NavigationView
    android:id="@+id/navigationView"
    android:layout_width="wrap_content"
    android:layout_height="match_parent"
    android:layout_gravity="start"
    app:menu="@menu/navigation_menu"
    app:headerLayout="@layout/nav_header"/>

ViewPager2

  • Description: Allows the user to flip left and right through pages of data.
  • Attributes: orientation, adapter, etc.
<androidx.viewpager2.widget.ViewPager2
    android:id="@+id/viewPager"
    android:layout_width="match_parent"
    android:layout_height="match_parent"
    android:orientation="horizontal"/>

ConstraintLayout

  • Description: A layout that allows you to create large and complex layouts with a flat view hierarchy.
  • Attributes: constraintSet, layout_constraint, etc.
<androidx.constraintlayout.widget.ConstraintLayout
    android:id="@+id/constraintLayout"
    android:layout_width="match_parent"
    android:layout_height="match_parent">

    <Button
        android:id="@+id/button"
        android:layout_width="wrap_content"
        android:layout_height="wrap_content"
        android:text="Click Me"
        app:layout_constraintTop_toTopOf="parent"
        app:layout_constraintStart_toStartOf="parent"
        app:layout_constraintEnd_toEndOf="parent"/>
</androidx.constraintlayout.widget.ConstraintLayout>

Summary

In this topic, we covered a range of common UI components in Android development, from basic elements like TextView and Button to more advanced components like RecyclerView and ConstraintLayout. Each component serves a specific purpose and can be customized through various attributes to fit the needs of your application. By understanding and utilizing these components, you can create more dynamic and user-friendly interfaces in your Android apps.
