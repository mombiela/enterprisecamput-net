Descriptive analysis is one of the first stages in the business analytics process. Its main objective is to summarize and describe the characteristics of a dataset, providing a clear and understandable view of what the data represents. This type of analysis is fundamental for understanding the context and trends before proceeding to more complex analyses.

Objectives of Descriptive Analysis

  • Summarize data: Provide an overview of the data through summary statistics.
  • Identify patterns: Detect trends and patterns in the data.
  • Data visualization: Use charts and tables to represent the data visually.
  • Preparation for advanced analyses: Establish a solid foundation for predictive and prescriptive analyses.

Key Concepts

  • Measures of Central Tendency:

    • Mean: The average of all values.
    • Median: The central value when the data is ordered.
    • Mode: The value that appears most frequently.

  • Measures of Dispersion:

    • Range: The difference between the maximum and minimum values.
    • Variance: The measure of how much the data varies from the mean.
    • Standard deviation: The square root of the variance, indicating the dispersion of the data.

  • Frequency Distribution:

    • Frequency tables: Tables that show the frequency of each value or range of values.
    • Histograms: Bar charts that represent the frequency distribution.

Tools and Techniques

Descriptive Statistics

Descriptive statistics are fundamental for summarizing and describing the data. Below is an example of how to calculate some basic descriptive statistics using Python and the pandas library.

import pandas as pd

# Create an example DataFrame
data = {'Sales': [200, 220, 250, 270, 300, 320, 350]}
df = pd.DataFrame(data)

# Calculate descriptive statistics
mean = df['Sales'].mean()
median = df['Sales'].median()
mode = df['Sales'].mode()[0]
range_ = df['Sales'].max() - df['Sales'].min()
variance = df['Sales'].var()
standard_deviation = df['Sales'].std()

# Display results
print(f"Mean: {mean}")
print(f"Median: {median}")
print(f"Mode: {mode}")
print(f"Range: {range_}")
print(f"Variance: {variance}")
print(f"Standard Deviation: {standard_deviation}")

Data Visualization

Data visualization is crucial for interpreting and communicating the results of descriptive analysis. Below is an example of how to create a histogram using matplotlib.

import matplotlib.pyplot as plt

# Create a histogram
plt.hist(df['Sales'], bins=5, edgecolor='black')
plt.title('Sales Distribution')
plt.xlabel('Sales')
plt.ylabel('Frequency')
plt.show()

Practical Example

Suppose a company wants to analyze the monthly sales of its products. The sales data is presented in the following table:

Using the techniques and tools described above, we can calculate the descriptive statistics and visualize the sales distribution to gain a clear understanding of the data.

Conclusion

Descriptive analysis is an essential stage in business analytics, as it provides a solid foundation for any subsequent analysis. By summarizing and visualizing the data, we can identify patterns and trends that are crucial for informed decision-making.
