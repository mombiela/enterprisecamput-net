Reading from Files in Fortran
Writing to Files ►

Reading from files is a fundamental skill in Fortran programming, enabling you to handle data input efficiently. This topic will guide you through the basics to advanced techniques of reading from files in Fortran.

Introduction to File Handling in Fortran

Before diving into reading from files, it's essential to understand some basic concepts of file handling in Fortran.

  • File Unit Numbers: Fortran uses unit numbers to refer to files. These are integer values that uniquely identify each file during program execution.
  • OPEN Statement: This statement is used to open a file and associate it with a unit number.
  • READ Statement: This statement reads data from a file.
  • CLOSE Statement: This statement closes a file that has been opened.

Basic File Reading

Opening a File

To read from a file, you first need to open it using the OPEN statement. Hereâ€™s a simple example:

PROGRAM ReadFile
  INTEGER :: unit_number
  unit_number = 10
  OPEN(unit=unit_number, file='data.txt', status='old')
  ! Now the file 'data.txt' is associated with unit number 10
END PROGRAM ReadFile
  • unit=unit_number: Specifies the unit number.
  • file='data.txt': Specifies the file name.
  • status='old': Indicates that the file already exists.

Reading Data

Once the file is opened, you can read data using the READ statement.

PROGRAM ReadFile
  INTEGER :: unit_number, number
  unit_number = 10
  OPEN(unit=unit_number, file='data.txt', status='old')
  READ(unit_number, *) number
  PRINT *, 'Number read from file:', number
  CLOSE(unit_number)
END PROGRAM ReadFile
  • READ(unit_number, *) number: Reads data from the file associated with unit_number into the variable number.

Closing a File

After reading the data, it is good practice to close the file using the CLOSE statement.

CLOSE(unit_number)

Reading Different Data Types

Fortran allows you to read various data types from a file, such as integers, real numbers, and strings.

Example: Reading Multiple Data Types

PROGRAM ReadFile
  INTEGER :: unit_number, int_val
  REAL :: real_val
  CHARACTER(len=20) :: str_val

  unit_number = 10
  OPEN(unit=unit_number, file='data.txt', status='old')
  READ(unit_number, *) int_val, real_val, str_val
  PRINT *, 'Integer:', int_val
  PRINT *, 'Real:', real_val
  PRINT *, 'String:', str_val
  CLOSE(unit_number)
END PROGRAM ReadFile

Advanced File Reading Techniques

Reading Data in a Loop

Often, you need to read multiple lines of data. This can be done using a loop.

PROGRAM ReadFile
  INTEGER :: unit_number, int_val, i
  unit_number = 10
  OPEN(unit=unit_number, file='data.txt', status='old')

  DO i = 1, 10
    READ(unit_number, *) int_val
    PRINT *, 'Read value:', int_val
  END DO

  CLOSE(unit_number)
END PROGRAM ReadFile

Handling End-of-File

To handle the end-of-file (EOF) condition, you can use the IOSTAT specifier.

PROGRAM ReadFile
  INTEGER :: unit_number, int_val, io_status
  unit_number = 10
  OPEN(unit=unit_number, file='data.txt', status='old')

  DO
    READ(unit_number, *, IOSTAT=io_status) int_val
    IF (io_status /= 0) EXIT
    PRINT *, 'Read value:', int_val
  END DO

  CLOSE(unit_number)
END PROGRAM ReadFile
  • IOSTAT=io_status: Stores the status of the I/O operation. A non-zero value indicates an error or EOF.

Summary

In this topic, we covered the basics and advanced techniques of reading from files in Fortran:

  • Opening and closing files using OPEN and CLOSE statements.
  • Reading data using the READ statement.
  • Handling different data types.
  • Reading data in loops and handling EOF conditions.

Understanding these concepts will enable you to efficiently manage data input in your Fortran programs. Practice these techniques with various data files to become proficient in file handling.
