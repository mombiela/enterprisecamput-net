Introduction

Fortran, short for "Formula Translation," is one of the oldest high-level programming languages, primarily used for numerical and scientific computing. Understanding variables and data types is fundamental to programming in Fortran. This section will cover the basics of declaring variables, understanding different data types, and their usage in Fortran.

Variables in Fortran

Variables are used to store data that can be manipulated throughout a program. In Fortran, variables must be declared before they are used.

Declaring Variables

To declare a variable in Fortran, you specify the type followed by the variable name. Optionally, you can also initialize the variable at the time of declaration.

INTEGER :: age
REAL :: height = 5.9
CHARACTER(LEN=20) :: name = "John Doe"
  • INTEGER declares an integer variable.
  • REAL declares a floating-point variable.
  • CHARACTER declares a string variable with a specified length.

Variable Naming Rules

  • Variable names must start with a letter.
  • They can contain letters, digits, and underscores.
  • They are case-insensitive (e.g., Age and age are considered the same).

Data Types in Fortran

Fortran supports several data types, each suited for different kinds of data. Understanding these types is crucial for efficient programming.

Basic Data Types

  1. INTEGER: Used for whole numbers. 
    INTEGER :: count = 10
  2. REAL: Used for floating-point numbers. 
    REAL :: temperature = 36.6
  3. DOUBLE PRECISION: Used for double-precision floating-point numbers. 
    DOUBLE PRECISION :: largeNumber = 1.0D+300
  4. COMPLEX: Used for complex numbers. 
    COMPLEX :: z = (3.0, 4.0)
  5. CHARACTER: Used for strings. 
    CHARACTER(LEN=10) :: greeting = "Hello"
  6. LOGICAL: Used for boolean values. 
    LOGICAL :: isValid = .TRUE.

Derived Data Types

Fortran allows the creation of custom data types using the TYPE keyword. This is useful for grouping related variables together.

TYPE :: Person
    CHARACTER(LEN=20) :: name
    INTEGER :: age
    REAL :: height
END TYPE Person

TYPE(Person) :: student
student%name = "Alice"
student%age = 21
student%height = 5.7

Type Conversion

Fortran provides intrinsic functions to convert between different data types.

  • INT(): Converts to integer.
  • REAL(): Converts to real.
  • DBLE(): Converts to double precision.
  • CMPLX(): Converts to complex.
  • CHAR(): Converts to character.

Example

INTEGER :: i = 5
REAL :: r
r = REAL(i)  ! Converts integer to real

Arrays

Arrays are used to store multiple values of the same type. They can be declared with specified dimensions.

Declaring Arrays

INTEGER, DIMENSION(5) :: numbers = (/1, 2, 3, 4, 5/)
REAL, DIMENSION(3, 3) :: matrix

Accessing Array Elements

Array elements are accessed using indices.

numbers(1) = 10  ! Sets the first element to 10
matrix(2, 3) = 5.5  ! Sets the element at row 2, column 3 to 5.5

Summary

In this section, we covered the basics of variables and data types in Fortran. We learned how to declare variables, understand different data types, and use arrays. Mastering these concepts is essential for writing efficient and effective Fortran programs. As you progress, you will encounter more complex data structures and operations, but a solid understanding of these fundamentals will provide a strong foundation for your Fortran programming journey.
