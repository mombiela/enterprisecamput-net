Introduction to Scientific Computing

Scientific computing involves using computational methods and software to solve complex scientific and engineering problems. Fortran, short for "Formula Translation," is a high-level programming language that has been widely used in scientific computing due to its efficiency in numerical computation and array manipulation.

Getting Started with Fortran

Before diving into scientific computing, it's essential to understand the basics of Fortran.

Basic Syntax and Structure

  • Program Structure: A Fortran program typically consists of a program name, declarations, executable statements, and an end statement.
  • Variables and Data Types: Common data types include INTEGER, REAL, DOUBLE PRECISION, COMPLEX, LOGICAL, and CHARACTER.
  • Control Structures: Includes IF statements, DO loops, and SELECT CASE statements.

Example: Simple Fortran Program

PROGRAM HelloWorld
  PRINT *, 'Hello, World!'
END PROGRAM HelloWorld

Compiling and Running Fortran Programs

  • Compiling: Use a Fortran compiler like gfortran to compile your code. 
    gfortran HelloWorld.f90 -o HelloWorld
  • Running: Execute the compiled program. 
    ./HelloWorld

Numerical Methods in Fortran

Numerical methods are algorithms used for solving mathematical problems numerically.

Solving Linear Equations

  • Gauss Elimination: A method for solving a system of linear equations.
  • LU Decomposition: Decomposes a matrix into a lower triangular matrix (L) and an upper triangular matrix (U).

Example: Solving Linear Equations using Gauss Elimination

PROGRAM GaussElimination
  IMPLICIT NONE
  INTEGER, PARAMETER :: n = 3
  REAL :: A(n,n), b(n), x(n)
  INTEGER :: i, j, k

  ! Initialize matrix A and vector b
  A = RESHAPE([2.0, -1.0, 1.0, 3.0, 3.0, 9.0, 3.0, 3.0, 5.0], SHAPE(A))
  b = [8.0, 0.0, 6.0]

  ! Forward elimination
  DO k = 1, n-1
    DO i = k+1, n
      A(i,k:n) = A(i,k:n) - A(i,k)/A(k,k) * A(k,k:n)
      b(i) = b(i) - A(i,k)/A(k,k) * b(k)
    END DO
  END DO

  ! Back substitution
  x(n) = b(n) / A(n,n)
  DO i = n-1, 1, -1
    x(i) = (b(i) - SUM(A(i,i+1:n) * x(i+1:n))) / A(i,i)
  END DO

  PRINT *, 'Solution: ', x
END PROGRAM GaussElimination

Numerical Integration

  • Trapezoidal Rule: Approximates the integral of a function by dividing the area under the curve into trapezoids.
  • Simpson's Rule: Provides a more accurate approximation by using parabolic segments.

Example: Numerical Integration using Trapezoidal Rule

PROGRAM TrapezoidalRule
  IMPLICIT NONE
  REAL :: a, b, h, integral
  INTEGER :: n, i
  REAL, EXTERNAL :: f

  a = 0.0
  b = 1.0
  n = 100
  h = (b - a) / n
  integral = 0.5 * (f(a) + f(b))

  DO i = 1, n-1
    integral = integral + f(a + i*h)
  END DO

  integral = integral * h
  PRINT *, 'Integral: ', integral
END PROGRAM TrapezoidalRule

REAL FUNCTION f(x)
  REAL, INTENT(IN) :: x
  f = x**2
END FUNCTION f

Advanced Topics in Fortran

As you advance, you will encounter more complex and specialized topics.

Parallel Computing with Fortran

  • OpenMP: A widely used API for parallel programming in shared memory environments.
  • MPI: Message Passing Interface for parallel programming in distributed memory systems.

Example: Parallel Computing with OpenMP

PROGRAM ParallelSum
  USE omp_lib
  IMPLICIT NONE
  INTEGER :: i, n
  REAL :: sum
  REAL, ALLOCATABLE :: a(:)

  n = 1000000
  ALLOCATE(a(n))
  a = 1.0

  sum = 0.0
  !$OMP PARALLEL DO REDUCTION(+:sum)
  DO i = 1, n
    sum = sum + a(i)
  END DO
  !$OMP END PARALLEL DO

  PRINT *, 'Sum: ', sum
END PROGRAM ParallelSum

Fortran Libraries for Scientific Computing

  • LAPACK: Linear Algebra PACKage for solving linear algebra problems.
  • BLAS: Basic Linear Algebra Subprograms for performing basic vector and matrix operations.
  • FFTW: Fastest Fourier Transform in the West for computing Fourier transforms.

Conclusion

Fortran remains a powerful tool for scientific computing due to its efficiency and extensive libraries. By mastering Fortran, you can tackle a wide range of numerical and scientific problems, from basic linear algebra to advanced parallel computing. This course has provided a foundation in Fortran programming, numerical methods, and advanced topics, equipping you with the skills needed for scientific computing.
