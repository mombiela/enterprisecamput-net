Introduction to Pointers

Pointers are a powerful feature in Fortran that allow you to reference and manipulate memory locations directly. This can be particularly useful for dynamic memory allocation, creating complex data structures, and optimizing performance.

Key Concepts

  • Pointer Declaration: How to declare a pointer variable.
  • Pointer Association: Associating a pointer with a target.
  • Pointer Operations: Basic operations you can perform with pointers.
  • Dynamic Memory Allocation: Using pointers to allocate and deallocate memory dynamically.
  • Pointer Arrays: Working with arrays of pointers.

Pointer Declaration

To declare a pointer in Fortran, you use the POINTER attribute in the type declaration.

INTEGER, POINTER :: ptr
REAL, POINTER :: rptr

In this example:

  • ptr is a pointer to an integer.
  • rptr is a pointer to a real number.

Pointer Association

Pointers need to be associated with a target before they can be used. This is done using the => operator.

INTEGER, TARGET :: target_var
INTEGER, POINTER :: ptr

ptr => target_var

In this example:

  • target_var is an integer variable that can be targeted by pointers.
  • ptr is associated with target_var.

Pointer Operations

Once a pointer is associated with a target, you can perform various operations on it.

Example: Basic Pointer Operations

INTEGER, TARGET :: target_var = 10
INTEGER, POINTER :: ptr

ptr => target_var
PRINT *, "Value of target_var: ", target_var
PRINT *, "Value via pointer: ", ptr

ptr = 20
PRINT *, "New value of target_var: ", target_var

Output:

Value of target_var: 10
Value via pointer: 10
New value of target_var: 20

Dynamic Memory Allocation

Pointers are essential for dynamic memory allocation in Fortran. You can allocate and deallocate memory using the ALLOCATE and DEALLOCATE statements.

Example: Dynamic Memory Allocation

INTEGER, POINTER :: ptr
ALLOCATE(ptr)
ptr = 42
PRINT *, "Dynamically allocated value: ", ptr
DEALLOCATE(ptr)

Output:

Dynamically allocated value: 42

Pointer Arrays

You can also create arrays of pointers, which can be useful for managing collections of dynamically allocated memory.

Example: Pointer Arrays

INTEGER, POINTER :: ptr(:)
ALLOCATE(ptr(5))

ptr(1) = 1
ptr(2) = 2
ptr(3) = 3
ptr(4) = 4
ptr(5) = 5

PRINT *, "Pointer array values: ", ptr

DEALLOCATE(ptr)

Output:

Pointer array values: 1 2 3 4 5

Summary

Pointers in Fortran provide a way to directly reference and manipulate memory locations, which is useful for dynamic memory allocation and creating complex data structures. Key concepts include pointer declaration, association, basic operations, dynamic memory allocation, and pointer arrays. Understanding these concepts will allow you to write more flexible and efficient Fortran programs.
