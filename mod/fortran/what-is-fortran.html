What is Fortran?
Setting Up the Development Environment ►

Fortran, short for "Formula Translation," is a high-level programming language that was developed in the 1950s by IBM. It is one of the oldest programming languages and is particularly well-suited for numerical computation and scientific computing.

History of Fortran

  • 1950s: Developed by IBM, Fortran was the first high-level programming language.
  • 1960s-1970s: Fortran II, Fortran IV, and Fortran 66 were introduced, adding more features and improving the language.
  • 1980s: Fortran 77 became the standard, introducing structured programming constructs.
  • 1990s: Fortran 90 and Fortran 95 added modern programming features like array programming and modularity.
  • 2000s-Present: Fortran 2003, Fortran 2008, and Fortran 2018 have continued to evolve, adding object-oriented programming and other advanced features.

Key Features of Fortran

  • Numerical Computation: Optimized for mathematical and scientific calculations.
  • Array Handling: Efficient manipulation of arrays.
  • Portability: Code can be run on different types of computer systems with minimal changes.
  • Performance: High performance for computational tasks.
  • Legacy Code: Extensive libraries and codebases that have been developed over decades.

Basic Syntax and Structure

Fortran code is organized into programs and subprograms. Here is a simple example of a Fortran program:

program HelloWorld
  print *, 'Hello, World!'
end program HelloWorld

Explanation:

  • program HelloWorld: Declares the start of a program named HelloWorld.
  • print *, 'Hello, World!': Prints the string 'Hello, World!' to the console.
  • end program HelloWorld: Marks the end of the program.

Data Types

Fortran supports several data types, including:

  • Integer: Whole numbers.
  • Real: Floating-point numbers.
  • Complex: Complex numbers.
  • Character: Strings of text.
  • Logical: Boolean values (true or false).

Example:

program DataTypes
  integer :: i = 10
  real :: x = 3.14
  complex :: z = (1.0, 2.0)
  character(len=20) :: str = 'Fortran'
  logical :: flag = .true.

  print *, 'Integer:', i
  print *, 'Real:', x
  print *, 'Complex:', z
  print *, 'Character:', str
  print *, 'Logical:', flag
end program DataTypes

Control Structures

Fortran includes standard control structures such as:

  • If Statements: Conditional execution.
  • Loops: Repetition of code blocks (e.g., do loops).

Example:

program ControlStructures
  integer :: i

  ! If statement
  if (i > 0) then
    print *, 'i is positive'
  else
    print *, 'i is non-positive'
  end if

  ! Do loop
  do i = 1, 5
    print *, 'Iteration:', i
  end do
end program ControlStructures

Arrays

Fortran excels in array handling, allowing for efficient and concise operations on arrays.

Example:

program Arrays
  integer, dimension(5) :: arr = [1, 2, 3, 4, 5]
  integer :: i

  ! Print array elements
  do i = 1, 5
    print *, 'Element', i, ':', arr(i)
  end do
end program Arrays

Conclusion

Fortran is a powerful language for numerical and scientific computing, with a rich history and a strong focus on performance and array handling. Understanding its basic syntax, data types, control structures, and array operations provides a solid foundation for further exploration and advanced programming in Fortran.
