File Positioning in Fortran
Binary File Operations ►

File positioning is an essential aspect of file handling in Fortran. It allows programmers to control where in a file data is read from or written to. This topic will cover the basics of file positioning, including how to move the file pointer, and provide examples and exercises to solidify your understanding.

Key Concepts

  • File Pointer: A marker that indicates the current position in a file where the next read or write operation will occur.
  • Sequential Access: Reading or writing data in a sequence, from the beginning to the end of the file.
  • Direct Access: Reading or writing data at specific positions in the file, allowing for non-sequential access.

Sequential Access

In sequential access, the file pointer moves automatically as data is read or written. This is the simplest form of file positioning.

Example: Sequential Read

PROGRAM SequentialRead
  IMPLICIT NONE
  INTEGER :: i, io_status
  OPEN(UNIT=10, FILE='data.txt', STATUS='OLD', ACTION='READ')

  DO
    READ(10, *, IOSTAT=io_status) i
    IF (io_status /= 0) EXIT
    PRINT *, 'Read value:', i
  END DO

  CLOSE(10)
END PROGRAM SequentialRead

In this example:

  • The file data.txt is opened for reading.
  • A loop reads integers from the file until the end is reached.
  • The file pointer moves automatically to the next integer after each read operation.

Direct Access

Direct access allows you to read or write data at specific positions in the file. This is useful for random access to data.

Example: Direct Access Write and Read

PROGRAM DirectAccess
  IMPLICIT NONE
  INTEGER :: i, rec_num
  INTEGER, DIMENSION(5) :: data = (/1, 2, 3, 4, 5/)
  INTEGER :: io_status

  ! Open file for direct access
  OPEN(UNIT=20, FILE='direct_data.dat', STATUS='NEW', ACCESS='DIRECT', RECL=4)

  ! Write data to specific records
  DO rec_num = 1, 5
    WRITE(20, REC=rec_num, IOSTAT=io_status) data(rec_num)
    IF (io_status /= 0) THEN
      PRINT *, 'Error writing to record', rec_num
      STOP
    END IF
  END DO

  ! Read data from specific records
  DO rec_num = 1, 5
    READ(20, REC=rec_num, IOSTAT=io_status) i
    IF (io_status /= 0) THEN
      PRINT *, 'Error reading from record', rec_num
      STOP
    END IF
    PRINT *, 'Read value from record', rec_num, ':', i
  END DO

  CLOSE(20)
END PROGRAM DirectAccess

In this example:

  • The file direct_data.dat is opened for direct access with a record length of 4 bytes.
  • Data is written to specific records in the file.
  • Data is then read from those records, demonstrating random access.

Summary

File positioning in Fortran is a crucial skill for managing file I/O operations effectively. Understanding both sequential and direct access methods allows for flexible and efficient data handling. Sequential access is straightforward and suitable for simple read/write operations, while direct access provides the ability to manipulate data at specific positions, which is useful for more complex applications.

By mastering these techniques, you can handle a wide range of file I/O scenarios in Fortran, enhancing your ability to develop robust and efficient programs.
