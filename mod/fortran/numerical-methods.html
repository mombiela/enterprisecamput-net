Introduction to Numerical Methods

Numerical methods are techniques used to solve mathematical problems numerically, i.e., using numbers and algorithms rather than symbolic manipulations. This section will introduce you to the basics of numerical methods and how they can be implemented in Fortran.

  • Definition: Numerical methods involve algorithms for solving mathematical problems.
  • Applications: Engineering, physics, finance, and other fields requiring numerical solutions.

Basic Fortran Syntax and Structures

Before diving into numerical methods, it's essential to understand the basic syntax and structures of Fortran.

  • Variables and Data Types: INTEGER, REAL, DOUBLE PRECISION, COMPLEX, etc.
  • Control Structures: IF statements, DO loops, etc.
  • Functions and Subroutines: Modularize code for reusability.

Example: Basic Fortran Program

PROGRAM HelloWorld
  PRINT *, 'Hello, World!'
END PROGRAM HelloWorld

Root Finding Methods

Root finding methods are used to find the roots of equations. Common methods include the Bisection Method and Newton-Raphson Method.

Bisection Method

The Bisection Method is a simple and robust method for finding roots of a function.

  • Algorithm:
    1. Choose two points \(a\) and \(b\) such that \(f(a)\) and \(f(b)\) have opposite signs.
    2. Compute the midpoint \(c = (a + b) / 2\).
    3. Check the sign of \(f(c)\).
    4. Replace \(a\) or \(b\) with \(c\) based on the sign.
    5. Repeat until the root is found within a desired tolerance.

Example: Bisection Method in Fortran

PROGRAM BisectionMethod
  IMPLICIT NONE
  REAL :: a, b, c, tol
  INTEGER :: max_iter, iter

  a = 1.0
  b = 2.0
  tol = 1.0e-6
  max_iter = 100

  DO iter = 1, max_iter
    c = (a + b) / 2.0
    IF (ABS(f(c)) < tol) EXIT
    IF (f(a) * f(c) < 0.0) THEN
      b = c
    ELSE
      a = c
    END IF
  END DO

  PRINT *, 'Root is approximately: ', c

CONTAINS
  REAL FUNCTION f(x)
    REAL, INTENT(IN) :: x
    f = x**2 - 2.0
  END FUNCTION f
END PROGRAM BisectionMethod

Numerical Integration

Numerical integration involves approximating the integral of a function. Common methods include the Trapezoidal Rule and Simpson's Rule.

Trapezoidal Rule

The Trapezoidal Rule approximates the integral by dividing the area under the curve into trapezoids.

  • Algorithm:
    1. Divide the interval \([a, b]\) into \(n\) subintervals.
    2. Compute the sum of the areas of the trapezoids.

Example: Trapezoidal Rule in Fortran

PROGRAM TrapezoidalRule
  IMPLICIT NONE
  REAL :: a, b, h, integral
  INTEGER :: n, i

  a = 0.0
  b = 1.0
  n = 100
  h = (b - a) / n
  integral = 0.5 * (f(a) + f(b))

  DO i = 1, n-1
    integral = integral + f(a + i * h)
  END DO

  integral = integral * h
  PRINT *, 'Approximate integral: ', integral

CONTAINS
  REAL FUNCTION f(x)
    REAL, INTENT(IN) :: x
    f = x**2
  END FUNCTION f
END PROGRAM TrapezoidalRule

Solving Systems of Linear Equations

Solving systems of linear equations is a fundamental problem in numerical methods. Techniques include Gaussian Elimination and LU Decomposition.

Gaussian Elimination

Gaussian Elimination is a method for solving linear systems by transforming the system's matrix into an upper triangular form.

  • Algorithm:
    1. Forward elimination to form an upper triangular matrix.
    2. Back substitution to solve for the variables.

Example: Gaussian Elimination in Fortran

PROGRAM GaussianElimination
  IMPLICIT NONE
  REAL, DIMENSION(3,3) :: A
  REAL, DIMENSION(3) :: b, x
  INTEGER :: i, j, k

  A = RESHAPE([2.0, -1.0, 1.0, 3.0, 3.0, 9.0, 3.0, 3.0, 5.0], SHAPE(A))
  b = [1.0, 0.0, 4.0]

  CALL gauss_elimination(A, b, x)

  PRINT *, 'Solution: ', x

CONTAINS
  SUBROUTINE gauss_elimination(A, b, x)
    REAL, DIMENSION(:,:), INTENT(INOUT) :: A
    REAL, DIMENSION(:), INTENT(INOUT) :: b
    REAL, DIMENSION(:), INTENT(OUT) :: x
    INTEGER :: n, i, j, k
    REAL :: factor

    n = SIZE(b)
    DO k = 1, n-1
      DO i = k+1, n
        factor = A(i,k) / A(k,k)
        A(i,k:n) = A(i,k:n) - factor * A(k,k:n)
        b(i) = b(i) - factor * b(k)
      END DO
    END DO

    x(n) = b(n) / A(n,n)
    DO i = n-1, 1, -1
      x(i) = (b(i) - SUM(A(i,i+1:n) * x(i+1:n))) / A(i,i)
    END DO
  END SUBROUTINE gauss_elimination
END PROGRAM GaussianElimination

Conclusion

  • Basic Fortran syntax and structures.

  • Numerical integration using the Trapezoidal Rule.
  • Solving systems of linear equations with Gaussian Elimination.
