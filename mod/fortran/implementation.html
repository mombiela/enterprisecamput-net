Introduction to Implementation in Fortran

In this section, we will cover how to implement programs in Fortran, from writing simple scripts to developing more complex applications. We will start with basic concepts and gradually move to advanced topics, ensuring that you understand each step along the way.

Writing Your First Fortran Program

Let's begin with a simple "Hello, World!" program to get you familiar with the basic structure of a Fortran program.

program hello
    print *, 'Hello, World!'
end program hello
  • program: This keyword starts the program.
  • print: This statement outputs text to the screen.
  • end program: This keyword ends the program.

Exercise: Modify the Program

Try modifying the program to print your name instead of "Hello, World!".

program hello
    print *, 'Hello, [Your Name]!'
end program hello

Variables and Data Types

Fortran supports several data types, including integers, real numbers, and characters. Let's look at how to declare and use these variables.

program variables
    integer :: i
    real :: x
    character(len=20) :: name

    i = 10
    x = 3.14
    name = 'Fortran'

    print *, 'Integer:', i
    print *, 'Real:', x
    print *, 'Character:', name
end program variables
  • integer: Declares an integer variable.
  • real: Declares a floating-point variable.
  • character: Declares a string variable with a specified length.

Exercise: Declare and Print Variables

Declare variables of different types and print their values.

program my_variables
    integer :: age
    real :: height
    character(len=30) :: hobby

    age = 25
    height = 5.9
    hobby = 'Programming'

    print *, 'Age:', age
    print *, 'Height:', height
    print *, 'Hobby:', hobby
end program my_variables

Control Structures

Control structures like loops and conditionals are essential for implementing logic in your programs.

If Statements

program if_example
    integer :: num

    num = 10

    if (num > 5) then
        print *, 'Number is greater than 5'
    else
        print *, 'Number is 5 or less'
    end if
end program if_example

Do Loops

program loop_example
    integer :: i

    do i = 1, 5
        print *, 'Iteration:', i
    end do
end program loop_example

Exercise: Implement Control Structures

Write a program that uses both if statements and do loops.

program control_structures
    integer :: i

    do i = 1, 10
        if (mod(i, 2) == 0) then
            print *, i, 'is even'
        else
            print *, i, 'is odd'
        end if
    end do
end program control_structures

Functions and Subroutines

Functions and subroutines allow you to modularize your code, making it more readable and reusable.

Functions

program function_example
    real :: result

    result = square(4.0)
    print *, 'Square of 4.0 is', result

contains
    real function square(x)
        real, intent(in) :: x
        square = x * x
    end function square
end program function_example

Subroutines

program subroutine_example
    real :: a, b, result

    a = 3.0
    b = 4.0
    call add(a, b, result)
    print *, 'Sum of', a, 'and', b, 'is', result

contains
    subroutine add(x, y, sum)
        real, intent(in) :: x, y
        real, intent(out) :: sum
        sum = x + y
    end subroutine add
end program subroutine_example

Exercise: Create Functions and Subroutines

Create a program that includes both a function and a subroutine.

program func_sub_example
    real :: num, sq, sum

    num = 5.0
    sq = square(num)
    call add(num, sq, sum)

    print *, 'Square of', num, 'is', sq
    print *, 'Sum of', num, 'and its square is', sum

contains
    real function square(x)
        real, intent(in) :: x
        square = x * x
    end function square

    subroutine add(x, y, result)
        real, intent(in) :: x, y
        real, intent(out) :: result
        result = x + y
    end subroutine add
end program func_sub_example

Advanced Topics

Modules

Modules are used to group related procedures and data together.

module math_operations
    contains
        real function square(x)
            real, intent(in) :: x
            square = x * x
        end function square

        subroutine add(x, y, sum)
            real, intent(in) :: x, y
            real, intent(out) :: sum
            sum = x + y
        end subroutine add
end module math_operations

program use_module
    use math_operations
    real :: a, b, result

    a = 2.0
    b = 3.0
    result = square(a)
    print *, 'Square of', a, 'is', result

    call add(a, b, result)
    print *, 'Sum of', a, 'and', b, 'is', result
end program use_module

Exercise: Create and Use Modules

Create a module with at least two functions and use it in a program.

module my_module
    contains
        real function cube(x)
            real, intent(in) :: x
            cube = x * x * x
        end function cube

        subroutine multiply(x, y, product)
            real, intent(in) :: x, y
            real, intent(out) :: product
            product = x * y
        end subroutine multiply
end module my_module

program use_my_module
    use my_module
    real :: num, cubed, product

    num = 3.0
    cubed = cube(num)
    call multiply(num, cubed, product)

    print *, 'Cube of', num, 'is', cubed
    print *, 'Product of', num, 'and its cube is', product
end program use_my_module

Conclusion

In this section, we covered the basics of implementing programs in Fortran, including writing simple programs, using variables and data types, implementing control structures, and creating functions and subroutines. We also touched on advanced topics like modules. By practicing these concepts, you will be well on your way to becoming proficient in Fortran programming.
