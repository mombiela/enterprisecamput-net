Introduction to Functions

Functions in Fortran are essential building blocks that allow you to encapsulate code for reuse, improve readability, and manage complexity. In this section, we will cover the basics of defining and using functions in Fortran, progressing from simple examples to more advanced concepts.

Basic Function Definition

A function in Fortran is a block of code that takes input arguments, performs a specific task, and returns a value. Here's the basic syntax for defining a function:

FUNCTION function_name(arguments)
  ! Declarations
  IMPLICIT NONE
  ! Function body
  function_name = result
END FUNCTION function_name

Example: Simple Function

Let's start with a simple example of a function that adds two numbers:

PROGRAM main
  IMPLICIT NONE
  REAL :: result

  result = add(2.0, 3.0)
  PRINT *, "The sum is: ", result

CONTAINS

  FUNCTION add(a, b)
    REAL :: add
    REAL, INTENT(IN) :: a, b

    add = a + b
  END FUNCTION add

END PROGRAM main
  • Explanation:
    • PROGRAM main: The main program block.
    • IMPLICIT NONE: Ensures all variables must be explicitly declared.
    • REAL :: result: Declares a real variable to store the result.
    • result = add(2.0, 3.0): Calls the add function with arguments 2.0 and 3.0.
    • CONTAINS: Indicates that the following block contains internal procedures.
    • FUNCTION add(a, b): Defines the add function.
    • REAL, INTENT(IN) :: a, b: Declares input arguments a and b as real numbers.
    • add = a + b: Computes the sum and assigns it to the function name.

Function Arguments and Return Types

Functions can take various types of arguments and return different types of values. The return type of a function is specified in its declaration.

Example: Function with Integer Arguments

Here's an example of a function that multiplies two integers:

PROGRAM main
  IMPLICIT NONE
  INTEGER :: result

  result = multiply(4, 5)
  PRINT *, "The product is: ", result

CONTAINS

  FUNCTION multiply(a, b)
    INTEGER :: multiply
    INTEGER, INTENT(IN) :: a, b

    multiply = a * b
  END FUNCTION multiply

END PROGRAM main
  • Explanation:
    • INTEGER :: result: Declares an integer variable to store the result.
    • result = multiply(4, 5): Calls the multiply function with arguments 4 and 5.
    • FUNCTION multiply(a, b): Defines the multiply function.
    • INTEGER, INTENT(IN) :: a, b: Declares input arguments a and b as integers.
    • multiply = a * b: Computes the product and assigns it to the function name.

Advanced Function Concepts

Recursive Functions

Fortran supports recursive functions, which are functions that call themselves. To define a recursive function, you must use the RECURSIVE keyword.

Example: Recursive Factorial Function

Here's an example of a recursive function to compute the factorial of a number:

PROGRAM main
  IMPLICIT NONE
  INTEGER :: result

  result = factorial(5)
  PRINT *, "The factorial is: ", result

CONTAINS

  RECURSIVE FUNCTION factorial(n) RESULT(fact)
    INTEGER :: fact
    INTEGER, INTENT(IN) :: n

    IF (n <= 1) THEN
      fact = 1
    ELSE
      fact = n * factorial(n - 1)
    END IF
  END FUNCTION factorial

END PROGRAM main
  • Explanation:
    • RECURSIVE FUNCTION factorial(n) RESULT(fact): Defines a recursive function factorial with result fact.
    • IF (n <= 1) THEN fact = 1 ELSE fact = n * factorial(n - 1) END IF: Implements the recursive logic for computing the factorial.

Functions with Array Arguments

Functions can also take arrays as arguments and return arrays.

Example: Function with Array Arguments

Here's an example of a function that adds two arrays element-wise:

PROGRAM main
  IMPLICIT NONE
  REAL, DIMENSION(3) :: array1 = (/1.0, 2.0, 3.0/)
  REAL, DIMENSION(3) :: array2 = (/4.0, 5.0, 6.0/)
  REAL, DIMENSION(3) :: result

  result = add_arrays(array1, array2)
  PRINT *, "The sum of arrays is: ", result

CONTAINS

  FUNCTION add_arrays(a, b) RESULT(sum)
    REAL, DIMENSION(3) :: sum
    REAL, DIMENSION(3), INTENT(IN) :: a, b

    sum = a + b
  END FUNCTION add_arrays

END PROGRAM main
  • Explanation:
    • REAL, DIMENSION(3) :: array1, array2, result: Declares arrays with three elements.
    • result = add_arrays(array1, array2): Calls the add_arrays function with array1 and array2.
    • FUNCTION add_arrays(a, b) RESULT(sum): Defines the add_arrays function with result sum.
    • REAL, DIMENSION(3), INTENT(IN) :: a, b: Declares input arguments a and b as arrays.
    • sum = a + b: Adds the arrays element-wise and assigns the result to sum.

Conclusion

Functions in Fortran are powerful tools that help you organize and reuse code efficiently. We covered the basics of defining functions, using different argument types, and advanced concepts like recursion and array arguments. By mastering functions, you can write more modular, readable, and maintainable Fortran programs.
