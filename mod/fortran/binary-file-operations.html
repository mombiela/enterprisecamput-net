Binary File Operations in Fortran
Modules and Packages ►

Binary file operations are crucial for efficiently handling large datasets and performing low-level data manipulation. This section will guide you through the basics to advanced techniques of binary file operations in Fortran.

Introduction to Binary Files

  • Binary Files: Unlike text files, binary files store data in a format that is not human-readable. They are used for storing data such as images, audio, and compiled programs.
  • Advantages:
    • Faster read/write operations.
    • More compact storage.
    • Direct access to data.

Opening and Closing Binary Files

To work with binary files in Fortran, you need to open the file in a specific mode and close it after operations are complete.

Opening a Binary File

Use the OPEN statement to open a binary file.

OPEN(UNIT=10, FILE='data.bin', STATUS='UNKNOWN', ACCESS='DIRECT', FORM='UNFORMATTED', RECL=4)
  • UNIT: Logical unit number.
  • FILE: Name of the file.
  • STATUS: File status ('NEW', 'OLD', 'UNKNOWN').
  • ACCESS: Access mode ('SEQUENTIAL', 'DIRECT').
  • FORM: File format ('FORMATTED', 'UNFORMATTED').
  • RECL: Record length (in bytes).

Closing a Binary File

Use the CLOSE statement to close the file.

CLOSE(UNIT=10)

Writing to a Binary File

To write data to a binary file, use the WRITE statement with the appropriate unit number.

Example: Writing Integers to a Binary File

PROGRAM WriteBinary
  INTEGER :: i, unit_num

  unit_num = 10
  OPEN(UNIT=unit_num, FILE='data.bin', STATUS='REPLACE', ACCESS='DIRECT', FORM='UNFORMATTED', RECL=4)

  DO i = 1, 10
    WRITE(UNIT=unit_num, REC=i) i
  END DO

  CLOSE(UNIT=unit_num)
END PROGRAM WriteBinary
  • REC: Record number for direct access.

Reading from a Binary File

To read data from a binary file, use the READ statement with the appropriate unit number.

Example: Reading Integers from a Binary File

PROGRAM ReadBinary
  INTEGER :: i, value, unit_num

  unit_num = 10
  OPEN(UNIT=unit_num, FILE='data.bin', STATUS='OLD', ACCESS='DIRECT', FORM='UNFORMATTED', RECL=4)

  DO i = 1, 10
    READ(UNIT=unit_num, REC=i) value
    PRINT *, 'Record', i, ':', value
  END DO

  CLOSE(UNIT=unit_num)
END PROGRAM ReadBinary

Advanced Binary File Operations

Handling Different Data Types

Fortran allows you to write and read various data types to/from binary files.

Example: Writing and Reading Real Numbers

PROGRAM WriteReadReal
  REAL :: x, y
  INTEGER :: unit_num

  unit_num = 10
  OPEN(UNIT=unit_num, FILE='realdata.bin', STATUS='REPLACE', ACCESS='DIRECT', FORM='UNFORMATTED', RECL=4)

  x = 3.14
  WRITE(UNIT=unit_num, REC=1) x

  CLOSE(UNIT=unit_num)

  OPEN(UNIT=unit_num, FILE='realdata.bin', STATUS='OLD', ACCESS='DIRECT', FORM='UNFORMATTED', RECL=4)

  READ(UNIT=unit_num, REC=1) y
  PRINT *, 'Read value:', y

  CLOSE(UNIT=unit_num)
END PROGRAM WriteReadReal

Error Handling

Proper error handling ensures that your program can gracefully handle issues such as file not found or read/write errors.

Example: Using IOSTAT for Error Handling

PROGRAM ErrorHandling
  INTEGER :: i, unit_num, io_status

  unit_num = 10
  OPEN(UNIT=unit_num, FILE='data.bin', STATUS='OLD', ACCESS='DIRECT', FORM='UNFORMATTED', RECL=4, IOSTAT=io_status)

  IF (io_status /= 0) THEN
    PRINT *, 'Error opening file. IOSTAT =', io_status
    STOP
  END IF

  DO i = 1, 10
    READ(UNIT=unit_num, REC=i, IOSTAT=io_status) value
    IF (io_status /= 0) THEN
      PRINT *, 'Error reading record', i, '. IOSTAT =', io_status
      EXIT
    END IF
    PRINT *, 'Record', i, ':', value
  END DO

  CLOSE(UNIT=unit_num)
END PROGRAM ErrorHandling

Summary

Binary file operations in Fortran are essential for efficient data handling. This section covered:

  • Basics of binary files.
  • Opening and closing files.
  • Writing and reading different data types.
  • Advanced techniques and error handling.

By mastering these operations, you can significantly enhance the performance and capabilities of your Fortran programs.
