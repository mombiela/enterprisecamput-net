Introduction to Linked Lists

Linked lists are a fundamental data structure used in computer science to store collections of elements. Unlike arrays, linked lists consist of nodes where each node contains data and a reference (or link) to the next node in the sequence. This structure allows for efficient insertion and deletion of elements.

Key Concepts

  • Node: The basic unit of a linked list, containing data and a reference to the next node.
  • Head: The first node in a linked list.
  • Tail: The last node in a linked list, which points to null or nil.
  • Pointer: A variable that holds the memory address of another variable, used to link nodes in a linked list.

Creating a Simple Linked List

To create a linked list in Fortran, we need to define a node structure and implement basic operations such as insertion and traversal.

Defining a Node

In Fortran, we can define a node using a type construct.

type :: Node
    integer :: data
    type(Node), pointer :: next => null()
end type Node

Initializing the Linked List

We start by creating the head of the linked list.

program LinkedListExample
    implicit none
    type(Node), pointer :: head, newNode

    ! Initialize the head node
    allocate(head)
    head%data = 10
    head%next => null()

    ! Print the head node data
    print *, "Head node data: ", head%data
end program LinkedListExample

Inserting Nodes

We can insert new nodes at the beginning, end, or any position in the linked list.

Inserting at the Beginning

To insert a node at the beginning, we need to update the head pointer to point to the new node.

subroutine insertAtBeginning(head, value)
    type(Node), pointer :: head
    integer, intent(in) :: value
    type(Node), pointer :: newNode

    allocate(newNode)
    newNode%data = value
    newNode%next => head
    head => newNode
end subroutine insertAtBeginning

Inserting at the End

To insert a node at the end, we traverse the list to find the last node and update its next pointer.

subroutine insertAtEnd(head, value)
    type(Node), pointer :: head
    integer, intent(in) :: value
    type(Node), pointer :: newNode, temp

    allocate(newNode)
    newNode%data = value
    newNode%next => null()

    if (.not. associated(head)) then
        head => newNode
    else
        temp => head
        do while (associated(temp%next))
            temp => temp%next
        end do
        temp%next => newNode
    end if
end subroutine insertAtEnd

Traversing the Linked List

Traversal involves visiting each node in the linked list and performing some operation, such as printing the data.

subroutine traverse(head)
    type(Node), pointer :: head
    type(Node), pointer :: temp

    temp => head
    do while (associated(temp))
        print *, "Node data: ", temp%data
        temp => temp%next
    end do
end subroutine traverse

Advanced Operations

Advanced operations on linked lists include deletion, searching, and reversing the list.

Deleting a Node

To delete a node, we need to update the pointers to bypass the node to be deleted.

subroutine deleteNode(head, value)
    type(Node), pointer :: head
    integer, intent(in) :: value
    type(Node), pointer :: temp, prev

    temp => head
    prev => null()

    do while (associated(temp))
        if (temp%data == value) then
            if (associated(prev)) then
                prev%next => temp%next
            else
                head => temp%next
            end if
            deallocate(temp)
            return
        end if
        prev => temp
        temp => temp%next
    end do
end subroutine deleteNode

Searching for a Node

To search for a node, we traverse the list and check each node's data.

function searchNode(head, value) result(found)
    type(Node), pointer :: head
    integer, intent(in) :: value
    logical :: found
    type(Node), pointer :: temp

    found = .false.
    temp => head
    do while (associated(temp))
        if (temp%data == value) then
            found = .true.
            return
        end if
        temp => temp%next
    end do
end function searchNode

Reversing the Linked List

Reversing a linked list involves changing the direction of the pointers.

subroutine reverseList(head)
    type(Node), pointer :: head
    type(Node), pointer :: prev, current, next

    prev => null()
    current => head

    do while (associated(current))
        next => current%next
        current%next => prev
        prev => current
        current => next
    end do
    head => prev
end subroutine reverseList

Conclusion

Linked lists are a versatile and fundamental data structure in programming. In Fortran, linked lists can be implemented using pointers and custom types. Understanding how to create, manipulate, and traverse linked lists is essential for solving various computational problems efficiently. By mastering these concepts, you can handle more complex data structures and algorithms in your programming journey.
