Introduction

In Fortran, control structures are essential for directing the flow of a program. One of the most fundamental control structures is the if statement, which allows you to execute certain sections of code based on specific conditions. This topic will cover the basics of if statements, including syntax, examples, and exercises to help you understand and apply this control structure effectively.

Basic If Statement

The simplest form of an if statement in Fortran checks a condition and executes a block of code if the condition is true.

Syntax

if (condition) then
    ! Code to execute if condition is true
end if

Example

program basic_if
    implicit none
    integer :: a

    a = 5

    if (a > 3) then
        print *, "a is greater than 3"
    end if
end program basic_if

In this example, the program checks if a is greater than 3. Since a is 5, the condition is true, and the message "a is greater than 3" is printed.

If-Else Statement

Sometimes, you need to execute one block of code if a condition is true and another block if the condition is false. This is where the if-else statement comes in.

Syntax

if (condition) then
    ! Code to execute if condition is true
else
    ! Code to execute if condition is false
end if

Example

program if_else
    implicit none
    integer :: a

    a = 2

    if (a > 3) then
        print *, "a is greater than 3"
    else
        print *, "a is not greater than 3"
    end if
end program if_else

In this example, the program checks if a is greater than 3. Since a is 2, the condition is false, and the message "a is not greater than 3" is printed.

If-Else If-Else Statement

For more complex decision-making, you can use multiple conditions with else if.

Syntax

if (condition1) then
    ! Code to execute if condition1 is true
else if (condition2) then
    ! Code to execute if condition2 is true
else
    ! Code to execute if none of the above conditions are true
end if

Example

program if_elseif_else
    implicit none
    integer :: a

    a = 5

    if (a > 10) then
        print *, "a is greater than 10"
    else if (a > 3) then
        print *, "a is greater than 3 but less than or equal to 10"
    else
        print *, "a is 3 or less"
    end if
end program if_elseif_else

In this example, the program checks multiple conditions. Since a is 5, the second condition is true, and the message "a is greater than 3 but less than or equal to 10" is printed.

Nested If Statements

You can also nest if statements within each other to handle more complex conditions.

Example

program nested_if
    implicit none
    integer :: a, b

    a = 5
    b = 7

    if (a > 3) then
        if (b > 6) then
            print *, "a is greater than 3 and b is greater than 6"
        else
            print *, "a is greater than 3 but b is not greater than 6"
        end if
    else
        print *, "a is not greater than 3"
    end if
end program nested_if

In this example, the program first checks if a is greater than 3. If true, it then checks if b is greater than 6. Since both conditions are true, the message "a is greater than 3 and b is greater than 6" is printed.

Exercises

  1. Write a program that checks if a number is positive, negative, or zero and prints an appropriate message.
  2. Modify the if_elseif_else example to include a condition that checks if a is exactly 5 and prints a special message.
  3. Create a nested if statement that checks two variables and prints different messages based on their values.

Conclusion

Understanding if statements in Fortran is crucial for controlling the flow of your programs. By mastering basic if, if-else, if-else if-else, and nested if statements, you can handle a wide range of decision-making scenarios in your code. Practice with the provided examples and exercises to solidify your understanding and become proficient in using if statements in Fortran.
