Multidimensional Arrays in Fortran

Multidimensional arrays are a fundamental concept in Fortran, allowing you to store and manipulate data in more than one dimension. This section will guide you from the basics to advanced usage of multidimensional arrays in Fortran.

Introduction to Multidimensional Arrays

Multidimensional arrays in Fortran are arrays with more than one index. They are used to represent data in a grid or matrix form, which is common in scientific and engineering computations.

Key Concepts

  • Dimensions: The number of indices required to access an element in the array.
  • Rank: The number of dimensions an array has.
  • Shape: The size of each dimension.

Declaring Multidimensional Arrays

To declare a multidimensional array in Fortran, you specify the dimensions in parentheses after the array name.

Example: Declaring a 2D Array

PROGRAM TwoDArray
  IMPLICIT NONE
  INTEGER, DIMENSION(3, 4) :: matrix
END PROGRAM TwoDArray

In this example, matrix is a 2D array with 3 rows and 4 columns.

Example: Declaring a 3D Array

PROGRAM ThreeDArray
  IMPLICIT NONE
  REAL, DIMENSION(2, 3, 4) :: tensor
END PROGRAM ThreeDArray

Here, tensor is a 3D array with dimensions 2x3x4.

Initializing Multidimensional Arrays

You can initialize multidimensional arrays at the time of declaration or later in the program.

Example: Initialization at Declaration

PROGRAM InitArray
  IMPLICIT NONE
  INTEGER, DIMENSION(2, 2) :: matrix = RESHAPE([1, 2, 3, 4], SHAPE(matrix))
END PROGRAM InitArray

In this example, the RESHAPE function is used to initialize the matrix with values.

Example: Initialization in the Program

PROGRAM InitArrayInProgram
  IMPLICIT NONE
  INTEGER, DIMENSION(2, 2) :: matrix
  matrix = RESHAPE([1, 2, 3, 4], SHAPE(matrix))
END PROGRAM InitArrayInProgram

Accessing Elements in Multidimensional Arrays

Elements in a multidimensional array are accessed using their indices.

Example: Accessing Elements

PROGRAM AccessElements
  IMPLICIT NONE
  INTEGER, DIMENSION(2, 2) :: matrix
  matrix = RESHAPE([1, 2, 3, 4], SHAPE(matrix))
  PRINT *, 'Element at (1,1): ', matrix(1, 1)
  PRINT *, 'Element at (2,2): ', matrix(2, 2)
END PROGRAM AccessElements

This program prints the elements at positions (1,1) and (2,2) of the matrix.

Operations on Multidimensional Arrays

You can perform various operations on multidimensional arrays, such as addition, subtraction, and multiplication.

Example: Array Addition

PROGRAM ArrayAddition
  IMPLICIT NONE
  INTEGER, DIMENSION(2, 2) :: matrix1, matrix2, result
  matrix1 = RESHAPE([1, 2, 3, 4], SHAPE(matrix1))
  matrix2 = RESHAPE([5, 6, 7, 8], SHAPE(matrix2))
  result = matrix1 + matrix2
  PRINT *, 'Result of addition:'
  PRINT *, result
END PROGRAM ArrayAddition

This program adds two 2D arrays and prints the result.

Advanced Topics

Dynamic Allocation

Fortran allows dynamic allocation of arrays, which is useful when the size of the array is not known at compile time.

Example: Dynamic Allocation

PROGRAM DynamicAllocation
  IMPLICIT NONE
  INTEGER, ALLOCATABLE, DIMENSION(:,:) :: matrix
  INTEGER :: rows, cols
  PRINT *, 'Enter number of rows and columns:'
  READ *, rows, cols
  ALLOCATE(matrix(rows, cols))
  matrix = 0  ! Initialize all elements to zero
  PRINT *, 'Matrix after allocation:'
  PRINT *, matrix
  DEALLOCATE(matrix)
END PROGRAM DynamicAllocation

This program dynamically allocates a 2D array based on user input.

Array Sections

You can work with sections of arrays, which is useful for manipulating subarrays.

Example: Array Sections

PROGRAM ArraySections
  IMPLICIT NONE
  INTEGER, DIMENSION(4, 4) :: matrix
  INTEGER :: i, j
  ! Initialize the matrix
  DO i = 1, 4
    DO j = 1, 4
      matrix(i, j) = i * 10 + j
    END DO
  END DO
  PRINT *, 'Original matrix:'
  PRINT *, matrix
  PRINT *, 'Submatrix (2:3, 2:3):'
  PRINT *, matrix(2:3, 2:3)
END PROGRAM ArraySections

This program demonstrates how to access and print a submatrix.

Conclusion

Multidimensional arrays are a powerful feature in Fortran, enabling complex data manipulation and storage. From basic declarations and initializations to advanced topics like dynamic allocation and array sections, understanding multidimensional arrays is crucial for effective Fortran programming. Practice with the provided examples and exercises to solidify your understanding.
