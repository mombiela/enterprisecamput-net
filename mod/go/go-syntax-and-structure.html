Introduction

Go, also known as Golang, is a statically typed, compiled programming language designed for simplicity and efficiency. Understanding the syntax and structure of Go is crucial for writing effective and efficient programs. This section will cover the basics of Go syntax and structure, providing a foundation for more advanced topics.

Basic Syntax

Hello World

The traditional starting point for learning any programming language is the "Hello, World!" program. Here's how it looks in Go:

package main

import "fmt"

func main() {
    fmt.Println("Hello, World!")
}
  • package main: Defines the package name. The main package is special because it defines a standalone executable program.
  • import "fmt": Imports the fmt package, which contains functions for formatted I/O.
  • func main() { ... }: The main function is the entry point of the program.

Variables and Constants

Variables in Go are declared using the var keyword, while constants are declared using the const keyword.

var name string = "John"
var age int = 30
const pi float64 = 3.14159
  • var name string = "John": Declares a variable name of type string and initializes it with "John".
  • const pi float64 = 3.14159: Declares a constant pi of type float64.

Short Variable Declarations

Go also supports short variable declarations using the := syntax.

name := "John"
age := 30
  • name := "John": Declares and initializes name with "John". The type is inferred.

Basic Data Types

Go has several basic data types:

  • int, float64, string, bool
  • Example:
var a int = 10
var b float64 = 3.14
var c string = "Hello"
var d bool = true

Control Structures

If-Else Statements

Conditional statements in Go are similar to other languages.

if age > 18 {
    fmt.Println("Adult")
} else {
    fmt.Println("Minor")
}

For Loops

Go uses the for keyword for loops.

for i := 0; i < 10; i++ {
    fmt.Println(i)
}

Switch Statements

Switch statements provide a way to execute different code based on the value of a variable.

switch day := "Monday"; day {
case "Monday":
    fmt.Println("Start of the week")
case "Friday":
    fmt.Println("End of the week")
default:
    fmt.Println("Midweek")
}

Functions

Basic Function Declaration

Functions in Go are declared using the func keyword.

func add(a int, b int) int {
    return a + b
}
  • func add(a int, b int) int: Declares a function add that takes two int parameters and returns an int.

Multiple Return Values

Go functions can return multiple values.

func swap(x, y string) (string, string) {
    return y, x
}
  • func swap(x, y string) (string, string): Declares a function swap that takes two string parameters and returns two string values.

Structs and Methods

Structs

Structs are used to group related data together.

type Person struct {
    Name string
    Age  int
}
  • type Person struct { ... }: Defines a new struct type Person.

Methods

Methods are functions with a receiver argument.

func (p Person) greet() string {
    return "Hello, my name is " + p.Name
}
  • func (p Person) greet() string: Declares a method greet on the Person type.

Packages and Modules

Creating a Package

Packages are used to organize code into reusable components.

// file: mathutil/mathutil.go
package mathutil

func Add(a, b int) int {
    return a + b
}

Using a Package

To use a package, you need to import it.

package main

import (
    "fmt"
    "path/to/mathutil"
)

func main() {
    result := mathutil.Add(1, 2)
    fmt.Println(result)
}

Conclusion

Understanding the syntax and structure of Go is essential for writing efficient and readable code. This section covered the basics, including variable declarations, control structures, functions, structs, and packages. With this foundation, you are well-equipped to explore more advanced topics in Go programming.
