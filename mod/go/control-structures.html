Control Structures in Go
Functions ►

Control structures are fundamental to programming, allowing you to dictate the flow of your program based on conditions and loops. In Go, control structures are straightforward and easy to understand. This section will cover the essential control structures in Go, including if statements, for loops, switch statements, and select statements.

If Statements

The if statement allows you to execute a block of code only if a specified condition is true.

Basic If Statement

package main

import "fmt"

func main() {
    x := 10
    if x > 5 {
        fmt.Println("x is greater than 5")
    }
}
  • Explanation: The code checks if x is greater than 5. If true, it prints "x is greater than 5".

If-Else Statement

package main

import "fmt"

func main() {
    x := 3
    if x > 5 {
        fmt.Println("x is greater than 5")
    } else {
        fmt.Println("x is not greater than 5")
    }
}
  • Explanation: If x is not greater than 5, the else block executes, printing "x is not greater than 5".

If-Else If-Else Statement

package main

import "fmt"

func main() {
    x := 7
    if x > 10 {
        fmt.Println("x is greater than 10")
    } else if x > 5 {
        fmt.Println("x is greater than 5 but less than or equal to 10")
    } else {
        fmt.Println("x is 5 or less")
    }
}
  • Explanation: This structure allows multiple conditions to be checked in sequence.

For Loops

The for loop is the only looping construct in Go, but it is versatile enough to handle all looping needs.

Basic For Loop

package main

import "fmt"

func main() {
    for i := 0; i < 5; i++ {
        fmt.Println(i)
    }
}
  • Explanation: This loop prints numbers from 0 to 4.

For Loop with Condition

package main

import "fmt"

func main() {
    x := 0
    for x < 5 {
        fmt.Println(x)
        x++
    }
}
  • Explanation: This loop continues as long as x is less than 5.

Infinite Loop

package main

import "fmt"

func main() {
    for {
        fmt.Println("Infinite loop")
        break // To prevent an actual infinite loop
    }
}
  • Explanation: An infinite loop can be created using for without any conditions. The break statement is used to exit the loop.

Switch Statements

The switch statement provides a way to execute one block of code among many based on the value of a variable.

Basic Switch Statement

package main

import "fmt"

func main() {
    day := "Tuesday"
    switch day {
    case "Monday":
        fmt.Println("Start of the work week")
    case "Tuesday":
        fmt.Println("Second day of the work week")
    default:
        fmt.Println("Another day")
    }
}
  • Explanation: The code prints a message based on the value of day.

Switch with Multiple Cases

package main

import "fmt"

func main() {
    day := "Saturday"
    switch day {
    case "Saturday", "Sunday":
        fmt.Println("Weekend")
    default:
        fmt.Println("Weekday")
    }
}
  • Explanation: Multiple cases can be combined to handle the same block of code.

Select Statements

The select statement is used with channels to handle multiple channel operations.

Basic Select Statement

package main

import (
    "fmt"
    "time"
)

func main() {
    ch1 := make(chan string)
    ch2 := make(chan string)

    go func() {
        time.Sleep(1 * time.Second)
        ch1 <- "one"
    }()

    go func() {
        time.Sleep(2 * time.Second)
        ch2 <- "two"
    }()

    select {
    case msg1 := <-ch1:
        fmt.Println("Received", msg1)
    case msg2 := <-ch2:
        fmt.Println("Received", msg2)
    }
}
  • Explanation: The select statement waits for one of the channels to send a message and then executes the corresponding case.

Conclusion

Control structures in Go are essential for managing the flow of your program. By mastering if statements, for loops, switch statements, and select statements, you can write more efficient and readable code. Practice using these constructs in various scenarios to become proficient in controlling the flow of your Go programs.
