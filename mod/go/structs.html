Introduction to Structs

Structs in Go are composite data types that group together variables under a single name. These variables, known as fields, can be of different types. Structs are useful for creating more complex data structures.

Key Concepts

  • Definition: A struct is defined using the type and struct keywords.
  • Fields: Variables within a struct, each with a name and type.
  • Instantiation: Creating an instance of a struct.
  • Accessing Fields: Using dot notation to access or modify fields.
  • Anonymous Fields: Fields without a name, allowing for more concise definitions.
  • Embedded Structs: Structs within structs, enabling composition.

Defining and Using Structs

Basic Struct Definition

To define a struct, you use the type keyword followed by the name of the struct and the struct keyword.

type Person struct {
    Name string
    Age  int
}

Instantiating a Struct

You can create an instance of a struct using the struct name followed by curly braces.

func main() {
    var p1 Person
    p1.Name = "Alice"
    p1.Age = 30

    p2 := Person{Name: "Bob", Age: 25}
    p3 := Person{"Charlie", 35} // Field names can be omitted if values are in order

    fmt.Println(p1)
    fmt.Println(p2)
    fmt.Println(p3)
}

Accessing and Modifying Fields

Fields of a struct can be accessed and modified using the dot notation.

func main() {
    p := Person{Name: "Alice", Age: 30}
    fmt.Println(p.Name) // Output: Alice
    p.Age = 31
    fmt.Println(p.Age)  // Output: 31
}

Advanced Struct Features

Anonymous Fields

Anonymous fields allow you to define a struct without explicitly naming the fields.

type Point struct {
    int
    int
}

func main() {
    p := Point{1, 2}
    fmt.Println(p.int) // Error: ambiguous selector p.int
    fmt.Println(p.int1) // Output: 1
    fmt.Println(p.int2) // Output: 2
}

Embedded Structs

Structs can be embedded within other structs, allowing for composition and reuse of fields.

type Address struct {
    City, State string
}

type Employee struct {
    Name    string
    Age     int
    Address // Embedded struct
}

func main() {
    e := Employee{
        Name: "John",
        Age:  28,
        Address: Address{
            City:  "New York",
            State: "NY",
        },
    }

    fmt.Println(e.Name)    // Output: John
    fmt.Println(e.City)    // Output: New York
    fmt.Println(e.Address) // Output: {New York NY}
}

Struct Methods

Methods can be associated with structs, allowing you to define behaviors for your data types.

type Rectangle struct {
    Width, Height int
}

// Method with value receiver
func (r Rectangle) Area() int {
    return r.Width * r.Height
}

// Method with pointer receiver
func (r *Rectangle) Scale(factor int) {
    r.Width *= factor
    r.Height *= factor
}

func main() {
    rect := Rectangle{Width: 10, Height: 5}
    fmt.Println(rect.Area()) // Output: 50

    rect.Scale(2)
    fmt.Println(rect.Area()) // Output: 200
}

Conclusion

Structs in Go provide a powerful way to group related data and define complex data structures. By understanding how to define, instantiate, and manipulate structs, as well as how to use advanced features like anonymous fields and embedded structs, you can write more organized and maintainable Go code. Struct methods further enhance the capabilities of structs by allowing you to define behaviors associated with your data types.
