Pandas is a powerful and flexible Python library used for data manipulation and analysis. It provides data structures and functions needed to manipulate structured data seamlessly. This topic will guide you through the basics to advanced features of Pandas, ensuring you can handle various data manipulation tasks efficiently.

Introduction to Pandas

  • What is Pandas?

    • Pandas is an open-source data analysis and manipulation library for Python.
    • It provides data structures like Series and DataFrame.

  • Installing Pandas

    !pip install pandas

  • Importing Pandas

    import pandas as pd

Pandas Data Structures

Series

  • Creating a Series

    import pandas as pd

data = [1, 2, 3, 4, 5]
series = pd.Series(data)
print(series)

  • Series with Custom Index

    data = [1, 2, 3, 4, 5]
index = ['a', 'b', 'c', 'd', 'e']
series = pd.Series(data, index=index)
print(series)

DataFrame

  • Creating a DataFrame

    data = {
    'Name': ['Alice', 'Bob', 'Charlie'],
    'Age': [25, 30, 35],
    'City': ['New York', 'Los Angeles', 'Chicago']
}
df = pd.DataFrame(data)
print(df)

  • Reading Data from a CSV File

    df = pd.read_csv('data.csv')
print(df.head())

Data Manipulation with Pandas

Selecting Data

  • Selecting Columns

    # Single column
print(df['Name'])

# Multiple columns
print(df[['Name', 'Age']])

  • Selecting Rows

    # By index
print(df.iloc[0])

# By label
print(df.loc[0])

Filtering Data

  • Using Conditions 
    # Filter rows where Age > 30
filtered_df = df[df['Age'] > 30]
print(filtered_df)

Adding and Modifying Columns

  • Adding a New Column

    df['Salary'] = [50000, 60000, 70000]
print(df)

  • Modifying an Existing Column

    df['Age'] = df['Age'] + 1
print(df)

Handling Missing Data

  • Checking for Missing Data

    print(df.isnull())
print(df.isnull().sum())

  • Filling Missing Data

    df['Age'].fillna(df['Age'].mean(), inplace=True)
print(df)

  • Dropping Missing Data

    df.dropna(inplace=True)
print(df)

Grouping and Aggregating Data

  • Grouping Data

    grouped = df.groupby('City')
print(grouped.mean())

  • Aggregating Data

    agg_df = df.groupby('City').agg({'Age': 'mean', 'Salary': 'sum'})
print(agg_df)

Merging and Joining DataFrames

  • Merging DataFrames

    df1 = pd.DataFrame({
    'Name': ['Alice', 'Bob'],
    'Age': [25, 30]
})

df2 = pd.DataFrame({
    'Name': ['Alice', 'Bob'],
    'Salary': [50000, 60000]
})

merged_df = pd.merge(df1, df2, on='Name')
print(merged_df)

  • Joining DataFrames

    df1 = df1.set_index('Name')
df2 = df2.set_index('Name')

joined_df = df1.join(df2)
print(joined_df)

Advanced Pandas Techniques

Pivot Tables

  • Creating a Pivot Table 
    pivot_table = df.pivot_table(values='Salary', index='City', columns='Age', aggfunc='mean')
print(pivot_table)

Time Series Analysis

  • Handling Date and Time Data

    df['Date'] = pd.to_datetime(df['Date'])
print(df)

  • Resampling Time Series Data

    ts = df.set_index('Date')
resampled_ts = ts.resample('M').mean()
print(resampled_ts)

Conclusion

Pandas is an essential tool for data manipulation in Python. It provides powerful data structures and functions that make data analysis tasks straightforward and efficient. By mastering Pandas, you can handle a wide range of data manipulation tasks, from simple data cleaning to complex data transformations and aggregations. Keep practicing with real datasets to enhance your skills and become proficient in using Pandas for data manipulation.
