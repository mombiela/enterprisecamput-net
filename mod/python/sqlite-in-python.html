Introduction to SQLite

SQLite is a C-language library that implements a small, fast, self-contained, high-reliability, full-featured, SQL database engine. It is the most widely deployed database engine in the world. In this section, we will learn how to use SQLite with Python.

Setting Up SQLite in Python

To use SQLite in Python, you need to import the sqlite3 module, which is included in the Python Standard Library.

import sqlite3

Creating a Database

You can create a new SQLite database by connecting to a database file. If the file does not exist, SQLite will create it.

conn = sqlite3.connect('example.db')
  • example.db is the name of the database file.
  • conn is the connection object.

Creating a Table

To create a table, you need to execute an SQL command using the connection object.

conn = sqlite3.connect('example.db')
cursor = conn.cursor()

cursor.execute('''
CREATE TABLE users (
    id INTEGER PRIMARY KEY,
    name TEXT,
    age INTEGER
)
''')

conn.commit()
conn.close()
  • cursor.execute runs the SQL command.
  • conn.commit saves the changes.
  • conn.close closes the connection.

Inserting Data

You can insert data into the table using the INSERT INTO SQL command.

conn = sqlite3.connect('example.db')
cursor = conn.cursor()

cursor.execute('''
INSERT INTO users (name, age) VALUES (?, ?)
''', ('Alice', 30))

conn.commit()
conn.close()
  • The ? placeholders are used to prevent SQL injection.
  • The values are provided as a tuple.

Querying Data

To retrieve data, you can use the SELECT SQL command.

conn = sqlite3.connect('example.db')
cursor = conn.cursor()

cursor.execute('SELECT * FROM users')
rows = cursor.fetchall()

for row in rows:
    print(row)

conn.close()
  • cursor.fetchall retrieves all rows from the result set.

Updating Data

To update existing data, use the UPDATE SQL command.

conn = sqlite3.connect('example.db')
cursor = conn.cursor()

cursor.execute('''
UPDATE users SET age = ? WHERE name = ?
''', (31, 'Alice'))

conn.commit()
conn.close()

Deleting Data

To delete data, use the DELETE SQL command.

conn = sqlite3.connect('example.db')
cursor = conn.cursor()

cursor.execute('''
DELETE FROM users WHERE name = ?
''', ('Alice',))

conn.commit()
conn.close()

Using Context Managers

Using context managers (with statement) ensures that the connection is properly closed even if an error occurs.

with sqlite3.connect('example.db') as conn:
    cursor = conn.cursor()
    cursor.execute('SELECT * FROM users')
    rows = cursor.fetchall()
    for row in rows:
        print(row)

Handling Exceptions

It's important to handle exceptions to avoid crashes and ensure the program runs smoothly.

try:
    conn = sqlite3.connect('example.db')
    cursor = conn.cursor()
    cursor.execute('SELECT * FROM users')
    rows = cursor.fetchall()
    for row in rows:
        print(row)
except sqlite3.Error as e:
    print(f"An error occurred: {e}")
finally:
    if conn:
        conn.close()

Summary

In this section, we covered the basics of using SQLite with Python, including:

  • Setting up SQLite in Python.
  • Creating a database and tables.
  • Inserting, querying, updating, and deleting data.
  • Using context managers and handling exceptions.

SQLite is a powerful tool for managing small to medium-sized databases, and its integration with Python makes it a great choice for many applications.
