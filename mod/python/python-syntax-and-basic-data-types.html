Introduction

Python is a versatile and widely-used programming language known for its readability and simplicity. This section will cover the fundamental syntax and basic data types in Python, providing a solid foundation for beginners.

Python Syntax

Python syntax refers to the set of rules that define how a Python program is written and interpreted.

Key Concepts

  • Indentation: Python uses indentation to define code blocks. Consistent indentation is crucial.
  • Comments: Use # for single-line comments and triple quotes (''' or """) for multi-line comments.
  • Variables: Variables do not require explicit declaration and can change type dynamically.

Code Examples

Indentation

if True:
    print("This is indented")
    if True:
        print("This is further indented")

Comments

# This is a single-line comment

"""
This is a
multi-line comment
"""

Variables

x = 5       # x is an integer
x = "Hello" # Now x is a string

Basic Data Types

Python supports several basic data types that are essential for any programming task.

Key Data Types

  • Integers: Whole numbers, e.g., 1, 42, -7
  • Floats: Decimal numbers, e.g., 3.14, 0.001, -2.5
  • Strings: Text, e.g., "Hello, World!", 'Python'
  • Booleans: True or False values, e.g., True, False

Code Examples

Integers and Floats

a = 10      # Integer
b = 3.14    # Float
c = a + b   # Result is a float: 13.14

Strings

greeting = "Hello"
name = 'Alice'
message = greeting + ", " + name + "!"  # Concatenation
print(message)  # Output: Hello, Alice!

Booleans

is_python_fun = True
is_sky_blue = False

print(is_python_fun)  # Output: True
print(is_sky_blue)    # Output: False

Summary

Understanding Python syntax and basic data types is essential for writing effective Python programs. Key points to remember:

  • Python uses indentation to define code blocks.
  • Comments are created using # for single-line and triple quotes for multi-line.
  • Variables can change type dynamically.
  • Basic data types include integers, floats, strings, and booleans.

By mastering these fundamentals, you will be well-prepared to tackle more complex programming concepts and tasks in Python.
