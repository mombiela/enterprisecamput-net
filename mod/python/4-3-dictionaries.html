4-3: Dictionaries

Introduction to Dictionaries

Dictionaries in Python are a collection of key-value pairs. They are unordered, mutable, and indexed by keys, which can be of any immutable type (e.g., strings, numbers, tuples). Dictionaries are highly efficient for lookups, insertions, and deletions.

Key Concepts

  • Dictionary Creation
  • Accessing Values
  • Adding and Modifying Entries
  • Removing Entries
  • Dictionary Methods
  • Iterating Through Dictionaries
  • Dictionary Comprehensions

Dictionary Creation

You can create a dictionary using curly braces {} or the dict() function.

# Using curly braces
my_dict = {
    "name": "Alice",
    "age": 25,
    "city": "New York"
}

# Using the dict() function
my_dict = dict(name="Alice", age=25, city="New York")

Accessing Values

You can access values in a dictionary by using the keys.

print(my_dict["name"])  # Output: Alice
print(my_dict["age"])   # Output: 25

Adding and Modifying Entries

You can add new key-value pairs or modify existing ones by assigning a value to a key.

# Adding a new key-value pair
my_dict["email"] = "alice@example.com"

# Modifying an existing value
my_dict["age"] = 26

print(my_dict)
# Output: {'name': 'Alice', 'age': 26, 'city': 'New York', 'email': 'alice@example.com'}

Removing Entries

You can remove entries using the del statement or the pop() method.

# Using del statement
del my_dict["city"]

# Using pop() method
email = my_dict.pop("email")

print(my_dict)  # Output: {'name': 'Alice', 'age': 26}
print(email)    # Output: alice@example.com

Dictionary Methods

Here are some useful dictionary methods:

Method Description
dict.keys() Returns a view object of all keys in the dictionary
dict.values() Returns a view object of all values in the dictionary
dict.items() Returns a view object of all key-value pairs in the dictionary
dict.get(key) Returns the value for the specified key, or None if the key is not found
dict.update(other) Updates the dictionary with key-value pairs from another dictionary or iterable of key-value pairs

Iterating Through Dictionaries

You can iterate through keys, values, or key-value pairs in a dictionary.

# Iterating through keys
for key in my_dict.keys():
    print(key)

# Iterating through values
for value in my_dict.values():
    print(value)

# Iterating through key-value pairs
for key, value in my_dict.items():
    print(f"{key}: {value}")

Dictionary Comprehensions

Dictionary comprehensions provide a concise way to create dictionaries.

# Creating a dictionary with squares of numbers
squares = {x: x**2 for x in range(1, 6)}
print(squares)  # Output: {1: 1, 2: 4, 3: 9, 4: 16, 5: 25}

Practical Exercises

Exercise 1: Basic Dictionary Operations

  1. Create a dictionary with the following key-value pairs: {"name": "Bob", "age": 30, "job": "Developer"}.
  2. Add a new key-value pair: "city": "San Francisco".
  3. Modify the value of the "age" key to 31.
  4. Remove the "job" key from the dictionary.
  5. Print all keys, values, and key-value pairs in the dictionary.

Solution:

# Step 1
person = {"name": "Bob", "age": 30, "job": "Developer"}

# Step 2
person["city"] = "San Francisco"

# Step 3
person["age"] = 31

# Step 4
del person["job"]

# Step 5
print("Keys:", person.keys())
print("Values:", person.values())
print("Items:", person.items())

Exercise 2: Dictionary Comprehension

  1. Create a dictionary that maps each character in the string "hello" to its ASCII value.

Solution:

ascii_map = {char: ord(char) for char in "hello"}
print(ascii_map)  # Output: {'h': 104, 'e': 101, 'l': 108, 'o': 111}

Common Mistakes and Tips

  • Using mutable types as keys: Only immutable types (e.g., strings, numbers, tuples) can be used as dictionary keys.
  • KeyError: Accessing a non-existent key will raise a KeyError. Use the get() method to avoid this.
  • Dictionary Views: Methods like keys(), values(), and items() return view objects, which reflect changes to the dictionary.

Conclusion

In this section, you learned about dictionaries in Python, including how to create, access, modify, and delete entries. You also explored dictionary methods, iteration, and comprehensions. These concepts are fundamental for efficient data management and manipulation in Python. In the next section, we will delve into another essential data structure: Sets.
