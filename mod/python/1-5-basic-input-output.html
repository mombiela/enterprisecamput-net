1-5: Basic Input and Output

In this section, we will cover the basics of input and output in Python. This includes how to get input from the user and how to display output to the screen. Understanding these concepts is crucial for interacting with users and displaying results.

Key Concepts

  1. Output using print() function
  2. Input using input() function
  3. Formatting output

Output using `print()` function

The print() function is used to display output to the screen. It can take multiple arguments and will convert them to strings before displaying them.

Syntax

print(*objects, sep=' ', end='\n', file=sys.stdout, flush=False)
  • *objects: Any number of objects to be printed.
  • sep: Separator between objects (default is a space).
  • end: String appended after the last value (default is a newline).
  • file: The file where the output is written (default is sys.stdout).
  • flush: Whether to forcibly flush the stream.

Example

print("Hello, World!")
print("Python", "is", "fun", sep="-")
print("This is the end of the line.", end=" END\n")

Explanation:

  • The first print() outputs "Hello, World!" followed by a newline.
  • The second print() outputs "Python-is-fun" with hyphens as separators.
  • The third print() outputs "This is the end of the line. END" with " END" appended instead of a newline.

Input using `input()` function

The input() function is used to get input from the user. It reads a line from input, converts it to a string, and returns it.

Syntax

input(prompt)
  • prompt: A string that is displayed to the user before input.

Example

name = input("Enter your name: ")
print("Hello, " + name + "!")

Explanation:

  • The input() function displays the prompt "Enter your name: " and waits for the user to input their name.
  • The entered name is stored in the variable name.
  • The print() function then outputs "Hello, " followed by the entered name and an exclamation mark.

Formatting Output

Python provides several ways to format output, making it easier to create readable and well-structured output.

Using str.format()

name = "Alice"
age = 30
print("Name: {}, Age: {}".format(name, age))

Explanation:

  • The str.format() method replaces the {} placeholders with the values of name and age.

Using f-strings (Python 3.6+)

name = "Alice"
age = 30
print(f"Name: {name}, Age: {age}")

Explanation:

  • F-strings provide a way to embed expressions inside string literals, using {}.

Practical Exercises

Exercise 1: Basic Output

Write a program that prints "Welcome to Python Programming!" to the screen.

Solution:

print("Welcome to Python Programming!")

Exercise 2: User Input

Write a program that asks the user for their favorite color and then prints a message saying "Your favorite color is [color]".

Solution:

color = input("Enter your favorite color: ")
print(f"Your favorite color is {color}")

Exercise 3: Formatting Output

Write a program that asks the user for their name and age, and then prints a message saying "Name: [name], Age: [age]".

Solution:

name = input("Enter your name: ")
age = input("Enter your age: ")
print(f"Name: {name}, Age: {age}")

Common Mistakes and Tips

  • Forgetting to convert input: The input() function always returns a string. If you need a different type (e.g., integer), you must convert it. 
    age = int(input("Enter your age: "))  # Convert input to integer
  • Using the wrong separator or end character: Remember that print() can take optional sep and end parameters to customize output.

Conclusion

In this section, we learned how to use the print() function to display output and the input() function to get input from the user. We also explored different ways to format output for better readability. These basic input and output operations are fundamental for creating interactive Python programs. In the next module, we will dive into control structures, which will allow us to make decisions and repeat actions in our programs.
