3-2: Function Arguments and Return Values

In this section, we will delve into the details of function arguments and return values in Python. Understanding these concepts is crucial for writing reusable and modular code.

Key Concepts

  1. Function Arguments:

    • Positional Arguments: Arguments that need to be passed in the correct order.
    • Keyword Arguments: Arguments that are passed by explicitly stating the parameter name.
    • Default Arguments: Arguments that assume a default value if a value is not provided.
    • Variable-Length Arguments: Functions that accept an arbitrary number of arguments using *args and **kwargs.

  2. Return Values:

    • Returning Single Values: Functions that return a single value.
    • Returning Multiple Values: Functions that return multiple values using tuples.

Function Arguments

Positional Arguments

Positional arguments are the most common type of arguments. They must be passed in the correct order.

def greet(name, message):
    print(f"Hello, {name}! {message}")

# Calling the function with positional arguments
greet("Alice", "Welcome to the Python course!")

Keyword Arguments

Keyword arguments are passed by explicitly stating the parameter name, which makes the function call more readable.

def greet(name, message):
    print(f"Hello, {name}! {message}")

# Calling the function with keyword arguments
greet(name="Bob", message="Good to see you!")

Default Arguments

Default arguments are used when you want to provide a default value for a parameter. If the caller does not provide a value, the default value is used.

def greet(name, message="Welcome!"):
    print(f"Hello, {name}! {message}")

# Calling the function with and without the default argument
greet("Charlie")
greet("David", "How are you?")

Variable-Length Arguments

Variable-length arguments allow a function to accept an arbitrary number of arguments. This is useful when you do not know beforehand how many arguments will be passed.

  • *args: Used to pass a variable number of non-keyword arguments.
  • **kwargs: Used to pass a variable number of keyword arguments.
def greet(*names):
    for name in names:
        print(f"Hello, {name}!")

# Calling the function with variable-length arguments
greet("Eve", "Frank", "Grace")

def greet_with_messages(**kwargs):
    for name, message in kwargs.items():
        print(f"Hello, {name}! {message}")

# Calling the function with variable-length keyword arguments
greet_with_messages(Alice="Welcome!", Bob="Good to see you!")

Return Values

Returning Single Values

A function can return a single value using the return statement.

def add(a, b):
    return a + b

# Calling the function and storing the return value
result = add(3, 5)
print(result)  # Output: 8

Returning Multiple Values

A function can return multiple values by returning a tuple.

def get_user_info():
    name = "Alice"
    age = 30
    return name, age

# Calling the function and unpacking the return values
name, age = get_user_info()
print(name)  # Output: Alice
print(age)   # Output: 30

Practical Exercises

Exercise 1: Positional and Keyword Arguments

Write a function describe_pet that takes two arguments: animal_type and pet_name. The function should print a message describing the pet. Call the function using both positional and keyword arguments.

def describe_pet(animal_type, pet_name):
    print(f"I have a {animal_type} named {pet_name}.")

# Your code here

Solution:

def describe_pet(animal_type, pet_name):
    print(f"I have a {animal_type} named {pet_name}.")

# Calling the function with positional arguments
describe_pet("dog", "Buddy")

# Calling the function with keyword arguments
describe_pet(animal_type="cat", pet_name="Whiskers")

Exercise 2: Default Arguments

Write a function make_shirt that takes two arguments: size and message. The function should print a message about the shirt being made. The size should have a default value of "Large" and the message should have a default value of "I love Python".

def make_shirt(size="Large", message="I love Python"):
    print(f"Making a {size} shirt with the message: '{message}'")

# Your code here

Solution:

def make_shirt(size="Large", message="I love Python"):
    print(f"Making a {size} shirt with the message: '{message}'")

# Calling the function with default arguments
make_shirt()

# Calling the function with custom arguments
make_shirt(size="Medium", message="Hello World!")

Exercise 3: Variable-Length Arguments

Write a function summarize_order that takes a variable number of arguments representing items in an order. The function should print each item in the order.

def summarize_order(*items):
    print("Order summary:")
    for item in items:
        print(f"- {item}")

# Your code here

Solution:

def summarize_order(*items):
    print("Order summary:")
    for item in items:
        print(f"- {item}")

# Calling the function with variable-length arguments
summarize_order("Pizza", "Burger", "Salad")

Conclusion

In this section, we covered the different types of function arguments in Python, including positional, keyword, default, and variable-length arguments. We also explored how to return single and multiple values from functions. Understanding these concepts will help you write more flexible and reusable functions.

Next, we will learn about lambda functions, which are a concise way to define small, anonymous functions.
