Introduction to Django

Django is a high-level Python web framework that encourages rapid development and clean, pragmatic design. It is built by experienced developers to take care of much of the hassle of web development, so you can focus on writing your app without needing to reinvent the wheel.

  • Key Features:
    • Fast development
    • Secure
    • Scalable
    • Versatile

Setting Up Django

Before diving into Django, you need to set up your environment.

  • Installation:

    pip install django

  • Creating a Project:

    django-admin startproject myproject

  • Running the Development Server:

    cd myproject
python manage.py runserver

Django Project Structure

Understanding the structure of a Django project is crucial for effective development.

  • Key Components:
    • manage.py: A command-line utility that lets you interact with your project.
    • settings.py: Configuration for your Django project.
    • urls.py: URL declarations for your project.
    • wsgi.py: An entry-point for WSGI-compatible web servers to serve your project.

Creating a Django App

Django projects are composed of multiple apps, each serving a specific purpose.

  • Creating an App:

    python manage.py startapp myapp

  • Registering the App: Add your app to the INSTALLED_APPS list in settings.py:

    INSTALLED_APPS = [
    ...
    'myapp',
]

Models and Databases

Django models define the structure of your database tables.

  • Creating a Model:

    from django.db import models

class Article(models.Model):
    title = models.CharField(max_length=100)
    content = models.TextField()
    published_date = models.DateTimeField(auto_now_add=True)

  • Applying Migrations:

    python manage.py makemigrations
python manage.py migrate

Views and Templates

Views handle the logic of your application, while templates handle the presentation.

  • Creating a View:

    from django.shortcuts import render
from .models import Article

def index(request):
    articles = Article.objects.all()
    return render(request, 'index.html', {'articles': articles})

  • Creating a Template:

    <!-- templates/index.html -->
<!DOCTYPE html>
<html>
<head>
    <title>Article List</title>
</head>
<body>
    <h1>Articles</h1>
    <ul>
        {% for article in articles %}
            <li>{{ article.title }} - {{ article.published_date }}</li>
        {% endfor %}
    </ul>
</body>
</html>

URL Routing

URL routing maps URLs to views.

  • Defining URLs: 
    from django.urls import path
from . import views

urlpatterns = [
    path('', views.index, name='index'),
]

Forms and User Input

Django forms handle user input and validation.

  • Creating a Form:

    from django import forms

class ArticleForm(forms.ModelForm):
    class Meta:
        model = Article
        fields = ['title', 'content']

  • Handling Form Submission:

    from django.shortcuts import render, redirect
from .forms import ArticleForm

def create_article(request):
    if request.method == 'POST':
        form = ArticleForm(request.POST)
        if form.is_valid():
            form.save()
            return redirect('index')
    else:
        form = ArticleForm()
    return render(request, 'create_article.html', {'form': form})

  • Template for Form:

    <!-- templates/create_article.html -->
<!DOCTYPE html>
<html>
<head>
    <title>Create Article</title>
</head>
<body>
    <h1>Create Article</h1>
    <form method="post">
        {% csrf_token %}
        {{ form.as_p }}
        <button type="submit">Save</button>
    </form>
</body>
</html>

Admin Interface

Django provides a built-in admin interface to manage your models.

  • Enabling the Admin Interface:

    from django.contrib import admin
from .models import Article

admin.site.register(Article)

  • Accessing the Admin Interface:

    • Run the development server and navigate to /admin.
    • Log in using the superuser credentials you created.

Conclusion

Django is a powerful framework that simplifies web development by providing a robust set of tools and features. By understanding its core components—models, views, templates, and URL routing—you can build scalable and maintainable web applications efficiently. Continue exploring Django's extensive documentation and community resources to deepen your knowledge and skills.
