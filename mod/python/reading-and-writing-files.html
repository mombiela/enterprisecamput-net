Introduction

Reading from and writing to files are fundamental tasks in programming. In Python, these operations are straightforward and can be performed using built-in functions and methods. This section will guide you through the basics of file handling, from opening and closing files to more advanced techniques like handling different file formats.

Basics of File Handling

Opening and Closing Files

  • Opening a File: Use the open() function. 
    file = open('example.txt', 'r')  # 'r' for read mode
  • Closing a File: Always close the file after operations using the close() method. 
    file.close()

File Modes

  • Read (r): Opens a file for reading.
  • Write (w): Opens a file for writing (creates a new file or truncates an existing file).
  • Append (a): Opens a file for appending (creates a new file if it doesn't exist).
  • Read and Write (r+): Opens a file for both reading and writing.

Example: Reading a File

with open('example.txt', 'r') as file:
    content = file.read()
    print(content)
  • Explanation: The with statement ensures the file is properly closed after its suite finishes, even if an exception is raised.

Example: Writing to a File

with open('example.txt', 'w') as file:
    file.write('Hello, World!')
  • Explanation: This will write the string 'Hello, World!' to example.txt. If the file already exists, it will be overwritten.

Reading Files

Reading Entire File

  • Using read(): Reads the entire file content. 
    with open('example.txt', 'r') as file:
    content = file.read()
    print(content)

Reading Line by Line

  • Using readline(): Reads one line at a time.

    with open('example.txt', 'r') as file:
    line = file.readline()
    while line:
        print(line, end='')
        line = file.readline()

  • Using readlines(): Reads all lines into a list.

    with open('example.txt', 'r') as file:
    lines = file.readlines()
    for line in lines:
        print(line, end='')

Writing Files

Writing Multiple Lines

  • Using writelines(): Writes a list of strings to a file. 
    lines = ['First line\n', 'Second line\n', 'Third line\n']
with open('example.txt', 'w') as file:
    file.writelines(lines)

Appending to a File

  • Using a mode: Appends content to the end of the file. 
    with open('example.txt', 'a') as file:
    file.write('This is an appended line.\n')

Advanced File Handling

Working with Binary Files

  • Reading Binary Files: Use rb mode.

    with open('example.bin', 'rb') as file:
    content = file.read()
    print(content)

  • Writing Binary Files: Use wb mode.

    with open('example.bin', 'wb') as file:
    file.write(b'\x00\x01\x02\x03')

Handling Different File Formats

  • CSV Files: Use the csv module.

    import csv

# Reading a CSV file
with open('example.csv', 'r') as file:
    reader = csv.reader(file)
    for row in reader:
        print(row)

# Writing to a CSV file
with open('example.csv', 'w', newline='') as file:
    writer = csv.writer(file)
    writer.writerow(['Name', 'Age'])
    writer.writerow(['Alice', 30])

  • JSON Files: Use the json module.

    import json

# Reading a JSON file
with open('example.json', 'r') as file:
    data = json.load(file)
    print(data)

# Writing to a JSON file
data = {'name': 'Alice', 'age': 30}
with open('example.json', 'w') as file:
    json.dump(data, file)

Summary

In this section, you learned the basics and some advanced techniques for reading and writing files in Python. You now know how to:

  • Open and close files.
  • Read from and write to text files.
  • Handle different file modes.
  • Work with binary files.
  • Read and write CSV and JSON files.

Understanding file handling is crucial for many applications, such as data processing, configuration management, and logging. Practice these concepts to become proficient in managing files in Python.
