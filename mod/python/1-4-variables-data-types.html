1-4 Variables and Data Types

Introduction

In this section, we will cover the basics of variables and data types in Python. Understanding these concepts is crucial as they form the foundation of any programming language. By the end of this section, you will be able to declare variables, understand different data types, and perform basic operations with them.

Key Concepts

  1. Variables
  2. Data Types
  3. Type Conversion
  4. Basic Operations

Variables

Variables are used to store data that can be referenced and manipulated in a program. In Python, you don't need to declare the type of a variable explicitly. The type is inferred from the value assigned to it.

Syntax

variable_name = value

Example

# Assigning values to variables
name = "Alice"
age = 25
height = 5.6
is_student = True

print(name)       # Output: Alice
print(age)        # Output: 25
print(height)     # Output: 5.6
print(is_student) # Output: True

Data Types

Python supports various data types. The most commonly used ones are:

  1. Integers: Whole numbers without a decimal point.
  2. Floats: Numbers with a decimal point.
  3. Strings: Sequence of characters enclosed in quotes.
  4. Booleans: Represents True or False values.

Examples

# Integer
num1 = 10
print(type(num1))  # Output: <class 'int'>

# Float
num2 = 10.5
print(type(num2))  # Output: <class 'float'>

# String
text = "Hello, World!"
print(type(text))  # Output: <class 'str'>

# Boolean
is_valid = True
print(type(is_valid))  # Output: <class 'bool'>

Type Conversion

You can convert one data type to another using type conversion functions.

Common Type Conversion Functions

  • int(): Converts to integer
  • float(): Converts to float
  • str(): Converts to string
  • bool(): Converts to boolean

Examples

# Convert float to int
num = 10.5
num_int = int(num)
print(num_int)  # Output: 10

# Convert int to float
num = 10
num_float = float(num)
print(num_float)  # Output: 10.0

# Convert int to string
num = 10
num_str = str(num)
print(num_str)  # Output: '10'

# Convert string to int
text = "123"
text_int = int(text)
print(text_int)  # Output: 123

# Convert string to float
text = "123.45"
text_float = float(text)
print(text_float)  # Output: 123.45

Basic Operations

You can perform basic arithmetic operations on variables.

Examples

# Addition
a = 10
b = 5
result = a + b
print(result)  # Output: 15

# Subtraction
result = a - b
print(result)  # Output: 5

# Multiplication
result = a * b
print(result)  # Output: 50

# Division
result = a / b
print(result)  # Output: 2.0

# Modulus (remainder)
result = a % b
print(result)  # Output: 0

Practical Exercises

Exercise 1

Create variables to store your name, age, and whether you are a student. Print these variables.

Solution:

name = "John Doe"
age = 20
is_student = True

print("Name:", name)
print("Age:", age)
print("Is Student:", is_student)

Exercise 2

Convert the string "100" to an integer and add 50 to it. Print the result.

Solution:

text = "100"
num = int(text)
result = num + 50
print(result)  # Output: 150

Exercise 3

Perform the following operations and print the results:

  • Subtract 15 from 45
  • Multiply 7 by 8
  • Divide 56 by 7
  • Find the remainder when 29 is divided by 5

Solution:

# Subtraction
result = 45 - 15
print(result)  # Output: 30

# Multiplication
result = 7 * 8
print(result)  # Output: 56

# Division
result = 56 / 7
print(result)  # Output: 8.0

# Modulus
result = 29 % 5
print(result)  # Output: 4

Common Mistakes and Tips

  • Mistake: Using the wrong data type for operations.
    • Tip: Ensure you are using compatible data types for operations. For example, you can't add a string and an integer directly.
  • Mistake: Forgetting to convert data types.
    • Tip: Use type conversion functions when necessary to avoid errors.

Conclusion

In this section, we covered the basics of variables and data types in Python. You learned how to declare variables, understand different data types, perform type conversions, and execute basic operations. These concepts are fundamental and will be used throughout your programming journey. In the next section, we will explore basic input and output operations in Python.
