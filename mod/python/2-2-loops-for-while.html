Topic 2-2: Loops: for and while

Introduction

Loops are fundamental control structures in programming that allow you to execute a block of code repeatedly. Python supports two main types of loops: for loops and while loops. This section will cover both types, their syntax, and practical examples to help you understand how to use them effectively.

1. `for` Loops

Syntax

The for loop in Python is used to iterate over a sequence (such as a list, tuple, dictionary, set, or string) and execute a block of code for each item in the sequence.

for item in sequence:
    # Code to execute for each item

Example

Let's start with a simple example of iterating over a list of numbers.

numbers = [1, 2, 3, 4, 5]
for number in numbers:
    print(number)

Explanation:

  • numbers is a list containing five integers.
  • The for loop iterates over each item in the list and prints it.

Using range()

The range() function generates a sequence of numbers, which is useful for iterating a specific number of times.

for i in range(5):
    print(i)

Explanation:

  • range(5) generates numbers from 0 to 4.
  • The for loop iterates over these numbers and prints each one.

2. `while` Loops

Syntax

The while loop in Python repeatedly executes a block of code as long as a given condition is True.

while condition:
    # Code to execute while the condition is True

Example

Here's an example of a while loop that prints numbers from 1 to 5.

i = 1
while i <= 5:
    print(i)
    i += 1

Explanation:

  • The loop starts with i equal to 1.
  • The condition i <= 5 is checked before each iteration.
  • The loop prints the value of i and then increments it by 1.
  • The loop stops when i becomes greater than 5.

3. Practical Examples

Example 1: Summing Numbers with a for Loop

Calculate the sum of all numbers in a list.

numbers = [1, 2, 3, 4, 5]
total = 0
for number in numbers:
    total += number
print("Total sum:", total)

Explanation:

  • Initialize total to 0.
  • Iterate over each number in the numbers list.
  • Add each number to total.
  • Print the final sum.

Example 2: Finding the First Even Number with a while Loop

Find the first even number in a list.

numbers = [1, 3, 5, 6, 7, 8]
i = 0
while i < len(numbers):
    if numbers[i] % 2 == 0:
        print("First even number:", numbers[i])
        break
    i += 1

Explanation:

  • Initialize i to 0.
  • Use a while loop to iterate through the list.
  • Check if the current number is even.
  • If an even number is found, print it and break out of the loop.
  • Increment i to move to the next number.

4. Exercises

Exercise 1: Print Odd Numbers

Write a for loop to print all odd numbers from 1 to 10.

# Your code here

Solution:

for i in range(1, 11):
    if i % 2 != 0:
        print(i)

Exercise 2: Factorial Calculation

Write a while loop to calculate the factorial of a given number n.

n = 5
# Your code here

Solution:

n = 5
factorial = 1
i = 1
while i <= n:
    factorial *= i
    i += 1
print("Factorial of", n, "is", factorial)

Common Mistakes and Tips

  • Off-by-One Errors: Ensure your loop conditions are correctly set to avoid iterating one time too many or too few.
  • Infinite Loops: Make sure the condition in a while loop will eventually become False to avoid infinite loops.
  • Using break and continue: Use break to exit a loop early and continue to skip the current iteration and proceed to the next one.

Conclusion

In this section, you learned about for and while loops in Python. You explored their syntax, practical examples, and common use cases. Loops are powerful tools for automating repetitive tasks and are essential for efficient programming. In the next topic, we will delve into control flow statements like break, continue, and pass to further control the execution of loops.
