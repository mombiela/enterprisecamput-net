8-1: NumPy for Numerical Computing

Introduction to NumPy

NumPy (Numerical Python) is a powerful library for numerical computing in Python. It provides support for arrays, matrices, and many mathematical functions to operate on these data structures efficiently.

Key Concepts

  1. NumPy Arrays:

    • One-dimensional arrays (vectors)
    • Two-dimensional arrays (matrices)
    • Multi-dimensional arrays

  2. Array Creation:

    • Using np.array()
    • Using built-in functions like np.zeros(), np.ones(), np.arange(), np.linspace()

  3. Array Operations:

    • Element-wise operations
    • Matrix operations
    • Broadcasting

  4. Indexing and Slicing:

    • Accessing elements
    • Slicing arrays
    • Boolean indexing

  5. Mathematical Functions:

    • Basic arithmetic operations
    • Statistical functions
    • Linear algebra functions

Practical Examples

1. Importing NumPy

import numpy as np

2. Creating Arrays

# Creating a one-dimensional array
arr1 = np.array([1, 2, 3, 4, 5])
print(arr1)

# Creating a two-dimensional array
arr2 = np.array([[1, 2, 3], [4, 5, 6]])
print(arr2)

# Creating an array of zeros
zeros = np.zeros((3, 3))
print(zeros)

# Creating an array of ones
ones = np.ones((2, 4))
print(ones)

# Creating an array with a range of values
range_arr = np.arange(0, 10, 2)
print(range_arr)

# Creating an array with linearly spaced values
linspace_arr = np.linspace(0, 1, 5)
print(linspace_arr)

3. Array Operations

# Element-wise addition
arr1 = np.array([1, 2, 3])
arr2 = np.array([4, 5, 6])
sum_arr = arr1 + arr2
print(sum_arr)

# Element-wise multiplication
prod_arr = arr1 * arr2
print(prod_arr)

# Matrix multiplication
mat1 = np.array([[1, 2], [3, 4]])
mat2 = np.array([[5, 6], [7, 8]])
mat_mul = np.dot(mat1, mat2)
print(mat_mul)

4. Indexing and Slicing

arr = np.array([1, 2, 3, 4, 5])

# Accessing elements
print(arr[0])  # Output: 1
print(arr[4])  # Output: 5

# Slicing arrays
print(arr[1:4])  # Output: [2 3 4]

# Boolean indexing
print(arr[arr > 3])  # Output: [4 5]

5. Mathematical Functions

arr = np.array([1, 2, 3, 4, 5])

# Basic arithmetic operations
print(np.sum(arr))  # Output: 15
print(np.mean(arr))  # Output: 3.0
print(np.max(arr))  # Output: 5
print(np.min(arr))  # Output: 1

# Statistical functions
print(np.std(arr))  # Output: 1.4142135623730951

# Linear algebra functions
mat = np.array([[1, 2], [3, 4]])
print(np.linalg.inv(mat))  # Output: [[-2.   1. ]
                           #          [ 1.5 -0.5]]

Practical Exercises

Exercise 1: Array Creation and Operations

Task: Create a 3x3 matrix with values ranging from 1 to 9. Perform element-wise addition with another 3x3 matrix of ones. Then, calculate the sum of all elements in the resulting matrix.

Solution:

import numpy as np

# Create a 3x3 matrix with values from 1 to 9
matrix = np.arange(1, 10).reshape(3, 3)
print("Original Matrix:\n", matrix)

# Create a 3x3 matrix of ones
ones_matrix = np.ones((3, 3))
print("Ones Matrix:\n", ones_matrix)

# Perform element-wise addition
result_matrix = matrix + ones_matrix
print("Resulting Matrix:\n", result_matrix)

# Calculate the sum of all elements
total_sum = np.sum(result_matrix)
print("Sum of all elements:", total_sum)

Exercise 2: Indexing and Slicing

Task: Given a 1D array of integers from 10 to 20, extract the sub-array from index 3 to 7 and find the mean of this sub-array.

Solution:

import numpy as np

# Create a 1D array of integers from 10 to 20
arr = np.arange(10, 21)
print("Original Array:", arr)

# Extract the sub-array from index 3 to 7
sub_array = arr[3:8]
print("Sub-array:", sub_array)

# Find the mean of the sub-array
mean_value = np.mean(sub_array)
print("Mean of sub-array:", mean_value)

Common Mistakes and Tips

  • Mistake: Confusing element-wise multiplication with matrix multiplication.

    • Tip: Use np.dot() for matrix multiplication and * for element-wise multiplication.

  • Mistake: Forgetting to import NumPy before using its functions.

    • Tip: Always start your script with import numpy as np.

  • Mistake: Misunderstanding the shape of arrays when reshaping.

    • Tip: Ensure the total number of elements matches when reshaping arrays.

Conclusion

In this section, we covered the basics of NumPy, including array creation, operations, indexing, slicing, and mathematical functions. These fundamental concepts are essential for numerical computing and data manipulation in Python. In the next section, we will explore Pandas, a powerful library for data analysis.
