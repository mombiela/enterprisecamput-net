8-2: Pandas for Data Analysis

Introduction to Pandas

Pandas is a powerful and flexible open-source data analysis and manipulation library for Python. It provides data structures like DataFrame and Series, which are essential for handling structured data efficiently.

Key Concepts

  1. Series: A one-dimensional labeled array capable of holding any data type.
  2. DataFrame: A two-dimensional labeled data structure with columns of potentially different types.
  3. Indexing and Selecting Data: Techniques to access and manipulate data within Series and DataFrames.
  4. Data Cleaning: Handling missing data, duplicates, and data transformation.
  5. Data Aggregation and Grouping: Summarizing data using group operations.
  6. Merging and Joining: Combining multiple DataFrames.

Setting Up Pandas

To start using Pandas, you need to install it using pip:

pip install pandas

Importing Pandas

import pandas as pd

Series

A Series is a one-dimensional array-like object containing an array of data and an associated array of data labels, called its index.

Creating a Series

import pandas as pd

data = [1, 2, 3, 4, 5]
series = pd.Series(data)
print(series)

Output:

0    1
1    2
2    3
3    4
4    5
dtype: int64

DataFrame

A DataFrame is a two-dimensional, size-mutable, and potentially heterogeneous tabular data structure with labeled axes (rows and columns).

Creating a DataFrame

import pandas as pd

data = {
    'Name': ['Alice', 'Bob', 'Charlie'],
    'Age': [25, 30, 35],
    'City': ['New York', 'Los Angeles', 'Chicago']
}
df = pd.DataFrame(data)
print(df)

Output:

      Name  Age         City
0    Alice   25     New York
1      Bob   30  Los Angeles
2  Charlie   35      Chicago

Indexing and Selecting Data

Selecting a Column

print(df['Name'])

Output:

0      Alice
1        Bob
2    Charlie
Name: Name, dtype: object

Selecting Multiple Columns

print(df[['Name', 'City']])

Output:

      Name         City
0    Alice     New York
1      Bob  Los Angeles
2  Charlie      Chicago

Selecting Rows by Index

print(df.iloc[0])  # First row

Output:

Name      Alice
Age          25
City    New York
Name: 0, dtype: object

Data Cleaning

Handling Missing Data

import numpy as np

data = {
    'Name': ['Alice', 'Bob', 'Charlie', 'David'],
    'Age': [25, 30, np.nan, 40],
    'City': ['New York', 'Los Angeles', 'Chicago', np.nan]
}
df = pd.DataFrame(data)

# Drop rows with missing values
df_cleaned = df.dropna()
print(df_cleaned)

# Fill missing values
df_filled = df.fillna({'Age': df['Age'].mean(), 'City': 'Unknown'})
print(df_filled)

Data Aggregation and Grouping

Grouping Data

data = {
    'Name': ['Alice', 'Bob', 'Charlie', 'Alice', 'Bob'],
    'Sales': [200, 150, 300, 400, 500]
}
df = pd.DataFrame(data)

grouped = df.groupby('Name').sum()
print(grouped)

Output:

         Sales
Name          
Alice      600
Bob        650
Charlie    300

Merging and Joining

Merging DataFrames

data1 = {
    'Name': ['Alice', 'Bob', 'Charlie'],
    'Age': [25, 30, 35]
}
data2 = {
    'Name': ['Alice', 'Bob', 'David'],
    'City': ['New York', 'Los Angeles', 'San Francisco']
}
df1 = pd.DataFrame(data1)
df2 = pd.DataFrame(data2)

merged_df = pd.merge(df1, df2, on='Name', how='inner')
print(merged_df)

Output:

    Name  Age         City
0  Alice   25     New York
1    Bob   30  Los Angeles

Practical Exercises

Exercise 1: Creating and Manipulating DataFrames

  1. Create a DataFrame with the following data:
    • Names: John, Jane, Jim
    • Ages: 28, 34, 29
    • Cities: Boston, Seattle, Austin
  2. Select the 'Ages' column.
  3. Select the first row.
  4. Add a new column 'Salary' with values 70000, 80000, 75000.

Solution

data = {
    'Name': ['John', 'Jane', 'Jim'],
    'Age': [28, 34, 29],
    'City': ['Boston', 'Seattle', 'Austin']
}
df = pd.DataFrame(data)

# Select the 'Ages' column
print(df['Age'])

# Select the first row
print(df.iloc[0])

# Add a new column 'Salary'
df['Salary'] = [70000, 80000, 75000]
print(df)

Conclusion

In this section, we covered the basics of using Pandas for data analysis, including creating Series and DataFrames, indexing and selecting data, data cleaning, aggregation, and merging. These foundational skills will enable you to handle and analyze data efficiently in Python. Next, we will explore data visualization with Matplotlib.
