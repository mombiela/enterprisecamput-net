9-1: Introduction to Flask

Overview

Flask is a lightweight WSGI web application framework in Python. It is designed to make getting started quick and easy, with the ability to scale up to complex applications. Flask is known for its simplicity, flexibility, and fine-grained control.

Key Concepts

  1. WSGI (Web Server Gateway Interface): A specification for a simple and universal interface between web servers and web applications or frameworks for the Python programming language.
  2. Flask Application: The central object of the Flask framework, which handles all the requests and responses.
  3. Routes: URL patterns that are mapped to view functions in Flask.
  4. View Functions: Functions in Flask that are mapped to routes and handle the logic for what should be displayed on a web page.

Setting Up Flask

Step 1: Install Flask

To install Flask, you can use pip, the Python package installer. Open your terminal and run:

pip install Flask

Step 2: Create a Basic Flask Application

Create a new directory for your Flask project and navigate into it. Then, create a new Python file, e.g., app.py.

# app.py
from flask import Flask

app = Flask(__name__)

@app.route('/')
def hello_world():
    return 'Hello, World!'

if __name__ == '__main__':
    app.run(debug=True)

Explanation

  • Importing Flask: The Flask class is imported from the flask module.
  • Creating an Application Instance: app = Flask(__name__) creates an instance of the Flask class. The __name__ variable is passed to help Flask determine the root path for the application.
  • Defining a Route: The @app.route('/') decorator defines the route for the URL /. The function hello_world is called when this URL is accessed.
  • Running the Application: app.run(debug=True) starts the Flask application. The debug=True argument enables debug mode, which provides helpful error messages and auto-reloads the server when code changes.

Running the Application

To run the Flask application, navigate to the directory containing app.py and execute:

python app.py

Open your web browser and go to http://127.0.0.1:5000/. You should see "Hello, World!" displayed.

Practical Exercise

Exercise 1: Create a New Route

  1. Add a new route to your Flask application that returns "Welcome to Flask!" when the URL /welcome is accessed.
# app.py
from flask import Flask

app = Flask(__name__)

@app.route('/')
def hello_world():
    return 'Hello, World!'

@app.route('/welcome')
def welcome():
    return 'Welcome to Flask!'

if __name__ == '__main__':
    app.run(debug=True)
  1. Run the application and navigate to http://127.0.0.1:5000/welcome in your web browser.

Common Mistakes and Tips

  • Indentation Errors: Ensure that your code is properly indented, as Python is sensitive to indentation.
  • Debug Mode: Always use debug=True during development to get detailed error messages.
  • Port Conflicts: If port 5000 is already in use, you can specify a different port by modifying app.run(port=5001).

Conclusion

In this lesson, you learned the basics of setting up a Flask application, defining routes, and creating view functions. You also practiced creating a new route and running your Flask application. In the next lesson, you will learn how to create a simple web application using Flask.
