Topic 3-4: Modules and Packages

Introduction

In Python, modules and packages are essential for organizing and managing code. They allow you to break down large programs into smaller, manageable, and reusable components. This topic will cover the basics of creating and using modules and packages in Python.

Key Concepts

  1. Modules:

    • A module is a file containing Python definitions and statements.
    • The file name is the module name with the suffix .py added.
    • Modules can define functions, classes, and variables, and can also include runnable code.

  2. Packages:

    • A package is a way of structuring Python’s module namespace by using "dotted module names".
    • A package is a directory containing a special file __init__.py and can contain sub-packages and modules.

Creating and Using Modules

Creating a Module

To create a module, simply save the code you want in a file with a .py extension.

Example: mymodule.py

# mymodule.py

def greet(name):
    return f"Hello, {name}!"

def add(a, b):
    return a + b

PI = 3.14159

Using a Module

You can import the module into another script using the import statement.

Example: main.py

# main.py

import mymodule

print(mymodule.greet("Alice"))  # Output: Hello, Alice!
print(mymodule.add(5, 3))       # Output: 8
print(mymodule.PI)              # Output: 3.14159

Importing Specific Attributes

You can import specific functions, classes, or variables from a module using the from ... import ... statement.

Example: main.py

# main.py

from mymodule import greet, PI

print(greet("Bob"))  # Output: Hello, Bob!
print(PI)            # Output: 3.14159

Aliasing

You can give a module or an attribute an alias using the as keyword.

Example: main.py

# main.py

import mymodule as mm

print(mm.greet("Charlie"))  # Output: Hello, Charlie!
print(mm.add(10, 5))        # Output: 15

Creating and Using Packages

Creating a Package

A package is a directory that contains a special file __init__.py and can contain sub-packages and modules.

Example: Directory Structure

mypackage/
    __init__.py
    module1.py
    module2.py

Example: mypackage/module1.py

# module1.py

def func1():
    return "Function 1 from module1"

Example: mypackage/module2.py

# module2.py

def func2():
    return "Function 2 from module2"

Example: mypackage/__init__.py

# __init__.py

from .module1 import func1
from .module2 import func2

Using a Package

You can import the package and use its modules and functions.

Example: main.py

# main.py

import mypackage

print(mypackage.func1())  # Output: Function 1 from module1
print(mypackage.func2())  # Output: Function 2 from module2

Practical Exercises

Exercise 1: Creating and Using a Module

  1. Create a module named math_operations.py with the following functions:

    • add(a, b): Returns the sum of a and b.
    • subtract(a, b): Returns the difference between a and b.
    • multiply(a, b): Returns the product of a and b.
    • divide(a, b): Returns the quotient of a divided by b.

  2. Create a script named test_math_operations.py that imports the math_operations module and tests all its functions.

Solution: math_operations.py

# math_operations.py

def add(a, b):
    return a + b

def subtract(a, b):
    return a - b

def multiply(a, b):
    return a * b

def divide(a, b):
    if b == 0:
        return "Cannot divide by zero"
    return a / b

Solution: test_math_operations.py

# test_math_operations.py

import math_operations

print(math_operations.add(10, 5))       # Output: 15
print(math_operations.subtract(10, 5))  # Output: 5
print(math_operations.multiply(10, 5))  # Output: 50
print(math_operations.divide(10, 5))    # Output: 2.0
print(math_operations.divide(10, 0))    # Output: Cannot divide by zero

Exercise 2: Creating and Using a Package

  1. Create a package named string_operations with the following structure: 
    string_operations/
    __init__.py
    uppercase.py
    lowercase.py
  2. In uppercase.py, create a function to_uppercase(s) that returns the uppercase version of the string s.
  3. In lowercase.py, create a function to_lowercase(s) that returns the lowercase version of the string s.
  4. In __init__.py, import the functions from uppercase.py and lowercase.py.
  5. Create a script named test_string_operations.py that imports the string_operations package and tests all its functions.

Solution: string_operations/uppercase.py

# uppercase.py

def to_uppercase(s):
    return s.upper()

Solution: string_operations/lowercase.py

# lowercase.py

def to_lowercase(s):
    return s.lower()

Solution: string_operations/__init__.py

# __init__.py

from .uppercase import to_uppercase
from .lowercase import to_lowercase

Solution: test_string_operations.py

# test_string_operations.py

import string_operations

print(string_operations.to_uppercase("hello"))  # Output: HELLO
print(string_operations.to_lowercase("WORLD"))  # Output: world

Conclusion

In this topic, you learned how to create and use modules and packages in Python. Modules and packages help you organize your code into manageable and reusable components.
