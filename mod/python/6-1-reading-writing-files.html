6-1: Reading and Writing Files

Introduction

File handling is an essential skill in Python programming. It allows you to read from and write to files, enabling data persistence and manipulation. This section will cover the basics of file operations, including opening, reading, writing, and closing files.

Key Concepts

  1. File Modes: Different modes for opening files.
  2. Reading Files: Techniques for reading file content.
  3. Writing Files: Methods for writing data to files.
  4. Closing Files: Importance of closing files.
  5. Context Managers: Using with statement for file operations.

File Modes

Python provides several modes for opening files:

  • 'r': Read (default mode)
  • 'w': Write (creates a new file or truncates an existing file)
  • 'a': Append (adds to the end of the file)
  • 'b': Binary mode (used with other modes, e.g., 'rb', 'wb')
  • '+': Read and write (used with other modes, e.g., 'r+', 'w+')
Mode Description
'r' Read mode. Opens the file for reading.
'w' Write mode. Opens the file for writing (truncates if exists).
'a' Append mode. Opens the file for appending.
'b' Binary mode. Used with other modes for binary files.
'+' Read and write mode. Used with other modes.

Reading Files

To read a file, you need to open it in read mode ('r'). You can use methods like read(), readline(), and readlines().

Example: Reading a File

# Open the file in read mode
file = open('example.txt', 'r')

# Read the entire content of the file
content = file.read()
print(content)

# Close the file
file.close()

Explanation

  • open('example.txt', 'r'): Opens the file in read mode.
  • file.read(): Reads the entire content of the file.
  • file.close(): Closes the file to free up resources.

Example: Reading Line by Line

# Open the file in read mode
file = open('example.txt', 'r')

# Read the file line by line
for line in file:
    print(line.strip())  # strip() removes the newline character

# Close the file
file.close()

Writing Files

To write to a file, you need to open it in write mode ('w') or append mode ('a').

Example: Writing to a File

# Open the file in write mode
file = open('example.txt', 'w')

# Write some content to the file
file.write('Hello, World!\n')
file.write('This is a new line.\n')

# Close the file
file.close()

Explanation

  • open('example.txt', 'w'): Opens the file in write mode.
  • file.write('...'): Writes the specified string to the file.
  • file.close(): Closes the file to ensure data is saved.

Using Context Managers

Using the with statement is a best practice for file operations as it ensures the file is properly closed after its suite finishes, even if an exception is raised.

Example: Using with Statement

# Using with statement to open and read a file
with open('example.txt', 'r') as file:
    content = file.read()
    print(content)

# Using with statement to open and write to a file
with open('example.txt', 'w') as file:
    file.write('Hello, World!\n')
    file.write('This is a new line.\n')

Practical Exercises

Exercise 1: Reading from a File

  1. Create a text file named sample.txt with the following content: 
    Line 1
Line 2
Line 3
  2. Write a Python script to read and print each line from the file.

Solution

with open('sample.txt', 'r') as file:
    for line in file:
        print(line.strip())

Exercise 2: Writing to a File

  1. Write a Python script to create a new file named output.txt and write the following lines to it: 
    First line of text.
Second line of text.

Solution

with open('output.txt', 'w') as file:
    file.write('First line of text.\n')
    file.write('Second line of text.\n')

Common Mistakes and Tips

  • Not closing files: Always close files to free up system resources. Use the with statement to handle this automatically.
  • Incorrect file paths: Ensure the file path is correct. Use absolute paths if necessary.
  • File modes: Use the correct mode for your operation to avoid data loss or errors.

Conclusion

In this section, you learned how to handle files in Python, including reading, writing, and using context managers. These skills are fundamental for data persistence and manipulation. In the next section, we will explore working with CSV files, which are commonly used for data storage and exchange.
