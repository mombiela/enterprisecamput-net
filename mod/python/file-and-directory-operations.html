Introduction

File and directory operations are essential skills for any programmer. In Python, the os and shutil modules provide a wide range of functions to interact with the file system. This section will cover the basics of file and directory operations, from reading and writing files to more advanced tasks like traversing directories and handling file permissions.

Working with Files

Opening and Closing Files

  • Opening a file: Use the open() function to open a file.
  • Closing a file: Always close the file using the close() method to free up system resources.
# Open a file for reading
file = open('example.txt', 'r')

# Perform file operations

# Close the file
file.close()

Reading Files

  • Read entire file: Use the read() method.
  • Read line by line: Use the readline() or readlines() methods.
# Read entire file
with open('example.txt', 'r') as file:
    content = file.read()
    print(content)

# Read file line by line
with open('example.txt', 'r') as file:
    for line in file:
        print(line.strip())

Writing to Files

  • Write text to a file: Use the write() method.
  • Append text to a file: Open the file in append mode ('a').
# Write to a file
with open('example.txt', 'w') as file:
    file.write('Hello, World!\n')

# Append to a file
with open('example.txt', 'a') as file:
    file.write('Appending a new line.\n')

Working with Directories

Creating and Removing Directories

  • Create a directory: Use os.mkdir() or os.makedirs() for nested directories.
  • Remove a directory: Use os.rmdir() or shutil.rmtree() for non-empty directories.
import os
import shutil

# Create a directory
os.mkdir('new_directory')

# Create nested directories
os.makedirs('parent_directory/child_directory')

# Remove an empty directory
os.rmdir('new_directory')

# Remove a non-empty directory
shutil.rmtree('parent_directory')

Listing Directory Contents

  • List files and directories: Use os.listdir() or os.scandir() for more detailed information.
# List files and directories
contents = os.listdir('.')
print(contents)

# List with detailed information
with os.scandir('.') as entries:
    for entry in entries:
        print(entry.name, entry.is_file(), entry.is_dir())

Advanced File and Directory Operations

Traversing Directories

  • Walk through directories: Use os.walk() to generate file names in a directory tree.
# Traverse directories
for dirpath, dirnames, filenames in os.walk('.'):
    print(f'Found directory: {dirpath}')
    for filename in filenames:
        print(f'\t{filename}')

File Permissions

  • Change file permissions: Use os.chmod() to change the mode of a file.
  • Check file permissions: Use os.access() to check the file's access permissions.
# Change file permissions
os.chmod('example.txt', 0o644)  # Read and write for owner, read for others

# Check file permissions
if os.access('example.txt', os.R_OK):
    print('File is readable')
else:
    print('File is not readable')

Summary

In this section, we covered the basics and some advanced topics related to file and directory operations in Python. You learned how to open, read, write, and close files, as well as how to create, remove, and list directories. We also touched on more advanced operations like traversing directories and handling file permissions. Mastering these skills will enable you to effectively manage files and directories in your Python applications.
