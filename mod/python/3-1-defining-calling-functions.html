Topic 3-1: Defining and Calling Functions

Introduction

Functions are a fundamental building block in Python programming. They allow you to encapsulate code into reusable blocks, making your code more organized and modular. In this section, we will cover how to define and call functions in Python.

Key Concepts

  1. Function Definition
  2. Function Calling
  3. Function Parameters
  4. Return Statement

1. Function Definition

A function in Python is defined using the def keyword, followed by the function name and parentheses (). The code block within every function starts with a colon : and is indented.

Syntax:

def function_name(parameters):
    """docstring"""
    statement(s)

Example:

def greet():
    """This function prints a greeting message."""
    print("Hello, welcome to the Python course!")

2. Function Calling

Once a function is defined, you can call it by using its name followed by parentheses ().

Example:

# Calling the greet function
greet()

Output:

Hello, welcome to the Python course!

3. Function Parameters

Functions can accept parameters (also known as arguments) to make them more flexible and reusable.

Example:

def greet(name):
    """This function prints a personalized greeting message."""
    print(f"Hello, {name}! Welcome to the Python course!")

# Calling the greet function with a parameter
greet("Alice")

Output:

Hello, Alice! Welcome to the Python course!

4. Return Statement

Functions can return a value using the return statement. This allows you to use the result of the function in other parts of your code.

Example:

def add(a, b):
    """This function returns the sum of two numbers."""
    return a + b

# Calling the add function and storing the result in a variable
result = add(3, 5)
print(result)

Output:

8

Practical Exercises

Exercise 1: Define and Call a Function

Define a function named square that takes a number as a parameter and returns its square. Then, call the function with the number 4 and print the result.

Solution:

def square(number):
    """This function returns the square of a number."""
    return number * number

# Calling the square function
result = square(4)
print(result)

Output:

16

Exercise 2: Function with Multiple Parameters

Define a function named multiply that takes two numbers as parameters and returns their product. Call the function with the numbers 3 and 7, and print the result.

Solution:

def multiply(a, b):
    """This function returns the product of two numbers."""
    return a * b

# Calling the multiply function
result = multiply(3, 7)
print(result)

Output:

21

Common Mistakes and Tips

  • Indentation Errors: Ensure that the code inside the function is properly indented.
  • Parameter Mismatch: When calling a function, make sure to pass the correct number of arguments.
  • Return Statement: If a function does not have a return statement, it will return None by default.

Conclusion

In this section, you learned how to define and call functions in Python. Functions help you organize your code into reusable blocks, making it more modular and easier to maintain. You also learned about function parameters and the return statement, which allow you to create more flexible and powerful functions.

Next, we will delve deeper into function arguments and return values, exploring different types of arguments and how to handle them in your functions.
