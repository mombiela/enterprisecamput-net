Topic 4-4: Sets

Introduction to Sets

Sets are an unordered collection of unique elements. They are useful when you need to store multiple items but want to ensure that there are no duplicates. Sets are mutable, meaning you can add or remove elements after the set has been created.

Key Concepts

  • Unordered: The elements in a set do not have a specific order.
  • Unique: Each element in a set must be unique.
  • Mutable: You can add or remove elements from a set.

Creating Sets

You can create a set by using curly braces {} or the set() function.

# Using curly braces
my_set = {1, 2, 3, 4, 5}
print(my_set)  # Output: {1, 2, 3, 4, 5}

# Using the set() function
another_set = set([1, 2, 3, 4, 5])
print(another_set)  # Output: {1, 2, 3, 4, 5}

# Creating an empty set
empty_set = set()
print(empty_set)  # Output: set()

Adding and Removing Elements

You can add elements to a set using the add() method and remove elements using the remove() or discard() methods.

# Adding elements
my_set.add(6)
print(my_set)  # Output: {1, 2, 3, 4, 5, 6}

# Removing elements
my_set.remove(3)
print(my_set)  # Output: {1, 2, 4, 5, 6}

# Removing elements using discard (no error if element not found)
my_set.discard(10)  # No error even though 10 is not in the set
print(my_set)  # Output: {1, 2, 4, 5, 6}

Set Operations

Sets support various operations like union, intersection, difference, and symmetric difference.

Operation Description Example
Union Combines elements from both sets set1.union(set2) or `set1
Intersection Elements common to both sets set1.intersection(set2) or set1 & set2
Difference Elements in the first set but not in the second set1.difference(set2) or set1 - set2
Symmetric Difference Elements in either set, but not in both set1.symmetric_difference(set2) or set1 ^ set2 
set1 = {1, 2, 3, 4}
set2 = {3, 4, 5, 6}

# Union
print(set1.union(set2))  # Output: {1, 2, 3, 4, 5, 6}
print(set1 | set2)       # Output: {1, 2, 3, 4, 5, 6}

# Intersection
print(set1.intersection(set2))  # Output: {3, 4}
print(set1 & set2)              # Output: {3, 4}

# Difference
print(set1.difference(set2))  # Output: {1, 2}
print(set1 - set2)            # Output: {1, 2}

# Symmetric Difference
print(set1.symmetric_difference(set2))  # Output: {1, 2, 5, 6}
print(set1 ^ set2)                      # Output: {1, 2, 5, 6}

Practical Examples

Example 1: Removing Duplicates from a List

You can use a set to remove duplicates from a list.

my_list = [1, 2, 2, 3, 4, 4, 5]
unique_elements = set(my_list)
print(unique_elements)  # Output: {1, 2, 3, 4, 5}

Example 2: Checking for Subsets and Supersets

You can check if a set is a subset or superset of another set using issubset() and issuperset() methods.

set1 = {1, 2, 3}
set2 = {1, 2, 3, 4, 5}

# Check if set1 is a subset of set2
print(set1.issubset(set2))  # Output: True

# Check if set2 is a superset of set1
print(set2.issuperset(set1))  # Output: True

Exercises

Exercise 1: Basic Set Operations

Create two sets, set_a and set_b, and perform the following operations:

  1. Union
  2. Intersection
  3. Difference
  4. Symmetric Difference
# Solution
set_a = {1, 2, 3}
set_b = {3, 4, 5}

# Union
print(set_a.union(set_b))  # Output: {1, 2, 3, 4, 5}

# Intersection
print(set_a.intersection(set_b))  # Output: {3}

# Difference
print(set_a.difference(set_b))  # Output: {1, 2}

# Symmetric Difference
print(set_a.symmetric_difference(set_b))  # Output: {1, 2, 4, 5}

Exercise 2: Removing Duplicates

Given a list of numbers, remove all duplicates using a set.

# Solution
numbers = [1, 2, 2, 3, 4, 4, 5, 5, 6]
unique_numbers = set(numbers)
print(unique_numbers)  # Output: {1, 2, 3, 4, 5, 6}

Common Mistakes and Tips

  • Mistake: Trying to access elements by index.
    • Tip: Remember that sets are unordered, so you cannot access elements by index.
  • Mistake: Assuming sets can contain mutable elements like lists.
    • Tip: Sets can only contain immutable elements like numbers, strings, and tuples.

Conclusion

In this section, you learned about sets in Python, including how to create them, add and remove elements, and perform various set operations. Sets are a powerful tool for managing collections of unique elements and can be particularly useful for tasks like removing duplicates from a list.
