Topic 9-3: Templates and Static Files

Introduction

In this section, we will explore how to use templates and static files in Flask to create dynamic web applications. Templates allow you to separate the HTML structure from your Python code, making your application more modular and easier to maintain. Static files, such as CSS, JavaScript, and images, are essential for enhancing the user experience.

Key Concepts

  1. Templates:

    • Templates are HTML files that can include placeholders for dynamic content.
    • Flask uses the Jinja2 templating engine to render templates.
    • Templates are stored in the templates directory.

  2. Static Files:

    • Static files include CSS, JavaScript, images, and other assets that do not change dynamically.
    • These files are stored in the static directory.

Setting Up Templates

Creating a Template

  1. Create a directory named templates in your Flask project.
  2. Inside the templates directory, create an HTML file, for example, index.html.
<!-- templates/index.html -->
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>{{ title }}</title>
    <link rel="stylesheet" href="{{ url_for('static', filename='styles.css') }}">
</head>
<body>
    <h1>{{ heading }}</h1>
    <p>{{ content }}</p>
</body>
</html>

Rendering a Template

  1. In your Flask application, import the render_template function.
  2. Use render_template to render the HTML file and pass dynamic data to it.
# app.py
from flask import Flask, render_template

app = Flask(__name__)

@app.route('/')
def home():
    return render_template('index.html', title='Home Page', heading='Welcome to Flask', content='This is a Flask web application.')

if __name__ == '__main__':
    app.run(debug=True)

Setting Up Static Files

Creating Static Files

  1. Create a directory named static in your Flask project.
  2. Inside the static directory, create a CSS file, for example, styles.css.
/* static/styles.css */
body {
    font-family: Arial, sans-serif;
    background-color: #f0f0f0;
    color: #333;
}

h1 {
    color: #007BFF;
}

Linking Static Files in Templates

  1. Use the url_for function to link static files in your templates.
<!-- templates/index.html -->
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>{{ title }}</title>
    <link rel="stylesheet" href="{{ url_for('static', filename='styles.css') }}">
</head>
<body>
    <h1>{{ heading }}</h1>
    <p>{{ content }}</p>
</body>
</html>

Practical Example

Let's create a simple Flask application that uses templates and static files.

Common Mistakes and Tips

  • Common Mistake: Forgetting to create the templates and static directories.

    • Tip: Always ensure that your directory structure is correctly set up before running your Flask application.

  • Common Mistake: Not using url_for to link static files.

    • Tip: Always use url_for('static', filename='your_file') to ensure correct linking of static files.

Conclusion

In this section, we learned how to use templates and static files in Flask to create dynamic and visually appealing web applications. We covered how to set up templates, render them with dynamic data, and link static files such as CSS. By mastering these concepts, you can build more complex and maintainable web applications with Flask.
