Introduction to JSON

  • JSON (JavaScript Object Notation) is a lightweight data interchange format.
  • It is easy for humans to read and write, and easy for machines to parse and generate.
  • JSON is often used to transmit data between a server and a web application.

JSON Basics

  • JSON data is written as key/value pairs.
  • Keys are strings, and values can be strings, numbers, objects, arrays, true, false, or null.

Example of JSON Data

{
  "name": "John",
  "age": 30,
  "isStudent": false,
  "courses": ["Math", "Science"],
  "address": {
    "street": "123 Main St",
    "city": "Anytown"
  }
}

Working with JSON in Python

  • Python provides a built-in module called json to handle JSON data.

Importing the JSON Module

import json

Reading JSON Data

  • JSON data can be read from a string or a file.

Reading JSON from a String

import json

json_string = '{"name": "John", "age": 30, "isStudent": false}'
data = json.loads(json_string)

print(data)
# Output: {'name': 'John', 'age': 30, 'isStudent': False}

Reading JSON from a File

import json

with open('data.json', 'r') as file:
    data = json.load(file)

print(data)
# Output: {'name': 'John', 'age': 30, 'isStudent': False}

Writing JSON Data

  • JSON data can be written to a string or a file.

Writing JSON to a String

import json

data = {
    "name": "John",
    "age": 30,
    "isStudent": False
}

json_string = json.dumps(data)
print(json_string)
# Output: {"name": "John", "age": 30, "isStudent": false}

Writing JSON to a File

import json

data = {
    "name": "John",
    "age": 30,
    "isStudent": False
}

with open('data.json', 'w') as file:
    json.dump(data, file)

Advanced JSON Handling

  • Customizing JSON encoding and decoding.
  • Handling complex data types.

Customizing JSON Encoding

  • Use the default parameter in json.dumps() to handle non-serializable objects.
import json
from datetime import datetime

def datetime_handler(x):
    if isinstance(x, datetime):
        return x.isoformat()
    raise TypeError("Unknown type")

data = {
    "name": "John",
    "joined": datetime(2021, 5, 17)
}

json_string = json.dumps(data, default=datetime_handler)
print(json_string)
# Output: {"name": "John", "joined": "2021-05-17T00:00:00"}

Customizing JSON Decoding

  • Use the object_hook parameter in json.loads() to handle custom decoding.
import json
from datetime import datetime

def datetime_parser(dct):
    for k, v in dct.items():
        if isinstance(v, str) and len(v) == 10:
            try:
                dct[k] = datetime.strptime(v, '%Y-%m-%d')
            except ValueError:
                pass
    return dct

json_string = '{"name": "John", "joined": "2021-05-17"}'
data = json.loads(json_string, object_hook=datetime_parser)
print(data)
# Output: {'name': 'John', 'joined': datetime.datetime(2021, 5, 17, 0, 0)}

Conclusion

  • JSON is a versatile format for data interchange.
  • Python's json module provides simple methods for reading and writing JSON data.
  • Advanced customization allows handling complex data types and custom encoding/decoding.
  • Mastering JSON handling in Python is essential for working with APIs and web services.
