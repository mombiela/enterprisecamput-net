Loops in Swift
Control Transfer Statements ►

Loops are fundamental constructs in programming that allow you to execute a block of code multiple times. Swift provides several types of loops to handle different scenarios. This section will guide you from the basics to advanced usage of loops in Swift.

Types of Loops

Swift supports three primary types of loops:

  • for-in loops
  • while loops
  • repeat-while loops

`for-in` Loops

The for-in loop is used to iterate over a sequence, such as items in an array, ranges of numbers, or characters in a string.

Syntax

for item in collection {
    // Code to execute
}

Example

let fruits = ["Apple", "Banana", "Cherry"]

for fruit in fruits {
    print(fruit)
}

In this example, the loop iterates over each element in the fruits array and prints it.

`while` Loops

The while loop continues to execute a block of code as long as a specified condition is true.

Syntax

while condition {
    // Code to execute
}

Example

var count = 5

while count > 0 {
    print("Count is \(count)")
    count -= 1
}

This loop will print the value of count and decrement it until count is no longer greater than 0.

`repeat-while` Loops

The repeat-while loop is similar to the while loop, but it guarantees that the code block will execute at least once before the condition is evaluated.

Syntax

repeat {
    // Code to execute
} while condition

Example

var count = 5

repeat {
    print("Count is \(count)")
    count -= 1
} while count > 0

This loop behaves similarly to the while loop example but ensures that the code block runs at least once.

Loop Control Statements

Swift provides control statements to alter the flow of loops:

  • break
  • continue

`break` Statement

The break statement terminates the loop immediately.

Example

for number in 1...5 {
    if number == 3 {
        break
    }
    print(number)
}

This loop will print 1 and 2, then terminate when number equals 3.

`continue` Statement

The continue statement skips the current iteration and proceeds to the next iteration of the loop.

Example

for number in 1...5 {
    if number == 3 {
        continue
    }
    print(number)
}

This loop will print 1, 2, 4, and 5, skipping the iteration where number equals 3.

Nested Loops

Loops can be nested inside other loops to handle more complex scenarios.

Example

for i in 1...3 {
    for j in 1...3 {
        print("i = \(i), j = \(j)")
    }
}

This example demonstrates a nested loop where the outer loop runs three times, and for each iteration of the outer loop, the inner loop also runs three times.

Looping Through Collections

Swift's for-in loop is particularly useful for iterating through collections like arrays, dictionaries, and sets.

Iterating Through an Array

let numbers = [1, 2, 3, 4, 5]

for number in numbers {
    print(number)
}

Iterating Through a Dictionary

let studentGrades = ["John": "A", "Jane": "B", "Jim": "C"]

for (student, grade) in studentGrades {
    print("\(student) received a grade of \(grade)")
}

Iterating Through a Set

let uniqueNumbers: Set = [1, 2, 3, 4, 5]

for number in uniqueNumbers {
    print(number)
}

Advanced Looping Techniques

Using `stride` for Custom Increments

The stride function allows you to create ranges with custom increments.

Example

for number in stride(from: 0, to: 10, by: 2) {
    print(number)
}

This loop will print 0, 2, 4, 6, and 8.

Looping with Enumerated Sequences

The enumerated() method returns a sequence of pairs (index, element), useful for when you need both the index and the value.

Example

let animals = ["Dog", "Cat", "Bird"]

for (index, animal) in animals.enumerated() {
    print("Animal \(index + 1): \(animal)")
}

This loop will print:

Animal 1: Dog
Animal 2: Cat
Animal 3: Bird

Conclusion

Loops are a powerful tool in Swift that allow you to perform repetitive tasks efficiently. Understanding the different types of loops and their control statements will enable you to write more effective and concise code. Practice using loops with various collections and conditions to become proficient in their use.
