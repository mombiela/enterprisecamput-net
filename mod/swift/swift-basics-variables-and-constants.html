Introduction

In Swift, variables and constants are fundamental building blocks for storing and manipulating data. Understanding how to declare and use them is essential for any Swift programmer. This section will cover the basics of variables and constants, including their declaration, usage, and differences.

Variables

Variables are used to store values that can change over time. In Swift, you declare a variable using the var keyword.

Declaring Variables

To declare a variable, you use the var keyword followed by the variable name and an optional type annotation.

var variableName: Type = initialValue

Example

var age: Int = 25
var name: String = "John"

Key Concepts

  • Type Inference: Swift can often infer the type of a variable from the initial value, so you can omit the type annotation.
  • Mutability: Variables can be changed after they are declared.
var age = 25 // Swift infers that age is of type Int
age = 26 // Changing the value of age

Constants

Constants are used to store values that do not change once they are set. In Swift, you declare a constant using the let keyword.

Declaring Constants

To declare a constant, you use the let keyword followed by the constant name and an optional type annotation.

let constantName: Type = initialValue

Example

let birthYear: Int = 1995
let pi: Double = 3.14159

Key Concepts

  • Immutability: Constants cannot be changed after they are declared.
  • Type Inference: Just like variables, Swift can infer the type of a constant from the initial value.
let birthYear = 1995 // Swift infers that birthYear is of type Int
// birthYear = 1996 // This will cause an error because birthYear is a constant

Comparing Variables and Constants

Feature Variables (var) Constants (let)
Mutability Can be changed Cannot be changed
Declaration var keyword let keyword
Use Case Values that change Values that remain constant

Code Examples

Using Variables

var greeting = "Hello, World!"
print(greeting) // Output: Hello, World!
greeting = "Hello, Swift!"
print(greeting) // Output: Hello, Swift!

Using Constants

let maximumNumberOfLoginAttempts = 3
var currentLoginAttempt = 0

// Uncommenting the next line will cause an error
// maximumNumberOfLoginAttempts = 4

Conclusion

Understanding the difference between variables and constants is crucial for writing effective Swift code. Variables (var) are used for values that can change, while constants (let) are used for values that remain the same. By using these appropriately, you can make your code more predictable and easier to understand. Remember to leverage Swift's type inference to write cleaner and more concise code.
