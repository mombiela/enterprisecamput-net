Iterating Over Collections
Introduction to Classes and Structures ►

In this section, we will explore how to iterate over collections in Swift. Collections in Swift include arrays, dictionaries, sets, and ranges. Understanding how to efficiently iterate over these collections is fundamental for any Swift programmer.

Key Concepts

  • Collections in Swift: Arrays, dictionaries, sets, and ranges.
  • For-in Loop: The primary loop used for iterating over collections.
  • Enumerated Method: Used to get both the index and the value.
  • Higher-Order Functions: forEach, map, filter, reduce.

Iterating Over Arrays

Arrays are ordered collections of elements. Let's start with the basics of iterating over arrays.

Using For-in Loop

The for-in loop is the most common way to iterate over an array.

let fruits = ["Apple", "Banana", "Cherry"]

for fruit in fruits {
    print(fruit)
}

Explanation:

  • fruits is an array of strings.
  • The for-in loop iterates over each element in the array and prints it.

Using Enumerated Method

The enumerated method provides both the index and the value of each element in the array.

let fruits = ["Apple", "Banana", "Cherry"]

for (index, fruit) in fruits.enumerated() {
    print("Fruit \(index + 1): \(fruit)")
}

Explanation:

  • enumerated returns a sequence of pairs (index, element).
  • The loop prints both the index and the value.

Iterating Over Dictionaries

Dictionaries are collections of key-value pairs. Iterating over dictionaries is slightly different from arrays.

Using For-in Loop

let capitals = ["France": "Paris", "Japan": "Tokyo", "USA": "Washington D.C."]

for (country, capital) in capitals {
    print("The capital of \(country) is \(capital).")
}

Explanation:

  • capitals is a dictionary with country names as keys and capital cities as values.
  • The for-in loop iterates over each key-value pair.

Iterating Over Sets

Sets are unordered collections of unique elements. Iterating over sets is similar to arrays but without guaranteed order.

Using For-in Loop

let uniqueNumbers: Set = [1, 2, 3, 4, 5]

for number in uniqueNumbers {
    print(number)
}

Explanation:

  • uniqueNumbers is a set of integers.
  • The for-in loop iterates over each element in the set.

Iterating Over Ranges

Ranges represent a sequence of values. They are often used in loops.

Using For-in Loop

for number in 1...5 {
    print(number)
}

Explanation:

  • 1...5 is a closed range that includes numbers from 1 to 5.
  • The for-in loop iterates over each number in the range.

Higher-Order Functions

Swift provides several higher-order functions that can be used to iterate over collections in a more functional programming style.

forEach

The forEach method is an alternative to the for-in loop.

let fruits = ["Apple", "Banana", "Cherry"]

fruits.forEach { fruit in
    print(fruit)
}

Explanation:

  • forEach takes a closure that is executed for each element in the array.

map

The map method transforms each element in the collection.

let numbers = [1, 2, 3, 4, 5]
let squaredNumbers = numbers.map { $0 * $0 }

print(squaredNumbers)

Explanation:

  • map applies the closure to each element and returns a new array with the transformed elements.

filter

The filter method returns a new collection containing elements that satisfy a given condition.

let numbers = [1, 2, 3, 4, 5]
let evenNumbers = numbers.filter { $0 % 2 == 0 }

print(evenNumbers)

Explanation:

  • filter applies the closure to each element and returns a new array with elements that meet the condition.

reduce

The reduce method combines all elements of the collection into a single value.

let numbers = [1, 2, 3, 4, 5]
let sum = numbers.reduce(0, +)

print(sum)

Explanation:

  • reduce takes an initial value and a closure that combines the initial value with each element of the array.

Conclusion

Iterating over collections is a fundamental skill in Swift programming. Whether you are working with arrays, dictionaries, sets, or ranges, understanding the various methods and techniques for iteration will help you write more efficient and readable code. Additionally, leveraging higher-order functions like forEach, map, filter, and reduce can lead to more expressive and concise code. Practice these concepts to become proficient in handling collections in Swift.
