Conditional Statements in Swift
Loops ►

Conditional statements are fundamental in programming as they allow you to execute different code based on certain conditions. In Swift, the primary conditional statements are if, else if, else, and switch.

Key Concepts

  • If Statement: Executes a block of code if a specified condition is true.
  • Else If Statement: Provides an additional condition to check if the initial if condition is false.
  • Else Statement: Executes a block of code if none of the preceding conditions are true.
  • Switch Statement: Evaluates a value and executes corresponding code blocks based on matching cases.

If Statement

The if statement is used to execute a block of code if a condition is true.

let temperature = 30

if temperature > 25 {
    print("It's a hot day!")
}

In this example, the message "It's a hot day!" will be printed because the condition temperature > 25 is true.

Else If Statement

The else if statement allows you to check multiple conditions.

let temperature = 20

if temperature > 25 {
    print("It's a hot day!")
} else if temperature > 15 {
    print("It's a warm day!")
}

Here, the message "It's a warm day!" will be printed because the first condition is false, but the second condition temperature > 15 is true.

Else Statement

The else statement is used to execute a block of code if none of the preceding conditions are true.

let temperature = 10

if temperature > 25 {
    print("It's a hot day!")
} else if temperature > 15 {
    print("It's a warm day!")
} else {
    print("It's a cold day!")
}

In this case, the message "It's a cold day!" will be printed because both previous conditions are false.

Switch Statement

The switch statement evaluates a value and executes the corresponding case block.

let dayOfWeek = 3

switch dayOfWeek {
case 1:
    print("Monday")
case 2:
    print("Tuesday")
case 3:
    print("Wednesday")
case 4:
    print("Thursday")
case 5:
    print("Friday")
case 6:
    print("Saturday")
case 7:
    print("Sunday")
default:
    print("Invalid day")
}

In this example, the message "Wednesday" will be printed because dayOfWeek is 3.

Switch Statement with Ranges and Tuples

Swift's switch statement is powerful and supports ranges and tuples.

let score = 85

switch score {
case 0..<50:
    print("Fail")
case 50..<75:
    print("Pass")
case 75...100:
    print("Excellent")
default:
    print("Invalid score")
}

Here, the message "Excellent" will be printed because score falls within the range 75 to 100.

Summary

  • If Statement: Executes code if a condition is true.
  • Else If Statement: Checks additional conditions if the previous ones are false.
  • Else Statement: Executes code if none of the conditions are true.
  • Switch Statement: Evaluates a value and matches it against multiple cases.

Understanding and using conditional statements effectively allows you to control the flow of your Swift programs, making them more dynamic and responsive to different inputs and conditions.
