File manipulation is an essential skill for any programmer. In this section, we will explore how to handle files using REXX, from basic operations to more advanced techniques.

Introduction to File Manipulation

File manipulation involves various operations such as reading from files, writing to files, and managing file attributes. In REXX, these operations are performed using built-in functions and commands.

Key Concepts

  • File Handling: Opening, reading, writing, and closing files.
  • File I/O Functions: Functions provided by REXX for file input and output.
  • Error Handling: Managing errors that occur during file operations.

Basic File Operations

Opening and Closing Files

To perform any operation on a file, you first need to open it. In REXX, files are typically opened using the OPEN function and closed using the CLOSE function.

Example: Opening and Closing a File

/* Open a file for reading */
file = 'example.txt'
fileHandle = open(file, 'r')

/* Perform operations on the file */

/* Close the file */
call close(fileHandle)

Reading from Files

Reading from a file can be done line by line or in chunks. The LINEIN function is commonly used to read lines from a file.

Example: Reading a File Line by Line

file = 'example.txt'
fileHandle = open(file, 'r')

do while lines(fileHandle) > 0
    line = linein(fileHandle)
    say line
end

call close(fileHandle)

Writing to Files

Writing to a file is accomplished using the LINEOUT function. You can write lines to a file one at a time.

Example: Writing to a File

file = 'output.txt'
fileHandle = open(file, 'w')

call lineout fileHandle, 'This is the first line.'
call lineout fileHandle, 'This is the second line.'

call close(fileHandle)

Advanced File Operations

Appending to Files

Appending data to an existing file can be done using the OPEN function with the append mode (a).

Example: Appending to a File

file = 'output.txt'
fileHandle = open(file, 'a')

call lineout fileHandle, 'This line is appended.'

call close(fileHandle)

Error Handling in File Operations

Proper error handling ensures that your program can gracefully handle issues like missing files or permission errors.

Example: Error Handling

file = 'nonexistent.txt'
fileHandle = open(file, 'r')

if fileHandle = '' then do
    say 'Error: Unable to open file.'
    exit
end

/* Perform operations on the file */

call close(fileHandle)

File Management Functions

REXX provides several functions for managing files, such as DELETE to remove a file and EXISTS to check if a file exists.

Example: Deleting a File

file = 'output.txt'

if exists(file) then do
    call delete file
    say 'File deleted successfully.'
end
else do
    say 'File does not exist.'
end

Summary

In this section, we covered the basics and some advanced techniques for file manipulation in REXX. You learned how to:

  • Open and close files.
  • Read from and write to files.
  • Append data to files.
  • Handle errors during file operations.
  • Manage files using built-in functions.

Understanding these concepts will enable you to effectively manage files in your REXX programs, making your scripts more robust and versatile.
