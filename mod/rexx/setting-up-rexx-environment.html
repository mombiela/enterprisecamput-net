Setting Up the REXX Environment
Hello World in REXX ►

Setting up the REXX environment is the first step towards mastering this powerful scripting language. This section will guide you through the process of installing and configuring REXX on your system, ensuring you have everything you need to start writing and executing REXX scripts.

Installing REXX

Before you can start writing REXX scripts, you need to install a REXX interpreter. There are several options available, depending on your operating system.

Windows

For Windows users, Regina REXX is a popular choice. Follow these steps to install it:

  1. Download Regina REXX:

    • Visit the Regina REXX website.
    • Download the appropriate installer for your version of Windows.

  2. Install Regina REXX:

    • Run the downloaded installer.
    • Follow the on-screen instructions to complete the installation.

  3. Verify Installation:

    • Open Command Prompt.
    • Type rexx and press Enter. If installed correctly, you should see the REXX prompt.

macOS

For macOS users, you can use the Homebrew package manager to install Regina REXX:

  1. Install Homebrew (if not already installed):

    • Open Terminal.
    • Run the command: 
      /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

  2. Install Regina REXX:

    • Run the command: 
      brew install regina-rexx

  3. Verify Installation:

    • Type rexx in Terminal and press Enter. You should see the REXX prompt.

Linux

For Linux users, Regina REXX can be installed using the package manager of your distribution:

  1. Debian/Ubuntu:

    • Open Terminal.
    • Run the command: 
      sudo apt-get install regina-rexx

  2. Fedora:

    • Open Terminal.
    • Run the command: 
      sudo dnf install regina-rexx

  3. Verify Installation:

    • Type rexx in Terminal and press Enter. You should see the REXX prompt.

Configuring the Environment

After installing REXX, you may need to configure your environment to make it easier to run REXX scripts.

Setting Environment Variables

Setting environment variables can help you run REXX scripts from any directory without specifying the full path to the interpreter.

Windows

  1. Open Environment Variables:

    • Right-click on 'This PC' or 'Computer' on the desktop or in File Explorer.
    • Select 'Properties'.
    • Click on 'Advanced system settings'.
    • Click on 'Environment Variables'.

  2. Add REXX to PATH:

    • In the 'System variables' section, find and select the 'Path' variable, then click 'Edit'.
    • Click 'New' and add the path to the directory where Regina REXX is installed (e.g., C:\Program Files\Regina).

  3. Apply Changes:

    • Click 'OK' to close all dialog boxes.

macOS and Linux

  1. Open Terminal.

  2. Edit Shell Configuration File:

    • For Bash, open .bashrc or .bash_profile: 
      nano ~/.bashrc
    • For Zsh, open .zshrc: 
      nano ~/.zshrc

  3. Add REXX to PATH:

    • Add the following line to the file: 
      export PATH=$PATH:/usr/local/bin

  4. Apply Changes:

    • Save the file and run: 
      source ~/.bashrc
    • Or for Zsh: 
      source ~/.zshrc

Writing Your First REXX Script

Now that your environment is set up, let's write a simple REXX script to ensure everything is working correctly.

  1. Create a New File:

    • Open your preferred text editor.
    • Create a new file named hello.rexx.

  2. Write the Script:

    • Add the following code to the file: 
      /* Hello World in REXX */
say "Hello, World!"

  3. Save the File.

  4. Run the Script:

    • Open Command Prompt (Windows) or Terminal (macOS/Linux).
    • Navigate to the directory where you saved hello.rexx.
    • Run the script by typing: 
      rexx hello.rexx
    • You should see the output: 
      Hello, World!

Conclusion

Setting up the REXX environment involves installing the interpreter and configuring your system to easily run REXX scripts. By following the steps outlined in this section, you should now have a fully functional REXX environment. You also wrote and executed your first REXX script, confirming that everything is set up correctly. In the next sections, we will dive deeper into the syntax and features of the REXX language.
