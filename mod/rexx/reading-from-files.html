Introduction

Reading from files is a fundamental skill in programming, allowing you to process and analyze data stored in external files. In this section, we will cover how to read from files using REXX, starting from the basics and moving towards more advanced techniques.

Basic File Reading

Opening a File

To read from a file in REXX, you first need to open it. This is done using the OPEN function.

/* Open a file for reading */
file = 'example.txt'
file_handle = OPEN(file, 'READ')
IF file_handle = 0 THEN
  SAY 'Error opening file'
ELSE
  SAY 'File opened successfully'
  • file: The name of the file you want to open.
  • file_handle: A handle that represents the opened file. If the file cannot be opened, file_handle will be 0.

Reading Lines from a File

Once the file is open, you can read its contents line by line using the LINEIN function.

/* Read lines from the file */
DO WHILE LINES(file_handle) > 0
  line = LINEIN(file_handle)
  SAY line
END
  • LINES(file_handle): Returns the number of lines remaining in the file.
  • LINEIN(file_handle): Reads the next line from the file.

Closing a File

After you are done reading from the file, it is important to close it using the CLOSE function.

/* Close the file */
CALL CLOSE file_handle

Advanced File Reading

Reading Specific Lines

You can read specific lines from a file by specifying the line number in the LINEIN function.

/* Read the 5th line from the file */
line = LINEIN(file_handle, 5)
SAY line

Reading with Error Handling

It's good practice to include error handling when working with files to manage unexpected issues.

/* Open a file with error handling */
file = 'example.txt'
file_handle = OPEN(file, 'READ')
IF file_handle = 0 THEN
  SAY 'Error opening file'
ELSE DO
  /* Read lines with error handling */
  DO WHILE LINES(file_handle) > 0
    line = LINEIN(file_handle)
    IF RC \= 0 THEN
      SAY 'Error reading line'
    ELSE
      SAY line
  END
  /* Close the file */
  CALL CLOSE file_handle
END

Reading Large Files

For large files, you might want to read chunks of data instead of line by line to improve performance.

/* Read chunks of data from the file */
DO WHILE LINES(file_handle) > 0
  chunk = CHARIN(file_handle, 1, 1024) /* Read 1024 characters */
  SAY chunk
END
  • CHARIN(file_handle, start, length): Reads a specified number of characters starting from a given position.

Summary

In this section, we covered the basics of reading from files in REXX, including opening, reading, and closing files. We also explored advanced techniques such as reading specific lines, error handling, and reading large files in chunks. Mastering these skills will enable you to efficiently process and analyze data stored in external files using REXX.
