In this section, we will explore the various looping constructs available in REXX. Looping constructs are essential for performing repetitive tasks efficiently. We will start with basic loops and progress to more advanced looping techniques.

1. Introduction to Loops

Loops allow you to execute a block of code multiple times. REXX provides several types of loops to handle different scenarios.

2. The `DO` Loop

The DO loop is the most commonly used loop in REXX. It can be used in several forms:

2.1 Simple `DO` Loop

A simple DO loop repeats a block of code a specified number of times.

/* Simple DO loop example */
DO i = 1 TO 5
    SAY 'Iteration:' i
END
  • Explanation:
    • DO i = 1 TO 5: Initializes i to 1 and increments it until it reaches 5.
    • SAY 'Iteration:' i: Prints the current value of i.
    • END: Marks the end of the loop.

2.2 `DO WHILE` Loop

The DO WHILE loop continues to execute as long as a specified condition is true.

/* DO WHILE loop example */
i = 1
DO WHILE i <= 5
    SAY 'Iteration:' i
    i = i + 1
END
  • Explanation:
    • i = 1: Initializes i to 1.
    • DO WHILE i <= 5: Continues the loop while i is less than or equal to 5.
    • SAY 'Iteration:' i: Prints the current value of i.
    • i = i + 1: Increments i by 1.
    • END: Marks the end of the loop.

2.3 `DO UNTIL` Loop

The DO UNTIL loop continues to execute until a specified condition becomes true.

/* DO UNTIL loop example */
i = 1
DO UNTIL i > 5
    SAY 'Iteration:' i
    i = i + 1
END
  • Explanation:
    • i = 1: Initializes i to 1.
    • DO UNTIL i > 5: Continues the loop until i is greater than 5.
    • SAY 'Iteration:' i: Prints the current value of i.
    • i = i + 1: Increments i by 1.
    • END: Marks the end of the loop.

2.4 `DO FOREVER` Loop

The DO FOREVER loop continues indefinitely until explicitly terminated with a LEAVE statement.

/* DO FOREVER loop example */
i = 1
DO FOREVER
    SAY 'Iteration:' i
    i = i + 1
    IF i > 5 THEN LEAVE
END
  • Explanation:
    • i = 1: Initializes i to 1.
    • DO FOREVER: Starts an infinite loop.
    • SAY 'Iteration:' i: Prints the current value of i.
    • i = i + 1: Increments i by 1.
    • IF i > 5 THEN LEAVE: Exits the loop if i is greater than 5.
    • END: Marks the end of the loop.

3. Nested Loops

You can nest loops within each other to handle more complex scenarios.

/* Nested loops example */
DO i = 1 TO 3
    SAY 'Outer loop iteration:' i
    DO j = 1 TO 2
        SAY '  Inner loop iteration:' j
    END
END
  • Explanation:
    • The outer loop runs from 1 to 3.
    • The inner loop runs from 1 to 2 for each iteration of the outer loop.
    • The SAY statements print the current iteration of both loops.

4. Loop Control Statements

4.1 `LEAVE`

The LEAVE statement exits the current loop immediately.

/* LEAVE statement example */
DO i = 1 TO 5
    IF i = 3 THEN LEAVE
    SAY 'Iteration:' i
END
  • Explanation:
    • The loop exits when i equals 3, skipping the remaining iterations.

4.2 `ITERATE`

The ITERATE statement skips the rest of the current iteration and proceeds to the next iteration.

/* ITERATE statement example */
DO i = 1 TO 5
    IF i = 3 THEN ITERATE
    SAY 'Iteration:' i
END
  • Explanation:
    • The loop skips the SAY statement when i equals 3 and continues with the next iteration.

Conclusion

In this section, we covered the various looping constructs available in REXX, including simple DO loops, DO WHILE, DO UNTIL, and DO FOREVER loops. We also explored nested loops and loop control statements like LEAVE and ITERATE. Understanding these constructs will help you write more efficient and readable REXX programs. Practice these examples and try creating your own loops to solidify your understanding.
