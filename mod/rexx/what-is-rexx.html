Introduction to REXX

REXX (Restructured Extended Executor) is a high-level programming language developed by IBM. It is designed for ease of learning and reading, making it an excellent choice for beginners and experienced programmers alike. REXX is widely used for scripting, data processing, and automating tasks on IBM mainframes, as well as on other platforms.

Key Features of REXX

  • Ease of Use: REXX syntax is straightforward and easy to read, which reduces the learning curve for new programmers.
  • Portability: REXX is available on multiple platforms, including IBM mainframes, Windows, Unix, and Linux.
  • Interoperability: REXX can interact with other programming languages and systems, making it versatile for various applications.
  • Built-in Functions: REXX includes a rich set of built-in functions for string manipulation, arithmetic operations, and more.
  • Extensibility: Users can extend REXX by writing their own functions and procedures.

Basic Syntax of REXX

Let's look at a simple REXX program to understand its basic syntax:

/* This is a comment in REXX */
say "Hello, World!"
  • Comments: Comments in REXX are enclosed within /* and */.
  • Output: The say instruction is used to print text to the screen.

Variables and Data Types

In REXX, variables are dynamically typed, meaning you don't need to declare their type explicitly. Here's an example:

name = "John Doe"
age = 30
say "Name:" name
say "Age:" age
  • String Variables: Variables like name can hold string values.
  • Numeric Variables: Variables like age can hold numeric values.

Control Structures

REXX supports standard control structures such as loops and conditional statements. Here are some examples:

Conditional Statements

age = 20
if age >= 18 then
    say "You are an adult."
else
    say "You are a minor."

Loops

do i = 1 to 5
    say "Iteration:" i
end

Built-in Functions

REXX provides a variety of built-in functions to perform common tasks. Here are a few examples:

  • String Manipulation: length, substr, pos
  • Arithmetic Operations: abs, max, min
  • Date and Time: date, time

Example of using built-in functions:

text = "Hello, World!"
say "Length of text:" length(text)
say "Substring from position 8:" substr(text, 8)

Conclusion

REXX is a powerful yet easy-to-learn programming language that is ideal for scripting and automating tasks. Its simplicity, portability, and rich set of built-in functions make it a versatile tool for programmers of all levels. In the upcoming sections of this course, we will delve deeper into REXX's advanced features and explore practical applications to enhance your programming skills.
