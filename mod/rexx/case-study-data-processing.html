Introduction

In this case study, we will explore how to process data using REXX, a powerful scripting language. We will cover the following key concepts:

  • Reading data from files
  • Processing and manipulating data
  • Writing data to files
  • Error handling

By the end of this case study, you will have a solid understanding of how to handle data processing tasks in REXX.

Reading Data from Files

Reading data from files is a fundamental task in data processing. In REXX, this can be done using the LINEIN function.

Example

/* Read data from a file */
filename = 'data.txt'
file = open(filename, 'r')
if file = '' then
  say 'Error: Unable to open file'
else
  do while lines(file) > 0
    line = linein(file)
    say 'Read line:' line
  end
call close file
  • open(filename, 'r'): Opens the file in read mode.
  • lines(file): Returns the number of lines left to read in the file.
  • linein(file): Reads a line from the file.
  • call close file: Closes the file.

Processing and Manipulating Data

Once data is read from a file, it often needs to be processed or manipulated. This can include tasks such as filtering, transforming, or aggregating data.

Example

/* Process data: Convert to uppercase and filter lines containing 'ERROR' */
filename = 'data.txt'
file = open(filename, 'r')
if file = '' then
  say 'Error: Unable to open file'
else
  do while lines(file) > 0
    line = linein(file)
    line = translate(line) /* Convert to uppercase */
    if pos('ERROR', line) > 0 then
      say 'Filtered line:' line
  end
call close file
  • translate(line): Converts the line to uppercase.
  • pos('ERROR', line): Checks if the string 'ERROR' is present in the line.

Writing Data to Files

After processing data, the results often need to be written to a file. This can be done using the LINEOUT function.

Example

/* Write processed data to a new file */
inputFile = 'data.txt'
outputFile = 'processed_data.txt'
inFile = open(inputFile, 'r')
outFile = open(outputFile, 'w')
if inFile = '' | outFile = '' then
  say 'Error: Unable to open file(s)'
else
  do while lines(inFile) > 0
    line = linein(inFile)
    line = translate(line) /* Convert to uppercase */
    if pos('ERROR', line) > 0 then
      call lineout outFile, line
  end
call close inFile
call close outFile
  • open(outputFile, 'w'): Opens the file in write mode.
  • call lineout outFile, line: Writes a line to the output file.

Error Handling

Error handling is crucial in data processing to ensure the robustness of the script. REXX provides mechanisms to handle errors gracefully.

Example

/* Error handling in data processing */
filename = 'data.txt'
file = open(filename, 'r')
if file = '' then
  say 'Error: Unable to open file'
else
  do while lines(file) > 0
    line = linein(file)
    if rc \= 0 then
      do
        say 'Error reading line'
        leave
      end
    line = translate(line) /* Convert to uppercase */
    if pos('ERROR', line) > 0 then
      say 'Filtered line:' line
  end
call close file
  • if rc \= 0: Checks if the return code (rc) indicates an error.

Conclusion

In this case study, we have covered the essential aspects of data processing in REXX:

  • Reading data from files
  • Processing and manipulating data
  • Writing data to files
  • Implementing error handling

By mastering these concepts, you will be well-equipped to handle various data processing tasks using REXX. Practice these examples and try to apply them to your own data processing needs to reinforce your understanding.
