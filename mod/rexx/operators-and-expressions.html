Introduction

In this section, we will explore the various operators and expressions in REXX. Operators are special symbols that perform operations on variables and values. Expressions are combinations of variables, values, and operators that REXX evaluates to produce another value. Understanding operators and expressions is fundamental to writing effective REXX programs.

Arithmetic Operators

Arithmetic operators are used to perform mathematical calculations.

  • Addition (+): Adds two numbers.
  • Subtraction (-): Subtracts the second number from the first.
  • Multiplication (*): Multiplies two numbers.
  • Division (/): Divides the first number by the second.
  • Integer Division (%): Divides the first number by the second and returns the integer part of the quotient.
  • Modulus (//): Divides the first number by the second and returns the remainder.

Examples

/* Addition */
result = 5 + 3
say result  /* Output: 8 */

/* Subtraction */
result = 10 - 4
say result  /* Output: 6 */

/* Multiplication */
result = 7 * 2
say result  /* Output: 14 */

/* Division */
result = 20 / 4
say result  /* Output: 5 */

/* Integer Division */
result = 20 % 3
say result  /* Output: 6 */

/* Modulus */
result = 20 // 3
say result  /* Output: 2 */

Comparison Operators

Comparison operators are used to compare two values. They return a boolean value (1 for true, 0 for false).

  • Equal (= or ==): Checks if two values are equal.
  • Not Equal (\= or <>): Checks if two values are not equal.
  • Less Than (<): Checks if the first value is less than the second.
  • Greater Than (>): Checks if the first value is greater than the second.
  • Less Than or Equal (<=): Checks if the first value is less than or equal to the second.
  • Greater Than or Equal (>=): Checks if the first value is greater than or equal to the second.

Examples

/* Equal */
result = 5 = 5
say result  /* Output: 1 */

/* Not Equal */
result = 5 \= 3
say result  /* Output: 1 */

/* Less Than */
result = 3 < 5
say result  /* Output: 1 */

/* Greater Than */
result = 7 > 4
say result  /* Output: 1 */

/* Less Than or Equal */
result = 5 <= 5
say result  /* Output: 1 */

/* Greater Than or Equal */
result = 6 >= 3
say result  /* Output: 1 */

Logical Operators

Logical operators are used to combine multiple conditions.

  • AND (&): Returns true if both conditions are true.
  • OR (|): Returns true if at least one condition is true.
  • NOT (\): Returns the opposite of the condition.

Examples

/* AND */
result = (5 = 5) & (3 < 4)
say result  /* Output: 1 */

/* OR */
result = (5 = 5) | (3 > 4)
say result  /* Output: 1 */

/* NOT */
result = \ (5 = 3)
say result  /* Output: 1 */

String Operators

String operators are used to manipulate strings.

  • Concatenation (||): Joins two strings together.
  • Substring (substr): Extracts a substring from a string.

Examples

/* Concatenation */
result = "Hello" || " " || "World"
say result  /* Output: Hello World */

/* Substring */
result = substr("Hello World", 1, 5)
say result  /* Output: Hello */

Assignment Operators

Assignment operators are used to assign values to variables.

  • Simple Assignment (=): Assigns the value on the right to the variable on the left.
  • Compound Assignment (+=, -=, *=, /=, %=, //=): Combines an arithmetic operation with assignment.

Examples

/* Simple Assignment */
a = 10
say a  /* Output: 10 */

/* Compound Assignment */
a += 5
say a  /* Output: 15 */

a -= 3
say a  /* Output: 12 */

a *= 2
say a  /* Output: 24 */

a /= 4
say a  /* Output: 6 */

a %= 2
say a  /* Output: 3 */

a //= 2
say a  /* Output: 1 */

Summary

In this section, we covered the various operators and expressions in REXX, including arithmetic, comparison, logical, string, and assignment operators. Understanding these operators and how to use them in expressions is crucial for writing effective REXX programs. Practice using these operators in different scenarios to become more comfortable with their usage.
