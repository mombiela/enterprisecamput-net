Introduction to REXX

REXX (Restructured Extended Executor) is a programming language that is easy to learn and use. It is designed for both beginners and experienced programmers. This section will cover the basic syntax and structure of REXX programs, providing a foundation for more advanced topics.

Key Concepts

  • Case Insensitivity: REXX is not case-sensitive, meaning say, SAY, and Say are treated the same.
  • Whitespace: Whitespace is generally ignored in REXX, making the code more readable.
  • Comments: Comments are used to explain code and are ignored during execution.

Basic Program Structure

A REXX program typically consists of a sequence of instructions that are executed in order. Here is a simple example:

/* This is a comment */
say "Hello, World!"

Comments

  • Single-line comments start with /* and end with */.
  • Comments can span multiple lines.
/* This is a 
   multi-line comment */

Variables

  • Variables in REXX are dynamically typed and do not need to be declared before use.
  • Variable names can contain letters, numbers, and special characters like . and _.
name = "John Doe"
age = 30
say "Name:" name
say "Age:" age

Operators

  • Arithmetic Operators: +, -, *, /, // (integer division), ** (exponentiation)
  • Comparison Operators: =, <>, >, <, >=, <=
  • Logical Operators: & (AND), | (OR), \ (NOT)
a = 10
b = 20
c = a + b
say "Sum:" c

Control Structures

If-Else

The if-else statement is used for conditional execution.

age = 18
if age >= 18 then
    say "You are an adult."
else
    say "You are a minor."

Select-When

The select-when structure is used for multiple conditions.

grade = "B"
select
    when grade = "A" then say "Excellent!"
    when grade = "B" then say "Good job!"
    when grade = "C" then say "You passed."
    otherwise say "Better luck next time."
end

Loops

  • Do-Loop: Used for iteration.
do i = 1 to 5
    say "Iteration:" i
end
  • Do While/Until: Used for conditional loops.
i = 1
do while i <= 5
    say "While Loop Iteration:" i
    i = i + 1
end

i = 1
do until i > 5
    say "Until Loop Iteration:" i
    i = i + 1
end

Built-in Functions

REXX provides several built-in functions for common tasks.

String Functions

  • Length: Returns the length of a string.
str = "Hello"
say length(str)  /* Output: 5 */
  • Substr: Extracts a substring.
str = "Hello, World!"
say substr(str, 1, 5)  /* Output: Hello */

Numeric Functions

  • Abs: Returns the absolute value.
num = -10
say abs(num)  /* Output: 10 */
  • Max/Min: Returns the maximum or minimum value.
say max(10, 20, 30)  /* Output: 30 */
say min(10, 20, 30)  /* Output: 10 */

Conclusion

Understanding the basic syntax and structure of REXX is crucial for writing effective programs. This section covered the essentials, including comments, variables, operators, control structures, and built-in functions. With this foundation, you are now prepared to delve into more advanced topics and features of the REXX programming language.
