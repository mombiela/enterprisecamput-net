Introduction

String manipulation is a fundamental skill in REXX programming. This topic will cover various techniques and functions for handling strings, from basic operations to advanced manipulations.

Basic String Operations

Concatenation

  • Concept: Combining two or more strings into one.
  • Example: 
    string1 = "Hello"
string2 = "World"
combinedString = string1 || " " || string2
say combinedString  /* Output: Hello World */

Substring Extraction

  • Concept: Extracting a part of a string.
  • Example: 
    fullString = "Hello World"
subString = substr(fullString, 1, 5)
say subString  /* Output: Hello */

String Length

  • Concept: Finding the length of a string.
  • Example: 
    myString = "Hello"
length = length(myString)
say length  /* Output: 5 */

Intermediate String Operations

Trimming Strings

  • Concept: Removing leading and/or trailing spaces.
  • Example: 
    myString = "  Hello World  "
trimmedString = strip(myString)
say trimmedString  /* Output: Hello World */

Changing Case

  • Concept: Converting strings to upper or lower case.
  • Example: 
    myString = "Hello World"
upperCaseString = translate(myString)
lowerCaseString = translate(myString, "abcdefghijklmnopqrstuvwxyz", "ABCDEFGHIJKLMNOPQRSTUVWXYZ")
say upperCaseString  /* Output: HELLO WORLD */
say lowerCaseString  /* Output: hello world */

Replacing Substrings

  • Concept: Replacing parts of a string with another string.
  • Example: 
    myString = "Hello World"
newString = overlay("REXX", myString, 7)
say newString  /* Output: Hello REXX */

Advanced String Operations

Pattern Matching and Substitution

  • Concept: Finding and replacing patterns within strings.
  • Example: 
    myString = "The quick brown fox jumps over the lazy dog"
newString = changestr("fox", "cat", myString)
say newString  /* Output: The quick brown cat jumps over the lazy dog */

Splitting Strings

  • Concept: Dividing a string into multiple parts based on a delimiter.
  • Example: 
    myString = "apple,banana,cherry"
parse var myString fruit1 "," fruit2 "," fruit3
say fruit1  /* Output: apple */
say fruit2  /* Output: banana */
say fruit3  /* Output: cherry */

String Formatting

  • Concept: Formatting strings to a specific layout or structure.
  • Example: 
    name = "John"
age = 30
formattedString = "Name: " || name || ", Age: " || age
say formattedString  /* Output: Name: John, Age: 30 */

Summary

String manipulation in REXX involves a variety of operations, from basic concatenation and substring extraction to more advanced pattern matching and string formatting. Mastering these techniques is essential for effective REXX programming. Practice these concepts with different examples to become proficient in handling strings in REXX.
