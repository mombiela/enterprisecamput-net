Introduction to Data Structures

Data structures are essential for organizing and managing data efficiently. In REXX, understanding how to use and manipulate various data structures will help you write more effective and efficient programs.

Basic Data Types

Before diving into complex data structures, it's important to understand the basic data types in REXX:

  • Strings: The most common data type in REXX, used for text and numbers.
  • Numbers: REXX treats numbers as strings but allows arithmetic operations.

Example: Basic Data Types

/* String Example */
name = "John Doe"
say "Name: " name

/* Number Example */
age = 30
say "Age: " age

Arrays

Arrays in REXX are associative arrays, also known as stem variables. They allow you to store and retrieve data using keys.

Creating and Using Arrays

  • Defining an Array: Use a stem variable followed by a period.
  • Assigning Values: Use the stem variable with an index.
  • Accessing Values: Use the index to retrieve the value.

Example: Arrays

/* Define an array */
students.1 = "Alice"
students.2 = "Bob"
students.3 = "Charlie"

/* Access array elements */
say "Student 1: " students.1
say "Student 2: " students.2
say "Student 3: " students.3

Lists

REXX does not have built-in list data structures, but you can simulate lists using arrays or strings.

Example: Lists Using Arrays

/* Define a list using an array */
list.1 = "apple"
list.2 = "banana"
list.3 = "cherry"

/* Iterate over the list */
do i = 1 to 3
    say "Item " i ": " list.i
end

Example: Lists Using Strings

/* Define a list using a string */
list = "apple banana cherry"

/* Split the string into words */
parse var list item1 item2 item3

say "Item 1: " item1
say "Item 2: " item2
say "Item 3: " item3

Stacks and Queues

Stacks and queues can be implemented using arrays or lists. A stack follows Last In First Out (LIFO) order, while a queue follows First In First Out (FIFO) order.

Example: Stack Implementation

/* Define a stack */
stack.1 = "first"
stack.2 = "second"
stack.3 = "third"

/* Pop from the stack */
do i = 3 to 1 by -1
    say "Popped: " stack.i
end

Example: Queue Implementation

/* Define a queue */
queue.1 = "first"
queue.2 = "second"
queue.3 = "third"

/* Dequeue */
do i = 1 to 3
    say "Dequeued: " queue.i
end

Hash Tables

REXX's associative arrays can be used to create hash tables, where each key maps to a value.

Example: Hash Table

/* Define a hash table */
hashTable.["name"] = "John Doe"
hashTable.["age"] = 30
hashTable.["city"] = "New York"

/* Access hash table elements */
say "Name: " hashTable.["name"]
say "Age: " hashTable.["age"]
say "City: " hashTable.["city"]

Advanced Data Structures

For more complex data structures, you may need to combine basic structures or use external libraries.

Example: Nested Arrays

/* Define a nested array */
students.1.name = "Alice"
students.1.age = 20
students.2.name = "Bob"
students.2.age = 22

/* Access nested array elements */
say "Student 1: " students.1.name ", Age: " students.1.age
say "Student 2: " students.2.name ", Age: " students.2.age

Conclusion

Understanding and utilizing data structures in REXX is crucial for efficient programming. From basic arrays to more complex nested structures, mastering these concepts will enhance your ability to manage and manipulate data effectively. Practice with the provided examples and explore further to deepen your knowledge.
