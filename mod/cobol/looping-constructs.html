Introduction to Looping Constructs

Looping constructs are essential in programming as they allow you to execute a block of code multiple times. In COBOL, loops are primarily implemented using the PERFORM statement. This section will cover the basics of looping constructs in COBOL, progressing from beginner to advanced levels.

Basic Looping with PERFORM

The PERFORM statement is the primary way to create loops in COBOL. It can be used in various ways to achieve different looping behaviors.

Simple PERFORM

The simplest form of the PERFORM statement executes a specified paragraph or section once.

PERFORM PRINT-DETAIL-LINE.

PERFORM with Times

To execute a block of code a specific number of times, use the PERFORM ... TIMES construct.

PERFORM 5 TIMES
    DISPLAY 'Hello, World!'.
END-PERFORM.
  • Explanation: This loop will display "Hello, World!" five times.

PERFORM with UNTIL

The PERFORM ... UNTIL construct allows you to execute a block of code until a condition is met.

MOVE 1 TO COUNTER.
PERFORM UNTIL COUNTER > 5
    DISPLAY 'Counter: ' COUNTER
    ADD 1 TO COUNTER
END-PERFORM.
  • Explanation: This loop will display the counter value from 1 to 5.

Advanced Looping Constructs

As you become more comfortable with basic loops, you can explore more advanced looping constructs in COBOL.

Nested Loops

You can nest PERFORM statements to create more complex looping structures.

PERFORM 3 TIMES
    PERFORM 2 TIMES
        DISPLAY 'Nested Loop'.
    END-PERFORM
END-PERFORM.
  • Explanation: This will display "Nested Loop" six times (3 outer loops * 2 inner loops).

PERFORM VARYING

The PERFORM VARYING construct is used for more complex loops where you need to vary a control variable.

PERFORM VARYING COUNTER FROM 1 BY 1 UNTIL COUNTER > 5
    DISPLAY 'Counter: ' COUNTER
END-PERFORM.
  • Explanation: This loop will display the counter value from 1 to 5, similar to the PERFORM ... UNTIL example but using a varying control variable.

Table Processing with PERFORM

COBOL often uses tables (arrays) for data storage and manipulation. The PERFORM statement can be used to iterate over table elements.

01 STUDENT-TABLE.
   05 STUDENT-ENTRY OCCURS 5 TIMES.
      10 STUDENT-NAME PIC X(20).

MOVE 'Alice' TO STUDENT-NAME(1).
MOVE 'Bob' TO STUDENT-NAME(2).
MOVE 'Charlie' TO STUDENT-NAME(3).
MOVE 'Diana' TO STUDENT-NAME(4).
MOVE 'Eve' TO STUDENT-NAME(5).

PERFORM VARYING INDEX FROM 1 BY 1 UNTIL INDEX > 5
    DISPLAY 'Student Name: ' STUDENT-NAME(INDEX)
END-PERFORM.
  • Explanation: This loop will display the names of the students stored in the STUDENT-TABLE.

Conclusion

Looping constructs in COBOL are powerful tools that allow you to execute code repeatedly based on specific conditions. Starting with simple loops using PERFORM, you can progress to more advanced constructs like nested loops and PERFORM VARYING. Understanding these constructs will enable you to write more efficient and effective COBOL programs.

By mastering these looping constructs, you will be well-equipped to handle repetitive tasks and complex data processing in your COBOL applications.
