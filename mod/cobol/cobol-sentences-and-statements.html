Introduction

COBOL (Common Business-Oriented Language) is a high-level programming language primarily used in business, finance, and administrative systems. Understanding COBOL sentences and statements is crucial for writing effective COBOL programs. This section will cover the basics of COBOL sentences and statements, providing clear explanations, examples, and exercises to reinforce learning.

COBOL Sentences

A COBOL sentence is a sequence of one or more statements that end with a period (.). Sentences are used to perform specific tasks within a COBOL program.

Key Concepts

  • Sentence Structure: A sentence consists of one or more statements and ends with a period.
  • Purpose: Sentences are used to group related statements that perform a specific task.

Example

MOVE 'HELLO' TO GREETING.
DISPLAY GREETING.

In this example, there are two statements within one sentence. The period after DISPLAY GREETING marks the end of the sentence.

COBOL Statements

Statements are the basic building blocks of COBOL programs. Each statement performs a specific operation, such as moving data, displaying output, or performing arithmetic operations.

Key Concepts

  • MOVE Statement: Transfers data from one field to another.
  • DISPLAY Statement: Outputs data to the screen or another output device.
  • COMPUTE Statement: Performs arithmetic operations and assigns the result to a variable.
  • IF Statement: Used for conditional processing.
  • PERFORM Statement: Executes a series of statements repeatedly.

MOVE Statement

The MOVE statement transfers data from one variable to another.

Example

MOVE 'HELLO' TO GREETING.

This statement moves the string 'HELLO' into the variable GREETING.

DISPLAY Statement

The DISPLAY statement outputs data to the screen or another output device.

Example

DISPLAY 'HELLO, WORLD'.

This statement displays the string 'HELLO, WORLD' on the screen.

COMPUTE Statement

The COMPUTE statement performs arithmetic operations and assigns the result to a variable.

Example

COMPUTE TOTAL = PRICE * QUANTITY.

This statement multiplies PRICE by QUANTITY and stores the result in TOTAL.

IF Statement

The IF statement is used for conditional processing.

Example

IF PRICE > 100
    DISPLAY 'EXPENSIVE ITEM'
ELSE
    DISPLAY 'AFFORDABLE ITEM'.

This statement checks if PRICE is greater than 100. If true, it displays 'EXPENSIVE ITEM'; otherwise, it displays 'AFFORDABLE ITEM'.

PERFORM Statement

The PERFORM statement executes a series of statements repeatedly.

Example

PERFORM 10 TIMES
    DISPLAY 'HELLO, WORLD'.

This statement displays 'HELLO, WORLD' ten times.

Summary

In this section, we covered the basics of COBOL sentences and statements. We learned that a COBOL sentence is a sequence of one or more statements ending with a period. We also explored various types of COBOL statements, including MOVE, DISPLAY, COMPUTE, IF, and PERFORM statements, with examples to illustrate their usage.

Understanding these fundamental concepts is essential for writing effective COBOL programs. Practice using these statements in different scenarios to become proficient in COBOL programming.
