String Handling in COBOL

String handling is a fundamental aspect of programming in COBOL, as it is in many other languages. This topic will cover the basics of string handling, including how to define, manipulate, and perform operations on strings in COBOL. By the end of this section, you will have a solid understanding of how to work with strings in COBOL, from beginner to advanced levels.

Introduction to Strings in COBOL

  • Definition: In COBOL, strings are referred to as alphanumeric data items.
  • Usage: Strings are used to store and manipulate text data.

Defining Strings

In COBOL, strings are defined using the PIC (Picture) clause in the Data Division.

01 CUSTOMER-NAME PIC A(30).
01 ADDRESS       PIC X(50).
  • PIC A(n): Defines an alphanumeric string of length n.
  • PIC X(n): Defines a string that can contain any character, including spaces.

Initializing Strings

Strings can be initialized using the VALUE clause.

01 GREETING PIC X(20) VALUE 'Hello, World!'.

Basic String Operations

Concatenation

Concatenation in COBOL can be done using the STRING statement.

STRING FIRST-NAME DELIMITED BY SPACE
       LAST-NAME DELIMITED BY SPACE
       INTO FULL-NAME.
  • Explanation: This code concatenates FIRST-NAME and LAST-NAME into FULL-NAME, with a space delimiter.

Substring Extraction

Extracting a substring can be done using the REFERENCE MODIFIER.

MOVE CUSTOMER-NAME(1:5) TO SHORT-NAME.
  • Explanation: This code extracts the first 5 characters from CUSTOMER-NAME and moves them to SHORT-NAME.

String Length

The length of a string can be determined using the FUNCTION LENGTH.

COMPUTE NAME-LENGTH = FUNCTION LENGTH(CUSTOMER-NAME).
  • Explanation: This code computes the length of CUSTOMER-NAME and stores it in NAME-LENGTH.

Advanced String Operations

String Searching

Searching for a substring within a string can be done using the INSPECT statement.

INSPECT FULL-NAME TALLYING COUNT-OF-A FOR ALL 'A'.
  • Explanation: This code counts the number of occurrences of the character 'A' in FULL-NAME and stores the count in COUNT-OF-A.

String Replacement

Replacing parts of a string can be done using the INSPECT statement with the REPLACING clause.

INSPECT FULL-NAME REPLACING ALL 'A' BY 'E'.
  • Explanation: This code replaces all occurrences of 'A' in FULL-NAME with 'E'.

String Comparison

String comparison can be done using relational operators.

IF CUSTOMER-NAME = 'JOHN DOE'
   DISPLAY 'Match Found'.
  • Explanation: This code checks if CUSTOMER-NAME is equal to 'JOHN DOE' and displays a message if it is.

Practical Examples

Example 1: Concatenating First and Last Name

IDENTIFICATION DIVISION.
PROGRAM-ID. CONCAT-NAME.

DATA DIVISION.
WORKING-STORAGE SECTION.
01 FIRST-NAME PIC X(10) VALUE 'John'.
01 LAST-NAME  PIC X(10) VALUE 'Doe'.
01 FULL-NAME  PIC X(21).

PROCEDURE DIVISION.
    STRING FIRST-NAME DELIMITED BY SPACE
           LAST-NAME DELIMITED BY SPACE
           INTO FULL-NAME
    END-STRING.
    DISPLAY 'Full Name: ' FULL-NAME.
    STOP RUN.

Example 2: Counting Characters in a String

IDENTIFICATION DIVISION.
PROGRAM-ID. COUNT-CHAR.

DATA DIVISION.
WORKING-STORAGE SECTION.
01 FULL-NAME  PIC X(20) VALUE 'Johnathan Doe'.
01 COUNT-OF-A PIC 9(2).

PROCEDURE DIVISION.
    INSPECT FULL-NAME TALLYING COUNT-OF-A FOR ALL 'A'.
    DISPLAY 'Number of A's: ' COUNT-OF-A.
    STOP RUN.

Conclusion

String handling in COBOL involves defining, initializing, and performing various operations on strings. From basic operations like concatenation and substring extraction to advanced operations like searching and replacing, COBOL provides robust tools for managing text data. By mastering these string handling techniques, you will be well-equipped to handle a wide range of text processing tasks in your COBOL programs.
