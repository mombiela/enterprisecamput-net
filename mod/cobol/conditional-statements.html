Introduction

Conditional statements are fundamental in any programming language, including COBOL. They allow the program to make decisions and execute different code blocks based on certain conditions. In this section, we will cover the basics of conditional statements in COBOL, progressing to more advanced concepts.

Basic Conditional Statements

IF Statement

The IF statement is the most basic form of conditional statement in COBOL. It allows the program to execute a block of code only if a specified condition is true.

Syntax

IF condition
    statement-1
    statement-2
    ...
END-IF.

Example

IF AGE > 18
    DISPLAY 'You are an adult.'
END-IF.

In this example, the message "You are an adult." will be displayed only if the value of AGE is greater than 18.

IF-ELSE Statement

The IF-ELSE statement extends the IF statement by providing an alternative block of code to execute if the condition is false.

Syntax

IF condition
    statement-1
ELSE
    statement-2
END-IF.

Example

IF AGE > 18
    DISPLAY 'You are an adult.'
ELSE
    DISPLAY 'You are not an adult.'
END-IF.

In this example, the program will display "You are an adult." if AGE is greater than 18, otherwise, it will display "You are not an adult."

Nested IF Statements

You can nest IF statements within each other to check multiple conditions.

Example

IF AGE > 18
    IF AGE < 65
        DISPLAY 'You are an adult.'
    ELSE
        DISPLAY 'You are a senior.'
    END-IF
ELSE
    DISPLAY 'You are not an adult.'
END-IF.

In this example, the program checks if AGE is greater than 18 and less than 65 to determine if the person is an adult or a senior.

Advanced Conditional Statements

EVALUATE Statement

The EVALUATE statement in COBOL is similar to the SWITCH statement in other programming languages. It allows for multiple conditions to be evaluated in a more readable and organized manner.

Syntax

EVALUATE TRUE
    WHEN condition-1
        statement-1
    WHEN condition-2
        statement-2
    ...
    WHEN OTHER
        statement-n
END-EVALUATE.

Example

EVALUATE TRUE
    WHEN AGE < 13
        DISPLAY 'You are a child.'
    WHEN AGE < 18
        DISPLAY 'You are a teenager.'
    WHEN AGE < 65
        DISPLAY 'You are an adult.'
    WHEN OTHER
        DISPLAY 'You are a senior.'
END-EVALUATE.

In this example, the program evaluates the value of AGE and displays the appropriate message based on the range in which AGE falls.

Complex Conditions

You can use logical operators such as AND and OR to form complex conditions in IF and EVALUATE statements.

Example

IF AGE > 18 AND AGE < 65
    DISPLAY 'You are an adult.'
ELSE
    DISPLAY 'You are either a minor or a senior.'
END-IF.

In this example, the program checks if AGE is between 18 and 65 to determine if the person is an adult.

Conclusion

Conditional statements in COBOL are essential for controlling the flow of a program based on different conditions. Starting from basic IF statements to more advanced EVALUATE statements, understanding these constructs will enable you to write more dynamic and flexible COBOL programs. Practice writing various conditional statements to become proficient in making your COBOL programs responsive to different scenarios.
