Introduction

In this section, we will explore the fundamental concepts of data types and variables in COBOL. Understanding these concepts is crucial for writing effective COBOL programs. We will cover the basic data types, how to declare variables, and how to use them in your programs.

Data Types in COBOL

COBOL supports several data types, each designed to handle different kinds of data. The primary data types in COBOL are:

  • Numeric: Used for numbers.
  • Alphabetic: Used for letters.
  • Alphanumeric: Used for a combination of letters and numbers.

Numeric Data Types

Numeric data types are used to store numbers. They can be further classified into:

  • Integer: Whole numbers without a decimal point.
  • Decimal: Numbers with a decimal point.

Example:

01  NUM-VAR1  PIC 9(5).        * Integer with 5 digits
01  NUM-VAR2  PIC 9(3)V9(2).   * Decimal number with 3 digits before and 2 digits after the decimal point

Alphabetic Data Types

Alphabetic data types are used to store letters only.

Example:

01  ALPHA-VAR  PIC A(10).      * Alphabetic variable with 10 characters

Alphanumeric Data Types

Alphanumeric data types can store a combination of letters and numbers.

Example:

01  ALNUM-VAR  PIC X(20).      * Alphanumeric variable with 20 characters

Declaring Variables

In COBOL, variables are declared in the DATA DIVISION. The WORKING-STORAGE SECTION is commonly used for variable declarations.

Example:

DATA DIVISION.
WORKING-STORAGE SECTION.
01  NUM-VAR1  PIC 9(5).
01  ALPHA-VAR  PIC A(10).
01  ALNUM-VAR  PIC X(20).

Using Variables

Variables in COBOL are used in the PROCEDURE DIVISION to perform operations such as arithmetic calculations, string manipulations, and data movement.

Example:

PROCEDURE DIVISION.
    MOVE 12345 TO NUM-VAR1.
    MOVE 'HELLO' TO ALPHA-VAR.
    MOVE 'HELLO123' TO ALNUM-VAR.
    DISPLAY NUM-VAR1.
    DISPLAY ALPHA-VAR.
    DISPLAY ALNUM-VAR.

Explanation:

  • MOVE: The MOVE statement is used to assign values to variables.
  • DISPLAY: The DISPLAY statement is used to print the value of a variable to the console.

Summary

In this section, we covered the basic data types in COBOL, including numeric, alphabetic, and alphanumeric types. We also learned how to declare variables in the DATA DIVISION and how to use them in the PROCEDURE DIVISION. Understanding these concepts is essential for writing effective COBOL programs. As you progress, you will encounter more complex data structures and operations, but a solid grasp of data types and variables will serve as a strong foundation.
