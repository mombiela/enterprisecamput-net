Introduction to Perform Statements

Perform statements in COBOL are used to control the flow of the program by executing a set of instructions repeatedly or conditionally. They are essential for implementing loops and modularizing code.

Key Concepts

  • Perform Statement: Executes a specified paragraph or section.
  • Inline Perform: Executes a block of code within the perform statement itself.
  • Perform Until: Repeats execution until a condition is met.
  • Perform Varying: Repeats execution with a counter variable that changes value each iteration.

Basic Perform Statement

The basic perform statement executes a specified paragraph or section once.

Example

PERFORM CALCULATE-TOTAL.

In this example, the paragraph CALCULATE-TOTAL will be executed once.

Explanation

  • PERFORM: Keyword to start the perform statement.
  • CALCULATE-TOTAL: The paragraph to be executed.

Inline Perform

Inline perform allows you to execute a block of code directly within the perform statement.

Example

PERFORM
    DISPLAY "Hello, World!"
    MOVE 0 TO COUNTER
END-PERFORM.

Explanation

  • The code between PERFORM and END-PERFORM is executed as a block.
  • Useful for small, self-contained operations.

Perform Until

The perform until statement repeats execution until a specified condition is true.

Example

PERFORM UNTIL COUNTER > 10
    ADD 1 TO COUNTER
    DISPLAY "Counter: " COUNTER
END-PERFORM.

Explanation

  • UNTIL COUNTER > 10: The loop continues until COUNTER is greater than 10.
  • ADD 1 TO COUNTER: Increments the counter by 1 each iteration.
  • DISPLAY "Counter: " COUNTER: Displays the current value of the counter.

Perform Varying

The perform varying statement repeats execution with a counter variable that changes value each iteration.

Example

PERFORM VARYING I FROM 1 BY 1 UNTIL I > 10
    DISPLAY "Iteration: " I
END-PERFORM.

Explanation

  • VARYING I FROM 1 BY 1 UNTIL I > 10: Initializes I to 1, increments I by 1 each iteration, and stops when I is greater than 10.
  • DISPLAY "Iteration: " I: Displays the current iteration number.

Advanced Perform Statements

Advanced perform statements include nested performs and perform with test after.

Nested Perform

You can nest perform statements to create complex loops.

Example

PERFORM VARYING I FROM 1 BY 1 UNTIL I > 5
    PERFORM VARYING J FROM 1 BY 1 UNTIL J > 3
        DISPLAY "I: " I " J: " J
    END-PERFORM
END-PERFORM.

Explanation

  • Outer loop runs from I = 1 to I = 5.
  • Inner loop runs from J = 1 to J = 3 for each iteration of the outer loop.
  • Displays the values of I and J for each combination.

Perform with Test After

Executes the block of code at least once before testing the condition.

Example

PERFORM WITH TEST AFTER
    DISPLAY "This will always display at least once."
    ADD 1 TO COUNTER
UNTIL COUNTER > 5.

Explanation

  • WITH TEST AFTER: Ensures the block is executed at least once before checking the condition.
  • Useful when the loop body must execute at least once regardless of the condition.

Conclusion

Perform statements in COBOL are powerful tools for controlling program flow. They allow for repeated execution of code blocks, conditional execution, and modularization of code. Understanding and using perform statements effectively can greatly enhance the readability and maintainability of COBOL programs.
