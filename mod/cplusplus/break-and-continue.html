Break and Continue in C++
Introduction to Functions ►

Introduction

In this section, we will explore the break and continue statements in C++. These control flow statements are used to alter the normal flow of loops, providing more control over how and when loops are executed.

Break Statement

The break statement is used to exit a loop prematurely. When a break statement is encountered inside a loop, the loop is immediately terminated, and the program control resumes at the next statement following the loop.

Key Concepts

  • Immediate Termination: The break statement causes the loop to terminate immediately.
  • Applicable Loops: Can be used in for, while, and do-while loops.
  • Nested Loops: When used in nested loops, break only exits the innermost loop.

Example

#include <iostream>
using namespace std;

int main() {
    for (int i = 0; i < 10; i++) {
        if (i == 5) {
            break; // Exit the loop when i is 5
        }
        cout << i << " ";
    }
    cout << "Loop terminated." << endl;
    return 0;
}

Explanation: In this example, the loop will print numbers from 0 to 4. When i equals 5, the break statement is executed, terminating the loop.

Continue Statement

The continue statement skips the current iteration of the loop and proceeds with the next iteration. Unlike break, it does not terminate the loop but simply skips the remaining code in the current iteration.

Key Concepts

  • Skip Iteration: The continue statement skips the rest of the code in the current loop iteration.
  • Applicable Loops: Can be used in for, while, and do-while loops.
  • Nested Loops: When used in nested loops, continue only affects the innermost loop.

Example

#include <iostream>
using namespace std;

int main() {
    for (int i = 0; i < 10; i++) {
        if (i % 2 == 0) {
            continue; // Skip the rest of the loop body for even numbers
        }
        cout << i << " ";
    }
    cout << "Loop completed." << endl;
    return 0;
}

Explanation: In this example, the loop will print only odd numbers from 1 to 9. When i is even, the continue statement is executed, skipping the cout statement for that iteration.

Comparison of Break and Continue

Feature Break Continue
Effect Terminates the loop Skips the current iteration
Loop Termination Yes No
Nested Loop Effect Affects only the innermost loop Affects only the innermost loop

Exercises

  1. Exercise 1: Write a program that uses a for loop to print numbers from 1 to 20. Use the break statement to exit the loop when the number is 15.
  2. Exercise 2: Write a program that uses a while loop to print numbers from 1 to 10. Use the continue statement to skip printing the number 5.

Exercise 1 Solution

#include <iostream>
using namespace std;

int main() {
    for (int i = 1; i <= 20; i++) {
        if (i == 15) {
            break; // Exit the loop when i is 15
        }
        cout << i << " ";
    }
    cout << "Loop terminated at 15." << endl;
    return 0;
}

Exercise 2 Solution

#include <iostream>
using namespace std;

int main() {
    int i = 1;
    while (i <= 10) {
        if (i == 5) {
            i++;
            continue; // Skip the rest of the loop body when i is 5
        }
        cout << i << " ";
        i++;
    }
    cout << "Loop completed, skipped 5." << endl;
    return 0;
}

Conclusion

The break and continue statements are powerful tools for controlling the flow of loops in C++. The break statement allows you to exit a loop prematurely, while the continue statement lets you skip the current iteration and proceed with the next one. Understanding how and when to use these statements can help you write more efficient and readable code.
