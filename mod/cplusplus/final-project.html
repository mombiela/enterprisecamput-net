Final Project

Overview

The final project is designed to consolidate your knowledge and skills acquired throughout this C++ course. This project will challenge you to apply various concepts, from basic syntax to advanced programming techniques, in a cohesive and practical manner. By the end of this project, you should have a comprehensive understanding of C++ and be able to develop a fully functional application.

Project Description

You will create a simple text-based game using C++. The game will involve a player navigating through a series of rooms, collecting items, and solving puzzles to achieve a specific goal. This project will require you to use:

  • Basic syntax and control structures
  • Functions and modular programming
  • Object-Oriented Programming (OOP)
  • File I/O operations
  • Dynamic memory management
  • Error handling

Requirements

Basic Game Structure

  1. Main Menu:

    • Display options to start a new game, load a saved game, or exit.
    • Example: 
      void displayMainMenu() {
    std::cout << "1. Start New Game\n";
    std::cout << "2. Load Game\n";
    std::cout << "3. Exit\n";
    std::cout << "Enter your choice: ";
}

  2. Game Loop:

    • Continuously run until the player wins or chooses to exit.
    • Example: 
      bool gameRunning = true;
while (gameRunning) {
    // Game logic here
    if (playerWins || playerExits) {
        gameRunning = false;
    }
}

Player and Room Classes

  1. Player Class:

    • Attributes: name, health, inventory.
    • Methods: move, pick up item, use item.
    • Example: 
      class Player {
public:
    std::string name;
    int health;
    std::vector<std::string> inventory;

    void move();
    void pickUpItem(const std::string& item);
    void useItem(const std::string& item);
};

  2. Room Class:

    • Attributes: description, items, exits.
    • Methods: add item, remove item, describe room.
    • Example: 
      class Room {
public:
    std::string description;
    std::vector<std::string> items;
    std::map<std::string, Room*> exits;

    void addItem(const std::string& item);
    void removeItem(const std::string& item);
    void describeRoom();
};

Game Mechanics

  1. Navigation:

    • Allow the player to move between rooms.
    • Example: 
      void Player::move(Room* currentRoom, const std::string& direction) {
    if (currentRoom->exits.find(direction) != currentRoom->exits.end()) {
        currentRoom = currentRoom->exits[direction];
        std::cout << "You moved to the " << direction << ".\n";
    } else {
        std::cout << "You can't go that way.\n";
    }
}

  2. Item Management:

    • Pick up and use items.
    • Example: 
      void Player::pickUpItem(const std::string& item) {
    inventory.push_back(item);
    std::cout << "You picked up a " << item << ".\n";
}

void Player::useItem(const std::string& item) {
    auto it = std::find(inventory.begin(), inventory.end(), item);
    if (it != inventory.end()) {
        inventory.erase(it);
        std::cout << "You used the " << item << ".\n";
    } else {
        std::cout << "You don't have a " << item << ".\n";
    }
}

Saving and Loading Game State

  1. File I/O:
    • Save the current game state to a file.
    • Load a saved game state from a file.
    • Example: 
      void saveGame(const Player& player, const Room& room) {
    std::ofstream outFile("savegame.txt");
    outFile << player.name << "\n" << player.health << "\n";
    for (const auto& item : player.inventory) {
        outFile << item << "\n";
    }
    outFile.close();
}

void loadGame(Player& player, Room& room) {
    std::ifstream inFile("savegame.txt");
    inFile >> player.name >> player.health;
    std::string item;
    while (inFile >> item) {
        player.inventory.push_back(item);
    }
    inFile.close();
}

Conclusion

The final project is an excellent opportunity to put your C++ skills to the test. By creating a text-based game, you will gain hands-on experience with various programming concepts and techniques. This project will not only solidify your understanding of C++ but also enhance your problem-solving and critical-thinking abilities. Good luck, and have fun coding!
