Input and Output in C++

Introduction

Input and output (I/O) are fundamental concepts in any programming language. In C++, I/O operations are primarily handled using streams. This section will cover the basics of I/O in C++, including how to read from the keyboard and write to the console, as well as more advanced topics like file I/O and custom stream manipulators.

Basic Console I/O

Reading from the Console

To read input from the console, C++ provides the cin object, which is part of the standard library.

Example:

#include <iostream>

int main() {
    int number;
    std::cout << "Enter a number: ";
    std::cin >> number;
    std::cout << "You entered: " << number << std::endl;
    return 0;
}
  • std::cin is used to read input from the standard input (keyboard).
  • std::cout is used to write output to the standard output (console).
  • std::endl is used to insert a newline character and flush the stream.

Writing to the Console

To write output to the console, C++ uses the cout object.

Example:

#include <iostream>

int main() {
    std::cout << "Hello, World!" << std::endl;
    return 0;
}
  • << is the insertion operator used to send data to the output stream.

Advanced Console I/O

Formatting Output

C++ provides several manipulators to format the output.

Example:

#include <iostream>
#include <iomanip>

int main() {
    double pi = 3.14159;
    std::cout << "Default: " << pi << std::endl;
    std::cout << "Fixed: " << std::fixed << pi << std::endl;
    std::cout << "Scientific: " << std::scientific << pi << std::endl;
    std::cout << "Precision: " << std::setprecision(2) << pi << std::endl;
    return 0;
}
  • std::fixed and std::scientific are used to format floating-point numbers.
  • std::setprecision sets the number of digits to display after the decimal point.

File I/O

Reading from a File

To read from a file, C++ uses the ifstream class.

Example:

#include <iostream>
#include <fstream>
#include <string>

int main() {
    std::ifstream inputFile("example.txt");
    std::string line;

    if (inputFile.is_open()) {
        while (getline(inputFile, line)) {
            std::cout << line << std::endl;
        }
        inputFile.close();
    } else {
        std::cout << "Unable to open file" << std::endl;
    }

    return 0;
}
  • std::ifstream is used to read from files.
  • getline reads a line from the file.

Writing to a File

To write to a file, C++ uses the ofstream class.

Example:

#include <iostream>
#include <fstream>

int main() {
    std::ofstream outputFile("example.txt");

    if (outputFile.is_open()) {
        outputFile << "This is a line.\n";
        outputFile << "This is another line.\n";
        outputFile.close();
    } else {
        std::cout << "Unable to open file" << std::endl;
    }

    return 0;
}
  • std::ofstream is used to write to files.
  • The << operator is used to send data to the file.

Custom Stream Manipulators

C++ allows you to create custom manipulators to format output in specific ways.

Example:

#include <iostream>
#include <iomanip>

std::ostream& customManipulator(std::ostream& os) {
    return os << std::setw(10) << std::setfill('*');
}

int main() {
    std::cout << customManipulator << 123 << std::endl;
    return 0;
}
  • std::setw sets the width of the output field.
  • std::setfill sets the fill character.

Conclusion

Understanding input and output in C++ is crucial for effective programming. This section covered the basics of console I/O, advanced formatting techniques, file I/O, and custom stream manipulators. Mastering these concepts will enable you to handle data efficiently and present it in a user-friendly manner.
