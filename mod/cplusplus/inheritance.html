Inheritance in C++
Polymorphism ►

Inheritance is a fundamental concept in object-oriented programming (OOP) that allows a class to inherit properties and behaviors (methods) from another class. This promotes code reusability and establishes a natural hierarchy between classes.

What is Inheritance?

Inheritance enables a new class (derived class) to inherit attributes and methods from an existing class (base class). This allows the derived class to reuse code from the base class and extend or modify its behavior.

Key Concepts

  • Base Class: The class whose properties and methods are inherited.
  • Derived Class: The class that inherits properties and methods from the base class.
  • Access Specifiers: Keywords that define the accessibility of the base class members in the derived class (public, protected, private).
  • Overriding: Redefining a base class method in the derived class.
  • Polymorphism: The ability to call derived class methods through a base class reference.

Basic Syntax

Defining a Base Class

class Base {
public:
    int baseValue;
    void display() {
        std::cout << "Base class display function" << std::endl;
    }
};

Defining a Derived Class

class Derived : public Base {
public:
    int derivedValue;
    void show() {
        std::cout << "Derived class show function" << std::endl;
    }
};

Creating Objects and Accessing Members

int main() {
    Derived obj;
    obj.baseValue = 10; // Accessing base class member
    obj.derivedValue = 20; // Accessing derived class member
    obj.display(); // Calling base class method
    obj.show(); // Calling derived class method
    return 0;
}

Access Specifiers in Inheritance

Access specifiers control the accessibility of the base class members in the derived class.

Access Specifier Base Class Member Access in Derived Class Outside Access
public Public Public
protected Protected Private
private Private Private

Example with Access Specifiers

class Base {
protected:
    int protectedValue;
public:
    int publicValue;
};

class Derived : public Base {
public:
    void setValues() {
        protectedValue = 10; // Accessible
        publicValue = 20; // Accessible
    }
};

Overriding Methods

A derived class can override a base class method to provide a specific implementation.

Example of Method Overriding

class Base {
public:
    virtual void display() {
        std::cout << "Base class display function" << std::endl;
    }
};

class Derived : public Base {
public:
    void display() override {
        std::cout << "Derived class display function" << std::endl;
    }
};

int main() {
    Base* basePtr;
    Derived derivedObj;
    basePtr = &derivedObj;
    basePtr->display(); // Calls Derived class display function
    return 0;
}

Polymorphism

Polymorphism allows the same interface to be used for different data types. In C++, this is often achieved through function overriding and pointers or references to base class types.

Example of Polymorphism

class Base {
public:
    virtual void display() {
        std::cout << "Base class display function" << std::endl;
    }
};

class Derived : public Base {
public:
    void display() override {
        std::cout << "Derived class display function" << std::endl;
    }
};

void showDisplay(Base* basePtr) {
    basePtr->display();
}

int main() {
    Base baseObj;
    Derived derivedObj;
    showDisplay(&baseObj); // Calls Base class display function
    showDisplay(&derivedObj); // Calls Derived class display function
    return 0;
}

Conclusion

Inheritance is a powerful feature in C++ that promotes code reuse and establishes a natural hierarchy between classes. By understanding and utilizing inheritance, you can create more modular, maintainable, and scalable code. Key concepts such as access specifiers, method overriding, and polymorphism are essential to mastering inheritance in C++. Practice these concepts with various examples to solidify your understanding.
