Regular Expressions in C++
Project Setup and Planning ►

Regular expressions (regex) are powerful tools for pattern matching and text processing. In C++, the <regex> library provides support for working with regular expressions. This topic will guide you from the basics to advanced usage of regular expressions in C++.

Introduction to Regular Expressions

  • Definition: Regular expressions are sequences of characters that define search patterns, mainly for use in pattern matching with strings.
  • Use Cases: Validating input, searching within strings, replacing substrings, and parsing text.

Basics of Regular Expressions

Including the Regex Library

To use regular expressions in C++, you need to include the <regex> header:

#include <regex>

Basic Syntax

  • Literal Characters: Match themselves.
  • Metacharacters: Characters with special meanings (e.g., . for any character, ^ for the start of a string, $ for the end of a string).

Simple Example

#include <iostream>
#include <regex>

int main() {
    std::string text = "Hello, World!";
    std::regex pattern("World");

    if (std::regex_search(text, pattern)) {
        std::cout << "Pattern found!" << std::endl;
    } else {
        std::cout << "Pattern not found." << std::endl;
    }

    return 0;
}

Explanation

  • std::regex: Represents a compiled regular expression.
  • std::regex_search: Searches for the pattern within the text.

Common Regex Patterns

  • Character Classes: [abc] matches any single character a, b, or c.
  • Predefined Character Classes: \d for digits, \w for word characters, \s for whitespace.
  • Quantifiers: * (0 or more), + (1 or more), ? (0 or 1), {n} (exactly n), {n,} (n or more), {n,m} (between n and m).

Example with Quantifiers

#include <iostream>
#include <regex>

int main() {
    std::string text = "The number is 12345.";
    std::regex pattern("\\d+");

    if (std::regex_search(text, pattern)) {
        std::cout << "Number found!" << std::endl;
    } else {
        std::cout << "Number not found." << std::endl;
    }

    return 0;
}

Advanced Regular Expressions

Capturing Groups

  • Definition: Parentheses () are used to create capturing groups.
  • Use Case: Extracting specific parts of a match.

Example with Capturing Groups

#include <iostream>
#include <regex>

int main() {
    std::string text = "My email is example@example.com.";
    std::regex pattern("(\\w+)@(\\w+\\.\\w+)");

    std::smatch matches;
    if (std::regex_search(text, matches, pattern)) {
        std::cout << "Username: " << matches[1] << std::endl;
        std::cout << "Domain: " << matches[2] << std::endl;
    } else {
        std::cout << "No match found." << std::endl;
    }

    return 0;
}

Explanation

  • std::smatch: Holds the results of the search.
  • matches[n]: Accesses the nth capturing group.

Regex Replace

  • Function: std::regex_replace replaces occurrences of the pattern with a specified string.

Example of Regex Replace

#include <iostream>
#include <regex>

int main() {
    std::string text = "I have 2 apples and 3 oranges.";
    std::regex pattern("\\d");
    std::string replacement = "#";

    std::string result = std::regex_replace(text, pattern, replacement);
    std::cout << "Result: " << result << std::endl;

    return 0;
}

Explanation

  • std::regex_replace: Replaces all occurrences of the pattern with the replacement string.

Performance Considerations

  • Compilation: Compiling a regex can be expensive. Reuse compiled regex objects when possible.
  • Complexity: Be cautious with complex patterns as they can lead to performance issues.

Conclusion

Regular expressions are a versatile tool in C++ for text processing. Starting from basic pattern matching to advanced text manipulation, understanding regex can significantly enhance your ability to handle strings effectively. Practice with different patterns and use cases to become proficient in using regular expressions in your C++ programs.
