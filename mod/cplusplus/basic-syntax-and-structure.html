Introduction

C++ is a powerful, high-performance programming language that supports both procedural and object-oriented programming paradigms. Understanding the basic syntax and structure of C++ is essential for writing efficient and effective code. This section will cover the fundamental elements of C++ syntax and structure, providing a solid foundation for further learning.

Key Concepts

  • Comments
  • Data Types
  • Variables
  • Operators
  • Control Structures
  • Functions

Comments

Comments are used to explain code and make it more readable. They are ignored by the compiler.

  • Single-line comments: Use // to comment out a single line.
  • Multi-line comments: Use /* */ to comment out multiple lines.
// This is a single-line comment

/*
This is a 
multi-line comment
*/

Data Types

C++ supports several data types, which can be broadly categorized into:

  • Primitive Data Types: int, float, double, char, bool
  • Derived Data Types: Arrays, Pointers, References
  • User-defined Data Types: Structures, Unions, Classes, Enums

Example:

int age = 25;         // Integer type
float height = 5.9;   // Floating-point type
char grade = 'A';     // Character type
bool isStudent = true; // Boolean type

Variables

Variables are used to store data that can be manipulated by the program. They must be declared before use.

Syntax:

data_type variable_name = value;

Example:

int number = 10;
float temperature = 36.6;
char initial = 'J';
bool isValid = false;

Operators

Operators are symbols that perform operations on variables and values. Common operators include:

  • Arithmetic Operators: +, -, *, /, %
  • Relational Operators: ==, !=, >, <, >=, <=
  • Logical Operators: &&, ||, !
  • Assignment Operators: =, +=, -=, *=, /=, %=

Example:

int a = 10, b = 20;
int sum = a + b; // Arithmetic operator
bool result = (a == b); // Relational operator
bool logicalResult = (a > 5 && b < 30); // Logical operator
a += 5; // Assignment operator

Control Structures

Control structures dictate the flow of the program. Common control structures include:

  • Conditional Statements: if, else if, else, switch
  • Loops: for, while, do-while

Example:

// Conditional Statement
int num = 10;
if (num > 0) {
    cout << "Positive number";
} else {
    cout << "Non-positive number";
}

// Loop
for (int i = 0; i < 5; i++) {
    cout << i << " ";
}

Functions

Functions are blocks of code that perform a specific task. They help in modularizing the code and reusing it.

Syntax:

return_type function_name(parameters) {
    // Function body
}

Example:

int add(int a, int b) {
    return a + b;
}

int main() {
    int result = add(5, 3);
    cout << "Sum: " << result;
    return 0;
}

Conclusion

Understanding the basic syntax and structure of C++ is crucial for any programmer. This section covered the essential elements, including comments, data types, variables, operators, control structures, and functions. Mastering these basics will provide a strong foundation for more advanced topics in C++.
