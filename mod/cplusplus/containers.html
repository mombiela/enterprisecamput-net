Introduction to Containers

Containers are fundamental components in C++ that allow you to store and manage collections of data. They are part of the Standard Template Library (STL) and provide various ways to organize and manipulate data efficiently.

Key Concepts

  • Sequence Containers: Store data in a linear fashion.
  • Associative Containers: Store data in a way that allows fast retrieval using keys.
  • Container Adapters: Provide a different interface for sequential containers.

Sequence Containers

Sequence containers maintain the ordering of elements.

Vector

A dynamic array that can resize itself automatically when an element is added or removed.

#include <iostream>
#include <vector>

int main() {
    std::vector<int> numbers = {1, 2, 3, 4, 5};
    numbers.push_back(6); // Add an element to the end

    for (int num : numbers) {
        std::cout << num << " ";
    }
    return 0;
}

List

A doubly linked list that allows efficient insertion and deletion of elements.

#include <iostream>
#include <list>

int main() {
    std::list<int> numbers = {1, 2, 3, 4, 5};
    numbers.push_back(6); // Add an element to the end

    for (int num : numbers) {
        std::cout << num << " ";
    }
    return 0;
}

Deque

A double-ended queue that allows insertion and deletion at both ends.

#include <iostream>
#include <deque>

int main() {
    std::deque<int> numbers = {1, 2, 3, 4, 5};
    numbers.push_front(0); // Add an element to the front

    for (int num : numbers) {
        std::cout << num << " ";
    }
    return 0;
}

Associative Containers

Associative containers store elements formed by a combination of a key value and a mapped value, and allow fast retrieval of elements based on the keys.

Map

A collection of key-value pairs with unique keys.

#include <iostream>
#include <map>

int main() {
    std::map<int, std::string> students;
    students[1] = "Alice";
    students[2] = "Bob";

    for (const auto& pair : students) {
        std::cout << pair.first << ": " << pair.second << std::endl;
    }
    return 0;
}

Set

A collection of unique keys.

#include <iostream>
#include <set>

int main() {
    std::set<int> numbers = {1, 2, 3, 4, 5};
    numbers.insert(6); // Add an element

    for (int num : numbers) {
        std::cout << num << " ";
    }
    return 0;
}

Container Adapters

Container adapters provide a different interface for sequential containers.

Stack

A LIFO (Last In, First Out) data structure.

#include <iostream>
#include <stack>

int main() {
    std::stack<int> numbers;
    numbers.push(1);
    numbers.push(2);
    numbers.push(3);

    while (!numbers.empty()) {
        std::cout << numbers.top() << " ";
        numbers.pop();
    }
    return 0;
}

Queue

A FIFO (First In, First Out) data structure.

#include <iostream>
#include <queue>

int main() {
    std::queue<int> numbers;
    numbers.push(1);
    numbers.push(2);
    numbers.push(3);

    while (!numbers.empty()) {
        std::cout << numbers.front() << " ";
        numbers.pop();
    }
    return 0;
}

Priority Queue

A data structure where elements are processed based on priority.

#include <iostream>
#include <queue>

int main() {
    std::priority_queue<int> numbers;
    numbers.push(3);
    numbers.push(1);
    numbers.push(2);

    while (!numbers.empty()) {
        std::cout << numbers.top() << " ";
        numbers.pop();
    }
    return 0;
}

Summary

In this section, we covered the different types of containers available in C++:

  • Sequence Containers: Vector, List, Deque.
  • Associative Containers: Map, Set.
  • Container Adapters: Stack, Queue, Priority Queue.

Understanding these containers and their appropriate use cases is crucial for efficient data management and manipulation in C++. Each container type offers unique advantages and is suited for specific scenarios, making them powerful tools in a C++ programmer's toolkit.
