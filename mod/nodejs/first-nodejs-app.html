Introduction

Welcome to your first Node.js application! In this section, we will guide you through setting up a basic Node.js environment and creating a simple application. By the end of this lesson, you will have a solid foundation to build more complex Node.js applications.

Prerequisites

Before we start, make sure you have the following:

  • Node.js installed on your computer. You can download it from nodejs.org.
  • A code editor like Visual Studio Code, Sublime Text, or any other of your choice.

Setting Up Your Environment

  1. Install Node.js: Follow the instructions on the Node.js website to download and install Node.js.
  2. Verify Installation: Open your terminal or command prompt and type: 
    node -v
    This command should display the version of Node.js installed on your system.

Creating Your First Node.js Application

Let's create a simple "Hello, World!" application to get started.

Step 1: Create a Project Directory

First, create a new directory for your project and navigate into it:

mkdir my-first-node-app
cd my-first-node-app

Step 2: Initialize a Node.js Project

Initialize your project with npm (Node Package Manager):

npm init -y

This command will create a package.json file with default settings.

Step 3: Create the Application File

Create a new file named app.js in your project directory. This will be the main file for your application.

Step 4: Write Your First Node.js Code

Open app.js in your code editor and add the following code:

// app.js
console.log('Hello, World!');

Step 5: Run Your Application

Go back to your terminal and run the application using the following command:

node app.js

You should see the output:

Hello, World!

Understanding the Code

  • console.log('Hello, World!');: This line of code prints the string 'Hello, World!' to the console. The console object in Node.js provides a simple debugging console that is similar to the JavaScript console in web browsers.

Adding More Functionality

Let's extend our application to create a simple HTTP server that responds with "Hello, World!" when accessed via a web browser.

Step 1: Import the HTTP Module

Update your app.js file to include the HTTP module:

const http = require('http');

Step 2: Create the Server

Add the following code to create an HTTP server:

const server = http.createServer((req, res) => {
  res.statusCode = 200;
  res.setHeader('Content-Type', 'text/plain');
  res.end('Hello, World!\n');
});

Step 3: Listen on a Port

Add the following code to make the server listen on port 3000:

const port = 3000;
server.listen(port, () => {
  console.log(`Server running at http://localhost:${port}/`);
});

Complete Code Example

Here is the complete app.js file:

const http = require('http');

const server = http.createServer((req, res) => {
  res.statusCode = 200;
  res.setHeader('Content-Type', 'text/plain');
  res.end('Hello, World!\n');
});

const port = 3000;
server.listen(port, () => {
  console.log(`Server running at http://localhost:${port}/`);
});

Running the Server

Run your application again:

node app.js

Open your web browser and navigate to http://localhost:3000/. You should see "Hello, World!" displayed in your browser.

Summary

Congratulations! You've created your first Node.js application. In this lesson, you learned how to:

  • Set up a Node.js project.
  • Write a simple "Hello, World!" application.
  • Create an HTTP server using Node.js.

This foundational knowledge will be crucial as you move on to more advanced topics in Node.js. Keep experimenting and building on what you've learned!
