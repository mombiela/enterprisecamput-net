HTTP Module in Node.js
Creating a Simple Web Server ►

The HTTP module in Node.js allows you to create web servers and handle HTTP requests and responses. This module is fundamental for building web applications and APIs in Node.js.

Introduction to the HTTP Module

  • The HTTP module is built into Node.js, so no additional installation is required.
  • It provides functionalities to create both HTTP servers and clients.

Basic HTTP Server

To create a basic HTTP server, you need to import the HTTP module and use the createServer method.

const http = require('http');

const server = http.createServer((req, res) => {
  res.statusCode = 200;
  res.setHeader('Content-Type', 'text/plain');
  res.end('Hello, World!\n');
});

const PORT = 3000;
server.listen(PORT, () => {
  console.log(`Server running at http://localhost:${PORT}/`);
});
  • http.createServer(callback): Creates a new HTTP server.
  • req: Represents the request object.
  • res: Represents the response object.
  • res.statusCode: Sets the HTTP status code.
  • res.setHeader(): Sets the response headers.
  • res.end(): Ends the response process.

Handling Different HTTP Methods

HTTP methods like GET, POST, PUT, DELETE, etc., are used to perform different actions on the server.

Example: Handling GET and POST Requests

const http = require('http');

const server = http.createServer((req, res) => {
  if (req.method === 'GET') {
    res.statusCode = 200;
    res.setHeader('Content-Type', 'text/plain');
    res.end('Received a GET request\n');
  } else if (req.method === 'POST') {
    let body = '';
    req.on('data', chunk => {
      body += chunk.toString();
    });
    req.on('end', () => {
      res.statusCode = 200;
      res.setHeader('Content-Type', 'application/json');
      res.end(`Received a POST request with data: ${body}\n`);
    });
  } else {
    res.statusCode = 405;
    res.end('Method Not Allowed\n');
  }
});

const PORT = 3000;
server.listen(PORT, () => {
  console.log(`Server running at http://localhost:${PORT}/`);
});
  • req.method: Checks the HTTP method of the request.
  • req.on('data', callback): Listens for data events when receiving data chunks.
  • req.on('end', callback): Listens for the end of the data transmission.

Working with Query Parameters and URL Routing

Parsing Query Parameters

To parse query parameters, you can use the url module.

const http = require('http');
const url = require('url');

const server = http.createServer((req, res) => {
  const parsedUrl = url.parse(req.url, true);
  const query = parsedUrl.query;

  res.statusCode = 200;
  res.setHeader('Content-Type', 'application/json');
  res.end(JSON.stringify(query));
});

const PORT = 3000;
server.listen(PORT, () => {
  console.log(`Server running at http://localhost:${PORT}/`);
});
  • url.parse(urlString, true): Parses the URL and returns an object with query parameters.

Basic URL Routing

Routing allows you to handle different endpoints.

const http = require('http');

const server = http.createServer((req, res) => {
  const url = req.url;

  if (url === '/') {
    res.statusCode = 200;
    res.setHeader('Content-Type', 'text/plain');
    res.end('Home Page\n');
  } else if (url === '/about') {
    res.statusCode = 200;
    res.setHeader('Content-Type', 'text/plain');
    res.end('About Page\n');
  } else {
    res.statusCode = 404;
    res.setHeader('Content-Type', 'text/plain');
    res.end('Page Not Found\n');
  }
});

const PORT = 3000;
server.listen(PORT, () => {
  console.log(`Server running at http://localhost:${PORT}/`);
});

Advanced Topics

Handling JSON Data

When working with APIs, you often need to handle JSON data.

const http = require('http');

const server = http.createServer((req, res) => {
  if (req.method === 'POST' && req.headers['content-type'] === 'application/json') {
    let body = '';
    req.on('data', chunk => {
      body += chunk.toString();
    });
    req.on('end', () => {
      const data = JSON.parse(body);
      res.statusCode = 200;
      res.setHeader('Content-Type', 'application/json');
      res.end(JSON.stringify({ message: 'Data received', data }));
    });
  } else {
    res.statusCode = 400;
    res.end('Bad Request\n');
  }
});

const PORT = 3000;
server.listen(PORT, () => {
  console.log(`Server running at http://localhost:${PORT}/`);
});

HTTPS Server

For secure communication, you can create an HTTPS server.

const https = require('https');
const fs = require('fs');

const options = {
  key: fs.readFileSync('path/to/private-key.pem'),
  cert: fs.readFileSync('path/to/certificate.pem')
};

const server = https.createServer(options, (req, res) => {
  res.statusCode = 200;
  res.setHeader('Content-Type', 'text/plain');
  res.end('Hello, Secure World!\n');
});

const PORT = 3000;
server.listen(PORT, () => {
  console.log(`Server running at https://localhost:${PORT}/`);
});
  • https.createServer(options, callback): Creates an HTTPS server with the provided options.

Conclusion

The HTTP module in Node.js is a powerful tool for creating web servers and handling HTTP requests and responses. By understanding the basics and advanced features, you can build robust web applications and APIs. Practice by creating different types of servers and handling various HTTP methods and data formats to solidify your understanding.
