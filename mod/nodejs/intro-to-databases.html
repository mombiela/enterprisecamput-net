Overview

In this section, we will explore the fundamentals of databases and how to interact with them using Node.js. Databases are essential for storing, retrieving, and managing data in web applications. We will cover both SQL and NoSQL databases, their differences, and how to integrate them with Node.js.

Key Concepts

  • Database: A structured collection of data.
  • SQL (Structured Query Language): A language used to manage and manipulate relational databases.
  • NoSQL (Not Only SQL): A category of database management systems that do not use SQL as their primary query language.
  • CRUD Operations: The four basic operations for managing data - Create, Read, Update, Delete.

Types of Databases

SQL Databases

  • Examples: MySQL, PostgreSQL, SQLite
  • Characteristics:
    • Use tables to store data.
    • Data is structured in rows and columns.
    • Support for ACID (Atomicity, Consistency, Isolation, Durability) transactions.

NoSQL Databases

  • Examples: MongoDB, CouchDB, Redis
  • Characteristics:
    • Use various data models like document, key-value, graph, or column-family.
    • Schema-less or flexible schema.
    • Designed for scalability and performance.

Setting Up a Database with Node.js

Installing Required Packages

To interact with databases in Node.js, you need to install specific packages. Here are some common ones:

  • For MySQL:

    npm install mysql

  • For MongoDB:

    npm install mongodb

Connecting to a Database

MySQL Example

const mysql = require('mysql');

const connection = mysql.createConnection({
  host: 'localhost',
  user: 'root',
  password: 'password',
  database: 'test_db'
});

connection.connect((err) => {
  if (err) {
    console.error('Error connecting: ' + err.stack);
    return;
  }
  console.log('Connected as id ' + connection.threadId);
});
  • Explanation:
    • mysql.createConnection: Creates a connection to the MySQL database.
    • connection.connect: Establishes the connection.

MongoDB Example

const { MongoClient } = require('mongodb');

const url = 'mongodb://localhost:27017';
const dbName = 'test_db';

MongoClient.connect(url, { useNewUrlParser: true, useUnifiedTopology: true }, (err, client) => {
  if (err) {
    console.error('Error connecting: ' + err.stack);
    return;
  }
  console.log('Connected successfully to server');
  const db = client.db(dbName);
  client.close();
});
  • Explanation:
    • MongoClient.connect: Connects to the MongoDB server.
    • client.db: Accesses the specific database.

Performing CRUD Operations

Create

MySQL

const sql = 'INSERT INTO users (name, age) VALUES (?, ?)';
const values = ['John Doe', 30];

connection.query(sql, values, (err, result) => {
  if (err) throw err;
  console.log('Record inserted, ID: ' + result.insertId);
});

MongoDB

const collection = db.collection('users');
collection.insertOne({ name: 'John Doe', age: 30 }, (err, result) => {
  if (err) throw err;
  console.log('Record inserted, ID: ' + result.insertedId);
});

Read

MySQL

const sql = 'SELECT * FROM users WHERE age > ?';
const age = 25;

connection.query(sql, [age], (err, results) => {
  if (err) throw err;
  console.log(results);
});

MongoDB

const collection = db.collection('users');
collection.find({ age: { $gt: 25 } }).toArray((err, docs) => {
  if (err) throw err;
  console.log(docs);
});

Update

MySQL

const sql = 'UPDATE users SET age = ? WHERE name = ?';
const values = [35, 'John Doe'];

connection.query(sql, values, (err, result) => {
  if (err) throw err;
  console.log('Record updated, affected rows: ' + result.affectedRows);
});

MongoDB

const collection = db.collection('users');
collection.updateOne({ name: 'John Doe' }, { $set: { age: 35 } }, (err, result) => {
  if (err) throw err;
  console.log('Record updated, modified count: ' + result.modifiedCount);
});

Delete

MySQL

const sql = 'DELETE FROM users WHERE name = ?';
const name = 'John Doe';

connection.query(sql, [name], (err, result) => {
  if (err) throw err;
  console.log('Record deleted, affected rows: ' + result.affectedRows);
});

MongoDB

const collection = db.collection('users');
collection.deleteOne({ name: 'John Doe' }, (err, result) => {
  if (err) throw err;
  console.log('Record deleted, deleted count: ' + result.deletedCount);
});

Conclusion

In this section, we covered the basics of databases, including the differences between SQL and NoSQL databases. We also demonstrated how to set up and connect to both types of databases using Node.js, and how to perform CRUD operations. Understanding these fundamentals is crucial for building robust and scalable web applications.
