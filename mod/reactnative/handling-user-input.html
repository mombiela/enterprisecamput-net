Introduction

Handling user input is a fundamental aspect of building interactive applications. In React Native, the TextInput component is used to capture text input from the user. This section will guide you through the basics of using TextInput, handling its state, and implementing advanced features.

Basic Usage of TextInput

To get started with TextInput, you need to import it from the react-native library and include it in your component.

Example

import React, { useState } from 'react';
import { View, TextInput, Text, StyleSheet } from 'react-native';

const BasicTextInput = () => {
  const [text, setText] = useState('');

  return (
    <View style={styles.container}>
      <TextInput
        style={styles.input}
        placeholder="Type here..."
        onChangeText={setText}
        value={text}
      />
      <Text style={styles.text}>You typed: {text}</Text>
    </View>
  );
};

const styles = StyleSheet.create({
  container: {
    padding: 20,
  },
  input: {
    height: 40,
    borderColor: 'gray',
    borderWidth: 1,
    marginBottom: 10,
    paddingHorizontal: 10,
  },
  text: {
    fontSize: 18,
  },
});

export default BasicTextInput;

Explanation

  • Importing Components: TextInput, View, Text, and StyleSheet are imported from react-native.
  • State Management: The useState hook is used to manage the text input state.
  • TextInput Component:
    • style: Applies styles to the TextInput.
    • placeholder: Displays a placeholder text when the input is empty.
    • onChangeText: Updates the state whenever the text changes.
    • value: Binds the state to the TextInput value.
  • Text Component: Displays the text typed by the user.

Handling Different Input Types

TextInput can handle various types of input by setting the keyboardType property.

Example

<TextInput
  style={styles.input}
  placeholder="Enter your email"
  keyboardType="email-address"
  onChangeText={setEmail}
  value={email}
/>

Common `keyboardType` Values

| Value | Description | |----------------|--------------------------------------| | default | Default keyboard | | email-address| Keyboard optimized for email input | | numeric | Numeric keyboard | | phone-pad | Keyboard optimized for phone numbers |

Managing Focus and Blur Events

You can handle focus and blur events to enhance user experience.

Example

<TextInput
  style={styles.input}
  placeholder="Focus to see effect"
  onFocus={() => console.log('Input focused')}
  onBlur={() => console.log('Input blurred')}
/>

Explanation

  • onFocus: Triggered when the TextInput gains focus.
  • onBlur: Triggered when the TextInput loses focus.

Advanced Features

Controlling TextInput Programmatically

You can use ref to control the TextInput programmatically.

Example

import React, { useRef } from 'react';
import { View, TextInput, Button, StyleSheet } from 'react-native';

const ControlledTextInput = () => {
  const inputRef = useRef(null);

  const focusInput = () => {
    inputRef.current.focus();
  };

  return (
    <View style={styles.container}>
      <TextInput
        ref={inputRef}
        style={styles.input}
        placeholder="Controlled input"
      />
      <Button title="Focus the input" onPress={focusInput} />
    </View>
  );
};

const styles = StyleSheet.create({
  container: {
    padding: 20,
  },
  input: {
    height: 40,
    borderColor: 'gray',
    borderWidth: 1,
    marginBottom: 10,
    paddingHorizontal: 10,
  },
});

export default ControlledTextInput;

Explanation

  • useRef Hook: Used to create a reference to the TextInput.
  • focus Method: Called to programmatically focus the TextInput.

Conclusion

In this section, you learned how to handle user input using the TextInput component in React Native. We covered basic usage, handling different input types, managing focus and blur events, and controlling TextInput programmatically. Mastering these concepts will help you build more interactive and user-friendly applications.
