Introduction

In this course, we will build a To-Do App using React Native. This project will help you understand the fundamentals of React Native and how to apply them to create a functional mobile application. We will start from the basics and gradually move to more advanced topics.

Setting Up the Environment

Before we start coding, we need to set up our development environment.

  • Install Node.js: Ensure you have Node.js installed. You can download it from nodejs.org.
  • Install Expo CLI: Expo is a framework and platform for universal React applications. Install it using npm: 
    npm install -g expo-cli
  • Initialize a New Project: Create a new React Native project using Expo CLI: 
    expo init ToDoApp
cd ToDoApp
expo start

Basic Structure of a React Native App

Understanding the basic structure of a React Native app is crucial.

  • App.js: The main entry point of the application.
  • Components: Reusable pieces of UI.
  • State Management: Managing the state of the application.

Example: Basic App Structure

import React from 'react';
import { StyleSheet, Text, View } from 'react-native';

export default function App() {
  return (
    <View style={styles.container}>
      <Text>Welcome to the To-Do App!</Text>
    </View>
  );
}

const styles = StyleSheet.create({
  container: {
    flex: 1,
    backgroundColor: '#fff',
    alignItems: 'center',
    justifyContent: 'center',
  },
});

Creating the To-Do List Component

We will create a To-Do List component that will display our tasks.

  • State Management: Use React's useState hook to manage the list of tasks.
  • Rendering List: Use FlatList to render the list of tasks.

Example: To-Do List Component

import React, { useState } from 'react';
import { View, Text, FlatList, StyleSheet } from 'react-native';

const ToDoList = () => {
  const [tasks, setTasks] = useState([
    { key: '1', task: 'Buy groceries' },
    { key: '2', task: 'Walk the dog' },
    { key: '3', task: 'Read a book' },
  ]);

  return (
    <View style={styles.container}>
      <FlatList
        data={tasks}
        renderItem={({ item }) => <Text style={styles.item}>{item.task}</Text>}
      />
    </View>
  );
};

const styles = StyleSheet.create({
  container: {
    flex: 1,
    paddingTop: 22,
  },
  item: {
    padding: 10,
    fontSize: 18,
    height: 44,
  },
});

export default ToDoList;

Adding New Tasks

We need a way to add new tasks to our list.

  • TextInput: Use TextInput to capture user input.
  • Button: Use Button to add the task to the list.

Example: Adding New Tasks

import React, { useState } from 'react';
import { View, TextInput, Button, FlatList, Text, StyleSheet } from 'react-native';

const ToDoApp = () => {
  const [tasks, setTasks] = useState([]);
  const [text, setText] = useState('');

  const addTask = () => {
    setTasks([...tasks, { key: Math.random().toString(), task: text }]);
    setText('');
  };

  return (
    <View style={styles.container}>
      <TextInput
        style={styles.input}
        placeholder="Add new task"
        value={text}
        onChangeText={setText}
      />
      <Button title="Add Task" onPress={addTask} />
      <FlatList
        data={tasks}
        renderItem={({ item }) => <Text style={styles.item}>{item.task}</Text>}
      />
    </View>
  );
};

const styles = StyleSheet.create({
  container: {
    flex: 1,
    paddingTop: 22,
  },
  input: {
    height: 40,
    borderColor: 'gray',
    borderWidth: 1,
    marginBottom: 10,
    paddingLeft: 8,
  },
  item: {
    padding: 10,
    fontSize: 18,
    height: 44,
  },
});

export default ToDoApp;

Deleting Tasks

To make our app more functional, we need to add the ability to delete tasks.

  • TouchableOpacity: Use TouchableOpacity to make each task item clickable.
  • Filter Function: Use the filter function to remove the task from the list.

Example: Deleting Tasks

import React, { useState } from 'react';
import { View, TextInput, Button, FlatList, Text, TouchableOpacity, StyleSheet } from 'react-native';

const ToDoApp = () => {
  const [tasks, setTasks] = useState([]);
  const [text, setText] = useState('');

  const addTask = () => {
    setTasks([...tasks, { key: Math.random().toString(), task: text }]);
    setText('');
  };

  const deleteTask = (key) => {
    setTasks(tasks.filter(task => task.key !== key));
  };

  return (
    <View style={styles.container}>
      <TextInput
        style={styles.input}
        placeholder="Add new task"
        value={text}
        onChangeText={setText}
      />
      <Button title="Add Task" onPress={addTask} />
      <FlatList
        data={tasks}
        renderItem={({ item }) => (
          <TouchableOpacity onPress={() => deleteTask(item.key)}>
            <Text style={styles.item}>{item.task}</Text>
          </TouchableOpacity>
        )}
      />
    </View>
  );
};

const styles = StyleSheet.create({
  container: {
    flex: 1,
    paddingTop: 22,
  },
  input: {
    height: 40,
    borderColor: 'gray',
    borderWidth: 1,
    marginBottom: 10,
    paddingLeft: 8,
  },
  item: {
    padding: 10,
    fontSize: 18,
    height: 44,
  },
});

export default ToDoApp;
