Introduction

In this section, we will explore the fundamentals of image classification using PyTorch. This project will guide you from setting up your environment to building and training a deep learning model to classify images. By the end of this project, you will have a solid understanding of how to implement image classification models using PyTorch.

Setting Up the Environment

Before we start coding, we need to set up our development environment.

Installing PyTorch

To install PyTorch, you can use pip. Open your terminal and run the following command:

pip install torch torchvision

Importing Necessary Libraries

We will need several libraries for this project. Let's import them:

import torch
import torch.nn as nn
import torch.optim as optim
import torchvision
import torchvision.transforms as transforms

Data Preparation

Data preparation is a crucial step in any machine learning project. We will use the CIFAR-10 dataset for this project.

Loading the CIFAR-10 Dataset

PyTorch provides a convenient way to load the CIFAR-10 dataset using torchvision.datasets.

transform = transforms.Compose(
    [transforms.ToTensor(),
     transforms.Normalize((0.5, 0.5, 0.5), (0.5, 0.5, 0.5))])

trainset = torchvision.datasets.CIFAR10(root='./data', train=True,
                                        download=True, transform=transform)
trainloader = torch.utils.data.DataLoader(trainset, batch_size=4,
                                          shuffle=True, num_workers=2)

testset = torchvision.datasets.CIFAR10(root='./data', train=False,
                                       download=True, transform=transform)
testloader = torch.utils.data.DataLoader(testset, batch_size=4,
                                         shuffle=False, num_workers=2)

classes = ('plane', 'car', 'bird', 'cat',
           'deer', 'dog', 'frog', 'horse', 'ship', 'truck')

Building the Model

We will build a Convolutional Neural Network (CNN) for our image classification task.

Defining the CNN

Here is a simple CNN model:

class Net(nn.Module):
    def __init__(self):
        super(Net, self).__init__()
        self.conv1 = nn.Conv2d(3, 6, 5)
        self.pool = nn.MaxPool2d(2, 2)
        self.conv2 = nn.Conv2d(6, 16, 5)
        self.fc1 = nn.Linear(16 * 5 * 5, 120)
        self.fc2 = nn.Linear(120, 84)
        self.fc3 = nn.Linear(84, 10)

        x = self.pool(F.relu(self.conv2(x)))
        x = x.view(-1, 16 * 5 * 5)
        x = F.relu(self.fc1(x))
        x = F.relu(self.fc2(x))
        x = self.fc3(x)
        return x

net = Net()

Training the Model

Now that we have our model, we need to train it.

Defining the Loss Function and Optimizer

We will use Cross-Entropy Loss and Stochastic Gradient Descent (SGD) for this task.

criterion = nn.CrossEntropyLoss()
optimizer = optim.SGD(net.parameters(), lr=0.001, momentum=0.9)

Training Loop

Here is the code for training the model:

for epoch in range(2):  # loop over the dataset multiple times

    running_loss = 0.0
    for i, data in enumerate(trainloader, 0):
        # get the inputs; data is a list of [inputs, labels]
        inputs, labels = data

        # zero the parameter gradients
        optimizer.zero_grad()

        # forward + backward + optimize
        outputs = net(inputs)
        loss = criterion(outputs, labels)
        loss.backward()
        optimizer.step()

        # print statistics
        running_loss += loss.item()
        if i % 2000 == 1999:    # print every 2000 mini-batches
            print(f'[{epoch + 1}, {i + 1}] loss: {running_loss / 2000:.3f}')
            running_loss = 0.0

print('Finished Training')

Evaluating the Model

After training, we need to evaluate the model's performance on the test data.

Testing the Model

Here is the code to test the model:

correct = 0
total = 0
with torch.no_grad():
    for data in testloader:
        images, labels = data
        outputs = net(images)
        _, predicted = torch.max(outputs.data, 1)
        total += labels.size(0)
        correct += (predicted == labels).sum().item()

print(f'Accuracy of the network on the 10000 test images: {100 * correct / total}%')

Class-wise Performance

We can also check the performance of the model for each class.

class_correct = list(0. for i in range(10))
class_total = list(0. for i in range(10))
with torch.no_grad():
    for data in testloader:
        images, labels = data
        outputs = net(images)
        _, predicted = torch.max(outputs, 1)
        c = (predicted == labels).squeeze()
        for i in range(4):
            label = labels[i]
            class_correct[label] += c[i].item()
            class_total[label] += 1

for i in range(10):
    print(f'Accuracy of {classes[i]} : {100 * class_correct[i] / class_total[i]}%')

Conclusion

In this project, we covered the basics of image classification using PyTorch. We started by setting up the environment, loading and preparing the CIFAR-10 dataset, building a CNN model, training the model, and finally evaluating its performance. This project provides a solid foundation for more advanced image classification tasks and can be extended further with more complex models and techniques.
