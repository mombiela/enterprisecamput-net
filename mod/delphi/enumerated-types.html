Enumerated Types in Delphi/Object Pascal
Strings and String Handling ►

Enumerated types, or enums, are a powerful feature in Delphi/Object Pascal that allow you to define a set of named values. These values are constants and can be used to make your code more readable and maintainable. This topic will guide you from the basics to advanced usage of enumerated types in Delphi/Object Pascal.

Introduction to Enumerated Types

Enumerated types are user-defined data types that consist of a set of named values called elements. Each element in an enumerated type is associated with an integer value, starting from 0 by default.

Defining Enumerated Types

To define an enumerated type, you use the type keyword followed by the name of the type and the list of possible values enclosed in parentheses.

type
  TColor = (Red, Green, Blue);

In this example, TColor is an enumerated type with three possible values: Red, Green, and Blue.

Using Enumerated Types

Once you have defined an enumerated type, you can declare variables of that type and assign them any of the values defined in the enumeration.

var
  MyColor: TColor;
begin
  MyColor := Red;
  if MyColor = Green then
    WriteLn('The color is green.')
  else
    WriteLn('The color is not green.');
end;

Advanced Enumerated Types

Assigning Specific Values

By default, the first element in an enumerated type is assigned the value 0, the second element the value 1, and so on. However, you can assign specific integer values to the elements.

type
  TStatus = (Pending = 1, Approved = 5, Rejected = 10);

In this example, Pending is assigned the value 1, Approved the value 5, and Rejected the value 10.

Enumerated Type Ranges

You can use enumerated types to define ranges, which can be useful in loops and array indexing.

type
  TDay = (Mon, Tue, Wed, Thu, Fri, Sat, Sun);

var
  Day: TDay;
begin
  for Day := Mon to Fri do
    WriteLn(Day);
end;

Enumerated Types and Sets

Enumerated types can be used to define sets, which are collections of values of the enumerated type.

type
  TDays = set of TDay;

var
  Weekend: TDays;
begin
  Weekend := [Sat, Sun];
  if Sun in Weekend then
    WriteLn('Sunday is a weekend day.');
end;

Enumerated Type Functions

Delphi provides several built-in functions to work with enumerated types, such as Ord, Pred, Succ, and High.

  • Ord: Returns the ordinal value of an enumerated type element.
  • Pred: Returns the predecessor of an enumerated type element.
  • Succ: Returns the successor of an enumerated type element.
  • High: Returns the highest value in an enumerated type.
begin
  WriteLn(Ord(Mon));  // Output: 0
  WriteLn(Pred(Wed)); // Output: Tue
  WriteLn(Succ(Tue)); // Output: Wed
  WriteLn(High(TDay)); // Output: Sun
end;

Conclusion

Enumerated types in Delphi/Object Pascal provide a way to define a set of named constants, making your code more readable and maintainable. You can assign specific values to elements, use them in ranges and sets, and leverage built-in functions to manipulate them. Understanding and utilizing enumerated types effectively can greatly enhance the clarity and functionality of your Delphi/Object Pascal programs.
