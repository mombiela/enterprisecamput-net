Dynamic Data Structures: Pointers and Linked Lists
Introduction to OOP ►

Introduction

Dynamic data structures are essential for efficient memory management and flexible data manipulation. In this topic, we will explore pointers and linked lists in Delphi/Object Pascal, starting from the basics and progressing to more advanced concepts.

Pointers in Delphi/Object Pascal

What is a Pointer?

  • A pointer is a variable that holds the memory address of another variable.
  • Pointers allow for dynamic memory allocation and manipulation of data structures like linked lists.

Declaring and Using Pointers

var
  p: ^Integer; // Declares a pointer to an integer
  num: Integer;
begin
  num := 10;
  p := @num; // p now holds the address of num
  WriteLn('Value of num: ', num); // Outputs 10
  WriteLn('Value pointed to by p: ', p^); // Outputs 10
end;
  • ^ is used to declare a pointer type.
  • @ operator is used to get the address of a variable.
  • p^ dereferences the pointer to access the value stored at the address.

Dynamic Memory Allocation

  • Use New and Dispose for dynamic memory allocation and deallocation.
var
  p: ^Integer;
begin
  New(p); // Allocates memory for an integer
  p^ := 20;
  WriteLn('Value pointed to by p: ', p^); // Outputs 20
  Dispose(p); // Deallocates the memory
end;

Linked Lists

What is a Linked List?

  • A linked list is a dynamic data structure consisting of nodes.
  • Each node contains data and a pointer to the next node.

Types of Linked Lists

  • Singly Linked List: Each node points to the next node.
  • Doubly Linked List: Each node points to both the next and the previous node.

Implementing a Singly Linked List

Node Structure

type
  PNode = ^TNode;
  TNode = record
    Data: Integer;
    Next: PNode;
  end;
  • PNode is a pointer to TNode.
  • TNode contains an integer data field and a pointer to the next node.

Creating and Manipulating a Linked List

var
  Head, Temp: PNode;

procedure AddNode(var Head: PNode; Value: Integer);
var
  NewNode: PNode;
begin
  New(NewNode);
  NewNode^.Data := Value;
  NewNode^.Next := Head;
  Head := NewNode;
end;

procedure PrintList(Head: PNode);
begin
  Temp := Head;
  while Temp <> nil do
  begin
    WriteLn(Temp^.Data);
    Temp := Temp^.Next;
  end;
end;

begin
  Head := nil;
  AddNode(Head, 10);
  AddNode(Head, 20);
  AddNode(Head, 30);
  PrintList(Head);
end;
  • AddNode procedure adds a new node to the beginning of the list.
  • PrintList procedure prints all the nodes in the list.

Implementing a Doubly Linked List

Node Structure

type
  PDNode = ^TDNode;
  TDNode = record
    Data: Integer;
    Prev, Next: PDNode;
  end;
  • PDNode is a pointer to TDNode.
  • TDNode contains an integer data field, a pointer to the previous node, and a pointer to the next node.

Creating and Manipulating a Doubly Linked List

var
  Head, Tail, Temp: PDNode;

procedure AddNode(var Head, Tail: PDNode; Value: Integer);
var
  NewNode: PDNode;
begin
  New(NewNode);
  NewNode^.Data := Value;
  NewNode^.Next := nil;
  NewNode^.Prev := Tail;
  if Tail <> nil then
    Tail^.Next := NewNode;
  Tail := NewNode;
  if Head = nil then
    Head := NewNode;
end;

procedure PrintListForward(Head: PDNode);
begin
  Temp := Head;
  while Temp <> nil do
  begin
    WriteLn(Temp^.Data);
    Temp := Temp^.Next;
  end;
end;

procedure PrintListBackward(Tail: PDNode);
begin
  Temp := Tail;
  while Temp <> nil do
  begin
    WriteLn(Temp^.Data);
    Temp := Temp^.Prev;
  end;
end;

begin
  Head := nil;
  Tail := nil;
  AddNode(Head, Tail, 10);
  AddNode(Head, Tail, 20);
  AddNode(Head, Tail, 30);
  PrintListForward(Head);
  PrintListBackward(Tail);
end;
  • AddNode procedure adds a new node to the end of the list.
  • PrintListForward procedure prints all the nodes from head to tail.
  • PrintListBackward procedure prints all the nodes from tail to head.

Conclusion

Pointers and linked lists are powerful tools in Delphi/Object Pascal for managing dynamic data structures. Understanding how to use pointers for dynamic memory allocation and how to implement and manipulate linked lists is crucial for advanced programming. Practice these concepts with various exercises to solidify your understanding and improve your programming skills.
