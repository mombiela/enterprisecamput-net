Introduction to Sets

Sets in Delphi/Object Pascal are collections of elements of the same type, typically used to represent a group of related values. They are particularly useful for handling a fixed range of values efficiently.

Key Concepts

  • Definition: A set is a collection of values of the same type.
  • Element Type: The type of elements in a set must be an ordinal type (e.g., integers, characters, enumerations).
  • Range: Sets can only contain values within a specific range.

Declaring Sets

To declare a set in Delphi/Object Pascal, you use the set of keyword followed by the type of elements the set will contain.

Example

type
  TDays = (Mon, Tue, Wed, Thu, Fri, Sat, Sun);
  TWorkDays = set of TDays;

In this example:

  • TDays is an enumeration representing the days of the week.
  • TWorkDays is a set that can contain any combination of TDays values.

Working with Sets

Creating and Initializing Sets

You can create and initialize sets using set constructors, which are enclosed in square brackets.

Example

var
  WorkDays: TWorkDays;
begin
  WorkDays := [Mon, Tue, Wed, Thu, Fri];
end;

In this example, WorkDays is initialized to contain Monday through Friday.

Adding and Removing Elements

You can add or remove elements from a set using the + and - operators.

Example

begin
  WorkDays := WorkDays + [Sat];  // Add Saturday
  WorkDays := WorkDays - [Mon];  // Remove Monday
end;

Checking Membership

To check if an element is part of a set, use the in operator.

Example

begin
  if Mon in WorkDays then
    WriteLn('Monday is a workday.')
  else
    WriteLn('Monday is not a workday.');
end;

Set Operations

Delphi/Object Pascal supports several operations on sets, including union, intersection, and difference.

Union

The union of two sets contains all elements that are in either set.

Example

var
  Weekend: TWorkDays;
begin
  Weekend := [Sat, Sun];
  WorkDays := WorkDays + Weekend;  // Union of WorkDays and Weekend
end;

Intersection

The intersection of two sets contains only the elements that are in both sets.

Example

var
  CommonDays: TWorkDays;
begin
  CommonDays := WorkDays * [Mon, Tue, Wed];  // Intersection of WorkDays and [Mon, Tue, Wed]
end;

Difference

The difference of two sets contains the elements that are in the first set but not in the second.

Example

begin
  WorkDays := WorkDays - [Fri];  // Difference of WorkDays and [Fri]
end;

Advanced Set Features

Subrange Types

Sets can be defined over subranges of ordinal types to limit their range.

Example

type
  TSmallNumbers = 1..10;
  TSmallNumberSet = set of TSmallNumbers;

Set of Characters

Sets can also be used with characters, which is useful for tasks like checking for valid input.

Example

var
  Vowels: set of Char;
begin
  Vowels := ['A', 'E', 'I', 'O', 'U', 'a', 'e', 'i', 'o', 'u'];
  if 'A' in Vowels then
    WriteLn('A is a vowel.');
end;

Conclusion

Sets in Delphi/Object Pascal provide a powerful way to handle collections of ordinal values efficiently. They support various operations like union, intersection, and difference, making them versatile for many programming tasks. Understanding how to declare, initialize, and manipulate sets is essential for effective Delphi/Object Pascal programming.
