Introduction

The "Hello, World!" program is a simple yet fundamental program that is often used to introduce a new programming language. In this section, we will cover how to write and run a "Hello, World!" program in Rust, explaining each part of the code in detail.

Writing the "Hello, World!" Program

Setting Up Your Environment

Before we write our first Rust program, ensure you have Rust installed on your machine. You can install Rust by following the instructions on the official Rust website.

Creating a New Project

Rust uses a build system and package manager called Cargo. To create a new project, open your terminal and run:

cargo new hello_world
cd hello_world

This will create a new directory named hello_world with the following structure:

hello_world
├── Cargo.toml
└── src
    └── main.rs

Writing the Code

Open the src/main.rs file in your favorite text editor and replace its contents with the following code:

fn main() {
    println!("Hello, world!");
}

Explanation

  • fn main() { ... }: This defines the main function, which is the entry point of a Rust program.
  • println!("Hello, world!");: This line prints "Hello, world!" to the console. println! is a macro in Rust used for printing text.

Running the Program

To run your program, use the following command in your terminal:

cargo run

You should see the output:

Hello, world!

Key Concepts

  • Functions: In Rust, functions are defined using the fn keyword. The main function is special because it is the entry point of the program.
  • Macros: The println! macro is used to print text to the console. Macros in Rust are invoked with a ! and can perform code generation at compile time.
  • Cargo: Cargo is Rust's build system and package manager, which simplifies managing dependencies and building projects.

Conclusion

In this section, we covered how to write and run a simple "Hello, World!" program in Rust. We introduced the basic structure of a Rust program, including the main function and the use of the println! macro. Understanding these basics is crucial as they form the foundation for more complex Rust programs.
