Introduction to Slices

Slices are a view into a contiguous sequence of elements in a collection rather than the collection itself. They are a reference to a portion of an array or a vector, allowing you to work with a subset of the collection without copying data.

Key Concepts

  • Slices: A reference to a contiguous sequence of elements in a collection.
  • Array Slices: Slices that reference a portion of an array.
  • Vector Slices: Slices that reference a portion of a vector.
  • Immutability: Slices are immutable by default, but mutable slices can be created.

Creating and Using Slices

Array Slices

To create a slice from an array, you use a range within square brackets.

fn main() {
    let array = [1, 2, 3, 4, 5];
    let slice = &array[1..4]; // Slicing from index 1 to 3 (4 is exclusive)
    
    println!("Slice: {:?}", slice); // Output: Slice: [2, 3, 4]
}

Vector Slices

Similarly, you can create a slice from a vector.

fn main() {
    let vector = vec![10, 20, 30, 40, 50];
    let slice = &vector[2..]; // Slicing from index 2 to the end
    
    println!("Slice: {:?}", slice); // Output: Slice: [30, 40, 50]
}

Mutable Slices

Mutable slices allow you to modify the elements of the slice.

fn main() {
    let mut array = [1, 2, 3, 4, 5];
    let slice = &mut array[1..4];
    
    slice[0] = 10; // Modifying the first element of the slice
    println!("Modified array: {:?}", array); // Output: Modified array: [1, 10, 3, 4, 5]
}

Slices vs. Arrays and Vectors

| Feature | Array | Vector | Slice | |---------------|----------------|----------------|----------------| | Fixed Size | Yes | No | No | | Dynamic Size | No | Yes | No | | Ownership | Yes | Yes | No | | Mutability | Yes | Yes | Yes (if mutable slice) | | Syntax | [T; N] | Vec<T> | &[T] or &mut [T] |

Common Operations with Slices

Iterating Over Slices

You can iterate over slices using a for loop.

fn main() {
    let array = [1, 2, 3, 4, 5];
    let slice = &array[..];
    
    for element in slice {
        println!("{}", element);
    }
}

Slicing Strings

Slices can also be used with strings, but be careful with UTF-8 character boundaries.

fn main() {
    let s = String::from("hello");
    let slice = &s[0..2]; // Slicing the first two bytes
    
    println!("Slice: {}", slice); // Output: Slice: he
}

Conclusion

Slices in Rust provide a powerful way to reference and work with a portion of a collection without copying data. They are versatile, allowing for both immutable and mutable references, and can be used with arrays, vectors, and strings. Understanding slices is crucial for efficient data manipulation and memory management in Rust.
