Introduction

Arithmetic operators are fundamental in any programming language, including Ruby. They allow you to perform basic mathematical operations such as addition, subtraction, multiplication, division, and more. Understanding these operators is crucial for manipulating numerical data and performing calculations in your programs.

Basic Arithmetic Operators

Ruby supports the following basic arithmetic operators:

  • Addition (+)
  • Subtraction (-)
  • Multiplication (*)
  • Division (/)
  • Modulus (%)
  • Exponentiation (**)

Addition (`+`)

The addition operator adds two numbers together.

a = 5
b = 3
sum = a + b
puts sum  # Output: 8

Subtraction (`-`)

The subtraction operator subtracts the second number from the first.

a = 5
b = 3
difference = a - b
puts difference  # Output: 2

Multiplication (`*`)

The multiplication operator multiplies two numbers.

a = 5
b = 3
product = a * b
puts product  # Output: 15

Division (`/`)

The division operator divides the first number by the second. Note that in Ruby, dividing two integers will result in an integer.

a = 5
b = 3
quotient = a / b
puts quotient  # Output: 1

To get a floating-point result, at least one of the numbers must be a float.

a = 5.0
b = 3
quotient = a / b
puts quotient  # Output: 1.6666666666666667

Modulus (`%`)

The modulus operator returns the remainder of the division of the first number by the second.

a = 5
b = 3
remainder = a % b
puts remainder  # Output: 2

Exponentiation (`**`)

The exponentiation operator raises the first number to the power of the second number.

a = 5
b = 3
power = a ** b
puts power  # Output: 125

Advanced Usage

Combining Operators

You can combine arithmetic operators to perform more complex calculations.

a = 5
b = 3
c = 2
result = (a + b) * c - (a / b)
puts result  # Output: 15

Operator Precedence

Ruby follows standard mathematical precedence rules (PEMDAS/BODMAS). Parentheses can be used to explicitly define the order of operations.

a = 5
b = 3
c = 2
result = a + b * c  # Multiplication before addition
puts result  # Output: 11

result = (a + b) * c  # Parentheses change the order
puts result  # Output: 16

Using Arithmetic Operators with Variables

Arithmetic operators can be used directly with variables to update their values.

a = 5
a += 3  # Equivalent to a = a + 3
puts a  # Output: 8

a -= 2  # Equivalent to a = a - 2
puts a  # Output: 6

a *= 2  # Equivalent to a = a * 2
puts a  # Output: 12

a /= 4  # Equivalent to a = a / 4
puts a  # Output: 3

a %= 2  # Equivalent to a = a % 2
puts a  # Output: 1

a **= 3  # Equivalent to a = a ** 3
puts a  # Output: 1

Conclusion

Arithmetic operators are essential tools in Ruby for performing mathematical operations. From basic addition and subtraction to more complex expressions involving multiple operators, understanding how to use these operators effectively is key to writing efficient and accurate Ruby code. Practice using these operators in different scenarios to become more comfortable with their behavior and precedence.
