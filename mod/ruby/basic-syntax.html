Introduction

Ruby is a dynamic, open-source programming language with a focus on simplicity and productivity. Its elegant syntax is natural to read and easy to write. In this section, we will cover the basic syntax of Ruby, which is essential for writing and understanding Ruby code.

Comments

  • Single-line comments start with #.
  • Multi-line comments are enclosed within =begin and =end.
# This is a single-line comment

=begin
This is a
multi-line comment
=end

Variables

  • Variables in Ruby are dynamically typed.
  • Variable names should start with a lowercase letter or an underscore.
name = "Alice"
age = 30
is_student = true

Data Types

Ruby supports several basic data types:

  • Numbers (Integer, Float)
  • Strings
  • Symbols
  • Arrays
  • Hashes
  • Booleans
# Integer
num = 42

# Float
pi = 3.14

# String
greeting = "Hello, World!"

# Symbol
status = :active

# Array
numbers = [1, 2, 3, 4, 5]

# Hash
person = { name: "Alice", age: 30 }

# Boolean
is_ruby_fun = true

Operators

Arithmetic Operators

  • + Addition
  • - Subtraction
  • * Multiplication
  • / Division
  • % Modulus
sum = 5 + 3       # 8
difference = 5 - 3 # 2
product = 5 * 3    # 15
quotient = 5 / 3   # 1
remainder = 5 % 3  # 2

Comparison Operators

  • == Equal to
  • != Not equal to
  • > Greater than
  • < Less than
  • >= Greater than or equal to
  • <= Less than or equal to
a = 5
b = 3

puts a == b  # false
puts a != b  # true
puts a > b   # true
puts a < b   # false
puts a >= b  # true
puts a <= b  # false

Logical Operators

  • && Logical AND
  • || Logical OR
  • ! Logical NOT
x = true
y = false

puts x && y  # false
puts x || y  # true
puts !x      # false

Control Structures

Conditional Statements

  • if, elsif, else
age = 18

if age < 18
  puts "You are a minor."
elsif age == 18
  puts "You just became an adult."
else
  puts "You are an adult."
end

Loops

  • while loop
  • for loop
  • each method for iterating over collections
# while loop
i = 0
while i < 5
  puts i
  i += 1
end

# for loop
for i in 0..4
  puts i
end

# each method
[0, 1, 2, 3, 4].each do |i|
  puts i
end

Methods

  • Define methods using the def keyword.
  • Methods can take arguments and return values.
def greet(name)
  return "Hello, #{name}!"
end

puts greet("Alice")  # Output: Hello, Alice!

Conclusion

In this section, we covered the basic syntax of Ruby, including comments, variables, data types, operators, control structures, and methods. Understanding these fundamentals is crucial for writing and comprehending Ruby code. As you progress through the course, these basics will serve as the foundation for more advanced topics.
