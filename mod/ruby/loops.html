Introduction to Loops

Loops are fundamental in programming as they allow you to execute a block of code multiple times. In Ruby, there are several types of loops, each with its own use cases and syntax.

Types of Loops

`while` Loop

The while loop continues to execute as long as its condition is true.

i = 0
while i < 5 do
  puts "i is #{i}"
  i += 1
end
  • Explanation:
    • i = 0: Initialize the variable i.
    • while i < 5 do: The loop will run as long as i is less than 5.
    • puts "i is #{i}": Print the current value of i.
    • i += 1: Increment i by 1.

`until` Loop

The until loop is the opposite of the while loop. It continues to execute until its condition becomes true.

i = 0
until i == 5 do
  puts "i is #{i}"
  i += 1
end
  • Explanation:
    • i = 0: Initialize the variable i.
    • until i == 5 do: The loop will run until i equals 5.
    • puts "i is #{i}": Print the current value of i.
    • i += 1: Increment i by 1.

`for` Loop

The for loop iterates over a range or an array.

for i in 0..4 do
  puts "i is #{i}"
end
  • Explanation:
    • for i in 0..4 do: The loop will iterate over the range from 0 to 4.
    • puts "i is #{i}": Print the current value of i.

`times` Loop

The times loop executes a block of code a specified number of times.

5.times do |i|
  puts "i is #{i}"
end
  • Explanation:
    • 5.times do |i|: The loop will run 5 times, with i taking values from 0 to 4.
    • puts "i is #{i}": Print the current value of i.

`each` Loop

The each loop iterates over elements of an array or a hash.

[1, 2, 3, 4, 5].each do |i|
  puts "i is #{i}"
end
  • Explanation:
    • [1, 2, 3, 4, 5].each do |i|: The loop will iterate over each element of the array.
    • puts "i is #{i}": Print the current value of i.

Loop Control: `break`, `next`, and `redo`

  • break: Exits the loop immediately.
  • next: Skips to the next iteration.
  • redo: Repeats the current iteration.
for i in 0..5 do
  if i == 3
    next
  elsif i == 4
    break
  end
  puts "i is #{i}"
end
  • Explanation:
    • if i == 3: When i is 3, next skips to the next iteration.
    • elsif i == 4: When i is 4, break exits the loop.
    • puts "i is #{i}": Print the current value of i.

Advanced Loop Techniques

Nested Loops

Loops can be nested within each other to handle more complex scenarios.

for i in 1..3 do
  for j in 1..3 do
    puts "i is #{i}, j is #{j}"
  end
end
  • Explanation:
    • The outer loop iterates over i from 1 to 3.
    • The inner loop iterates over j from 1 to 3.
    • puts "i is #{i}, j is #{j}": Print the current values of i and j.

Looping with Enumerators

Enumerators provide a way to create custom iteration patterns.

enum = (1..5).each
loop do
  puts enum.next
end
  • Explanation:
    • enum = (1..5).each: Create an enumerator for the range 1 to 5.
    • loop do: Start an infinite loop.
    • puts enum.next: Print the next element from the enumerator.

Conclusion

Loops are a powerful tool in Ruby, allowing you to execute code multiple times efficiently. Understanding the different types of loops and their use cases will enable you to write more effective and concise code. Practice using these loops in various scenarios to become proficient in their application.
