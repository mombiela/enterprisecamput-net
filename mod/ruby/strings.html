Introduction to Strings

Strings are sequences of characters used to represent text in Ruby. They are one of the most commonly used data types and are essential for handling textual data.

Creating Strings

  • Single Quotes: Strings can be created using single quotes (').
  • Double Quotes: Strings can also be created using double quotes (").
single_quoted_string = 'Hello, World!'
double_quoted_string = "Hello, World!"

Differences Between Single and Double Quotes

  • Interpolation: Double-quoted strings support string interpolation, while single-quoted strings do not.
  • Escape Sequences: Double-quoted strings support more escape sequences than single-quoted strings.
name = "Alice"
single_quoted = 'Hello, #{name}'  # Outputs: Hello, #{name}
double_quoted = "Hello, #{name}"  # Outputs: Hello, Alice

single_quoted_escape = 'It\'s a sunny day.'  # Outputs: It's a sunny day.
double_quoted_escape = "Line1\nLine2"       # Outputs: Line1
                                            #          Line2

String Methods

Ruby provides a rich set of methods to manipulate strings.

Common String Methods

  • Length: Returns the number of characters in the string.
str = "Hello"
puts str.length  # Outputs: 5
  • Upcase/Downcase: Converts all characters to uppercase or lowercase.
puts str.upcase   # Outputs: HELLO
puts str.downcase # Outputs: hello
  • Substrings: Extracts a part of the string.
puts str[0, 2]  # Outputs: He
  • Concatenation: Combines strings together.
greeting = "Hello"
name = "Alice"
puts greeting + " " + name  # Outputs: Hello Alice
  • Interpolation: Embeds variables or expressions inside a string.
age = 25
puts "I am #{age} years old."  # Outputs: I am 25 years old.

Advanced String Methods

  • Split: Divides a string into an array based on a delimiter.
sentence = "Hello, World!"
words = sentence.split(", ")
puts words  # Outputs: ["Hello", "World!"]
  • Join: Combines elements of an array into a single string.
words = ["Hello", "World!"]
sentence = words.join(", ")
puts sentence  # Outputs: Hello, World!
  • Gsub: Substitutes all occurrences of a pattern with a replacement.
text = "The quick brown fox"
new_text = text.gsub("quick", "slow")
puts new_text  # Outputs: The slow brown fox

String Comparison

Strings can be compared using various operators.

Comparison Operators

  • Equality (==): Checks if two strings are identical.
puts "hello" == "hello"  # Outputs: true
puts "hello" == "world"  # Outputs: false
  • Inequality (!=): Checks if two strings are not identical.
puts "hello" != "world"  # Outputs: true
  • Lexicographical Comparison (<, >, <=, >=): Compares strings based on alphabetical order.
puts "apple" < "banana"  # Outputs: true
puts "apple" > "banana"  # Outputs: false

String Encoding

Ruby supports multiple string encodings, which is essential for handling internationalization.

Checking and Changing Encoding

  • Encoding: Returns the encoding of the string.
str = "Hello"
puts str.encoding  # Outputs: UTF-8
  • Force Encoding: Changes the encoding of the string.
str = "Hello"
str.force_encoding("ASCII-8BIT")
puts str.encoding  # Outputs: ASCII-8BIT

Conclusion

Strings are a fundamental part of Ruby programming. Understanding how to create, manipulate, and compare strings is crucial for handling textual data effectively. By mastering the various methods and techniques discussed, you can efficiently work with strings in your Ruby applications.
