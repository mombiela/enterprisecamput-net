Introduction to Ruby

Ruby is a dynamic, open-source programming language with a focus on simplicity and productivity. It has an elegant syntax that is natural to read and easy to write. Ruby was created in the mid-1990s by Yukihiro "Matz" Matsumoto in Japan.

Key Features of Ruby

  • Object-Oriented: Everything in Ruby is an object, including primitive data types.
  • Dynamic Typing: Variables in Ruby are not bound to a specific data type.
  • Garbage Collection: Ruby automatically manages memory allocation and deallocation.
  • Rich Standard Library: Ruby comes with a comprehensive standard library that makes it easy to perform a wide range of tasks.
  • Community and Ecosystem: Ruby has a vibrant community and a rich ecosystem of libraries and frameworks, such as Ruby on Rails.

Basic Syntax

Let's look at some basic Ruby syntax to get a feel for the language.

Hello World Example

puts 'Hello, World!'
  • puts is a method that outputs text to the console.
  • 'Hello, World!' is a string literal.

Variables and Data Types

name = "Alice"  # String
age = 30        # Integer
height = 5.5    # Float
is_student = true  # Boolean

puts name       # Outputs: Alice
puts age        # Outputs: 30
puts height     # Outputs: 5.5
puts is_student # Outputs: true
  • Variables are dynamically typed.
  • Common data types include String, Integer, Float, and Boolean.

Control Structures

If-Else Statement

age = 18

if age >= 18
  puts "You are an adult."
else
  puts "You are a minor."
end
  • if and else are used for conditional execution.
  • The condition is evaluated, and the corresponding block of code is executed.

Loops

While Loop
count = 0

while count < 5
  puts "Count is #{count}"
  count += 1
end
  • while loop runs as long as the condition is true.
  • #{} is used for string interpolation.
Each Loop
numbers = [1, 2, 3, 4, 5]

numbers.each do |number|
  puts number
end
  • each method iterates over elements in a collection, such as an array.
  • |number| is a block variable that takes on the value of each element in the array.

Object-Oriented Programming in Ruby

Ruby is a pure object-oriented language, meaning everything is an object. Let's create a simple class to illustrate this.

Defining a Class

class Person
  attr_accessor :name, :age

  def initialize(name, age)
    @name = name
    @age = age
  end

  def introduce
    puts "Hello, my name is #{@name} and I am #{@age} years old."
  end
end

person = Person.new("Alice", 30)
person.introduce
  • class keyword is used to define a class.
  • attr_accessor creates getter and setter methods for instance variables.
  • initialize method is the constructor.
  • @name and @age are instance variables.
  • introduce is an instance method.

Summary

Ruby is a versatile and beginner-friendly programming language known for its simplicity and productivity. It supports object-oriented programming and has a rich standard library. By understanding its basic syntax, control structures, and object-oriented features, you can start building powerful applications. As you progress, you'll discover more advanced features and libraries that make Ruby a powerful tool for developers.
