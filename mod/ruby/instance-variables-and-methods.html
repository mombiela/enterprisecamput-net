Introduction

In Ruby, instance variables and methods are fundamental concepts that help in defining the state and behavior of objects. This section will cover what instance variables and methods are, how to use them, and provide examples and exercises to solidify your understanding.

Instance Variables

What are Instance Variables?

  • Instance variables are variables that belong to an instance of a class.
  • They are used to store the state of an object.
  • Instance variables are prefixed with an @ symbol.

Defining Instance Variables

Instance variables are typically defined within instance methods. They are initialized when an object is created and can be accessed by any instance method within the same object.

class Person
  def initialize(name, age)
    @name = name
    @age = age
  end

  def display_info
    puts "Name: #{@name}, Age: #{@age}"
  end
end

person = Person.new("Alice", 30)
person.display_info  # Output: Name: Alice, Age: 30

Accessing Instance Variables

Instance variables can be accessed and modified within instance methods. However, they are not directly accessible from outside the class.

class Person
  def initialize(name, age)
    @name = name
    @age = age
  end

  def set_name(new_name)
    @name = new_name
  end

  def get_name
    @name
  end
end

person = Person.new("Alice", 30)
puts person.get_name  # Output: Alice
person.set_name("Bob")
puts person.get_name  # Output: Bob

Instance Methods

What are Instance Methods?

  • Instance methods are methods that belong to an instance of a class.
  • They define the behavior of an object.
  • Instance methods can access and modify instance variables.

Defining Instance Methods

Instance methods are defined using the def keyword within a class. They can perform operations using instance variables and other methods.

class Calculator
  def initialize
    @result = 0
  end

  def add(number)
    @result += number
  end

  def subtract(number)
    @result -= number
  end

  def result
    @result
  end
end

calc = Calculator.new
calc.add(10)
calc.subtract(3)
puts calc.result  # Output: 7

Example: A Simple Bank Account

Let's create a simple bank account class to demonstrate instance variables and methods.

class BankAccount
  def initialize(owner, balance = 0)
    @owner = owner
    @balance = balance
  end

  def deposit(amount)
    @balance += amount
    puts "Deposited #{amount}. New balance: #{@balance}"
  end

  def withdraw(amount)
    if amount > @balance
      puts "Insufficient funds!"
    else
      @balance -= amount
      puts "Withdrew #{amount}. New balance: #{@balance}"
    end
  end

  def display_balance
    puts "Account owner: #{@owner}, Balance: #{@balance}"
  end
end

account = BankAccount.new("John Doe", 100)
account.deposit(50)
account.withdraw(30)
account.display_balance

Exercises

  1. Create a Car class:

    • Define instance variables for make, model, and year.
    • Create instance methods to start the car, stop the car, and display the car's details.

  2. Enhance the Person class:

    • Add an instance method to update the age.
    • Add an instance method to display a greeting message that includes the person's name and age.

  3. Build a Library class:

    • Define instance variables for name and books (an array of book titles).
    • Create instance methods to add a book, remove a book, and list all books.

Conclusion

Instance variables and methods are crucial for defining the state and behavior of objects in Ruby. By understanding how to use them, you can create more complex and functional classes. Practice by creating your own classes and experimenting with instance variables and methods to deepen your understanding.
