Introduction to Conditional Statements

Conditional statements are fundamental in programming as they allow you to execute certain pieces of code based on specific conditions. In Ruby, conditional statements help control the flow of your program by making decisions.

Basic Conditional Statements

`if` Statement

The if statement is the most basic form of conditional statement in Ruby. It executes a block of code if a specified condition is true.

if condition
  # code to execute if condition is true
end

Example:

age = 18

if age >= 18
  puts "You are an adult."
end

`else` Statement

The else statement can be used alongside if to execute a block of code if the condition is false.

if condition
  # code to execute if condition is true
else
  # code to execute if condition is false
end

Example:

age = 16

if age >= 18
  puts "You are an adult."
else
  puts "You are a minor."
end

`elsif` Statement

The elsif statement allows you to check multiple conditions.

if condition1
  # code to execute if condition1 is true
elsif condition2
  # code to execute if condition2 is true
else
  # code to execute if none of the above conditions are true
end

Example:

age = 20

if age < 13
  puts "You are a child."
elsif age < 18
  puts "You are a teenager."
else
  puts "You are an adult."
end

Advanced Conditional Statements

`unless` Statement

The unless statement is the opposite of if. It executes a block of code if the condition is false.

unless condition
  # code to execute if condition is false
end

Example:

age = 15

unless age >= 18
  puts "You are not an adult."
end

Ternary Operator

The ternary operator is a shorthand for if-else statements. It is useful for simple conditions.

condition ? true_expression : false_expression

Example:

age = 20
message = age >= 18 ? "You are an adult." : "You are a minor."
puts message

Case Statement

The case statement is used for comparing a variable against multiple values. It is similar to the switch statement in other languages.

case variable
when value1
  # code to execute if variable == value1
when value2
  # code to execute if variable == value2
else
  # code to execute if variable does not match any value
end

Example:

day = "Saturday"

case day
when "Monday"
  puts "Start of the work week."
when "Friday"
  puts "End of the work week."
when "Saturday", "Sunday"
  puts "Weekend!"
else
  puts "Midweek days."
end

Combining Conditions

Logical Operators

Logical operators can be used to combine multiple conditions in a single if, elsif, or unless statement.

  • && (and): Both conditions must be true.
  • || (or): At least one condition must be true.
  • ! (not): Inverts the condition.

Example:

age = 25
has_id = true

if age >= 18 && has_id
  puts "You can enter the club."
else
  puts "You cannot enter the club."
end

Exercises

  1. Write a Ruby program that checks if a number is positive, negative, or zero.
  2. Create a Ruby script that categorizes a person's age group (child, teenager, adult, senior) based on their age.
  3. Implement a Ruby program that uses a case statement to print the day of the week based on a number input (1 for Monday, 2 for Tuesday, etc.).

Summary

Conditional statements in Ruby are essential for controlling the flow of your program. Starting with basic if, else, and elsif statements, you can handle simple conditions. For more complex scenarios, Ruby offers unless, ternary operators, and case statements. Combining conditions with logical operators allows for even more precise control. Practice these concepts to become proficient in managing program logic in Ruby.
