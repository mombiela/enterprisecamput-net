File Modes in Ruby
File Operations ►

Introduction

File handling is a crucial aspect of programming, allowing you to read from and write to files. In Ruby, file modes determine how a file is opened and what operations can be performed on it. This section will cover the various file modes available in Ruby, providing clear explanations, examples, and exercises to help you understand and use them effectively.

Key Concepts

  • File Modes: Different ways to open a file, determining the operations that can be performed.
  • Common File Modes: Modes like read, write, append, and their combinations.
  • File Class Methods: Methods provided by Ruby's File class to handle files.

Common File Modes

Ruby provides several file modes that you can use to open files. Here are the most commonly used ones:

Mode Description
r Read-only mode. The file must exist.
w Write-only mode. Creates a new file or truncates an existing file to zero length.
a Append mode. Write-only. Creates a new file if it doesn't exist.
r+ Read-write mode. The file must exist.
w+ Read-write mode. Creates a new file or truncates an existing file to zero length.
a+ Read-write mode. Creates a new file if it doesn't exist. Data is appended to the end of the file.

Examples

Read-Only Mode (`r`)

# Open a file in read-only mode
file = File.open("example.txt", "r")

# Read the content of the file
content = file.read
puts content

# Close the file
file.close

Explanation:

  • The file is opened in read-only mode using "r".
  • The read method reads the entire content of the file.
  • The close method closes the file.

Write-Only Mode (`w`)

# Open a file in write-only mode
file = File.open("example.txt", "w")

# Write content to the file
file.write("Hello, World!")

# Close the file
file.close

Explanation:

  • The file is opened in write-only mode using "w".
  • The write method writes the specified content to the file.
  • The close method closes the file.

Append Mode (`a`)

# Open a file in append mode
file = File.open("example.txt", "a")

# Append content to the file
file.write("\nAppended text.")

# Close the file
file.close

Explanation:

  • The file is opened in append mode using "a".
  • The write method appends the specified content to the end of the file.
  • The close method closes the file.

Read-Write Mode (`r+`)

# Open a file in read-write mode
file = File.open("example.txt", "r+")

# Read the current content
content = file.read
puts "Current content: #{content}"

# Write new content
file.write("\nNew content added.")

# Close the file
file.close

Explanation:

  • The file is opened in read-write mode using "r+".
  • The read method reads the current content of the file.
  • The write method writes new content to the file.
  • The close method closes the file.

Exercises

  1. Reading a File: Create a file named data.txt with some text in it. Write a Ruby script to read and print the content of the file.
  2. Writing to a File: Write a Ruby script to create a new file named output.txt and write the text "Ruby File Handling" to it.
  3. Appending to a File: Modify the script from Exercise 2 to append the text "Appending more text." to output.txt.
  4. Read-Write Operations: Write a Ruby script to open an existing file in read-write mode, read its content, and then append the text "End of file." to it.

Conclusion

Understanding file modes in Ruby is essential for effective file handling. By mastering the different modes, you can read from, write to, and append to files as needed. Practice the exercises provided to solidify your understanding and become proficient in handling files in Ruby.
