Introduction

File operations are essential for many applications, allowing you to read from and write to files. This section will cover the basics of file handling in Ruby, progressing from simple operations to more advanced techniques.

Basic File Operations

Opening and Closing Files

  • Opening a File: Use the File.open method to open a file. This method can take a block, which ensures the file is properly closed after the block is executed.
  • Closing a File: If you don't use a block, you must explicitly close the file using the close method.
# Opening a file with a block
File.open('example.txt', 'w') do |file|
  file.puts 'Hello, world!'
end
# The file is automatically closed after the block

# Opening a file without a block
file = File.open('example.txt', 'w')
file.puts 'Hello, world!'
file.close

Reading from Files

  • Reading Entire File: Use File.read to read the entire content of a file.
  • Reading Line by Line: Use File.foreach or IO.readlines to read a file line by line.
# Reading the entire file
content = File.read('example.txt')
puts content

# Reading line by line
File.foreach('example.txt') do |line|
  puts line
end

Writing to Files

  • Writing Modes: Use different modes like 'w' for write, 'a' for append, and 'r+' for read/write.
  • Writing Data: Use methods like puts, write, and print to write data to a file.
# Writing to a file
File.open('example.txt', 'w') do |file|
  file.puts 'Hello, world!'
end

# Appending to a file
File.open('example.txt', 'a') do |file|
  file.puts 'Appending this line.'
end

Intermediate File Operations

File Modes and Permissions

  • File Modes: Understand different file modes like 'r', 'w', 'a', 'r+', 'w+', and 'a+'.
  • Permissions: Use File.chmod to change file permissions.
# Changing file permissions
File.chmod(0644, 'example.txt')

File Positioning

  • Seek and Tell: Use seek to move the file pointer and tell to get the current position.
File.open('example.txt', 'r') do |file|
  file.seek(5, IO::SEEK_SET) # Move to the 5th byte
  puts file.tell # Outputs: 5
end

Working with Binary Files

  • Binary Mode: Open files in binary mode using 'rb', 'wb', or 'ab'.
# Reading a binary file
File.open('example.bin', 'rb') do |file|
  data = file.read
  puts data
end

Advanced File Operations

File Manipulation

  • Renaming and Deleting Files: Use File.rename and File.delete to rename and delete files.
# Renaming a file
File.rename('example.txt', 'new_example.txt')

# Deleting a file
File.delete('new_example.txt')

Directory Operations

  • Creating and Deleting Directories: Use Dir.mkdir and Dir.rmdir to create and delete directories.
  • Listing Directory Contents: Use Dir.entries or Dir.glob to list directory contents.
# Creating a directory
Dir.mkdir('new_directory')

# Listing directory contents
puts Dir.entries('.')

# Deleting a directory
Dir.rmdir('new_directory')

File and Directory Information

  • File Information: Use methods like File.size, File.exist?, and File.mtime to get information about files.
  • Directory Information: Use Dir.pwd to get the current directory and Dir.chdir to change the current directory.
# Getting file information
puts File.size('example.txt')
puts File.exist?('example.txt')
puts File.mtime('example.txt')

# Getting and changing the current directory
puts Dir.pwd
Dir.chdir('..')
puts Dir.pwd

Conclusion

File operations in Ruby are powerful and versatile, allowing you to handle files and directories efficiently. From basic reading and writing to advanced file manipulation and directory operations, mastering these skills will enable you to build robust applications that interact with the file system. Practice these concepts with various exercises to solidify your understanding and become proficient in file handling with Ruby.
