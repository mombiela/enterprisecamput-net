Introduction to Iterators

Iterators are a fundamental concept in Ruby, allowing you to traverse collections such as arrays and hashes. They provide a way to access each element in a collection without needing to manage the indexing manually.

Key Concepts

  • Iterator: A method that loops through a collection, yielding each element to a block.
  • Block: A chunk of code enclosed in do...end or curly braces {...} that can be passed to methods like iterators.

Basic Iterators

Ruby provides several built-in iterator methods, the most common being each.

Using `each` with Arrays

The each method is used to iterate over each element in an array.

numbers = [1, 2, 3, 4, 5]
numbers.each do |number|
  puts number
end
  • Explanation: The each method loops through the numbers array, yielding each element to the block. The block variable |number| represents the current element.

Using `each` with Hashes

The each method can also be used with hashes, yielding key-value pairs.

person = {name: "Alice", age: 30, city: "New York"}
person.each do |key, value|
  puts "#{key}: #{value}"
end
  • Explanation: The each method loops through the person hash, yielding each key-value pair to the block. The block variables |key, value| represent the current key and value.

Advanced Iterators

Ruby offers more advanced iterators for specific use cases.

`map`

The map method creates a new array containing the values returned by the block.

numbers = [1, 2, 3, 4, 5]
squares = numbers.map do |number|
  number * number
end
puts squares.inspect
  • Explanation: The map method iterates over the numbers array, applying the block to each element and returning a new array with the results.

`select`

The select method returns a new array containing elements for which the block returns true.

numbers = [1, 2, 3, 4, 5]
even_numbers = numbers.select do |number|
  number.even?
end
puts even_numbers.inspect
  • Explanation: The select method iterates over the numbers array, selecting elements that are even.

`reduce`

The reduce method combines all elements of an array by applying a binary operation, specified by a block or symbol.

numbers = [1, 2, 3, 4, 5]
sum = numbers.reduce(0) do |accumulator, number|
  accumulator + number
end
puts sum
  • Explanation: The reduce method iterates over the numbers array, accumulating the sum of all elements.

Custom Iterators

You can create custom iterators by defining methods that yield to a block.

def custom_each(array)
  for element in array
    yield(element)
  end
end

custom_each([1, 2, 3, 4, 5]) do |number|
  puts number
end
  • Explanation: The custom_each method takes an array and yields each element to the block, similar to the built-in each method.

Conclusion

Iterators are a powerful feature in Ruby, enabling you to traverse and manipulate collections with ease. By understanding and utilizing iterators like each, map, select, and reduce, you can write more concise and readable code. Custom iterators further extend this flexibility, allowing you to create specialized iteration behaviors tailored to your needs.
