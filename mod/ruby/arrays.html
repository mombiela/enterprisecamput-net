Introduction to Arrays

Arrays are ordered, integer-indexed collections of any object. They are a fundamental data structure in Ruby, allowing you to store and manipulate lists of data.

Key Concepts

  • Ordered Collection: Elements in an array are stored in a specific order.
  • Indexing: Arrays use zero-based indexing, meaning the first element is at index 0.
  • Dynamic Size: Arrays can grow or shrink as needed.

Creating Arrays

You can create arrays using square brackets [] or the Array class.

# Using square brackets
numbers = [1, 2, 3, 4, 5]

# Using the Array class
words = Array.new
words = Array.new(3) # Creates an array with 3 nil elements
words = Array.new(3, "hello") # Creates an array with 3 "hello" elements

Accessing Elements

Access elements using their index.

numbers = [1, 2, 3, 4, 5]
puts numbers[0] # Output: 1
puts numbers[4] # Output: 5

Modifying Arrays

You can modify arrays by assigning values to specific indices or using methods like push, pop, shift, and unshift.

numbers = [1, 2, 3, 4, 5]
numbers[2] = 10 # Modifies the third element
numbers.push(6) # Adds 6 to the end of the array
numbers.pop # Removes the last element
numbers.shift # Removes the first element
numbers.unshift(0) # Adds 0 to the beginning of the array

Array Methods

Ruby provides a rich set of methods to manipulate arrays.

Common Methods

  • length or size: Returns the number of elements.
  • empty?: Checks if the array is empty.
  • include?: Checks if an element is present.
  • sort: Sorts the array.
  • reverse: Reverses the array.
  • join: Converts the array to a string.
numbers = [3, 1, 4, 1, 5, 9]
puts numbers.length # Output: 6
puts numbers.empty? # Output: false
puts numbers.include?(4) # Output: true
puts numbers.sort # Output: [1, 1, 3, 4, 5, 9]
puts numbers.reverse # Output: [9, 5, 1, 4, 1, 3]
puts numbers.join(", ") # Output: "3, 1, 4, 1, 5, 9"

Iterating Over Arrays

Use loops or iterators to process each element in an array.

numbers = [1, 2, 3, 4, 5]

# Using a for loop
for number in numbers
  puts number
end

# Using the each method
numbers.each do |number|
  puts number
end

Advanced Array Techniques

Nested Arrays

Arrays can contain other arrays, creating multi-dimensional arrays.

matrix = [[1, 2, 3], [4, 5, 6], [7, 8, 9]]
puts matrix[1][2] # Output: 6

Array Transformations

Use methods like map, select, and reject to transform arrays.

numbers = [1, 2, 3, 4, 5]

# Using map to create a new array with each element squared
squares = numbers.map { |number| number ** 2 }
puts squares # Output: [1, 4, 9, 16, 25]

# Using select to filter elements
evens = numbers.select { |number| number.even? }
puts evens # Output: [2, 4]

# Using reject to filter elements
odds = numbers.reject { |number| number.even? }
puts odds # Output: [1, 3, 5]

Array Set Operations

Arrays support set operations like union, intersection, and difference.

a = [1, 2, 3]
b = [3, 4, 5]

# Union
puts (a | b) # Output: [1, 2, 3, 4, 5]

# Intersection
puts (a & b) # Output: [3]

# Difference
puts (a - b) # Output: [1, 2]

Conclusion

Arrays are a versatile and powerful data structure in Ruby, essential for handling collections of data. From basic operations like accessing and modifying elements to advanced techniques like transformations and set operations, mastering arrays will significantly enhance your Ruby programming skills. Practice using arrays in different scenarios to become proficient in their use.
