What is Cryptography?

Cryptography is the science and art of creating and analyzing protocols that prevent third parties or the general public from reading private messages. Modern cryptography is based on mathematical theories and the practice of computing.

Objectives of Cryptography

  • Confidentiality: Ensure that information is only accessible to authorized parties.
  • Integrity: Ensure that information has not been altered in an unauthorized manner.
  • Authentication: Verify the identity of the parties involved in the communication.
  • Non-repudiation: Prevent a party from denying having performed an action.

Types of Cryptography

Symmetric Cryptography

In symmetric cryptography, the same key is used both to encrypt and decrypt the information.

  • Advantages:
    • Fast and efficient.
    • Fewer computational resources.
  • Disadvantages:
    • Secure key distribution.
    • Not scalable for large systems.

Example of Symmetric Cryptography: AES Encryption

from Crypto.Cipher import AES
from Crypto.Random import get_random_bytes

# Generate a random key
key = get_random_bytes(16)

# Create an AES cipher object
cipher = AES.new(key, AES.MODE_EAX)

# Message to encrypt
data = b"Secret message"

# Encrypt the message
ciphertext, tag = cipher.encrypt_and_digest(data)

print("Encrypted text:", ciphertext)

Asymmetric Cryptography

In asymmetric cryptography, two different keys are used: a public key to encrypt and a private key to decrypt.

  • Advantages:
    • Does not require secure key distribution.
    • Scalable for large systems.
  • Disadvantages:
    • Slower and requires more computational resources.
    • Complexity in key management.

Example of Asymmetric Cryptography: RSA Encryption

from Crypto.PublicKey import RSA
from Crypto.Cipher import PKCS1_OAEP

# Generate an RSA key pair
key = RSA.generate(2048)
public_key = key.publickey()

# Create an RSA cipher object with the public key
cipher = PKCS1_OAEP.new(public_key)

# Message to encrypt
data = b"Secret message"

# Encrypt the message
ciphertext = cipher.encrypt(data)

print("Encrypted text:", ciphertext)

Common Cryptographic Algorithms

| Algorithm | Type | Main Use | |-----------|------|----------| | AES | Symmetric | Data encryption | | DES | Symmetric | Data encryption (obsolete) | | RSA | Asymmetric | Encryption and digital signature | | ECC | Asymmetric | Encryption and digital signature | | SHA-256 | Hash | Data integrity |

Applications of Cryptography

  • Communication security: Protect information transmitted over networks.
  • Digital signatures: Verify the authenticity and integrity of messages.
  • Authentication: Confirm the identity of users and devices.
  • Secure storage: Protect stored data against unauthorized access.

Conclusion

Cryptography is an essential tool in computer security that allows protecting the confidentiality, integrity, authentication, and non-repudiation of information. Understanding the different types of cryptography and their applications is fundamental for any IT professional who wishes to effectively secure systems and data.
