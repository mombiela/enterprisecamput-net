Introduction

Scala provides a rich set of collections and functional operations that allow developers to write concise and expressive code. This section will cover the basics of Scala collections, including lists, sets, maps, and sequences, as well as functional operations like map, filter, and reduce.

Collections in Scala

Scala collections are divided into three main categories:

  • Sequences: Ordered collections of elements.
  • Sets: Unordered collections of unique elements.
  • Maps: Collections of key-value pairs.

Sequences

Sequences are ordered collections of elements. The most common sequence types in Scala are List, Array, and Vector.

List

A List is an immutable sequence. Here’s how to create and use a list:

val fruits = List("apple", "banana", "cherry")
println(fruits.head) // Output: apple
println(fruits.tail) // Output: List(banana, cherry)

Array

An Array is a mutable sequence. Here’s how to create and use an array:

val numbers = Array(1, 2, 3, 4, 5)
numbers(0) = 10
println(numbers.mkString(", ")) // Output: 10, 2, 3, 4, 5

Vector

A Vector is an immutable sequence that provides fast random access. Here’s how to create and use a vector:

val vector = Vector(1, 2, 3, 4, 5)
println(vector(2)) // Output: 3

Sets

Sets are collections of unique elements. Scala provides both mutable and immutable sets.

Immutable Set

Here’s how to create and use an immutable set:

val set = Set(1, 2, 3, 4, 5)
println(set.contains(3)) // Output: true
println(set.contains(6)) // Output: false

Mutable Set

Here’s how to create and use a mutable set:

import scala.collection.mutable

val mutableSet = mutable.Set(1, 2, 3)
mutableSet += 4
println(mutableSet) // Output: Set(1, 2, 3, 4)

Maps

Maps are collections of key-value pairs. Scala provides both mutable and immutable maps.

Immutable Map

Here’s how to create and use an immutable map:

val map = Map("a" -> 1, "b" -> 2, "c" -> 3)
println(map("a")) // Output: 1

Mutable Map

Here’s how to create and use a mutable map:

import scala.collection.mutable

val mutableMap = mutable.Map("a" -> 1, "b" -> 2)
mutableMap("c") = 3
println(mutableMap) // Output: Map(a -> 1, b -> 2, c -> 3)

Functional Operations

Functional operations allow you to manipulate collections in a functional programming style. The most common functional operations are map, filter, and reduce.

Map

The map operation applies a function to each element of a collection and returns a new collection with the results.

val numbers = List(1, 2, 3, 4, 5)
val doubled = numbers.map(_ * 2)
println(doubled) // Output: List(2, 4, 6, 8, 10)

Filter

The filter operation selects elements of a collection that satisfy a predicate.

val numbers = List(1, 2, 3, 4, 5)
val evenNumbers = numbers.filter(_ % 2 == 0)
println(evenNumbers) // Output: List(2, 4)

Reduce

The reduce operation combines the elements of a collection using a binary function.

val numbers = List(1, 2, 3, 4, 5)
val sum = numbers.reduce(_ + _)
println(sum) // Output: 15

Conclusion

In this section, we covered the basics of Scala collections and functional operations. We explored different types of collections such as lists, arrays, vectors, sets, and maps, and learned how to perform functional operations like map, filter, and reduce. Understanding these concepts is crucial for writing efficient and expressive Scala code. Practice using these collections and operations to become more proficient in Scala programming.
