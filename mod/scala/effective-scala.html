Introduction to Scala

Scala is a powerful programming language that combines object-oriented and functional programming paradigms. It is designed to be concise, elegant, and type-safe, making it an excellent choice for both beginners and experienced developers.

Getting Started with Scala

  • Installation: How to install Scala on different operating systems.
  • Scala REPL: Introduction to the Scala Read-Eval-Print Loop (REPL) for interactive programming.
  • Hello World: Writing and running your first Scala program.
object HelloWorld {
  def main(args: Array[String]): Unit = {
    println("Hello, World!")
  }
}

Basic Syntax and Data Types

  • Variables and Values: Understanding var and val.
  • Basic Data Types: Int, Double, String, Boolean, etc.
  • Type Inference: Letting Scala infer types for you.
val immutableValue: Int = 10
var mutableVariable: String = "Hello"
mutableVariable = "World"

Control Structures

  • Conditional Statements: if, else if, else.
  • Loops: for, while, do-while.
  • Pattern Matching: A powerful feature for control flow.
val number = 10
if (number > 0) {
  println("Positive")
} else if (number < 0) {
  println("Negative")
} else {
  println("Zero")
}

for (i <- 1 to 5) {
  println(i)
}

number match {
  case 1 => println("One")
  case 2 => println("Two")
  case _ => println("Other")
}

Functions and Methods

  • Defining Functions: Syntax and usage.
  • Higher-Order Functions: Functions that take other functions as parameters.
  • Anonymous Functions: Lambda expressions.
def add(a: Int, b: Int): Int = {
  a + b
}

val sum = add(5, 3)

val multiply = (x: Int, y: Int) => x * y
val product = multiply(4, 2)

Collections

  • Lists: Immutable sequences.
  • Arrays: Mutable sequences.
  • Maps: Key-value pairs.
  • Tuples: Fixed-size collections of elements.
val list = List(1, 2, 3, 4, 5)
val array = Array(1, 2, 3, 4, 5)
val map = Map("a" -> 1, "b" -> 2)
val tuple = (1, "Scala", true)

Object-Oriented Programming

  • Classes and Objects: Defining and using classes.
  • Inheritance: Extending classes.
  • Traits: Similar to interfaces in other languages.
class Animal {
  def speak(): Unit = {
    println("Animal speaks")
  }
}

class Dog extends Animal {
  override def speak(): Unit = {
    println("Dog barks")
  }
}

val dog = new Dog()
dog.speak()

trait Friendly {
  def greet(): Unit = {
    println("Hello!")
  }
}

class Person extends Friendly

val person = new Person()
person.greet()

Functional Programming

  • Immutability: Benefits and usage.
  • Pure Functions: Functions without side effects.
  • Monads: Understanding Option, Either, and Future.
val numbers = List(1, 2, 3, 4, 5)
val doubled = numbers.map(_ * 2)

val maybeNumber: Option[Int] = Some(5)
val result = maybeNumber match {
  case Some(value) => value * 2
  case None => 0
}

Advanced Topics

  • Implicit Conversions: Automatically converting types.
  • Type Classes: Ad-hoc polymorphism.
  • Concurrency: Using Futures and Promises for asynchronous programming.
implicit def intToString(x: Int): String = x.toString

def printString(s: String): Unit = {
  println(s)
}

printString(123) // Implicitly converts Int to String

import scala.concurrent.Future
import scala.concurrent.ExecutionContext.Implicits.global

val futureResult = Future {
  Thread.sleep(1000)
  42
}

futureResult.map(result => println(s"Result: $result"))

Conclusion

Scala is a versatile language that offers a rich set of features for both object-oriented and functional programming. By mastering the basics and exploring advanced topics, you can write efficient, concise, and elegant code. Practice regularly and explore Scala's extensive libraries and frameworks to become proficient in this powerful language.
