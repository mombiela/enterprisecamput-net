Example: Setting Up a New Play Project

sbt new playframework/play-scala-seed.g8
cd play-scala-seed
sbt run
  • Explanation: This sequence of commands initializes a new Play project using the Play Scala seed template and runs the development server.

Basic Concepts

  • Understanding MVC Architecture

    • Model: Data and business logic
    • View: User interface
    • Controller: Request handling and response generation

  • Routing

    • Defining routes in conf/routes
    • Example: 
      GET     /                           controllers.HomeController.index
POST    /submit                     controllers.HomeController.submit
    • Explanation: This routes file maps HTTP requests to controller actions.

  • Controllers

    • Creating a basic controller
    • Example: 
      package controllers

import javax.inject._
import play.api.mvc._

@Singleton
class HomeController @Inject()(val controllerComponents: ControllerComponents) extends BaseController {

  def index() = Action { implicit request: Request[AnyContent] =>
    Ok("Hello, Play Framework!")
  }
}
    • Explanation: This controller handles a GET request to the root URL and responds with a simple text message.

Working with Views

  • Using Twirl Templates

    • Creating a basic HTML view
    • Example: 
      @(message: String)

<!DOCTYPE html>
<html>
  <head>
    <title>Welcome to Play</title>
  </head>
  <body>
    <h1>@message</h1>
  </body>
</html>
    • Explanation: This Twirl template takes a message parameter and displays it within an HTML structure.

  • Integrating Views with Controllers

    • Example: 
      def index() = Action { implicit request: Request[AnyContent] =>
  Ok(views.html.index("Welcome to Play Framework!"))
}
    • Explanation: This controller action renders the index.scala.html view with a message.

Advanced Topics

  • Form Handling

    • Creating and processing forms
    • Example: 
      import play.api.data._
import play.api.data.Forms._

case class UserData(name: String, age: Int)

val userForm = Form(
  mapping(
    "name" -> nonEmptyText,
    "age" -> number(min = 0, max = 100)
  )(UserData.apply)(UserData.unapply)
)

  • Database Integration

    • Using Slick for database operations
    • Example: 
      import slick.jdbc.H2Profile.api._

case class User(id: Long, name: String, age: Int)

class Users(tag: Tag) extends Table[User](tag, "USERS") {
  def id = column[Long]("ID", O.PrimaryKey, O.AutoInc)
  def name = column[String]("NAME")
  def age = column[Int]("AGE")
  def * = (id, name, age) <> (User.tupled, User.unapply)
}

val users = TableQuery[Users]
    • Explanation: This defines a User table schema using Slick.

  • Asynchronous Programming

    • Using Futures in Play
    • Example: 
      import scala.concurrent.Future
import scala.concurrent.ExecutionContext.Implicits.global

def asyncAction() = Action.async {
  val futureResult: Future[String] = Future {
    // Simulate a long-running computation
    Thread.sleep(1000)
    "Async Result"
  }
  futureResult.map(result => Ok(result))
}
    • Explanation: This controller action demonstrates handling asynchronous operations using Scala Futures.
