Introduction to Convolutional Neural Networks

Convolutional Neural Networks (CNN) are a type of neural network especially effective for image processing and pattern recognition tasks. Unlike traditional neural networks, CNNs use convolutional layers that allow extracting spatial and hierarchical features from input data.

Architecture of a CNN

A typical CNN is composed of several layers, each with a specific function:

  1. Convolutional Layer
  2. Pooling Layer
  3. Normalization Layer
  4. Fully Connected Layer

Convolutional Layer

The convolutional layer is the core of a CNN. It uses filters (or kernels) to perform convolutions on the input image, producing feature maps.

  • Filter/Kernels: Small matrices that slide over the input image.
  • Stride: The step with which the filter moves over the image.
  • Padding: Adding borders to the image to control the output size.

Code example:

import tensorflow as tf
from tensorflow.keras import layers

model = tf.keras.Sequential([
    layers.Conv2D(32, (3, 3), activation='relu', input_shape=(28, 28, 1)),
    layers.MaxPooling2D((2, 2)),
    layers.Conv2D(64, (3, 3), activation='relu'),
    layers.MaxPooling2D((2, 2)),
    layers.Flatten(),
    layers.Dense(64, activation='relu'),
    layers.Dense(10, activation='softmax')
])

Pooling Layer

The pooling layer reduces the spatial dimensions of the feature maps, decreasing the number of parameters and computational cost.

  • Max Pooling: Selects the maximum value in a window.
  • Average Pooling: Calculates the average of the values in a window.

Code example:

model.add(layers.MaxPooling2D((2, 2)))

Normalization Layer

Normalization helps accelerate training and improve network stability.

  • Batch Normalization: Normalizes the activations of a layer for each mini-batch.

Code example:

model.add(layers.BatchNormalization())

Fully Connected Layer

Fully connected layers are densely connected layers used at the end of the network to perform classification.

Code example:

model.add(layers.Dense(64, activation='relu'))
model.add(layers.Dense(10, activation='softmax'))

Complete Example of a CNN

Below is a complete example of a CNN for image classification using the MNIST dataset:

import tensorflow as tf
from tensorflow.keras import datasets, layers, models

# Load and preprocess data
(train_images, train_labels), (test_images, test_labels) = datasets.mnist.load_data()
train_images = train_images.reshape((60000, 28, 28, 1)).astype('float32') / 255
test_images = test_images.reshape((10000, 28, 28, 1)).astype('float32') / 255

# Define the model
model = models.Sequential([
    layers.Conv2D(32, (3, 3), activation='relu', input_shape=(28, 28, 1)),
    layers.MaxPooling2D((2, 2)),
    layers.Conv2D(64, (3, 3), activation='relu'),
    layers.MaxPooling2D((2, 2)),
    layers.Flatten(),
    layers.Dense(64, activation='relu'),
    layers.Dense(10, activation='softmax')
])

# Compile the model
model.compile(optimizer='adam',
              loss='sparse_categorical_crossentropy',
              metrics=['accuracy'])

# Train the model
model.fit(train_images, train_labels, epochs=5, validation_data=(test_images, test_labels))

Conclusion

Convolutional Neural Networks are a powerful tool for image processing and other pattern recognition tasks. Their architecture, which includes convolutional, pooling, and fully connected layers, allows extracting and learning complex features from input data. With practice and experimentation, professionals can adjust and optimize CNNs for a variety of applications in the field of artificial intelligence.
