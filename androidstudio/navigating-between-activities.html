Navigating between activities is a fundamental concept in Android development. This topic will guide you through the basics of activity navigation, from creating intents to passing data between activities. By the end of this section, you will be able to create seamless transitions between different screens in your Android app.

Introduction to Activities

  • Activity: A single, focused thing that the user can do. Almost all activities interact with the user, so the Activity class takes care of creating a window for you in which you can place your UI with setContentView(View).
  • Intent: An abstract description of an operation to be performed. It can be used with startActivity to launch an activity.

Creating and Starting an Activity

To navigate from one activity to another, you need to create an intent and use it to start the new activity.

Example: Starting a New Activity

  1. Create a new Activity: Suppose you have a main activity (MainActivity) and you want to navigate to a second activity (SecondActivity).
// MainActivity.java
package com.example.myfirstapp;

import android.content.Intent;
import android.os.Bundle;
import android.view.View;
import androidx.appcompat.app.AppCompatActivity;

public class MainActivity extends AppCompatActivity {
    @Override
    protected void onCreate(Bundle savedInstanceState) {
        super.onCreate(savedInstanceState);
        setContentView(R.layout.activity_main);
    }

    public void navigateToSecondActivity(View view) {
        Intent intent = new Intent(this, SecondActivity.class);
        startActivity(intent);
    }
}
  1. Define the new Activity in the Manifest: Ensure that SecondActivity is declared in your AndroidManifest.xml.
<!-- AndroidManifest.xml -->
<application
    ... >
    <activity android:name=".MainActivity">
        <intent-filter>
            <action android:name="android.intent.action.MAIN" />
            <category android:name="android.intent.category.LAUNCHER" />
        </intent-filter>
    </activity>
    <activity android:name=".SecondActivity"></activity>
</application>
  1. Create the new Activity class: Implement the SecondActivity.
// SecondActivity.java
package com.example.myfirstapp;

import android.os.Bundle;
import androidx.appcompat.app.AppCompatActivity;

public class SecondActivity extends AppCompatActivity {
    @Override
    protected void onCreate(Bundle savedInstanceState) {
        super.onCreate(savedInstanceState);
        setContentView(R.layout.activity_second);
    }
}
  1. Create the layout for the new Activity: Define the UI for SecondActivity in activity_second.xml.
<!-- activity_second.xml -->
<LinearLayout xmlns:android="http://schemas.android.com/apk/res/android"
    android:layout_width="match_parent"
    android:layout_height="match_parent"
    android:orientation="vertical"
    android:padding="16dp">

    <TextView
        android:layout_width="wrap_content"
        android:layout_height="wrap_content"
        android:text="Welcome to the Second Activity!" />
</LinearLayout>

Passing Data Between Activities

Often, you need to pass data from one activity to another. This can be done using Intent extras.

Example: Passing Data with Intent Extras

  1. Send Data from MainActivity:
// MainActivity.java
public void navigateToSecondActivity(View view) {
    Intent intent = new Intent(this, SecondActivity.class);
    intent.putExtra("EXTRA_MESSAGE", "Hello, Second Activity!");
    startActivity(intent);
}
  1. Receive Data in SecondActivity:
// SecondActivity.java
@Override
protected void onCreate(Bundle savedInstanceState) {
    super.onCreate(savedInstanceState);
    setContentView(R.layout.activity_second);

    Intent intent = getIntent();
    String message = intent.getStringExtra("EXTRA_MESSAGE");

    TextView textView = findViewById(R.id.textView);
    textView.setText(message);
}
  1. Update the layout to include a TextView:
<!-- activity_second.xml -->
<LinearLayout xmlns:android="http://schemas.android.com/apk/res/android"
    android:layout_width="match_parent"
    android:layout_height="match_parent"
    android:orientation="vertical"
    android:padding="16dp">

    <TextView
        android:id="@+id/textView"
        android:layout_width="wrap_content"
        android:layout_height="wrap_content"
        android:text="Welcome to the Second Activity!" />
</LinearLayout>

Navigating Back to the Previous Activity

Android handles the back navigation for you, but you can also explicitly finish an activity.

Example: Finishing an Activity

// SecondActivity.java
public void finishActivity(View view) {
    finish();
}

Conclusion

Navigating between activities is a crucial part of Android development. By understanding how to create intents, start activities, and pass data between them, you can create more dynamic and interactive applications. Practice these concepts by creating multiple activities and navigating between them, passing different types of data to get comfortable with the process.
