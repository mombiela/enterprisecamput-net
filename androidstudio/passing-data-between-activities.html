Passing Data Between Activities
Introduction to Fragments ►

Passing data between activities is a fundamental concept in Android development. This topic will guide you from the basics to advanced techniques for sharing data between activities in an Android application.

Introduction to Intents

Intents are the primary method for starting new activities and passing data between them.

  • Intent: An abstract description of an operation to be performed.
  • Explicit Intent: Specifies the exact activity to start.
  • Implicit Intent: Declares a general action to perform, allowing the system to find an appropriate activity.

Creating an Intent

To start a new activity, you create an Intent object and call startActivity().

Intent intent = new Intent(CurrentActivity.this, TargetActivity.class);
startActivity(intent);

Passing Data with Intents

You can pass data between activities using the putExtra method of the Intent class.

Basic Data Types

You can pass basic data types such as int, String, boolean, etc.

Intent intent = new Intent(CurrentActivity.this, TargetActivity.class);
intent.putExtra("EXTRA_MESSAGE", "Hello, World!");
intent.putExtra("EXTRA_NUMBER", 123);
startActivity(intent);

Retrieving Data in the Target Activity

In the target activity, you retrieve the data using the getIntent() method and the appropriate get methods.

Intent intent = getIntent();
String message = intent.getStringExtra("EXTRA_MESSAGE");
int number = intent.getIntExtra("EXTRA_NUMBER", 0);

Passing Complex Data Types

For more complex data types, such as custom objects, you need to implement the Parcelable or Serializable interface.

Using Parcelable

Parcelable is the preferred method for passing complex data types because it is more efficient than Serializable.

Creating a Parcelable Class

public class MyObject implements Parcelable {
    private int id;
    private String name;

    // Constructor
    public MyObject(int id, String name) {
        this.id = id;
        this.name = name;
    }

    // Parcelable implementation
    protected MyObject(Parcel in) {
        id = in.readInt();
        name = in.readString();
    }

    public static final Creator<MyObject> CREATOR = new Creator<MyObject>() {
        @Override
        public MyObject createFromParcel(Parcel in) {
            return new MyObject(in);
        }

        @Override
        public MyObject[] newArray(int size) {
            return new MyObject[size];
        }
    };

    @Override
    public int describeContents() {
        return 0;
    }

    @Override
    public void writeToParcel(Parcel dest, int flags) {
        dest.writeInt(id);
        dest.writeString(name);
    }
}

Passing a Parcelable Object

MyObject myObject = new MyObject(1, "Sample");
Intent intent = new Intent(CurrentActivity.this, TargetActivity.class);
intent.putExtra("EXTRA_MY_OBJECT", myObject);
startActivity(intent);

Retrieving a Parcelable Object

Intent intent = getIntent();
MyObject myObject = intent.getParcelableExtra("EXTRA_MY_OBJECT");

Advanced Techniques

Using Bundles

Bundles are useful for passing multiple pieces of data together.

Bundle bundle = new Bundle();
bundle.putString("EXTRA_MESSAGE", "Hello, World!");
bundle.putInt("EXTRA_NUMBER", 123);

Intent intent = new Intent(CurrentActivity.this, TargetActivity.class);
intent.putExtras(bundle);
startActivity(intent);

Using Shared Preferences

For persistent data that needs to be accessed across multiple activities, consider using Shared Preferences.

// Saving data
SharedPreferences sharedPreferences = getSharedPreferences("MyPrefs", MODE_PRIVATE);
SharedPreferences.Editor editor = sharedPreferences.edit();
editor.putString("EXTRA_MESSAGE", "Hello, World!");
editor.apply();

// Retrieving data
SharedPreferences sharedPreferences = getSharedPreferences("MyPrefs", MODE_PRIVATE);
String message = sharedPreferences.getString("EXTRA_MESSAGE", "Default Value");

Conclusion

Passing data between activities is a crucial skill in Android development. By mastering intents, bundles, and shared preferences, you can efficiently share data and create a seamless user experience. Practice these techniques to become proficient in managing data flow in your Android applications.
