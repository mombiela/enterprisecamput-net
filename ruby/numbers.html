Introduction

Numbers are a fundamental part of any programming language, and Ruby is no exception. In this section, we will cover the basics of numbers in Ruby, including different types of numbers, basic arithmetic operations, and some advanced topics like working with large numbers and complex numbers.

Types of Numbers

Ruby supports several types of numbers, each with its own characteristics and use cases.

  • Integer: Whole numbers without a decimal point.
  • Float: Numbers with a decimal point.
  • Rational: Numbers represented as fractions.
  • Complex: Numbers with a real and an imaginary part.

Integer

Integers are whole numbers, both positive and negative, including zero.

a = 42
b = -7
c = 0

Float

Floats are numbers that contain a decimal point.

a = 3.14
b = -0.001
c = 2.0

Rational

Rational numbers are represented as fractions.

a = Rational(2, 3)  # 2/3
b = Rational(7, 4)  # 7/4

Complex

Complex numbers have a real part and an imaginary part.

a = Complex(2, 3)  # 2 + 3i
b = Complex(1, -4) # 1 - 4i

Basic Arithmetic Operations

Ruby supports basic arithmetic operations such as addition, subtraction, multiplication, and division.

# Addition
sum = 5 + 3  # 8

# Subtraction
difference = 10 - 4  # 6

# Multiplication
product = 7 * 6  # 42

# Division
quotient = 20 / 4  # 5

# Modulus (remainder)
remainder = 10 % 3  # 1

# Exponentiation
power = 2 ** 3  # 8

Advanced Topics

Working with Large Numbers

Ruby can handle very large numbers with ease. You don't need to worry about integer overflow.

large_number = 123456789012345678901234567890
puts large_number * large_number

Converting Between Number Types

You can convert between different number types using Ruby's built-in methods.

# Integer to Float
int_num = 10
float_num = int_num.to_f  # 10.0

# Float to Integer
float_num = 3.14
int_num = float_num.to_i  # 3

# Integer to Rational
int_num = 5
rational_num = int_num.to_r  # 5/1

# Float to Rational
float_num = 2.5
rational_num = float_num.to_r  # 5/2

# Integer to Complex
int_num = 4
complex_num = int_num.to_c  # 4+0i

Complex Number Operations

Ruby provides methods to perform arithmetic operations on complex numbers.

a = Complex(2, 3)
b = Complex(1, -4)

# Addition
sum = a + b  # 3 - 1i

# Subtraction
difference = a - b  # 1 + 7i

# Multiplication
product = a * b  # 14 - 5i

# Division
quotient = a / b  # -0.6470588235294118 + 0.7058823529411765i

Summary

In this section, we covered the basics of numbers in Ruby, including different types of numbers (Integer, Float, Rational, Complex), basic arithmetic operations, and some advanced topics like working with large numbers and converting between number types. Understanding these concepts is crucial for performing mathematical operations and handling numerical data in your Ruby programs.
