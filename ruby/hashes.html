Introduction to Hashes

Hashes, also known as dictionaries, maps, or associative arrays, are a fundamental data structure in Ruby. They store key-value pairs, allowing you to retrieve values based on their corresponding keys.

Key Concepts

  • Key-Value Pairs: Each element in a hash is a pair consisting of a unique key and a value.
  • Unordered Collection: Unlike arrays, hashes do not maintain any specific order for their elements.
  • Flexible Keys and Values: Keys and values can be of any data type.

Creating Hashes

You can create hashes using different syntaxes in Ruby.

Using Literal Notation

# Creating an empty hash
empty_hash = {}

# Creating a hash with initial key-value pairs
person = {
  "name" => "Alice",
  "age" => 30,
  "city" => "New York"
}

Using the `Hash` Class

# Creating an empty hash
empty_hash = Hash.new

# Creating a hash with initial key-value pairs
person = Hash.new
person["name"] = "Alice"
person["age"] = 30
person["city"] = "New York"

Accessing and Modifying Hashes

Accessing Values

You can access values in a hash using their keys.

person = {
  "name" => "Alice",
  "age" => 30,
  "city" => "New York"
}

# Accessing values
puts person["name"]  # Output: Alice
puts person["age"]   # Output: 30

Modifying Values

You can modify existing values or add new key-value pairs.

person["age"] = 31  # Modifying an existing value
person["country"] = "USA"  # Adding a new key-value pair

puts person
# Output: {"name"=>"Alice", "age"=>31, "city"=>"New York", "country"=>"USA"}

Common Hash Methods

Ruby provides several useful methods for working with hashes.

`keys` and `values`

Retrieve all keys or values from a hash.

person = {
  "name" => "Alice",
  "age" => 30,
  "city" => "New York"
}

puts person.keys   # Output: ["name", "age", "city"]
puts person.values # Output: ["Alice", 30, "New York"]

`each`

Iterate over each key-value pair in a hash.

person.each do |key, value|
  puts "#{key}: #{value}"
end
# Output:
# name: Alice
# age: 30
# city: New York

`delete`

Remove a key-value pair from a hash.

person.delete("age")
puts person
# Output: {"name"=>"Alice", "city"=>"New York"}

`merge`

Combine two hashes into one.

person = {
  "name" => "Alice",
  "age" => 30
}

address = {
  "city" => "New York",
  "country" => "USA"
}

person.merge!(address)
puts person
# Output: {"name"=>"Alice", "age"=>30, "city"=>"New York", "country"=>"USA"}

Advanced Hash Techniques

Default Values

You can set a default value for a hash, which will be returned when accessing a non-existent key.

hash_with_default = Hash.new("Not Found")
puts hash_with_default["missing_key"]  # Output: Not Found

Symbols as Keys

Using symbols as keys can be more efficient and readable.

person = {
  name: "Alice",
  age: 30,
  city: "New York"
}

puts person[:name]  # Output: Alice

Nested Hashes

Hashes can contain other hashes, allowing for more complex data structures.

person = {
  name: "Alice",
  age: 30,
  address: {
    city: "New York",
    country: "USA"
  }
}

puts person[:address][:city]  # Output: New York

Conclusion

Hashes are a versatile and powerful data structure in Ruby, suitable for a wide range of applications. Understanding how to create, access, and manipulate hashes will significantly enhance your ability to write efficient and effective Ruby code. Practice using the methods and techniques discussed in this section to become proficient in working with hashes.
