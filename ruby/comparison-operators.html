Introduction

Comparison operators in Ruby are used to compare two values. They return a boolean value (true or false) based on the comparison. Understanding these operators is crucial for controlling the flow of your programs through conditional statements and loops.

Key Comparison Operators

Equality Operators

  • ==: Checks if two values are equal.
  • !=: Checks if two values are not equal.
a = 5
b = 5
c = 10

puts a == b  # Output: true
puts a != c  # Output: true

Relational Operators

  • >: Greater than.
  • <: Less than.
  • >=: Greater than or equal to.
  • <=: Less than or equal to.
x = 7
y = 10

puts x > y   # Output: false
puts x < y   # Output: true
puts x >= 7  # Output: true
puts y <= 10 # Output: true

Combined Comparison Operator

  • <=>: Also known as the spaceship operator, it returns:
    • -1 if the left operand is less than the right operand.
    • 0 if the left operand is equal to the right operand.
    • 1 if the left operand is greater than the right operand.
puts 5 <=> 10  # Output: -1
puts 10 <=> 10 # Output: 0
puts 15 <=> 10 # Output: 1

Case Equality Operator

  • ===: Used in case statements to test equality.
range = (1..10)

puts range === 5  # Output: true
puts range === 15 # Output: false

Object Identity

  • equal?: Checks if two objects are the same object.
str1 = "hello"
str2 = "hello"
str3 = str1

puts str1.equal?(str2) # Output: false
puts str1.equal?(str3) # Output: true

Practical Examples

Using Comparison Operators in Conditional Statements

age = 18

if age >= 18
  puts "You are eligible to vote."
else
  puts "You are not eligible to vote."
end

Using the Spaceship Operator for Sorting

numbers = [5, 2, 9, 3, 8]

sorted_numbers = numbers.sort { |a, b| a <=> b }
puts sorted_numbers.inspect  # Output: [2, 3, 5, 8, 9]

Using Case Equality in Case Statements

grade = 'B'

case grade
when 'A'
  puts "Excellent!"
when 'B'
  puts "Good job!"
when 'C'
  puts "Well done!"
else
  puts "Keep trying!"
end

Summary

Comparison operators are fundamental in Ruby for making decisions and controlling the flow of your programs. They allow you to compare values for equality, relational order, and object identity. Understanding how to use these operators effectively will enable you to write more dynamic and responsive code.

By mastering these operators, you can handle a wide range of programming tasks, from simple conditional checks to complex sorting algorithms. Practice using these operators in different scenarios to become proficient in their application.
