Setting Up Rails
MVC Architecture ►

Setting up Rails is an essential step for any Ruby developer looking to build web applications. This section will guide you through the process of installing and configuring Rails on your system, ensuring you have a solid foundation to start developing your applications.

Prerequisites

Before setting up Rails, ensure you have the following prerequisites installed on your system:

  • Ruby: Rails is a Ruby framework, so you need Ruby installed. Preferably, use a version manager like RVM or rbenv.
  • Node.js: Rails uses Node.js for managing JavaScript dependencies.
  • Yarn: A package manager for JavaScript, used by Rails for managing front-end dependencies.
  • SQLite3: A lightweight database used by default in Rails.

Installing Ruby

First, install Ruby using a version manager. Here, we'll use rbenv:

  1. Install rbenv and ruby-build:

    $ brew install rbenv
$ brew install ruby-build

  2. Set up rbenv in your shell:

    $ echo 'export PATH="$HOME/.rbenv/bin:$PATH"' >> ~/.bash_profile
$ echo 'eval "$(rbenv init -)"' >> ~/.bash_profile
$ source ~/.bash_profile

  3. Install Ruby:

    $ rbenv install 3.0.0
$ rbenv global 3.0.0

  4. Verify the installation:

    $ ruby -v

Installing Node.js and Yarn

Next, install Node.js and Yarn:

  1. Install Node.js:

    $ brew install node

  2. Install Yarn:

    $ brew install yarn

Installing SQLite3

SQLite3 is the default database for Rails development. Install it using Homebrew:

$ brew install sqlite3

Installing Rails

With Ruby, Node.js, Yarn, and SQLite3 installed, you can now install Rails:

  1. Install Rails:

    $ gem install rails

  2. Verify the installation:

    $ rails -v

Creating a New Rails Application

Once Rails is installed, you can create a new Rails application:

  1. Create a new Rails application:

    $ rails new myapp

  2. Navigate to your application directory:

    $ cd myapp

  3. Start the Rails server:

    $ rails server

  4. Open your browser and navigate to http://localhost:3000 to see your new Rails application running.

Conclusion

Setting up Rails involves installing several dependencies and configuring your development environment. By following the steps outlined in this section, you should have a fully functional Rails setup ready for development. This foundation will allow you to start building powerful web applications using the Rails framework.
