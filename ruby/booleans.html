Introduction to Booleans

Booleans are a fundamental data type in Ruby, representing true or false values. They are essential for making decisions in your code through conditional statements and loops.

  • True: Represents a true value.
  • False: Represents a false value.

In Ruby, booleans are instances of the TrueClass and FalseClass.

Boolean Literals

In Ruby, the boolean literals are true and false.

is_ruby_fun = true
is_sky_green = false

puts is_ruby_fun  # Output: true
puts is_sky_green # Output: false

Boolean Expressions

Boolean expressions evaluate to either true or false. They often involve comparison operators.

Comparison Operators

  • == : Equal to
  • != : Not equal to
  • > : Greater than
  • < : Less than
  • >= : Greater than or equal to
  • <= : Less than or equal to
a = 10
b = 20

puts a == b  # Output: false
puts a != b  # Output: true
puts a > b   # Output: false
puts a < b   # Output: true
puts a >= 10 # Output: true
puts b <= 15 # Output: false

Logical Operators

Logical operators are used to combine multiple boolean expressions.

  • && : Logical AND
  • || : Logical OR
  • ! : Logical NOT
x = true
y = false

puts x && y  # Output: false
puts x || y  # Output: true
puts !x      # Output: false
puts !y      # Output: true

Conditional Statements

Conditional statements use boolean expressions to control the flow of the program.

if Statement

age = 18

if age >= 18
  puts "You are an adult."
else
  puts "You are a minor."
end

unless Statement

The unless statement is the opposite of if.

age = 15

unless age >= 18
  puts "You are a minor."
else
  puts "You are an adult."
end

Ternary Operator

The ternary operator is a shorthand for if-else.

age = 20
message = age >= 18 ? "You are an adult." : "You are a minor."
puts message  # Output: You are an adult.

Boolean Methods

Ruby provides several methods that return boolean values.

.nil?

Checks if an object is nil.

name = nil
puts name.nil?  # Output: true

name = "Alice"
puts name.nil?  # Output: false

.empty?

Checks if a string or array is empty.

str = ""
puts str.empty?  # Output: true

arr = []
puts arr.empty?  # Output: true

.include?

Checks if a string or array includes a specific element.

str = "Hello, world!"
puts str.include?("world")  # Output: true

arr = [1, 2, 3, 4, 5]
puts arr.include?(3)  # Output: true

Summary

Booleans are a crucial part of Ruby programming, enabling decision-making and control flow. Understanding boolean literals, expressions, and logical operators is essential for writing effective Ruby code. Conditional statements like if, unless, and the ternary operator allow you to execute code based on boolean conditions. Additionally, Ruby provides several methods that return boolean values, helping you to write more expressive and readable code.
