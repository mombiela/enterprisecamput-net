Introduction to Ranges

Ranges in Ruby represent a sequence of values, which can be numbers, letters, or objects. They are commonly used for iteration and defining intervals.

Key Concepts

  • Range Operators: .. (inclusive) and ... (exclusive).
  • Range Methods: Methods to manipulate and query ranges.
  • Use Cases: Iteration, condition checking, and slicing.

Creating Ranges

Ranges can be created using the .. and ... operators.

Inclusive Ranges

An inclusive range includes the end value.

inclusive_range = 1..5
puts inclusive_range.to_a  # Output: [1, 2, 3, 4, 5]

Exclusive Ranges

An exclusive range excludes the end value.

exclusive_range = 1...5
puts exclusive_range.to_a  # Output: [1, 2, 3, 4]

Range Methods

Ruby provides several methods to work with ranges.

`to_a` Method

Converts a range to an array.

range = 1..3
array = range.to_a
puts array.inspect  # Output: [1, 2, 3]

`include?` Method

Checks if a value is within the range.

range = 1..5
puts range.include?(3)  # Output: true
puts range.include?(6)  # Output: false

`each` Method

Iterates over each element in the range.

(1..3).each do |number|
  puts number
end
# Output:
# 1
# 2
# 3

Advanced Range Features

Ranges in Ruby can be used in more advanced ways, such as with custom objects and in conditional statements.

Custom Objects in Ranges

You can define ranges with custom objects if they implement the <=> (spaceship) operator.

class Custom
  attr_reader :value

  def initialize(value)
    @value = value
  end

  def <=>(other)
    @value <=> other.value
  end
end

range = Custom.new(1)..Custom.new(3)
puts range.include?(Custom.new(2))  # Output: true

Ranges in Conditional Statements

Ranges can be used in conditional statements to check if a value falls within a specific range.

age = 25

case age
when 0..12
  puts "Child"
when 13..19
  puts "Teenager"
when 20..64
  puts "Adult"
else
  puts "Senior"
end
# Output: Adult

Summary

Ranges in Ruby are a powerful tool for representing sequences and intervals. They can be created using inclusive (..) and exclusive (...) operators, and offer various methods for manipulation and iteration. Understanding ranges is essential for effective Ruby programming, especially for tasks involving iteration, condition checking, and custom object handling.
