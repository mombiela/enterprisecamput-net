Assignment Operators in Ruby
Operator Precedence ►

Assignment operators are used to assign values to variables. In Ruby, assignment operators not only include the basic = operator but also a variety of compound operators that perform an operation and assignment in one step.

Basic Assignment Operator

The most fundamental assignment operator is the = operator.

x = 10
y = "Hello, Ruby!"
  • x = 10 assigns the integer value 10 to the variable x.
  • y = "Hello, Ruby!" assigns the string "Hello, Ruby!" to the variable y.

Compound Assignment Operators

Compound assignment operators combine an arithmetic operation with assignment. These operators are shorthand for performing an operation and then assigning the result to a variable.

Addition Assignment (`+=`)

x = 5
x += 3  # Equivalent to x = x + 3
puts x  # Output: 8

Subtraction Assignment (`-=`)

x = 5
x -= 2  # Equivalent to x = x - 2
puts x  # Output: 3

Multiplication Assignment (`*=`)

x = 5
x *= 4  # Equivalent to x = x * 4
puts x  # Output: 20

Division Assignment (`/=`)

x = 20
x /= 4  # Equivalent to x = x / 4
puts x  # Output: 5

Modulus Assignment (`%=`)

x = 20
x %= 3  # Equivalent to x = x % 3
puts x  # Output: 2

Exponentiation Assignment (`**=`)

x = 2
x **= 3  # Equivalent to x = x ** 3
puts x  # Output: 8

Bitwise Assignment Operators

Ruby also supports bitwise assignment operators, which perform bitwise operations and then assign the result to the variable.

Bitwise AND Assignment (`&=`)

x = 6  # Binary: 110
x &= 3  # Binary: 011, Result: 010 (2 in decimal)
puts x  # Output: 2

Bitwise OR Assignment (`|=`)

x = 6  # Binary: 110
x |= 3  # Binary: 011, Result: 111 (7 in decimal)
puts x  # Output: 7

Bitwise XOR Assignment (`^=`)

x = 6  # Binary: 110
x ^= 3  # Binary: 011, Result: 101 (5 in decimal)
puts x  # Output: 5

Bitwise Left Shift Assignment (`<<=`)

x = 3  # Binary: 11
x <<= 2  # Binary: 1100 (12 in decimal)
puts x  # Output: 12

Bitwise Right Shift Assignment (`>>=`)

x = 12  # Binary: 1100
x >>= 2  # Binary: 11 (3 in decimal)
puts x  # Output: 3

Comparison of Assignment Operators

Conclusion

Assignment operators in Ruby are essential tools for managing and manipulating variable values efficiently. Understanding both basic and compound assignment operators allows you to write more concise and readable code. Practice using these operators in different scenarios to become comfortable with their functionality and applications.
