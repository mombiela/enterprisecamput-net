Introduction to Case Statements

Case statements in Ruby are a powerful tool for controlling the flow of your program based on different conditions. They are similar to switch statements in other programming languages and provide a more readable alternative to multiple if-elsif-else statements.

Basic Syntax

The basic syntax of a case statement in Ruby is as follows:

case expression
when value1
  # code to execute if expression == value1
when value2
  # code to execute if expression == value2
else
  # code to execute if none of the above conditions are met
end

Example

Here's a simple example to illustrate the use of a case statement:

day = "Saturday"

case day
when "Monday"
  puts "Start of the work week"
when "Friday"
  puts "End of the work week"
when "Saturday", "Sunday"
  puts "Weekend!"
else
  puts "Midweek day"
end

In this example:

  • If day is "Monday", it prints "Start of the work week".
  • If day is "Friday", it prints "End of the work week".
  • If day is "Saturday" or "Sunday", it prints "Weekend!".
  • For any other value of day, it prints "Midweek day".

Advanced Usage

Using Ranges

You can also use ranges in case statements to handle a range of values.

score = 85

case score
when 90..100
  puts "Excellent"
when 75..89
  puts "Good"
when 50..74
  puts "Pass"
else
  puts "Fail"
end

Using Regular Expressions

Case statements can also match regular expressions.

name = "Alice"

case name
when /Alic/
  puts "Name contains 'Alic'"
when /^A/
  puts "Name starts with 'A'"
else
  puts "Name does not match any pattern"
end

Using Multiple Conditions

You can specify multiple conditions in a single when clause by separating them with commas.

fruit = "Apple"

case fruit
when "Apple", "Banana", "Cherry"
  puts "This is a common fruit"
else
  puts "This is an uncommon fruit"
end

Best Practices

  • Readability: Use case statements to improve the readability of your code, especially when dealing with multiple conditions.
  • Simplicity: Keep the conditions simple and avoid overly complex logic within a single case statement.
  • Default Case: Always include an else clause to handle unexpected values.

Summary

Case statements in Ruby provide a clean and readable way to handle multiple conditions. They can be used with simple values, ranges, regular expressions, and multiple conditions. By following best practices, you can make your code more maintainable and easier to understand.

Key Points

  • Case statements are an alternative to multiple if-elsif-else statements.
  • They improve code readability and maintainability.
  • They can handle ranges, regular expressions, and multiple conditions.
  • Always include an else clause to handle unexpected values.

By mastering case statements, you can write more efficient and readable Ruby code.
