Introduction

In this section, we will cover the steps required to install Hadoop on your local machine. This guide will help you set up a single-node Hadoop cluster, which is ideal for learning and development purposes.

Prerequisites

Before installing Hadoop, ensure you have the following:

  • A machine running a Unix-based operating system (Linux, macOS).
  • Java Development Kit (JDK) installed (version 8 or higher).
  • SSH installed and configured.

Step-by-Step Installation Guide

1. Install Java

Hadoop requires Java to run. You can check if Java is installed by running:

java -version

If Java is not installed, you can install it using the following commands:

For Ubuntu/Debian:

sudo apt update
sudo apt install openjdk-8-jdk

For CentOS/RHEL:

sudo yum install java-1.8.0-openjdk

2. Download Hadoop

Download the latest stable release of Hadoop from the official Apache Hadoop website:

wget https://downloads.apache.org/hadoop/common/hadoop-3.3.1/hadoop-3.3.1.tar.gz

3. Extract Hadoop

Extract the downloaded tarball to a directory of your choice:

tar -xzvf hadoop-3.3.1.tar.gz
mv hadoop-3.3.1 /usr/local/hadoop

4. Configure Environment Variables

Add Hadoop and Java to your system's PATH by editing the .bashrc file:

nano ~/.bashrc

Add the following lines at the end of the file:

export JAVA_HOME=/usr/lib/jvm/java-8-openjdk-amd64
export HADOOP_HOME=/usr/local/hadoop
export PATH=$PATH:$HADOOP_HOME/bin:$HADOOP_HOME/sbin

Apply the changes:

source ~/.bashrc

5. Configure Hadoop

Edit the Hadoop configuration files located in the $HADOOP_HOME/etc/hadoop directory.

core-site.xml

<configuration>
    <property>
        <name>fs.defaultFS</name>
        <value>hdfs://localhost:9000</value>
    </property>
</configuration>

hdfs-site.xml

<configuration>
    <property>
        <name>dfs.replication</name>
        <value>1</value>
    </property>
    <property>
        <name>dfs.namenode.name.dir</name>
        <value>file:///usr/local/hadoop/hadoop_data/hdfs/namenode</value>
    </property>
    <property>
        <name>dfs.datanode.data.dir</name>
        <value>file:///usr/local/hadoop/hadoop_data/hdfs/datanode</value>
    </property>
</configuration>

mapred-site.xml

<configuration>
    <property>
        <name>mapreduce.framework.name</name>
        <value>yarn</value>
    </property>
</configuration>

yarn-site.xml

<configuration>
    <property>
        <name>yarn.nodemanager.aux-services</name>
        <value>mapreduce_shuffle</value>
    </property>
</configuration>

6. Format the Hadoop Filesystem

Before starting Hadoop, format the Hadoop filesystem (HDFS):

hdfs namenode -format

7. Start Hadoop

Start the Hadoop daemons:

start-dfs.sh
start-yarn.sh

8. Verify Installation

Check if the Hadoop daemons are running:

jps

You should see the following processes:

  • NameNode
  • DataNode
  • ResourceManager
  • NodeManager

Conclusion

You have successfully installed Hadoop on your local machine. This single-node setup is perfect for learning and experimenting with Hadoop. In the next sections, we will explore how to use Hadoop for various data processing tasks.
