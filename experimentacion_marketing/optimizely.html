Introduction to Optimizely

Optimizely is an experimentation and optimization platform that allows marketing and development teams to conduct A/B and multivariate tests to improve user experience and increase conversions. In this topic, we will learn how to use Optimizely to conduct effective marketing experiments.

Main Features of Optimizely

  • A/B and Multivariate Testing: Allows comparing two or more versions of a web page or application to determine which one performs better.
  • Advanced Segmentation: Offers the ability to segment audiences based on behaviors, demographics, and other criteria.
  • Personalization: Allows real-time content personalization for different user segments.
  • Analytics and Reporting: Provides analytics tools to measure the impact of experiments and make data-driven decisions.
  • Integrations: Integrates with various marketing and analytics tools like Google Analytics, Adobe Analytics, and more.

Initial Setup

Create an Account and Project in Optimizely

  1. Sign Up: Sign up on Optimizely and create an account.
  2. Create Project: Once registered, create a new project from the dashboard.
  3. Snippet Installation: Add the Optimizely snippet to your website or application to enable experimentation.
<!-- Optimizely Snippet -->
<script src="https://cdn.optimizely.com/js/PROJECT_ID.js"></script>

Setting Up Goals

Define the goals you want to measure, such as button clicks, conversions, or time on page.

  1. Navigate to the Goals section.
  2. Create New Goal: Select the type of goal (e.g., button click).
  3. Configure Details: Enter the necessary details, such as the CSS selector of the element to be measured.

Creating Experiments

Basic A/B Test

  1. Create Experiment: In the dashboard, select "Create Experiment".
  2. Define Variants: Create the variants you want to test (e.g., different button texts).
  3. Configure Segmentation: Define the audience that will see the experiment.
  4. Launch Experiment: Publish the experiment and start collecting data.
// Example code to change the text of a button
optimizely.push(["activate", {
  experimentId: "EXPERIMENT_ID",
  variationId: "VARIATION_ID"
}]);

document.querySelector('#my-button').innerText = 'New Text';

Multivariate Testing

  1. Create Multivariate Experiment: Select the multivariate test option.
  2. Define Factors and Levels: Specify the different elements and their variations.
  3. Configure and Launch: Configure segmentation and launch the experiment.

Analyzing Results

Data Interpretation

  • Conversion Rate: Measures the percentage of visitors who complete the goal.
  • Statistical Significance: Ensure that the results are statistically significant before making decisions.
  • Variant Comparison: Use Optimizely's analysis tools to compare the performance of different variants.

Reporting

  • Experiment Summary: Review the summary of all experiments conducted.
  • Goal Details: Analyze the specific details of each measured goal.
  • Data Export: Export data for deeper analysis in other tools.

Conclusion

Optimizely is a powerful tool for marketing experimentation, allowing teams to make data-driven decisions and continuously optimize the user experience. By mastering Optimizely's features and functionalities, marketing professionals can significantly improve their strategies and results.

Always remember to clearly define your goals, properly segment your audience, and carefully analyze the results to get the most benefit from your experiments.
