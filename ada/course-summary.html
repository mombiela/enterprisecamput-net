Example: Hello World in Ada

with Ada.Text_IO; use Ada.Text_IO;

procedure Hello is
begin
   Put_Line("Hello, World!");
end Hello;

Explanation:

  • with Ada.Text_IO; use Ada.Text_IO;: Importing the Ada Text_IO package for input/output operations.
  • procedure Hello is: Declaring a procedure named Hello.
  • begin ... end Hello;: The body of the procedure, where Put_Line prints "Hello, World!" to the console.

Example: Conditional Statement

procedure Check_Number is
   Number : Integer := 10;
begin
   if Number > 0 then
      Put_Line("Number is positive");
   elsif Number < 0 then
      Put_Line("Number is negative");
   else
      Put_Line("Number is zero");
   end if;
end Check_Number;

Explanation:

  • if Number > 0 then ... end if;: Checks if Number is positive, negative, or zero and prints the corresponding message.

Example: Exception Handling

procedure Divide is
   A, B, Result : Integer;
begin
   A := 10;
   B := 0;
   begin
      Result := A / B;
   exception
      when Constraint_Error =>
         Put_Line("Division by zero is not allowed");
   end;
end Divide;

Explanation:

  • begin ... exception ... end;: Attempts to divide A by B and catches a Constraint_Error if B is zero.

Example: Tasking

task type Worker is
end Worker;

task body Worker is
begin
   Put_Line("Worker task is running");
end Worker;

procedure Main is
   W : Worker;
begin
   null;
end Main;

Explanation:

  • task type Worker is ... end Worker;: Declares a task type named Worker.
  • task body Worker is ... end Worker;: Defines the behavior of the Worker task.
  • W : Worker;: Instantiates a Worker task in the Main procedure.
