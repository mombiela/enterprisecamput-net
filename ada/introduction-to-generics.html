Introduction to Generics
Generic Procedures and Functions ►

Generics in Ada provide a powerful way to create flexible and reusable code components. They allow you to write algorithms and data structures that can operate on any data type, making your code more modular and easier to maintain.

Key Concepts

  • Generic Units: These are templates for creating subprograms or packages that can operate on different data types.
  • Generic Parameters: These are the placeholders for the types or values that will be specified when the generic unit is instantiated.
  • Instantiation: This is the process of creating a specific instance of a generic unit by providing actual parameters.

Generic Units

In Ada, you can create generic procedures, functions, and packages. Here's a basic example of a generic procedure:

generic
   type Item_Type is private;
procedure Swap (X, Y : in out Item_Type);
  • generic keyword indicates the start of a generic unit.
  • type Item_Type is private; declares a generic type parameter.
  • procedure Swap is the generic procedure that will operate on the Item_Type.

Instantiating Generics

To use a generic unit, you need to instantiate it with actual parameters. Here's how you can instantiate the Swap procedure for integers:

procedure Swap_Integer is new Swap (Item_Type => Integer);

Now you can use Swap_Integer to swap integer values:

declare
   A, B : Integer := 10;
begin
   Swap_Integer (A, B);
   -- Now A = 10 and B = 10
end;

Generic Packages

Generic packages allow you to create reusable components that can be customized with different types or values. Here's an example of a generic package for a stack:

generic
   type Element_Type is private;
package Generic_Stack is
   procedure Push (Item : in Element_Type);
   procedure Pop (Item : out Element_Type);
end Generic_Stack;

To instantiate this package for integers:

package Integer_Stack is new Generic_Stack (Element_Type => Integer);

Now you can use Integer_Stack to create and manipulate stacks of integers.

Example: Generic Sorting Algorithm

Let's create a generic package for a sorting algorithm:

generic
   type Element_Type is private;
   with function "<" (Left, Right : Element_Type) return Boolean;
package Generic_Sort is
   procedure Sort (Array : in out array (Positive range <>) of Element_Type);
end Generic_Sort;

To instantiate and use this package for sorting an array of integers:

package Integer_Sort is new Generic_Sort (Element_Type => Integer, "<" => "<");

declare
   Arr : array (1 .. 5) of Integer := (5, 3, 4, 1, 2);
begin
   Integer_Sort.Sort (Arr);
   -- Now Arr is sorted: (1, 2, 3, 4, 5)
end;

Conclusion

Generics in Ada provide a robust mechanism for creating flexible and reusable code components. By understanding how to define and instantiate generic units, you can write more modular and maintainable code. Practice creating and using generics with different types and see how they can simplify your programming tasks.
