Overview

Object-Oriented Programming (OOP) is a programming paradigm that uses "objects" to design applications and computer programs. Ada, a statically typed, structured, and imperative programming language, supports OOP principles such as encapsulation, inheritance, and polymorphism.

Key Concepts of OOP

  • Encapsulation: Bundling data and methods that operate on the data within one unit, e.g., a class.
  • Inheritance: Mechanism by which one class can inherit attributes and methods from another class.
  • Polymorphism: Ability to process objects differently based on their data type or class.

Encapsulation in Ada

Encapsulation in Ada is achieved using packages and private types. Here's a simple example:

-- Define a package for a Stack
package Stack_Pkg is
    type Stack is private;
    procedure Push(S : in out Stack; Element : Integer);
    procedure Pop(S : in out Stack; Element : out Integer);
    function Is_Empty(S : Stack) return Boolean;
private
    type Stack is record
        Elements : array(1..100) of Integer;
        Top : Integer := 0;
    end record;
end Stack_Pkg;

package body Stack_Pkg is
    procedure Push(S : in out Stack; Element : Integer) is
    begin
        S.Top := S.Top + 1;
        S.Elements(S.Top) := Element;
    end Push;

    procedure Pop(S : in out Stack; Element : out Integer) is
    begin
        Element := S.Elements(S.Top);
        S.Top := S.Top - 1;
    end Pop;

    function Is_Empty(S : Stack) return Boolean is
    begin
        return S.Top = 0;
    end Is_Empty;
end Stack_Pkg;

Explanation:

  • Package Specification: Defines the interface of the stack.
  • Private Type: The actual implementation of the stack is hidden from the user.
  • Package Body: Implements the methods defined in the package specification.

Inheritance in Ada

Inheritance allows a new type to be defined based on an existing type. Here's an example:

package Shape_Pkg is
    type Shape is tagged record
        X, Y : Integer;
    end record;

    procedure Move(S : in out Shape; New_X, New_Y : Integer);

    type Circle is new Shape with record
        Radius : Integer;
    end record;

    procedure Resize(C : in out Circle; New_Radius : Integer);
end Shape_Pkg;

package body Shape_Pkg is
    procedure Move(S : in out Shape; New_X, New_Y : Integer) is
    begin
        S.X := New_X;
        S.Y := New_Y;
    end Move;

    procedure Resize(C : in out Circle; New_Radius : Integer) is
    begin
        C.Radius := New_Radius;
    end Resize;
end Shape_Pkg;

Explanation:

  • Tagged Record: Shape is a tagged record, which means it can be extended.
  • Derived Type: Circle is derived from Shape and adds a new attribute Radius.
  • Method Overriding: Methods like Move and Resize can be defined for both Shape and Circle.

Polymorphism in Ada

Polymorphism in Ada is achieved through dynamic dispatching. Here's an example:

with Ada.Text_IO; use Ada.Text_IO;

package Shape_Pkg is
    type Shape is tagged record
        X, Y : Integer;
    end record;

    procedure Move(S : in out Shape; New_X, New_Y : Integer) is abstract;
    procedure Display(S : in Shape) is abstract;

    type Circle is new Shape with record
        Radius : Integer;
    end record;

    procedure Move(S : in out Circle; New_X, New_Y : Integer);
    procedure Display(S : in Circle);
end Shape_Pkg;

package body Shape_Pkg is
    procedure Move(S : in out Circle; New_X, New_Y : Integer) is
    begin
        S.X := New_X;
        S.Y := New_Y;
    end Move;

    procedure Display(S : in Circle) is
    begin
        Put_Line("Circle at (" & Integer'Image(S.X) & ", " & Integer'Image(S.Y) & ") with radius " & Integer'Image(S.Radius));
    end Display;
end Shape_Pkg;

with Shape_Pkg; use Shape_Pkg;

procedure Main is
    C : Circle := (X => 0, Y => 0, Radius => 10);
begin
    C.Move(5, 5);
    C.Display;
end Main;

Explanation:

  • Abstract Methods: Move and Display are abstract methods in Shape.
  • Dynamic Dispatching: When Move and Display are called on a Circle, the specific implementations for Circle are used.

Conclusion

In this topic, we introduced the fundamental concepts of Object-Oriented Programming in Ada, including encapsulation, inheritance, and polymorphism. We provided examples to illustrate how these concepts are implemented in Ada. Understanding these principles is crucial for designing robust and maintainable software systems. As you progress, you will see how these concepts are applied in more complex scenarios.
