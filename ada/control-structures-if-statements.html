Introduction

In Ada, control structures are essential for directing the flow of a program. One of the most fundamental control structures is the if statement, which allows a program to make decisions based on conditions. This section will cover the basics of if statements, including syntax, usage, and examples.

Basic Syntax

The if statement in Ada is used to execute a block of code only if a specified condition is true. The basic syntax is as follows:

if condition then
   -- statements to execute if condition is true
end if;

Example

with Ada.Text_IO; use Ada.Text_IO;

procedure Check_Number is
   Number : Integer := 10;
begin
   if Number > 5 then
      Put_Line("Number is greater than 5");
   end if;
end Check_Number;

In this example, the program checks if the variable Number is greater than 5. If the condition is true, it prints "Number is greater than 5".

Adding Else and Elsif

To handle multiple conditions, you can use else and elsif clauses.

Syntax with Else

if condition then
   -- statements to execute if condition is true
else
   -- statements to execute if condition is false
end if;

Syntax with Elsif

if condition1 then
   -- statements to execute if condition1 is true
elsif condition2 then
   -- statements to execute if condition2 is true
else
   -- statements to execute if none of the above conditions are true
end if;

Example with Else and Elsif

with Ada.Text_IO; use Ada.Text_IO;

procedure Check_Number is
   Number : Integer := 3;
begin
   if Number > 5 then
      Put_Line("Number is greater than 5");
   elsif Number = 5 then
      Put_Line("Number is equal to 5");
   else
      Put_Line("Number is less than 5");
   end if;
end Check_Number;

In this example, the program checks multiple conditions:

  • If Number is greater than 5, it prints "Number is greater than 5".
  • If Number is equal to 5, it prints "Number is equal to 5".
  • If none of the above conditions are true, it prints "Number is less than 5".

Nested If Statements

You can nest if statements within each other to handle more complex conditions.

Example of Nested If Statements

with Ada.Text_IO; use Ada.Text_IO;

procedure Check_Number is
   Number : Integer := 7;
begin
   if Number > 0 then
      if Number < 10 then
         Put_Line("Number is between 1 and 9");
      else
         Put_Line("Number is 10 or greater");
      end if;
   else
      Put_Line("Number is 0 or negative");
   end if;
end Check_Number;

In this example, the program first checks if Number is greater than 0. If true, it then checks if Number is less than 10. Depending on the result, it prints the appropriate message.

Best Practices

  • Readability: Ensure your if statements are easy to read and understand.
  • Avoid Deep Nesting: Deeply nested if statements can be hard to follow. Consider using logical operators or breaking down complex conditions into simpler ones.
  • Consistent Formatting: Use consistent indentation and formatting to make your code more readable.

Summary

The if statement is a powerful tool in Ada for making decisions based on conditions. By understanding its basic syntax, as well as how to use else, elsif, and nested if statements, you can write more complex and flexible programs. Always aim for readability and maintainability in your code to ensure it is easy to understand and modify.
