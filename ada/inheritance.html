Inheritance in Ada
Polymorphism ►

Inheritance is a fundamental concept in object-oriented programming (OOP) that allows a new class to inherit properties and behaviors (methods) from an existing class. In Ada, inheritance is achieved through the use of tagged types and type extensions.

Key Concepts

  • Tagged Types: These are types that can be extended to create new types.
  • Type Extension: The mechanism by which a new type inherits properties and behaviors from an existing type.
  • Overriding: The ability to redefine methods in the derived type.

Tagged Types

Tagged types are the foundation of inheritance in Ada. A tagged type is a type that can be extended to create new types.

Example

package Shapes is
    type Shape is tagged record
        X, Y : Float;
    end record;

    procedure Move(S : in out Shape; New_X, New_Y : Float);
end Shapes;

In this example, Shape is a tagged type with two fields, X and Y. The Move procedure is a method that can be applied to any Shape.

Type Extension

Type extension allows you to create a new type that inherits the properties and behaviors of an existing type.

Example

package Shapes is
    type Circle is new Shape with record
        Radius : Float;
    end record;

    procedure Move(S : in out Circle; New_X, New_Y : Float);
end Shapes;

Here, Circle is a new type that extends Shape by adding a Radius field. The Move procedure is overridden to handle Circle specifically.

Overriding Methods

When you extend a type, you can override its methods to provide new behavior.

Example

package body Shapes is
    procedure Move(S : in out Shape; New_X, New_Y : Float) is
    begin
        S.X := New_X;
        S.Y := New_Y;
    end Move;

    procedure Move(S : in out Circle; New_X, New_Y : Float) is
    begin
        S.X := New_X;
        S.Y := New_Y;
        -- Additional behavior for Circle can be added here
    end Move;
end Shapes;

In this example, the Move procedure is overridden for the Circle type to provide specific behavior for circles.

Polymorphism

Polymorphism allows you to use a derived type wherever the base type is expected. This is useful for writing generic code that can operate on different types of objects.

Example

procedure Draw(Shape : in Shape'Class) is
begin
    -- Code to draw the shape
end Draw;

procedure Draw_All(Shapes : in Shape'Class) is
begin
    for S of Shapes loop
        Draw(S);
    end loop;
end Draw_All;

In this example, Draw is a procedure that can operate on any type derived from Shape. The Draw_All procedure demonstrates how to use polymorphism to draw a collection of shapes.

Conclusion

Inheritance in Ada is a powerful feature that allows you to create new types based on existing ones, promoting code reuse and extensibility. By understanding tagged types, type extension, method overriding, and polymorphism, you can effectively use inheritance to build robust and maintainable Ada programs.
