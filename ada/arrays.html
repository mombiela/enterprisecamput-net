Introduction to Arrays

Arrays are a fundamental data structure in Ada that allow you to store and manipulate collections of elements. Each element in an array is identified by an index, and all elements must be of the same type.

Key Concepts

  • Indexing: Arrays in Ada are indexed, meaning each element is accessed using an index.
  • Bounds: Arrays have defined bounds, specifying the range of valid indices.
  • Types: All elements in an array must be of the same type.
  • Static and Dynamic Arrays: Arrays can be either static (fixed size) or dynamic (size can change).

Declaring Arrays

In Ada, arrays are declared by specifying the type of elements and the range of indices.

Static Arrays

Static arrays have a fixed size defined at compile-time.

-- Declaration of a static array with indices from 1 to 5
type Index_Type is range 1 .. 5;
type Array_Type is array (Index_Type) of Integer;

-- Initializing the array
My_Array : Array_Type := (1, 2, 3, 4, 5);

Dynamic Arrays

Dynamic arrays can change size during runtime using the Unbounded_Array type from the Ada.Containers package.

with Ada.Containers.Indefinite_Vectors;
procedure Dynamic_Array_Example is
   package Integer_Vectors is new Ada.Containers.Indefinite_Vectors (Index_Type => Natural, Element_Type => Integer);
   My_Vector : Integer_Vectors.Vector;
begin
   -- Adding elements to the dynamic array
   Integer_Vectors.Append (My_Vector, 10);
   Integer_Vectors.Append (My_Vector, 20);
end Dynamic_Array_Example;

Accessing Array Elements

Elements in an array are accessed using their indices.

-- Accessing elements in a static array
My_Array(1) := 10;
Put_Line (Integer'Image (My_Array(1))); -- Output: 10

Iterating Over Arrays

You can iterate over arrays using loops.

Using a `for` Loop

-- Iterating over a static array
for Index in My_Array'Range loop
   Put_Line (Integer'Image (My_Array(Index)));
end loop;

Using `while` Loop

-- Iterating over a dynamic array
Index : Natural := My_Vector.First_Index;
while Index /= My_Vector.No_Element loop
   Put_Line (Integer'Image (My_Vector.Element (Index)));
   Index := My_Vector.Next_Index (Index);
end loop;

Multidimensional Arrays

Ada supports multidimensional arrays, which are arrays of arrays.

-- Declaration of a 2D array
type Matrix_Type is array (1 .. 3, 1 .. 3) of Integer;
Matrix : Matrix_Type := ((1, 2, 3), (4, 5, 6), (7, 8, 9));

-- Accessing elements in a 2D array
Put_Line (Integer'Image (Matrix(2, 3))); -- Output: 6

Array Attributes

Ada provides several attributes to work with arrays.

  • 'First: Returns the first index of the array.
  • 'Last: Returns the last index of the array.
  • 'Length: Returns the number of elements in the array.
  • 'Range: Returns the range of indices.
Put_Line (Integer'Image (My_Array'First));  -- Output: 1
Put_Line (Integer'Image (My_Array'Last));   -- Output: 5
Put_Line (Integer'Image (My_Array'Length)); -- Output: 5

Summary

Arrays in Ada are versatile and powerful structures that allow you to store and manipulate collections of elements efficiently. Understanding how to declare, access, and iterate over arrays is crucial for effective programming in Ada. Whether you are working with static or dynamic arrays, Ada provides robust features and attributes to manage them effectively.
