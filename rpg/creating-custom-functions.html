Introduction

Creating custom functions is a fundamental skill in RPG (Report Program Generator) programming. Functions allow you to encapsulate reusable code, making your programs more modular, readable, and maintainable. This section will guide you through the process of creating custom functions in RPG, from basic to advanced levels.

Key Concepts

  • Function Definition: How to define a function in RPG.
  • Parameters: Passing data to functions.
  • Return Values: Returning data from functions.
  • Scope: Understanding local and global variables within functions.
  • Modularity: Breaking down complex tasks into simpler functions.

Basic Function Definition

In RPG, functions are defined using the DCL-PROC and END-PROC keywords. Here is a simple example of a function that adds two numbers:

DCL-PROC AddNumbers;
  DCL-PI AddNumbers INT(10);
    num1 INT(10);
    num2 INT(10);
  END-PI;

  DCL-S result INT(10);

  result = num1 + num2;

  RETURN result;
END-PROC;

Explanation

  • DCL-PROC AddNumbers;: Declares the start of a procedure named AddNumbers.
  • DCL-PI AddNumbers INT(10);: Declares the procedure interface, specifying that AddNumbers returns an integer of length 10.
  • num1 INT(10); and num2 INT(10);: Parameters of the function, both integers of length 10.
  • DCL-S result INT(10);: Declares a local variable result to store the sum.
  • result = num1 + num2;: Adds the two parameters and stores the result.
  • RETURN result;: Returns the result to the caller.
  • END-PROC;: Marks the end of the procedure.

Parameters and Return Values

Functions can take multiple parameters and return a single value. Here is an example of a function that calculates the area of a rectangle:

DCL-PROC CalculateArea;
  DCL-PI CalculateArea INT(10);
    length INT(10);
    width INT(10);
  END-PI;

  DCL-S area INT(10);

  area = length * width;

  RETURN area;
END-PROC;

Explanation

  • The function CalculateArea takes two parameters: length and width.
  • It calculates the area by multiplying length and width.
  • The result is stored in the local variable area and returned to the caller.

Scope of Variables

Understanding the scope of variables is crucial when working with functions. Variables declared within a function are local to that function and cannot be accessed outside of it.

DCL-PROC ExampleScope;
  DCL-PI ExampleScope;
  END-PI;

  DCL-S localVar INT(10);

  localVar = 100;

  RETURN;
END-PROC;

Explanation

  • localVar is a local variable and can only be accessed within the ExampleScope function.
  • Attempting to access localVar outside of ExampleScope will result in an error.

Modularity

Breaking down complex tasks into simpler functions makes your code more modular and easier to maintain. Here is an example of a program that uses multiple functions to perform a series of calculations:

DCL-PROC Main;
  DCL-PI Main;
  END-PI;

  DCL-S length INT(10) INZ(5);
  DCL-S width INT(10) INZ(10);
  DCL-S area INT(10);
  DCL-S perimeter INT(10);

  area = CalculateArea(length: width);
  perimeter = CalculatePerimeter(length: width);

  RETURN;
END-PROC;

DCL-PROC CalculateArea;
  DCL-PI CalculateArea INT(10);
    length INT(10);
    width INT(10);
  END-PI;

  DCL-S area INT(10);

  area = length * width;

  RETURN area;
END-PROC;

DCL-PROC CalculatePerimeter;
  DCL-PI CalculatePerimeter INT(10);
    length INT(10);
    width INT(10);
  END-PI;

  DCL-S perimeter INT(10);

  perimeter = 2 * (length + width);

  RETURN perimeter;
END-PROC;

Explanation

  • The Main procedure initializes length and width.
  • It calls CalculateArea and CalculatePerimeter to compute the area and perimeter of a rectangle.
  • Each function performs a specific task, making the code more modular and easier to understand.

Conclusion

Creating custom functions in RPG allows you to write more modular, maintainable, and reusable code. By understanding how to define functions, pass parameters, return values, and manage variable scope, you can break down complex tasks into simpler, more manageable pieces. Practice creating your own functions to become proficient in this essential programming skill.
