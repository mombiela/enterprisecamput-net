Interfacing with Other Languages
Security and Best Practices ►

Interfacing with other programming languages is a crucial skill for RPG (Report Program Generator) developers. This topic will guide you through the basics and advanced techniques of integrating RPG with other languages, such as C, Java, and SQL.

Introduction to Interfacing

Interfacing allows RPG programs to leverage functionalities from other languages, enhancing their capabilities and performance. This section covers the fundamental concepts and benefits of interfacing.

  • Key Concepts:
    • Interoperability: The ability of different systems and organizations to work together.
    • APIs (Application Programming Interfaces): Sets of routines, protocols, and tools for building software and applications.
    • Data Exchange: Methods for transferring data between different programming environments.

Calling C Functions from RPG

RPG can call C functions to perform tasks that are either not possible or inefficient in RPG. This section explains how to call C functions from RPG.

  • Steps to Call C Functions:

    1. Define the C Function Prototype in RPG: 
      D myCFunction     PR              10I 0 EXTPROC('myCFunction')
D  param1                         10I 0 VALUE
D  param2                         10I 0 VALUE
    2. Call the C Function in RPG: 
      C                   eval      result = myCFunction(10 : 20)

  • Example:

    // C function definition
int myCFunction(int a, int b) {
    return a + b;
}

    // RPG program calling the C function
D myCFunction     PR              10I 0 EXTPROC('myCFunction')
D  param1                         10I 0 VALUE
D  param2                         10I 0 VALUE

D result          S               10I 0

C                   eval      result = myCFunction(10 : 20)
C                   dsply     result
C                   eval      *inlr = *on

Integrating Java with RPG

Java offers a wide range of libraries and functionalities that can be utilized in RPG programs. This section covers the integration of Java with RPG.

  • Steps to Call Java Methods:

    1. Define the Java Class and Method: 
      public class MyJavaClass {
    public static int addNumbers(int a, int b) {
        return a + b;
    }
}
    2. Use RPG to Call the Java Method: 
      D MyJavaClass     PR              10I 0 EXTPROC(*JAVA : 'MyJavaClass' : 'addNumbers')
D  param1                         10I 0 VALUE
D  param2                         10I 0 VALUE

D result          S               10I 0

C                   eval      result = MyJavaClass(10 : 20)
C                   dsply     result
C                   eval      *inlr = *on

  • Example:

    // Java class definition
public class MyJavaClass {
    public static int addNumbers(int a, int b) {
        return a + b;
    }
}

    // RPG program calling the Java method
D MyJavaClass     PR              10I 0 EXTPROC(*JAVA : 'MyJavaClass' : 'addNumbers')
D  param1                         10I 0 VALUE
D  param2                         10I 0 VALUE

D result          S               10I 0

C                   eval      result = MyJavaClass(10 : 20)
C                   dsply     result
C                   eval      *inlr = *on

Using SQL in RPG

SQL can be embedded in RPG programs to perform database operations efficiently. This section explains how to use SQL within RPG.

  • Steps to Embed SQL in RPG:

    1. Include the SQL Precompiler Directive: 
      /EXEC SQL
    2. Write SQL Statements within the RPG Program: 
      /EXEC SQL
    SELECT COUNT(*) INTO :recordCount FROM myTable
/END-EXEC

  • Example:

    // RPG program with embedded SQL
D recordCount     S               10I 0

/EXEC SQL
    SELECT COUNT(*) INTO :recordCount FROM myTable
/END-EXEC

C                   dsply     recordCount
C                   eval      *inlr = *on

Conclusion

Interfacing RPG with other languages like C, Java, and SQL significantly enhances the functionality and performance of RPG applications. By understanding and applying these interfacing techniques, RPG developers can create more robust and versatile programs. Practice these examples and explore further to master the art of interfacing in RPG.
