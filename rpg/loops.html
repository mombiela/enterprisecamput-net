Loops: DO, FOR, and WHILE

Introduction

Loops are fundamental constructs in programming that allow you to execute a block of code multiple times. In RPG (Report Program Generator), loops help automate repetitive tasks, making your code more efficient and easier to manage. This section will cover the three primary types of loops in RPG: DO, FOR, and WHILE.

DO Loop

The DO loop in RPG is used to execute a block of code a specific number of times. It is particularly useful when you know in advance how many iterations are required.

Syntax

DCL VAR(&COUNTER) TYPE(*INT) VALUE(1)

DO &COUNTER = 1 TO 10
    /* Your code here */
    CHGVAR VAR(&COUNTER) VALUE(&COUNTER + 1)
ENDDO

Explanation

  • DCL VAR(&COUNTER) TYPE(*INT) VALUE(1): Declares a variable &COUNTER of type integer and initializes it to 1.
  • DO &COUNTER = 1 TO 10: Starts the loop, setting &COUNTER to 1 and running until &COUNTER is 10.
  • CHGVAR VAR(&COUNTER) VALUE(&COUNTER + 1): Increments the counter by 1.
  • ENDDO: Ends the loop.

Example

DCL VAR(&I) TYPE(*INT) VALUE(1)

DO &I = 1 TO 5
    SNDPGMMSG MSG('Iteration ' *CAT &I)
    CHGVAR VAR(&I) VALUE(&I + 1)
ENDDO

This example sends a program message for each iteration from 1 to 5.

FOR Loop

The FOR loop is another way to execute a block of code multiple times, often used when the number of iterations is known.

Syntax

FOR &COUNTER = 1 TO 10
    /* Your code here */
ENDFOR

Explanation

  • FOR &COUNTER = 1 TO 10: Starts the loop, setting &COUNTER to 1 and running until &COUNTER is 10.
  • ENDFOR: Ends the loop.

Example

DCL VAR(&I) TYPE(*INT) VALUE(1)

FOR &I = 1 TO 5
    SNDPGMMSG MSG('Iteration ' *CAT &I)
ENDFOR

This example sends a program message for each iteration from 1 to 5, similar to the DO loop example but using the FOR loop syntax.

WHILE Loop

The WHILE loop is used to execute a block of code as long as a specified condition is true. It is particularly useful when the number of iterations is not known in advance.

Syntax

DCL VAR(&COUNTER) TYPE(*INT) VALUE(1)

WHILE &COUNTER <= 10
    /* Your code here */
    CHGVAR VAR(&COUNTER) VALUE(&COUNTER + 1)
ENDWHILE

Explanation

  • DCL VAR(&COUNTER) TYPE(*INT) VALUE(1): Declares a variable &COUNTER of type integer and initializes it to 1.
  • WHILE &COUNTER <= 10: Starts the loop, running as long as &COUNTER is less than or equal to 10.
  • CHGVAR VAR(&COUNTER) VALUE(&COUNTER + 1): Increments the counter by 1.
  • ENDWHILE: Ends the loop.

Example

DCL VAR(&I) TYPE(*INT) VALUE(1)

WHILE &I <= 5
    SNDPGMMSG MSG('Iteration ' *CAT &I)
    CHGVAR VAR(&I) VALUE(&I + 1)
ENDWHILE

This example sends a program message for each iteration from 1 to 5, similar to the previous examples but using the WHILE loop syntax.

Comparison of Loops

| Loop Type | Use Case | Syntax Complexity | Flexibility | |-----------|----------|-------------------|-------------| | DO | Known number of iterations | Medium | Medium | | FOR | Known number of iterations | Simple | Low | | WHILE | Unknown number of iterations | Medium | High |

Conclusion

Understanding loops is crucial for efficient programming in RPG. The DO, FOR, and WHILE loops each have their own use cases and advantages:

  • DO Loop: Best when you know the number of iterations and need more control over the loop variable.
  • FOR Loop: Simplest syntax for a known number of iterations.
  • WHILE Loop: Most flexible, ideal for situations where the number of iterations is not known in advance.

By mastering these loops, you can write more efficient and readable RPG code. Practice using each type of loop in different scenarios to become proficient in their use.
