Introduction

Built-in functions in RPG (Report Program Generator) are pre-defined functions provided by the language to perform common tasks. These functions can simplify your code, make it more readable, and reduce the likelihood of errors. In this section, we will explore various built-in functions, understand their usage, and see how they can be applied in real-world scenarios.

Key Concepts

  • Definition: Built-in functions are predefined routines available in RPG that perform specific tasks.
  • Categories: These functions can be categorized into string manipulation, mathematical operations, date and time handling, and more.
  • Syntax: Understanding the syntax and parameters of each function is crucial for effective use.

Common Built-in Functions

String Manipulation Functions

String manipulation functions help in handling and modifying string data.

  • %TRIM: Removes leading and trailing spaces from a string.

    Dcl-S myString Char(20) Inz('  Hello World  ');
Dcl-S trimmedString Char(20);

trimmedString = %Trim(myString);
// trimmedString now contains 'Hello World'

  • %SUBST: Extracts a substring from a string.

    Dcl-S myString Char(20) Inz('Hello World');
Dcl-S subString Char(5);

subString = %Subst(myString: 1: 5);
// subString now contains 'Hello'

  • %SCAN: Searches for a substring within a string and returns its position.

    Dcl-S myString Char(20) Inz('Hello World');
Dcl-S position Int(10);

position = %Scan('World': myString);
// position now contains 7

Mathematical Functions

Mathematical functions perform arithmetic operations and other mathematical tasks.

  • %ABS: Returns the absolute value of a number.

    Dcl-S myNumber Int(10) Inz(-5);
Dcl-S absValue Int(10);

absValue = %Abs(myNumber);
// absValue now contains 5

  • %REM: Returns the remainder of a division operation.

    Dcl-S dividend Int(10) Inz(10);
Dcl-S divisor Int(10) Inz(3);
Dcl-S remainder Int(10);

remainder = %Rem(dividend: divisor);
// remainder now contains 1

Date and Time Functions

Date and time functions help in handling and manipulating date and time values.

  • %DATE: Converts a string to a date.

    Dcl-S dateString Char(10) Inz('2023-10-01');
Dcl-S myDate Date;

myDate = %Date(dateString);
// myDate now contains the date 2023-10-01

  • %DIFF: Calculates the difference between two dates.

    Dcl-S startDate Date Inz(D'2023-01-01');
Dcl-S endDate Date Inz(D'2023-10-01');
Dcl-S dateDiff Int(10);

dateDiff = %Diff(endDate: startDate: *D);
// dateDiff now contains 273 (days)

Examples and Exercises

Example 1: Using String Functions

Dcl-S fullName Char(50) Inz('  John Doe  ');
Dcl-S firstName Char(20);
Dcl-S lastName Char(20);
Dcl-S position Int(10);

// Trim the full name
fullName = %Trim(fullName);

// Find the position of the space
position = %Scan(' ': fullName);

// Extract first and last names
firstName = %Subst(fullName: 1: position - 1);
lastName = %Subst(fullName: position + 1);

// Output: firstName = 'John', lastName = 'Doe'

Exercise 1: Calculate Age

Write a program that calculates the age of a person given their birth date.

Dcl-S birthDate Date Inz(D'1990-05-15');
Dcl-S currentDate Date Inz(%Date());
Dcl-S age Int(10);

age = %Diff(currentDate: birthDate: *Y);
// Output: age will contain the number of years since birthDate

Conclusion

Built-in functions in RPG are powerful tools that can greatly enhance your programming efficiency. By understanding and utilizing these functions, you can write cleaner, more efficient code. Practice using these functions in various scenarios to become proficient in their application.
