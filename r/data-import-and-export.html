Introduction

Data import and export are fundamental skills in R programming, enabling you to read data from various sources and save your results. This section will guide you through the basics to advanced techniques of importing and exporting data in R.

Importing Data

Reading Data from CSV Files

CSV (Comma-Separated Values) files are one of the most common formats for data storage.

  • Basic CSV Import:

    data <- read.csv("path/to/your/file.csv")
    • read.csv() is a function that reads a file in table format and creates a data frame from it.
    • "path/to/your/file.csv" is the path to your CSV file.

  • Specifying Delimiters:

    data <- read.csv("path/to/your/file.csv", sep = ";")
    • sep parameter allows you to specify the delimiter used in your file.

Reading Data from Excel Files

Excel files are widely used in data analysis. R provides several packages to read Excel files, such as readxl and openxlsx.

  • Using readxl Package: 
    library(readxl)
data <- read_excel("path/to/your/file.xlsx", sheet = "Sheet1")
    • read_excel() function reads an Excel file.
    • sheet parameter specifies the sheet to read from.

Reading Data from Databases

R can connect to various databases using packages like DBI and RSQLite.

  • Connecting to SQLite Database: 
    library(DBI)
con <- dbConnect(RSQLite::SQLite(), "path/to/your/database.sqlite")
data <- dbGetQuery(con, "SELECT * FROM your_table")
dbDisconnect(con)
    • dbConnect() establishes a connection to the database.
    • dbGetQuery() executes a SQL query and returns the result as a data frame.
    • dbDisconnect() closes the connection.

Reading Data from Web

R can also read data directly from the web using functions like read.csv() for CSV files hosted online.

  • Reading CSV from URL: 
    url <- "http://example.com/data.csv"
data <- read.csv(url)

Exporting Data

Writing Data to CSV Files

Exporting data to CSV is straightforward with the write.csv() function.

  • Basic CSV Export: 
    write.csv(data, "path/to/your/output.csv")
    • write.csv() writes a data frame to a CSV file.
    • "path/to/your/output.csv" is the path where the file will be saved.

Writing Data to Excel Files

To write data to Excel files, you can use the writexl or openxlsx packages.

  • Using writexl Package: 
    library(writexl)
write_xlsx(data, "path/to/your/output.xlsx")
    • write_xlsx() writes a data frame to an Excel file.

Writing Data to Databases

You can also export data to databases using the DBI package.

  • Writing to SQLite Database: 
    library(DBI)
con <- dbConnect(RSQLite::SQLite(), "path/to/your/database.sqlite")
dbWriteTable(con, "your_table", data)
dbDisconnect(con)
    • dbWriteTable() writes a data frame to a database table.

Writing Data to Web

While R does not directly support writing data to web locations, you can use APIs or other web services to upload data.

Summary

In this section, we covered the essential techniques for importing and exporting data in R:

  • Importing Data:

    • Reading from CSV files using read.csv().
    • Reading from Excel files using readxl package.
    • Reading from databases using DBI and RSQLite.
    • Reading from web using URLs.

  • Exporting Data:

    • Writing to CSV files using write.csv().
    • Writing to Excel files using writexl package.
    • Writing to databases using DBI and RSQLite.

Mastering these skills will enable you to handle data from various sources and formats, making your data analysis workflow more efficient and versatile.
