Introduction to Base R Graphics

Base R graphics provide a powerful and flexible system for creating a wide variety of plots and visualizations directly within R. This section will introduce you to the fundamental concepts and functions used in base R graphics.

  • Key Concepts:
    • Graphics Devices: The medium where the plot is displayed (e.g., screen, file).
    • Plot Functions: Functions used to create different types of plots.
    • Plot Customization: Modifying the appearance of plots using various parameters.

Basic Plotting Functions

Base R offers several functions to create basic plots. These functions are easy to use and provide a good starting point for beginners.

  • Plotting a Simple Scatter Plot:

    # Generate some sample data
x <- 1:10
y <- x^2

# Create a scatter plot
plot(x, y, main="Scatter Plot", xlab="X-axis", ylab="Y-axis")

    Explanation:

    • plot(x, y): Creates a scatter plot with x and y values.
    • main, xlab, ylab: Add a title and labels to the axes.

  • Line Plot:

    # Create a line plot
plot(x, y, type="l", main="Line Plot", xlab="X-axis", ylab="Y-axis")

    Explanation:

    • type="l": Specifies that the plot should be a line plot.

  • Bar Plot:

    # Create a bar plot
barplot(y, names.arg=x, main="Bar Plot", xlab="X-axis", ylab="Y-axis")

    Explanation:

    • barplot(y): Creates a bar plot with y values.
    • names.arg=x: Labels the bars with x values.

Customizing Plots

Customizing plots allows you to enhance the readability and aesthetics of your visualizations.

  • Changing Colors and Plot Symbols:

    # Scatter plot with custom colors and symbols
plot(x, y, col="blue", pch=19, main="Custom Scatter Plot", xlab="X-axis", ylab="Y-axis")

    Explanation:

    • col: Changes the color of the points.
    • pch: Changes the plotting symbol.

  • Adding Grid Lines:

    # Scatter plot with grid lines
plot(x, y, main="Scatter Plot with Grid", xlab="X-axis", ylab="Y-axis")
grid()

    Explanation:

    • grid(): Adds grid lines to the plot.

  • Adding Legends:

    # Scatter plot with a legend
plot(x, y, col="blue", pch=19, main="Scatter Plot with Legend", xlab="X-axis", ylab="Y-axis")
legend("topleft", legend="Data Points", col="blue", pch=19)

    Explanation:

    • legend(): Adds a legend to the plot. The position ("topleft") and other parameters specify the legend's appearance.

Advanced Plotting Techniques

For more complex visualizations, base R provides additional functions and techniques.

  • Multiple Plots in One Figure:

    # Set up the plotting area to have 2 rows and 1 column
par(mfrow=c(2,1))

# First plot
plot(x, y, main="Plot 1")

# Second plot
plot(y, x, main="Plot 2")

    Explanation:

    • par(mfrow=c(2,1)): Divides the plotting area into a 2x1 grid.

  • Overlaying Plots:

    # Create the first plot
plot(x, y, type="l", col="blue", main="Overlaying Plots", xlab="X-axis", ylab="Y-axis")

# Add a second plot on top
lines(x, y/2, col="red")

    Explanation:

    • lines(): Adds a line plot to an existing plot.

Saving Plots

You can save your plots to various file formats for use in reports or presentations.

  • Saving to a PNG File:

    # Open a PNG device
png("plot.png")

# Create a plot
plot(x, y, main="Saved Plot", xlab="X-axis", ylab="Y-axis")

# Close the device
dev.off()

    Explanation:

    • png("plot.png"): Opens a PNG device to save the plot.
    • dev.off(): Closes the device and writes the file.

  • Saving to a PDF File:

    # Open a PDF device
pdf("plot.pdf")

# Create a plot
plot(x, y, main="Saved Plot", xlab="X-axis", ylab="Y-axis")

# Close the device
dev.off()

    Explanation:

    • pdf("plot.pdf"): Opens a PDF device to save the plot.

Conclusion

Base R graphics provide a robust set of tools for creating a wide range of visualizations. From basic plots to advanced customizations and saving options, understanding these tools is essential for effective data visualization in R. By mastering these techniques, you can create informative and visually appealing graphics that enhance your data analysis and presentation.
