Introduction to Regression

Regression is a fundamental concept in supervised learning, where the goal is to predict a continuous target variable based on one or more predictor variables. In this section, we will cover the basics of regression, focusing on how to implement and interpret regression models in R.

Key Concepts

  • Dependent Variable (Target): The variable we are trying to predict.
  • Independent Variables (Predictors): The variables used to make predictions.
  • Linear Regression: A method to model the relationship between the dependent variable and one or more independent variables by fitting a linear equation.
  • Residuals: The differences between the observed and predicted values.

Simple Linear Regression

Simple linear regression involves predicting a dependent variable using a single independent variable.

Example

Let's start with a simple example to understand how to perform linear regression in R.

# Load necessary library
library(ggplot2)

# Sample data
data <- data.frame(
  x = c(1, 2, 3, 4, 5),
  y = c(2, 4, 5, 4, 5)
)

# Plot the data
ggplot(data, aes(x = x, y = y)) +
  geom_point() +
  geom_smooth(method = "lm", se = FALSE) +
  labs(title = "Simple Linear Regression", x = "Independent Variable (x)", y = "Dependent Variable (y)")

Explanation

  • Data Preparation: We create a data frame with two variables x and y.
  • Visualization: We use ggplot2 to plot the data points and the linear regression line.

Fitting the Model

# Fit the linear model
model <- lm(y ~ x, data = data)

# Summary of the model
summary(model)

Explanation

  • Model Fitting: The lm function is used to fit a linear model.
  • Model Summary: The summary function provides detailed information about the model, including coefficients, R-squared value, and p-values.

Multiple Linear Regression

Multiple linear regression involves predicting a dependent variable using two or more independent variables.

Example

Let's extend our example to include multiple predictors.

# Sample data
data <- data.frame(
  x1 = c(1, 2, 3, 4, 5),
  x2 = c(5, 4, 3, 2, 1),
  y = c(2, 3, 4, 5, 6)
)

# Fit the multiple linear model
model <- lm(y ~ x1 + x2, data = data)

# Summary of the model
summary(model)

Explanation

  • Data Preparation: We create a data frame with three variables x1, x2, and y.
  • Model Fitting: The lm function is used to fit a multiple linear regression model.
  • Model Summary: The summary function provides detailed information about the model.

Model Evaluation

Evaluating the performance of a regression model is crucial to understand its predictive power.

Key Metrics

  • R-squared: Indicates the proportion of the variance in the dependent variable that is predictable from the independent variables.
  • Mean Squared Error (MSE): The average of the squares of the residuals.
  • Root Mean Squared Error (RMSE): The square root of the MSE, providing a measure of the average magnitude of the residuals.

Example

# Calculate predictions
predictions <- predict(model, data)

# Calculate residuals
residuals <- data$y - predictions

# Calculate MSE and RMSE
mse <- mean(residuals^2)
rmse <- sqrt(mse)

# Print the metrics
cat("MSE:", mse, "\n")
cat("RMSE:", rmse, "\n")

Explanation

  • Predictions: The predict function is used to generate predictions from the model.
  • Residuals: Calculated as the difference between the observed and predicted values.
  • MSE and RMSE: Calculated to evaluate the model's performance.

Conclusion

In this topic, we covered the basics of supervised learning with a focus on regression. We explored simple and multiple linear regression, learned how to fit models in R, and discussed key metrics for model evaluation. By understanding these concepts, you can start building and evaluating regression models to make predictions based on continuous data.
