Vectors in R
Matrices ►

Vectors are one of the most fundamental data structures in R. They are essential for storing and manipulating data. This section will cover everything from the basics of vectors to advanced operations.

Introduction to Vectors

  • Definition: A vector is a sequence of data elements of the same basic type. In R, vectors can be numeric, character, logical, or complex.
  • Creation: Vectors can be created using the c() function.

Creating Vectors

# Numeric vector
numeric_vector <- c(1, 2, 3, 4, 5)
print(numeric_vector)

# Character vector
char_vector <- c("apple", "banana", "cherry")
print(char_vector)

# Logical vector
logical_vector <- c(TRUE, FALSE, TRUE)
print(logical_vector)

Accessing Elements

  • Indexing: Elements in a vector can be accessed using square brackets [].
# Accessing the first element of a numeric vector
first_element <- numeric_vector[1]
print(first_element)

# Accessing multiple elements
subset_vector <- numeric_vector[1:3]
print(subset_vector)

Vector Operations

  • Arithmetic Operations: You can perform arithmetic operations on vectors element-wise.
# Adding two vectors
vector1 <- c(1, 2, 3)
vector2 <- c(4, 5, 6)
sum_vector <- vector1 + vector2
print(sum_vector)

# Multiplying a vector by a scalar
scalar_mult <- vector1 * 2
print(scalar_mult)
  • Logical Operations: Logical operations can also be performed element-wise.
# Logical comparison
logical_result <- vector1 > 2
print(logical_result)

Advanced Vector Operations

Vector Functions

  • Common Functions: R provides several built-in functions to work with vectors.
# Sum of elements
sum_vector <- sum(numeric_vector)
print(sum_vector)

# Mean of elements
mean_vector <- mean(numeric_vector)
print(mean_vector)

# Sorting a vector
sorted_vector <- sort(numeric_vector)
print(sorted_vector)

Vector Recycling

  • Recycling Rule: When performing operations on vectors of different lengths, R recycles the shorter vector to match the length of the longer one.
# Recycling example
short_vector <- c(1, 2)
long_vector <- c(10, 20, 30, 40)
recycled_sum <- short_vector + long_vector
print(recycled_sum)

Named Vectors

  • Naming Elements: You can assign names to the elements of a vector for better readability.
# Creating a named vector
named_vector <- c(a = 1, b = 2, c = 3)
print(named_vector)

# Accessing elements by name
element_b <- named_vector["b"]
print(element_b)

Conclusion

Vectors are a versatile and essential data structure in R. They allow for efficient storage and manipulation of data. Understanding how to create, access, and perform operations on vectors is crucial for any R programmer. As you advance, mastering vector functions, recycling, and naming will enable you to write more efficient and readable code.
