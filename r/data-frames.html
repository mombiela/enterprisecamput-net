Introduction to Data Frames

Data frames are a crucial data structure in R, used for storing tabular data. They are similar to tables in a database or Excel spreadsheets. Each column in a data frame can contain different types of data (numeric, character, factor, etc.), making them versatile for data analysis.

Key Concepts

  • Data Frame Structure: Rows represent observations, and columns represent variables.
  • Creating Data Frames: Using the data.frame() function.
  • Accessing Data: Using indexing, column names, and functions like subset().

Creating Data Frames

To create a data frame, you can use the data.frame() function. Here’s a simple example:

# Creating a data frame
df <- data.frame(
  Name = c("Alice", "Bob", "Charlie"),
  Age = c(25, 30, 35),
  Height = c(5.5, 6.0, 5.8)
)

# Display the data frame
print(df)

Explanation

  • Name: A character vector representing names.
  • Age: A numeric vector representing ages.
  • Height: A numeric vector representing heights.

Accessing Data in Data Frames

You can access data in a data frame using various methods:

By Index

# Accessing the first row
print(df[1, ])

# Accessing the first column
print(df[, 1])

# Accessing a specific element
print(df[1, 2])

By Column Name

# Accessing the 'Name' column
print(df$Name)

# Accessing the 'Age' column
print(df[["Age"]])

Using the `subset()` Function

# Subsetting data where Age is greater than 25
subset_df <- subset(df, Age > 25)
print(subset_df)

Modifying Data Frames

You can modify data frames by adding or removing columns and rows.

Adding Columns

# Adding a new column 'Weight'
df$Weight <- c(120, 150, 140)
print(df)

Removing Columns

# Removing the 'Height' column
df$Height <- NULL
print(df)

Adding Rows

# Adding a new row
new_row <- data.frame(Name = "David", Age = 40, Weight = 160)
df <- rbind(df, new_row)
print(df)

Removing Rows

# Removing the first row
df <- df[-1, ]
print(df)

Advanced Data Frame Operations

As you become more comfortable with data frames, you can perform more complex operations.

Merging Data Frames

You can merge two data frames using the merge() function.

# Creating another data frame
df2 <- data.frame(
  Name = c("Alice", "Bob", "David"),
  Salary = c(50000, 60000, 70000)
)

# Merging data frames by 'Name'
merged_df <- merge(df, df2, by = "Name")
print(merged_df)

Applying Functions to Data Frames

You can apply functions to data frames using apply(), lapply(), and sapply().

# Applying a function to each column
mean_values <- sapply(df[, -1], mean)
print(mean_values)

Summary

Data frames are a fundamental data structure in R, essential for data manipulation and analysis. They allow you to store and operate on tabular data efficiently. By understanding how to create, access, modify, and perform advanced operations on data frames, you can handle a wide range of data analysis tasks in R.

Key Takeaways

  • Data frames store tabular data with rows and columns.
  • You can create data frames using the data.frame() function.
  • Access data using indexing, column names, and the subset() function.
  • Modify data frames by adding/removing columns and rows.
  • Perform advanced operations like merging and applying functions to data frames.

By mastering these concepts, you'll be well-equipped to handle data analysis tasks in R.
