Overview

Machine Learning (ML) is a subset of artificial intelligence that involves the use of algorithms and statistical models to enable computers to perform tasks without explicit instructions. This course will guide you through the basics of machine learning using R, a powerful language for statistical computing and graphics.

What is Machine Learning?

  • Definition: Machine Learning is the study of algorithms that improve automatically through experience.
  • Types of Machine Learning:
    • Supervised Learning: Learning from labeled data (e.g., classification, regression).
    • Unsupervised Learning: Learning from unlabeled data (e.g., clustering, association).
    • Reinforcement Learning: Learning by interacting with an environment to maximize some notion of cumulative reward.

Setting Up R for Machine Learning

Before diving into machine learning, ensure you have R and RStudio installed. You will also need to install some essential packages.

Installing R and RStudio

  1. Download and install R from CRAN.
  2. Download and install RStudio from RStudio.

Installing Essential Packages

install.packages(c("tidyverse", "caret", "randomForest", "e1071"))

Data Preprocessing

Data preprocessing is a crucial step in machine learning. It involves cleaning and transforming raw data into a format suitable for modeling.

Loading Data

library(tidyverse)

data <- read.csv("path/to/your/dataset.csv")

Handling Missing Values

# Remove rows with missing values
data <- na.omit(data)

Feature Scaling

# Normalize the data
data_scaled <- scale(data)

Supervised Learning

Supervised learning involves training a model on labeled data. We will cover two main types: classification and regression.

Classification

Example: Predicting Species of Iris Flowers

# Load the dataset
data(iris)

# Split the data into training and testing sets
set.seed(123)
training_indices <- sample(1:nrow(iris), 0.7 * nrow(iris))
train_data <- iris[training_indices, ]
test_data <- iris[-training_indices, ]

# Train a random forest model
library(randomForest)
model <- randomForest(Species ~ ., data = train_data)

# Make predictions
predictions <- predict(model, test_data)

# Evaluate the model
confusion_matrix <- table(predictions, test_data$Species)
print(confusion_matrix)

Regression

Example: Predicting House Prices

# Load the dataset
data <- read.csv("path/to/house_prices.csv")

# Split the data into training and testing sets
set.seed(123)
training_indices <- sample(1:nrow(data), 0.7 * nrow(data))
train_data <- data[training_indices, ]
test_data <- data[-training_indices, ]

# Train a linear regression model
model <- lm(price ~ ., data = train_data)

# Make predictions
predictions <- predict(model, test_data)

# Evaluate the model
mse <- mean((predictions - test_data$price)^2)
print(mse)

Unsupervised Learning

Unsupervised learning involves finding hidden patterns in unlabeled data. We will cover clustering and dimensionality reduction.

Clustering

Example: K-means Clustering

# Load the dataset
data <- read.csv("path/to/your/dataset.csv")

# Perform K-means clustering
set.seed(123)
clusters <- kmeans(data, centers = 3)

# Add cluster assignments to the data
data$cluster <- clusters$cluster

# Visualize the clusters
library(ggplot2)
ggplot(data, aes(x = feature1, y = feature2, color = as.factor(cluster))) +
  geom_point() +
  labs(color = "Cluster")

Dimensionality Reduction

Example: Principal Component Analysis (PCA)

# Load the dataset
data <- read.csv("path/to/your/dataset.csv")

# Perform PCA
pca <- prcomp(data, scale. = TRUE)

# Visualize the first two principal components
pca_data <- data.frame(pca$x)
ggplot(pca_data, aes(x = PC1, y = PC2)) +
  geom_point()
