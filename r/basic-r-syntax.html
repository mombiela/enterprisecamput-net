Introduction

R is a powerful language for statistical computing and graphics. Understanding the basic syntax is crucial for writing efficient and readable R code. This section will cover the fundamental elements of R syntax, including variables, data types, operators, and basic functions.

Variables and Assignment

Variables in R are used to store data that can be referenced and manipulated. The assignment operator <- is commonly used to assign values to variables.

# Assigning a value to a variable
x <- 10
y <- 5
  • Assignment Operator: <- is preferred, but = can also be used.
  • Variable Names: Should start with a letter and can contain letters, numbers, and underscores.

Data Types

R supports various data types, including numeric, integer, character, logical, and complex.

# Numeric
num <- 3.14

# Integer
int <- 42L

# Character
char <- "Hello, R!"

# Logical
bool <- TRUE

# Complex
comp <- 2 + 3i

| Data Type | Example | Description | |-----------|---------------|-------------------------------| | Numeric | 3.14 | Decimal numbers | | Integer | 42L | Whole numbers with L suffix | | Character | "Hello, R!" | Text strings | | Logical | TRUE | Boolean values (TRUE, FALSE) | | Complex | 2 + 3i | Complex numbers |

Operators

Operators in R are used to perform arithmetic, comparison, and logical operations.

Arithmetic Operators

  • + : Addition
  • - : Subtraction
  • * : Multiplication
  • / : Division
  • ^ : Exponentiation
# Arithmetic operations
sum <- x + y
diff <- x - y
prod <- x * y
quot <- x / y
exp <- x ^ y

Comparison Operators

  • == : Equal to
  • != : Not equal to
  • < : Less than
  • > : Greater than
  • <= : Less than or equal to
  • >= : Greater than or equal to
# Comparison operations
is_equal <- x == y
is_not_equal <- x != y
is_less <- x < y
is_greater <- x > y

Logical Operators

  • & : AND
  • | : OR
  • ! : NOT
# Logical operations
and_op <- (x > 5) & (y < 10)
or_op <- (x > 5) | (y < 10)
not_op <- !(x > 5)

Basic Functions

Functions are used to perform specific tasks. R has many built-in functions, and you can also create your own.

Built-in Functions

  • print(): Prints output to the console.
  • sqrt(): Computes the square root.
  • abs(): Computes the absolute value.
# Using built-in functions
print("Hello, R!")
sqrt_val <- sqrt(16)
abs_val <- abs(-5)

User-defined Functions

You can create your own functions using the function keyword.

# Defining a function
add_numbers <- function(a, b) {
  result <- a + b
  return(result)
}

# Calling the function
sum_result <- add_numbers(3, 4)
print(sum_result)

Conclusion

Understanding the basic syntax of R is the first step towards mastering the language. This section covered the essentials, including variables, data types, operators, and basic functions. With these fundamentals, you can start writing simple R scripts and gradually move on to more complex tasks. Practice these concepts with exercises to reinforce your learning and build a strong foundation in R programming.
