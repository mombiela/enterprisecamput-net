Introduction

In this section, we will explore how to draw basic shapes using DirectX. This is fundamental for anyone looking to create graphics applications or games. We will start with simple shapes like lines and rectangles and gradually move to more complex shapes.

Setting Up DirectX

Before we start drawing shapes, ensure that you have DirectX set up in your development environment. If you haven't done this yet, refer to the setup guide provided in the course introduction.

Drawing a Line

Drawing a line is one of the simplest tasks in DirectX. We'll use the ID3D11DeviceContext::Draw method to render a line.

Key Concepts

  • Vertex Buffer: Stores the vertices of the shape.
  • Input Layout: Describes the vertex format to the GPU.
  • Shaders: Programs that run on the GPU to process vertices and pixels.

Code Example

// Define the vertices for a line
struct Vertex {
    DirectX::XMFLOAT3 position;
    DirectX::XMFLOAT4 color;
};

Vertex vertices[] = {
    { DirectX::XMFLOAT3(0.0f, 0.0f, 0.0f), DirectX::XMFLOAT4(1.0f, 0.0f, 0.0f, 1.0f) },
    { DirectX::XMFLOAT3(1.0f, 1.0f, 0.0f), DirectX::XMFLOAT4(0.0f, 1.0f, 0.0f, 1.0f) }
};

// Create the vertex buffer
D3D11_BUFFER_DESC bufferDesc = {};
bufferDesc.Usage = D3D11_USAGE_DEFAULT;
bufferDesc.ByteWidth = sizeof(vertices);
bufferDesc.BindFlags = D3D11_BIND_VERTEX_BUFFER;

D3D11_SUBRESOURCE_DATA initData = {};
initData.pSysMem = vertices;

ID3D11Buffer* vertexBuffer;
device->CreateBuffer(&bufferDesc, &initData, &vertexBuffer);

// Set the vertex buffer
UINT stride = sizeof(Vertex);
UINT offset = 0;
context->IASetVertexBuffers(0, 1, &vertexBuffer, &stride, &offset);

// Draw the line
context->Draw(2, 0);

Explanation

  • Vertex Structure: Defines the position and color of each vertex.
  • Vertex Buffer Creation: Allocates memory on the GPU and copies the vertex data.
  • Setting the Vertex Buffer: Binds the vertex buffer to the input assembler stage.
  • Drawing: Issues a draw call to render the line.

Drawing a Rectangle

Drawing a rectangle involves defining four vertices and two triangles.

Key Concepts

  • Index Buffer: Specifies the order in which vertices are used to form triangles.

Code Example

// Define the vertices for a rectangle
Vertex vertices[] = {
    { DirectX::XMFLOAT3(-0.5f, -0.5f, 0.0f), DirectX::XMFLOAT4(1.0f, 0.0f, 0.0f, 1.0f) },
    { DirectX::XMFLOAT3(-0.5f,  0.5f, 0.0f), DirectX::XMFLOAT4(0.0f, 1.0f, 0.0f, 1.0f) },
    { DirectX::XMFLOAT3( 0.5f,  0.5f, 0.0f), DirectX::XMFLOAT4(0.0f, 0.0f, 1.0f, 1.0f) },
    { DirectX::XMFLOAT3( 0.5f, -0.5f, 0.0f), DirectX::XMFLOAT4(1.0f, 1.0f, 0.0f, 1.0f) }
};

// Define the indices for the rectangle
unsigned short indices[] = {
    0, 1, 2,
    0, 2, 3
};

// Create the vertex buffer
D3D11_BUFFER_DESC bufferDesc = {};
bufferDesc.Usage = D3D11_USAGE_DEFAULT;
bufferDesc.ByteWidth = sizeof(vertices);
bufferDesc.BindFlags = D3D11_BIND_VERTEX_BUFFER;

D3D11_SUBRESOURCE_DATA initData = {};
initData.pSysMem = vertices;

ID3D11Buffer* vertexBuffer;
device->CreateBuffer(&bufferDesc, &initData, &vertexBuffer);

// Create the index buffer
bufferDesc.Usage = D3D11_USAGE_DEFAULT;
bufferDesc.ByteWidth = sizeof(indices);
bufferDesc.BindFlags = D3D11_BIND_INDEX_BUFFER;

initData.pSysMem = indices;

ID3D11Buffer* indexBuffer;
device->CreateBuffer(&bufferDesc, &initData, &indexBuffer);

// Set the vertex buffer
UINT stride = sizeof(Vertex);
UINT offset = 0;
context->IASetVertexBuffers(0, 1, &vertexBuffer, &stride, &offset);

// Set the index buffer
context->IASetIndexBuffer(indexBuffer, DXGI_FORMAT_R16_UINT, 0);

// Draw the rectangle
context->DrawIndexed(6, 0, 0);

Explanation

  • Vertices and Indices: Defines the rectangle's shape and how to connect the vertices.
  • Index Buffer Creation: Allocates memory on the GPU and copies the index data.
  • Setting Buffers: Binds the vertex and index buffers to the input assembler stage.
  • Drawing: Issues a draw call to render the rectangle using the indices.

Drawing a Circle

Drawing a circle is more complex and involves approximating the circle with a polygon.

Key Concepts

  • Circle Approximation: Using a polygon with many sides to approximate a circle.

Code Example

#include <vector>
#include <cmath>

// Define the number of segments
const int numSegments = 36;
const float radius = 1.0f;
std::vector<Vertex> vertices(numSegments + 1);

// Center vertex
vertices[0] = { DirectX::XMFLOAT3(0.0f, 0.0f, 0.0f), DirectX::XMFLOAT4(1.0f, 1.0f, 1.0f, 1.0f) };

// Define the vertices for the circle
for (int i = 1; i <= numSegments; ++i) {
    float angle = i * (2.0f * DirectX::XM_PI / numSegments);
    vertices[i] = { DirectX::XMFLOAT3(radius * cosf(angle), radius * sinf(angle), 0.0f), DirectX::XMFLOAT4(1.0f, 0.0f, 0.0f, 1.0f) };
}

// Create the vertex buffer
D3D11_BUFFER_DESC bufferDesc = {};
bufferDesc.Usage = D3D11_USAGE_DEFAULT;
bufferDesc.ByteWidth = sizeof(Vertex) * vertices.size();
bufferDesc.BindFlags = D3D11_BIND_VERTEX_BUFFER;

D3D11_SUBRESOURCE_DATA initData = {};
initData.pSysMem = vertices.data();

ID3D11Buffer* vertexBuffer;
device->CreateBuffer(&bufferDesc, &initData, &vertexBuffer);

// Set the vertex buffer
UINT stride = sizeof(Vertex);
UINT offset = 0;
context->IASetVertexBuffers(0, 1, &vertexBuffer, &stride, &offset);

// Draw the circle
context->Draw(vertices.size(), 0);

Explanation

  • Circle Vertices: Calculates the vertices around the circle's circumference.
  • Vertex Buffer Creation: Allocates memory on the GPU and copies the vertex data.
  • Setting the Vertex Buffer: Binds the vertex buffer to the input assembler stage.
  • Drawing: Issues a draw call to render the circle.

Conclusion

Drawing basic shapes in DirectX involves understanding vertex buffers, index buffers, and how to issue draw calls. By mastering these concepts, you can create more complex shapes and eventually build sophisticated graphics applications. Practice by experimenting with different shapes and configurations to deepen your understanding.
