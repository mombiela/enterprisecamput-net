Introduction

In this project, we will build a portfolio website using Vue.js. This project will help you consolidate your knowledge of Vue.js by applying it to a real-world scenario. By the end of this project, you will have a fully functional portfolio website to showcase your skills and projects.

Project Setup

Before we start building the portfolio website, let's set up our project.

  1. Install Vue CLI: If you haven't already installed Vue CLI, you can do so by running:

    npm install -g @vue/cli

  2. Create a New Vue Project: Create a new Vue project by running:

    vue create portfolio-website

  3. Navigate to the Project Directory:

    cd portfolio-website

  4. Run the Development Server:

    npm run serve

Project Structure

Understanding the project structure is crucial for efficient development.

  • src: Contains the main application code.
    • assets: Static assets like images and styles.
    • components: Reusable Vue components.
    • views: Different views or pages of the application.
    • router: Configuration for Vue Router.
    • store: Vuex store for state management (if needed).

Creating Components

We will create several components for our portfolio website:

  • Header: Contains the navigation menu.
  • Footer: Contains contact information and social media links.
  • Home: The landing page of the portfolio.
  • About: A page with information about yourself.
  • Projects: A page showcasing your projects.
  • Contact: A page with a contact form.

Routing

To navigate between different pages, we need to set up Vue Router.

  1. Install Vue Router:

    npm install vue-router

  2. Configure Vue Router in src/router/index.js:

    import Vue from 'vue'
import VueRouter from 'vue-router'
import Home from '../views/Home.vue'
import About from '../views/About.vue'
import Projects from '../views/Projects.vue'
import Contact from '../views/Contact.vue'

Vue.use(VueRouter)

const routes = [
  { path: '/', component: Home },
  { path: '/about', component: About },
  { path: '/projects', component: Projects },
  { path: '/contact', component: Contact }
]

const router = new VueRouter({
  mode: 'history',
  routes
})

export default router

  3. Import and Use Router in src/main.js:

    import Vue from 'vue'
import App from './App.vue'
import router from './router'

Vue.config.productionTip = false

new Vue({
  router,
  render: h => h(App)
}).$mount('#app')
