Introduction

In Vue.js, class and style bindings are powerful features that allow you to dynamically apply CSS classes and inline styles to your components. This topic will cover how to use these bindings effectively, from basic to advanced usage.

Class Bindings

Basic Class Binding

Vue.js allows you to bind classes to elements using the v-bind:class directive or its shorthand :class.

Example:

<div :class="{'active': isActive}">Hello, Vue!</div>

In this example, the active class will be applied to the div element if isActive is true.

Object Syntax

You can use an object to conditionally apply multiple classes.

Example:

<div :class="{'active': isActive, 'disabled': isDisabled}">Hello, Vue!</div>

Here, the active class will be applied if isActive is true, and the disabled class will be applied if isDisabled is true.

Array Syntax

You can also use an array to apply multiple classes.

Example:

<div :class="[classA, classB]">Hello, Vue!</div>

In this case, both classA and classB will be applied to the div element.

Computed Classes

You can use computed properties to dynamically generate class names.

Example:

<div :class="computedClass">Hello, Vue!</div>

<script>
export default {
  data() {
    return {
      isActive: true,
      isDisabled: false
    };
  },
  computed: {
    computedClass() {
      return {
        active: this.isActive,
        disabled: this.isDisabled
      };
    }
  }
};
</script>

This approach keeps your template clean and makes your class logic easier to manage.

Style Bindings

Basic Style Binding

Vue.js allows you to bind inline styles using the v-bind:style directive or its shorthand :style.

Example:

<div :style="{ color: activeColor, fontSize: fontSize + 'px' }">Hello, Vue!</div>

In this example, the div element will have its color and fontSize styles dynamically set based on the values of activeColor and fontSize.

Object Syntax

You can use an object to apply multiple styles.

Example:

<div :style="{ color: activeColor, fontSize: fontSize + 'px' }">Hello, Vue!</div>

Here, both color and fontSize styles are applied to the div element.

Array Syntax

You can also use an array to apply multiple style objects.

Example:

<div :style="[baseStyles, overridingStyles]">Hello, Vue!</div>

In this case, styles from both baseStyles and overridingStyles will be applied to the div element. If there are conflicting styles, the latter in the array will take precedence.

Computed Styles

You can use computed properties to dynamically generate styles.

Example:

<div :style="computedStyles">Hello, Vue!</div>

<script>
export default {
  data() {
    return {
      activeColor: 'red',
      fontSize: 14
    };
  },
  computed: {
    computedStyles() {
      return {
        color: this.activeColor,
        fontSize: this.fontSize + 'px'
      };
    }
  }
};
</script>

This approach keeps your template clean and makes your style logic easier to manage.

Summary

Class and style bindings in Vue.js provide a flexible and powerful way to dynamically apply CSS classes and inline styles to your components. By understanding and utilizing object and array syntax, as well as computed properties, you can create more dynamic and maintainable Vue.js applications.
