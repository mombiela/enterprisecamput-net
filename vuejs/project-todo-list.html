Introduction

In this project, we will build a simple To-Do List application using Vue.js. This project will help you understand the core concepts of Vue.js, including data binding, event handling, and component-based architecture.

Key Concepts

  • Vue Instance: The root of every Vue application.
  • Data Binding: Synchronizing data between the model and the view.
  • Event Handling: Responding to user actions.
  • Components: Reusable Vue instances with their own scope.
  • Directives: Special tokens in the markup that tell the library to do something to a DOM element.

Setting Up the Project

  1. Install Vue CLI: 
    npm install -g @vue/cli
  2. Create a New Project: 
    vue create todo-list
  3. Navigate to the Project Directory: 
    cd todo-list
  4. Run the Development Server: 
    npm run serve

Building the To-Do List Application

Creating the Vue Instance

Create a new Vue instance in src/main.js:

import Vue from 'vue';
import App from './App.vue';

Vue.config.productionTip = false;

new Vue({
  render: h => h(App),
}).$mount('#app');

Designing the Application Layout

In src/App.vue, set up the basic structure:

<template>
  <div id="app">
    <h1>To-Do List</h1>
    <AddTodo @add-todo="addTodo"></AddTodo>
    <TodoList :todos="todos" @delete-todo="deleteTodo"></TodoList>
  </div>
</template>

<script>
import AddTodo from './components/AddTodo.vue';
import TodoList from './components/TodoList.vue';

export default {
  name: 'App',
  components: {
    AddTodo,
    TodoList
  },
  data() {
    return {
      todos: []
    };
  },
  methods: {
    addTodo(todo) {
      this.todos.push(todo);
    },
    deleteTodo(index) {
      this.todos.splice(index, 1);
    }
  }
};
</script>

/* Add some basic styling */
</style>

Creating the AddTodo Component

Create a new file src/components/AddTodo.vue:

<template>
  <div>
    <input v-model="newTodo" @keyup.enter="submitTodo" placeholder="Add a new task" />
    <button @click="submitTodo">Add</button>
  </div>
</template>

<script>
export default {
  data() {
    return {
      newTodo: ''
    };
  },
  methods: {
    submitTodo() {
      if (this.newTodo.trim()) {
        this.$emit('add-todo', this.newTodo.trim());
        this.newTodo = '';
      }
    }
  }
};
</script>

<style scoped>
/* Add some basic styling */
</style>

Creating the TodoList Component

Create a new file src/components/TodoList.vue:

<template>
  <ul>
    <li v-for="(todo, index) in todos" :key="index">
      {{ todo }}
      <button @click="$emit('delete-todo', index)">Delete</button>
    </li>
  </ul>
</template>

<script>
export default {
  props: ['todos']
};
</script>

Conclusion

In this project, you have learned how to create a simple To-Do List application using Vue.js. You have explored the basics of Vue instances, data binding, event handling, and components. This foundational knowledge will be crucial as you move on to more advanced topics in Vue.js. Keep experimenting with the code and try adding new features to the To-Do List application to deepen your understanding.
