Introduction

In COBOL, data items are the fundamental units of data storage and manipulation. Understanding how to define, use, and manipulate data items is crucial for writing effective COBOL programs. This section will cover the basics of data items, including their types, declaration, and usage.

Data Item Types

COBOL supports various types of data items, each suited for different kinds of data. The primary types are:

  • Numeric Data Items: Used to store numbers.
  • Alphabetic Data Items: Used to store alphabetic characters.
  • Alphanumeric Data Items: Used to store a combination of letters, numbers, and special characters.

Numeric Data Items

Numeric data items are used to store numbers and can be defined with different levels of precision.

01 NUMERIC-VAR PIC 9(5).
  • 01 is the level number.
  • NUMERIC-VAR is the data item name.
  • PIC 9(5) specifies that the data item can hold a numeric value up to 5 digits long.

Alphabetic Data Items

Alphabetic data items are used to store alphabetic characters (A-Z).

01 ALPHA-VAR PIC A(10).
  • 01 is the level number.
  • ALPHA-VAR is the data item name.
  • PIC A(10) specifies that the data item can hold up to 10 alphabetic characters.

Alphanumeric Data Items

Alphanumeric data items can store a combination of letters, numbers, and special characters.

01 ALNUM-VAR PIC X(20).
  • 01 is the level number.
  • ALNUM-VAR is the data item name.
  • PIC X(20) specifies that the data item can hold up to 20 alphanumeric characters.

Declaring Data Items

Data items are declared in the Data Division of a COBOL program, specifically within the Working-Storage Section.

DATA DIVISION.
WORKING-STORAGE SECTION.
01 NUMERIC-VAR PIC 9(5).
01 ALPHA-VAR PIC A(10).
01 ALNUM-VAR PIC X(20).

Usage of Data Items

Data items are used in the Procedure Division to perform operations such as arithmetic calculations, string manipulations, and data movement.

Arithmetic Operations

You can perform arithmetic operations on numeric data items.

PROCEDURE DIVISION.
    ADD 5 TO NUMERIC-VAR.
    SUBTRACT 2 FROM NUMERIC-VAR.
    MULTIPLY NUMERIC-VAR BY 3.
    DIVIDE NUMERIC-VAR BY 2 GIVING RESULT-VAR.

String Manipulations

String manipulations can be performed on alphanumeric data items.

PROCEDURE DIVISION.
    MOVE "HELLO" TO ALPHA-VAR.
    MOVE "WORLD" TO ALNUM-VAR(6:5).
  • MOVE "HELLO" TO ALPHA-VAR. moves the string "HELLO" to ALPHA-VAR.
  • MOVE "WORLD" TO ALNUM-VAR(6:5). moves the string "WORLD" starting at position 6 for 5 characters in ALNUM-VAR.

Data Movement

Data movement involves transferring data from one data item to another.

PROCEDURE DIVISION.
    MOVE NUMERIC-VAR TO ANOTHER-NUMERIC-VAR.
    MOVE ALPHA-VAR TO ANOTHER-ALPHA-VAR.
    MOVE ALNUM-VAR TO ANOTHER-ALNUM-VAR.

Summary

In this section, we covered the basics of working with data items in COBOL. We discussed the different types of data items, how to declare them, and how to use them in various operations. Understanding these concepts is essential for effective COBOL programming, as data items are the building blocks for data manipulation and storage in your programs.
