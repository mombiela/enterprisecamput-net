Loops and Iterations in Scala
Pattern Matching ►

Introduction

Loops and iterations are fundamental concepts in programming that allow you to execute a block of code multiple times. In Scala, there are several ways to perform loops and iterations, each with its own use cases and advantages. This section will cover the basic to advanced looping constructs available in Scala.

Basic Loop Constructs

`while` Loop

The while loop repeatedly executes a block of code as long as a given condition is true.

Syntax

while (condition) {
  // code block to be executed
}

Example

var i = 0
while (i < 5) {
  println(s"i = $i")
  i += 1
}

In this example, the loop will print the value of i from 0 to 4.

`do-while` Loop

The do-while loop is similar to the while loop, but it guarantees that the code block is executed at least once.

Syntax

do {
  // code block to be executed
} while (condition)

Example

var i = 0
do {
  println(s"i = $i")
  i += 1
} while (i < 5)

This loop will also print the value of i from 0 to 4, but the code block is executed before the condition is checked.

For Loops

Basic `for` Loop

The for loop in Scala is more powerful and flexible compared to traditional for loops in other languages.

Syntax

for (i <- range) {
  // code block to be executed
}

Example

for (i <- 0 until 5) {
  println(s"i = $i")
}

This loop will print the value of i from 0 to 4.

Nested `for` Loop

You can nest for loops to iterate over multiple ranges.

Example

for (i <- 1 to 3; j <- 1 to 3) {
  println(s"i = $i, j = $j")
}

This will print all combinations of i and j from 1 to 3.

`for` Loop with Conditions (Guards)

You can add conditions to for loops to filter the iterations.

Example

for (i <- 1 to 10 if i % 2 == 0) {
  println(s"Even number: $i")
}

This loop will print only the even numbers between 1 and 10.

Advanced Loop Constructs

`for` Comprehensions

for comprehensions are a powerful feature in Scala that allow you to generate and transform collections.

Example

val numbers = List(1, 2, 3, 4, 5)
val doubled = for (n <- numbers) yield n * 2
println(doubled)  // Output: List(2, 4, 6, 8, 10)

In this example, the for comprehension creates a new list where each element is doubled.

Using `foreach`

The foreach method is used to apply a function to each element of a collection.

Example

val numbers = List(1, 2, 3, 4, 5)
numbers.foreach(n => println(n))

This will print each number in the list.

Using `map`

The map method transforms each element of a collection using a given function.

Example

val numbers = List(1, 2, 3, 4, 5)
val squared = numbers.map(n => n * n)
println(squared)  // Output: List(1, 4, 9, 16, 25)

This example creates a new list where each element is squared.

Conclusion

Loops and iterations are essential tools in any programmer's toolkit. Scala offers a variety of looping constructs, from basic while and for loops to advanced for comprehensions and collection methods like foreach and map. Understanding these constructs will help you write more efficient and readable code. Practice using these loops in different scenarios to become proficient in Scala programming.
