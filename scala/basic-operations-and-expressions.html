Introduction

In this section, we will cover the fundamental operations and expressions in Scala. Understanding these basics is crucial for writing effective Scala code. We will explore arithmetic operations, relational and logical operations, and how to use expressions to create more complex logic.

Arithmetic Operations

Arithmetic operations are the basic building blocks of any programming language. In Scala, you can perform standard arithmetic operations such as addition, subtraction, multiplication, division, and modulus.

Key Concepts

  • Addition (+): Adds two numbers.
  • Subtraction (-): Subtracts the second number from the first.
  • Multiplication (*): Multiplies two numbers.
  • Division (/): Divides the first number by the second.
  • Modulus (%): Returns the remainder of the division of the first number by the second.

Code Examples

val a = 10
val b = 3

val sum = a + b        // 13
val difference = a - b // 7
val product = a * b    // 30
val quotient = a / b   // 3
val remainder = a % b  // 1

Relational Operations

Relational operations are used to compare two values. The result of a relational operation is a Boolean value (true or false).

Key Concepts

  • Equal to (==): Checks if two values are equal.
  • Not equal to (!=): Checks if two values are not equal.
  • Greater than (>): Checks if the first value is greater than the second.
  • Less than (<): Checks if the first value is less than the second.
  • Greater than or equal to (>=): Checks if the first value is greater than or equal to the second.
  • Less than or equal to (<=): Checks if the first value is less than or equal to the second.

Code Examples

val x = 5
val y = 10

val isEqual = x == y       // false
val isNotEqual = x != y    // true
val isGreater = x > y      // false
val isLess = x < y         // true
val isGreaterOrEqual = x >= y // false
val isLessOrEqual = x <= y // true

Logical Operations

Logical operations are used to combine multiple Boolean expressions. The result of a logical operation is also a Boolean value.

Key Concepts

  • Logical AND (&&): Returns true if both expressions are true.
  • Logical OR (||): Returns true if at least one of the expressions is true.
  • Logical NOT (!): Inverts the Boolean value.

Code Examples

val a = true
val b = false

val andResult = a && b // false
val orResult = a || b  // true
val notResult = !a     // false

Expressions

Expressions are combinations of variables, operators, and values that produce a result. In Scala, almost everything is an expression, including if statements and code blocks.

Key Concepts

  • Simple Expressions: Direct calculations or evaluations.
  • Complex Expressions: Combinations of multiple operations and functions.

Code Examples

// Simple expression
val simpleExpression = 5 * 2 + 3 // 13

// Complex expression
val complexExpression = {
  val x = 10
  val y = 20
  x + y * 2 // 50
}

// Using if as an expression
val condition = true
val result = if (condition) "Yes" else "No" // "Yes"

Summary

In this section, we covered the basic operations and expressions in Scala. We explored arithmetic, relational, and logical operations, and how to use expressions to create more complex logic. Understanding these fundamentals is essential for progressing to more advanced topics in Scala programming. Practice these concepts with various examples to solidify your understanding.
