Introduction

Functions are fundamental building blocks in Scala, allowing you to encapsulate logic and reuse code efficiently. This section will guide you through the basics of defining and invoking functions, progressing from simple to more advanced concepts.

Defining Functions

Basic Function Definition

In Scala, a function is defined using the def keyword followed by the function name, parameter list, and return type.

def add(a: Int, b: Int): Int = {
  a + b
}
  • def: Keyword to define a function.
  • add: Function name.
  • (a: Int, b: Int): Parameter list with types.
  • : Int: Return type.
  • { a + b }: Function body.

Single Expression Functions

For functions with a single expression, you can omit the curly braces.

def subtract(a: Int, b: Int): Int = a - b

Functions Without Parameters

Functions can also be defined without parameters.

def greet(): String = "Hello, World!"

Functions Without Return Type

If a function does not return a value, its return type is Unit.

def printMessage(message: String): Unit = {
  println(message)
}

Invoking Functions

Basic Invocation

To invoke a function, use the function name followed by arguments in parentheses.

val sum = add(5, 3) // sum will be 8

Omitting Parentheses

For functions without parameters, you can omit the parentheses when invoking them.

val greeting = greet // greeting will be "Hello, World!"

Named Arguments

You can use named arguments to make the invocation clearer.

val result = add(a = 10, b = 5) // result will be 15

Default Parameters

Functions can have default parameter values.

def multiply(a: Int, b: Int = 2): Int = a * b

val product1 = multiply(4) // product1 will be 8
val product2 = multiply(4, 3) // product2 will be 12

Higher-Order Functions

Functions as Parameters

Scala supports higher-order functions, which can take other functions as parameters.

def applyOperation(a: Int, b: Int, operation: (Int, Int) => Int): Int = {
  operation(a, b)
}

val sum = applyOperation(5, 3, add) // sum will be 8

Anonymous Functions

You can pass anonymous functions (lambdas) as arguments.

val result = applyOperation(5, 3, (x, y) => x * y) // result will be 15

Recursion

Recursive Functions

Functions can call themselves, which is known as recursion.

def factorial(n: Int): Int = {
  if (n == 0) 1
  else n * factorial(n - 1)
}

val fact5 = factorial(5) // fact5 will be 120

Tail Recursion

Tail-recursive functions are optimized by the compiler to prevent stack overflow.

def factorialTailRec(n: Int, acc: Int = 1): Int = {
  if (n == 0) acc
  else factorialTailRec(n - 1, n * acc)
}

val fact5TailRec = factorialTailRec(5) // fact5TailRec will be 120

Conclusion

In this section, you learned how to define and invoke functions in Scala, including basic definitions, higher-order functions, and recursion. Understanding these concepts is crucial for writing efficient and reusable code. Practice by defining your own functions and experimenting with different types of function invocations to solidify your understanding.
