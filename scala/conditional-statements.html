Introduction

Conditional statements are fundamental in programming, allowing developers to execute different code blocks based on certain conditions. In Scala, conditional statements are similar to those in other programming languages but come with some unique features and syntax. This section will cover the basics of conditional statements in Scala, progressing to more advanced concepts.

Basic Conditional Statements

`if` Statement

The if statement is used to execute a block of code only if a specified condition is true.

Syntax

if (condition) {
  // code to be executed if the condition is true
}

Example

val x = 10
if (x > 5) {
  println("x is greater than 5")
}

`if-else` Statement

The if-else statement allows you to execute one block of code if the condition is true and another block if it is false.

Syntax

if (condition) {
  // code to be executed if the condition is true
} else {
  // code to be executed if the condition is false
}

Example

val x = 3
if (x > 5) {
  println("x is greater than 5")
} else {
  println("x is not greater than 5")
}

`if-else if-else` Ladder

When you have multiple conditions to check, you can use an if-else if-else ladder.

Syntax

if (condition1) {
  // code to be executed if condition1 is true
} else if (condition2) {
  // code to be executed if condition2 is true
} else {
  // code to be executed if both conditions are false
}

Example

val x = 7
if (x > 10) {
  println("x is greater than 10")
} else if (x > 5) {
  println("x is greater than 5 but less than or equal to 10")
} else {
  println("x is 5 or less")
}

Advanced Conditional Statements

Ternary-like `if` Expression

Scala does not have a ternary operator like some other languages (e.g., condition ? trueExpr : falseExpr), but you can achieve similar functionality using an if expression.

Syntax

val result = if (condition) trueExpr else falseExpr

Example

val x = 8
val result = if (x > 5) "Greater" else "Lesser"
println(result) // Output: Greater

Pattern Matching

Pattern matching is a powerful feature in Scala that can be used as an alternative to complex conditional statements.

Syntax

value match {
  case pattern1 => // code to execute if value matches pattern1
  case pattern2 => // code to execute if value matches pattern2
  case _ => // code to execute if value matches none of the above patterns
}

Example

val x = 3
x match {
  case 1 => println("x is 1")
  case 2 => println("x is 2")
  case 3 => println("x is 3")
  case _ => println("x is something else")
}

Guard Conditions in Pattern Matching

Guard conditions can be used in pattern matching to add additional conditions to a case.

Syntax

value match {
  case pattern if condition => // code to execute if pattern matches and condition is true
  case _ => // code to execute if no other cases match
}

Example

val x = 10
x match {
  case y if y > 5 => println("x is greater than 5")
  case _ => println("x is 5 or less")
}

Conclusion

Conditional statements in Scala provide a robust way to control the flow of your program. Starting from basic if statements to more advanced pattern matching, Scala offers a variety of tools to handle different scenarios effectively. Understanding and mastering these concepts will significantly enhance your ability to write efficient and readable Scala code.
