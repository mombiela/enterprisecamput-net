Using Filters in Google Analytics
Multi-Channel Funnels ►

Filters in Google Analytics are powerful tools that allow you to include, exclude, or modify the data that appears in your reports. This topic will guide you from the basics to advanced techniques for using filters effectively.

Introduction to Filters

Filters help you refine and customize the data in your Google Analytics account. They can be applied to views to control which data is included in your reports.

Key Concepts

  • Views: Different perspectives of your data, which can be customized using filters.
  • Predefined Filters: Simple filters provided by Google Analytics for common tasks.
  • Custom Filters: Advanced filters that allow for more specific data manipulation.

Creating and Applying Filters

Steps to Create a Filter

  1. Navigate to Admin: Go to the Admin section of your Google Analytics account.
  2. Select the View: Choose the view you want to apply the filter to.
  3. Create Filter: Click on 'Filters' and then 'Add Filter'.

Types of Filters

  • Predefined Filters:
    • Exclude/Include Traffic: Based on ISP domain, IP address, subdirectories, or hostname.
    • Lowercase/Uppercase: Convert data to lowercase or uppercase.
  • Custom Filters:
    • Exclude/Include: Exclude or include data based on specific criteria.
    • Lowercase/Uppercase: Convert specific fields to lowercase or uppercase.
    • Search and Replace: Modify data by searching for a pattern and replacing it.
    • Advanced: Combine fields and apply complex conditions.

Example: Excluding Internal Traffic

To exclude internal traffic from your reports, you can create a filter based on IP address.

  1. Navigate to Filters: Admin > View > Filters > Add Filter.
  2. Filter Type: Choose 'Predefined'.
  3. Select Filter Type: Exclude > traffic from the IP addresses > that are equal to.
  4. Enter IP Address: Input your internal IP address.
  5. Save: Click 'Save' to apply the filter.

Filter Type: Predefined
Select Filter Type: Exclude
Select Source or Destination: traffic from the IP addresses
Select Expression: that are equal to
IP Address: 123.456.789.0

Advanced Filtering Techniques

Custom Filters

Custom filters provide more flexibility and control over your data. They allow you to create complex conditions and transformations.

Example: Creating a Custom Filter to Include Only Mobile Traffic

  1. Navigate to Filters: Admin > View > Filters > Add Filter.
  2. Filter Type: Choose 'Custom'.
  3. Include: Select 'Include' from the filter type options.
  4. Filter Field: Choose 'Mobile Device (including tablet)'.
  5. Filter Pattern: Use a regular expression to match mobile devices.
Filter Name: Include Mobile Traffic
Filter Type: Custom
Select Filter Type: Include
Filter Field: Mobile Device (including tablet)
Filter Pattern: Yes

Combining Filters

You can combine multiple filters to refine your data further. For example, you can exclude internal traffic and include only mobile traffic.

  1. Create Filters: Follow the steps to create each filter individually.
  2. Apply Filters: Apply both filters to the same view.

Testing and Verifying Filters

Before applying filters to your main view, it's crucial to test them in a test view to ensure they work as expected.

Steps to Test Filters

  1. Create a Test View: Duplicate your main view and name it 'Test View'.
  2. Apply Filters: Apply the filters to the test view.
  3. Verify Data: Check the data in the test view to ensure the filters are working correctly.

Conclusion

Filters in Google Analytics are essential for customizing your data and ensuring that your reports are accurate and relevant. By understanding and applying both predefined and custom filters, you can gain deeper insights into your website's performance. Always remember to test filters in a test view before applying them to your main view to avoid data loss or inaccuracies.
