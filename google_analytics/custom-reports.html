Custom Reports in Google Analytics
Segmentation ►

Introduction

Custom Reports in Google Analytics allow you to tailor the data you see to fit your specific needs. This flexibility enables you to focus on the metrics and dimensions that are most relevant to your business goals.

Key Concepts

  • Metrics: Quantitative measurements such as sessions, users, and pageviews.
  • Dimensions: Qualitative attributes such as city, browser, and landing page.
  • Filters: Criteria to include or exclude data.
  • Views: Different perspectives of your data.

Creating a Custom Report

  1. Navigate to Customization: In the left-hand menu, click on "Customization" and then "Custom Reports."
  2. Click on + New Custom Report: This will open the report builder interface.
  3. Define Report Content:
    • Report Tab Name: Name your report tab.
    • Type: Choose between Explorer, Flat Table, Map Overlay, and Funnel.
    • Metrics: Select the metrics you want to include.
    • Dimensions: Select the dimensions you want to include.
    • Filters: Apply filters to refine your data.

Example: Creating a Simple Custom Report

Let's create a custom report that shows the number of sessions and pageviews by city.

1. **Report Tab Name**: City Performance
2. **Type**: Explorer
3. **Metrics**:
    - Sessions
    - Pageviews
4. **Dimensions**:
    - City
5. **Filters**: None

Advanced Custom Reports

For more advanced users, custom reports can include multiple tabs, complex filters, and calculated metrics.

Adding Multiple Tabs

You can add multiple tabs to a custom report to view different sets of data within the same report.

1. **First Tab**: City Performance
    - **Metrics**: Sessions, Pageviews
    - **Dimensions**: City
2. **Second Tab**: Device Performance
    - **Metrics**: Sessions, Bounce Rate
    - **Dimensions**: Device Category

Using Filters

Filters allow you to include or exclude specific data points. For example, you might want to exclude internal traffic from your custom report.

1. **Filter Name**: Exclude Internal Traffic
2. **Filter Type**: Exclude
3. **Dimension**: IP Address
4. **Condition**: Matches regex
5. **Value**: ^123\.456\.789\.0$

Calculated Metrics

Calculated metrics allow you to create new metrics based on existing ones. For example, you can create a metric for "Average Session Duration per User."

1. **Metric Name**: Avg. Session Duration per User
2. **Formula**: `{{Session Duration}} / {{Users}}`

Comparing Custom Reports and Standard Reports

| Feature | Standard Reports | Custom Reports | |------------------------|------------------|----------------| | Predefined Metrics | Yes | No | | Predefined Dimensions | Yes | No | | Customizable Layout | No | Yes | | Filters | Limited | Extensive | | Multiple Tabs | No | Yes | | Calculated Metrics | No | Yes |

Conclusion

Custom Reports in Google Analytics provide a powerful way to tailor your data analysis to your specific needs. By understanding how to create and configure custom reports, you can gain deeper insights into your website's performance and make more informed decisions. Whether you're a beginner or an advanced user, mastering custom reports will significantly enhance your analytics capabilities.
